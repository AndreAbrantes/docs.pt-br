# <a name="scaling-cloud-native-applications"></a>Dimensionamento de aplicativos nativos de nuvem

[!INCLUDE [book-preview](../../../includes/book-preview.md)]

Uma das vantagens mais frequentes de mudar para um ambiente de Hospedagem de nuvem é a escalabilidade. A escalabilidade ou a capacidade de um aplicativo de aceitar carga de usuário adicional sem indevidamente degradação do desempenho para cada usuário, geralmente é obtida com a divisão de aplicativos em pequenas partes que podem receber quaisquer recursos necessários. Neste capítulo, apresentamos as tecnologias que permitem que aplicativos nativos de nuvem sejam dimensionados para atender à demanda do usuário. Essas tecnologias incluem:

- Contêineres
- Orquestradores
- Computação sem servidor

# <a name="introduction-tothe-docker-application-life-cycle"></a>Introdução ao ciclo de vida de aplicativos do Docker

*O ciclo de vida das aplicações contêinerizadas é uma jornada que começa com o desenvolvedor. O desenvolvedor opta por implementar contêineres e Docker porque elimina atritos em implantações e operações de TI, o que, em última análise, ajuda a todos a serem mais ágeis, mais produtivos de ponta a ponta e mais rápido.*

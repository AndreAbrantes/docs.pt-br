# <a name="grpc-in-production"></a>gRPC em produção

ASP.NET Core aplicativos 3,0, incluindo os serviços gRPCs, podem ser executados no Windows, no Linux e em contêineres usando plataformas modernas, como Docker e kubernetes. Este capítulo explorará as várias opções para executar seus serviços gRPC em produção e examinará as opções de monitoramento e registro em log para garantir a operação ideal dos sistemas.

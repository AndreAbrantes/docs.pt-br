# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Quando implantar contêineres do Windows em sua infraestrutura de VM de IaaS local

- Sua organização pode não estar pronta para se mover para a nuvem, ou pode não ser capaz de se mover para a nuvem por uma razão comercial. Mas, você ainda pode obter os benefícios de usar o Windows Containers em seus próprios data centers.

- Você pode ter outros artefatos que estão sendo usados no local, e que podem atrasá-lo quando você tentar se mover para a nuvem. Por exemplo, dependências de segurança ou autenticação com o Windows Server Active Directory ou qualquer outro ativo no local.

- Se você começar a usar o Windows Containers hoje, você pode fazer uma migração em fases para a nuvem amanhã a partir de uma posição muito melhor. O Windows Containers está se tornando uma unidade de implantação para qualquer nuvem, sem bloqueio.

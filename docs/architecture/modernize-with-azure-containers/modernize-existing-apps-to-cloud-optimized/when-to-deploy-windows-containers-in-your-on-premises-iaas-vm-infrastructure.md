# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Quando implantar contêineres do Windows em sua infraestrutura de VM de IaaS local

- Sua organização pode não estar pronta para mudar para a nuvem ou talvez não consiga mudar para a nuvem por um motivo comercial. No entanto, você ainda pode obter os benefícios de usar contêineres do Windows em seus próprios data centers.

- Você pode ter outros artefatos que estão sendo usados localmente e que podem ficar lentos quando você tenta migrar para a nuvem. Por exemplo, dependências de segurança ou autenticação com o Windows Server Active Directory local ou qualquer outro ativo local.

- Se começar a usar os contêineres do Windows hoje, você poderá fazer uma migração em fases para a nuvem amanhã de uma posição muito melhor. Os contêineres do Windows estão se tornando uma unidade de implantação para qualquer nuvem, sem nenhum bloqueio.

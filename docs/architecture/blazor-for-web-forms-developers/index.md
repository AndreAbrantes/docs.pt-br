<span data-ttu-id="3509b-127">O .NET há muito apoia o desenvolvimento de aplicativos web através de ASP.NET, um conjunto abrangente de frameworks e ferramentas para a construção de qualquer tipo de class="sxs-lookup"><span data-stu-id="3509b-129">Frameworks like ASP.NET Web Forms, ASP.NET MVC, ASP.NET Web Pages, and more recently ASP.NET Core, provide a productive and powerful way to build data-stu-id="3509b-139">While Blazor shares many commonalities with ASP.NET Web Forms, like having a reusable component model and a simple way to handle user events, it also builds on the foundations <span data-ttu-id="3509b-140">Este livro apresenta ASP.NET desenvolvedores da Web Forms ao Blazor de uma forma familiar e conveniente.</span><span class="sxs-lookup"><span data-stu-id="3509b-140">This book introduces ASP.NET Web Forms developers to Blazor in a way that is familiar and convenient.</span></span> <span data-ttu-id="3509b-141">Introduz conceitos blazor em paralelo com conceitos análogos em ASP.NET Web Forms, ao mesmo tempo em que explica novos conceitos que podem ser menos familiares.</span><span class="sxs-lookup"><span data-stu-id="3509b-141">It introduces Blazor concepts in parallel with analogous concepts in ASP.NET Web Forms while also explaining new concepts that may be less familiar.</span></span> <span data-ttu-id="3509b-142">Ele abrange uma ampla gama de tópicos e preocupações, incluindo autoria de componentes, roteamento, layout, configuração e segurança.</span><span class="sxs-lookup"><span data-stu-id="3509b-142">It covers a broad range of topics and concerns including component authoring, routing, layout, configuration, and security.</span></span> <span data-ttu-id="3509b-143">E embora o conteúdo deste livro seja principalmente para permitir um novo desenvolvimento, ele também abrange diretrizes e estratégias para migrar ASP.NET Formulários web existentes para Blazor para quando você quiser modernizar um aplicativo existente.</span><span class="sxs-lookup"><span data-stu-id="3509b-143">And while the content of this book is primarily for enabling new development, it also covers guidelines and strategies for migrating existing ASP.NET Web Forms to Blazor for when you want to modernize an existing app.</span></span>

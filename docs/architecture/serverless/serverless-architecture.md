- <span data-ttu-id="09c9a-191">Acomoda a adição de novos pontos de extremidade para dar suporte a novos dispositivos e sensores.</span><span class="sxs-lookup"><span data-stu-id="09c9a-191">Accommodates adding new endpoints to support new devices and sensors.</span></span>

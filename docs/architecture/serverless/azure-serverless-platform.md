# <a name="azure-serverless-platform"></a>Plataforma sem servidor Azure

A plataforma sem servidor Azure inclui funções do Azure, aplicativos lógicos e grade de eventos. Esses serviços trabalham em conjunto e se conectam com uma infinidade de outros recursos. A plataforma sem servidor funciona com tudo, desde bancos de dados e armazenamento até análises e aprendizado de máquina/inteligência artificial.

Você também pode usar o Application Insights, uma plataforma sem servidor para capturar rastreamentos de diagnóstico e telemetria. Os Insights de aplicativos estão disponíveis para aplicativos de todos os tipos (desktop, mobile ou web), bem como implementações sem servidor. A plataforma é visualizada no seguinte diagrama:

![Plataforma sem servidor Azure](./media/azure-serverless-platform.png)

# <a name="azure-serverless-platform"></a>Plataforma sem servidor do Azure

A plataforma sem servidor do Azure inclui Azure Functions, aplicativos lógicos e grade de eventos. Esses serviços funcionam juntos e se conectam com inúmeros outros recursos. A plataforma sem servidor funciona com tudo, desde bancos de dados e armazenamento até análise e aprendizado de máquina/inteligência artificial.

Você também pode usar Application Insights, uma plataforma sem servidor para capturar rastreamentos de diagnóstico e telemetria. Application Insights estão disponíveis para aplicativos de todos os tipos (área de trabalho, móvel ou Web), bem como implementações sem servidor. A plataforma é visualizada no diagrama a seguir:

![Plataforma sem servidor do Azure](./media/azure-serverless-platform.png)

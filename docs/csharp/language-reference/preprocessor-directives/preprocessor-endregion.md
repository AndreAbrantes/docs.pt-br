# <a name="endregion-c-reference"></a>#endregion (Referência de C#)
O `#endregion` marca o fim de um bloco [#region](./preprocessor-region.md). Por exemplo:  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```

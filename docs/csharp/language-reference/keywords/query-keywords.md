# <a name="query-keywords-c-reference"></a>Palavras-chave de consulta (Referência de C#)

Esta seção contém as palavras-chave contextuais usadas em expressões de consulta.

## <a name="in-this-section"></a>Nesta seção

|Cláusula|Descrição|
|------------|-----------------|
|[De](from-clause.md)|Especifica uma fonte de dados e uma variável de intervalo (semelhante a uma variável de iteração).|
|[Onde](where-clause.md)|Filtra elementos de origem baseados em uma ou mais expressões boolianas separadas por operadores AND e OR lógicos ( `&&` ou <code>&#124;&#124;</code> ).|
|[Selecione](select-clause.md)|Especifica o tipo e a forma que os elementos na sequência retornada terão quando a consulta for executada.|
|[grupo](group-clause.md)|Agrupa os resultados da consulta de acordo com um valor de chave especificado.|
|[Em](into.md)|Fornece um identificador que pode funcionar como uma referência aos resultados de uma cláusula join, group ou select.|
|[Orderby](orderby-clause.md)|Classifica os resultados da consulta em ordem crescente ou decrescente com base no comparador padrão para o tipo de elemento.|
|[Juntar](join-clause.md)|Une duas fontes de dados com base em uma comparação de igualdade entre dois critérios de correspondência especificados.|
|[Deixe](let-clause.md)|Introduz uma variável de intervalo para armazenar os resultados de subexpressão em uma expressão de consulta.|
|[Em](in.md)|Palavra-chave contextual em uma cláusula [join](join-clause.md).|
|[Em](on.md)|Palavra-chave contextual em uma cláusula [join](join-clause.md).|
|[Equals](equals.md)|Palavra-chave contextual em uma cláusula [join](join-clause.md).|
|[Por](by.md)|Palavra-chave contextual em uma cláusula [group](group-clause.md).|
|[Crescente](ascending.md)|Palavra-chave contextual em uma cláusula [ordery](orderby-clause.md).|
|[Descendente](descending.md)|Palavra-chave contextual em uma cláusula [ordery](orderby-clause.md).|

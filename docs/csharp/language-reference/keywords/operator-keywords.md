# <a name="operator-keywords-c-reference"></a>Palavras-chave de operador (Referência de C#)

Utilizada para executar diversas ações, como a criação de objetos, a verificação do tipo de tempo de execução de um objeto, a obtenção do tamanho de um tipo e outras ações. Esta seção apresenta as seguintes palavras-chave:

- [as](as.md) Converte um objeto em um tipo compatível.

- [await](await.md) Suspende um método [assíncrono](async.md) até que uma tarefa esperada seja concluída.

- [is](is.md) Verifica o tipo de execução de um objeto, ou (começando com o C# 7.0) testa uma expressão em relação a um padrão.

- [new](new.md)

  - [Operador new](new-operator.md) Cria objetos.

  - [Modificador new](new-modifier.md) Oculta um membro herdado.

  - [Restrição new](new-constraint.md) Qualifica um parâmetro de tipo.

- [nameof](nameof.md) Obtém o nome de cadeia de caracteres simples (não qualificado) de uma variável, tipo ou membro.

- [sizeof](sizeof.md) Obtém o tamanho de um tipo não gerenciado.  

- [typeof](typeof.md) Obtém o objeto <xref:System.Type?displayProperty=nameWithType> de um tipo.  

- [true](true.md)  

  - [true Operator](true-false-operators.md) Retorna o valor [bool](bool.md) `true` para indicar que um operando é definitivamente true.

  - [true Literal](true-literal.md) Representa o valor [bool](bool.md) `true`.

- [false](false.md)  

  - [false Operator](true-false-operators.md) Retorna o valor [bool](bool.md) `true` para indicar que um operando é definitivamente false.

  - [false Literal](false-literal.md) Representa o valor [bool](bool.md) `false`.

- [stackalloc](stackalloc.md) Aloca um bloco de memória na pilha.  

As palavras-chave a seguir, que podem ser usadas como operadores e instruções, são abordadas na seção [Instruções](statement-keywords.md):

- [checked](checked.md) Especifica o contexto marcado.  

- [unchecked](unchecked.md) Especifica o contexto desmarcado.

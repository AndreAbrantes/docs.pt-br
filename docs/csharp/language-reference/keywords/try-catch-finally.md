# <a name="try-catch-finally-c-reference"></a>try-catch-finally (Referência de C#)
Um uso comum de `catch` e `finally` juntos é obter e usar recursos em um bloco `try`, lidar com circunstâncias excepcionais em um bloco `catch` e liberar os recursos no bloco `finally`.  
  
 Para obter mais informações e exemplos sobre como lançar exceções, consulte [try-catch](../../../csharp/language-reference/keywords/try-catch.md) e [Lançando exceções](../../../standard/exceptions/index.md). Para obter mais informações sobre o bloco `finally`, consulte [try-finally](../../../csharp/language-reference/keywords/try-finally.md).

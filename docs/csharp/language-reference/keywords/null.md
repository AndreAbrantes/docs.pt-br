# <a name="null-c-reference"></a>null (Referência de C#)

A palavra-chave `null` é um literal que representa uma referência nula, que não faz referência a qualquer objeto. `null` é o valor padrão de variáveis do tipo de referência. Os tipos de valor ordinários não podem ser nulos, exceto para [tipos de valor anulados.](../builtin-types/nullable-value-types.md)

# <a name="method-parameters-c-reference"></a>Parâmetros de método (Referência de C#)
Se um parâmetro for declarado para um método sem [ref](../../../csharp/language-reference/keywords/ref.md) ou [out](../../../csharp/language-reference/keywords/out.md), o parâmetro pode ter um valor associado a ele. Esse valor pode ser alterado no método, mas o valor alterado não será mantido quando o controle passá-lo de volta para o procedimento de chamada. É possível alterar esse comportamento usando uma palavra-chave de parâmetro de método.  
  
 Esta seção descreve as palavras-chave que podem ser usadas ao declarar parâmetros de método:  
  
-   [params](../../../csharp/language-reference/keywords/params.md)  
  
-   [ref](../../../csharp/language-reference/keywords/ref.md)  
  
-   [out](../../../csharp/language-reference/keywords/out.md)

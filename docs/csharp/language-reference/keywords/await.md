<span data-ttu-id="1f5e5-103">O operador `await` é aplicado a uma tarefa em um método assíncrono para inserir um ponto de suspensão na execução do método até que a tarefa aguardada seja concluída.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-103">The `await` operator is applied to a task in an asynchronous method to insert a suspension point in the execution of the method until the awaited task completes.</span></span> <span data-ttu-id="1f5e5-104">A tarefa representa um trabalho em andamento.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-104">The task represents ongoing work.</span></span>

<span data-ttu-id="1f5e5-109">A tarefa à qual o operador `await` é aplicado, normalmente é retornada por uma chamada a um método que implementa o [Padrão assíncrono baseado em tarefa](../../../standard/asynchronous-programming-patterns/task-based-asynchronous-pattern-tap.md).</span><span class="sxs-lookup"><span data-stu-id="1f5e5-109">The task to which the `await` operator is applied typically is returned by a call to a method that implements the [Task-Based Asynchronous Pattern](../../../standard/asynchronous-programming-patterns/task-based-asynchronous-pattern-tap.md).</span></span> <span data-ttu-id="1f5e5-110">Eles incluem métodos que retornam objetos <xref:System.Threading.Tasks.Task>, <xref:System.Threading.Tasks.Task%601> e `System.Threading.Tasks.ValueType<TResult>`.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-110">They include methods that return <xref:System.Threading.Tasks.Task>, <xref:System.Threading.Tasks.Task%601>, and `System.Threading.Tasks.ValueType<TResult>` objects.</span></span>  

  
 <span data-ttu-id="1f5e5-111">No exemplo a seguir, o método <xref:System.Net.Http.HttpClient.GetByteArrayAsync%2A?displayProperty=nameWithType> retorna um `Task<byte[]>`.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-111">In the following example, the <xref:System.Net.Http.HttpClient.GetByteArrayAsync%2A?displayProperty=nameWithType> method returns a `Task<byte[]>`.</span></span> <span data-ttu-id="1f5e5-112">A tarefa é uma promessa de produzir a matriz de bytes real quando a tarefa for concluída.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-112">The task is a promise to produce the actual byte array when the task is complete.</span></span> <span data-ttu-id="1f5e5-113">O operador `await` suspende a execução até que o trabalho do método <xref:System.Net.Http.HttpClient.GetByteArrayAsync%2A> esteja concluído.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-113">The `await` operator suspends execution until the work of the <xref:System.Net.Http.HttpClient.GetByteArrayAsync%2A> method is complete.</span></span> <span data-ttu-id="1f5e5-114">Enquanto isso, o controle é retornado ao chamador de `GetPageSizeAsync`.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-114">In the meantime, control is returned to the caller of `GetPageSizeAsync`.</span></span> <span data-ttu-id="1f5e5-115">Quando a tarefa termina a execução, a expressão `await` é avaliada para uma matriz de bytes.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-115">When the task finishes execution, the `await` expression evaluates to a byte array.</span></span>

<span data-ttu-id="1f5e5-122">Um expressão `await` não bloqueia o thread no qual ela está em execução.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-122">An `await` expression does not block the thread on which it is executing.</span></span> <span data-ttu-id="1f5e5-123">Em vez disso, ela faz com que o compilador inscreva o restante do método assíncrono como uma continuação da tarefa aguardada.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-123">Instead, it causes the compiler to sign up the rest of the async method as a continuation on the awaited task.</span></span> <span data-ttu-id="1f5e5-124">Em seguida, o controle retorna para o chamador do método assíncrono.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-124">Control then returns to the caller of the async method.</span></span> <span data-ttu-id="1f5e5-125">Quando a tarefa for concluída, ela invoca a sua continuação e a execução do método assíncrono continua de onde parou.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-125">When the task completes, it invokes its continuation, and execution of the async method resumes where it left off.</span></span>  
  
<span data-ttu-id="1f5e5-126">Uma expressão `await` só pode ocorrer no corpo do respectivo método delimitador, de uma expressão lambda ou de um método anônimo que deve estar marcado com um modificador `async`.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-126">An `await` expression can occur only in the body of its enclosing method, lambda expression, or anonymous method, which must be marked with an `async` modifier.</span></span> <span data-ttu-id="1f5e5-127">O termo *await* só serve como uma palavra-chave nesse contexto.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-127">The term *await* serves as a keyword only in that context.</span></span> <span data-ttu-id="1f5e5-128">Em outro local, ele será interpretado como um identificador.</span><span class="sxs-lookup"><span data-stu-id="1f5e5-128">Elsewhere, it is interpreted as an identifier.</span></span> <span data-ttu-id="1f5e5-129">No método, na expressão lambda ou no método anônimo, uma expressão `await` não pode ocorrer no corpo de uma função síncrona, em uma expressão de consulta, no bloco de uma [instrução lock](../../../csharp/language-reference/keywords/lock-statement.md) ou em um contexto [inseguro](../../../csharp/language-reference/keywords/unsafe.md).</span><span class="sxs-lookup"><span data-stu-id="1f5e5-129">Within the method, lambda expression, or anonymous method, an `await` expression cannot occur in the body of a synchronous function, in a query expression, in the block of a [lock statement](../../../csharp/language-reference/keywords/lock-statement.md), or in an [unsafe](../../../csharp/language-reference/keywords/unsafe.md) context.</span></span>  
  
## <a name="exceptions"></a><span data-ttu-id="1f5e5-130">Exceções</span><span class="sxs-lookup"><span data-stu-id="1f5e5-130">Exceptions</span></span>

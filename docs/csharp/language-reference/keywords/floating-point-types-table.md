# <a name="floating-point-types-table-c-reference"></a>Tabela de tipos de ponto flutuante (referência em C#)

A tabela a seguir mostra a precisão e os intervalos aproximados dos tipos de ponto flutuante.  
  
|Tipo|Intervalo aproximado|Precisão|  
|----------|-----------------------|---------------|  
|[float](float.md)|±1,5 x 10<sup>−45</sup> para ±3,4 x 10<sup>38</sup>|7 dígitos|  
|[double](double.md)|±5.0 × 10<sup>−324</sup> to ±1.7 × 10<sup>308</sup>|15 a 16 dígitos|  
|[decimal](decimal.md)|±1,0 x 10<sup>-28</sup> para ±7,9228 x 10<sup>28</sup>|28 a 29 dígitos|

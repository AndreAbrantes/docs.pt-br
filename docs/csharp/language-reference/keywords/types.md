# <a name="types-c-reference"></a>Tipos (Referência de C#)

O sistema de digitação do C# contém as seguintes categorias:

- [Tipos de valor](value-types.md)

- [Tipos de referência](reference-types.md)

- [Tipos de ponteiro](../../programming-guide/unsafe-code-pointers/pointer-types.md)

 Variáveis que são tipos de valor armazenam dados e as que são tipos de referência armazenam referências aos dados reais. As instâncias dos tipos de referência também são chamados de objetos. Os tipos de ponteiro podem ser usados somente em modo [unsafe](unsafe.md).

 É possível converter um tipo de valor em um tipo de referência e, novamente, em um tipo de valor usando [conversões boxing e unboxing](../../../csharp/programming-guide/types/boxing-and-unboxing.md). Com exceção de um tipo de valor demarcado, não é possível converter um tipo de referência em um tipo de valor.

 Esta seção também apresenta [nulo](void.md).

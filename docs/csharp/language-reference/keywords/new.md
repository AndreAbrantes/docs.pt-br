# <a name="new-c-reference"></a>new (Referência de C#)
No C#, a palavra-chave `new` pode ser usada como um operador, um modificador ou uma restrição.  
  
 [Operador new](../../../csharp/language-reference/keywords/new-operator.md)  
 Usado para criar objetos e invocar construtores.  
  
 [Modificador new](../../../csharp/language-reference/keywords/new-modifier.md)  
 Usado para ocultar um membro herdado de um membro da classe base.  
  
 [Restrição new](../../../csharp/language-reference/keywords/new-constraint.md)  
 Usado para restringir os tipos que podem ser usados como argumentos para um parâmetro de tipo em uma declaração genérica.

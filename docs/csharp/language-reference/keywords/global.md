# <a name="global-c-reference"></a>global (Referência de C#)

Quando a palavra-chave contextual `global` vem antes do [operador ::](../operators/namespace-alias-qualifer.md), refere-se ao namespace global, que é o namespace padrão para qualquer programa em C#, que não foi nomeado de outra maneira. Confira mais informações em [Como: usar o alias de namespace global](../../programming-guide/namespaces/how-to-use-the-global-namespace-alias.md).

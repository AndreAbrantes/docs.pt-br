# <a name="default-c-reference"></a>default (Referência de C#)

A palavra-chave `default` pode ser usada na instrução `switch` ou na expressão de valor padrão:

- [A instrução switch](switch.md): especifica o modo padrão.

- [Expressão de valor padrão](../../programming-guide/statements-expressions-operators/default-value-expressions.md): produz o valor padrão de um tipo.

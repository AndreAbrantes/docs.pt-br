# <a name="compiler-error-cs0413"></a><span data-ttu-id="29334-103">Erro do Compilador CS0413</span><span class="sxs-lookup"><span data-stu-id="29334-103">Compiler Error CS0413</span></span>
<span data-ttu-id="29334-104">O parâmetro de tipo 'type parameter' não pode ser usado com o operador 'as', pois não tem uma restrição de tipo de classe nem uma restrição 'class'</span><span class="sxs-lookup"><span data-stu-id="29334-104">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="29334-105">Esse erro ocorrerá se um tipo genérico usar o operador [as](../operators/type-testing-and-cast.md#as-operator), mas não tiver uma restrição de tipo de classe.</span><span class="sxs-lookup"><span data-stu-id="29334-105">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="29334-106">O operador `as` será permitido somente com tipos de referência e tipos de valor anuláveis, assim, o parâmetro de tipo deve estar restringido para assegurar que não é um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="29334-106">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="29334-107">Para evitar esse erro, use uma restrição de tipo de classe ou uma restrição de tipo de referência.</span><span class="sxs-lookup"><span data-stu-id="29334-107">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="29334-108">Isso ocorre porque o operador `as` pode retornar `null`, que não é um valor possível para um tipo de valor e o parâmetro de tipo deve ser tratado como um tipo de valor, a menos que ele seja uma restrição de tipo de classe ou uma restrição de tipo de referência.</span><span class="sxs-lookup"><span data-stu-id="29334-108">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="29334-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="29334-109">Example</span></span>

<span data-ttu-id="29334-110">O exemplo a seguir gera o erro CS0413.</span><span class="sxs-lookup"><span data-stu-id="29334-110">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```

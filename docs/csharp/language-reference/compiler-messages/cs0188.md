# <a name="compiler-error-cs0188"></a><span data-ttu-id="b6a3f-102">Erro do Compilador CS0188</span><span class="sxs-lookup"><span data-stu-id="b6a3f-102">Compiler Error CS0188</span></span>

<span data-ttu-id="b6a3f-103">O objeto 'this' não pode ser usado antes que todos os seus campos sejam atribuídos a</span><span class="sxs-lookup"><span data-stu-id="b6a3f-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="b6a3f-104">Todos os campos em um `struct` devem ser atribuídos por um construtor antes que ele possa chamar um método no `struct`.</span><span class="sxs-lookup"><span data-stu-id="b6a3f-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="b6a3f-105">Se você vir esse erro ao tentar inicializar uma propriedade em um construtor de struct, a solução será alterar o parâmetro de construtor para especificar o campo de suporte em vez da própria propriedade.</span><span class="sxs-lookup"><span data-stu-id="b6a3f-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="b6a3f-106">Propriedades autoimplementadas devem ser evitadas em struct, pois eles não têm nenhum campo de suporte e, portanto, não podem ser inicializados de forma alguma do construtor.</span><span class="sxs-lookup"><span data-stu-id="b6a3f-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="b6a3f-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="b6a3f-108">Example</span></span>

 <span data-ttu-id="b6a3f-109">O exemplo a seguir gera o erro CS0188:</span><span class="sxs-lookup"><span data-stu-id="b6a3f-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```

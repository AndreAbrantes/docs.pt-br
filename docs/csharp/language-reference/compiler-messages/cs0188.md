# <a name="compiler-error-cs0188"></a><span data-ttu-id="9974e-103">Erro do Compilador CS0188</span><span class="sxs-lookup"><span data-stu-id="9974e-103">Compiler Error CS0188</span></span>

 <span data-ttu-id="9974e-105">Todos os campos em um `struct` devem ser atribuídos por um construtor antes que ele possa chamar um método no `struct`.</span><span class="sxs-lookup"><span data-stu-id="9974e-105">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="9974e-106">Se você vir esse erro ao tentar inicializar uma propriedade em um construtor de struct, a solução será inicializar o campo de backup dessa propriedade diretamente, em vez da própria propriedade.</span><span class="sxs-lookup"><span data-stu-id="9974e-106">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="9974e-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9974e-107">Example</span></span>

 <span data-ttu-id="9974e-108">O exemplo a seguir gera o erro CS0188:</span><span class="sxs-lookup"><span data-stu-id="9974e-108">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```

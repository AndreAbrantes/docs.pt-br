## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="c6874-107">Exemplo de conversão ambígua</span><span class="sxs-lookup"><span data-stu-id="c6874-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="c6874-108">Esse erro pode ocorrer quando não há nenhuma conversão ou se as conversões implícitas disponíveis resultam em uma situação ambígua.</span><span class="sxs-lookup"><span data-stu-id="c6874-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="c6874-109">O exemplo a seguir gera o erro CS0151:</span><span class="sxs-lookup"><span data-stu-id="c6874-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="c6874-110">Exemplo de método void</span><span class="sxs-lookup"><span data-stu-id="c6874-110">Example of void method</span></span>

```csharp
    static void Main()
    {
        switch (M()) // CS0151
        {

                break;

    {
    }
}
```

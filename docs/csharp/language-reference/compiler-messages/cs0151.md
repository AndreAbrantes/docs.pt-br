# <a name="compiler-error-cs0151"></a><span data-ttu-id="a775a-102">Erro do Compilador CS0151</span><span class="sxs-lookup"><span data-stu-id="a775a-102">Compiler Error CS0151</span></span>
<span data-ttu-id="a775a-103">Um valor de tipo integral é esperado</span><span class="sxs-lookup"><span data-stu-id="a775a-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="a775a-104">Uma variável foi usada em uma situação em que um tipo de dados integral era necessário.</span><span class="sxs-lookup"><span data-stu-id="a775a-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="a775a-105">Para obter mais informações, consulte [Tipos](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="a775a-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="a775a-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a775a-106">Example</span></span>  
 <span data-ttu-id="a775a-107">Esse erro pode ocorrer quando não há nenhuma conversão ou se as conversões implícitas disponíveis resultam em uma situação ambígua.</span><span class="sxs-lookup"><span data-stu-id="a775a-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="a775a-108">O exemplo a seguir gera o erro CS0151.</span><span class="sxs-lookup"><span data-stu-id="a775a-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```

```csharp  
    }  
  
    static void M()  
    {  
    }  
}  
```

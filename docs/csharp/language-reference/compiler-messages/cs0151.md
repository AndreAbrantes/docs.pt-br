## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="ea934-106">Exemplo de conversão ambígua</span><span class="sxs-lookup"><span data-stu-id="ea934-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="ea934-107">Esse erro pode ocorrer quando não há nenhuma conversão ou se as conversões implícitas disponíveis resultam em uma situação ambígua.</span><span class="sxs-lookup"><span data-stu-id="ea934-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="ea934-108">O exemplo a seguir gera o erro CS0151:</span><span class="sxs-lookup"><span data-stu-id="ea934-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="ea934-109">Exemplo de método void</span><span class="sxs-lookup"><span data-stu-id="ea934-109">Example of void method</span></span>

<span data-ttu-id="ea934-110">Uma invocação do método [void](../../../csharp/language-reference/keywords/void.md) em uma expressão de correspondência [switch](../keywords/switch.md) gera o erro CS0151.</span><span class="sxs-lookup"><span data-stu-id="ea934-110">A [void](../../../csharp/language-reference/keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="ea934-111">Você pode corrigir o erro chamando um método que retorne um tipo integral, como [int](../../../csharp/language-reference/keywords/int.md) ou [long](../../../csharp/language-reference/keywords/long.md).</span><span class="sxs-lookup"><span data-stu-id="ea934-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/keywords/int.md) or [long](../../../csharp/language-reference/keywords/long.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```

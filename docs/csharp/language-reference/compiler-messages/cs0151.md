## <a name="example"></a><span data-ttu-id="e86b4-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e86b4-106">Example</span></span>  
 <span data-ttu-id="e86b4-107">Esse erro pode ocorrer quando não há nenhuma conversão ou se as conversões implícitas disponíveis resultam em uma situação ambígua.</span><span class="sxs-lookup"><span data-stu-id="e86b4-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="e86b4-108">O exemplo a seguir gera o erro CS0151.</span><span class="sxs-lookup"><span data-stu-id="e86b4-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```

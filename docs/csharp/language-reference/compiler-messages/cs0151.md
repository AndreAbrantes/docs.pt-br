## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="53e75-106">Exemplo de conversão ambígua</span><span class="sxs-lookup"><span data-stu-id="53e75-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="53e75-107">Esse erro pode ocorrer quando não há nenhuma conversão ou se as conversões implícitas disponíveis resultam em uma situação ambígua.</span><span class="sxs-lookup"><span data-stu-id="53e75-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="53e75-108">O exemplo a seguir gera o erro CS0151:</span><span class="sxs-lookup"><span data-stu-id="53e75-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="53e75-109">Exemplo de método void</span><span class="sxs-lookup"><span data-stu-id="53e75-109">Example of void method</span></span>

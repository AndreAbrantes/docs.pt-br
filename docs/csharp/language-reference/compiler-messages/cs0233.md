<span data-ttu-id="ece40-103">'identifier' não tem um tamanho predefinido, por isso, sizeof só pode ser usado em um contexto não seguro (considere o uso de System.Runtime.InteropServices.Marshal.SizeOf)</span><span class="sxs-lookup"><span data-stu-id="ece40-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>

## <a name="example"></a><span data-ttu-id="ece40-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ece40-107">Example</span></span>  
 <span data-ttu-id="ece40-108">O exemplo a seguir gera o erro CS0233:</span><span class="sxs-lookup"><span data-stu-id="ece40-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="9244f-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9244f-107">Example</span></span>  
 <span data-ttu-id="9244f-108">O exemplo a seguir gera o erro CS0233:</span><span class="sxs-lookup"><span data-stu-id="9244f-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```

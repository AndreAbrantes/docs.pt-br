Portanto, sizeof só poderá ser usado em contextos não seguros</span><span class="sxs-lookup"><span data-stu-id="3f10b-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="3f10b-105">Sem contextos [não seguros](../keywords/unsafe.md), o operador [sizeof](../operators/sizeof.md) só poderá ser usado para tipos cujo tamanho seja uma constante do tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="3f10b-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="3f10b-106">Se estiver recebendo esse erro, use um contexto não seguro.</span><span class="sxs-lookup"><span data-stu-id="3f10b-106">If you are getting this error, use an unsafe context.</span></span>

```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```

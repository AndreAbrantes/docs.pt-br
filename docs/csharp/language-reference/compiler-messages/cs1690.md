# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="06e65-102">Aviso do compilador (nível 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="06e65-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="06e65-103">O acesso a um membro em 'membro' pode gerar uma exceção em tempo de execução por ele ser um campo de uma classe de marshaling por referência</span><span class="sxs-lookup"><span data-stu-id="06e65-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="06e65-104">Este aviso ocorre quando você tentar chamar um método, uma propriedade ou um indexador em um membro de uma classe que deriva de <xref:System.MarshalByRefObject> e o membro é um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="06e65-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="06e65-105">Os objetos que herdam de `MarshalByRefObject` geralmente são destinados a realizar marshaling por referência em um domínio de aplicativo.</span><span class="sxs-lookup"><span data-stu-id="06e65-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="06e65-106">Se algum código tentar acessar diretamente o membro de tipo de valor desse tipo de objeto em um domínio de aplicativo, ocorrerá uma exceção em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="06e65-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="06e65-107">Para resolver o aviso, primeiro copie o membro para uma variável local e chame o método nessa variável.</span><span class="sxs-lookup"><span data-stu-id="06e65-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="06e65-108">O exemplo a seguir gera o erro CS1690:</span><span class="sxs-lookup"><span data-stu-id="06e65-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```

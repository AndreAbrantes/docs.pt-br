# <a name="compiler-error-cs0845"></a><span data-ttu-id="64c36-103">Erro do Compilador CS0845</span><span class="sxs-lookup"><span data-stu-id="64c36-103">Compiler Error CS0845</span></span>

<span data-ttu-id="64c36-104">Uma árvore de expressão lambda não pode conter um operador de união com um lado esquerdo literal nulo.</span><span class="sxs-lookup"><span data-stu-id="64c36-104">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="64c36-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="64c36-106">To correct this error</span></span>  
  
1. <span data-ttu-id="64c36-107">Converta o literal nulo para um objeto.</span><span class="sxs-lookup"><span data-stu-id="64c36-107">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="64c36-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="64c36-108">Example</span></span>  

 <span data-ttu-id="64c36-109">O código a seguir gera o erro CS0845:</span><span class="sxs-lookup"><span data-stu-id="64c36-109">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```

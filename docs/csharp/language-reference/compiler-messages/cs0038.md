<span data-ttu-id="7da31-103">Não é possível acessar um membro não estático do tipo externo 'type1' por meio do tipo aninhado 'type2'</span><span class="sxs-lookup"><span data-stu-id="7da31-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="7da31-104">Um campo em uma classe não está automaticamente disponível para uma classe aninhada.</span><span class="sxs-lookup"><span data-stu-id="7da31-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="7da31-105">Para estar disponível para uma classe aninhada, o campo deve ser [estático](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="7da31-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="7da31-106">Caso contrário, será necessário criar uma instância da classe externa.</span><span class="sxs-lookup"><span data-stu-id="7da31-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="7da31-107">Para obter mais informações, consulte [Tipos aninhados](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="7da31-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="7da31-108">O exemplo a seguir gera o erro CS0038:</span><span class="sxs-lookup"><span data-stu-id="7da31-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

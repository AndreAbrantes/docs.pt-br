 <span data-ttu-id="da952-104">O compilador requer uma conversão explícita.</span><span class="sxs-lookup"><span data-stu-id="da952-104">The compiler requires an explicit conversion.</span></span> <span data-ttu-id="da952-105">Por exemplo, talvez seja necessário converter um valor r para ser do mesmo tipo de valor l.</span><span class="sxs-lookup"><span data-stu-id="da952-105">For example, you may need to cast an r-value to be the same type as an l-value.</span></span> <span data-ttu-id="da952-106">De outro modo, é necessário fornecer rotinas de conversão para dar suporte a determinadas sobrecargas de operador.</span><span class="sxs-lookup"><span data-stu-id="da952-106">Or, you must provide conversion routines to support certain operator overloads.</span></span>  
  
 <span data-ttu-id="da952-107">As conversões devem ocorrer ao atribuir uma variável de um tipo a uma variável de um tipo diferente.</span><span class="sxs-lookup"><span data-stu-id="da952-107">Conversions must occur when assigning a variable of one type to a variable of a different type.</span></span> <span data-ttu-id="da952-108">Ao atribuir entre variáveis de tipos diferentes, o compilador deve converter o tipo ao lado direito do operador de atribuição para o tipo ao lado esquerdo do operador de atribuição.</span><span class="sxs-lookup"><span data-stu-id="da952-108">When making an assignment between variables of different types, the compiler must convert the type on the right-hand side of the assignment operator to the type on the left-hand side of the assignment operator.</span></span> <span data-ttu-id="da952-109">Use os códigos a seguir:</span><span class="sxs-lookup"><span data-stu-id="da952-109">Take the following the code:</span></span>

 <span data-ttu-id="da952-110">`i = lng;` faz uma atribuição, mas os tipos de dados das variáveis dos lados esquerdo e direito do operador de atribuição não são correspondentes.</span><span class="sxs-lookup"><span data-stu-id="da952-110">`i = lng;` makes an assignment, but the data types of the variables on the left and right-hand side of the assignment operator don't match.</span></span> <span data-ttu-id="da952-111">Antes de fazer a atribuição, o compilador converte implicitamente a variável `lng`, que é do tipo long, para um int.  <span data-ttu-id="da952-113">Uma conversão de estreitamento existirá ao converter para um tipo de dados que ocupa menos espaço de armazenamento na memória do que o tipo de dados que estão sendo convertidos.</span><span class="sxs-lookup"><span data-stu-id="da952-113">A narrowing conversion exists when converting to a data type that occupies less storage space in memory than the data type we are converting from.</span></span> <span data-ttu-id="da952-114">Por exemplo, converter um long para um int seria considerado uma conversão de estreitamento.</span><span class="sxs-lookup"><span data-stu-id="da952-114">For example, converting a long to an int would be considered a narrowing conversion.</span></span> <span data-ttu-id="da952-115">Um long ocupa 8 bytes de memória, enquanto um int ocupa 4 bytes.</span><span class="sxs-lookup"><span data-stu-id="da952-115">A long occupies 8 bytes of memory while an int occupies 4 bytes.</span></span> <span data-ttu-id="da952-116">Para saber como a perda de dados pode ocorrer, considere o seguinte exemplo:</span><span class="sxs-lookup"><span data-stu-id="da952-116">To see how data loss can occur, consider the following sample:</span></span>  

```csharp

 <span data-ttu-id="da952-117">Agora, a variável `lng` contém um valor que não pode ser armazenado na variável `i`, porque ele é muito grande.</span><span class="sxs-lookup"><span data-stu-id="da952-117">The variable `lng` now contains a value that cannot be stored in the variable `i` because it is too large.</span></span> <span data-ttu-id="da952-118">Se convertêssemos esse valor para um tipo int, perderíamos alguns de nossos dados e o valor convertido não seria o mesmo de antes da conversão.</span><span class="sxs-lookup"><span data-stu-id="da952-118">If we were to convert this value to an int type we would be losing some of our data and the converted value would not be the same as the value before the conversion.</span></span>  
  
 <span data-ttu-id="da952-119">Uma conversão de expansão seria o oposto de uma conversão de estreitamento.</span><span class="sxs-lookup"><span data-stu-id="da952-119">A widening conversion would be the opposite of a narrowing conversion.</span></span> <span data-ttu-id="da952-120">Nas conversões de expansão, converte-se para um tipo de dados que ocupa mais espaço de armazenamento na memória do que o tipo de dados que estão sendo convertidos.</span><span class="sxs-lookup"><span data-stu-id="da952-120">With widening conversions, we are converting to a data type that occupies more storage space in memory than the data type we are converting from.</span></span> <span data-ttu-id="da952-121">Aqui está um exemplo de uma conversão de expansão:</span><span class="sxs-lookup"><span data-stu-id="da952-121">Here is an example of a widening conversion:</span></span>

```

 <span data-ttu-id="da952-122">Observe a diferença entre este exemplo de código e o primeiro.</span><span class="sxs-lookup"><span data-stu-id="da952-122">Notice the difference between this code sample and the first.</span></span> <span data-ttu-id="da952-123">Dessa vez, a variável `lng` está à esquerda do operador de atribuição, assim, ela será o destino da atribuição.</span><span class="sxs-lookup"><span data-stu-id="da952-123">This time the variable `lng` is on the left-hand side of the assignment operator, so it is the target of our assignment.</span></span> <span data-ttu-id="da952-124">Antes de realizar a atribuição, o compilador deve converter implicitamente a variável `i`, que é do tipo int, para o tipo long.</span><span class="sxs-lookup"><span data-stu-id="da952-124">Before the assignment can be made, the compiler must implicitly convert the variable `i`, which is of type int, to type long.</span></span> <span data-ttu-id="da952-125">Essa é uma conversão de expansão, pois a conversão é de um tipo que ocupa 4 bytes de memória (um int) para um tipo que ocupa 8 bytes de memória (um long).</span><span class="sxs-lookup"><span data-stu-id="da952-125">This is a widening conversion since we are converting from a type that occupies 4 bytes of memory (an int) to a type that occupies 8 bytes of memory (a long).</span></span> <span data-ttu-id="da952-126">Conversões de expansão implícitas são permitidas porque não há nenhuma perda de dados.</span><span class="sxs-lookup"><span data-stu-id="da952-126">Implicit widening conversions are allowed because there is no potential loss of data.</span></span> <span data-ttu-id="da952-127">Qualquer valor que pode ser armazenado em um int também pode ser armazenado em um long.</span><span class="sxs-lookup"><span data-stu-id="da952-127">Any value that can be stored in an int can also be stored in a long.</span></span>

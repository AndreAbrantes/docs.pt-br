CS0019 also is generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="94353-110">O operador `+` de adição pode ser usado com operands de corda porque esse operador está sobrecarregado pela `String` classe para executar a concatenação de cordas.</span><span class="sxs-lookup"><span data-stu-id="94353-110">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp

{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```

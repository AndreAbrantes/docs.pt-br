# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="158f2-102">Aviso do compilador (nível 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="158f2-102">Compiler Warning (level 2) CS0108</span></span>
<span data-ttu-id="158f2-103">'member1' oculta o membro herdado 'member2'.</span><span class="sxs-lookup"><span data-stu-id="158f2-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="158f2-104">Use a nova palavra-chave se a ocultação foi intencional.</span><span class="sxs-lookup"><span data-stu-id="158f2-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="158f2-105">Uma variável foi declarada com o mesmo nome que uma variável em uma classe base.</span><span class="sxs-lookup"><span data-stu-id="158f2-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="158f2-106">No entanto, a palavra-chave [new](../../../csharp/language-reference/keywords/new.md) não foi usada.</span><span class="sxs-lookup"><span data-stu-id="158f2-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="158f2-107">Este aviso informa **new** deve ser usada; a variável será declarada como se **new** tivesse sido usada na declaração.</span><span class="sxs-lookup"><span data-stu-id="158f2-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="158f2-108">O exemplo a seguir gera o erro CS0108:</span><span class="sxs-lookup"><span data-stu-id="158f2-108">The following sample generates CS0108:</span></span>  
  
```csharp  
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```

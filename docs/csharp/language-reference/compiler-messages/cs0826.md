## <a name="example"></a><span data-ttu-id="51299-112">Exemplo</span><span class="sxs-lookup"><span data-stu-id="51299-112">Example</span></span>  
 <span data-ttu-id="51299-113">O código a seguir gera o erro CS0826 porque os elementos da matriz não são todos do mesmo tipo e a lógica de inferência de tipos do compilador não encontra um único tipo melhor:</span><span class="sxs-lookup"><span data-stu-id="51299-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```

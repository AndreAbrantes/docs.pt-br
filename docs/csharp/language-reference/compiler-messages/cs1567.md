# <a name="compiler-error-cs1567"></a>Erro do Compilador CS1567
Erro ao gerar o recurso do Win32: 'file'  
  
 A compilação usou a opção do compilador [/win32icon](../compiler-options/win32icon-compiler-option.md) ou não usou [/win32res](../compiler-options/win32res-compiler-option.md), o que faz com que o compilador gere um arquivo que contém informações de recurso, porém, o compilador não pôde criar o arquivo em razão do espaço em disco insuficiente ou algum outro erro.  
  
 Se não for possível resolver o problema de geração de arquivo, será possível usar [/win32res](../compiler-options/win32res-compiler-option.md), que não gera um arquivo que contém informações sobre o recurso.

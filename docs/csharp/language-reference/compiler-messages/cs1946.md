# <a name="compiler-error-cs1946"></a><span data-ttu-id="b53f0-102">Erro do Compilador CS1946</span><span class="sxs-lookup"><span data-stu-id="b53f0-102">Compiler Error CS1946</span></span>

<span data-ttu-id="b53f0-103">Uma expressão de método anônimo não pode ser convertida em uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="b53f0-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="b53f0-104">Um [método anônimo](../operators/delegate-operator.md) representa um conjunto de instruções, mas uma árvore de expressão não deve conter uma instrução.</span><span class="sxs-lookup"><span data-stu-id="b53f0-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="b53f0-105">Portanto, um método anônimo não pode ser representado por uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="b53f0-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="b53f0-106">Para corrigir esse erro, altere o método anônimo para uma [expressão lambda](../operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="b53f0-106">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="b53f0-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="b53f0-107">Example</span></span>

<span data-ttu-id="b53f0-108">O exemplo a seguir gera o erro CS1946:</span><span class="sxs-lookup"><span data-stu-id="b53f0-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```

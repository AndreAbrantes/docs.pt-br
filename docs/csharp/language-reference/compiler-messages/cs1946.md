# <a name="compiler-error-cs1946"></a><span data-ttu-id="fd998-102">Erro do Compilador CS1946</span><span class="sxs-lookup"><span data-stu-id="fd998-102">Compiler Error CS1946</span></span>
<span data-ttu-id="fd998-103">Uma expressão de método anônimo não pode ser convertida em uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="fd998-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
 <span data-ttu-id="fd998-104">Um método anônimo representa um conjunto de instruções, mas uma árvore de expressão não deve conter uma instrução.</span><span class="sxs-lookup"><span data-stu-id="fd998-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="fd998-105">Portanto, um método anônimo não pode ser representado por uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="fd998-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fd998-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="fd998-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="fd998-107">Altere o método anônimo para uma expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="fd998-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fd998-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="fd998-108">Example</span></span>  
 <span data-ttu-id="fd998-109">O exemplo a seguir gera o erro CS1946:</span><span class="sxs-lookup"><span data-stu-id="fd998-109">The following example generates CS1946:</span></span>

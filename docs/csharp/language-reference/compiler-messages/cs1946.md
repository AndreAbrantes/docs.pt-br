# <a name="compiler-error-cs1946"></a><span data-ttu-id="e527c-102">Erro do Compilador CS1946</span><span class="sxs-lookup"><span data-stu-id="e527c-102">Compiler Error CS1946</span></span>

<span data-ttu-id="e527c-103">Uma expressão de método anônimo não pode ser convertida em uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="e527c-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="e527c-104">Um método anônimo representa um conjunto de instruções, mas uma árvore de expressão não deve conter uma instrução.</span><span class="sxs-lookup"><span data-stu-id="e527c-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="e527c-105">Portanto, um método anônimo não pode ser representado por uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="e527c-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="e527c-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="e527c-106">To correct this error</span></span>

1. <span data-ttu-id="e527c-107">Altere o método anônimo para uma expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="e527c-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e527c-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e527c-108">Example</span></span>

<span data-ttu-id="e527c-109">O exemplo a seguir gera o erro CS1946:</span><span class="sxs-lookup"><span data-stu-id="e527c-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```

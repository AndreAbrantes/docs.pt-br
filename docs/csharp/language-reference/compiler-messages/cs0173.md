## <a name="example"></a><span data-ttu-id="70817-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="70817-110">Example</span></span>

<span data-ttu-id="70817-111">Os exemplos a seguir geram o erro do compilador CS0173:</span><span class="sxs-lookup"><span data-stu-id="70817-111">The following examples generate compiler error CS0173:</span></span>

```csharp
public class C {}

public class A
{
    // The following code defines an implicit conversion operator from
    // type C to type A.
    //public static implicit operator A(C c)
    //{
    //    A a = new A();
    //    a = c;
    //    return a;
    //}
}

public class MyClass
{
    public static void F(bool b)
    {
        A a = new A();
        C c = new C();

        // The following line causes CS0173 because there is no implicit
        // conversion from a to c or from c to a.
        object o = b ? a : c;

        // To resolve the error, you can cast a and c.
        // object o = b ? (object)a : (object)c;

        // Alternatively, you can add a conversion operator from class C to
        // class A, or from class A to class C, but not both.
    }

   public static void Main()
   {
      F(true);
   }
}
```

```csharp
class M
{
    static int Main ()

        // The following line causes CS0173.
        object o = (X == 0) ? null : null;

    }
}
```

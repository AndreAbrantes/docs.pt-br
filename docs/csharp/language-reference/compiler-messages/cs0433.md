## <a name="example"></a><span data-ttu-id="2e4b9-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2e4b9-110">Example</span></span>  
 <span data-ttu-id="2e4b9-111">O exemplo a seguir gera o erro CS0433.</span><span class="sxs-lookup"><span data-stu-id="2e4b9-111">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="2e4b9-112">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2e4b9-112">Example</span></span>

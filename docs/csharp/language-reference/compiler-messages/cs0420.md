# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="3d34d-103">Aviso do compilador (nível 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="3d34d-103">Compiler Warning (level 1) CS0420</span></span>

<span data-ttu-id="3d34d-104">'identifier': uma referência a um campo volátil não será tratada como volátil</span><span class="sxs-lookup"><span data-stu-id="3d34d-104">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="3d34d-105">Normalmente, um campo volátil não deve ser passado usando os parâmetros `ref` ou **out**, uma vez que ele não será tratado como volátil dentro do escopo da função.</span><span class="sxs-lookup"><span data-stu-id="3d34d-105">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="3d34d-106">Há exceções, como ao chamar uma API interligada.</span><span class="sxs-lookup"><span data-stu-id="3d34d-106">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="3d34d-107">Como com qualquer aviso, é possível usar o [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) para desabilitar esse aviso naqueles casos raros em que um campo volátil está sendo usado como um parâmetro de referência intencionalmente.</span><span class="sxs-lookup"><span data-stu-id="3d34d-107">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="3d34d-108">O exemplo a seguir gera o erro CS0420:</span><span class="sxs-lookup"><span data-stu-id="3d34d-108">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```

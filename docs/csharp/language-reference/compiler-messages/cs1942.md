# <a name="compiler-error-cs1942"></a><span data-ttu-id="be3c0-103">Erro do Compilador CS1942</span><span class="sxs-lookup"><span data-stu-id="be3c0-103">Compiler Error CS1942</span></span>

<span data-ttu-id="be3c0-104">O tipo de uma das expressões na cláusula 'clause' está incorreto.</span><span class="sxs-lookup"><span data-stu-id="be3c0-104">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="be3c0-105">A inferência de tipos falhou na chamada para 'method'.</span><span class="sxs-lookup"><span data-stu-id="be3c0-105">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="be3c0-106">Normalmente, esse erro é gerado quando a variável de intervalo tem um tipo explícito incorreto.</span><span class="sxs-lookup"><span data-stu-id="be3c0-106">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="be3c0-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="be3c0-107">To correct this error</span></span>

1. <span data-ttu-id="be3c0-108">Se a variável de intervalo for tipada explicitamente, certifique-se de que o tipo é igual ou implicitamente conversível do tipo dos elementos na coleção em que a iteração é realizada.</span><span class="sxs-lookup"><span data-stu-id="be3c0-108">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="be3c0-109">Se a variável de intervalo for precedida pela palavra-chave `var`, remova `var`.</span><span class="sxs-lookup"><span data-stu-id="be3c0-109">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="be3c0-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="be3c0-110">Example</span></span>

<span data-ttu-id="be3c0-111">O código a seguir gera o erro CS1942:</span><span class="sxs-lookup"><span data-stu-id="be3c0-111">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="be3c0-112">O CS1942 está relacionado ao CS1949, pois o uso do `var` com uma variável de intervalo faz com que a operação `Cast<T>` subjacente falhe, pois `var` não é um tipo.</span><span class="sxs-lookup"><span data-stu-id="be3c0-112">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>

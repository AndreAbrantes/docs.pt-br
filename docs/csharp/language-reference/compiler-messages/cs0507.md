# <a name="compiler-error-cs0507"></a><span data-ttu-id="5b4d0-102">Erro do Compilador CS0507</span><span class="sxs-lookup"><span data-stu-id="5b4d0-102">Compiler Error CS0507</span></span>

 <span data-ttu-id="5b4d0-104">Houve uma tentativa de alterar a especificação de acesso em uma substituição de método.</span><span class="sxs-lookup"><span data-stu-id="5b4d0-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5b4d0-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5b4d0-105">Example</span></span>  
 <span data-ttu-id="5b4d0-106">O exemplo a seguir gera o erro CS0507.</span><span class="sxs-lookup"><span data-stu-id="5b4d0-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="5b4d0-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5b4d0-107">Example</span></span>  
 <span data-ttu-id="5b4d0-108">O erro CS0507 também poderá ocorrer se uma classe tentar substituir um método marcado como `protected internal`, conforme definido nos metadados referenciados.</span><span class="sxs-lookup"><span data-stu-id="5b4d0-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="5b4d0-109">Nessa situação, o método de substituição deve ser marcado como `protected`.</span><span class="sxs-lookup"><span data-stu-id="5b4d0-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="5b4d0-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5b4d0-110">Example</span></span>  
 <span data-ttu-id="5b4d0-111">O exemplo a seguir gera o erro CS0507.</span><span class="sxs-lookup"><span data-stu-id="5b4d0-111">The following sample generates CS0507.</span></span>

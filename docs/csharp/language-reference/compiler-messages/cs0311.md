# <a name="compiler-error-cs0311"></a><span data-ttu-id="1da11-103">Erro do Compilador CS0311</span><span class="sxs-lookup"><span data-stu-id="1da11-103">Compiler Error CS0311</span></span>

<span data-ttu-id="1da11-104">O tipo ' type1 ' não pode ser usado como parâmetro de tipo ' T' no tipo ou método genérico ' \<name> '.</span><span class="sxs-lookup"><span data-stu-id="1da11-104">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="1da11-105">Não há nenhuma conversão de referência implícita de 'type1' para 'type2'.</span><span class="sxs-lookup"><span data-stu-id="1da11-105">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="1da11-106">Quando uma restrição é aplicada a um parâmetro de tipo genérico, uma conversão implícita de identidade ou referência deve existir do argumento concreto para o tipo de restrição.</span><span class="sxs-lookup"><span data-stu-id="1da11-106">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1da11-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="1da11-107">To correct this error</span></span>  
  
1. <span data-ttu-id="1da11-108">Altere o argumento usado para criar a classe.</span><span class="sxs-lookup"><span data-stu-id="1da11-108">Change the argument you are using to create the class.</span></span>

## <a name="example"></a><span data-ttu-id="1da11-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="1da11-111">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="1da11-112">Caso esse erro ocorra ao tentar usar um argumento de tipo de valor, observe que uma conversão numérica implícita, por exemplo, de `short` para `int`, não atende a um parâmetro de tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="1da11-112">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>

<span data-ttu-id="daeaf-104">'member' obsoleto: 'text'</span><span class="sxs-lookup"><span data-stu-id="daeaf-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="daeaf-105">Um membro de classe foi marcado com o atributo `Obsolete`, de modo que um aviso será emitido quando o membro da classe for referenciado.</span><span class="sxs-lookup"><span data-stu-id="daeaf-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="daeaf-106">Para obter mais informações, consulte [Atributos Comuns](../attributes/global.md).</span><span class="sxs-lookup"><span data-stu-id="daeaf-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="daeaf-107">O exemplo a seguir gera o erro CS0618:</span><span class="sxs-lookup"><span data-stu-id="daeaf-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```

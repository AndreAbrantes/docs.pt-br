# <a name="compiler-error-cs0840"></a><span data-ttu-id="04dd8-102">Erro do Compilador CS0840</span><span class="sxs-lookup"><span data-stu-id="04dd8-102">Compiler Error CS0840</span></span>
<span data-ttu-id="04dd8-103">'Property name' deve declarar um corpo, pois não está marcado como abstract ou extern.</span><span class="sxs-lookup"><span data-stu-id="04dd8-103">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="04dd8-104">Propriedades implementadas automaticamente devem definir os acessadores get e set.</span><span class="sxs-lookup"><span data-stu-id="04dd8-104">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="04dd8-105">A menos que uma propriedade esteja marcada como `abstract` ou `extern` ou seja um membro de um tipo `partial`, ela deve fornecer um corpo.</span><span class="sxs-lookup"><span data-stu-id="04dd8-105">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="04dd8-106">Propriedades autoimplementadas não fornecem corpos de acessador, mas devem especificar ambos os acessadores.</span><span class="sxs-lookup"><span data-stu-id="04dd8-106">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="04dd8-107">Para criar uma propriedade autoimplementada somente leitura, defina o acessador set como `private`.</span><span class="sxs-lookup"><span data-stu-id="04dd8-107">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04dd8-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="04dd8-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="04dd8-109">Forneça o corpo ausente ou o acessador ou então, use os modificadores [abstract](../../../csharp/language-reference/keywords/abstract.md), [extern](../../../csharp/language-reference/keywords/extern.md) ou [partial (tipo)](../../../csharp/language-reference/keywords/partial-type.md) nele e/ou em seu tipo delimitador.</span><span class="sxs-lookup"><span data-stu-id="04dd8-109">Supply the missing body or accessor or else use the [abstract](../../../csharp/language-reference/keywords/abstract.md), [extern](../../../csharp/language-reference/keywords/extern.md), or [partial (Type)](../../../csharp/language-reference/keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04dd8-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="04dd8-110">Example</span></span>  
 <span data-ttu-id="04dd8-111">O exemplo a seguir gera o erro CS0840:</span><span class="sxs-lookup"><span data-stu-id="04dd8-111">The following example generates CS0840:</span></span>  
  
```csharp  
// cs0840.cs

class Test  
{  
    public int myProp { get; } // CS0840  
  
    // to create a read-only property  
    // try the following line instead  
    public int myProp2 { get; private set; }  
  
}  
```

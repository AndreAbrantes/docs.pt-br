<span data-ttu-id="bb3b9-104">'Property name' deve declarar um corpo, pois não está marcado como abstract ou extern.</span><span class="sxs-lookup"><span data-stu-id="bb3b9-104">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="bb3b9-105">Propriedades implementadas automaticamente devem definir os acessadores get e set.</span><span class="sxs-lookup"><span data-stu-id="bb3b9-105">Automatically implemented properties must define both get and set accessors.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="bb3b9-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="bb3b9-109">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="bb3b9-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="bb3b9-111">Example</span></span>

    public int myProp { get; } // CS0840

# <a name="compiler-error-cs1936"></a><span data-ttu-id="336d8-102">Erro do Compilador CS1936</span><span class="sxs-lookup"><span data-stu-id="336d8-102">Compiler Error CS1936</span></span>

<span data-ttu-id="336d8-103">Não foi possível encontrar uma implementação do padrão de consulta para o tipo de origem 'type'.</span><span class="sxs-lookup"><span data-stu-id="336d8-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="336d8-104">'method' não encontrado.</span><span class="sxs-lookup"><span data-stu-id="336d8-104">'method' not found.</span></span>  
  
<span data-ttu-id="336d8-105">Para consultar um tipo de origem, esse tipo deve implementar os métodos de operador de consulta padrão invocados na consulta.</span><span class="sxs-lookup"><span data-stu-id="336d8-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="336d8-106">A implementação pode ser na forma de membros de classe ou métodos de extensão colocados no escopo com as diretiva `using` apropriada.</span><span class="sxs-lookup"><span data-stu-id="336d8-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="336d8-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="336d8-107">To correct this error</span></span>

- <span data-ttu-id="336d8-108">Certifique-se de que está consultando uma coleção de objetos e não um objeto individual.</span><span class="sxs-lookup"><span data-stu-id="336d8-108">Make sure that you are querying a collection of objects, not an individual object.</span></span>  
  
- <span data-ttu-id="336d8-109">Certifique-se de que especificou as diretivas `using` necessárias.</span><span class="sxs-lookup"><span data-stu-id="336d8-109">Make sure that you have specified the necessary `using` directives.</span></span>  

## <a name="example"></a><span data-ttu-id="336d8-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="336d8-110">Example</span></span>

<span data-ttu-id="336d8-111">O exemplo a seguir produz o CS1936:</span><span class="sxs-lookup"><span data-stu-id="336d8-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="336d8-112">Normalmente, esse erro ocorre quando há uma tentativa acidental de consultar um objeto de algum tipo, em vez de uma coleção desses objetos.</span><span class="sxs-lookup"><span data-stu-id="336d8-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>

## <a name="example"></a><span data-ttu-id="9ccfa-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9ccfa-108">Example</span></span>  
 <span data-ttu-id="9ccfa-109">O exemplo a seguir gera o erro CS0592:</span><span class="sxs-lookup"><span data-stu-id="9ccfa-109">The following sample generates CS0592:</span></span>

public class MyAttribute : Attribute   
{  
}

    }  
}

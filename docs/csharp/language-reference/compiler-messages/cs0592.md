## <a name="example"></a><span data-ttu-id="9c827-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9c827-108">Example</span></span>

 <span data-ttu-id="9c827-109">O exemplo a seguir gera o erro CS0592:</span><span class="sxs-lookup"><span data-stu-id="9c827-109">The following sample generates CS0592:</span></span>

```csharp
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.

{  
    public static void Main()  
    {  
    }  
}  
```

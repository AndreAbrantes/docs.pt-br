# <a name="compiler-error-cs0201"></a><span data-ttu-id="ef547-102">Erro do Compilador CS0201</span><span class="sxs-lookup"><span data-stu-id="ef547-102">Compiler Error CS0201</span></span>

## <a name="example"></a><span data-ttu-id="ef547-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ef547-107">Example</span></span>  
 <span data-ttu-id="ef547-108">O exemplo a seguir gera o erro CS0201, pois 2 \* 3 é uma expressão e não uma instrução.</span><span class="sxs-lookup"><span data-stu-id="ef547-108">The following sample generates CS0201 because 2 \* 3 is an expression, not a statement.</span></span> <span data-ttu-id="ef547-109">Para fazer o código compilar, tente atribuir o valor da expressão a uma variável.</span><span class="sxs-lookup"><span data-stu-id="ef547-109">To make the code compile, try assigning the value of the expression to a  variable.</span></span>  
  
```csharp  
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ef547-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ef547-110">Example</span></span>  
 <span data-ttu-id="ef547-111">O exemplo a seguir gera o erro CS0201, pois a verificação em si não é uma instrução, embora seja parametrizada por uma operação de incremento.</span><span class="sxs-lookup"><span data-stu-id="ef547-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>  
  
```csharp  
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>   
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;   
         }  
      }  
   }  
}  
```

# <a name="compiler-error-cs0270"></a>Erro do Compilador CS0270
O tamanho de matriz não pode ser especificado em uma declaração de variável (tente inicializar com uma expressão 'new')  
  
 Esse erro ocorre quando um tamanho é especificado como parte de uma declaração de matriz. Para resolver, use a nova expressão do [operador.](../operators/new-operator.md)  
  
 O exemplo a seguir gera o erro CS0270:  
  
```csharp  
// CS0270.cs  
// compile with: /t:module  
  
public class Test  
{  
   int[10] a;   // CS0270  
   // To resolve, use the following line instead:  
   // int[] a = new int[10];  
}  
```

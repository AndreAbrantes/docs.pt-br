## <a name="example"></a><span data-ttu-id="f47ac-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f47ac-111">Example</span></span>  
 <span data-ttu-id="f47ac-112">O código a seguir gera o erro CS1060 porque o tipo de classe `U` é um membro de `struct S`, mas nunca é inicializado.</span><span class="sxs-lookup"><span data-stu-id="f47ac-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```

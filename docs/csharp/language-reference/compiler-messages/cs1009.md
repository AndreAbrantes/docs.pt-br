# <a name="compiler-error-cs1009"></a><span data-ttu-id="01f7d-102">Erro do Compilador CS1009</span><span class="sxs-lookup"><span data-stu-id="01f7d-102">Compiler Error CS1009</span></span>
<span data-ttu-id="01f7d-103">Sequência de escape não reconhecida</span><span class="sxs-lookup"><span data-stu-id="01f7d-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="01f7d-104">Um caractere inesperado após uma barra invertida (\\) em uma [cadeia de caracteres](../../../csharp/language-reference/keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="01f7d-104">An unexpected character follows a backslash (\\) in a [string](../../../csharp/language-reference/keywords/string.md).</span></span> <span data-ttu-id="01f7d-105">O compilador espera um dos caracteres de escape válidos.</span><span class="sxs-lookup"><span data-stu-id="01f7d-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="01f7d-106">Para obter mais informações, consulte [Escapes de Caracteres](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="01f7d-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="01f7d-107">O exemplo a seguir gera o erro CS1009.</span><span class="sxs-lookup"><span data-stu-id="01f7d-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="01f7d-108">Uma causa comum desse erro é usar o caractere de barra invertida em um nome de arquivo, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="01f7d-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="01f7d-109">Para resolver esse erro, use "\\\\" ou o literal @-quoted da cadeia de caracteres, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="01f7d-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```

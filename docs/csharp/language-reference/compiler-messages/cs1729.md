# <a name="compiler-error-cs1729"></a><span data-ttu-id="b16eb-102">Erro do Compilador CS1729</span><span class="sxs-lookup"><span data-stu-id="b16eb-102">Compiler Error CS1729</span></span>
<span data-ttu-id="b16eb-103">'tipo' não contém um construtor que aceita 'número' argumentos.</span><span class="sxs-lookup"><span data-stu-id="b16eb-103">'type' does not contain a constructor that takes 'number' arguments.</span></span>  
  
 <span data-ttu-id="b16eb-104">Esse erro ocorre quando você invoca o construtor de uma classe direta ou indiretamente, mas o compilador não encontra nenhum construtor com o mesmo número de parâmetros.</span><span class="sxs-lookup"><span data-stu-id="b16eb-104">This error occurs when you either directly or indirectly invoke the constructor of a class but the compiler cannot find any constructors with the same number of parameters.</span></span> <span data-ttu-id="b16eb-105">No exemplo a seguir, a classe `test` não tem nenhum construtor que aceita argumentos.</span><span class="sxs-lookup"><span data-stu-id="b16eb-105">In the following example, the `test` class has no constructors that take any arguments.</span></span> <span data-ttu-id="b16eb-106">Portanto, ela tem apenas um construtor padrão que aceita zero argumentos.</span><span class="sxs-lookup"><span data-stu-id="b16eb-106">It therefore has only a default constructor that takes zero arguments.</span></span> <span data-ttu-id="b16eb-107">Como na segunda linha em que o erro é gerado a classe derivada não declara nenhum construtor próprio, o compilador fornece um construtor padrão.</span><span class="sxs-lookup"><span data-stu-id="b16eb-107">Because in the second line in which the error is generated, the derived class declares no constructors of its own, the compiler provides a default constructor.</span></span> <span data-ttu-id="b16eb-108">Esse construtor invoca um construtor sem parâmetros na classe base.</span><span class="sxs-lookup"><span data-stu-id="b16eb-108">That constructor invokes a parameterless constructor in the base class.</span></span> <span data-ttu-id="b16eb-109">Como a classe base não tem esse construtor, o erro CS1729 será gerado.</span><span class="sxs-lookup"><span data-stu-id="b16eb-109">Because the base class has no such constructor, CS1729 is generated.</span></span>

## <a name="example"></a><span data-ttu-id="b16eb-114">Exemplo</span><span class="sxs-lookup"><span data-stu-id="b16eb-114">Example</span></span>  
 <span data-ttu-id="b16eb-115">O exemplo a seguir gera o erro CS1729:</span><span class="sxs-lookup"><span data-stu-id="b16eb-115">The following example generates CS1729:</span></span>  
  
```csharp  
// cs1729.cs  
class Test  
{  
    static int Main()  
    {  
        // Class Test has only a default constructor, which takes no arguments.  
        Test test1 = new Test(2); // CS1729  
        // The following line resolves the error.  
        Test test2 = new Test();  
  
        // Class Parent has only one constructor, which takes two int parameters.  
        Parent exampleParent1 = new Parent(10); // CS1729  
        // The following line resolves the error.  
        Parent exampleParent2 = new Parent(10, 1);  
  
        return 1;  
    }  
}  
  
public class Parent  
{  
    // The only constructor for this class has two parameters.  
    public Parent(int i, int j) { }  
}  
  
// The following declaration causes a compiler error because class Parent  
// does not have a constructor that takes no arguments. The declaration of  
// class Child2 shows how to resolve this error.  
public class Child : Parent { } // CS1729  
  
public class Child2 : Parent  
{  
    // The constructor for Child2 has only one parameter. To access the   
    // constructor in Parent, and prevent this compiler error, you must provide   
    // a value for the second parameter of Parent. The following example provides 0.  
    public Child2(int k)  
        : base(k, 0)  
    {  
        // Add the body of the constructor here.  
    }  
}  
```

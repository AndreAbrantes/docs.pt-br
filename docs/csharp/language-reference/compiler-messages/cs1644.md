# <a name="compiler-error-cs1644"></a>Erro do Compilador CS1644
O recurso 'recurso' não faz parte da especificação de linguagem ISO C# padronizada e pode não ser aceito por outros compiladores  
  
 Esse erro ocorre se você tiver especificado a opção [/langversion](../compiler-options/langversion-compiler-option.md) ISO-1 e o código que você está compilando usa recursos que não fazem parte do padrão ISO 1.0. Para resolver esse erro, não use nenhum dos recursos do compilador do C# 2.0, como genéricos ou métodos anônimos, com a opção de compatibilidade ISO-1.  
  
 O exemplo a seguir gera o erro CS1644:  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```

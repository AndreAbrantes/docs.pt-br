## <a name="example"></a><span data-ttu-id="14c89-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="14c89-109">Example</span></span>  

 <span data-ttu-id="14c89-110">O exemplo a seguir gera o erro CS0229:</span><span class="sxs-lookup"><span data-stu-id="14c89-110">The following example generates CS0229:</span></span>  
  
```csharp  
// CS0229.cs  
  
interface IList  
{  
    int Count  
    {  
        get;  
        set;  
    }  
  
    void Counter();  
}  
  
interface ICounter  
{  
    double Count  
    {  
        get;  
        set;  
    }  
}  
  
interface IListCounter : IList, ICounter {}  
  
class MyClass  
{  
    void Test(IListCounter x)  
    {  
        x.Count = 1;  // CS0229  
        // Try one of the following lines instead:  
        // ((IList)x).Count = 1;  
        // or  
        // ((ICounter)x).Count = 1;  
    }  
  
    public static void Main() {}  
}  
```

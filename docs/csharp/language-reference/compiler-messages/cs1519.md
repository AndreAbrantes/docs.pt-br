# <a name="compiler-error-cs1519"></a><span data-ttu-id="4b10c-102">Erro do Compilador CS1519</span><span class="sxs-lookup"><span data-stu-id="4b10c-102">Compiler Error CS1519</span></span>
<span data-ttu-id="4b10c-103">Token 'token' inválido em classe, struct ou na declaração de membro de interface</span><span class="sxs-lookup"><span data-stu-id="4b10c-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="4b10c-104">Esse erro é gerado sempre que um token é encontrado em um local ao qual ele não pertence.</span><span class="sxs-lookup"><span data-stu-id="4b10c-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="4b10c-105">Um *token* é uma palavra-chave, um identificador (o nome de uma classe, struct, método e assim por diante), uma cadeia de caracteres, um caractere ou valor literal numérico como 108, "Hello", ou 'A', ou um operador ou pontuador como `==` ou `;`.</span><span class="sxs-lookup"><span data-stu-id="4b10c-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="4b10c-106">Qualquer [classe](../../../csharp/language-reference/keywords/class.md), struct ou declaração de membro de interface que contém modificadores inválidos antes do tipo gerará esse erro.</span><span class="sxs-lookup"><span data-stu-id="4b10c-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="4b10c-107">Para corrigir o erro, remova os modificadores inválidos.</span><span class="sxs-lookup"><span data-stu-id="4b10c-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="4b10c-108">O exemplo a seguir geram o erro CS1519 em cinco locais porque os tokens são colocados em locais em que eles não são válidos:</span><span class="sxs-lookup"><span data-stu-id="4b10c-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```

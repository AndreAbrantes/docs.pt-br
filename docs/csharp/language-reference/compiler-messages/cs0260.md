# <a name="compiler-error-cs0260"></a><span data-ttu-id="0068e-102">Erro do Compilador CS0260</span><span class="sxs-lookup"><span data-stu-id="0068e-102">Compiler Error CS0260</span></span>
<span data-ttu-id="0068e-103">Modificador parcial inexistente na declaração de tipo 'type'; existe outra declaração parcial desse tipo</span><span class="sxs-lookup"><span data-stu-id="0068e-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="0068e-104">Esse erro indica que várias classes que têm o mesmo nome foram declaradas.</span><span class="sxs-lookup"><span data-stu-id="0068e-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="0068e-105">Além disso, pelo menos uma, mas não todas as declarações, contém o modificador `partial`.</span><span class="sxs-lookup"><span data-stu-id="0068e-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="0068e-106">Caso deseje definir uma classe em várias partes, será necessário declarar cada parte usando a palavra-chave `partial`.</span><span class="sxs-lookup"><span data-stu-id="0068e-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="0068e-107">Esse erro também ocorrerá se a classe for declarada e acidentalmente nomeada com o mesmo nome de uma classe parcial declarada em outro local no mesmo namespace.</span><span class="sxs-lookup"><span data-stu-id="0068e-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="0068e-108">O exemplo a seguir gera o erro CS0260:</span><span class="sxs-lookup"><span data-stu-id="0068e-108">The following sample generates CS0260:</span></span>  
  
```csharp  
// CS0260.cs  
// You must mark both parts of the definition of class C   
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C    
{  
}  
  
partial class C  
{  
}  
```

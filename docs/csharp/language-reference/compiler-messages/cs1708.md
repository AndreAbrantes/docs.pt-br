# <a name="compiler-error-cs1708"></a><span data-ttu-id="225e0-102">Erro do Compilador CS1708</span><span class="sxs-lookup"><span data-stu-id="225e0-102">Compiler Error CS1708</span></span>
<span data-ttu-id="225e0-103">Buffers de tamanho fixo só podem ser acessados por meio de locais ou campos</span><span class="sxs-lookup"><span data-stu-id="225e0-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="225e0-104">Um novo recurso do C# 2.0 é a capacidade de definir uma matriz em linha dentro de um `struct`.</span><span class="sxs-lookup"><span data-stu-id="225e0-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="225e0-105">Essas matrizes só podem ser acessadas por meio de variáveis locais ou campos e não podem ser referenciadas como valores intermediários no lado esquerdo de uma expressão.</span><span class="sxs-lookup"><span data-stu-id="225e0-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="225e0-106">Além disso, as matrizes não podem ser acessadas por campos que sejam `static` ou `readonly`.</span><span class="sxs-lookup"><span data-stu-id="225e0-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="225e0-107">Para resolver esse erro, defina uma variável de matriz e atribua a matriz em linha à variável.</span><span class="sxs-lookup"><span data-stu-id="225e0-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="225e0-108">Também é possível remover o modificador `static` ou `readonly` do campo que representa a matriz em linha.</span><span class="sxs-lookup"><span data-stu-id="225e0-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="225e0-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="225e0-109">Example</span></span>  
 <span data-ttu-id="225e0-110">O exemplo a seguir gera o erro CS1708.</span><span class="sxs-lookup"><span data-stu-id="225e0-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```

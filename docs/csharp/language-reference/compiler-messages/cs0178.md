## <a name="example"></a><span data-ttu-id="52fe2-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="52fe2-111">Example</span></span>

```csharp  
   public static void Main()  
   {  
      int a = new int[5][,][][5;   // CS0178

   }  
}  
```

<span data-ttu-id="a38d8-103">Apresentar um método 'Finalize' pode interferir na invocação do destruidor.</span><span class="sxs-lookup"><span data-stu-id="a38d8-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="a38d8-104">Você pretendia declarar um destruidor?</span><span class="sxs-lookup"><span data-stu-id="a38d8-104">Did you intend to declare a destructor?</span></span>

 <span data-ttu-id="a38d8-106">Caso essa classe seja usada como classe base e a classe derivada definir um destruidor, o destruidor substituirá a método `Finalize` da classe base e não <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="a38d8-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a38d8-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a38d8-107">Example</span></span>  
 <span data-ttu-id="a38d8-108">O exemplo a seguir gera o erro CS0465.</span><span class="sxs-lookup"><span data-stu-id="a38d8-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```

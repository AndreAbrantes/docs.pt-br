## <a name="example"></a><span data-ttu-id="227a3-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="227a3-106">Example</span></span>  
 <span data-ttu-id="227a3-107">O exemplo a seguir gera o erro CS0115.</span><span class="sxs-lookup"><span data-stu-id="227a3-107">The following sample generates CS0115.</span></span> <span data-ttu-id="227a3-108">É possível resolver o erro CS0115 em uma das duas maneiras:</span><span class="sxs-lookup"><span data-stu-id="227a3-108">You can resolve CS0115 in one of two ways:</span></span>  
  
-   <span data-ttu-id="227a3-109">Remova a palavra-chave `override` do método em `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="227a3-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
-   <span data-ttu-id="227a3-110">Use `MyClass1` como classe base para `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="227a3-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```

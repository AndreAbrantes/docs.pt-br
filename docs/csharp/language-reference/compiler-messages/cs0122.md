# <a name="compiler-error-cs0122"></a>Erro do Compilador CS0122
'member' está inacessível em razão de seu nível de proteção  
  
 O [modificador de acesso](../keywords/modifiers.md) de um membro de classe impede o acesso ao membro. Para obter mais informações, consulte [Modificadores de Acesso](../../programming-guide/classes-and-structs/access-modifiers.md).  
  
 Uma causa disso (não mostrada no exemplo abaixo) pode ser a ausência do sinalizador de compilador **/out** no destino de um assembly amigável. Para obter mais informações, consulte [Assemblies Amigáveis](../../../standard/assembly/friend-assemblies.md) e [/out (Opções do Compilador C#)](../compiler-options/out-compiler-option.md)  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera o erro CS0122:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```

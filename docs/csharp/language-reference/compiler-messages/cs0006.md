# <a name="compiler-error-cs0006"></a>Erro do Compilador CS0006
Não foi possível encontrar o arquivo de metadados 'dll_name'  
  
 O programa foi compilado e passou explicitamente o nome de um arquivo que continha metadados; no entanto, o arquivo .dll não foi encontrado. Para obter mais informações, consulte [/reference (opções do compilador C#)](../compiler-options/reference-compiler-option.md).

# <a name="compiler-error-cs1001"></a><span data-ttu-id="2322c-102">Erro do Compilador CS1001</span><span class="sxs-lookup"><span data-stu-id="2322c-102">Compiler Error CS1001</span></span>
<span data-ttu-id="2322c-103">Identificador esperado</span><span class="sxs-lookup"><span data-stu-id="2322c-103">Identifier expected</span></span>  
  
 <span data-ttu-id="2322c-104">O identificador não foi fornecido.</span><span class="sxs-lookup"><span data-stu-id="2322c-104">You did not supply an identifier.</span></span> <span data-ttu-id="2322c-105">Um identificador é o nome que você fornecer a uma classe, struct, namespace, método, variável e assim por diante.</span><span class="sxs-lookup"><span data-stu-id="2322c-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="2322c-106">O exemplo a seguir declara uma classe simples, mas não a nomeia:</span><span class="sxs-lookup"><span data-stu-id="2322c-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="2322c-107">O exemplo a seguir gera o erro CS1001, pois, ao declarar uma enumeração, é necessário especificar membros:</span><span class="sxs-lookup"><span data-stu-id="2322c-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="2322c-108">Os nomes de parâmetro são necessários, mesmo que o compilador não os use, por exemplo, em uma definição de interface.</span><span class="sxs-lookup"><span data-stu-id="2322c-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="2322c-109">Esses parâmetros são necessários para que os programadores que estão consumindo a interface saibam alguma coisa sobre o significado dos parâmetros.</span><span class="sxs-lookup"><span data-stu-id="2322c-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```

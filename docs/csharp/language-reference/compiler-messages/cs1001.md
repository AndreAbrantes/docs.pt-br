# <a name="compiler-error-cs1001"></a><span data-ttu-id="98ad6-103">Erro do Compilador CS1001</span><span class="sxs-lookup"><span data-stu-id="98ad6-103">Compiler Error CS1001</span></span>

<span data-ttu-id="98ad6-104">Identificador esperado</span><span class="sxs-lookup"><span data-stu-id="98ad6-104">Identifier expected</span></span>

<span data-ttu-id="98ad6-105">O identificador não foi fornecido.</span><span class="sxs-lookup"><span data-stu-id="98ad6-105">You did not supply an identifier.</span></span> <span data-ttu-id="98ad6-106">Um identificador é o nome que você fornecer a uma classe, struct, namespace, método, variável e assim por diante.</span><span class="sxs-lookup"><span data-stu-id="98ad6-106">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="98ad6-107">O exemplo a seguir declara uma classe simples, mas não a nomeia:</span><span class="sxs-lookup"><span data-stu-id="98ad6-107">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="98ad6-108">O exemplo a seguir gera o erro CS1001, pois, ao declarar uma enumeração, é necessário especificar membros:</span><span class="sxs-lookup"><span data-stu-id="98ad6-108">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="98ad6-109">Os nomes de parâmetro são necessários, mesmo que o compilador não os use, por exemplo, em uma definição de interface.</span><span class="sxs-lookup"><span data-stu-id="98ad6-109">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="98ad6-110">Esses parâmetros são necessários para que os programadores que estão consumindo a interface saibam alguma coisa sobre o significado dos parâmetros.</span><span class="sxs-lookup"><span data-stu-id="98ad6-110">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```

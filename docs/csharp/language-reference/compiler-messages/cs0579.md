# <a name="compiler-error-cs0579"></a><span data-ttu-id="2a40f-102">Erro do Compilador CS0579</span><span class="sxs-lookup"><span data-stu-id="2a40f-102">Compiler Error CS0579</span></span>
<span data-ttu-id="2a40f-103">'attribute' é um atributo duplicado</span><span class="sxs-lookup"><span data-stu-id="2a40f-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="2a40f-104">Não é possível especificar o mesmo atributo mais de uma vez, a menos que o atributo especifique **AllowMultiple=true** em seu [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="2a40f-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2a40f-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2a40f-105">Example</span></span>  
 <span data-ttu-id="2a40f-106">O exemplo a seguir gera o erro CS0579.</span><span class="sxs-lookup"><span data-stu-id="2a40f-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

# <a name="compiler-error-cs0579"></a>Erro do Compilador CS0579
'attribute' é um atributo duplicado  
  
 Não é possível especificar o mesmo atributo mais de uma vez, a menos que o atributo especifique **AllowMultiple=true** em seu [AttributeUsage](../../language-reference/attributes/general.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera o erro CS0579.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

# <a name="compiler-error-cs1943"></a><span data-ttu-id="04766-103">Erro do Compilador CS1943</span><span class="sxs-lookup"><span data-stu-id="04766-103">Compiler Error CS1943</span></span>
<span data-ttu-id="04766-104">Uma expressão do tipo 'tipo' não é permitida em uma cláusula From subsequente em uma expressão de consulta com o tipo de origem 'tipo'.</span><span class="sxs-lookup"><span data-stu-id="04766-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="04766-105">A inferência de tipos falhou na chamada para 'method'.</span><span class="sxs-lookup"><span data-stu-id="04766-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="04766-106">Todas as variáveis de intervalo devem representar tipos passíveis de consulta.</span><span class="sxs-lookup"><span data-stu-id="04766-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04766-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="04766-107">To correct this error</span></span>  
  
1. <span data-ttu-id="04766-108">Verifique se o tipo é um tipo passível de consulta que implementa `IEnumerable`, `IEnumerable<T>` ou uma interface derivada ou qualquer outro tipo que tem um padrão de consulta definido para ele.</span><span class="sxs-lookup"><span data-stu-id="04766-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>

## <a name="example"></a><span data-ttu-id="04766-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="04766-110">Example</span></span>  
 <span data-ttu-id="04766-111">O código a seguir gera o erro CS1943:</span><span class="sxs-lookup"><span data-stu-id="04766-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```

# <a name="compiler-error-cs1943"></a><span data-ttu-id="46973-102">Erro do Compilador CS1943</span><span class="sxs-lookup"><span data-stu-id="46973-102">Compiler Error CS1943</span></span>
<span data-ttu-id="46973-103">Uma expressão do tipo 'tipo' não é permitida em uma cláusula From subsequente em uma expressão de consulta com o tipo de origem 'tipo'.</span><span class="sxs-lookup"><span data-stu-id="46973-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="46973-104">A inferência de tipos falhou na chamada para 'method'.</span><span class="sxs-lookup"><span data-stu-id="46973-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="46973-105">Todas as variáveis de intervalo devem representar tipos passíveis de consulta.</span><span class="sxs-lookup"><span data-stu-id="46973-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="46973-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="46973-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="46973-107">Verifique se o tipo é um tipo passível de consulta que implementa `IEnumerable`, `IEnumerable<T>` ou uma interface derivada ou qualquer outro tipo que tem um padrão de consulta definido para ele.</span><span class="sxs-lookup"><span data-stu-id="46973-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>

## <a name="example"></a><span data-ttu-id="46973-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="46973-109">Example</span></span>  
 <span data-ttu-id="46973-110">O código a seguir gera o erro CS1943:</span><span class="sxs-lookup"><span data-stu-id="46973-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```

# <a name="compiler-error-cs1721"></a><span data-ttu-id="866da-102">Erro do Compilador CS1721</span><span class="sxs-lookup"><span data-stu-id="866da-102">Compiler Error CS1721</span></span>

## <a name="example"></a><span data-ttu-id="866da-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="866da-107">Example</span></span>  
 <span data-ttu-id="866da-108">O exemplo a seguir mostra uma maneira na qual CS1721 é gerado e, em seguida, mostra duas maneiras possíveis de evitar o erro.</span><span class="sxs-lookup"><span data-stu-id="866da-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```

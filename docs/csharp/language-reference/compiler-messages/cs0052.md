# <a name="compiler-error-cs0052"></a><span data-ttu-id="4e580-103">Erro do Compilador CS0052</span><span class="sxs-lookup"><span data-stu-id="4e580-103">Compiler Error CS0052</span></span>

<span data-ttu-id="4e580-104">Acessibilidade inconsistente: tipo de campo 'type' é menos acessível do que o campo 'field'</span><span class="sxs-lookup"><span data-stu-id="4e580-104">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="4e580-105">O tipo de um campo não pode ser menos acessível que o próprio campo, pois todos os constructos públicos devem retornar um objeto acessível publicamente.</span><span class="sxs-lookup"><span data-stu-id="4e580-105">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="4e580-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4e580-106">Example</span></span>

 <span data-ttu-id="4e580-107">O exemplo a seguir gera o erro CS0052:</span><span class="sxs-lookup"><span data-stu-id="4e580-107">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```

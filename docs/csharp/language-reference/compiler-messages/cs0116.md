# <a name="compiler-error-cs0116"></a><span data-ttu-id="77a4c-102">Erro do Compilador CS0116</span><span class="sxs-lookup"><span data-stu-id="77a4c-102">Compiler Error CS0116</span></span>
<span data-ttu-id="77a4c-103">Um namespace não contém membros diretamente, como campos ou métodos</span><span class="sxs-lookup"><span data-stu-id="77a4c-103">A namespace does not directly contain members such as fields or methods</span></span>  
  
 <span data-ttu-id="77a4c-104">Dentro de um [namespace](../../../csharp/language-reference/keywords/namespace.md), o compilador aceita apenas classes, estruturas, uniões, enumerações, interfaces e delegados.</span><span class="sxs-lookup"><span data-stu-id="77a4c-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="77a4c-105">Geralmente, esse erro é cometido por desenvolvedores com experiência no C/C++, que se esquecem de que no C# métodos e variáveis devem ser declarados e definidos dentro de uma classe ou struct.</span><span class="sxs-lookup"><span data-stu-id="77a4c-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="77a4c-106">Para obter mais informações, consulte [Estrutura Geral de um Programa em C#](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="77a4c-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="77a4c-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="77a4c-107">Example</span></span>  
 <span data-ttu-id="77a4c-108">O exemplo a seguir gera o erro CS0116:</span><span class="sxs-lookup"><span data-stu-id="77a4c-108">The following sample generates CS0116:</span></span>  
  
```csharp  
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```

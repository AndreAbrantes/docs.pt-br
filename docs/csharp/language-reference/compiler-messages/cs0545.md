<span data-ttu-id="335a8-103">'função': não pode substituir porque 'propriedade' não tem um acessador get substituível</span><span class="sxs-lookup"><span data-stu-id="335a8-103">'function' : cannot override because 'property' does not have an overridable get accessor</span></span>  
  
 <span data-ttu-id="335a8-104">Foi feita uma tentativa para definir uma substituição para um acessador de propriedade quando a classe base não tem essa definição para substituir.</span><span class="sxs-lookup"><span data-stu-id="335a8-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="335a8-105">Você pode resolver esse erro da seguinte maneira:</span><span class="sxs-lookup"><span data-stu-id="335a8-105">You can resolve this error by:</span></span>

- <span data-ttu-id="335a8-107">Remoção do acessador `set` da classe derivada.</span><span class="sxs-lookup"><span data-stu-id="335a8-107">Removing the `set` accessor from the derived class.</span></span>

## <a name="example"></a><span data-ttu-id="335a8-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="335a8-111">Example</span></span>  
 <span data-ttu-id="335a8-112">O exemplo a seguir gera o erro CS0545.</span><span class="sxs-lookup"><span data-stu-id="335a8-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```

<span data-ttu-id="37f43-103">Código de expressões inacessível detectado</span><span class="sxs-lookup"><span data-stu-id="37f43-103">Unreachable expression code detected</span></span>  
  
 <span data-ttu-id="37f43-104">Esse erro ocorre sempre que parte de uma expressão do código estiver inacessível.</span><span class="sxs-lookup"><span data-stu-id="37f43-104">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="37f43-105">No exemplo a seguir, a condição `false && myTest()` atende a esse critério, pois o método `myTest()` nunca será avaliado devido ao fato de que o lado esquerdo da operação `&&` sempre é false.</span><span class="sxs-lookup"><span data-stu-id="37f43-105">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="37f43-106">Assim que o operador `&&` avaliar a instrução `false` como false, ele interromperá a avaliação e nunca avaliará o lado direito.</span><span class="sxs-lookup"><span data-stu-id="37f43-106">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>

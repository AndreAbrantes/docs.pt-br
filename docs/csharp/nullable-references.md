# <a name="nullable-reference-types"></a><span data-ttu-id="0345c-104">Tipos de referência anuláveis</span><span class="sxs-lookup"><span data-stu-id="0345c-104">Nullable reference types</span></span>

<span data-ttu-id="0345c-105">O C# 8.0 apresenta **tipos de referência que permitem valor nulo** e **tipos de referência que não permitem valor nulo** que permitem que você crie importantes instruções sobre as propriedades para variáveis de tipo de referência:</span><span class="sxs-lookup"><span data-stu-id="0345c-105">C# 8.0 introduces **nullable reference types** and **non-nullable reference types** that enable you to make important statements about the properties for reference type variables:</span></span>

- <span data-ttu-id="0345c-106">**Uma referência não deve ser nula**.</span><span class="sxs-lookup"><span data-stu-id="0345c-106">**A reference is not supposed to be null**.</span></span> <span data-ttu-id="0345c-107">Quando as variáveis não devem ser nulas, o compilador impõe regras que garantem que seja seguro desreferenciar essas variáveis sem verificar primeiro que ela não está nula:</span><span class="sxs-lookup"><span data-stu-id="0345c-107">When variables aren't supposed to be null, the compiler enforces rules that ensure it is safe to dereference these variables without first checking that it isn't null:</span></span>
  - <span data-ttu-id="0345c-108">A variável deve ser inicializada para um valor não nulo.</span><span class="sxs-lookup"><span data-stu-id="0345c-108">The variable must be initialized to a non-null value.</span></span>
  - <span data-ttu-id="0345c-109">A variável nunca pode receber o valor `null`.</span><span class="sxs-lookup"><span data-stu-id="0345c-109">The variable can never be assigned the value `null`.</span></span>
- <span data-ttu-id="0345c-110">**Uma referência pode ser nula**.</span><span class="sxs-lookup"><span data-stu-id="0345c-110">**A reference may be null**.</span></span> <span data-ttu-id="0345c-111">Quando variáveis puderem ser nulas, o compilador imporá diferentes regras para garantir que você verificou corretamente se há uma referência nula:</span><span class="sxs-lookup"><span data-stu-id="0345c-111">When variables may be null, the compiler enforces different rules to ensure that you've correctly checked for a null reference:</span></span>
  - <span data-ttu-id="0345c-112">a variável só poderá ser desreferenciada quando o compilador puder garantir que o valor não é nulo.</span><span class="sxs-lookup"><span data-stu-id="0345c-112">The variable may only be dereferenced when the compiler can guarantee that the value isn't null.</span></span>
  - <span data-ttu-id="0345c-113">Essas variáveis podem ser inicializadas com o valor `null` padrão e receber o valor `null` em outro código.</span><span class="sxs-lookup"><span data-stu-id="0345c-113">These variables may be initialized with the default `null` value and may be assigned the value `null` in other code.</span></span>

<span data-ttu-id="0345c-114">Esse novo recurso oferece benefícios significativos com relação à manipulação de variáveis de referência em versões anteriores do C#, em que a intenção do design não poderia ser determinada na declaração da variável.</span><span class="sxs-lookup"><span data-stu-id="0345c-114">This new feature provides significant benefits over the handling of reference variables in earlier versions of C# where the design intent couldn't be determined from the variable declaration.</span></span> <span data-ttu-id="0345c-115">O compilador não forneceu segurança com relação a exceções de referência nula para tipos de referência:</span><span class="sxs-lookup"><span data-stu-id="0345c-115">The compiler didn't provide safety against null reference exceptions for reference types:</span></span>

- <span data-ttu-id="0345c-116">**Uma referência pode ser nula**.</span><span class="sxs-lookup"><span data-stu-id="0345c-116">**A reference can be null**.</span></span> <span data-ttu-id="0345c-117">Nenhum aviso é emitido quando um tipo de referência é inicializado para nulo ou atribuído posteriormente a nulo.</span><span class="sxs-lookup"><span data-stu-id="0345c-117">No warnings are issued when a reference type is initialized to null, or later assigned to null.</span></span>
- <span data-ttu-id="0345c-118">**Uma referência é considerada não nula**.</span><span class="sxs-lookup"><span data-stu-id="0345c-118">**A reference is assumed to be not null**.</span></span> <span data-ttu-id="0345c-119">O compilador não emite nenhum aviso quando os tipos de referência são desreferenciados.</span><span class="sxs-lookup"><span data-stu-id="0345c-119">The compiler doesn't issue any warnings when reference types are dereferenced.</span></span> <span data-ttu-id="0345c-120">(Com referências que permitem valor nulo, o compilador emite avisos sempre que você desreferencia uma variável que pode ser nula).</span><span class="sxs-lookup"><span data-stu-id="0345c-120">(With nullable references,  the compiler issues warnings whenever you dereference a variable that may be null).</span></span>

<span data-ttu-id="0345c-121">Com a adição de tipos de referência que permitem valor nulo, é possível declarar sua intenção mais claramente.</span><span class="sxs-lookup"><span data-stu-id="0345c-121">With the addition of nullable reference types, you can declare your intent more clearly.</span></span> <span data-ttu-id="0345c-122">O valor `null` é a maneira correta de representar que uma variável não se refere a um valor.</span><span class="sxs-lookup"><span data-stu-id="0345c-122">The `null` value is the correct way to represent that a variable doesn't refer to a value.</span></span> <span data-ttu-id="0345c-123">Não use esse recurso para remover todos os valores `null` do seu código.</span><span class="sxs-lookup"><span data-stu-id="0345c-123">Don't use this feature to remove all `null` values from your code.</span></span> <span data-ttu-id="0345c-124">Em vez disso, você deve declarar sua intenção par ao compilador e para outros desenvolvedores que leem seu código.</span><span class="sxs-lookup"><span data-stu-id="0345c-124">Rather, you should declare your intent to the compiler and other developers that read your code.</span></span> <span data-ttu-id="0345c-125">Ao declarar sua intenção, o compilador informa quando você escreve um código inconsistente com essa intenção.</span><span class="sxs-lookup"><span data-stu-id="0345c-125">By declaring your intent, the compiler informs you when you write code that is inconsistent with that intent.</span></span>

<span data-ttu-id="0345c-126">Um **tipo de referência que permite valor nulo** é indicado usando a mesma sintaxe que [tipos de valor que permitem valor nulo](programming-guide/nullable-types/index.md): um `?` é acrescentado ao tipo da variável.</span><span class="sxs-lookup"><span data-stu-id="0345c-126">A **nullable reference type** is noted using the same syntax as [nullable value types](programming-guide/nullable-types/index.md): a `?` is appended to the type of the variable.</span></span> <span data-ttu-id="0345c-127">Por exemplo, a seguinte declaração de variável representa uma variável de cadeia de caracteres que permite valor nulo, `name`:</span><span class="sxs-lookup"><span data-stu-id="0345c-127">For example, the following variable declaration represents a nullable string variable, `name`:</span></span>


<span data-ttu-id="0345c-128">Qualquer variável em que o `?` não é acrescentado ao nome do tipo é um **tipo de referência que não permite valor nulo**.</span><span class="sxs-lookup"><span data-stu-id="0345c-128">Any variable where the `?` is not appended to the type name is a **non-nullable reference type**.</span></span> <span data-ttu-id="0345c-129">Isso inclui todas as variáveis de tipo de referência no código existente quando você habilitou esse recurso.</span><span class="sxs-lookup"><span data-stu-id="0345c-129">That includes all reference type variables in existing code when you have enabled this feature.</span></span>

<span data-ttu-id="0345c-130">O compilador usa uma análise estática para determinar se uma referência que permite valor nulo é conhecida como não nula.</span><span class="sxs-lookup"><span data-stu-id="0345c-130">The compiler uses static analysis to determine if a nullable reference is known to be non-null.</span></span> <span data-ttu-id="0345c-131">O compilador informa se você desreferencia uma referência quer permite valor nulo quando ela pode ser nula.</span><span class="sxs-lookup"><span data-stu-id="0345c-131">The compiler warns you if you dereference a nullable reference when it may be null.</span></span> <span data-ttu-id="0345c-132">É possível substituir esse comportamento usando o **operador tolerante a nulo** (`!`) seguindo um nome de variável.</span><span class="sxs-lookup"><span data-stu-id="0345c-132">You can override this behavior by using the **null-forgiving operator** (`!`) following a variable name.</span></span> <span data-ttu-id="0345c-133">Por exemplo, se você sabe que a variável `name` não é nula, mas o compilador emite um aviso, é possível escrever o seguinte código para substituir a análise do compilador:</span><span class="sxs-lookup"><span data-stu-id="0345c-133">For example, if you know the `name` variable isn't null but the compiler issues a warning, you can write the following code to override the compiler's analysis:</span></span>

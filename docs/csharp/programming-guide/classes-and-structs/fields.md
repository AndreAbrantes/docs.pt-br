# <a name="fields-c-programming-guide"></a><span data-ttu-id="e73e7-102">Campos (Guia de Programação em C#)</span><span class="sxs-lookup"><span data-stu-id="e73e7-102">Fields (C# Programming Guide)</span></span>
<span data-ttu-id="e73e7-103">Um *campo* é uma variável de qualquer tipo que é declarada diretamente em uma [classe](../../../csharp/language-reference/keywords/class.md) ou [struct](../../../csharp/language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="e73e7-103">A *field* is a variable of any type that is declared directly in a [class](../../../csharp/language-reference/keywords/class.md) or [struct](../../../csharp/language-reference/keywords/struct.md).</span></span> <span data-ttu-id="e73e7-104">Os campos são *membros* do tipo que os contém.</span><span class="sxs-lookup"><span data-stu-id="e73e7-104">Fields are *members* of their containing type.</span></span>

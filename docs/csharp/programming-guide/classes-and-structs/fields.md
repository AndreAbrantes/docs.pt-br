# <a name="fields-c-programming-guide"></a><span data-ttu-id="4d7ce-102">Campos (Guia de Programação em C#)</span><span class="sxs-lookup"><span data-stu-id="4d7ce-102">Fields (C# Programming Guide)</span></span>
<span data-ttu-id="4d7ce-103">Um *campo* é uma variável de qualquer tipo que é declarada diretamente em uma [classe](../../../csharp/language-reference/keywords/class.md) ou [struct](../../../csharp/language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="4d7ce-103">A *field* is a variable of any type that is declared directly in a [class](../../../csharp/language-reference/keywords/class.md) or [struct](../../../csharp/language-reference/keywords/struct.md).</span></span> <span data-ttu-id="4d7ce-104">Os campos são *membros* do tipo que os contém.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-104">Fields are *members* of their containing type.</span></span>  
  
 <span data-ttu-id="4d7ce-105">Uma classe ou um struct podem ter campos de instância, campos estáticos ou ambos.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-105">A class or struct may have instance fields or static fields or both.</span></span> <span data-ttu-id="4d7ce-106">Os campos de instância são específicos a uma instância de um tipo.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-106">Instance fields are specific to an instance of a type.</span></span> <span data-ttu-id="4d7ce-107">Se você tem uma classe T, com um campo de instância F, você pode criar dois objetos do tipo T e modificar o valor de F em cada objeto sem afetar o valor no outro objeto.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-107">If you have a class T, with an instance field F, you can create two objects of type T, and modify the value of F in each object without affecting the value in the other object.</span></span> <span data-ttu-id="4d7ce-108">Por outro lado, um campo estático pertence à própria classe e é compartilhado entre todas as instâncias dessa classe.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-108">By contrast, a static field belongs to the class itself, and is shared among all instances of that class.</span></span> <span data-ttu-id="4d7ce-109">As alterações feitas na instância A serão imediatamente visíveis para as instâncias B e C se elas acessarem o campo.</span><span class="sxs-lookup"><span data-stu-id="4d7ce-109">Changes made from instance A will be visibly immediately to instances B and C if they access the field.</span></span>

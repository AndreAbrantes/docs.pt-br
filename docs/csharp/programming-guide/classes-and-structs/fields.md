<span data-ttu-id="963a6-103">Um *campo* é uma variável de qualquer tipo que é declarada diretamente em uma [classe](../../language-reference/keywords/class.md) ou [struct](../../language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="963a6-103">A *field* is a variable of any type that is declared directly in a [class](../../language-reference/keywords/class.md) or [struct](../../language-reference/keywords/struct.md).</span></span> <span data-ttu-id="963a6-104">Os campos são *membros* do tipo que os contém.</span><span class="sxs-lookup"><span data-stu-id="963a6-104">Fields are *members* of their containing type.</span></span>  
  
 <span data-ttu-id="963a6-105">Uma classe ou um struct podem ter campos de instância, campos estáticos ou ambos.</span><span class="sxs-lookup"><span data-stu-id="963a6-105">A class or struct may have instance fields or static fields or both.</span></span> <span data-ttu-id="963a6-106">Os campos de instância são específicos a uma instância de um tipo.</span><span class="sxs-lookup"><span data-stu-id="963a6-106">Instance fields are specific to an instance of a type.</span></span> <span data-ttu-id="963a6-107">Se você tem uma classe T, com um campo de instância F, você pode criar dois objetos do tipo T e modificar o valor de F em cada objeto sem afetar o valor no outro objeto.</span><span class="sxs-lookup"><span data-stu-id="963a6-107">If you have a class T, with an instance field F, you can create two objects of type T, and modify the value of F in each object without affecting the value in the other object.</span></span> <span data-ttu-id="963a6-108">Por outro lado, um campo estático pertence à própria classe e é compartilhado entre todas as instâncias dessa classe.</span><span class="sxs-lookup"><span data-stu-id="963a6-108">By contrast, a static field belongs to the class itself, and is shared among all instances of that class.</span></span> <span data-ttu-id="963a6-109">As alterações feitas na instância A serão imediatamente visíveis para as instâncias B e C se elas acessarem o campo.</span><span class="sxs-lookup"><span data-stu-id="963a6-109">Changes made from instance A will be visibly immediately to instances B and C if they access the field.</span></span>

 <span data-ttu-id="963a6-114">Os campos normalmente armazenam os dados que devem estar acessíveis a mais de um método de classe e devem ser armazenados por mais tempo que o tempo de vida de qualquer método único.</span><span class="sxs-lookup"><span data-stu-id="963a6-114">Fields typically store the data that must be accessible to more than one class method and must be stored for longer than the lifetime of any single method.</span></span> <span data-ttu-id="963a6-115">Por exemplo, uma classe que representa uma data do calendário pode ter três campos de inteiros: um para o mês, um para o dia e outro para o ano.</span><span class="sxs-lookup"><span data-stu-id="963a6-115">For example, a class that represents a calendar date might have three integer fields: one for the month, one for the day, and one for the year.</span></span> <span data-ttu-id="963a6-116">As variáveis que não são usadas fora do escopo de um método único devem ser declaradas como *variáveis locais* dentro do próprio corpo do método.</span><span class="sxs-lookup"><span data-stu-id="963a6-116">Variables that are not used outside the scope of a single method should be declared as *local variables* within the method body itself.</span></span>

 <span data-ttu-id="963a6-127">Os campos podem ser marcados como [public](../../language-reference/keywords/public.md), [private](../../language-reference/keywords/private.md), [protected](../../language-reference/keywords/protected.md), [internal](../../language-reference/keywords/internal.md), [protected internal](../../language-reference/keywords/protected-internal.md) ou [private protected](../../language-reference/keywords/private-protected.md).</span><span class="sxs-lookup"><span data-stu-id="963a6-127">Fields can be marked as [public](../../language-reference/keywords/public.md), [private](../../language-reference/keywords/private.md), [protected](../../language-reference/keywords/protected.md), [internal](../../language-reference/keywords/internal.md), [protected internal](../../language-reference/keywords/protected-internal.md) or [private protected](../../language-reference/keywords/private-protected.md).</span></span> <span data-ttu-id="963a6-128">Esses modificadores de acesso definem como os usuários da classe podem acessar os campos.</span><span class="sxs-lookup"><span data-stu-id="963a6-128">These access modifiers define how users of the class can access the fields.</span></span> <span data-ttu-id="963a6-129">Para obter mais informações, consulte [Modificadores de Acesso](./access-modifiers.md).</span><span class="sxs-lookup"><span data-stu-id="963a6-129">For more information, see [Access Modifiers](./access-modifiers.md).</span></span>

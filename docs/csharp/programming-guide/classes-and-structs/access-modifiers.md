## <a name="class-and-struct-member-accessibility"></a><span data-ttu-id="adb8e-129">Acessibilidade de classe e membro de struct</span><span class="sxs-lookup"><span data-stu-id="adb8e-129">Class and struct member accessibility</span></span>

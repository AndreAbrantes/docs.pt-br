# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Genéricos na biblioteca de classes .NET Framework (Guia de Programação em C#)
A versão 2.0 da biblioteca de classes .NET Framework fornece um novo namespace, <xref:System.Collections.Generic>, que inclui várias classes de coleção genérica prontas para uso e interfaces associadas. Outros namespaces, como <xref:System>, também fornecem novas interfaces genéricas, como <xref:System.IComparable%601>. Essas classes e interfaces são mais eficientes e fortemente tipadas que as classes de coleção não genéricas fornecidas em versões anteriores do .NET Framework. Antes de criar e implementar suas próprias classes de coleção personalizadas, considere se você pode usar ou derivar uma classe de uma das classes fornecidas na biblioteca de classes .NET Framework.

## <a name="delegates-overview"></a><span data-ttu-id="8be16-119">Visão geral de delegados</span><span class="sxs-lookup"><span data-stu-id="8be16-119">Delegates Overview</span></span>  
 <span data-ttu-id="8be16-120">Os delegados têm as seguintes propriedades:</span><span class="sxs-lookup"><span data-stu-id="8be16-120">Delegates have the following properties:</span></span>

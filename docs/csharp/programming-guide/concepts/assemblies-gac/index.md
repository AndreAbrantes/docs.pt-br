## <a name="assembly-manifest"></a><span data-ttu-id="15639-121">Manifesto de um assembly</span><span class="sxs-lookup"><span data-stu-id="15639-121">Assembly Manifest</span></span>  
 <span data-ttu-id="15639-122">Dentro de todo assembly está um *manifesto do assembly*.</span><span class="sxs-lookup"><span data-stu-id="15639-122">Within every assembly is an *assembly manifest*.</span></span> <span data-ttu-id="15639-123">Semelhante a um sumário, o manifesto do assembly contém o seguinte:</span><span class="sxs-lookup"><span data-stu-id="15639-123">Similar to a table of contents, the assembly manifest contains the following:</span></span>

# <a name="how-to-build-linq-to-xml-examples-c"></a>Como compilar exemplos de LINQ to XML (C#)
Os vários trechos e exemplos nesta documentação usam classes e tipos de uma variedade de namespaces. Para compilar o código em C#, você precisará fornecer diretivas apropriadas de `using`.  
  
## <a name="example"></a>Exemplo  
 O código a seguir contém diretivas de `using` que os exemplos de C# exigem para compilar e executar. Nem todas as diretivas de `using` são necessárias para todos os exemplos.  
  
```csharp  
using System;  
using System.Diagnostics;  
using System.Collections;  
using System.Collections.Generic;  
using System.Collections.Specialized;  
using System.Text;  
using System.Linq;  
using System.Xml;  
using System.Xml.Linq;  
using System.Xml.Schema;  
using System.Xml.XPath;  
using System.Xml.Xsl;  
using System.IO;  
using System.Threading;  
using System.Reflection;  
using System.IO.Packaging;  
```

  - <span data-ttu-id="1c238-125">A propriedade <xref:System.Nullable%601.HasValue%2A> retornará `true` se a variável contiver um valor ou `false` se for `null`.</span><span class="sxs-lookup"><span data-stu-id="1c238-125">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>

  - <span data-ttu-id="1c238-126">A propriedade <xref:System.Nullable%601.Value%2A> retornará um valor se <xref:System.Nullable%601.HasValue%2A> retornar `true`.</span><span class="sxs-lookup"><span data-stu-id="1c238-126">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="1c238-127">Caso contrário, um <xref:System.InvalidOperationException> será gerado.</span><span class="sxs-lookup"><span data-stu-id="1c238-127">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>

- <span data-ttu-id="1c238-133">Use o método <xref:System.Nullable%601.GetValueOrDefault(%600)> para retornar o valor atribuído ou o valor padrão fornecido se o valor do tipo que permite valor nulo for `null`.</span><span class="sxs-lookup"><span data-stu-id="1c238-133">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>

# <a name="compiler-error-cs1561"></a>CS1561 de erro do compilador
Nome do arquivo de saída é muito longo ou inválido  
  
 O nome do arquivo de saída não pode ter mais de 256 caracteres e não pode conter caracteres inválidos, como o símbolo de Euro, um ponto de interrogação ou uma barra invertida, entre outros.

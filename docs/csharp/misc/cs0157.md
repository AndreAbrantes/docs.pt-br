 <span data-ttu-id="27c5e-107">O exemplo a seguir gera CS0157:</span><span class="sxs-lookup"><span data-stu-id="27c5e-107">The following sample generates CS0157:</span></span>  
  
```csharp

{  
   public class MyClass2 : Exception  
   {  
   }

      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```

 <span data-ttu-id="d5c0f-106">O exemplo a seguir gera CS0157:</span><span class="sxs-lookup"><span data-stu-id="d5c0f-106">The following sample generates CS0157:</span></span>  
  
```csharp

{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }

         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```

# <a name="compiler-error-cs0174"></a>CS0174 de erro do compilador
Uma classe base é necessária para uma referência "base"  
  
 Esse erro ocorre somente quando o .NET Framework common language runtime compila o código-fonte para o `System.Object` classe, que é a única classe que não tem classe base.

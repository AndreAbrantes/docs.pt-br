# <a name="compiler-error-cs0174"></a>Erro do Compilador CS0174
Uma classe base é necessária para uma referência de ' base '  
  
 Esse erro ocorre somente quando o .NET Framework Common Language Runtime compila o código-fonte da `System.Object` classe, que é a única classe que não tem nenhuma classe base.

<span data-ttu-id="435f9-104">Um acessador Add ou remove deve ter um corpo</span><span class="sxs-lookup"><span data-stu-id="435f9-104">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="435f9-105">Uma palavra-chave **Add** ou **Remove** em uma definição de [evento](../language-reference/keywords/event.md) deve ter um corpo.</span><span class="sxs-lookup"><span data-stu-id="435f9-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="435f9-106">Para obter mais informações, consulte [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="435f9-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="435f9-107">O exemplo a seguir gera CS0073:</span><span class="sxs-lookup"><span data-stu-id="435f9-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

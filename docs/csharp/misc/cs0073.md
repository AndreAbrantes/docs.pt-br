 <span data-ttu-id="dc319-104">Uma palavra-chave **Add** ou **Remove** em uma definição de [evento](../language-reference/keywords/event.md) deve ter um corpo.</span><span class="sxs-lookup"><span data-stu-id="dc319-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="dc319-105">Para obter mais informações, consulte [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="dc319-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="dc319-106">O exemplo a seguir gera CS0073:</span><span class="sxs-lookup"><span data-stu-id="dc319-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

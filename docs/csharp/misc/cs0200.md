# <a name="compiler-error-cs0200"></a>Erro do Compilador CS0200
A propriedade ou a propriedade 'propriedade' não pode ser atribuída a -- ela é lida apenas

Foi feita uma tentativa de atribuir um valor a um [imóvel,](../programming-guide/classes-and-structs/using-properties.md)mas o imóvel não tem um acessório definido ou a atribuição estava fora da construtora. Resolva o erro adicionando um acessório definido. Para obter mais informações, consulte [Como declarar e usar as propriedades de gravação de leitura](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).

## <a name="examples"></a>Exemplos
A seguinte amostra gera CS0200:

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

A amostra a seguir usa [propriedades auto-implementadas,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [inicializadores de objetos](../programming-guide/classes-and-structs/object-and-collection-initializers.md)e ainda gera CS0200:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

A atribuição a uma propriedade ou 'propriedade' indexador que é apenas lida, pode ser alcançada através da adição de um acessório conjunto ou atribuindo à propriedade no construtor do objeto.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```

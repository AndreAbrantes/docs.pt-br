## <a name="examples"></a><span data-ttu-id="89e04-107">Exemplos</span><span class="sxs-lookup"><span data-stu-id="89e04-107">Examples</span></span>
<span data-ttu-id="89e04-108">A seguinte amostra gera CS0200:</span><span class="sxs-lookup"><span data-stu-id="89e04-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="89e04-109">A amostra a seguir usa [propriedades auto-implementadas,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [inicializadores de objetos](../programming-guide/classes-and-structs/object-and-collection-initializers.md)e ainda gera CS0200:</span><span class="sxs-lookup"><span data-stu-id="89e04-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="89e04-110">A atribuição a uma propriedade ou 'propriedade' indexador que é apenas lida, pode ser alcançada através da adição de um acessório conjunto ou atribuindo à propriedade no construtor do objeto.</span><span class="sxs-lookup"><span data-stu-id="89e04-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```

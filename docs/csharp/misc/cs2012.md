# <a name="compiler-error-cs2012"></a>Erro do compilador CS2012
Não é possível abrir ' arquivo ' para gravação  
  
 Ao usar a opção de compilador [-bugreport:](../language-reference/compiler-options/bugreport-compiler-option.md) `file`, o arquivo não pôde ser aberto para gravação. Verifique se você especificou um nome de arquivo válido e se o arquivo não é somente leitura.

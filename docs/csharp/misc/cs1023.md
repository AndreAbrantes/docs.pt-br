# <a name="compiler-error-cs1023"></a>Erro do compilador CS1023
Instrução inserida não pode ser uma declaração ou uma instrução rotulada  
  
 Uma declaração inserido, como os demonstrativos que seguem uma **se** instrução, pode conter nem declarações nem instruções rotuladas.  
  
 O exemplo a seguir gera CS1023 duas vezes:  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```

# <a name="compiler-warning-level-3-cs0168"></a>Compilador CS0168 de aviso (nível 3)
A variável 'var' está declarada mas nunca foi usada

O compilador emite um aviso de três níveis, quando você declara uma variável, mas não usá-lo.
  
O exemplo a seguir gera um aviso CS0168:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```

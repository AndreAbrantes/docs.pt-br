# <a name="compiler-error-cs0036"></a><span data-ttu-id="cf196-102">CS0036 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="cf196-102">Compiler Error CS0036</span></span>
<span data-ttu-id="cf196-103">Um parâmetro out não pode ter o atributo '[In]'</span><span class="sxs-lookup"><span data-stu-id="cf196-103">An out parameter cannot have the '[In]' attribute</span></span>  
  
 <span data-ttu-id="cf196-104">Atualmente, o **na** atributo não é permitido em uma [out](../../csharp/language-reference/keywords/out.md) parâmetro.</span><span class="sxs-lookup"><span data-stu-id="cf196-104">Currently, the **In** attribute is not permitted on an [out](../../csharp/language-reference/keywords/out.md) parameter.</span></span>  
  
 <span data-ttu-id="cf196-105">O exemplo a seguir gera CS0036:</span><span class="sxs-lookup"><span data-stu-id="cf196-105">The following sample generates CS0036:</span></span>  
  
```  
// CS0036.cs  
  
using System;  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   public static void TestOut([In] out char TestChar)   // CS0036  
   // try the following line instead  
   // public static void TestOut(out char TestChar)  
   {  
      TestChar = 'b';  
      Console.WriteLine(TestChar);  
   }  
  
   public static void Main()  
   {  
      char i;           //variable to receive the value  
      TestOut(out i);   // the arg must be passed as out  
      Console.WriteLine(i);  
   }  
}  
```

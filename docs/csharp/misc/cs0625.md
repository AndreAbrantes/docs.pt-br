# <a name="compiler-error-cs0625"></a><span data-ttu-id="4c6da-103">Erro do Compilador CS0625</span><span class="sxs-lookup"><span data-stu-id="4c6da-103">Compiler Error CS0625</span></span>
<span data-ttu-id="4c6da-104">' Field ': tipos de campo de instância marcados com StructLayout (LayoutKind. Explicit) devem ter um atributo FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="4c6da-104">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="4c6da-105">Quando uma struct é marcada com um atributo **StructLayout** explícito, todos os campos na struct devem ter o atributo [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) .</span><span class="sxs-lookup"><span data-stu-id="4c6da-105">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="4c6da-106">Para obter mais informações, consulte [classe StructLayoutAttribute](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="4c6da-106">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="4c6da-107">O exemplo a seguir gera CS0625:</span><span class="sxs-lookup"><span data-stu-id="4c6da-107">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```

## <a name="example"></a><span data-ttu-id="d8d92-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d8d92-106">Example</span></span>  
 <span data-ttu-id="d8d92-107">O exemplo a seguir gera CS0082:</span><span class="sxs-lookup"><span data-stu-id="d8d92-107">The following example generates CS0082:</span></span>  
  
```  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```

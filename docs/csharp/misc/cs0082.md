## <a name="example"></a><span data-ttu-id="9f1a2-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9f1a2-106">Example</span></span>  
 <span data-ttu-id="9f1a2-107">O exemplo a seguir gera CS0082:</span><span class="sxs-lookup"><span data-stu-id="9f1a2-107">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```

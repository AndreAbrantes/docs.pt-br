## <a name="example"></a><span data-ttu-id="5ba0e-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5ba0e-108">Example</span></span>

<span data-ttu-id="5ba0e-109">O código a seguir gera CS1955 porque o código está tentando invocar um campo e uma propriedade usando o [operador de invocação `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="5ba0e-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="5ba0e-110">Você não pode chamar um campo ou uma propriedade.</span><span class="sxs-lookup"><span data-stu-id="5ba0e-110">You cannot call a field or a property.</span></span> <span data-ttu-id="5ba0e-111">Use o [operador de acesso de membro `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) para acessar o valor que ele armazena.</span><span class="sxs-lookup"><span data-stu-id="5ba0e-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```

 <span data-ttu-id="5fe3c-104">Uma instrução yield não é permitida no corpo de uma cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="5fe3c-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="5fe3c-105">Para evitar esse erro, mover a instrução yield da cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="5fe3c-105">To avoid this error, move the yield statement out of the finally clause.</span></span>

```

using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}

{  
   public static void Main() { }  
}  
```

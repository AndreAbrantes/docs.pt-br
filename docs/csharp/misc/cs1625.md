# <a name="compiler-error-cs1625"></a><span data-ttu-id="0038f-103">Erro do Compilador CS1625</span><span class="sxs-lookup"><span data-stu-id="0038f-103">Compiler Error CS1625</span></span>
<span data-ttu-id="0038f-104">Não é possível produzir no corpo de uma cláusula finally</span><span class="sxs-lookup"><span data-stu-id="0038f-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="0038f-105">Uma instrução yield não é permitida no corpo de uma cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="0038f-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="0038f-106">Para evitar esse erro, mova a instrução yield para fora da cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="0038f-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="0038f-107">O exemplo a seguir gera CS1625:</span><span class="sxs-lookup"><span data-stu-id="0038f-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

 <span data-ttu-id="b8aa2-104">Uma instrução yield não é permitida no corpo de uma cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="b8aa2-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="b8aa2-105">Para evitar esse erro, mova a instrução yield da cláusula finally.</span><span class="sxs-lookup"><span data-stu-id="b8aa2-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="b8aa2-106">O exemplo a seguir gera CS1625:</span><span class="sxs-lookup"><span data-stu-id="b8aa2-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

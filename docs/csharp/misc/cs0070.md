# <a name="compiler-error-cs0070"></a><span data-ttu-id="19e61-103">Erro do Compilador CS0070</span><span class="sxs-lookup"><span data-stu-id="19e61-103">Compiler Error CS0070</span></span>
<span data-ttu-id="19e61-104">O evento ' Event ' só pode aparecer no lado esquerdo de + = ou-= (exceto quando usado de dentro do tipo ' type ')</span><span class="sxs-lookup"><span data-stu-id="19e61-104">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="19e61-105">Fora da classe em que ela está definida, um [evento](../language-reference/keywords/event.md) só pode adicionar ou subtrair referências.</span><span class="sxs-lookup"><span data-stu-id="19e61-105">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="19e61-106">Para obter mais informações, consulte [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="19e61-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="19e61-107">O exemplo a seguir gera CS0070:</span><span class="sxs-lookup"><span data-stu-id="19e61-107">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```

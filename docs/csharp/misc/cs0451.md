# <a name="compiler-error-cs0451"></a><span data-ttu-id="524e7-102">Erro do Compilador CS0451</span><span class="sxs-lookup"><span data-stu-id="524e7-102">Compiler Error CS0451</span></span>
<span data-ttu-id="524e7-103">A restrição 'new()' não pode ser usada com a restrição 'struct'</span><span class="sxs-lookup"><span data-stu-id="524e7-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="524e7-104">Ao especificar restrições sobre o tipo `new()` de genérico, a restrição só pode ser usada com restrições de tipo de classe, restrições de tipo de interface, restrições de tipo de referência e restrições de parâmetro de tipo, mas não com restrições de tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="524e7-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="524e7-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="524e7-105">Example</span></span>  
 <span data-ttu-id="524e7-106">A seguinte amostra gera CS0451.</span><span class="sxs-lookup"><span data-stu-id="524e7-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```

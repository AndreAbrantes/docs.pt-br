# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="a9afb-102">Compilador CS3021 de aviso (nível 2)</span><span class="sxs-lookup"><span data-stu-id="a9afb-102">Compiler Warning (level 2) CS3021</span></span>
<span data-ttu-id="a9afb-103">'type' não precisa de um atributo CLSCompliant porque o assembly não tem um atributo CLSCompliant</span><span class="sxs-lookup"><span data-stu-id="a9afb-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="a9afb-104">Este aviso ocorre se `[CLSCompliant(false)]` aparece em uma classe em um assembly que não tem um nível de assembly CLSCompliant atributo definido como verdadeiro (ou seja, a linha `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="a9afb-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="a9afb-105">Desde que o assembly é não declarar próprio CLS compatível, não é necessário para qualquer coisa dentro do assembly para declarar em si não compatíveis, pois ele é considerado não compatível.</span><span class="sxs-lookup"><span data-stu-id="a9afb-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="a9afb-106">Para obter mais informações sobre compatibilidade com CLS, consulte [escrevendo código compatível com CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="a9afb-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="a9afb-107">Para eliminar esse aviso, remova o atributo ou adicione o atributo de nível de assembly.</span><span class="sxs-lookup"><span data-stu-id="a9afb-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a9afb-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a9afb-108">Example</span></span>  
 <span data-ttu-id="a9afb-109">O exemplo a seguir gera CS3021:</span><span class="sxs-lookup"><span data-stu-id="a9afb-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```

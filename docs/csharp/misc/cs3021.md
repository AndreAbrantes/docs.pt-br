 <span data-ttu-id="7b0e8-104">Este aviso ocorre se `[CLSCompliant(false)]` aparece em uma classe em um assembly que não tem um nível de assembly CLSCompliant atributo definido como true (ou seja, a linha `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="7b0e8-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="7b0e8-105">Uma vez que o assembly é não declarar próprio CLS compatível, não é necessário para qualquer coisa dentro do assembly para declarar incompatíveis, pois ele é considerado incompatível.</span><span class="sxs-lookup"><span data-stu-id="7b0e8-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="7b0e8-106">Para obter mais informações sobre compatibilidade com CLS, consulte [escrevendo código compatível com CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="7b0e8-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>

## <a name="example"></a><span data-ttu-id="7b0e8-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7b0e8-108">Example</span></span>  
 <span data-ttu-id="7b0e8-109">O exemplo a seguir gera CS3021:</span><span class="sxs-lookup"><span data-stu-id="7b0e8-109">The following example generates CS3021:</span></span>  
  
```  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```

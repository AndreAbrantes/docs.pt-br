# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="7b1ba-102">Aviso do compilador (nível 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="7b1ba-102">Compiler Warning (level 2) CS3021</span></span>

<span data-ttu-id="7b1ba-103">' type ' não precisa de um atributo CLSCompliant porque o assembly não tem um atributo CLSCompliant</span><span class="sxs-lookup"><span data-stu-id="7b1ba-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="7b1ba-104">Esse aviso ocorre se `[CLSCompliant(false)]` aparecer em uma classe em um assembly que não tem um atributo CLSCompliant de nível de assembly definido como true (ou seja, a linha `[assembly: CLSCompliant(true)]` ).</span><span class="sxs-lookup"><span data-stu-id="7b1ba-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="7b1ba-105">Como o assembly não está declarando em conformidade com CLS, não há necessidade de nada dentro do assembly para declará-lo em não conformidade, pois supõe-se que não esteja em conformidade.</span><span class="sxs-lookup"><span data-stu-id="7b1ba-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="7b1ba-106">Para obter mais informações sobre a conformidade com CLS, consulte [independência de idioma e componentes independentes de linguagem](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="7b1ba-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>

## <a name="example"></a><span data-ttu-id="7b1ba-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7b1ba-108">Example</span></span>  

 <span data-ttu-id="7b1ba-109">O exemplo a seguir gera CS3021:</span><span class="sxs-lookup"><span data-stu-id="7b1ba-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```

# <a name="compiler-error-cs0517"></a>Erro do compilador CS0517
'class' não tem classe base e não é possível chamar um construtor base  
  
 CS0517 somente pode ocorrer quando o .NET Framework common language runtime compila o código-fonte para a classe de objeto, que é a única classe que não tem classe base.

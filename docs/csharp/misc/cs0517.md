# <a name="compiler-error-cs0517"></a>Erro do Compilador CS0517
' class ' não tem nenhuma classe base e não pode chamar um construtor base  
  
 CS0517 pode ocorrer somente quando o .NET Framework Common Language Runtime compila o código-fonte da classe de objeto, que é a única classe que não tem nenhuma classe base.

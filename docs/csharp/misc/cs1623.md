# <a name="compiler-error-cs1623"></a><span data-ttu-id="49323-102">CS1623 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="49323-102">Compiler Error CS1623</span></span>

 <span data-ttu-id="49323-104">Esse erro ocorre se um método iterador usa um `ref` ou `out` parâmetro.</span><span class="sxs-lookup"><span data-stu-id="49323-104">This error occurs if an iterator method takes a `ref` or `out` parameter.</span></span> <span data-ttu-id="49323-105">Para evitar esse erro, remova o `ref` ou `out` palavra-chave de assinatura de método.</span><span class="sxs-lookup"><span data-stu-id="49323-105">To avoid this error, remove the `ref` or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="49323-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="49323-106">Example</span></span>  
 <span data-ttu-id="49323-107">O exemplo a seguir gera CS1623:</span><span class="sxs-lookup"><span data-stu-id="49323-107">The following sample generates CS1623:</span></span>  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```

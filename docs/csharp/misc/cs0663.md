# <a name="compiler-error-cs0663"></a><span data-ttu-id="b8e8a-102">Erro do compilador CS0663</span><span class="sxs-lookup"><span data-stu-id="b8e8a-102">Compiler Error CS0663</span></span>
<span data-ttu-id="b8e8a-103">Não é possível definir métodos sobrecarregados que diferem somente em ref e out.</span><span class="sxs-lookup"><span data-stu-id="b8e8a-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="b8e8a-104">Métodos que diferem apenas em seu uso do [na](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) e [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) em um parâmetro não são permitidos.</span><span class="sxs-lookup"><span data-stu-id="b8e8a-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="b8e8a-105">O exemplo a seguir gera CS0663:</span><span class="sxs-lookup"><span data-stu-id="b8e8a-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```

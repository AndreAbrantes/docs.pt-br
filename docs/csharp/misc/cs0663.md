# <a name="compiler-error-cs0663"></a><span data-ttu-id="ada14-102">CS0663 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="ada14-102">Compiler Error CS0663</span></span>
<span data-ttu-id="ada14-103">Não é possível definir métodos sobrecarregados que diferem somente em ref e out.</span><span class="sxs-lookup"><span data-stu-id="ada14-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="ada14-104">Métodos que diferem apenas em termos de uso [ref](../../csharp/language-reference/keywords/ref.md) e [out](../../csharp/language-reference/keywords/out.md) em um parâmetro não são permitidos.</span><span class="sxs-lookup"><span data-stu-id="ada14-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="ada14-105">O exemplo a seguir gera CS0663:</span><span class="sxs-lookup"><span data-stu-id="ada14-105">The following sample generates CS0663:</span></span>  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```

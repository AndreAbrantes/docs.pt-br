# <a name="compiler-error-cs0663"></a><span data-ttu-id="9dbc5-103">Erro do Compilador CS0663</span><span class="sxs-lookup"><span data-stu-id="9dbc5-103">Compiler Error CS0663</span></span>
<span data-ttu-id="9dbc5-104">Não é possível definir métodos sobrecarregados que diferem apenas em ref e out.</span><span class="sxs-lookup"><span data-stu-id="9dbc5-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="9dbc5-105">Os métodos que diferem apenas no uso de [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) e [out](../language-reference/keywords/out-parameter-modifier.md) em um parâmetro não são permitidos.</span><span class="sxs-lookup"><span data-stu-id="9dbc5-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="9dbc5-106">O exemplo a seguir gera CS0663:</span><span class="sxs-lookup"><span data-stu-id="9dbc5-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```

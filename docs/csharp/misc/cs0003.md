# <a name="compiler-error-cs0003"></a>CS0003 de erro do compilador
Memória insuficiente  
  
 O compilador não pôde alocar memória virtual insuficiente para concluir a compilação. Feche todos os aplicativos desnecessários e compile novamente.  
  
 Talvez você queira aumentar o tamanho do arquivo de paginação e certifique-se de que você tenha espaço livre em disco.  
  
 Esse erro também pode aparecer como o resultado de ter diferentes versões do [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] e o compilador c# ou um ou mais arquivos corrompidos que oferece suporte para o compilador c#; reinstale o Visual Studio.

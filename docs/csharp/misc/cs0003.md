# <a name="compiler-error-cs0003"></a>Erro do compilador CS0003
Memória insuficiente  
  
 O compilador não pôde alocar memória virtual suficiente para concluir a compilação. Feche todos os aplicativos desnecessários e compile novamente.  
  
 Talvez você queira aumentar o tamanho do arquivo de paginação e certifique-se de que você tem espaço livre em disco.  
  
 Esse erro também pode aparecer como o resultado de ter incompatíveis versões do Windows Software Development Kit (SDK) e o C# arquivos de compilador ou um ou mais corrompido que dão suporte a C# compilador; Reinstale o Visual Studio.

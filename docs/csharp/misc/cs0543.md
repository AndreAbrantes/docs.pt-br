# <a name="compiler-error-cs0543"></a>Erro do compilador CS0543
' Enumeration ': o valor do enumerador é muito grande para se ajustar ao seu tipo  
  
 Um valor que foi atribuído a um elemento em uma [Enumeração](../language-reference/keywords/enum.md) está fora do intervalo do tipo de dados.  
  
 O exemplo a seguir gera CS0543:  
  
```csharp  
// CS0543.cs  
namespace x  
{  
   enum I : byte  
   {a = 255, b, c}   // CS0543  
   public class clx  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

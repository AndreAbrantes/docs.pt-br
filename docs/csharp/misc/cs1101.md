# <a name="compiler-error-cs1101"></a><span data-ttu-id="2defb-103">Erro do Compilador CS1101</span><span class="sxs-lookup"><span data-stu-id="2defb-103">Compiler Error CS1101</span></span>

<span data-ttu-id="2defb-104">O modificador de parâmetro ' ref ' não pode ser usado com ' this '.</span><span class="sxs-lookup"><span data-stu-id="2defb-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="2defb-105">Quando a `this` palavra-chave modifica o primeiro parâmetro de um método estático, ela é sinalizada ao compilador que o método é um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="2defb-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="2defb-106">Nenhum outro modificador é necessário ou permitido no primeiro parâmetro de um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="2defb-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2defb-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2defb-107">Example</span></span>  

 <span data-ttu-id="2defb-108">O exemplo a seguir gera CS1101:</span><span class="sxs-lookup"><span data-stu-id="2defb-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```

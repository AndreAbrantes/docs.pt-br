<span data-ttu-id="9408f-103">Não é possível obter o endereço de uma variável local somente leitura</span><span class="sxs-lookup"><span data-stu-id="9408f-103">Cannot take the address of a read-only local variable</span></span>

 <span data-ttu-id="9408f-104">Há três cenários comuns no C# idioma que geram variáveis locais somente leitura: `foreach`, `using` e `fixed`.</span><span class="sxs-lookup"><span data-stu-id="9408f-104">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="9408f-105">Em cada um desses casos, você não tem permissão para gravar na variável local somente leitura ou para obter seu endereço.</span><span class="sxs-lookup"><span data-stu-id="9408f-105">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="9408f-106">Esse erro é gerado quando o compilador percebe que você está tentando obter o endereço de uma variável local somente leitura.</span><span class="sxs-lookup"><span data-stu-id="9408f-106">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>

## <a name="example"></a><span data-ttu-id="9408f-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9408f-107">Example</span></span>

 <span data-ttu-id="9408f-108">O exemplo a seguir gera CS0459 quando é feita uma tentativa de obter o endereço de uma variável local somente leitura em um loop de `foreach` e em um bloco de instrução `fixed`:</span><span class="sxs-lookup"><span data-stu-id="9408f-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```

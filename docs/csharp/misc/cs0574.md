# <a name="compiler-error-cs0574"></a>CS0574 de erro do compilador
O nome do destruidor deve corresponder ao nome da classe  
  
 O nome de um destruidor deve ser o nome da classe precedido por um til (~).  
  
 O exemplo a seguir gera CS0574:  
  
```  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```

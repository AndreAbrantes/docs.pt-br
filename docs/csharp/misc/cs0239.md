 <span data-ttu-id="5dbe4-105">Um membro não pode [substituir](../language-reference/keywords/override.md) um membro herdado [lacrado](../language-reference/keywords/sealed.md) .</span><span class="sxs-lookup"><span data-stu-id="5dbe4-105">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="5dbe4-106">Para obter mais informações, consulte [marcada e desmarcada](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="5dbe4-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="5dbe4-107">O exemplo a seguir gera CS0239:</span><span class="sxs-lookup"><span data-stu-id="5dbe4-107">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```

 <span data-ttu-id="1d8e1-104">Não é um membro [substituir](../../csharp/language-reference/keywords/override.md) um [lacrado](../../csharp/language-reference/keywords/sealed.md) membro herdado.</span><span class="sxs-lookup"><span data-stu-id="1d8e1-104">A member cannot [override](../../csharp/language-reference/keywords/override.md) a [sealed](../../csharp/language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="1d8e1-105">Para obter mais informações, consulte [marcado e desmarcado](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="1d8e1-105">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>

```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```

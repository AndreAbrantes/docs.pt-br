# <a name="compiler-warning-level-4-cs0422"></a>Compilador aviso (nível 4) CS0422
O / que não há suporte para a opção incremental  
  
 Compilação incremental (**/incr** ou **/incremental**) não tem suporte no Visual C# 2005.

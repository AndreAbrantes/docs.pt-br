# <a name="compiler-warning-level-4-cs0422"></a>Aviso do compilador (nível 4) CS0422

A opção /incremental não é mais suportada

 A compilação incremental **(/incr** ou **/incremental)** não é suportada no C# 2.0.

> [!NOTE]
> Este aviso de compilador não é mais usado em Roslyn.

# <a name="compiler-warning-level-4-cs0422"></a>Compilador CS0422 de aviso (nível 4)
Não há mais suporte para a opção /incremental  
  
 Compilação incremental (**/incr** ou **/incremental**) não é suportado no Visual c# 2005.

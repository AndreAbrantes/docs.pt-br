# <a name="compiler-warning-level-4-cs0422"></a>Aviso do compilador (nível 4) CS0422

Não há mais suporte para a opção/incremental

 A compilação incremental ( **/incr** ou **/incremental**) não tem suporte C# no 2,0.
 
> [!NOTE]
> Este aviso do compilador não é mais usado no Roslyn.

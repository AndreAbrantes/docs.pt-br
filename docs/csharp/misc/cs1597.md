# <a name="compiler-error-cs1597"></a>Erro do compilador CS1597
-E-vírgula após bloco de acessador ou método não é válido  
  
 Ponto e vírgula não é necessário (ou permitida) ao final de um método ou o bloco de acessador.  
  
 O exemplo a seguir gera CS1597:  
  
```csharp  
// CS1597.cs  
class TestClass  
{  
   public static void Main()  
   {  
   };   // CS1597, remove semicolon  
}  
```

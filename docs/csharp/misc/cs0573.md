# <a name="compiler-error-cs0573"></a>Erro do compilador CS0573
' Declaração de campo ': não é possível ter inicializadores de campo de instância em structs  
  
 Não é possível inicializar um campo de instância de um [struct](../language-reference/builtin-types/struct.md). Os campos dos tipos de valor serão inicializados para seus valores padrão e os campos de tipo de referência serão inicializados para `null`.  
  
## <a name="example"></a>{1&gt;Exemplo&lt;1}  
 O exemplo a seguir gera CS0573:  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```

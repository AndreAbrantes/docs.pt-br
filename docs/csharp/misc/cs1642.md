## <a name="example"></a><span data-ttu-id="40c49-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="40c49-106">Example</span></span>  
 <span data-ttu-id="40c49-107">O exemplo a seguir gera CS1642.</span><span class="sxs-lookup"><span data-stu-id="40c49-107">The following sample generates CS1642.</span></span>

```  
// CS1642.cs

{  
   fixed int a[10];   // CS1642  
}

    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```

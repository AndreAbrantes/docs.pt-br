# <a name="compiler-error-cs0401"></a>CS0401 de erro do compilador
A restrição new() deve ser a última especificada  
  
 Ao usar várias restrições, liste todas as outras restrições antes da restrição New ().  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0401.  
  
```  
// CS0401.cs  
// compile with: /target:library  
using System;  
 class C<T> where T : new(), IDisposable {}  // CS0401  
  
class D<T> where T : IDisposable  
{  
   static void F<U>() where U : new(), IDisposable{}   // CS0401  
}  
```

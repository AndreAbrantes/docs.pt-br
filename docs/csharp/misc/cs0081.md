# <a name="compiler-error-cs0081"></a><span data-ttu-id="b3bba-102">CS0081 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="b3bba-102">Compiler Error CS0081</span></span>
<span data-ttu-id="b3bba-103">Declaração de parâmetro de tipo deve ser um identificador não é um tipo</span><span class="sxs-lookup"><span data-stu-id="b3bba-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="b3bba-104">Quando você declara um método genérico ou tipo, especifique o parâmetro de tipo como um identificador, por exemplo, "T" ou "inputType".</span><span class="sxs-lookup"><span data-stu-id="b3bba-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="b3bba-105">Quando o código do cliente chama o método, ele fornece o tipo, que substitui cada ocorrência do identificador no corpo do método ou classe.</span><span class="sxs-lookup"><span data-stu-id="b3bba-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="b3bba-106">Para obter mais informações, consulte [parâmetros de tipo genéricos](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="b3bba-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```

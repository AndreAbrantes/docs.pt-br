# <a name="compiler-error-cs1647"></a>Erro do compilador CS1647
Uma expressão é muito longa ou complexa para ser compilada próximo a 'code'  
  
 Houve um estouro de pilha no compilador seu código de processamento. Para resolver esse erro, simplificar seu código. Se seu código for válido, obtenha ajuda na [comunidade de desenvolvedores](https://developercommunity.visualstudio.com/spaces/61/index.html) site ou [StackOverflow](https://stackoverflow.com/).

# <a name="compiler-error-cs1647"></a>CS1647 de erro do compilador
Uma expressão é muito longa ou complexa para ser compilada próximo a 'code'  
  
 Houve um estouro de pilha no compilador seu código de processamento. Para resolver esse erro, simplificar seu código. Se seu código é válido, entre em contato com [suporte ao produto](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).

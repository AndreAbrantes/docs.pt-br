# <a name="compiler-error-cs2003"></a>CS2003 de erro do compilador
O arquivo de resposta 'file' incluído várias vezes  
  
 Um arquivo de resposta foi passado para o compilador mais de uma vez. Um arquivo de resposta somente pode ser passado para o compilador uma vez por arquivo de saída.  
  
 Para obter mais informações sobre arquivos de resposta, consulte [@ (especificar arquivo de resposta)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).

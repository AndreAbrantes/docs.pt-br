<span data-ttu-id="73376-103">Um membro estático 'function' não pode ser marcado como override, virtual ou abstract</span><span class="sxs-lookup"><span data-stu-id="73376-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="73376-104">Qualquer declaração de método que usa o `override`, **virtual**, ou **abstrata** palavra-chave não é possível usar também o **estático** palavra-chave.</span><span class="sxs-lookup"><span data-stu-id="73376-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="73376-105">Para saber mais, veja [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="73376-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="73376-106">O exemplo a seguir gera CS0112:</span><span class="sxs-lookup"><span data-stu-id="73376-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

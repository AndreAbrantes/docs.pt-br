# <a name="compiler-error-cs1626"></a><span data-ttu-id="36559-102">Erro do compilador CS1626</span><span class="sxs-lookup"><span data-stu-id="36559-102">Compiler Error CS1626</span></span>
<span data-ttu-id="36559-103">Não é possível usar a instrução yield no corpo de um bloco try com uma cláusula catch</span><span class="sxs-lookup"><span data-stu-id="36559-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="36559-104">Uma instrução yield não é permitida em um bloco try, se houver uma cláusula catch associada com o bloco try.</span><span class="sxs-lookup"><span data-stu-id="36559-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="36559-105">Para evitar esse erro, mova a instrução yield fora do bloco try/catch/finally ou remova o bloco catch.</span><span class="sxs-lookup"><span data-stu-id="36559-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="36559-106">O exemplo a seguir gera CS1626:</span><span class="sxs-lookup"><span data-stu-id="36559-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {
      
      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

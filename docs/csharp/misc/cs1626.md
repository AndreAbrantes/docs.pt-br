<span data-ttu-id="60702-103">Não é possível usar a instrução yield no corpo de um bloco try com uma cláusula catch</span><span class="sxs-lookup"><span data-stu-id="60702-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="60702-104">Uma instrução yield não é permitida em um bloco try, se houver uma cláusula catch associada ao bloco try.</span><span class="sxs-lookup"><span data-stu-id="60702-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="60702-105">Para evitar esse erro, mova a instrução yield sem a cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="60702-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="60702-106">O exemplo a seguir gera CS1626:</span><span class="sxs-lookup"><span data-stu-id="60702-106">The following sample generates CS1626:</span></span>  
  
```  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

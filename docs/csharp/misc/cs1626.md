 <span data-ttu-id="7ab95-105">Uma instrução yield não é permitida em um bloco try se houver uma cláusula catch associada ao bloco try.</span><span class="sxs-lookup"><span data-stu-id="7ab95-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="7ab95-106">Para evitar esse erro, mova a instrução yield para fora do bloco try/catch/finally ou remova o bloco catch.</span><span class="sxs-lookup"><span data-stu-id="7ab95-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>

class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626

      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}

{  
   public static void Main() { }  
}

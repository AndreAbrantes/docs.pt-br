# <a name="compiler-error-cs0040"></a>Erro do compilador CS0040
Erro inesperado ao criar o arquivo de informações de depuração — 'reason'  
  
 Esse erro pode ocorrer ao usar o [/Debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) opção de compilador e indica que o compilador não pôde gravar no arquivo. PDB. Possíveis resoluções para esse erro incluem reinstalando o Visual Studio, garantindo que o compilador tem acesso de gravação para um arquivo ou diretório ou não compilar com a opção /debug.

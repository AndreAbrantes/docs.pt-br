# <a name="compiler-error-cs0040"></a>CS0040 de erro do compilador
Erro inesperado ao criar o arquivo de informações de depuração — 'motivo'  
  
 Esse erro pode ocorrer ao usar o [/Debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) opção de compilador e indica que o compilador não pôde gravar no arquivo. PDB. Possíveis resoluções para esse erro incluem reinstalar o Visual Studio, garantindo que o compilador tenha acesso de gravação para um arquivo ou diretório ou não compilar com /debug.

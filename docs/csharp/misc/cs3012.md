# <a name="compiler-warning-level-1-cs3012"></a>Compilador CS3012 de aviso (nível 1)
Você não pode especificar o atributo CLSCompliant em um módulo que difira do atributo CLSCompliant no assembly  
  
 Para que seja compatível com o Common Language Specification (CLS) por meio de um módulo `[module:System.CLCSompliant(true)]`, que deve ser compilado com o [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) opção de compilador. Para obter mais informações sobre a CLS, consulte [independência da linguagem e componentes independentes da linguagem](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir, quando compilado sem `/target:module`, gera CS3012:  
  
```  
// CS3012.cs  
// compile with: /W:1  
  
[module:System.CLSCompliant(true)]   // CS3012  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```

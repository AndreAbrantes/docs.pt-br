# <a name="compiler-error-cs0014"></a>Erro do compilador CS0014
Não foi possível encontrar o arquivo necessário 'file'  
  
 Um arquivo é necessário pelo compilador, mas não está no sistema. Certifique-se de que o caminho está correto. Se o arquivo for um arquivo de sistema do Visual Studio, em seguida, você talvez precise reparar sua instalação ou então remover e reinstalar completamente o Visual Studio.

# <a name="compiler-error-cs0014"></a>CS0014 de erro do compilador
Não foi possível encontrar o arquivo necessário 'file'  
  
 Um arquivo é necessário pelo compilador, mas não está no sistema. Certifique-se de que o caminho está correto. Se o arquivo é um arquivo de sistema do Visual Studio, em seguida, você precisará repare a instalação ou remover e reinstalar completamente o Visual Studio.

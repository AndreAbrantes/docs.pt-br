# <a name="compiler-error-cs0014"></a>CS0014 de erro do compilador
Não foi possível encontrar o arquivo necessário 'file'  
  
 Um arquivo é necessário pelo compilador, mas não está no sistema. Certifique-se de que o caminho está correto. Se o arquivo for um [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] arquivo do sistema, talvez seja necessário reparar sua instalação ou então remova e reinstale [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] completamente.

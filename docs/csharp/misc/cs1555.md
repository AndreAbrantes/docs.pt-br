# <a name="compiler-error-cs1555"></a>CS1555 de erro do compilador
Não foi possível encontrar 'class' especificado para método Main  
  
 Uma classe foi especificada para o [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador, mas o nome da classe não foi encontrado no código-fonte.

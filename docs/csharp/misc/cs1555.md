# <a name="compiler-error-cs1555"></a>Erro do compilador CS1555
Não foi possível encontrar ' class ' especificado para o método Main  
  
 Uma classe foi especificada para a opção de compilador [/Main](../language-reference/compiler-options/main-compiler-option.md) , mas o nome de classe não foi encontrado no código-fonte.

# <a name="compiler-error-cs1555"></a>CS1555 de erro do compilador
Não foi possível encontrar o 'class' especificado para o método Main  
  
 Uma classe especificada para o [/principal](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador, mas o nome da classe não foi encontrada no código-fonte.

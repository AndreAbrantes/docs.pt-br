# <a name="compiler-error-cs1555"></a>Erro do compilador CS1555
Não foi possível encontrar o 'class' especificado para o método Main  
  
 Uma classe que foi especificada para o [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador, mas o nome de classe não foi encontrado no código-fonte.

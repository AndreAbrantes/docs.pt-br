## <a name="example"></a><span data-ttu-id="d2f69-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d2f69-109">Example</span></span>  
 <span data-ttu-id="d2f69-110">O exemplo a seguir gera CS0197.</span><span class="sxs-lookup"><span data-stu-id="d2f69-110">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```

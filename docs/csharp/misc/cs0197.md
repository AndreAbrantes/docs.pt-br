# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="3e52e-102">Compilador CS0197 de aviso (nível 1)</span><span class="sxs-lookup"><span data-stu-id="3e52e-102">Compiler Warning (level 1) CS0197</span></span>
<span data-ttu-id="3e52e-103">Passar 'argumento' como ref ou out ou obter seu endereço pode gerar uma exceção de tempo de execução porque ele é um campo de uma classe de marshaling por referência</span><span class="sxs-lookup"><span data-stu-id="3e52e-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="3e52e-104">Qualquer classe derivada, direta ou indiretamente, de <xref:System.MarshalByRefObject> é uma classe de marshaling por referência.</span><span class="sxs-lookup"><span data-stu-id="3e52e-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="3e52e-105">Essa classe pode ser empacotado por referência entre limites de processo e de máquina.</span><span class="sxs-lookup"><span data-stu-id="3e52e-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="3e52e-106">Portanto, as instâncias dessa classe podem ser proxies a objetos remotos.</span><span class="sxs-lookup"><span data-stu-id="3e52e-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="3e52e-107">Você não pode passar um campo de um objeto de proxy como [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../../csharp/language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3e52e-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="3e52e-108">Example</span></span>  
 <span data-ttu-id="3e52e-109">O exemplo a seguir gera CS0197.</span><span class="sxs-lookup"><span data-stu-id="3e52e-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```

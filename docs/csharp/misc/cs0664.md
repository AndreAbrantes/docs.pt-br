 <span data-ttu-id="f955e-104">Uma atribuição não pôde ser concluída; Use um sufixo para corrigir a instrução.</span><span class="sxs-lookup"><span data-stu-id="f955e-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="f955e-105">A documentação de cada tipo identifica o sufixo correspondente para o tipo.</span><span class="sxs-lookup"><span data-stu-id="f955e-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="f955e-106">Para obter mais informações sobre conversões, consulte [conversão e conversões de tipo](../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="f955e-106">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="f955e-107">O exemplo a seguir gera CS0664:</span><span class="sxs-lookup"><span data-stu-id="f955e-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```

# <a name="compiler-error-cs1906"></a>Erro do compilador CS1906
Opção inválida ' Option '; A visibilidade do recurso deve ser ' Public ' ou ' Private '  
  
 Esse erro indica um [/Resource inválido (inserir arquivo de recurso para saída) ou a](../language-reference/compiler-options/resource-compiler-option.md) opção de linha de comando [/linkresource (link para .NET Framework recurso)](../language-reference/compiler-options/linkresource-compiler-option.md) . Verifique a sintaxe da opção de linha de comando **/Resource** ou **/linkresource** e certifique-se de que o modificador de acessibilidade usado `private`seja **público** ou.

# <a name="compiler-error-cs0531"></a><span data-ttu-id="7fcd6-102">Erro do compilador CS0531</span><span class="sxs-lookup"><span data-stu-id="7fcd6-102">Compiler Error CS0531</span></span>
<span data-ttu-id="7fcd6-103">' member ': membros de interface não podem ter uma definição</span><span class="sxs-lookup"><span data-stu-id="7fcd6-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="7fcd6-104">Métodos que são declarados em uma [interface](../language-reference/keywords/interface.md) devem ser implementados em uma classe que herda dele e não na própria interface.</span><span class="sxs-lookup"><span data-stu-id="7fcd6-104">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="7fcd6-105">O exemplo a seguir gera CS0531:</span><span class="sxs-lookup"><span data-stu-id="7fcd6-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="cf639-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="cf639-106">Example</span></span>  

 <span data-ttu-id="cf639-107">O exemplo a seguir gera CS1529:</span><span class="sxs-lookup"><span data-stu-id="cf639-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };

    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}

```

# <a name="compiler-error-cs2006"></a>CS2006 de erro do compilador
Erro de sintaxe de linha de comando: 'texto' para a opção 'option' ausente  
  
 A sintaxe para *opção* requer texto adicional. Para obter informações, consulte [opções do compilador](../../csharp/language-reference/compiler-options/index.md).

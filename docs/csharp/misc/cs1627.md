# <a name="compiler-error-cs1627"></a><span data-ttu-id="f52c9-102">Erro do compilador CS1627</span><span class="sxs-lookup"><span data-stu-id="f52c9-102">Compiler Error CS1627</span></span>
<span data-ttu-id="f52c9-103">Expressão esperada após yield return</span><span class="sxs-lookup"><span data-stu-id="f52c9-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="f52c9-104">Esse erro ocorre se `yield` for usado sem uma expressão.</span><span class="sxs-lookup"><span data-stu-id="f52c9-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="f52c9-105">Para evitar esse erro, insira a expressão apropriada na instrução.</span><span class="sxs-lookup"><span data-stu-id="f52c9-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="f52c9-106">O exemplo a seguir gera CS1627:</span><span class="sxs-lookup"><span data-stu-id="f52c9-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

# <a name="compiler-error-cs0027"></a><span data-ttu-id="dcf05-102">Erro do Compilador CS0027</span><span class="sxs-lookup"><span data-stu-id="dcf05-102">Compiler Error CS0027</span></span>
<span data-ttu-id="dcf05-103">A palavra-chave 'isso' não está disponível no contexto atual</span><span class="sxs-lookup"><span data-stu-id="dcf05-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="dcf05-104">Esta [this](../language-reference/keywords/this.md) palavra-chave foi encontrada fora de uma propriedade, método ou construtor.</span><span class="sxs-lookup"><span data-stu-id="dcf05-104">The [this](../language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="dcf05-105">Para corrigir esse erro, modifique a `this` declaração para eliminar o uso da palavra-chave e/ou mova parte ou toda a declaração dentro de uma propriedade, método ou construtor.</span><span class="sxs-lookup"><span data-stu-id="dcf05-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="dcf05-106">O exemplo a seguir gera CS0027:</span><span class="sxs-lookup"><span data-stu-id="dcf05-106">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```

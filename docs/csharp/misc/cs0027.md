# <a name="compiler-error-cs0027"></a><span data-ttu-id="91d21-102">CS0027 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="91d21-102">Compiler Error CS0027</span></span>
<span data-ttu-id="91d21-103">A palavra-chave 'this' não está disponível no contexto atual</span><span class="sxs-lookup"><span data-stu-id="91d21-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="91d21-104">O [isso](../../csharp/language-reference/keywords/this.md) palavra-chave foi encontrado fora de uma propriedade, um método ou um construtor.</span><span class="sxs-lookup"><span data-stu-id="91d21-104">The [this](../../csharp/language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="91d21-105">Para corrigir esse erro, modifique a instrução para eliminar o uso de `this` palavra-chave, e/ou mover parte ou todos da instrução dentro de uma propriedade, um método ou um construtor.</span><span class="sxs-lookup"><span data-stu-id="91d21-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="91d21-106">O exemplo a seguir gera CS0027:</span><span class="sxs-lookup"><span data-stu-id="91d21-106">The following example generates CS0027:</span></span>  
  
```  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027   
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```

 <span data-ttu-id="932b5-104">Quando você declara um [propriedade](../../csharp/programming-guide/classes-and-structs/using-properties.md), você também deverá declarar seus métodos de acessador.</span><span class="sxs-lookup"><span data-stu-id="932b5-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="932b5-105">No entanto, uma propriedade não pode ter mais de um `get` método de acessador ou mais de um `set` método do acessador.</span><span class="sxs-lookup"><span data-stu-id="932b5-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="932b5-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="932b5-106">Example</span></span>  
 <span data-ttu-id="932b5-107">O exemplo a seguir gera CS1007:</span><span class="sxs-lookup"><span data-stu-id="932b5-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs

{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```

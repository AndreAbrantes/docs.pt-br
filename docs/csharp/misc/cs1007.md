## <a name="example"></a><span data-ttu-id="227e3-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="227e3-107">Example</span></span>  
 <span data-ttu-id="227e3-108">O exemplo a seguir gera CS1007:</span><span class="sxs-lookup"><span data-stu-id="227e3-108">The following sample generates CS1007:</span></span>

```csharp  
            return 0;

        }  
    }  
  
    public static void Main() {}  
}  
```

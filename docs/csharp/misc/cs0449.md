# <a name="compiler-error-cs0449"></a><span data-ttu-id="059af-103">Erro do Compilador CS0449</span><span class="sxs-lookup"><span data-stu-id="059af-103">Compiler Error CS0449</span></span>
<span data-ttu-id="059af-104">A restrição ' class ' ou ' struct ' deve vir antes de qualquer outra restrição</span><span class="sxs-lookup"><span data-stu-id="059af-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="059af-105">As restrições no parâmetro de tipo de um tipo ou método genérico devem ocorrer em uma ordem específica: `class` ou `struct` devem ser as primeiras, se presentes, em seguida quaisquer restrições de interface e, finalmente, quaisquer restrições de construtor.</span><span class="sxs-lookup"><span data-stu-id="059af-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="059af-106">Esse erro é causado pela `class` restrição or que `struct` não aparece primeiro.</span><span class="sxs-lookup"><span data-stu-id="059af-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="059af-107">Para resolver esse erro, reordene as cláusulas de restrição.</span><span class="sxs-lookup"><span data-stu-id="059af-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="059af-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="059af-108">Example</span></span>  
 <span data-ttu-id="059af-109">O exemplo a seguir gera CS0449.</span><span class="sxs-lookup"><span data-stu-id="059af-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```

<span data-ttu-id="fdea6-103">A restrição 'class' ou 'struct' deve vir antes de qualquer outra restrição</span><span class="sxs-lookup"><span data-stu-id="fdea6-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="fdea6-104">As restrições no parâmetro de tipo de um tipo genérico ou método devem ocorrer em uma ordem específica: `class` ou `struct` deve ser primeiro, se presente, em seguida, as restrições de interface e, finalmente, qualquer restrição de construtor.</span><span class="sxs-lookup"><span data-stu-id="fdea6-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="fdea6-105">Esse erro é causado pelo `class` ou `struct` restrição não aparecendo primeiro.</span><span class="sxs-lookup"><span data-stu-id="fdea6-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="fdea6-106">Para resolver esse erro, reordene as cláusulas de restrição.</span><span class="sxs-lookup"><span data-stu-id="fdea6-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fdea6-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="fdea6-107">Example</span></span>  
 <span data-ttu-id="fdea6-108">O exemplo a seguir gera CS0449.</span><span class="sxs-lookup"><span data-stu-id="fdea6-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```

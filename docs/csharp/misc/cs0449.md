## <a name="example"></a><span data-ttu-id="f7e85-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f7e85-107">Example</span></span>  
 <span data-ttu-id="f7e85-108">A seguinte amostra gera CS0449.</span><span class="sxs-lookup"><span data-stu-id="f7e85-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```

# <a name="compiler-error-cs1585"></a>Erro do compilador CS1585
Modificador de membro 'palavra-chave' deve preceder o nome e o tipo de membro  
  
 O especificador de acesso em uma assinatura de método não existia no local correto.  
  
 O exemplo a seguir gera CS1585:  
  
```csharp  
// CS1585.cs  
public class Class1  
{  
   public void static Main(string[] args)   // CS1585  
   // try the following line instead  
   // public static void Main(string[] args)  
   {  
   }  
}  
```

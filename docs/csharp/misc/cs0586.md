# <a name="compiler-error-cs0586"></a>CS0586 de erro do compilador
Erro interno do compilador: etapa 'estágio'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se não for o caso, tente [Fale conosco](https://docs.microsoft.com/visualstudio/ide/talk-to-us).

# <a name="compiler-error-cs1713"></a>CS1713 de erro do compilador
Erro inesperado ao criar nome de metadados para o tipo Typename1 — 'Motivo'  
  
 Este erro geralmente é causado por um erro interno do compilador. Fazer algumas pequenas alterações no código, como encurtar o comprimento do nome e, em seguida, a recompilação, pode resolver esse erro.

# <a name="compiler-error-cs1713"></a>CS1713 de erro do compilador
Erro inesperado ao criar nome de metadados para tipo Typename1 — 'Razão'  
  
 Esse erro geralmente é causado por um erro interno do compilador. Fazer algumas pequenas alterações no código, como encurtar o comprimento do nome e, em seguida, recompilar, pode resolver esse erro.

# <a name="compiler-error-cs0761"></a><span data-ttu-id="36feb-102">Erro do compilador CS0761</span><span class="sxs-lookup"><span data-stu-id="36feb-102">Compiler Error CS0761</span></span>
<span data-ttu-id="36feb-103">As declarações de método parcial de\<' method T > ' têm restrições de parâmetro de tipo inconsistentes.</span><span class="sxs-lookup"><span data-stu-id="36feb-103">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="36feb-104">Se um método parcial tiver uma implementação, a restrição de tipo genérico deverá ser idêntica à restrição definida na assinatura do método.</span><span class="sxs-lookup"><span data-stu-id="36feb-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="36feb-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="36feb-105">To correct this error</span></span>

# <a name="compiler-error-cs0761"></a><span data-ttu-id="727fe-103">Erro do Compilador CS0761</span><span class="sxs-lookup"><span data-stu-id="727fe-103">Compiler Error CS0761</span></span>
<span data-ttu-id="727fe-104">As declarações de método parcial de ' method \<T> ' têm restrições de parâmetro de tipo inconsistentes.</span><span class="sxs-lookup"><span data-stu-id="727fe-104">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="727fe-105">Se um método parcial tiver uma implementação, a restrição de tipo genérico deverá ser idêntica à restrição definida na assinatura do método.</span><span class="sxs-lookup"><span data-stu-id="727fe-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="727fe-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="727fe-106">To correct this error</span></span>  
  
1. <span data-ttu-id="727fe-107">Torne as restrições de tipo genérico idênticas em cada parte do método parcial.</span><span class="sxs-lookup"><span data-stu-id="727fe-107">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="727fe-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="727fe-108">Example</span></span>  
 <span data-ttu-id="727fe-109">O código a seguir gera CS0761:</span><span class="sxs-lookup"><span data-stu-id="727fe-109">The following code generates CS0761:</span></span>

```csharp  
{  
    partial void Part<T>() where T : class;  
    partial void Part<T>() where T : struct // CS0761

    }

    {  
        return 1;  
    }  
}  
```

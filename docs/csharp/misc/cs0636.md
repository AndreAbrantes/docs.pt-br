# <a name="compiler-error-cs0636"></a><span data-ttu-id="93d91-102">CS0636 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="93d91-102">Compiler Error CS0636</span></span>
<span data-ttu-id="93d91-103">O atributo FieldOffset só pode ser colocado em membros de tipos marcados com StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="93d91-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="93d91-104">Você deve usar o **StructLayout(LayoutKind.Explicit)** atributo na declaração de estrutura, se ele contém todos os membros marcados com o **FieldOffset** atributo.</span><span class="sxs-lookup"><span data-stu-id="93d91-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="93d91-105">Para obter mais informações, consulte [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="93d91-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="93d91-106">O exemplo a seguir gera CS0636:</span><span class="sxs-lookup"><span data-stu-id="93d91-106">The following sample generates CS0636:</span></span>  
  
```  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```

<span data-ttu-id="2438a-103">O atributo FieldOffset só pode ser colocado em membros de tipos marcados com o StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="2438a-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="2438a-104">Você deve usar o **StructLayout(LayoutKind.Explicit)** atributo na declaração de struct, se ele contém todos os membros marcados com o **FieldOffset** atributo.</span><span class="sxs-lookup"><span data-stu-id="2438a-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="2438a-105">Para obter mais informações, consulte [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="2438a-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="2438a-106">O exemplo a seguir gera CS0636:</span><span class="sxs-lookup"><span data-stu-id="2438a-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```

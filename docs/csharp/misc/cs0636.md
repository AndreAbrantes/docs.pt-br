 <span data-ttu-id="9345f-107">O exemplo a seguir gera CS0636:</span><span class="sxs-lookup"><span data-stu-id="9345f-107">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs

using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]

{  
   [FieldOffset(4)]public int i;   // CS0636
}

{  
   public static void Main ()  
   {  
   }  
}  
```

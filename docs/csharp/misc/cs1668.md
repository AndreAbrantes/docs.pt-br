# <a name="compiler-warning-level-2-cs1668"></a>Compilador CS1668 de aviso (nível 2)
Caminho de pesquisa inválido 'path' especificado em 'caminho cadeia de caracteres' - 'mensagem de erro do sistema'  
  
 O caminho fornecido para [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) no comando linha não era válida ou um caminho na variável de ambiente LIB é inválido. Verifique o caminho usado para verificar se ele existe e pode ser acessado. A mensagem de erro entre aspas simples é o erro retornado pelo sistema operacional.

# <a name="compiler-error-cs0031"></a><span data-ttu-id="522c0-102">CS0031 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="522c0-102">Compiler Error CS0031</span></span>
<span data-ttu-id="522c0-103">Valor constante 'value' não pode ser convertido em 'type'.</span><span class="sxs-lookup"><span data-stu-id="522c0-103">Constant value 'value' cannot be converted to a 'type'.</span></span> <span data-ttu-id="522c0-104">(use a sintaxe 'unchecked' para substituir)</span><span class="sxs-lookup"><span data-stu-id="522c0-104">(use 'unchecked' syntax to override)</span></span>  
  
 <span data-ttu-id="522c0-105">Foi feita uma tentativa para atribuir um valor a uma variável cujo tipo não é possível armazenar o valor.</span><span class="sxs-lookup"><span data-stu-id="522c0-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="522c0-106">Para obter mais informações, consulte [Tipos](../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="522c0-106">For more information, see [Types](../../csharp/programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="522c0-107">O exemplo a seguir gera CS0031 marcados e desmarcados contextos:</span><span class="sxs-lookup"><span data-stu-id="522c0-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>  
  
```csharp  
// CS0031.cs  
namespace CS0031  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031  
  
         const decimal d = -10M; // Decimal literal  
            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```

# <a name="compiler-error-cs0031"></a><span data-ttu-id="a9d5a-102">Erro do compilador CS0031</span><span class="sxs-lookup"><span data-stu-id="a9d5a-102">Compiler Error CS0031</span></span>

<span data-ttu-id="a9d5a-103">O valor constante ' value ' não pode ser convertido em um ' type '.</span><span class="sxs-lookup"><span data-stu-id="a9d5a-103">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="a9d5a-104">Foi feita uma tentativa de atribuir um valor a uma variável cujo tipo não pode armazenar o valor.</span><span class="sxs-lookup"><span data-stu-id="a9d5a-104">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="a9d5a-105">Para obter mais informações, consulte [Tipos](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="a9d5a-105">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="a9d5a-106">O exemplo a seguir gera CS0031 em contextos marcados e desmarcados:</span><span class="sxs-lookup"><span data-stu-id="a9d5a-106">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```

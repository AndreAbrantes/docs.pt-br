# <a name="compiler-error-cs0238"></a><span data-ttu-id="e99c8-102">CS0238 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="e99c8-102">Compiler Error CS0238</span></span>
<span data-ttu-id="e99c8-103">'member' não pode ser sealed porque não é uma substituição</span><span class="sxs-lookup"><span data-stu-id="e99c8-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="e99c8-104">[lacrado](../../csharp/language-reference/keywords/sealed.md) foi usado em um membro que também não foi marcado com [substituir](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="e99c8-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="e99c8-105">Para obter mais informações, consulte [Herança](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="e99c8-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="e99c8-106">O exemplo a seguir gera CS0238:</span><span class="sxs-lookup"><span data-stu-id="e99c8-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```

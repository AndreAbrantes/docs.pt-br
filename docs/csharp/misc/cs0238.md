<span data-ttu-id="051d8-104">' member ' não pode ser sealed porque não é uma substituição</span><span class="sxs-lookup"><span data-stu-id="051d8-104">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="051d8-105">[sealed](../language-reference/keywords/sealed.md) foi usado em um membro que não foi marcado também com [override](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="051d8-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="051d8-106">Para obter mais informações, consulte [Herança](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="051d8-106">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="051d8-107">O exemplo a seguir gera CS0238:</span><span class="sxs-lookup"><span data-stu-id="051d8-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```

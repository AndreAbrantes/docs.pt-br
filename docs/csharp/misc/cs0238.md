 <span data-ttu-id="30e70-104">[lacrado](../../csharp/language-reference/keywords/sealed.md) foi usado em um membro que também não foi marcado com [substituir](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="30e70-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="30e70-105">Para obter mais informações, consulte [Herança](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="30e70-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="30e70-106">O exemplo a seguir gera CS0238:</span><span class="sxs-lookup"><span data-stu-id="30e70-106">The following sample generates CS0238:</span></span>  
  
```csharp

}

   // Try the following definition instead:

   {  
   }  
}  
```

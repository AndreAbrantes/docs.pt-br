 <span data-ttu-id="05d2e-104">[sealed](../language-reference/keywords/sealed.md) foi usado em um membro que não foi marcado também com [override](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="05d2e-104">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="05d2e-105">Para obter mais informações, consulte [Herança](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="05d2e-105">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="05d2e-106">O exemplo a seguir gera CS0238:</span><span class="sxs-lookup"><span data-stu-id="05d2e-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs

   public abstract void f();  
}

{  
   public static void Main()  
   {  
   }

   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```

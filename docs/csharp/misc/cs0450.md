# <a name="compiler-error-cs0450"></a><span data-ttu-id="d424f-102">CS0450 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="d424f-102">Compiler Error CS0450</span></span>
<span data-ttu-id="d424f-103">'Nome do parâmetro de tipo': não é possível especificar uma classe de restrição e a restrição 'class' ou 'struct'</span><span class="sxs-lookup"><span data-stu-id="d424f-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="d424f-104">Se um parâmetro de tipo é restrito pela restrição de tipo de estrutura, é logicamente contraditória para que ele também ser restringidos por um tipo de classe específica, como struct e classe são mutuamente excludentes categorias.</span><span class="sxs-lookup"><span data-stu-id="d424f-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="d424f-105">Se um parâmetro de tipo é restrito por uma restrição de tipo de classe específica, em seguida, é por definição restringida pela restrição de tipo de classe e então especificar a restrição de tipo de classe é redundante.</span><span class="sxs-lookup"><span data-stu-id="d424f-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d424f-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d424f-106">Example</span></span>  
  
```  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```

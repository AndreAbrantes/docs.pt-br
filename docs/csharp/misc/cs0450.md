# <a name="compiler-error-cs0450"></a><span data-ttu-id="8de6f-102">Erro do compilador CS0450</span><span class="sxs-lookup"><span data-stu-id="8de6f-102">Compiler Error CS0450</span></span>
<span data-ttu-id="8de6f-103">'Nome do parâmetro de tipo': não é possível especificar uma classe de restrição e a restrição 'class' ou 'struct'</span><span class="sxs-lookup"><span data-stu-id="8de6f-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="8de6f-104">Se um parâmetro de tipo é restrito pela restrição de tipo de struct, é logicamente contraditória para que ele também ser restrito por um tipo de classe específica, como classe e struct são categorias mutuamente exclusivas.</span><span class="sxs-lookup"><span data-stu-id="8de6f-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="8de6f-105">Se um parâmetro de tipo é restrita por uma restrição de tipo de classe específica, em seguida, é por definição restringida pela restrição de tipo de classe e então, especificar a restrição de tipo de classe é redundante.</span><span class="sxs-lookup"><span data-stu-id="8de6f-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8de6f-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8de6f-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```

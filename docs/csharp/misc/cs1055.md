# <a name="compiler-error-cs1055"></a><span data-ttu-id="b418e-102">CS1055 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="b418e-102">Compiler Error CS1055</span></span>
<span data-ttu-id="b418e-103">Acessador add ou remove esperado</span><span class="sxs-lookup"><span data-stu-id="b418e-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="b418e-104">Se seu [evento](../../csharp/language-reference/keywords/event.md) não é declarado como um campo, ele deve definir ambos **adicionar** e **remover** funções do acessador.</span><span class="sxs-lookup"><span data-stu-id="b418e-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="b418e-105">O exemplo a seguir gera CS1055:</span><span class="sxs-lookup"><span data-stu-id="b418e-105">The following sample generates CS1055:</span></span>  
  
```  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="6f436-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="6f436-106">Example</span></span>  
 <span data-ttu-id="6f436-107">O exemplo a seguir gera CS0199:</span><span class="sxs-lookup"><span data-stu-id="6f436-107">The following sample generates CS0199:</span></span>  
  
```  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

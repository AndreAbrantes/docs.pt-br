<span data-ttu-id="bd995-103">Não há suporte para a propriedade, indexador ou evento 'property' de linguagem. Tente chamar diretamente o acessador métodos 'acessador set' ou 'acessador get'</span><span class="sxs-lookup"><span data-stu-id="bd995-103">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="bd995-104">O código está consumindo um objeto que tem um não-padrão [indexador](../../csharp/programming-guide/indexers/index.md) e tentou usar a sintaxe indexada.</span><span class="sxs-lookup"><span data-stu-id="bd995-104">The code is consuming an object that has a non-default [indexer](../../csharp/programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="bd995-105">Para resolver esse erro, chame a propriedade `get` ou `set` método do acessador.</span><span class="sxs-lookup"><span data-stu-id="bd995-105">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bd995-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="bd995-106">Example</span></span>  
  
```  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```

 <span data-ttu-id="bd995-108">O exemplo a seguir gera CS1545.</span><span class="sxs-lookup"><span data-stu-id="bd995-108">The following sample generates CS1545.</span></span>

```  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545

   }  
}  
```

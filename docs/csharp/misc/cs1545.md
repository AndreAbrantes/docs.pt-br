## <a name="example"></a><span data-ttu-id="5d91b-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5d91b-107">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="5d91b-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5d91b-108">Example</span></span>  
 <span data-ttu-id="5d91b-109">O exemplo a seguir gera CS1545.</span><span class="sxs-lookup"><span data-stu-id="5d91b-109">The following sample generates CS1545.</span></span>

```csharp  
class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545

   }  
}  
```

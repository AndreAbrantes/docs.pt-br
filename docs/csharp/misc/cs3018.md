# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="c9ccc-102">Compilador aviso (nível 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="c9ccc-102">Compiler Warning (level 1) CS3018</span></span>

 <span data-ttu-id="c9ccc-104">Este aviso ocorre se uma classe aninhada com o atributo CLSCompliant definido como `true` é declarado como um membro de uma classe declarada com o atributo CLSCompliant para `false`.</span><span class="sxs-lookup"><span data-stu-id="c9ccc-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="c9ccc-105">Isso não é permitido, como uma classe aninhada não pode ser compatível com CLS, se ele for um membro de uma classe externa que não é compatível com CLS.</span><span class="sxs-lookup"><span data-stu-id="c9ccc-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="c9ccc-106">Para resolver este aviso, remova o atributo CLSCompliant a classe aninhada, ou alterá-lo no `true` para `false`.</span><span class="sxs-lookup"><span data-stu-id="c9ccc-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="c9ccc-107">Para obter mais informações sobre a conformidade com CLS, consulte [independência de linguagem e componentes independentes de linguagem](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="c9ccc-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="c9ccc-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="c9ccc-108">Example</span></span>  
 <span data-ttu-id="c9ccc-109">O exemplo a seguir gera CS3018.</span><span class="sxs-lookup"><span data-stu-id="c9ccc-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```

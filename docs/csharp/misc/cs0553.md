# <a name="compiler-error-cs0553"></a><span data-ttu-id="c4e60-102">CS0553 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="c4e60-102">Compiler Error CS0553</span></span>
<span data-ttu-id="c4e60-103">'rotina de conversão': definidos pelo usuário conversão em/de classe base</span><span class="sxs-lookup"><span data-stu-id="c4e60-103">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="c4e60-104">Não são permitidas conversões definidas pelo usuário para valores de uma classe base; não é necessário um operador.</span><span class="sxs-lookup"><span data-stu-id="c4e60-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="c4e60-105">O exemplo a seguir gera CS0553:</span><span class="sxs-lookup"><span data-stu-id="c4e60-105">The following sample generates CS0553:</span></span>  
  
```  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```

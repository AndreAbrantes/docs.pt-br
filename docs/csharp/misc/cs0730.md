## <a name="example"></a><span data-ttu-id="09bc9-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="09bc9-106">Example</span></span>  

 <span data-ttu-id="09bc9-107">O exemplo a seguir gera CS0730.</span><span class="sxs-lookup"><span data-stu-id="09bc9-107">The following sample generates CS0730.</span></span> <span data-ttu-id="09bc9-108">Ele consiste em dois arquivos de origem.</span><span class="sxs-lookup"><span data-stu-id="09bc9-108">It consists of two source files.</span></span> <span data-ttu-id="09bc9-109">Primeiro, compile o arquivo de biblioteca `CS0730a.cs` e compile o arquivo que `CS0730.cs` faz referência ao arquivo de biblioteca.</span><span class="sxs-lookup"><span data-stu-id="09bc9-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```

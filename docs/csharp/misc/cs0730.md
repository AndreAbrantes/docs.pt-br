## <a name="example"></a><span data-ttu-id="553d5-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="553d5-105">Example</span></span>  
 <span data-ttu-id="553d5-106">O exemplo a seguir gera CS0730.</span><span class="sxs-lookup"><span data-stu-id="553d5-106">The following sample generates CS0730.</span></span> <span data-ttu-id="553d5-107">Ele consiste em dois arquivos de origem.</span><span class="sxs-lookup"><span data-stu-id="553d5-107">It consists of two source files.</span></span> <span data-ttu-id="553d5-108">Primeiro, compile o arquivo de biblioteca `CS0730a.cs`e o compilar o arquivo `CS0730.cs` referenciando o arquivo de biblioteca.</span><span class="sxs-lookup"><span data-stu-id="553d5-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```

<span data-ttu-id="840aa-103">Não é possível retornar um valor de um iterador.</span><span class="sxs-lookup"><span data-stu-id="840aa-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="840aa-104">Use a instrução yield return para retornar um valor ou yield break para finalizar a iteração.</span><span class="sxs-lookup"><span data-stu-id="840aa-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="840aa-105">Um iterador é uma função especial que retorna um valor por meio da instrução yield, em vez da instrução return.</span><span class="sxs-lookup"><span data-stu-id="840aa-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="840aa-106">Para obter mais informações, consulte **iteradores**.</span><span class="sxs-lookup"><span data-stu-id="840aa-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="840aa-107">O exemplo a seguir gera CS1622:</span><span class="sxs-lookup"><span data-stu-id="840aa-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```

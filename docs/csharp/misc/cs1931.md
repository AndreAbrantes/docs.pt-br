## <a name="example"></a><span data-ttu-id="41a9d-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="41a9d-107">Example</span></span>  
 <span data-ttu-id="41a9d-108">O código a seguir gera CS1931 porque o identificador `x` é usado como uma variável local em `Main` e como a variável de intervalo na expressão de consulta:</span><span class="sxs-lookup"><span data-stu-id="41a9d-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

        static void Main()

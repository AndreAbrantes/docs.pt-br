# <a name="compiler-error-cs1507"></a>Erro do compilador CS1507
Não é possível vincular o arquivo de recurso ' file ' ao compilar um módulo  
  
 [/linkresource](../language-reference/compiler-options/linkresource-compiler-option.md) foi usado na mesma compilação com [/target: module](../language-reference/compiler-options/target-module-compiler-option.md), o que não é permitido. Por exemplo, as opções a seguir gerarão CS1507:  
  
```console  
csc /linkresource:rf.resource /target:module in.cs  
```  
  
 A inserção de recursos ([/Resource](../language-reference/compiler-options/resource-compiler-option.md)), no entanto, é permitida.

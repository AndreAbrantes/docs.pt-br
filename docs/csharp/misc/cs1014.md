# <a name="compiler-error-cs1014"></a>Erro do compilador CS1014
Um acessador get ou set esperado  
  
 Uma declaração de método foi encontrada em uma declaração de propriedade. Somente é possível declarar `get` e `set` métodos em uma propriedade.  
  
 Para obter mais informações sobre as propriedades, consulte [usando as propriedades](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1014.  
  
```csharp  
// CS1014.cs  
// compile with: /target:library  
class Sample  
{  
   public int TestProperty  
   {  
      get  
      {  
         return 0;  
      }  
      int z;   // CS1014  not get or set  
   }  
}  
```

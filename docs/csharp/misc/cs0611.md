# <a name="compiler-error-cs0611"></a><span data-ttu-id="8dfbc-102">CS0611 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="8dfbc-102">Compiler Error CS0611</span></span>

 <span data-ttu-id="8dfbc-104">Há alguns tipos que não podem ser usados como o tipo de uma matriz.</span><span class="sxs-lookup"><span data-stu-id="8dfbc-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="8dfbc-105">Esses tipos incluem **System.TypedReference** e **System.ArgIterator**.</span><span class="sxs-lookup"><span data-stu-id="8dfbc-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>  
  
 <span data-ttu-id="8dfbc-106">O exemplo a seguir gera CS0611 como resultado do uso **System.TypedReference** como um elemento de matriz:</span><span class="sxs-lookup"><span data-stu-id="8dfbc-106">The following sample generates CS0611 as a result of using **System.TypedReference** as an array element:</span></span>  
  
```

{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```

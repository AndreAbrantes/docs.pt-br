# <a name="compiler-error-cs1922"></a><span data-ttu-id="387ae-102">CS1922 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="387ae-102">Compiler Error CS1922</span></span>
<span data-ttu-id="387ae-103">Inicializador de coleção exige o tipo 'type' para implementar System.Collections.IEnumerable.</span><span class="sxs-lookup"><span data-stu-id="387ae-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>  
  
 <span data-ttu-id="387ae-104">Para usar um inicializador de coleção com um tipo, o tipo deve implementar `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="387ae-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="387ae-105">Esse erro pode ocorrer se você usar acidentalmente sintaxe do inicializador de coleção quando você pretende usar um inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="387ae-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="387ae-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="387ae-106">To correct this error</span></span>  
  
-   <span data-ttu-id="387ae-107">Se o tipo não representa uma coleção, use a sintaxe do inicializador de objeto, em vez da sintaxe do inicializador de coleção.</span><span class="sxs-lookup"><span data-stu-id="387ae-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>  
  
-   <span data-ttu-id="387ae-108">Se o tipo representam uma coleção, modificá-lo para implementar `IEnumerable` antes de usar inicializadores de coleção para inicializar objetos desse tipo.</span><span class="sxs-lookup"><span data-stu-id="387ae-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>  
  
-   <span data-ttu-id="387ae-109">Se o tipo representa uma coleção e você não tem acesso ao código-fonte, apenas inicialize seus elementos usando seus construtores de classe ou outros métodos de inicialização.</span><span class="sxs-lookup"><span data-stu-id="387ae-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>  
  
## <a name="example"></a><span data-ttu-id="387ae-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="387ae-110">Example</span></span>  
 <span data-ttu-id="387ae-111">O código a seguir produz CS1922:</span><span class="sxs-lookup"><span data-stu-id="387ae-111">The following code produces CS1922:</span></span>  
  
```csharp  
// cs1922.cs  
public class Test  
{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922  
  
        // Object initalizer.  
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK  
    }  
}  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public string memberB { get; set; }  
}  
```

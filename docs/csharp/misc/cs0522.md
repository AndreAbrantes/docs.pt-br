# <a name="compiler-error-cs0522"></a><span data-ttu-id="3f931-102">Erro do compilador CS0522</span><span class="sxs-lookup"><span data-stu-id="3f931-102">Compiler Error CS0522</span></span>
<span data-ttu-id="3f931-103">"constructor": structs não podem chamar construtores de classe base</span><span class="sxs-lookup"><span data-stu-id="3f931-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="3f931-104">Uma [struct](../language-reference/keywords/struct.md) não pode chamar um construtor de classe base; Remova a chamada para o construtor da classe base.</span><span class="sxs-lookup"><span data-stu-id="3f931-104">A [struct](../language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="3f931-105">O exemplo a seguir gera CS0522:</span><span class="sxs-lookup"><span data-stu-id="3f931-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```

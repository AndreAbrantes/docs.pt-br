# <a name="compiler-error-cs0522"></a><span data-ttu-id="eb6b5-102">Erro do compilador CS0522</span><span class="sxs-lookup"><span data-stu-id="eb6b5-102">Compiler Error CS0522</span></span>
<span data-ttu-id="eb6b5-103">"constructor": structs não podem chamar construtores de classe base</span><span class="sxs-lookup"><span data-stu-id="eb6b5-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="eb6b5-104">Uma [struct](../language-reference/builtin-types/struct.md) não pode chamar um construtor de classe base; Remova a chamada para o construtor da classe base.</span><span class="sxs-lookup"><span data-stu-id="eb6b5-104">A [struct](../language-reference/builtin-types/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="eb6b5-105">O exemplo a seguir gera CS0522:</span><span class="sxs-lookup"><span data-stu-id="eb6b5-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```

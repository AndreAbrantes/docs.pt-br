# <a name="compiler-error-cs0522"></a><span data-ttu-id="fc643-102">CS0522 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="fc643-102">Compiler Error CS0522</span></span>
<span data-ttu-id="fc643-103">'construtor': structs não podem chamar construtores de classe base</span><span class="sxs-lookup"><span data-stu-id="fc643-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="fc643-104">Um [struct](../../csharp/language-reference/keywords/struct.md) não é possível chamar um construtor de classe base; remova a chamada para o construtor da classe base.</span><span class="sxs-lookup"><span data-stu-id="fc643-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="fc643-105">O exemplo a seguir gera CS0522:</span><span class="sxs-lookup"><span data-stu-id="fc643-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```

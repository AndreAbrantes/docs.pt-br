# <a name="compiler-warning-level-1-cs3016"></a>Compilador CS3016 de aviso (nível 1)
Matrizes como argumentos de atributo não são compatíveis com CLS  
  
 Não é compatível com o Common Language Specification (CLS) para passar uma matriz a um atributo. Para obter mais informações sobre compatibilidade com CLS, consulte [escrevendo código compatível com CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) e [independência da linguagem e componentes independentes de linguagem](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS3016:  
  
```  
// CS3016.cs  
  
using System;  
  
[assembly : CLSCompliant(true)]  
[C(new int[] {1, 2})]   // CS3016  
// try the following line instead  
// [C()]  
class C : Attribute  
{  
    public C()  
    {  
    }  
  
    public C(int[] a)  
    {  
    }  
  
    public static void Main ()  
    {  
    }  
}  
```

# <a name="compiler-error-cs1511"></a><span data-ttu-id="5cdb1-102">Erro do compilador CS1511</span><span class="sxs-lookup"><span data-stu-id="5cdb1-102">Compiler Error CS1511</span></span>

 <span data-ttu-id="5cdb1-104">O [base](../../csharp/language-reference/keywords/base.md) palavra-chave que foi usada em uma [estático](../../csharp/language-reference/keywords/static.md) método.</span><span class="sxs-lookup"><span data-stu-id="5cdb1-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="5cdb1-105">`base` só pode ser chamado em um construtor de instância, o método de instância ou o acessador de instância.</span><span class="sxs-lookup"><span data-stu-id="5cdb1-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5cdb1-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5cdb1-106">Example</span></span>  
 <span data-ttu-id="5cdb1-107">O exemplo a seguir gera CS1511.</span><span class="sxs-lookup"><span data-stu-id="5cdb1-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```

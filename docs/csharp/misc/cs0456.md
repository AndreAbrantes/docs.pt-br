# <a name="compiler-error-cs0456"></a><span data-ttu-id="da650-102">CS0456 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="da650-102">Compiler Error CS0456</span></span>
<span data-ttu-id="da650-103">O parâmetro de tipo '1 de nome de parâmetro de tipo' tem a restrição 'struct' para que '1 de nome de parâmetro de tipo' não pode ser usado como uma restrição para 'Nome de parâmetro de tipo 2'</span><span class="sxs-lookup"><span data-stu-id="da650-103">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>  
  
 <span data-ttu-id="da650-104">Restrições de tipo de valor são lacradas implicitamente para que essas restrições não podem ser usadas como restrições em um segundo parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="da650-104">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="da650-105">Isso ocorre porque os tipos de valor não podem ser substituídos.</span><span class="sxs-lookup"><span data-stu-id="da650-105">This is because value types cannot be overridden.</span></span> <span data-ttu-id="da650-106">Para resolver esse erro, coloque uma restrição de tipo de valor diretamente no segundo parâmetro de tipo, em vez de fazer assim indiretamente por meio do primeiro parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="da650-106">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="da650-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="da650-107">Example</span></span>  
 <span data-ttu-id="da650-108">O exemplo a seguir gera CS0456.</span><span class="sxs-lookup"><span data-stu-id="da650-108">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs

{  
   public class G5<T> where T : struct  
   {

   }  
}  
```

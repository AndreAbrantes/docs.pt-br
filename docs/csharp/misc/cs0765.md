# <a name="compiler-error-cs0765"></a><span data-ttu-id="00b49-103">Erro do Compilador CS0765</span><span class="sxs-lookup"><span data-stu-id="00b49-103">Compiler Error CS0765</span></span>

<span data-ttu-id="00b49-104">Os métodos parciais com apenas uma declaração de definição ou métodos condicionais removidos não podem ser usados em árvores de expressão</span><span class="sxs-lookup"><span data-stu-id="00b49-104">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="00b49-105">Embora uma chamada para um método parcial removido seja uma expressão, ela não é uma expressão aceitável em uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="00b49-105">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="00b49-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="00b49-106">To correct this error</span></span>  
  
1. <span data-ttu-id="00b49-107">Adicione uma declaração de implementação para o método parcial ou remova o código que está fazendo com que o método condicional seja excluído da compilação.</span><span class="sxs-lookup"><span data-stu-id="00b49-107">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="00b49-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="00b49-108">Example</span></span>  

 <span data-ttu-id="00b49-109">O código a seguir gera CS0765 em dois locais:</span><span class="sxs-lookup"><span data-stu-id="00b49-109">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```

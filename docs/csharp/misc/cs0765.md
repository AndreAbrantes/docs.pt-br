 <span data-ttu-id="8f0a3-104">Embora uma chamada para um método parcial removido é uma expressão, ele não é uma expressão aceitável em uma árvore de expressão.</span><span class="sxs-lookup"><span data-stu-id="8f0a3-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="8f0a3-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8f0a3-107">Example</span></span>  
 <span data-ttu-id="8f0a3-108">O código a seguir gera CS0765 em dois locais:</span><span class="sxs-lookup"><span data-stu-id="8f0a3-108">The following code generates CS0765 in two locations:</span></span>  
  
```  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```

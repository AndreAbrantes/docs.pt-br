<span data-ttu-id="ded9c-103">Não é possível obter o endereço da expressão fornecida</span><span class="sxs-lookup"><span data-stu-id="ded9c-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="ded9c-104">Você pode pegar o endereço de campos, variáveis locais e indireção de ponteiros, mas não pode tomar, por exemplo, o endereço da soma de duas variáveis locais.</span><span class="sxs-lookup"><span data-stu-id="ded9c-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="ded9c-105">Para obter mais informações, consulte [Código não seguro e ponteiros](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="ded9c-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="ded9c-106">O exemplo a seguir gera CS0211:</span><span class="sxs-lookup"><span data-stu-id="ded9c-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

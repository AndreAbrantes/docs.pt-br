# <a name="compiler-error-cs0211"></a><span data-ttu-id="1453f-102">CS0211 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="1453f-102">Compiler Error CS0211</span></span>
<span data-ttu-id="1453f-103">Não é possível obter o endereço da expressão especificada</span><span class="sxs-lookup"><span data-stu-id="1453f-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="1453f-104">Você pode obter o endereço de campos, variáveis locais e indireção de ponteiros, mas você não pode obter, por exemplo, o endereço da soma das duas variáveis locais.</span><span class="sxs-lookup"><span data-stu-id="1453f-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="1453f-105">Para obter mais informações, consulte [código não seguro e ponteiros](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="1453f-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="1453f-106">O exemplo a seguir gera CS0211:</span><span class="sxs-lookup"><span data-stu-id="1453f-106">The following sample generates CS0211:</span></span>  
  
```  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

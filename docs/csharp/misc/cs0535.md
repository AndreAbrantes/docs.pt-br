 <span data-ttu-id="e2560-104">Um [classe](../../csharp/language-reference/keywords/class.md) derivado de um [interface](../../csharp/language-reference/keywords/interface.md), mas a classe não implementou um ou mais dos membros da interface.</span><span class="sxs-lookup"><span data-stu-id="e2560-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="e2560-105">Uma classe deve implementar todos os membros das interfaces do qual ele deriva ou então ser declarado como `abstract`.</span><span class="sxs-lookup"><span data-stu-id="e2560-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e2560-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e2560-106">Example</span></span>  
 <span data-ttu-id="e2560-107">O exemplo a seguir gera CS0535.</span><span class="sxs-lookup"><span data-stu-id="e2560-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```

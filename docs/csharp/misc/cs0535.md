 <span data-ttu-id="50bbc-105">Uma [classe](../language-reference/keywords/class.md) derivada de uma [interface](../language-reference/keywords/interface.md), mas a classe não implementou um ou mais dos membros da interface.</span><span class="sxs-lookup"><span data-stu-id="50bbc-105">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="50bbc-106">Uma classe deve implementar todos os membros das interfaces das quais ela deriva ou deve ser declarada `abstract` .</span><span class="sxs-lookup"><span data-stu-id="50bbc-106">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="50bbc-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="50bbc-107">Example</span></span>  
 <span data-ttu-id="50bbc-108">O exemplo a seguir gera CS0535.</span><span class="sxs-lookup"><span data-stu-id="50bbc-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```

<span data-ttu-id="5981d-103">O programa 'nome do arquivo de saída' tem mais de um ponto de entrada definido.</span><span class="sxs-lookup"><span data-stu-id="5981d-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="5981d-104">Compile com /main para especificar o tipo que contém o ponto de entrada.</span><span class="sxs-lookup"><span data-stu-id="5981d-104">Compile with /main to specify the type that contains the entry point.</span></span>

 <span data-ttu-id="5981d-107">O exemplo a seguir gera CS0017:</span><span class="sxs-lookup"><span data-stu-id="5981d-107">The following sample generates CS0017:</span></span>

```  
{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```

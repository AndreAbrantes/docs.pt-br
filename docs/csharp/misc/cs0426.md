# <a name="compiler-error-cs0426"></a><span data-ttu-id="c7eab-102">Erro do Compilador CS0426</span><span class="sxs-lookup"><span data-stu-id="c7eab-102">Compiler Error CS0426</span></span>
<span data-ttu-id="c7eab-103">O nome de tipo 'identificador' não existe no tipo 'tipo'</span><span class="sxs-lookup"><span data-stu-id="c7eab-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="c7eab-104">Esse erro ocorre quando você faz referência a um tipo aninhado dentro de outro tipo, mas não existe esse tipo aninhado.</span><span class="sxs-lookup"><span data-stu-id="c7eab-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="c7eab-105">Isso pode ocorrer se você digitar mal o nome do tipo aninhado.</span><span class="sxs-lookup"><span data-stu-id="c7eab-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="c7eab-106">Verifique a ortografia dos nomes utilizados e verifique se o tipo de fechamento tem o membro esperado.</span><span class="sxs-lookup"><span data-stu-id="c7eab-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="c7eab-107">A seguinte amostra gera CS0426 porque a classe C não tem o tipo A aninhada:</span><span class="sxs-lookup"><span data-stu-id="c7eab-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```

 <span data-ttu-id="70ef2-107">O exemplo a seguir gera CS0426 porque a classe C não tem nenhum tipo aninhado r:</span><span class="sxs-lookup"><span data-stu-id="70ef2-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp

{  
    // No nested types are declared.     
}

{  
   public static void Main()

```

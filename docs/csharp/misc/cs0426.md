# <a name="compiler-error-cs0426"></a><span data-ttu-id="d98e1-103">Erro do Compilador CS0426</span><span class="sxs-lookup"><span data-stu-id="d98e1-103">Compiler Error CS0426</span></span>

<span data-ttu-id="d98e1-104">O nome do tipo ' identifier ' não existe no tipo ' type '</span><span class="sxs-lookup"><span data-stu-id="d98e1-104">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="d98e1-105">Esse erro ocorre quando você faz referência a um tipo aninhado dentro de outro tipo, mas esse tipo aninhado não existe.</span><span class="sxs-lookup"><span data-stu-id="d98e1-105">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="d98e1-106">Isso pode ocorrer se você digitar indigitadamente o nome do tipo aninhado.</span><span class="sxs-lookup"><span data-stu-id="d98e1-106">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="d98e1-107">Verifique a ortografia dos nomes usados e verifique se o tipo de delimitador tem o membro esperado.</span><span class="sxs-lookup"><span data-stu-id="d98e1-107">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="d98e1-108">O exemplo a seguir gera CS0426 porque A classe C não tem um tipo aninhado A:</span><span class="sxs-lookup"><span data-stu-id="d98e1-108">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```

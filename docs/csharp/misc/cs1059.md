## <a name="example"></a><span data-ttu-id="f0799-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f0799-110">Example</span></span>  
 <span data-ttu-id="f0799-111">O exemplo a seguir gera CS1059 porque `i` é uma constante, não é uma variável, e `E` é um `Enum` tipo, cujos elementos também são valores constantes.</span><span class="sxs-lookup"><span data-stu-id="f0799-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```

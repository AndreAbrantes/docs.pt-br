# <a name="compiler-error-cs0546"></a><span data-ttu-id="a876a-102">CS0546 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="a876a-102">Compiler Error CS0546</span></span>
<span data-ttu-id="a876a-103">'acessador': não é possível substituir porque 'property' não tem um acessador set substituível</span><span class="sxs-lookup"><span data-stu-id="a876a-103">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="a876a-104">Falha ao tentar substituir um dos métodos de acessador para uma propriedade, porque o acessador não pode ser substituído.</span><span class="sxs-lookup"><span data-stu-id="a876a-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="a876a-105">Esse erro pode ocorrer se:</span><span class="sxs-lookup"><span data-stu-id="a876a-105">This error can occur if:</span></span>  
  
-   <span data-ttu-id="a876a-106">a propriedade de classe base não está declarada como [virtual](../../csharp/language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="a876a-106">the base class property is not declared as [virtual](../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
-   <span data-ttu-id="a876a-107">a propriedade de classe base não declarar o [obter](../../csharp/language-reference/keywords/get.md) ou [definir](../../csharp/language-reference/keywords/set.md) acessador que você está tentando substituir.</span><span class="sxs-lookup"><span data-stu-id="a876a-107">the base class property does not declare the [get](../../csharp/language-reference/keywords/get.md) or [set](../../csharp/language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="a876a-108">Se você não deseja substituir a propriedade de classe base, você pode usar o [novo](../../csharp/language-reference/keywords/new.md) palavra-chave antes da propriedade na classe derivada.</span><span class="sxs-lookup"><span data-stu-id="a876a-108">If you do not want to override the base class property, you can use the [new](../../csharp/language-reference/keywords/new.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="a876a-109">Para obter mais informações, consulte [Propriedades using](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="a876a-109">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="a876a-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a876a-110">Example</span></span>  
 <span data-ttu-id="a876a-111">O exemplo a seguir gera CS0546 porque a classe base não declara um acessador set da propriedade.</span><span class="sxs-lookup"><span data-stu-id="a876a-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```

# <a name="compiler-error-cs0546"></a><span data-ttu-id="df83c-103">Erro do Compilador CS0546</span><span class="sxs-lookup"><span data-stu-id="df83c-103">Compiler Error CS0546</span></span>

<span data-ttu-id="df83c-104">' acessador ': não é possível substituir porque ' Property ' não tem um acessador set substituível</span><span class="sxs-lookup"><span data-stu-id="df83c-104">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="df83c-105">Falha ao tentar substituir um dos métodos de acessador de uma propriedade porque o acessador não pode ser substituído.</span><span class="sxs-lookup"><span data-stu-id="df83c-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="df83c-106">Esse erro pode acontecer se:</span><span class="sxs-lookup"><span data-stu-id="df83c-106">This error can occur if:</span></span>  
  
- <span data-ttu-id="df83c-107">a propriedade da classe base não está declarada como [virtual](../language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="df83c-107">the base class property is not declared as [virtual](../language-reference/keywords/virtual.md).</span></span>  
  
- <span data-ttu-id="df83c-108">a propriedade da classe base não declara o acessador [Get](../language-reference/keywords/get.md) ou [set](../language-reference/keywords/set.md) que você está tentando substituir.</span><span class="sxs-lookup"><span data-stu-id="df83c-108">the base class property does not declare the [get](../language-reference/keywords/get.md) or [set](../language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="df83c-109">Se você não quiser substituir a propriedade da classe base, poderá usar a [nova](../language-reference/keywords/new-modifier.md) palavra-chave antes da propriedade na classe derivada.</span><span class="sxs-lookup"><span data-stu-id="df83c-109">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="df83c-110">Para obter mais informações, consulte [Propriedades using](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="df83c-110">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="df83c-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="df83c-111">Example</span></span>  

 <span data-ttu-id="df83c-112">O exemplo a seguir gera CS0546 porque a classe base não declara um acessador set para a propriedade.</span><span class="sxs-lookup"><span data-stu-id="df83c-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```

<span data-ttu-id="ebf18-103">'acessador': não é possível substituir porque 'property' não tem um acessador set substituível</span><span class="sxs-lookup"><span data-stu-id="ebf18-103">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="ebf18-104">Falha ao tentar substituir um dos métodos de acessador para uma propriedade, porque o acessador não pode ser substituído.</span><span class="sxs-lookup"><span data-stu-id="ebf18-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="ebf18-105">Esse erro pode ocorrer se:</span><span class="sxs-lookup"><span data-stu-id="ebf18-105">This error can occur if:</span></span>  
  
- <span data-ttu-id="ebf18-106">a propriedade de classe base não está declarada como [virtual](../../csharp/language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="ebf18-106">the base class property is not declared as [virtual](../../csharp/language-reference/keywords/virtual.md).</span></span>

## <a name="example"></a><span data-ttu-id="ebf18-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ebf18-110">Example</span></span>  
 <span data-ttu-id="ebf18-111">O exemplo a seguir gera CS0546 porque a classe base não declara um acessador set da propriedade.</span><span class="sxs-lookup"><span data-stu-id="ebf18-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```

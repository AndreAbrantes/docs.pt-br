# <a name="compiler-error-cs0159"></a><span data-ttu-id="1569e-103">Erro do Compilador CS0159</span><span class="sxs-lookup"><span data-stu-id="1569e-103">Compiler Error CS0159</span></span>
<span data-ttu-id="1569e-104">Não há tal rótulo ' label ' dentro do escopo da instrução goto</span><span class="sxs-lookup"><span data-stu-id="1569e-104">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="1569e-105">O rótulo referenciado pela instrução [goto](../language-reference/keywords/goto.md) não pôde ser encontrado no escopo da `goto` instrução.</span><span class="sxs-lookup"><span data-stu-id="1569e-105">The label referenced by the [goto](../language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="1569e-106">O exemplo a seguir gera CS0159:</span><span class="sxs-lookup"><span data-stu-id="1569e-106">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```

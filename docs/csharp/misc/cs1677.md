 <span data-ttu-id="f6ec5-104">Esse erro ocorre quando o modificador de parâmetro de tipo em um método anônimo não corresponde ao usado na declaração de representante, ao qual você está convertendo o método.</span><span class="sxs-lookup"><span data-stu-id="f6ec5-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f6ec5-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f6ec5-105">Example</span></span>  
 <span data-ttu-id="f6ec5-106">O exemplo a seguir gera CS1677:</span><span class="sxs-lookup"><span data-stu-id="f6ec5-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```

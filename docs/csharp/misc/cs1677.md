 <span data-ttu-id="48c9f-104">Este erro ocorre quando o modificador de tipo de parâmetro em um método anônimo não corresponde ao usado na declaração do delegado, ao qual você está lançando o método.</span><span class="sxs-lookup"><span data-stu-id="48c9f-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="48c9f-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="48c9f-105">Example</span></span>  
 <span data-ttu-id="48c9f-106">A seguinte amostra gera CS1677:</span><span class="sxs-lookup"><span data-stu-id="48c9f-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```

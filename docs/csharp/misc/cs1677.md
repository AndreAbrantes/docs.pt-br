 <span data-ttu-id="af8fb-106">O exemplo a seguir gera CS1677:</span><span class="sxs-lookup"><span data-stu-id="af8fb-106">The following sample generates CS1677:</span></span>  
  
```

{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```

-   <span data-ttu-id="4ea22-108">Um campo estático privado ou interno é atribuído um valor constante apenas no inicializador.</span><span class="sxs-lookup"><span data-stu-id="4ea22-108">A private or internal static field is assigned a constant value only in the initializer.</span></span> <span data-ttu-id="4ea22-109">Considere alterar o campo para uma constante.</span><span class="sxs-lookup"><span data-stu-id="4ea22-109">Consider changing the field to a const.</span></span>

-   <span data-ttu-id="4ea22-112">Um campo particular ou interno é atribuído a valores constantes em vários locais, mas não for acessado.</span><span class="sxs-lookup"><span data-stu-id="4ea22-112">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="4ea22-113">Se o campo não é necessário, considere removê-lo.</span><span class="sxs-lookup"><span data-stu-id="4ea22-113">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="4ea22-114">Caso contrário, usá-lo de forma apropriada.</span><span class="sxs-lookup"><span data-stu-id="4ea22-114">Otherwise, use it in some appropriate way.</span></span>

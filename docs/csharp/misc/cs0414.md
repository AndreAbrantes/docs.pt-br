 <span data-ttu-id="156b3-104">Esse aviso pode ocorrer em vários cenários em que o compilador possa verificar que uma variável não é referenciada:</span><span class="sxs-lookup"><span data-stu-id="156b3-104">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>

-   <span data-ttu-id="156b3-108">Um campo estático privado ou interno é atribuído um valor constante apenas no inicializador.</span><span class="sxs-lookup"><span data-stu-id="156b3-108">A private or internal static field is assigned a constant value only in the initializer.</span></span> <span data-ttu-id="156b3-109">Considere alterar o campo para uma constante.</span><span class="sxs-lookup"><span data-stu-id="156b3-109">Consider changing the field to a const.</span></span>

-   <span data-ttu-id="156b3-112">Um campo privado ou interno é atribuído valores constantes em vários locais, mas não for acessado.</span><span class="sxs-lookup"><span data-stu-id="156b3-112">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="156b3-113">Se o campo não é necessário, considere removê-la.</span><span class="sxs-lookup"><span data-stu-id="156b3-113">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="156b3-114">Caso contrário, usá-lo de alguma maneira apropriada.</span><span class="sxs-lookup"><span data-stu-id="156b3-114">Otherwise, use it in some appropriate way.</span></span>

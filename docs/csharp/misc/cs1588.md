# <a name="compiler-error-cs1588"></a>CS1588 de erro do compilador
Não é possível determinar o diretório de tempo de execução de linguagem comum - 'razão'  
  
 O compilador não pode recuperar o common language runtime do .NET Framework instalado no diretório. Esse erro indica que o common language runtime está instalado incorretamente.

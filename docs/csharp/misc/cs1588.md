# <a name="compiler-error-cs1588"></a>Erro do compilador CS1588
Não é possível determinar o diretório de tempo de execução de linguagem comum--'reason'  
  
 O compilador não é possível recuperar o diretório que o .NET Framework common language runtime está instalado no. Esse erro indica que o common language runtime está instalado incorretamente.

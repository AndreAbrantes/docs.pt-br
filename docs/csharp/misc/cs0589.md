# <a name="compiler-error-cs0589"></a>Erro do compilador CS0589

Erro de compilador interno: estágio ' PARSE '

 Tente determinar se o compilador está falhando devido à sua incapacidade de analisar a sintaxe inesperada. Se esse não for o caso, tente [conversar conosco](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

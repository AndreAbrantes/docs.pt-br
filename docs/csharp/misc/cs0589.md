# <a name="compiler-error-cs0589"></a>Erro do compilador CS0589
Erro interno no compilador: etapa 'PARSE'  
  
 Tente determinar se o compilador está falhando devido à sua incapacidade de analisar sintaxes inesperadas. Se não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="85de4-103">Aviso do compilador (nível 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="85de4-103">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="85de4-104">O valor ' goto Case ' não é implicitamente conversível para o tipo ' type '</span><span class="sxs-lookup"><span data-stu-id="85de4-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="85de4-105">Quando você usa `goto case` , deve haver uma conversão implícita do valor do caso goto para o tipo da opção.</span><span class="sxs-lookup"><span data-stu-id="85de4-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="85de4-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="85de4-106">Example</span></span>  
 <span data-ttu-id="85de4-107">O exemplo a seguir gera CS0469.</span><span class="sxs-lookup"><span data-stu-id="85de4-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```

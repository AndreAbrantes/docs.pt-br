## <a name="example"></a><span data-ttu-id="36657-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="36657-105">Example</span></span>  
 <span data-ttu-id="36657-106">O exemplo a seguir gera CS0469.</span><span class="sxs-lookup"><span data-stu-id="36657-106">The following sample generates CS0469.</span></span>  
  
```  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```

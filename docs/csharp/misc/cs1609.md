# <a name="compiler-error-cs1609"></a>CS1609 de erro do compilador
Modificadores não podem ser colocados em declarações de acessador de evento  
  
 Modificadores só podem ser colocados em declarações de evento e não em declarações de acessador de evento. Para obter mais informações, consulte [usando propriedades](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1609.  
  
```  
// CS1609.cs  
// compile with: /target:library  
delegate int Del();  
class A  
{  
   public event Del MyEvent   
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```

# <a name="compiler-error-cs1609"></a>Erro do Compilador CS1609
Modificadores não podem ser colocados em declarações de acessório de evento  
  
 Os modificadores só podem ser colocados em declarações de eventos e não nas declarações do acessório do evento. Para obter mais informações, consulte [Propriedades using](../programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Exemplo  
 A seguinte amostra gera CS1609.  
  
```csharp  
// CS1609.cs  
// compile with: /target:library  
delegate int Del();  
class A  
{  
   public event Del MyEvent
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```

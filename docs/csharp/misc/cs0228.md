# <a name="compiler-error-cs0228"></a>Erro do compilador CS0228
'type' não contém uma definição para 'membro' ou não está acessível  
  
 A versão do sistema pode ter sido substituído **System.Hashtable**, **System. String**, ou **System. array**, e a substituição não contém `member`.  
  
 Caso contrário, talvez você precise reparar ou reinstalar o Visual Studio.

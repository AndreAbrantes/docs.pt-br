# <a name="compiler-error-cs0228"></a>CS0228 de erro do compilador
'type' não contém uma definição para 'member' ou não está acessível  
  
 A versão do sistema pode ter sido substituído **System.Hashtable**, **System. String**, ou **System. array**, e a substituição não contém `member`.  
  
 Caso contrário, você precisará repare ou reinstale o Visual Studio.

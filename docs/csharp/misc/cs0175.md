<span data-ttu-id="01181-103">O uso da palavra-chave "base" não é válido neste contexto</span><span class="sxs-lookup"><span data-stu-id="01181-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="01181-104">O [base](../../csharp/language-reference/keywords/base.md) palavra-chave deve ser usada para especificar um determinado membro da classe base.</span><span class="sxs-lookup"><span data-stu-id="01181-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="01181-105">Para obter mais informações, consulte [construtores](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="01181-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="01181-106">O exemplo a seguir gera CS0175:</span><span class="sxs-lookup"><span data-stu-id="01181-106">The following sample generates CS0175:</span></span>  
  
```  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```

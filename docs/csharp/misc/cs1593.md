# <a name="compiler-error-cs1593"></a><span data-ttu-id="9b8c2-102">CS1593 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="9b8c2-102">Compiler Error CS1593</span></span>
<span data-ttu-id="9b8c2-103">Delegate 'del' não obtém argumentos 'number'</span><span class="sxs-lookup"><span data-stu-id="9b8c2-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="9b8c2-104">O número de argumentos passados para uma [delegar](../../csharp/language-reference/keywords/delegate.md) invocação não concorda com o número de parâmetros na declaração do delegado.</span><span class="sxs-lookup"><span data-stu-id="9b8c2-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="9b8c2-105">O exemplo a seguir gera CS1593:</span><span class="sxs-lookup"><span data-stu-id="9b8c2-105">The following sample generates CS1593:</span></span>  
  
```  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```

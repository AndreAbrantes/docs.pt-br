# <a name="compiler-error-cs1593"></a><span data-ttu-id="09ef9-102">CS1593 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="09ef9-102">Compiler Error CS1593</span></span>
<span data-ttu-id="09ef9-103">Delegate 'del' não obtém argumentos 'número'</span><span class="sxs-lookup"><span data-stu-id="09ef9-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="09ef9-104">O número de argumentos passados para um [delegar](../../csharp/language-reference/keywords/delegate.md) invocação não concorda com o número de parâmetros na declaração de representante.</span><span class="sxs-lookup"><span data-stu-id="09ef9-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="09ef9-105">O exemplo a seguir gera CS1593:</span><span class="sxs-lookup"><span data-stu-id="09ef9-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```

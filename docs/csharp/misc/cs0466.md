# <a name="compiler-error-cs0466"></a><span data-ttu-id="22e4c-102">Erro do compilador CS0466</span><span class="sxs-lookup"><span data-stu-id="22e4c-102">Compiler Error CS0466</span></span>
<span data-ttu-id="22e4c-103">'method1' não deve ter um parâmetro params, pois 'method2' não tem</span><span class="sxs-lookup"><span data-stu-id="22e4c-103">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="22e4c-104">Não é possível usar `params` parâmetro em um membro de classe que não o use a interface implementada.</span><span class="sxs-lookup"><span data-stu-id="22e4c-104">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="22e4c-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="22e4c-105">Example</span></span>  
 <span data-ttu-id="22e4c-106">O exemplo a seguir gera CS0466.</span><span class="sxs-lookup"><span data-stu-id="22e4c-106">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```

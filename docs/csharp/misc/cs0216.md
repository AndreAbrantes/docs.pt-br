# <a name="compiler-error-cs0216"></a><span data-ttu-id="5eccd-102">Erro do compilador CS0216</span><span class="sxs-lookup"><span data-stu-id="5eccd-102">Compiler Error CS0216</span></span>
<span data-ttu-id="5eccd-103">O operador 'operator' requer que um operador correspondente 'missing_operator' também seja definido</span><span class="sxs-lookup"><span data-stu-id="5eccd-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="5eccd-104">Uma definida pelo usuário [verdadeira](../language-reference/keywords/true-false-operators.md) operador requer uma definida pelo usuário [falso](../language-reference/keywords/true-false-operators.md) operador e vice-versa.</span><span class="sxs-lookup"><span data-stu-id="5eccd-104">A user-defined [true](../language-reference/keywords/true-false-operators.md) operator requires a user-defined [false](../language-reference/keywords/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="5eccd-105">O exemplo a seguir gera CS0216:</span><span class="sxs-lookup"><span data-stu-id="5eccd-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```

# <a name="compiler-error-cs1920"></a><span data-ttu-id="21241-102">Erro do Compilador CS1920</span><span class="sxs-lookup"><span data-stu-id="21241-102">Compiler Error CS1920</span></span>
<span data-ttu-id="21241-103">O inicializador do elemento não pode estar vazio.</span><span class="sxs-lookup"><span data-stu-id="21241-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="21241-104">Um inicializador de coleção consiste em uma seqüência de iniciadores de elementos.</span><span class="sxs-lookup"><span data-stu-id="21241-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="21241-105">Os iniciadores do elemento não devem ser fechados em aparelhos, a menos que contenham uma expressão de atribuição.</span><span class="sxs-lookup"><span data-stu-id="21241-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="21241-106">No entanto, se você fornecer aparelhos, eles não podem estar vazios.</span><span class="sxs-lookup"><span data-stu-id="21241-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="21241-107">Se o inicializador do elemento for um inicializador de objetos, as chaves podem estar vazias desde que o inicializador contenha uma nova expressão de criação de objeto.</span><span class="sxs-lookup"><span data-stu-id="21241-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>

- <span data-ttu-id="21241-109">Adicione a expressão que falta entre os aparelhos.</span><span class="sxs-lookup"><span data-stu-id="21241-109">Add the missing expression between the braces.</span></span>

## <a name="example"></a><span data-ttu-id="21241-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="21241-111">Example</span></span>  
 <span data-ttu-id="21241-112">O exemplo a seguir gera CS1920:</span><span class="sxs-lookup"><span data-stu-id="21241-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```

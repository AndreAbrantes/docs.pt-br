<span data-ttu-id="a4576-103">O inicializador de elemento não pode estar vazio.</span><span class="sxs-lookup"><span data-stu-id="a4576-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="a4576-104">Um inicializador de coleção consiste em uma sequência de inicializadores de elemento.</span><span class="sxs-lookup"><span data-stu-id="a4576-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="a4576-105">Os inicializadores de elemento não possuem ser colocados entre chaves, a menos que eles contêm uma expressão de atribuição.</span><span class="sxs-lookup"><span data-stu-id="a4576-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="a4576-106">No entanto, se você fornecer chaves, eles não podem ser vazios.</span><span class="sxs-lookup"><span data-stu-id="a4576-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="a4576-107">Se o inicializador de elemento é um inicializador de objeto, as chaves podem estar vazias como o inicializador contém uma expressão de criação de objeto novo.</span><span class="sxs-lookup"><span data-stu-id="a4576-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>

## <a name="example"></a><span data-ttu-id="a4576-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a4576-111">Example</span></span>  
 <span data-ttu-id="a4576-112">O exemplo a seguir gera CS1920:</span><span class="sxs-lookup"><span data-stu-id="a4576-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer   
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized   
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```

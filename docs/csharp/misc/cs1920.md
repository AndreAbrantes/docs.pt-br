<span data-ttu-id="4af7a-103">O inicializador de elemento não pode estar vazio.</span><span class="sxs-lookup"><span data-stu-id="4af7a-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="4af7a-104">Um inicializador de coleção consiste em uma sequência de inicializadores de elemento.</span><span class="sxs-lookup"><span data-stu-id="4af7a-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="4af7a-105">Os inicializadores de elemento não precisam ser colocados entre chaves, a menos que contenham uma expressão de atribuição.</span><span class="sxs-lookup"><span data-stu-id="4af7a-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="4af7a-106">No entanto, se você fornecer chaves de suprimento, elas não poderão ficar vazias.</span><span class="sxs-lookup"><span data-stu-id="4af7a-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="4af7a-107">Se o inicializador de elemento for um inicializador de objeto, as chaves poderão estar vazias, desde que o inicializador contenha uma nova expressão de criação de objeto.</span><span class="sxs-lookup"><span data-stu-id="4af7a-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>

## <a name="example"></a><span data-ttu-id="4af7a-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4af7a-111">Example</span></span>  

 <span data-ttu-id="4af7a-112">O exemplo a seguir gera CS1920:</span><span class="sxs-lookup"><span data-stu-id="4af7a-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```

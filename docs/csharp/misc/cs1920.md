# <a name="compiler-error-cs1920"></a><span data-ttu-id="81411-102">CS1920 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="81411-102">Compiler Error CS1920</span></span>
<span data-ttu-id="81411-103">O inicializador de elemento não pode estar vazio.</span><span class="sxs-lookup"><span data-stu-id="81411-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="81411-104">Um inicializador de coleção consiste em uma sequência de inicializadores de elemento.</span><span class="sxs-lookup"><span data-stu-id="81411-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="81411-105">Os inicializadores de elemento não precisa ser colocado entre chaves, a menos que contenham uma expressão de atribuição.</span><span class="sxs-lookup"><span data-stu-id="81411-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="81411-106">No entanto, se você fornecer chaves, eles não podem estar vazios.</span><span class="sxs-lookup"><span data-stu-id="81411-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="81411-107">Se o inicializador de elemento é um inicializador de objeto, as chaves podem estar vazias como o inicializador contém uma nova expressão de criação de objeto.</span><span class="sxs-lookup"><span data-stu-id="81411-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="81411-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="81411-108">To correct this error</span></span>  
  
-   <span data-ttu-id="81411-109">Adicione a expressão ausente entre as chaves.</span><span class="sxs-lookup"><span data-stu-id="81411-109">Add the missing expression between the braces.</span></span>  
  
-   <span data-ttu-id="81411-110">Se a expressão deve ser um inicializador de objeto, adicione a nova expressão de criação de objeto na frente de chaves.</span><span class="sxs-lookup"><span data-stu-id="81411-110">If the expression is intended to be an object initializer, add the new object creation expression in front of the braces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="81411-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="81411-111">Example</span></span>  
 <span data-ttu-id="81411-112">O exemplo a seguir gera CS1920:</span><span class="sxs-lookup"><span data-stu-id="81411-112">The following example generates CS1920:</span></span>  
  
```  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer   
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized   
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```

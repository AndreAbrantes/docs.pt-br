# <a name="compiler-error-cs0587"></a>Erro do compilador CS0587
Erro interno no compilador: etapa "estágio"  
  
 Tente determinar se o compilador está falhando devido à sua incapacidade de analisar sintaxes inesperadas. Se isso não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

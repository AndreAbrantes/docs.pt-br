# <a name="compiler-error-cs0587"></a>Erro do Compilador CS0587

Erro do compilador interno: estágio 'estágio' do estágio

 Tente determinar se o compilador está falhando devido à sua incapacidade de analisar a sintaxe inesperada. Se este não for o caso, tente [falar conosco.](/visualstudio/ide/feedback-options)

> [!NOTE]
> Este erro do compilador não é mais usado em Roslyn.

# <a name="compiler-error-cs0587"></a>CS0587 de erro do compilador
Erro interno no compilador: etapa 'estágio'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se isso não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

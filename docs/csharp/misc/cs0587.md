# <a name="compiler-error-cs0587"></a>Erro do Compilador CS0587

Erro de compilador interno: estágio ' Stage '

 Tente determinar se o compilador está falhando devido à sua incapacidade de analisar a sintaxe inesperada. Se esse não for o caso, tente [conversar conosco](/visualstudio/ide/feedback-options).

> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

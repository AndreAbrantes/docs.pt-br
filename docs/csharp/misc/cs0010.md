# <a name="compiler-error-cs0010"></a>Erro do compilador CS0010
Erro fatal inesperado--'error'.  
  
 Esse erro é gerado quando algo completamente inesperado ocorre para interromper a compilação.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Recompile a outro projeto. Se você receber o mesmo erro, tente reinstalar o Visual Studio. Se você receber esse erro, envie um relatório de erros à Microsoft.

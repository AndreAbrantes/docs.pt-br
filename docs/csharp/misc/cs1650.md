# <a name="compiler-error-cs1650"></a><span data-ttu-id="d99cf-103">Erro do Compilador CS1650</span><span class="sxs-lookup"><span data-stu-id="d99cf-103">Compiler Error CS1650</span></span>
<span data-ttu-id="d99cf-104">Os campos do campo somente leitura estático ' identifier ' não podem ser atribuídos (exceto em um construtor estático ou inicializador de variável)</span><span class="sxs-lookup"><span data-stu-id="d99cf-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="d99cf-105">Esse erro ocorre quando você tenta modificar um membro de um campo que é ReadOnly e estático, onde não é permitido modificá-lo.</span><span class="sxs-lookup"><span data-stu-id="d99cf-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="d99cf-106">Para resolver esse erro, limite as atribuições a campos ReadOnly para o construtor ou inicializador de variável, ou remova a `readonly` palavra-chave da declaração do campo.</span><span class="sxs-lookup"><span data-stu-id="d99cf-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```

# <a name="compiler-error-cs1650"></a><span data-ttu-id="63590-102">CS1650 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="63590-102">Compiler Error CS1650</span></span>
<span data-ttu-id="63590-103">Campos de somente leitura e estático 'Identificador de campo' não podem ser atribuídos a (exceto em um construtor estático ou um inicializador de variável)</span><span class="sxs-lookup"><span data-stu-id="63590-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="63590-104">Esse erro ocorre quando você tenta modificar um membro de um campo que é somente leitura e estático onde não é permitido a ser modificado.</span><span class="sxs-lookup"><span data-stu-id="63590-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="63590-105">Para resolver esse erro, limite atribuições a campos somente leitura para o construtor ou inicializador de variável, ou remova o `readonly` palavra-chave da declaração do campo.</span><span class="sxs-lookup"><span data-stu-id="63590-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```

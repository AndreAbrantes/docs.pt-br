# <a name="compiler-error-cs0110"></a><span data-ttu-id="e6e3a-102">CS0110 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="e6e3a-102">Compiler Error CS0110</span></span>
<span data-ttu-id="e6e3a-103">A avaliação do valor de constante para 'declaração const' envolve uma definição circular</span><span class="sxs-lookup"><span data-stu-id="e6e3a-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="e6e3a-104">A declaração de um [const](../../csharp/language-reference/keywords/const.md) variável (`a`) não pode fazer referência a outra variável const (`b`) que também faz referência (`a`).</span><span class="sxs-lookup"><span data-stu-id="e6e3a-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="e6e3a-105">O exemplo a seguir gera CS0110:</span><span class="sxs-lookup"><span data-stu-id="e6e3a-105">The following sample generates CS0110:</span></span>  
  
```  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```

# <a name="compiler-error-cs0544"></a>Erro do compilador CS0544
'substituição de propriedade': não é possível substituir porque 'não property' não é uma propriedade  
  
 Foi feita uma tentativa para substituir um tipo de dados nonproperty como uma [propriedade](../../csharp/programming-guide/classes-and-structs/properties.md), que não é permitido.  
  
 O exemplo a seguir gera CS0544:  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```

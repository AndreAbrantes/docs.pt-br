# <a name="compiler-error-cs0541"></a>Erro do compilador CS0541
'declaração de ': declaração de interface explícita só pode ser declarada em uma classe ou struct  
  
 Um explícito [interface](../../csharp/language-reference/keywords/interface.md) declaração foi encontrada fora de um [classe](../../csharp/language-reference/keywords/class.md) ou [struct](../../csharp/language-reference/keywords/struct.md).  
  
 O exemplo a seguir gera CS0541:  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```

# <a name="compiler-error-cs1722"></a><span data-ttu-id="e9cba-102">CS1722 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="e9cba-102">Compiler Error CS1722</span></span>

## <a name="example"></a><span data-ttu-id="e9cba-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e9cba-105">Example</span></span>  
 <span data-ttu-id="e9cba-106">O exemplo a seguir gera CS1722.</span><span class="sxs-lookup"><span data-stu-id="e9cba-106">The following sample generates CS1722.</span></span>

public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722  
public class MyClass2 : A, I {}   // OK

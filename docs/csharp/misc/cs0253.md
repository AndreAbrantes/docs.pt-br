 <span data-ttu-id="c163d-105">O compilador está fazendo uma comparação de referência.</span><span class="sxs-lookup"><span data-stu-id="c163d-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="c163d-106">Se você quiser comparar o valor de cadeias de caracteres, converta o lado direito da expressão para `type` .</span><span class="sxs-lookup"><span data-stu-id="c163d-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="c163d-107">O exemplo a seguir gera CS0253:</span><span class="sxs-lookup"><span data-stu-id="c163d-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs

class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```

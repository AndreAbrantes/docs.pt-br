# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="92b86-102">Compilador CS0253 de aviso (nível 2)</span><span class="sxs-lookup"><span data-stu-id="92b86-102">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="92b86-103">Comparação de referência não intencional possível; Para obter uma comparação de valor, converta o lado direito no tipo 'type'</span><span class="sxs-lookup"><span data-stu-id="92b86-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="92b86-104">O compilador está fazendo uma comparação de referência.</span><span class="sxs-lookup"><span data-stu-id="92b86-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="92b86-105">Se você quiser comparar o valor de cadeias de caracteres, converter o lado direito da expressão `type`.</span><span class="sxs-lookup"><span data-stu-id="92b86-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="92b86-106">O exemplo a seguir gera CS0253:</span><span class="sxs-lookup"><span data-stu-id="92b86-106">The following sample generates CS0253:</span></span>  
  
```csharp 
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```

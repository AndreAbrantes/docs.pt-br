# <a name="compiler-error-cs0009"></a>Erro do compilador CS0009
Não foi possível abrir o arquivo de metadados 'file' — 'Descrição'  
  
 O arquivo especificado com o [/Reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) opção de compilador não contém metadados válidos.

# <a name="compiler-error-cs0009"></a>Erro do compilador CS0009
Não foi possível abrir o arquivo de metadados ' file ' — ' description '  
  
 O arquivo especificado com a opção de compilador [/Reference](../language-reference/compiler-options/reference-compiler-option.md) não contém metadados válidos.

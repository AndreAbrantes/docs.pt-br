# <a name="compiler-warning-level-3-cs1702"></a>Compilador CS1702 de aviso (nível 3)
Presumindo que a referência de assembly "Nome de Assembly nº 1" corresponda a "Nome de Assembly nº 2", talvez seja necessário fornecer a política em tempo de execução  
  
 As referências de duas assembly tem diferentes de compilação e/ou números de revisão, então não será unificar automaticamente. Talvez seja necessário fornecer a diretiva de tempo de execução para forçar a união usando diretivas no arquivo. config do aplicativo.

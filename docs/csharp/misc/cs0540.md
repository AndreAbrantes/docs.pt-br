 <span data-ttu-id="cc93c-104">Você tentou implementar um membro de interface em um [classe](../../csharp/language-reference/keywords/class.md) que não derivam de [interface](../../csharp/language-reference/keywords/interface.md).</span><span class="sxs-lookup"><span data-stu-id="cc93c-104">You attempted to implement an interface member in a [class](../../csharp/language-reference/keywords/class.md) that does not derive from the [interface](../../csharp/language-reference/keywords/interface.md).</span></span> <span data-ttu-id="cc93c-105">Você deve excluir a implementação do membro de interface ou adicionar a interface para a lista de classe base da classe.</span><span class="sxs-lookup"><span data-stu-id="cc93c-105">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cc93c-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="cc93c-106">Example</span></span>  
 <span data-ttu-id="cc93c-107">O exemplo a seguir gera CS0540.</span><span class="sxs-lookup"><span data-stu-id="cc93c-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```

## <a name="example"></a><span data-ttu-id="b7fd3-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="b7fd3-106">Example</span></span>  
 <span data-ttu-id="b7fd3-107">O exemplo a seguir gera CS0540.</span><span class="sxs-lookup"><span data-stu-id="b7fd3-107">The following sample generates CS0540.</span></span>  
  
```  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```

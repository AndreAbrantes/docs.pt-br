# <a name="compiler-error-cs0540"></a><span data-ttu-id="ee63a-103">Erro do Compilador CS0540</span><span class="sxs-lookup"><span data-stu-id="ee63a-103">Compiler Error CS0540</span></span>

 <span data-ttu-id="ee63a-105">Você tentou implementar um membro de interface em uma [classe](../language-reference/keywords/class.md) que não é derivada da [interface](../language-reference/keywords/interface.md).</span><span class="sxs-lookup"><span data-stu-id="ee63a-105">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="ee63a-106">Você deve excluir a implementação do membro da interface ou adicionar a interface à lista de classe base da classe.</span><span class="sxs-lookup"><span data-stu-id="ee63a-106">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ee63a-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ee63a-107">Example</span></span>  

 <span data-ttu-id="ee63a-108">O exemplo a seguir gera CS0540.</span><span class="sxs-lookup"><span data-stu-id="ee63a-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```

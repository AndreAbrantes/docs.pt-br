# <a name="compiler-error-cs0540"></a>Erro do compilador CS0540
'membro de interface ': tipo recipiente não implementa a interface 'interface'  
  
 Você tentou implementar um membro de interface em um [classe](../../csharp/language-reference/keywords/class.md) que não derivam de [interface](../../csharp/language-reference/keywords/interface.md). Você deve excluir a implementação de membro de interface ou adicionar a interface à lista de classe base da classe.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0540.  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0540.  
  
```csharp  
// CS0540_b.cs  
using System;  
class C {  
   void IDisposable.Dispose() {}   // CS0540  
}  
  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```

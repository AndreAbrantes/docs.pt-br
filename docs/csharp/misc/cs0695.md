# <a name="compiler-error-cs0695"></a><span data-ttu-id="0939b-102">CS0695 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="0939b-102">Compiler Error CS0695</span></span>
<span data-ttu-id="0939b-103">'tipo genérico' não pode implementar 'interface genérica' e 'interface genérica' porque eles podem se unificar em algumas substituições de parâmetro de tipo</span><span class="sxs-lookup"><span data-stu-id="0939b-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="0939b-104">Esse erro ocorre quando uma classe genérica implementa mais de uma parametrização da mesma interface genérica e existe uma substituição de parâmetro de tipo que criam as duas interfaces idênticas.</span><span class="sxs-lookup"><span data-stu-id="0939b-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="0939b-105">Para evitar esse erro, implementar apenas uma das interfaces ou alterar os parâmetros de tipo para evitar o conflito.</span><span class="sxs-lookup"><span data-stu-id="0939b-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="0939b-106">O exemplo a seguir gera CS0695:</span><span class="sxs-lookup"><span data-stu-id="0939b-106">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```

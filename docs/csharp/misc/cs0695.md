 <span data-ttu-id="15ea6-104">Esse erro ocorre quando uma classe genérica implementa mais de uma parametrização da mesma interface genérica e existe uma substituição de parâmetro de tipo que criam as duas interfaces idênticas.</span><span class="sxs-lookup"><span data-stu-id="15ea6-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="15ea6-105">Para evitar esse erro, implementar apenas uma das interfaces ou alterar os parâmetros de tipo para evitar o conflito.</span><span class="sxs-lookup"><span data-stu-id="15ea6-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>

# <a name="compiler-error-cs0077"></a><span data-ttu-id="5add2-102">Erro do compilador CS0077</span><span class="sxs-lookup"><span data-stu-id="5add2-102">Compiler Error CS0077</span></span>
<span data-ttu-id="5add2-103">O operador as deve ser usado com um tipo de referência ou tipo anulável (' int ' é um tipo de valor não anulável).</span><span class="sxs-lookup"><span data-stu-id="5add2-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="5add2-104">O [operador](../language-reference/operators/type-testing-and-cast.md#as-operator) as passou um [tipo de valor](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="5add2-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="5add2-105">Como `as` pode retornar [NULL](../language-reference/keywords/null.md), só pode ser passado [tipos de referência](../language-reference/keywords/reference-types.md) ou tipos de valor anulável.</span><span class="sxs-lookup"><span data-stu-id="5add2-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable value types.</span></span> <span data-ttu-id="5add2-106">Para obter mais informações sobre tipos de valor anulável, consulte [tipos de valor anulável](../programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="5add2-106">For more information about nullable value types, see [Nullable value types](../programming-guide/nullable-types/index.md).</span></span>
  
 <span data-ttu-id="5add2-107">O exemplo a seguir gera CS0077:</span><span class="sxs-lookup"><span data-stu-id="5add2-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```

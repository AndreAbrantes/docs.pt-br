# <a name="compiler-error-cs0077"></a><span data-ttu-id="8b5e7-102">Erro do compilador CS0077</span><span class="sxs-lookup"><span data-stu-id="8b5e7-102">Compiler Error CS0077</span></span>
<span data-ttu-id="8b5e7-103">O operador deve ser usado com um tipo de referência ou tipo anulável ('int' é um tipo de valor não anulável).</span><span class="sxs-lookup"><span data-stu-id="8b5e7-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="8b5e7-104">O [como](../../csharp/language-reference/keywords/as.md) operador foi passado um [tipo de valor](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="8b5e7-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="8b5e7-105">Porque `as` pode retornar [nulo](../../csharp/language-reference/keywords/null.md), só podem ser passada [tipos de referência](../../csharp/language-reference/keywords/reference-types.md) ou um tipo anulável.</span><span class="sxs-lookup"><span data-stu-id="8b5e7-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="8b5e7-106">Para obter mais informações sobre tipos anuláveis, consulte [tipos anuláveis](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="8b5e7-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="8b5e7-107">O exemplo a seguir gera CS0077:</span><span class="sxs-lookup"><span data-stu-id="8b5e7-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```

<span data-ttu-id="7644a-103">A variável ' Variable ' é atribuída, mas seu valor nunca é usado</span><span class="sxs-lookup"><span data-stu-id="7644a-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="7644a-104">O compilador emite um aviso de nível três, quando você declara e atribui uma variável, mas não a utiliza.</span><span class="sxs-lookup"><span data-stu-id="7644a-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="7644a-105">O compilador gera esse aviso somente quando o valor da variável é uma constante de tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="7644a-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="7644a-106">Atribuir uma expressão não constante ou um resultado de método a uma variável local torna mais fácil observar essas expressões no depurador.</span><span class="sxs-lookup"><span data-stu-id="7644a-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="7644a-107">Ele também torna o resultado acessível, impedindo a coleta de lixo enquanto essa variável estiver acessível.</span><span class="sxs-lookup"><span data-stu-id="7644a-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>
 
 <span data-ttu-id="7644a-108">O exemplo a seguir mostra os casos em que e quando o aviso não é gerado:</span><span class="sxs-lookup"><span data-stu-id="7644a-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }
   
   private static int GetZero()
   {
       return 0;
   }
}  
```

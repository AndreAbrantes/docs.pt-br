# <a name="compiler-error-cs0210"></a><span data-ttu-id="0482c-102">CS0210 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="0482c-102">Compiler Error CS0210</span></span>
<span data-ttu-id="0482c-103">Forneça um inicializador em uma declaração de instrução fixed ou using</span><span class="sxs-lookup"><span data-stu-id="0482c-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="0482c-104">Você deve declarar e inicializar a variável em uma [instrução fixa](../../csharp/language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="0482c-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="0482c-105">Para obter mais informações, consulte [código não seguro e ponteiros](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="0482c-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="0482c-106">O exemplo a seguir gera CS0210:</span><span class="sxs-lookup"><span data-stu-id="0482c-106">The following sample generates CS0210:</span></span>  
  
```  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="0482c-107">O exemplo a seguir também gera CS0210 porque o [usando a instrução](../../csharp/language-reference/keywords/using-statement.md) não tem nenhum inicializador.</span><span class="sxs-lookup"><span data-stu-id="0482c-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```

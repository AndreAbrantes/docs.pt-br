## <a name="example"></a><span data-ttu-id="f8c5c-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f8c5c-108">Example</span></span>  
 <span data-ttu-id="f8c5c-109">O código a seguir gera CS1928 e CS1929:</span><span class="sxs-lookup"><span data-stu-id="f8c5c-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```

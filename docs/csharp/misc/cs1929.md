# <a name="compiler-error-cs1929"></a><span data-ttu-id="52d67-102">CS1929 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="52d67-102">Compiler Error CS1929</span></span>
<span data-ttu-id="52d67-103">Argumento de instância: não é possível converter de 'typeA' em 'typeB'.</span><span class="sxs-lookup"><span data-stu-id="52d67-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="52d67-104">Esse erro é gerado quando você tentar chamar um método de extensão de uma classe que não estende.</span><span class="sxs-lookup"><span data-stu-id="52d67-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="52d67-105">No exemplo mostrado aqui, o método de extensão está definido para a classe derivada `A`, mas não para a classe base `B`.</span><span class="sxs-lookup"><span data-stu-id="52d67-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="52d67-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="52d67-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="52d67-107">Crie um novo método de extensão para o tipo em que você precisa chamá-la, caso contrário, mova a chamada para um objeto do tipo que estende o método existente.</span><span class="sxs-lookup"><span data-stu-id="52d67-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="52d67-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="52d67-108">Example</span></span>  
 <span data-ttu-id="52d67-109">O código a seguir gera CS1928 e CS1929:</span><span class="sxs-lookup"><span data-stu-id="52d67-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```

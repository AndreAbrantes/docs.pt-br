# <a name="compiler-error-cs1929"></a><span data-ttu-id="0feb8-102">Erro do Compilador CS1929</span><span class="sxs-lookup"><span data-stu-id="0feb8-102">Compiler Error CS1929</span></span>

<span data-ttu-id="0feb8-103">Argumento de instância: não é possível converter de ' TypeA ' em ' typeB '.</span><span class="sxs-lookup"><span data-stu-id="0feb8-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="0feb8-104">Esse erro é gerado quando você tenta invocar um método de extensão de uma classe que ele não estende.</span><span class="sxs-lookup"><span data-stu-id="0feb8-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="0feb8-105">No exemplo mostrado aqui, o método de extensão é definido para a classe derivada `A` , mas não para a classe base `B` .</span><span class="sxs-lookup"><span data-stu-id="0feb8-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0feb8-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="0feb8-106">To correct this error</span></span>  
  
1. <span data-ttu-id="0feb8-107">Crie um novo método de extensão para o tipo onde você precisa invocá-lo, ou então mova a chamada para um objeto do tipo que o método existente estende.</span><span class="sxs-lookup"><span data-stu-id="0feb8-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0feb8-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="0feb8-108">Example</span></span>  

 <span data-ttu-id="0feb8-109">O código a seguir gera CS1928 e CS1929:</span><span class="sxs-lookup"><span data-stu-id="0feb8-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```

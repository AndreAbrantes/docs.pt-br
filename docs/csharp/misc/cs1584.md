# <a name="compiler-warning-level-1-cs1584"></a>Compilador CS1584 de aviso (nível 1)
O comentário XML em 'member' tem atributo cref sintaticamente incorreto 'invalid_syntax'  
  
 Um dos parâmetros passados para uma marca de comentário de documentação tem sintaxe inválida. Para obter mais informações, consulte [marcas recomendadas para comentários de documentação](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1584.  
  
```  
// CS1584.cs  
// compile with: /W:1 /doc:CS1584.xml  
/// <remarks>Test class</remarks>  
public class Test  
{  
   /// <remarks>Called in <see cref="Test.Mai()n"/>.</remarks>   // CS1584  
   // try the following line instead  
   // /// <remarks>Called in <see cref="Test.Main()"/>.</remarks>  
   public static void Test2() {}  
  
   /// <remarks>Main method</remarks>  
   public static void Main() {}  
}  
```

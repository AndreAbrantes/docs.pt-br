 <span data-ttu-id="1de7e-104">O [params](../../csharp/language-reference/keywords/params.md) parâmetro oferece suporte a um número variável de argumentos e deve ser após todos os outros parâmetros.</span><span class="sxs-lookup"><span data-stu-id="1de7e-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="1de7e-105">Para saber mais, veja [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="1de7e-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="1de7e-106">O exemplo a seguir gera CS0231:</span><span class="sxs-lookup"><span data-stu-id="1de7e-106">The following sample generates CS0231:</span></span>

```  
// CS0231.cs

   // To resolve the error, use the following line instead:

   {  
   }  
}  
```

<span data-ttu-id="3c681-103">Um parâmetro params deve ser o último parâmetro em uma lista de parâmetros formais.</span><span class="sxs-lookup"><span data-stu-id="3c681-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="3c681-104">O [params](../../csharp/language-reference/keywords/params.md) parâmetro dá suporte a um número variável de argumentos e deve ser posterior a todos os outros parâmetros.</span><span class="sxs-lookup"><span data-stu-id="3c681-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="3c681-105">Para saber mais, veja [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="3c681-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="3c681-106">O exemplo a seguir gera CS0231:</span><span class="sxs-lookup"><span data-stu-id="3c681-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```

# <a name="compiler-error-cs0231"></a><span data-ttu-id="7345a-103">Erro do Compilador CS0231</span><span class="sxs-lookup"><span data-stu-id="7345a-103">Compiler Error CS0231</span></span>

<span data-ttu-id="7345a-104">Um parâmetro params deve ser o último parâmetro em uma lista de parâmetros formais.</span><span class="sxs-lookup"><span data-stu-id="7345a-104">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="7345a-105">O parâmetro [params](../language-reference/keywords/params.md) dá suporte a um número variável de argumentos e deve ser após todos os outros parâmetros.</span><span class="sxs-lookup"><span data-stu-id="7345a-105">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="7345a-106">Para saber mais, veja [Métodos](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="7345a-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="7345a-107">O exemplo a seguir gera CS0231:</span><span class="sxs-lookup"><span data-stu-id="7345a-107">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```

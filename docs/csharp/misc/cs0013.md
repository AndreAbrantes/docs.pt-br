# <a name="compiler-error-cs0013"></a>Erro do compilador CS0013
Erro inesperado ao gravar metadados no arquivo 'ModelStore.dll' – ' nenhum espaço lógico da esquerda para criar mais cadeias de caracteres do usuário.  
  
 O .NET Framework common language runtime não emitir metadados. Verifique se que o caminho está correto e que o disco não está cheio. Se o problema persistir, você terá que reparar ou reinstalar o Visual Studio e/ou o .NET Framework.

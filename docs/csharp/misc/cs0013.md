# <a name="compiler-error-cs0013"></a>Erro do Compilador CS0013
Erro inesperado ao gravar metadados no arquivo ' ModelStore.dll '--' nenhum espaço lógico deixado para criar mais cadeias de caracteres de usuário.  
  
 O .NET Framework Common Language Runtime não emitiu metadados. Verifique se o caminho está correto e se o disco não está cheio. Se o problema persistir, talvez seja necessário reparar ou reinstalar o Visual Studio e/ou o .NET Framework.

# <a name="compiler-error-cs0013"></a>CS0013 de erro do compilador
Erro inesperado ao gravar metadados no arquivo 'ModelStore.dll' - ' nenhum espaço lógico deixado para criar mais cadeias de caracteres do usuário.  
  
 O .NET Framework common language runtime não emitir metadados. Verifique se o caminho está correto e se o disco não está completo. Se o problema persistir, você precisará repare ou reinstale o Visual Studio e/ou o .NET Framework.

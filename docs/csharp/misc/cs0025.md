# <a name="compiler-error-cs0025"></a>CS0025 de erro do compilador
Não foi possível encontrar o arquivo de biblioteca padrão 'file'  
  
 Não foi encontrado um arquivo que é necessário pelo compilador. Verifique se o caminho está correto e se o arquivo existe.  
  
 Se o arquivo é um arquivo de sistema do Visual Studio, convém reparar sua instalação do Visual Studio ou reinstalá-lo completamente.

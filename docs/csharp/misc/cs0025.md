# <a name="compiler-error-cs0025"></a>Erro do compilador CS0025
Não foi possível encontrar o arquivo de biblioteca padrão 'file'  
  
 Não foi encontrado um arquivo que é necessário pelo compilador. Verifique se o caminho está correto e se o arquivo existe.  
  
 Se o arquivo for um arquivo de sistema do Visual Studio, você talvez precise reparar sua instalação do Visual Studio ou reinstalá-lo completamente.

# <a name="compiler-error-cs1606"></a>CS1606 de erro do compilador
Falha na assinatura de assembly; saída não pode ser assinada-- razão  
  
 O assembly foi criado, mas quando o compilador tentou concluir assiná-lo, houve uma falha.

# <a name="compiler-error-cs1606"></a>Erro do compilador CS1606
Falha na assinatura do assembly; saída não pode ser assinada-- motivo  
  
 O assembly foi gerado, mas quando o compilador tentou concluir assiná-lo, houve uma falha.

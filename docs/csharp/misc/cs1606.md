# <a name="compiler-error-cs1606"></a>CS1606 de erro do compilador
Assinatura de assembly com falha; saída não pode ser assinada-- motivo  
  
 O assembly foi gerado, mas quando o compilador tentou concluir assiná-lo, houve uma falha.

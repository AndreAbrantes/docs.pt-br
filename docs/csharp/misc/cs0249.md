# <a name="compiler-error-cs0249"></a>Erro do compilador CS0249
Não substitua o objeto. Finalize. Em vez disso, forneça um destruidor.  
  
 Use a sintaxe do destruidor para especificar as instruções para executar quando o objeto é destruído.  

 O exemplo a seguir gera CS0249:  
  
```csharp  
// CS0249.cs  
class MyClass  
{  
   protected override void Finalize()   // CS0249  
   // try the following line instead  
   // ~MyClass()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

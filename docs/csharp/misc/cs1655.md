 <span data-ttu-id="5cdd8-104">Esse erro ocorre se você estiver tentando passar um membro de uma variável [forcada,](../language-reference/keywords/foreach-in.md) uma variável [de uso](../language-reference/keywords/using-statement.md) ou uma variável [fixa](../language-reference/keywords/fixed-statement.md) para uma função como um argumento de ref ou out.</span><span class="sxs-lookup"><span data-stu-id="5cdd8-104">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="5cdd8-105">Como essas variáveis são consideradas somente leitura nesses contextos, isso não é permitido.</span><span class="sxs-lookup"><span data-stu-id="5cdd8-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>

```csharp  
// CS1655.cs

   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }

     S[] sa = new S[10];

     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```

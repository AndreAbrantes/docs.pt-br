 <span data-ttu-id="98caf-105">Uma declaração de classe tentou substituir um não método em uma classe base.</span><span class="sxs-lookup"><span data-stu-id="98caf-105">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="98caf-106">As substituições devem corresponder ao tipo de membro.</span><span class="sxs-lookup"><span data-stu-id="98caf-106">Overrides must match the member type.</span></span> <span data-ttu-id="98caf-107">Se for desejado um método com o mesmo nome de um método em uma classe base, use [New](../language-reference/keywords/new-modifier.md) (e não [override](../language-reference/keywords/override.md)) na declaração do método na classe base.</span><span class="sxs-lookup"><span data-stu-id="98caf-107">If a method with the same name as a method in a base class is desired, use [new](../language-reference/keywords/new-modifier.md) (and not [override](../language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>

   public int i;

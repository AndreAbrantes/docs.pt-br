# <a name="compiler-error-cs1541"></a>Erro do compilador CS1541
Opção de referência inválida: ' Symbol ' — não é possível referenciar diretórios  
  
 O compilador detectou uma tentativa de especificar um diretório em vez de um arquivo específico. Por exemplo, ao usar a opção de compilador [/Reference](../language-reference/compiler-options/reference-compiler-option.md) , você deve especificar um arquivo; Não é possível especificar um diretório.  
  
 Por exemplo, passar `/reference:c:\` para o compilador geraria CS1541.

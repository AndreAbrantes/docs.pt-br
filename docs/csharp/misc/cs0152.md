 <span data-ttu-id="1515c-106">O exemplo a seguir gera CS0152:</span><span class="sxs-lookup"><span data-stu-id="1515c-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```

<span data-ttu-id="7b710-104">O tipo não genérico-ou-método ' identifier ' não pode ser usado com argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="7b710-104">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="7b710-105">O método ou o tipo não é genérico, mas foi usado com argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="7b710-105">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="7b710-106">Para evitar esse erro, remova os colchetes angulares e os argumentos de tipo, ou redeclare o método ou o tipo como um método ou tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="7b710-106">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="7b710-107">O exemplo a seguir gera CS0308:</span><span class="sxs-lookup"><span data-stu-id="7b710-107">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="7b710-108">O exemplo a seguir também gera CS0308.</span><span class="sxs-lookup"><span data-stu-id="7b710-108">The following example also generates CS0308.</span></span> <span data-ttu-id="7b710-109">Para resolver o erro, use a diretiva "usando System. Collections. Generic."</span><span class="sxs-lookup"><span data-stu-id="7b710-109">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```

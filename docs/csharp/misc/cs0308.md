# <a name="compiler-error-cs0308"></a><span data-ttu-id="4ae0f-102">CS0308 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="4ae0f-102">Compiler Error CS0308</span></span>
<span data-ttu-id="4ae0f-103">O método não genérico tipo-ou-'Identificador' não pode ser usado com argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="4ae0f-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="4ae0f-104">O método ou tipo não é genérico mas foi usado com argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="4ae0f-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="4ae0f-105">Para evitar esse erro, remova os colchetes angulares e argumentos de tipo ou declarar novamente o método ou tipo como um método genérico ou o tipo.</span><span class="sxs-lookup"><span data-stu-id="4ae0f-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="4ae0f-106">O exemplo a seguir gera CS0308:</span><span class="sxs-lookup"><span data-stu-id="4ae0f-106">The following example generates CS0308:</span></span>  
  
```  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="4ae0f-107">O exemplo a seguir também gera CS0308.</span><span class="sxs-lookup"><span data-stu-id="4ae0f-107">The following example also generates CS0308.</span></span> <span data-ttu-id="4ae0f-108">Para resolver o erro, use a diretiva "using Generic".</span><span class="sxs-lookup"><span data-stu-id="4ae0f-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```

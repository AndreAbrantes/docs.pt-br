# <a name="compiler-warning-level-1-cs1202"></a>Compilador CS1202 de aviso (nível 1)
O recurso 'recurso inválido' é preterido. Em vez disso, use 'recurso válido' '.  
  
 O recurso que você está tentando usar é substituído. Atualize seu código para usar o recurso válido em vez disso.

# <a name="compiler-error-cs0035"></a><span data-ttu-id="a8544-102">Erro do compilador CS0035</span><span class="sxs-lookup"><span data-stu-id="a8544-102">Compiler Error CS0035</span></span>
<span data-ttu-id="a8544-103">O operador 'operator' é ambíguo em um operando do tipo 'type'</span><span class="sxs-lookup"><span data-stu-id="a8544-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="a8544-104">O compilador tem mais de uma conversão disponível e não sabe qual escolher antes de aplicar o operador.</span><span class="sxs-lookup"><span data-stu-id="a8544-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="a8544-105">O exemplo a seguir gera CS0035:</span><span class="sxs-lookup"><span data-stu-id="a8544-105">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```

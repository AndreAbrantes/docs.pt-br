# <a name="compiler-error-cs1562"></a>CS1562 de erro do compilador
Saídas sem origem devem ter a opção /out especificada  
  
 A compilação foi possível criar um arquivo de saída, mas não havia nenhum arquivo de código fonte como entrada do qual o nome do arquivo de saída foi indicado. Por exemplo, você pode estar tentando compilar um arquivo ou recurso-somente de metadados.  
  
 Use o [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) opção de compilador para especificar o nome do arquivo de saída.

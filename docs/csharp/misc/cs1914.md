## <a name="example"></a><span data-ttu-id="fd7aa-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="fd7aa-109">Example</span></span>  

 <span data-ttu-id="fd7aa-110">O código a seguir gera CS1914 porque o inicializador tenta inicializar o `TestClass.Number` campo, que é `static` :</span><span class="sxs-lookup"><span data-stu-id="fd7aa-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```

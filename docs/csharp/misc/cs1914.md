## <a name="example"></a><span data-ttu-id="e9fe6-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e9fe6-109">Example</span></span>  
 <span data-ttu-id="e9fe6-110">O código a seguir gera CS1914 porque o inicializador tenta inicializar o `TestClass.Number` campo, que é `static`:</span><span class="sxs-lookup"><span data-stu-id="e9fe6-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```

# <a name="compiler-error-cs0447"></a><span data-ttu-id="38ee2-102">CS0447 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="38ee2-102">Compiler Error CS0447</span></span>
<span data-ttu-id="38ee2-103">Atributos não podem ser usados em argumentos de tipo, somente em parâmetros de tipo</span><span class="sxs-lookup"><span data-stu-id="38ee2-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="38ee2-104">Esse erro ocorre quando você aplicar um atributo a um argumento de tipo que ocorre em uma instrução de chamada.</span><span class="sxs-lookup"><span data-stu-id="38ee2-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="38ee2-105">É aceitável para aplicar um atributo para um parâmetro de tipo em uma instrução de declaração de classe ou método, como o seguinte:</span><span class="sxs-lookup"><span data-stu-id="38ee2-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```  
  
 <span data-ttu-id="38ee2-106">A linha de código a seguir irá gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="38ee2-106">The following line of code will generate this error.</span></span> <span data-ttu-id="38ee2-107">Supõe-se que a classe `C`, definido na linha de código anterior tem um método estático chamado `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="38ee2-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```

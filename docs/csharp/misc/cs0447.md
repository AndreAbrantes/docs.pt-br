<span data-ttu-id="f18d8-104">Atributos não podem ser usados em argumentos de tipo, somente em parâmetros de tipo</span><span class="sxs-lookup"><span data-stu-id="f18d8-104">Attributes cannot be used on type arguments, only on type parameters</span></span>

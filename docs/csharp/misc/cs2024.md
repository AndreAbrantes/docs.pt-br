# <a name="compiler-error-cs2024"></a>Erro do compilador CS2024
Número de alinhamento de seção de arquivo inválido ' # '  
  
 Um valor inválido foi passado para a opção de compilador [/filealign](../language-reference/compiler-options/filealign-compiler-option.md) .  
  
 O exemplo a seguir gera CS2024:  
  
```csharp  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```

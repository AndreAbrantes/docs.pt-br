<span data-ttu-id="4c26a-104">A comparação com a constante integral é inútil; a constante está fora do intervalo do tipo ' type '</span><span class="sxs-lookup"><span data-stu-id="4c26a-104">Comparison to integral constant is useless; the constant is outside the range of type 'type'</span></span>  
  
 <span data-ttu-id="4c26a-105">O compilador detectou uma comparação entre uma constante e uma variável em que a constante está fora do intervalo da variável.</span><span class="sxs-lookup"><span data-stu-id="4c26a-105">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>  
  
 <span data-ttu-id="4c26a-106">O exemplo a seguir gera CS0652:</span><span class="sxs-lookup"><span data-stu-id="4c26a-106">The following sample generates CS0652:</span></span>  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```

# <a name="compiler-error-cs1620"></a><span data-ttu-id="0ded0-102">Erro do compilador CS1620</span><span class="sxs-lookup"><span data-stu-id="0ded0-102">Compiler Error CS1620</span></span>
<span data-ttu-id="0ded0-103">O argumento ' Number ' deve ser passado com a palavra-chave ' keyword '</span><span class="sxs-lookup"><span data-stu-id="0ded0-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="0ded0-104">Esse erro ocorre se você estiver passando um argumento para uma função que usa um parâmetro [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md) e você não inclui a `ref` palavra `out` -chave or no ponto de chamada, ou você inclui a palavra-chave incorreta.</span><span class="sxs-lookup"><span data-stu-id="0ded0-104">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="0ded0-105">O texto do erro indica a palavra-chave apropriada a ser usada e qual argumento causou a falha.</span><span class="sxs-lookup"><span data-stu-id="0ded0-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="0ded0-106">O exemplo a seguir gera CS1620:</span><span class="sxs-lookup"><span data-stu-id="0ded0-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```

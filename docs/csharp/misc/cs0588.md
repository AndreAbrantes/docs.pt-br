# <a name="compiler-error-cs0588"></a>CS0588 de erro do compilador
Erro interno no compilador: etapa 'LEX'  
  
 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se isso não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

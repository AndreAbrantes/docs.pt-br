# <a name="compiler-error-cs0588"></a>Erro do compilador CS0588

Erro de compilador interno: estágio ' LEX '

 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se esse não for o caso, tente [conversar conosco](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

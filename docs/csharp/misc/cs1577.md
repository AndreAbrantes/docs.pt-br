# <a name="compiler-error-cs1577"></a>CS1577 de erro do compilador
Falha na geração do assembly — motivo  
  
 A parte de geração de assembly da compilação falha depois que o compilador c# o código-fonte compilado com êxito. Consulte a documentação de erros para o utilitário ALink ([Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) para obter mais informações.

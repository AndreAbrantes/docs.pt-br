# <a name="compiler-error-cs1577"></a>Erro do compilador CS1577
Falha na geração do assembly — motivo  
  
 A parte de geração de assembly da compilação falhou após o C# compilador compilado com êxito o código-fonte. Consulte a documentação de erros para o utilitário ALink ([Al.exe Tool Errors and Warnings](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)) para obter mais informações.

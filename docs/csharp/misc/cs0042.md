# <a name="compiler-error-cs0042"></a>Erro do compilador CS0042
Erro inesperado ao criar informações de depuração 'arquivo' — 'reason'  
  
 O compilador não foi possível criar informações de depuração. *motivo* contém informações adicionais sobre a condição de erro.

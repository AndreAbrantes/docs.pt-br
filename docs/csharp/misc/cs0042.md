# <a name="compiler-error-cs0042"></a>CS0042 de erro do compilador
Erro inesperado ao criar informações de depuração 'arquivo', 'razão'  
  
 O compilador não pôde criar as informações de depuração. *motivo* contém informações adicionais sobre a condição de erro.

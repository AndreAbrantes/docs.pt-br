# <a name="compiler-error-cs1714"></a>CS1714 de erro do compilador
A classe base ou interface de TypeName1 não pôde ser resolvida ou é inválido  
  
 Você está implementando TypeName1 de uma classe ou interface que o não pôde ser resolvido, por exemplo, o compilador não foi possível localizá-lo, ou é inválido. A solução é determinar qual dos dois casos é e especifique o local do tipo, mais corretamente, ou corrija os erros de compilador na classe base.

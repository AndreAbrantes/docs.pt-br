# <a name="compiler-error-cs1714"></a>Erro do compilador CS1714
A classe base ou interface de TypeName1 não pôde ser resolvida ou é inválido  
  
 Você está implementando TypeName1 de uma classe ou interface que o não pôde ser resolvido, por exemplo, o compilador não pôde localizá-lo, ou é inválido. A solução é determinar qual desses dois casos é e especifique o local do tipo, mais corretamente ou corrigir erros de compilador na classe base.

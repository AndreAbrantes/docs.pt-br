# <a name="compiler-error-cs0170"></a><span data-ttu-id="086bc-102">CS0170 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="086bc-102">Compiler Error CS0170</span></span>
<span data-ttu-id="086bc-103">Uso de campo possivelmente não atribuído 'field'</span><span class="sxs-lookup"><span data-stu-id="086bc-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="086bc-104">Um campo em uma estrutura foi usado sem primeiro sendo inicializado.</span><span class="sxs-lookup"><span data-stu-id="086bc-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="086bc-105">Para resolver esse problema, primeiro determine qual campo foi inicializado e inicializá-lo antes de tentar acessá-lo.</span><span class="sxs-lookup"><span data-stu-id="086bc-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="086bc-106">Para obter mais informações sobre como inicializar estruturas, consulte [estruturas](../../csharp/programming-guide/classes-and-structs/structs.md) e [Structs usando](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="086bc-106">For more information about initializing structs, see [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) and [Using Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="086bc-107">O exemplo a seguir gera CS0170:</span><span class="sxs-lookup"><span data-stu-id="086bc-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```

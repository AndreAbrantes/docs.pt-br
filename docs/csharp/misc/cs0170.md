# <a name="compiler-error-cs0170"></a><span data-ttu-id="d04fa-103">Erro do Compilador CS0170</span><span class="sxs-lookup"><span data-stu-id="d04fa-103">Compiler Error CS0170</span></span>
<span data-ttu-id="d04fa-104">Uso do campo ' campo ' possivelmente não atribuído</span><span class="sxs-lookup"><span data-stu-id="d04fa-104">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="d04fa-105">Um campo em uma estrutura foi usado sem ser inicializado pela primeira vez.</span><span class="sxs-lookup"><span data-stu-id="d04fa-105">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="d04fa-106">Para resolver esse problema, primeiro determine qual campo não foi inicializado e, em seguida, inicialize-o antes de tentar acessá-lo.</span><span class="sxs-lookup"><span data-stu-id="d04fa-106">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="d04fa-107">Para obter mais informações sobre como inicializar structs, consulte [tipos de estrutura](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="d04fa-107">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="d04fa-108">O exemplo a seguir gera CS0170:</span><span class="sxs-lookup"><span data-stu-id="d04fa-108">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="f4363-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f4363-108">Example</span></span>  
 <span data-ttu-id="f4363-109">O exemplo a seguir gera CS0688.</span><span class="sxs-lookup"><span data-stu-id="f4363-109">The following sample generates CS0688.</span></span> <span data-ttu-id="f4363-110">Para resolver o aviso sem modificar a classe base, remova o atributo de segurança um método de substituição.</span><span class="sxs-lookup"><span data-stu-id="f4363-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="f4363-111">Isso não resolverá o problema de segurança.</span><span class="sxs-lookup"><span data-stu-id="f4363-111">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```

# <a name="compiler-error-cs0551"></a><span data-ttu-id="8dba6-102">CS0551 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="8dba6-102">Compiler Error CS0551</span></span>
<span data-ttu-id="8dba6-103">Implementação de interface explícita 'implementação' não encontrou o acessador 'acessador'</span><span class="sxs-lookup"><span data-stu-id="8dba6-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="8dba6-104">Uma classe que implementa explicitamente a propriedade da interface deve implementar acessadores que define a interface.</span><span class="sxs-lookup"><span data-stu-id="8dba6-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="8dba6-105">Para obter mais informações, consulte [Propriedades using](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="8dba6-105">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="8dba6-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8dba6-106">Example</span></span>  
 <span data-ttu-id="8dba6-107">O exemplo a seguir gera CS0551.</span><span class="sxs-lookup"><span data-stu-id="8dba6-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```

# <a name="compiler-error-cs0551"></a>Erro do Compilador CS0551
A implementação da implementação da interface explícita está faltando 'acessório' do acessório  
  
 Uma classe que implemente explicitamente a propriedade de uma interface deve implementar todos os acessórios que a interface define.  
  
 Para obter mais informações, consulte [Propriedades using](../programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Exemplo  
 A seguinte amostra gera CS0551.  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="62233-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="62233-106">Example</span></span>  
 <span data-ttu-id="62233-107">O exemplo a seguir gera CS0551.</span><span class="sxs-lookup"><span data-stu-id="62233-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```

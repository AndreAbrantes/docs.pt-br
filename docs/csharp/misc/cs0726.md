# <a name="compiler-error-cs0726"></a>CS0726 de erro do compilador
'o especificador de formato' não é um especificador de formato válido  
  
 Esse erro ocorre no depurador. Quando você digita um nome de variável em uma das janelas do depurador, você pode segui-lo com uma vírgula e um especificador de formato. Os exemplos são: `myInt, h` ou `myString,nq`. Esse erro surge quando o compilador não reconhece o [especificadores de formato em c#](/visualstudio/debugger/format-specifiers-in-csharp).

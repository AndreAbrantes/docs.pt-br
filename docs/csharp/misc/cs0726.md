# <a name="compiler-error-cs0726"></a>Erro do compilador CS0726
'especificador de formato ' não é um especificador de formato válido  
  
 Esse erro ocorre no depurador. Quando você digita um nome de variável em uma das janelas do depurador, siga-o com uma vírgula e, em seguida, um especificador de formato. Os exemplos são: `myInt, h` ou `myString,nq`. Esse erro ocorre quando o compilador não reconhece o [especificadores de formato em C# ](/visualstudio/debugger/format-specifiers-in-csharp).

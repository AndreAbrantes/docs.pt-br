# <a name="compiler-error-cs0726"></a>CS0726 de erro do compilador
'especificador de formato ' não é um especificador de formato válido  
  
 Esse erro ocorre no depurador. Quando você digita um nome de variável em uma das janelas do depurador, você pode segui-lo com uma vírgula e um especificador de formato. Os exemplos são: `myInt, h` ou `myString,nq`. Este erro ocorre quando o compilador não reconhece o [especificadores de formato em c#](https://docs.microsoft.com/visualstudio/debugger/format-specifiers-in-csharp).

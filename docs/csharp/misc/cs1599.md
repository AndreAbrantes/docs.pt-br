# <a name="compiler-error-cs1599"></a>Erro do compilador CS1599
Método ou delegate não pode retornar o tipo 'type'  
  
 Alguns tipos na biblioteca de classes do .NET Framework, por exemplo, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> e <xref:System.ArgIterator> não podem ser usados como tipos de retorno porque eles podem ser usados para executar operações não seguras.  
  
 O exemplo a seguir gera CS1599:  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```

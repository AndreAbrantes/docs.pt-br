# <a name="compiler-error-cs1565"></a>CS1565 de erro do compilador
Opções conflitantes especificadas: arquivo de recurso Win32; Ícone do Win32  
  
 Não é válido especificar tanto o [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opção de compilador e o [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) opção de compilador na mesma compilação.

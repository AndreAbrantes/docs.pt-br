# <a name="compiler-error-cs1565"></a>Erro do compilador CS1565
Opções conflitantes especificadas: Arquivo de recurso Win32; Ícone do Win32  
  
 Não é válido especificar ambos os [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opção de compilador e o [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) opção de compilador na mesma compilação.

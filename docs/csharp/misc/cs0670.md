 Um campo foi declarado para ser do tipo [void](../../csharp/language-reference/keywords/void.md).  
  
 O exemplo a seguir gera CS0670:  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```

# <a name="compiler-error-cs0754"></a><span data-ttu-id="43800-102">CS0754 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="43800-102">Compiler Error CS0754</span></span>
<span data-ttu-id="43800-103">Um método parcial não pode implementar explicitamente um método de interface.</span><span class="sxs-lookup"><span data-stu-id="43800-103">A partial method may not explicitly implement an interface method.</span></span>  
  
 <span data-ttu-id="43800-104">Um método parcial não pode ser declarado como uma implementação explícita de um método definido em uma interface.</span><span class="sxs-lookup"><span data-stu-id="43800-104">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="43800-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="43800-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="43800-106">Remova a qualificação de interface explícita de declaração de método.</span><span class="sxs-lookup"><span data-stu-id="43800-106">Remove the explicit interface qualification from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="43800-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="43800-107">Example</span></span>  
 <span data-ttu-id="43800-108">O código a seguir gera CS0754:</span><span class="sxs-lookup"><span data-stu-id="43800-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

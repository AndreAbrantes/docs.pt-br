## <a name="example"></a><span data-ttu-id="7c625-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7c625-107">Example</span></span>  
 <span data-ttu-id="7c625-108">O código a seguir gera CS0754:</span><span class="sxs-lookup"><span data-stu-id="7c625-108">The following code generates CS0754:</span></span>  
  
```  
// cs0754.cs

    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="example"></a><span data-ttu-id="e5495-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e5495-107">Example</span></span>  
 <span data-ttu-id="e5495-108">O código a seguir gera CS0754:</span><span class="sxs-lookup"><span data-stu-id="e5495-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

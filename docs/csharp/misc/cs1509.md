# <a name="compiler-error-cs1509"></a>Erro do compilador CS1509
O arquivo referenciado ' file ' não é um assembly; Use a opção '/addmodule ' em vez disso  
  
 Um arquivo de saída (arquivo de saída 1), produzido em uma compilação que usou [/target: module](../language-reference/compiler-options/target-module-compiler-option.md) (não tem um manifesto de assembly), foi especificado como [/Reference](../language-reference/compiler-options/reference-compiler-option.md). Portanto, em vez de acrescentar um assembly ao assembly para o programa atual, as informações de metadados no arquivo de saída 1 serão adicionadas ao assembly para o programa atual.

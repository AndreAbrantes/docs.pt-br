# <a name="compiler-error-cs1528"></a>Erro do compilador CS1528
Esperado; ou = (não é possível especificar argumentos de construtor na declaração)  
  
 Uma referência a uma classe foi formada como se um objeto para a classe estava sendo criado. Por exemplo, houve uma tentativa de passar uma variável para um construtor. Use o [novo](../../csharp/language-reference/keywords/new.md) operador para criar um objeto de uma classe.  
  
 O exemplo a seguir gera CS1528:  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```

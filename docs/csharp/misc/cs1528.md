<span data-ttu-id="64f0b-104">Vista or = (não é possível especificar argumentos de Construtor na declaração)</span><span class="sxs-lookup"><span data-stu-id="64f0b-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="64f0b-105">Uma referência a uma classe foi formada como se um objeto para a classe estivesse sendo criado.</span><span class="sxs-lookup"><span data-stu-id="64f0b-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="64f0b-106">Por exemplo, houve uma tentativa de passar uma variável para um construtor.</span><span class="sxs-lookup"><span data-stu-id="64f0b-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="64f0b-107">Use o operador [New](../language-reference/operators/new-operator.md) para criar um objeto de uma classe.</span><span class="sxs-lookup"><span data-stu-id="64f0b-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="64f0b-108">O exemplo a seguir gera CS1528:</span><span class="sxs-lookup"><span data-stu-id="64f0b-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```

# <a name="compiler-error-cs1004"></a>CS1004 de erro do compilador
Modificador 'Modificador' duplicado  
  
 Um modificador duplicado, como um **acesso** modificador, foi detectado.  
  
 O exemplo a seguir gera CS1004:  
  
```  
// CS1004.cs  
public class clx  
{  
   public public static void Main()   // CS1004, two public keywords  
   {  
   }  
}  
```

# <a name="compiler-error-cs0584"></a>Erro do Compilador CS0584

Erro de compilador interno: estágio ' Stage ' Symbol ' Symbol '
  
 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se esse não for o caso, tente [conversar conosco](/visualstudio/ide/feedback-options).

> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

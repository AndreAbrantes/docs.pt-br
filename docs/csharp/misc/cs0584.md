# <a name="compiler-error-cs0584"></a>CS0584 de erro do compilador
Erro do compilador interno: símbolo de "estágio" estágio 'symbol'  
  
 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se esse não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

## <a name="example"></a><span data-ttu-id="4c2e0-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4c2e0-108">Example</span></span>  
 <span data-ttu-id="4c2e0-109">O exemplo a seguir gera CS0131.</span><span class="sxs-lookup"><span data-stu-id="4c2e0-109">The following sample generates CS0131.</span></span>  
  
```  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="4c2e0-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4c2e0-110">Example</span></span>

{  
    public static int Main()  
    {

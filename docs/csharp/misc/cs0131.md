## <a name="example"></a><span data-ttu-id="41371-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="41371-108">Example</span></span>  
 <span data-ttu-id="41371-109">O exemplo a seguir gera CS0131.</span><span class="sxs-lookup"><span data-stu-id="41371-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```

    public static int Main()

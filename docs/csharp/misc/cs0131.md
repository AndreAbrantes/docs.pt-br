 <span data-ttu-id="2305f-105">Em uma instrução de atribuição, o valor do lado direito é atribuído ao lado esquerdo.</span><span class="sxs-lookup"><span data-stu-id="2305f-105">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="2305f-106">O lado esquerdo deve ser uma variável, uma propriedade ou um indexador.</span><span class="sxs-lookup"><span data-stu-id="2305f-106">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="2305f-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2305f-109">Example</span></span>  

 <span data-ttu-id="2305f-110">O exemplo a seguir gera CS0131.</span><span class="sxs-lookup"><span data-stu-id="2305f-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="2305f-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2305f-111">Example</span></span>

```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```

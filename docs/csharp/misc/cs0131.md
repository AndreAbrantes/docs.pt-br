 <span data-ttu-id="7507a-104">Em uma instrução de atribuição, o valor do lado direito é atribuído para o lado esquerdo.</span><span class="sxs-lookup"><span data-stu-id="7507a-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="7507a-105">O lado esquerdo deve ser uma variável, propriedade ou indexador.</span><span class="sxs-lookup"><span data-stu-id="7507a-105">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="7507a-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7507a-108">Example</span></span>  
 <span data-ttu-id="7507a-109">O exemplo a seguir gera CS0131.</span><span class="sxs-lookup"><span data-stu-id="7507a-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```

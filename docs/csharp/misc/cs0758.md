# <a name="compiler-error-cs0758"></a><span data-ttu-id="06678-103">Erro do Compilador CS0758</span><span class="sxs-lookup"><span data-stu-id="06678-103">Compiler Error CS0758</span></span>

## <a name="example"></a><span data-ttu-id="06678-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="06678-108">Example</span></span>

        }  
  
    }

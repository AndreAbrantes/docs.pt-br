# <a name="compiler-error-cs1676"></a><span data-ttu-id="3ef6d-102">CS1676 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="3ef6d-102">Compiler Error CS1676</span></span>
<span data-ttu-id="3ef6d-103">O parâmetro 'number' deve ser declarado com a palavra-chave 'palavra-chave'</span><span class="sxs-lookup"><span data-stu-id="3ef6d-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="3ef6d-104">Esse erro ocorre quando o modificador de parâmetro de tipo em um método anônimo é diferente daquele usado na declaração do delegado que está convertendo o método.</span><span class="sxs-lookup"><span data-stu-id="3ef6d-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="3ef6d-105">O exemplo a seguir gera CS1676:</span><span class="sxs-lookup"><span data-stu-id="3ef6d-105">The following sample generates CS1676:</span></span>  
  
```  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```

# <a name="compiler-error-cs1676"></a><span data-ttu-id="faf04-102">Erro do Compilador CS1676</span><span class="sxs-lookup"><span data-stu-id="faf04-102">Compiler Error CS1676</span></span>
<span data-ttu-id="faf04-103">O parâmetro 'número' deve ser declarado com a palavra-chave 'palavra-chave'</span><span class="sxs-lookup"><span data-stu-id="faf04-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="faf04-104">Este erro ocorre quando o modificador de tipo de parâmetro em um método anônimo é diferente do usado na declaração do delegado para o que você está lançando o método.</span><span class="sxs-lookup"><span data-stu-id="faf04-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="faf04-105">A seguinte amostra gera CS1676:</span><span class="sxs-lookup"><span data-stu-id="faf04-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```

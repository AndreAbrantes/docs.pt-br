# <a name="compiler-error-cs1002"></a>Erro do compilador CS1002
; vista  
  
 O compilador detectou um ponto-e-vírgula ausente. Um ponto-e-vírgula em obrigatório no final de C#cada instrução no. Uma instrução pode abranger mais de uma linha.  
  
 O exemplo a seguir gera CS1002:  
  
```csharp  
// CS1002.cs  
namespace x  
{  
   abstract public class clx  
   {  
      int i   // CS1002, missing semicolon  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

# <a name="compiler-error-cs0058"></a>Erro do compilador CS0058
Acessibilidade inconsistente: tipo de retorno 'type' é menos acessível que o delegado 'delegate'  
  
 Uma construção pública deve retornar um objeto acessível publicamente. Para obter mais informações, consulte [Modificadores de Acesso](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 O exemplo a seguir gera CS0058 porque nenhum modificador de acesso é aplicado para MyClass e, portanto, ele recebe acessibilidade privada por padrão:  
  
```csharp  
// CS0058.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public delegate MyClass MyClassDel();   // CS0058  
  
public class A  
{  
   public static void Main()  
   {  
   }  
}  
```

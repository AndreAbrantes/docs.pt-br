## <a name="example"></a><span data-ttu-id="eb44b-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="eb44b-108">Example</span></span>  
 <span data-ttu-id="eb44b-109">O exemplo a seguir gera CS1930 porque o `num` identificador é usado para a variável de intervalo `from` na cláusula e para a variável de `let` intervalo introduzida pela cláusula.</span><span class="sxs-lookup"><span data-stu-id="eb44b-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```

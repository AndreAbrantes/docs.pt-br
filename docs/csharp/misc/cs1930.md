<span data-ttu-id="df552-103">A variável de intervalo 'name' já foi declarada.</span><span class="sxs-lookup"><span data-stu-id="df552-103">The range variable 'name' has already been declared</span></span>

## <a name="example"></a><span data-ttu-id="df552-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="df552-108">Example</span></span>  
 <span data-ttu-id="df552-109">O exemplo a seguir gera CS1930 porque o identificador `num` é usado para a variável de intervalo na `from` cláusula e para a variável de intervalo introduzidos pelo `let` cláusula.</span><span class="sxs-lookup"><span data-stu-id="df552-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```

 <span data-ttu-id="f145f-104">Um tipo estático como um tipo de retorno não é significativo porque não não possível criar instâncias de tipos estáticos.</span><span class="sxs-lookup"><span data-stu-id="f145f-104">A static type as a return type is not meaningful since instances of static types cannot be created.</span></span>  
  
 <span data-ttu-id="f145f-105">O exemplo a seguir gera CS0722:</span><span class="sxs-lookup"><span data-stu-id="f145f-105">The following sample generates CS0722:</span></span>

```csharp  
{  
}

{  
   public SC F()  // CS0722

      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

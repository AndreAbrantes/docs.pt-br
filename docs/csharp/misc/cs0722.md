# <a name="compiler-error-cs0722"></a>Erro do compilador CS0722
'type': tipos static não podem ser usados como tipos de retorno  
  
 Um tipo estático como um tipo de retorno não é significativo, pois não não possível criar instâncias de tipos estáticos.  
  
 O exemplo a seguir gera CS0722:  
  
```csharp  
// CS0722.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public SC F()  // CS0722  
   {  
      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

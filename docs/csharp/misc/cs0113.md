# <a name="compiler-error-cs0113"></a><span data-ttu-id="ca9ca-103">Erro do Compilador CS0113</span><span class="sxs-lookup"><span data-stu-id="ca9ca-103">Compiler Error CS0113</span></span>

<span data-ttu-id="ca9ca-104">Um membro ' function ' marcado como override não pode ser marcado como New ou virtual</span><span class="sxs-lookup"><span data-stu-id="ca9ca-104">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="ca9ca-105">É mutuamente exclusivo marcar um método com as palavras-chave [New](../language-reference/keywords/new-modifier.md) e [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="ca9ca-105">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="ca9ca-106">O exemplo a seguir gera CS0113:</span><span class="sxs-lookup"><span data-stu-id="ca9ca-106">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

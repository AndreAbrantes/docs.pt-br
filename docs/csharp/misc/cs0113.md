# <a name="compiler-error-cs0113"></a><span data-ttu-id="73b73-102">CS0113 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="73b73-102">Compiler Error CS0113</span></span>
<span data-ttu-id="73b73-103">Um membro 'function' marcado como override não pode ser marcado como new ou virtual</span><span class="sxs-lookup"><span data-stu-id="73b73-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="73b73-104">É mutuamente exclusiva para marcar um método com o [novo](../../csharp/language-reference/keywords/new.md) e [substituir](../../csharp/language-reference/keywords/override.md) palavras-chave.</span><span class="sxs-lookup"><span data-stu-id="73b73-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="73b73-105">O exemplo a seguir gera CS0113:</span><span class="sxs-lookup"><span data-stu-id="73b73-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

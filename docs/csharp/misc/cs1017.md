# <a name="compiler-error-cs1017"></a>Erro do compilador CS1017
Cláusulas catch não podem seguir a cláusula catch geral de uma instrução try  
  
 Um `catch` bloco que não leve nenhum parâmetro deve ser o último em uma série de `catch` blocos. Para obter mais informações sobre exceções, consulte [instruções de tratamento de exceção](../../csharp/language-reference/keywords/exception-handling-statements.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1017:  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```

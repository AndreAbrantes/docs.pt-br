# <a name="compiler-error-cs1563"></a>CS1563 de erro do compilador
A saída 'arquivo de saída' não tem nenhum arquivo de origem  
  
 O [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) opção de compilador foi especificada, mas não siga nenhum arquivo de código fonte. Você deve usar não **/out**, ou você deve especificar os arquivos de código a seguir **/out**.

# <a name="compiler-error-cs1563"></a>Erro do compilador CS1563
A saída 'arquivo de saída' não tem quaisquer arquivos de origem  
  
 O [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) opção de compilador foi especificada, mas não há arquivos de código-fonte siga. Você deve usar não **/out**, ou você deve especificar os arquivos de código a seguir **/out**.

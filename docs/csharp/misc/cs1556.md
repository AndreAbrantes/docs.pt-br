# <a name="compiler-error-cs1556"></a>Erro do compilador CS1556
' Construct ' especificado para o método Main deve ser uma classe ou struct válida  
  
 A opção de compilador [/Main](../language-reference/compiler-options/main-compiler-option.md) passou um identificador que não era um nome de classe.

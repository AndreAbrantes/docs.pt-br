# <a name="compiler-error-cs1556"></a>CS1556 de erro do compilador
'Criar' especificado para método Main deve ser uma classe ou estrutura válida  
  
 O [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador foi passada um identificador que não é um nome de classe.

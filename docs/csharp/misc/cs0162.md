# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="6fac5-102">Compilador CS0162 de aviso (nível 2)</span><span class="sxs-lookup"><span data-stu-id="6fac5-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="6fac5-103">Código inacessível detectado</span><span class="sxs-lookup"><span data-stu-id="6fac5-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="6fac5-104">O compilador detectou código nunca será executado.</span><span class="sxs-lookup"><span data-stu-id="6fac5-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="6fac5-105">O exemplo a seguir gera CS0162:</span><span class="sxs-lookup"><span data-stu-id="6fac5-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```

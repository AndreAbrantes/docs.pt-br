# <a name="compiler-warning-level-2-cs0162"></a>Compilador aviso (nível 2) CS0162
Código inacessível detectado  
  
 O compilador detectou o código que nunca será executado.  
  
 O exemplo a seguir gera CS0162:  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```

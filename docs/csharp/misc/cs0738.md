## <a name="example"></a><span data-ttu-id="59f46-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="59f46-108">Example</span></span>  
 <span data-ttu-id="59f46-109">O código a seguir gera CS0738 porque o método da classe retorna `void` e retorna o membro de interface de mesmo nome `int`:</span><span class="sxs-lookup"><span data-stu-id="59f46-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```  
{  
    int TestMethod();  
}

{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();

```

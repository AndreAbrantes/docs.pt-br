<span data-ttu-id="85e13-103">' Type Name ' não implementa o membro de interface ' member Name '.</span><span class="sxs-lookup"><span data-stu-id="85e13-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="85e13-104">' Method Name ' não pode implementar ' interface member ' porque não tem o tipo de retorno correspondente de ' Type Name '.</span><span class="sxs-lookup"><span data-stu-id="85e13-104">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>

## <a name="example"></a><span data-ttu-id="85e13-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="85e13-108">Example</span></span>  
 <span data-ttu-id="85e13-109">O código a seguir gera CS0738 porque o método de `void` classe retorna e o membro de interface do mesmo `int`nome retorna:</span><span class="sxs-lookup"><span data-stu-id="85e13-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```

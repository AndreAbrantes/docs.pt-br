# <a name="compiler-error-cs0738"></a><span data-ttu-id="28670-103">Erro do Compilador CS0738</span><span class="sxs-lookup"><span data-stu-id="28670-103">Compiler Error CS0738</span></span>

<span data-ttu-id="28670-104">' Type Name ' não implementa o membro de interface ' member Name '.</span><span class="sxs-lookup"><span data-stu-id="28670-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="28670-105">' Method Name ' não pode implementar ' interface member ' porque não tem o tipo de retorno correspondente de ' Type Name '.</span><span class="sxs-lookup"><span data-stu-id="28670-105">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>  
  
 <span data-ttu-id="28670-106">O valor de retorno de um método de implementação em uma classe deve corresponder ao valor de retorno do membro da interface que ele implementa.</span><span class="sxs-lookup"><span data-stu-id="28670-106">The return value of an implementing method in a class must match the return value of the interface member that it implements.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="28670-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="28670-107">To correct this error</span></span>  
  
1. <span data-ttu-id="28670-108">Altere o tipo de retorno do método para corresponder ao membro da interface.</span><span class="sxs-lookup"><span data-stu-id="28670-108">Change the return type of the method to match that of the interface member.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28670-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="28670-109">Example</span></span>  

 <span data-ttu-id="28670-110">O código a seguir gera CS0738 porque o método de classe retorna `void` e o membro de interface do mesmo nome retorna `int` :</span><span class="sxs-lookup"><span data-stu-id="28670-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```

# <a name="compiler-error-cs1662"></a><span data-ttu-id="f0269-102">Erro do compilador CS1662</span><span class="sxs-lookup"><span data-stu-id="f0269-102">Compiler Error CS1662</span></span>

<span data-ttu-id="f0269-103">Não é possível converter bloco de método anônimo delegate tipo 'tipo de delegado' porque alguns dos tipos de retorno no bloco não são implicitamente conversíveis para o tipo de retorno do delegado</span><span class="sxs-lookup"><span data-stu-id="f0269-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="f0269-104">Esse erro ocorre se a instrução de retorno do bloco de método anônimo tinha um tipo que não foi implicitamente conversível para o tipo de retorno do delegado.</span><span class="sxs-lookup"><span data-stu-id="f0269-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="f0269-105">O exemplo a seguir gera CS1662:</span><span class="sxs-lookup"><span data-stu-id="f0269-105">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```

# <a name="compiler-error-cs1662"></a><span data-ttu-id="8f32c-102">CS1662 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="8f32c-102">Compiler Error CS1662</span></span>

 <span data-ttu-id="8f32c-104">Esse erro ocorre se a instrução de retorno do bloco de métodos anônimo tinha um tipo que não é implicitamente conversível para o tipo de retorno do delegado.</span><span class="sxs-lookup"><span data-stu-id="8f32c-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

class C  
{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662

     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}

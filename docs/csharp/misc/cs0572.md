<span data-ttu-id="27979-103">'type': não é possível fazer referência a um tipo por meio de uma expressão. Tente 'path_to_type' em vez disso</span><span class="sxs-lookup"><span data-stu-id="27979-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="27979-104">Foi feita uma tentativa para acessar um membro de uma classe por meio de um identificador, que não é permitido nessa situação.</span><span class="sxs-lookup"><span data-stu-id="27979-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="27979-105">O exemplo a seguir gera CS0572:</span><span class="sxs-lookup"><span data-stu-id="27979-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```

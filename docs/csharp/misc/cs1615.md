# <a name="compiler-error-cs1615"></a><span data-ttu-id="6b197-102">CS1615 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="6b197-102">Compiler Error CS1615</span></span>
<span data-ttu-id="6b197-103">'Número de argumento' não deve ser transmitido com a palavra-chave 'palavra-chave'</span><span class="sxs-lookup"><span data-stu-id="6b197-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="6b197-104">Uma das palavras-chave `ref` ou **out** foi usado quando a função não continha um `ref` ou **out** parâmetro para o argumento.</span><span class="sxs-lookup"><span data-stu-id="6b197-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="6b197-105">Para resolver esse erro, remova a palavra-chave incorreta e use a palavra-chave apropriada que corresponda a declaração da função, se houver.</span><span class="sxs-lookup"><span data-stu-id="6b197-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="6b197-106">O exemplo a seguir gera CS1615:</span><span class="sxs-lookup"><span data-stu-id="6b197-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```

# <a name="compiler-error-cs1621"></a><span data-ttu-id="254d0-102">Erro do compilador CS1621</span><span class="sxs-lookup"><span data-stu-id="254d0-102">Compiler Error CS1621</span></span>
<span data-ttu-id="254d0-103">A instrução yield não pode ser usada dentro de um método anônimo ou expressão lambda</span><span class="sxs-lookup"><span data-stu-id="254d0-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="254d0-104">Você não pode usar a instrução [yield](../language-reference/keywords/yield.md) em um [método anônimo](../language-reference/operators/delegate-operator.md) ou uma [expressão lambda](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="254d0-104">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="254d0-105">O exemplo a seguir gera CS1621:</span><span class="sxs-lookup"><span data-stu-id="254d0-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

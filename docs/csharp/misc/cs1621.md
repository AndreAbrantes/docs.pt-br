# <a name="compiler-error-cs1621"></a><span data-ttu-id="9899e-102">CS1621 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="9899e-102">Compiler Error CS1621</span></span>
<span data-ttu-id="9899e-103">A instrução yield não pode ser usada dentro de um método anônimo ou expressão lambda</span><span class="sxs-lookup"><span data-stu-id="9899e-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="9899e-104">A instrução yield não pode estar em um bloco de métodos anônimo em um iterador.</span><span class="sxs-lookup"><span data-stu-id="9899e-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9899e-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9899e-105">Example</span></span>  
 <span data-ttu-id="9899e-106">O exemplo a seguir gera CS1621:</span><span class="sxs-lookup"><span data-stu-id="9899e-106">The following sample generates CS1621:</span></span>  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

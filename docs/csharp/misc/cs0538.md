# <a name="compiler-error-cs0538"></a><span data-ttu-id="1d3c2-103">Erro do Compilador CS0538</span><span class="sxs-lookup"><span data-stu-id="1d3c2-103">Compiler Error CS0538</span></span>

<span data-ttu-id="1d3c2-104">' name ' na declaração de interface explícita não é uma interface</span><span class="sxs-lookup"><span data-stu-id="1d3c2-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="1d3c2-105">Foi feita uma tentativa de declarar explicitamente uma [interface](../language-reference/keywords/interface.md), mas uma interface não foi especificada.</span><span class="sxs-lookup"><span data-stu-id="1d3c2-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="1d3c2-106">O exemplo a seguir gera CS0538:</span><span class="sxs-lookup"><span data-stu-id="1d3c2-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```

# <a name="compiler-error-cs0538"></a><span data-ttu-id="be312-102">Erro do compilador CS0538</span><span class="sxs-lookup"><span data-stu-id="be312-102">Compiler Error CS0538</span></span>
<span data-ttu-id="be312-103">' name ' na declaração de interface explícita não é uma interface</span><span class="sxs-lookup"><span data-stu-id="be312-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="be312-104">Foi feita uma tentativa de declarar explicitamente uma [interface](../language-reference/keywords/interface.md), mas uma interface não foi especificada.</span><span class="sxs-lookup"><span data-stu-id="be312-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="be312-105">O exemplo a seguir gera CS0538:</span><span class="sxs-lookup"><span data-stu-id="be312-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```

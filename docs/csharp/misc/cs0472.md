 <span data-ttu-id="d1ddc-107">O exemplo a seguir gera CS0472.</span><span class="sxs-lookup"><span data-stu-id="d1ddc-107">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```

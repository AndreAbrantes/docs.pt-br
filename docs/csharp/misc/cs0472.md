## <a name="example"></a><span data-ttu-id="f82bf-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f82bf-105">Example</span></span>  
 <span data-ttu-id="f82bf-106">O exemplo a seguir gera CS0472.</span><span class="sxs-lookup"><span data-stu-id="f82bf-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```

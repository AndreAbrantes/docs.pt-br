## <a name="example"></a><span data-ttu-id="02632-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="02632-105">Example</span></span>

```  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```

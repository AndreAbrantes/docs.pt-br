# <a name="compiler-error-cs1902"></a>Erro do compilador CS1902
Opção inválida ' Option ' para/Debug; deve ser Full ou pdbonly  
  
 Uma opção inválida foi passada para a opção do compilador [/debug](../language-reference/compiler-options/debug-compiler-option.md) .  
  
 O exemplo a seguir gera CS1902:  
  
```csharp  
// CS1902.cs  
// compile with: /debug:x  
// CS1902 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```

# <a name="compiler-error-cs1566"></a>Erro do compilador CS1566
Erro ao ler o recurso de arquivo 'arquivo' — 'reason'  
  
 O compilador tinha problemas com o nome do arquivo passado para o [/resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md) opção de compilador.

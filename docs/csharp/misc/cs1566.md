# <a name="compiler-error-cs1566"></a>CS1566 de erro do compilador
Erro ao ler o recurso 'arquivo', 'razão'  
  
 O compilador teve problemas com o nome do arquivo passado para o [/resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md) opção de compilador.

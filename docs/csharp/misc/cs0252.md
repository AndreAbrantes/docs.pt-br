 <span data-ttu-id="7c506-104">O compilador está fazendo uma comparação de referência.</span><span class="sxs-lookup"><span data-stu-id="7c506-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="7c506-105">Se você quiser comparar o valor de cadeias de caracteres, converta o lado esquerdo da expressão `type`.</span><span class="sxs-lookup"><span data-stu-id="7c506-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="7c506-106">O exemplo a seguir gera CS0252:</span><span class="sxs-lookup"><span data-stu-id="7c506-106">The following sample generates CS0252:</span></span>  
  
```  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```

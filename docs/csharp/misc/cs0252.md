 <span data-ttu-id="4800f-106">O exemplo a seguir gera CS0252:</span><span class="sxs-lookup"><span data-stu-id="4800f-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs

using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;

      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```

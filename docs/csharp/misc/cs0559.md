## <a name="example"></a><span data-ttu-id="9d1e8-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="9d1e8-108">Example</span></span>  
 <span data-ttu-id="9d1e8-109">O exemplo a seguir gera CS0559.</span><span class="sxs-lookup"><span data-stu-id="9d1e8-109">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559

{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```

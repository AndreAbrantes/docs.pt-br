# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="229c0-102">Compilador CS0419 de aviso (nível 3)</span><span class="sxs-lookup"><span data-stu-id="229c0-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="229c0-103">Referência ambígua no atributo cref: 'Método Name1'.</span><span class="sxs-lookup"><span data-stu-id="229c0-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="229c0-104">Supondo que o 'Método Name2', mas também poderia ter correspondido outras sobrecargas, incluindo 'Método nome3'.</span><span class="sxs-lookup"><span data-stu-id="229c0-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="229c0-105">Em um comentário de documentação XML no código, uma referência não pôde ser resolvida.</span><span class="sxs-lookup"><span data-stu-id="229c0-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="229c0-106">Isso pode ocorrer se o método estiver sobrecarregado ou se forem encontrados dois identificadores diferentes com o mesmo nome.</span><span class="sxs-lookup"><span data-stu-id="229c0-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="229c0-107">Para resolver o aviso, use um nome qualificado para resolver a ambiguidade de referência ou incluir a sobrecarga específica entre parênteses.</span><span class="sxs-lookup"><span data-stu-id="229c0-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="229c0-108">O exemplo a seguir gera CS0419.</span><span class="sxs-lookup"><span data-stu-id="229c0-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```

<span data-ttu-id="1c69b-103">Referência ambígua no atributo cref: 'Método Nome1'.</span><span class="sxs-lookup"><span data-stu-id="1c69b-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="1c69b-104">Supondo que o 'Método nome2', mas também poderia ter correspondido outras sobrecargas, incluindo o 'Método nome3'.</span><span class="sxs-lookup"><span data-stu-id="1c69b-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="1c69b-105">Em um comentário de documentação XML no código, uma referência não pôde ser resolvida.</span><span class="sxs-lookup"><span data-stu-id="1c69b-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="1c69b-106">Isso pode ocorrer se o método está sobrecarregado ou se encontram-se dois identificadores diferentes com o mesmo nome.</span><span class="sxs-lookup"><span data-stu-id="1c69b-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="1c69b-107">Para resolver o aviso, use um nome qualificado para resolver a ambiguidade de referência ou incluir a sobrecarga específica entre parênteses.</span><span class="sxs-lookup"><span data-stu-id="1c69b-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>

```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```

# <a name="compiler-error-cs0757"></a><span data-ttu-id="92878-102">CS0757 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="92878-102">Compiler Error CS0757</span></span>

 <span data-ttu-id="92878-104">Um método parcial consiste em exatamente uma declaração de definição (assinatura) e uma ou zero (corpo) do declarações de implementação.</span><span class="sxs-lookup"><span data-stu-id="92878-104">A partial method consists of exactly one defining declaration (signature) and one or zero implementing declarations (body).</span></span> <span data-ttu-id="92878-105">Não são permitidas várias declarações de implementação para as mesmo declarações de definição idênticas.</span><span class="sxs-lookup"><span data-stu-id="92878-105">Multiple implementing declarations for the same identical defining declarations are not allowed.</span></span> <span data-ttu-id="92878-106">Métodos parciais podem ser sobrecarregados, e cada versão sobrecarregada pode ter uma ou zero declarações de implementação.</span><span class="sxs-lookup"><span data-stu-id="92878-106">Partial methods may be overloaded, and each overloaded version may have one or zero implementing declarations.</span></span>

## <a name="example"></a><span data-ttu-id="92878-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="92878-109">Example</span></span>  
 <span data-ttu-id="92878-110">O exemplo a seguir gera CS0757:</span><span class="sxs-lookup"><span data-stu-id="92878-110">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="example"></a><span data-ttu-id="a6b4c-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a6b4c-111">Example</span></span>  
 <span data-ttu-id="a6b4c-112">O código a seguir gera CS0736 porque `Program.testMethod` é declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="a6b4c-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```

## <a name="example"></a><span data-ttu-id="7c179-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7c179-110">Example</span></span>  
 <span data-ttu-id="7c179-111">O código a seguir gera CS0736 porque `Program.testMethod` está declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="7c179-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
{     
  
    interface ITest  
    {  
        int testMethod(int x);

    }      
}  
```

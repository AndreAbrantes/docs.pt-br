# <a name="compiler-error-cs0736"></a><span data-ttu-id="4152d-103">Erro do Compilador CS0736</span><span class="sxs-lookup"><span data-stu-id="4152d-103">Compiler Error CS0736</span></span>

- <span data-ttu-id="4152d-110">Redefina o tipo recipiente para que ele não herde da interface.</span><span class="sxs-lookup"><span data-stu-id="4152d-110">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4152d-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4152d-111">Example</span></span>  

 <span data-ttu-id="4152d-112">O código a seguir gera CS0736 porque `Program.testMethod` é declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="4152d-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```

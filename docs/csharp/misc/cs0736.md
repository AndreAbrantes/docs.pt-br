## <a name="example"></a><span data-ttu-id="0767d-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="0767d-110">Example</span></span>  
 <span data-ttu-id="0767d-111">O código a seguir gera CS0736 porque `Program.testMethod` é declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="0767d-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }

    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }

    }      
}  
```

# <a name="compiler-error-cs0736"></a><span data-ttu-id="52805-102">Erro do compilador CS0736</span><span class="sxs-lookup"><span data-stu-id="52805-102">Compiler Error CS0736</span></span>
<span data-ttu-id="52805-103">' Type Name ' não implementa o membro de interface ' member Name '.</span><span class="sxs-lookup"><span data-stu-id="52805-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="52805-104">' Method Name ' não pode implementar um membro de interface porque ele é estático.</span><span class="sxs-lookup"><span data-stu-id="52805-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="52805-105">Esse erro é gerado quando um método estático é implicitamente ou explicitamente declarado como uma implementação de um membro de interface.</span><span class="sxs-lookup"><span data-stu-id="52805-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="52805-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="52805-106">To correct this error</span></span>  
  
- <span data-ttu-id="52805-107">Remova o modificador [estático](../language-reference/keywords/static.md) da declaração do método.</span><span class="sxs-lookup"><span data-stu-id="52805-107">Remove the [static](../language-reference/keywords/static.md) modifier from the method declaration.</span></span>  
  
- <span data-ttu-id="52805-108">Altere o nome do método de interface.</span><span class="sxs-lookup"><span data-stu-id="52805-108">Change the name of the interface method.</span></span>  
  
- <span data-ttu-id="52805-109">Redefina o tipo recipiente para que ele não herde da interface.</span><span class="sxs-lookup"><span data-stu-id="52805-109">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="52805-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="52805-110">Example</span></span>  
 <span data-ttu-id="52805-111">O código a seguir gera CS0736 `Program.testMethod` porque é declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="52805-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```

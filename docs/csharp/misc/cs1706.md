## <a name="example"></a><span data-ttu-id="7486b-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7486b-107">Example</span></span>  
 <span data-ttu-id="7486b-108">O exemplo a seguir gera CS1706.</span><span class="sxs-lookup"><span data-stu-id="7486b-108">The following example generates CS1706.</span></span>

```csharp  
{  
    public MyAttribute(MyDelegate d) { }

{  
}  
```

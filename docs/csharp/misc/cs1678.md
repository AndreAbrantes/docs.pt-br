# <a name="compiler-error-cs1678"></a><span data-ttu-id="da969-102">CS1678 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="da969-102">Compiler Error CS1678</span></span>
<span data-ttu-id="da969-103">Parâmetro 'número' é declarado como tipo 'type1' mas deveria ser 'type2'</span><span class="sxs-lookup"><span data-stu-id="da969-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="da969-104">Esse erro ocorre quando o tipo de parâmetro em um método anônimo é diferente da declaração do delegado que está convertendo o método.</span><span class="sxs-lookup"><span data-stu-id="da969-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="da969-105">O exemplo a seguir gera CS1678:</span><span class="sxs-lookup"><span data-stu-id="da969-105">The following sample generates CS1678:</span></span>  
  
```  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```

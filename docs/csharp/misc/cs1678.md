# <a name="compiler-error-cs1678"></a>Erro do compilador CS1678
Parâmetro 'número' é declarado como tipo 'type1', mas deve ser 'type2'  
  
 Esse erro ocorre quando o tipo de parâmetro em um método anônimo é diferente da declaração de delegado que você está convertendo o método.  
  
 O exemplo a seguir gera CS1678:  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```

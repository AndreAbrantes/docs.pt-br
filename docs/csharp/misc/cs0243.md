# <a name="compiler-error-cs0243"></a><span data-ttu-id="04f80-102">Erro do compilador CS0243</span><span class="sxs-lookup"><span data-stu-id="04f80-102">Compiler Error CS0243</span></span>

<span data-ttu-id="04f80-103">O atributo Conditional não é válido em 'method' porque ele é um método de substituição</span><span class="sxs-lookup"><span data-stu-id="04f80-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="04f80-104">O [condicional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) atributo não é permitido em um método que é marcado com o [substituir](../../csharp/language-reference/keywords/override.md) palavra-chave.</span><span class="sxs-lookup"><span data-stu-id="04f80-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="04f80-105">Para obter mais informações, consulte [sabendo quando usar Override e novas palavras-chave](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="04f80-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="04f80-106">O compilador nunca se associa ao substituir métodos; ele se vincula apenas para o método base e o common language runtime chama a substituição, conforme apropriado.</span><span class="sxs-lookup"><span data-stu-id="04f80-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="04f80-107">O exemplo a seguir gera CS0243:</span><span class="sxs-lookup"><span data-stu-id="04f80-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```

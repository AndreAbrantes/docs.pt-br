# <a name="compiler-error-cs0243"></a><span data-ttu-id="64171-102">Erro do compilador CS0243</span><span class="sxs-lookup"><span data-stu-id="64171-102">Compiler error CS0243</span></span>

<span data-ttu-id="64171-103">O atributo Condicional não é válido em 'método' porque é um método de substituição</span><span class="sxs-lookup"><span data-stu-id="64171-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="64171-104">O <xref:System.Diagnostics.ConditionalAttribute> atributo não é permitido em um método marcado com a palavra-chave [de substituição.](../language-reference/keywords/override.md)</span><span class="sxs-lookup"><span data-stu-id="64171-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="64171-105">Para obter mais informações, consulte [Saber quando usar o Override e novas palavras-chave](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="64171-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="64171-106">O compilador nunca se liga aos métodos de substituição.</span><span class="sxs-lookup"><span data-stu-id="64171-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="64171-107">Ele só se liga ao método base, e o tempo de execução da linguagem comum chama a substituição, conforme apropriado.</span><span class="sxs-lookup"><span data-stu-id="64171-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="64171-108">O código a seguir gera CS0243:</span><span class="sxs-lookup"><span data-stu-id="64171-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```

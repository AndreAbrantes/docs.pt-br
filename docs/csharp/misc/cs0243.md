# <a name="compiler-error-cs0243"></a><span data-ttu-id="396a4-102">Erro do compilador CS0243</span><span class="sxs-lookup"><span data-stu-id="396a4-102">Compiler Error CS0243</span></span>

<span data-ttu-id="396a4-103">O atributo Conditional não é válido em ' method ' porque é um método override</span><span class="sxs-lookup"><span data-stu-id="396a4-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="396a4-104">O atributo [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) não é permitido em um método marcado com a palavra-chave [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="396a4-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="396a4-105">Para obter mais informações, consulte [sabendo quando usar substituições e novas palavras-chave](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="396a4-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="396a4-106">O compilador nunca é associado a métodos de substituição; Ele só liga ao método base e o Common Language Runtime chama a substituição, conforme apropriado.</span><span class="sxs-lookup"><span data-stu-id="396a4-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="396a4-107">O exemplo a seguir gera CS0243:</span><span class="sxs-lookup"><span data-stu-id="396a4-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```

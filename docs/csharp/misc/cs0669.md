# <a name="compiler-error-cs0669"></a><span data-ttu-id="59ddf-102">CS0669 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="59ddf-102">Compiler Error CS0669</span></span>

<span data-ttu-id="59ddf-103">Uma classe com o atributo ComImport não pode ter um construtor definido pelo usuário.</span><span class="sxs-lookup"><span data-stu-id="59ddf-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="59ddf-104">A camada de interoperabilidade COM no common language runtime fornece o construtor para [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span><span class="sxs-lookup"><span data-stu-id="59ddf-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="59ddf-105">Consequentemente, um objeto COM pode ser usado como um objeto gerenciado no tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="59ddf-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="59ddf-106">O exemplo a seguir gera CS0669:</span><span class="sxs-lookup"><span data-stu-id="59ddf-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```

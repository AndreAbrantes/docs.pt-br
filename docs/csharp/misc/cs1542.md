# <a name="compiler-error-cs1542"></a>Erro do compilador CS1542
'dll' não pode ser adicionado a este assembly porque já é um assembly. Use ' / R' em vez disso, a opção  
  
 O arquivo que foi referenciado com o [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md) opção de compilador não foi criada com [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md); use [/Reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) para referenciar o arquivo nessa compilação.

# <a name="compiler-error-cs2011"></a>CS2011 de erro do compilador
Não é possível abrir arquivo 'arquivo' de resposta  
  
 Foi especificado um arquivo de resposta em uma compilação, mas o compilador não foi possível localizar e abrir o arquivo.  
  
 Para obter mais informações sobre arquivos de resposta, consulte [@ (especificar arquivo de resposta)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).

# <a name="compiler-error-cs0534"></a><span data-ttu-id="a201f-103">Erro do Compilador CS0534</span><span class="sxs-lookup"><span data-stu-id="a201f-103">Compiler Error CS0534</span></span>

<span data-ttu-id="a201f-104">' function1 ' não implementa o membro abstrato herdado ' function2 '</span><span class="sxs-lookup"><span data-stu-id="a201f-104">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="a201f-105">Uma classe é necessária para implementar todos os membros [abstratos](../language-reference/keywords/abstract.md) na classe base, a menos que a classe também seja abstrata.</span><span class="sxs-lookup"><span data-stu-id="a201f-105">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="a201f-106">O exemplo a seguir gera CS0534:</span><span class="sxs-lookup"><span data-stu-id="a201f-106">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

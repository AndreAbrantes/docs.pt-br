# <a name="compiler-error-cs0534"></a><span data-ttu-id="bb91c-102">CS0534 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="bb91c-102">Compiler Error CS0534</span></span>
<span data-ttu-id="bb91c-103">'function1' não implementa membro abstrato herdado 'function2'</span><span class="sxs-lookup"><span data-stu-id="bb91c-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="bb91c-104">Uma classe é necessária para implementar todo o [abstrata](../../csharp/language-reference/keywords/abstract.md) os membros da classe base, a menos que a classe é também abstrata.</span><span class="sxs-lookup"><span data-stu-id="bb91c-104">A class is required to implement all the [abstract](../../csharp/language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="bb91c-105">O exemplo a seguir gera CS0534:</span><span class="sxs-lookup"><span data-stu-id="bb91c-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```

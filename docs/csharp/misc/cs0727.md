# <a name="compiler-error-cs0727"></a>CS0727 de erro do compilador
Especificador de formato inválido  
  
 Esse erro ocorre no depurador. Quando você digita um nome de variável em uma das janelas do depurador, você pode segui-lo com uma vírgula e um especificador de formato. Os exemplos são: myInt, h; ou myString, nq. Este erro ocorre quando o compilador é completamente não é possível analisar o que você digitou. Para resolver esse erro, digite novamente o nome da variável, opcionalmente seguido por uma vírgula e um [especificador de formato válido](/visualstudio/debugger/format-specifiers-in-csharp).

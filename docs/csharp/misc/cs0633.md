## <a name="example"></a><span data-ttu-id="b169c-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="b169c-109">Example</span></span>

```  
#define DEBUG  
using System.Runtime.CompilerServices;

{  
   [IndexerName("Invalid+Identifier")]   // CS0633

   {   
      get { return i; }   
   }  
}  
```

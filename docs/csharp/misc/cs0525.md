# <a name="compiler-error-cs0525"></a>Erro do compilador CS0525
Interfaces não podem conter campos  
  
 Uma [interface](../../csharp/language-reference/keywords/interface.md) pode conter métodos e propriedades, mas não de campos.  
  
 O exemplo a seguir gera CS0525:  
  
```csharp  
// CS0525.cs  
namespace x  
{  
   public interface clx  
   {  
      public int i;   // CS0525  
   }  
}  
```

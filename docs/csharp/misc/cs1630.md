# <a name="compiler-error-cs1630"></a>Erro do compilador CS1630
Opção inválida 'option' para /errorreport; deve ser prompt, send, fila ou none  
  
 A opção de linha de comando [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) deve ser seguido por **prompt**, **enviar**, **fila**, ou **none**, especificando a ação que você deseja feito quando ocorre um erro interno do compilador.

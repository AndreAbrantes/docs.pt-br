# <a name="compiler-error-cs1630"></a>Erro do compilador CS1630
Opção inválida ' Option ' para/errorreport; deve ser prompt, enviar, enfileirar ou nenhum  
  
 A opção de linha de comando [/errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) deve ser seguida por **prompt**, **Send**, **Queue**ou **None**, especificando a ação que você deseja tomar quando ocorre um erro interno do compilador.

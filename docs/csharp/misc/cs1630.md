# <a name="compiler-error-cs1630"></a>CS1630 de erro do compilador
Opção inválida 'option' para /errorreport; deve ser prompt, send, fila ou none  
  
 A opção de linha de comando [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) deve ser seguido por **prompt**, **enviar**, **fila**, ou **nenhum**, especificar a ação que você deseja obtido quando ocorre um erro interno do compilador.

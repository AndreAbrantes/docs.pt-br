## <a name="example"></a><span data-ttu-id="5d5d7-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5d5d7-108">Example</span></span>  
 <span data-ttu-id="5d5d7-109">O exemplo a seguir gera CS0315:</span><span class="sxs-lookup"><span data-stu-id="5d5d7-109">The following example generates CS0315:</span></span>

```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}

{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```

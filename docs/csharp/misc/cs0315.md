## <a name="example"></a><span data-ttu-id="f6df4-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f6df4-109">Example</span></span>  

 <span data-ttu-id="f6df4-110">O exemplo a seguir gera CS0315:</span><span class="sxs-lookup"><span data-stu-id="f6df4-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0315"></a><span data-ttu-id="ce044-103">Erro do Compilador CS0315</span><span class="sxs-lookup"><span data-stu-id="ce044-103">Compiler Error CS0315</span></span>
<span data-ttu-id="ce044-104">O tipo ' valueType ' não pode ser usado como parâmetro de tipo ' T' no tipo genérico ou no método ' TypeorMethod \<T> '.</span><span class="sxs-lookup"><span data-stu-id="ce044-104">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="ce044-105">Não há conversão de Boxing de ' valueType ' em ' ReferenceType '.</span><span class="sxs-lookup"><span data-stu-id="ce044-105">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="ce044-106">Esse erro ocorre quando você restringi um tipo genérico a uma classe específica e tenta construir uma instância dessa classe usando um tipo de valor que não pode ser digitado implicitamente.</span><span class="sxs-lookup"><span data-stu-id="ce044-106">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ce044-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="ce044-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="ce044-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ce044-109">Example</span></span>  
 <span data-ttu-id="ce044-110">O exemplo a seguir gera CS0315:</span><span class="sxs-lookup"><span data-stu-id="ce044-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0315"></a><span data-ttu-id="67006-102">Erro do Compilador CS0315</span><span class="sxs-lookup"><span data-stu-id="67006-102">Compiler Error CS0315</span></span>
<span data-ttu-id="67006-103">O tipo 'valueType' não pode ser usado como parâmetro de tipo 'T' no tipo genérico ou método 'TypeorMethod\<T>'.</span><span class="sxs-lookup"><span data-stu-id="67006-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="67006-104">Não há conversão de boxe de 'valueType' para 'referenceType'.</span><span class="sxs-lookup"><span data-stu-id="67006-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="67006-105">Esse erro ocorre quando você restringe um tipo genérico a uma classe específica e tenta construir uma instância dessa classe usando um tipo de valor que não pode ser implicitamente encaixotado a ele.</span><span class="sxs-lookup"><span data-stu-id="67006-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="67006-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="67006-106">To correct this error</span></span>  
  
1. <span data-ttu-id="67006-107">Uma solução é redefinir a estrutura como uma classe.</span><span class="sxs-lookup"><span data-stu-id="67006-107">One solution is to redefine the struct as a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="67006-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="67006-108">Example</span></span>  
 <span data-ttu-id="67006-109">O exemplo a seguir gera CS0315:</span><span class="sxs-lookup"><span data-stu-id="67006-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```

 <span data-ttu-id="3494a-105">Esse erro ocorre quando você restringe um tipo genérico para uma determinada classe e tentar construir uma instância dessa classe usando um tipo de valor não pode ser convertido implicitamente a ele.</span><span class="sxs-lookup"><span data-stu-id="3494a-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>

## <a name="example"></a><span data-ttu-id="3494a-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="3494a-108">Example</span></span>  
 <span data-ttu-id="3494a-109">O exemplo a seguir gera CS0315:</span><span class="sxs-lookup"><span data-stu-id="3494a-109">The following example generates CS0315:</span></span>  
  
```  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}

{  
    public static int Main()  
    {

    }
```  

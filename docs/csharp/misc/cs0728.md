## <a name="example"></a><span data-ttu-id="38c88-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="38c88-110">Example</span></span>  
 <span data-ttu-id="38c88-111">O código a seguir irá gerar um aviso CS0728.</span><span class="sxs-lookup"><span data-stu-id="38c88-111">The following code will generate warning CS0728.</span></span>

```  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)   
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```

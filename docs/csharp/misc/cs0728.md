## <a name="example"></a><span data-ttu-id="5075c-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5075c-110">Example</span></span>  
 <span data-ttu-id="5075c-111">O código a seguir irá gerar aviso CS0728.</span><span class="sxs-lookup"><span data-stu-id="5075c-111">The following code will generate warning CS0728.</span></span>  
  
```csharp 
// CS0728.cs  
  
using System;  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)   
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```

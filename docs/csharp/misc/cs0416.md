# <a name="compiler-error-cs0416"></a><span data-ttu-id="deaed-103">Erro do Compilador CS0416</span><span class="sxs-lookup"><span data-stu-id="deaed-103">Compiler Error CS0416</span></span>

 <span data-ttu-id="deaed-105">Um parâmetro de tipo foi usado como um argumento de atributo, o que não é permitido.</span><span class="sxs-lookup"><span data-stu-id="deaed-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="deaed-106">Use um tipo não genérico.</span><span class="sxs-lookup"><span data-stu-id="deaed-106">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="deaed-107">O exemplo a seguir gera CS0416:</span><span class="sxs-lookup"><span data-stu-id="deaed-107">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```

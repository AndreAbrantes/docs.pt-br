# <a name="compiler-error-cs0756"></a><span data-ttu-id="679ff-103">Erro do Compilador CS0756</span><span class="sxs-lookup"><span data-stu-id="679ff-103">Compiler Error CS0756</span></span>

<span data-ttu-id="679ff-104">Um método parcial não pode ter várias declarações de definição.</span><span class="sxs-lookup"><span data-stu-id="679ff-104">A partial method may not have multiple defining declarations.</span></span>  
  
 <span data-ttu-id="679ff-105">A declaração de definição de um método parcial é a parte que especifica a assinatura do método, mas não a implementação (corpo do método).</span><span class="sxs-lookup"><span data-stu-id="679ff-105">The defining declaration of a partial method is the part that specifies the method signature, but not the implementation (method body).</span></span> <span data-ttu-id="679ff-106">Um método parcial deve ter exatamente uma declaração de definição para cada assinatura exclusiva.</span><span class="sxs-lookup"><span data-stu-id="679ff-106">A partial method must have exactly one defining declaration for each unique signature.</span></span> <span data-ttu-id="679ff-107">Cada versão sobrecarregada de um método parcial deve ter sua própria declaração de definição.</span><span class="sxs-lookup"><span data-stu-id="679ff-107">Each overloaded version of a partial method must have its own defining declaration.</span></span>

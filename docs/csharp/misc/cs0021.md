## <a name="example"></a><span data-ttu-id="79de3-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="79de3-108">Example</span></span>  
 <span data-ttu-id="79de3-109">Esse arquivo é compilado em um arquivo. dll — com o `DefaultMember` atributo comentada — para gerar o erro.</span><span class="sxs-lookup"><span data-stu-id="79de3-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="79de3-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="79de3-110">Example</span></span>

```csharp  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```

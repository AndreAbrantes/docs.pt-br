# <a name="compiler-error-cs0021"></a>Erro do compilador CS0021
Não é possível aplicar a indexação com [] a uma expressão do tipo ' type '  
  
 Foi feita uma tentativa de acessar um valor por meio de um indexador em um tipo de dados que não oferece suporte a [indexadores](../programming-guide/indexers/index.md).  
  
 Você pode obter CS0021 ao tentar usar um indexador em um C++ assembly. Nesse caso, decorar a C++ classe com o `DefaultMember` atributo para que o C# compilador saiba qual indexador é o padrão. O exemplo a seguir gera CS0021.  
  
## <a name="example"></a>Exemplo  
 Esse arquivo é compilado em um arquivo. dll — com o `DefaultMember` atributo comentado — para gerar o erro.  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a>Exemplo  
 Este é o C# arquivo que chama o arquivo. dll. Esse arquivo tenta acessar a classe por meio de um indexador, mas como nenhum membro foi declarado como o indexador padrão a ser usado, o erro é gerado.  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```

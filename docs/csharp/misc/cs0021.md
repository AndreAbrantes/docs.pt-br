 <span data-ttu-id="e46d6-104">Foi feita uma tentativa para acessar um valor por meio de um indexador em um tipo de dados que não oferece suporte a [indexadores](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="e46d6-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="e46d6-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e46d6-108">Example</span></span>  
 <span data-ttu-id="e46d6-109">Esse arquivo é compilado em um arquivo. dll — com o `DefaultMember` atributo comentadas — para gerar o erro.</span><span class="sxs-lookup"><span data-stu-id="e46d6-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```

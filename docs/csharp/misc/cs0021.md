## <a name="example"></a><span data-ttu-id="72008-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="72008-108">Example</span></span>  

<span data-ttu-id="72008-109">O exemplo C++ a seguir compila em um arquivo. dll.</span><span class="sxs-lookup"><span data-stu-id="72008-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="72008-110">Observe que o atributo [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) é comentado para gerar o erro.</span><span class="sxs-lookup"><span data-stu-id="72008-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="72008-111">O exemplo C# a seguir chama o arquivo. dll.</span><span class="sxs-lookup"><span data-stu-id="72008-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="72008-112">Ele tenta acessar a classe por meio de um indexador, mas como nenhum membro foi declarado como o indexador padrão, o erro é gerado.</span><span class="sxs-lookup"><span data-stu-id="72008-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="72008-113">Você pode resolver o erro removendo a marca de comentário da linha `[DefaultMember("myItem")]`.</span><span class="sxs-lookup"><span data-stu-id="72008-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```

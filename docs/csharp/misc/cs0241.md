# <a name="compiler-error-cs0241"></a>Erro do Compilador CS0241
Especificadores de parâmetropadrão não são permitidos  
  
 [Os parâmetros do método](../language-reference/keywords/method-parameters.md) não podem ter valores padrão. Use sobrecargas de método se você quiser obter o mesmo efeito. Para obter mais informações, consulte [Passando Parâmetros](../programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Exemplo  
 A seguinte amostra gera CS0241. Além disso, a amostra mostra como simular, com sobrecarga, um método com argumentos padrão.  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```

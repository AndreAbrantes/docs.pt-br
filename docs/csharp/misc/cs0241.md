## <a name="example"></a><span data-ttu-id="d27d4-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d27d4-107">Example</span></span>  
 <span data-ttu-id="d27d4-108">O exemplo a seguir gera CS0241.</span><span class="sxs-lookup"><span data-stu-id="d27d4-108">The following sample generates CS0241.</span></span> <span data-ttu-id="d27d4-109">Além disso, o exemplo mostra como simular com sobrecarga, um método com argumentos padrão.</span><span class="sxs-lookup"><span data-stu-id="d27d4-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```

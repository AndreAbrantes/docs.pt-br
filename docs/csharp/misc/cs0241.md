## <a name="example"></a><span data-ttu-id="c3d03-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="c3d03-108">Example</span></span>  
 <span data-ttu-id="c3d03-109">O exemplo a seguir gera CS0241.</span><span class="sxs-lookup"><span data-stu-id="c3d03-109">The following sample generates CS0241.</span></span> <span data-ttu-id="c3d03-110">Além disso, o exemplo mostra como simular, com sobrecarga, um método com argumentos padrão.</span><span class="sxs-lookup"><span data-stu-id="c3d03-110">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```

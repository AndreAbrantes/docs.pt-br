# <a name="compiler-error-cs0241"></a>CS0241 de erro do compilador
Não são permitidos especificadores de parâmetro padrão  
  
 [Parâmetros de método](../../csharp/language-reference/keywords/method-parameters.md) não pode ter valores padrão. Use sobrecargas do método para obter o mesmo efeito. Para obter mais informações, consulte [passando parâmetros](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0241. Além disso, o exemplo mostra como simular, com sobrecarga, um método com argumentos padrão.  
  
```  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```

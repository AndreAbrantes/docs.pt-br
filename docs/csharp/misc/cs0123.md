# <a name="compiler-error-cs0123"></a><span data-ttu-id="32de5-103">Erro do Compilador CS0123</span><span class="sxs-lookup"><span data-stu-id="32de5-103">Compiler Error CS0123</span></span>
<span data-ttu-id="32de5-104">Nenhuma sobrecarga para ' method ' corresponde ao delegado ' delegate '</span><span class="sxs-lookup"><span data-stu-id="32de5-104">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="32de5-105">Falha ao tentar criar um delegado porque a assinatura correta não foi usada.</span><span class="sxs-lookup"><span data-stu-id="32de5-105">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="32de5-106">As instâncias de um delegado devem ser declaradas com a mesma assinatura que a declaração delegate.</span><span class="sxs-lookup"><span data-stu-id="32de5-106">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="32de5-107">Você pode resolver esse erro ajustando o método ou a assinatura delegada.</span><span class="sxs-lookup"><span data-stu-id="32de5-107">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="32de5-108">Para obter mais informações, consulte [Delegados](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="32de5-108">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="32de5-109">O exemplo a seguir gera CS0123.</span><span class="sxs-lookup"><span data-stu-id="32de5-109">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```

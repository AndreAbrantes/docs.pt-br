# <a name="compiler-error-cs0123"></a><span data-ttu-id="e46b8-102">CS0123 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="e46b8-102">Compiler Error CS0123</span></span>
<span data-ttu-id="e46b8-103">Nenhuma sobrecarga para 'method' corresponde ao delegate 'delegate'</span><span class="sxs-lookup"><span data-stu-id="e46b8-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="e46b8-104">Falha ao tentar criar um delegado porque a assinatura correta não foi usada.</span><span class="sxs-lookup"><span data-stu-id="e46b8-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="e46b8-105">Instâncias de um delegado devem ser declaradas com a mesma assinatura que a declaração de delegado.</span><span class="sxs-lookup"><span data-stu-id="e46b8-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="e46b8-106">Você pode resolver esse erro, ajustando o método ou delegar a assinatura.</span><span class="sxs-lookup"><span data-stu-id="e46b8-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="e46b8-107">Para obter mais informações, consulte [Delegados](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="e46b8-107">For more information, see [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="e46b8-108">O exemplo a seguir gera CS0123.</span><span class="sxs-lookup"><span data-stu-id="e46b8-108">The following sample generates CS0123.</span></span>  
  
```  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```

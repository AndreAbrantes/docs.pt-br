# <a name="compiler-error-cs0750"></a><span data-ttu-id="adb76-103">Erro do Compilador CS0750</span><span class="sxs-lookup"><span data-stu-id="adb76-103">Compiler Error CS0750</span></span>

<span data-ttu-id="adb76-104">Um método parcial não pode ter modificadores de acesso ou modificadores virtuais, abstratos, de substituição, novos, lacrados ou extern.</span><span class="sxs-lookup"><span data-stu-id="adb76-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="adb76-105">Devido a seu comportamento especial, os métodos parciais estão sujeitos a restrições quanto aos modificadores que eles podem aceitar.</span><span class="sxs-lookup"><span data-stu-id="adb76-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="adb76-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="adb76-106">To correct this error</span></span>  
  
1. <span data-ttu-id="adb76-107">Remova o modificador não autorizado da declaração do método.</span><span class="sxs-lookup"><span data-stu-id="adb76-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="adb76-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="adb76-108">Example</span></span>  

 <span data-ttu-id="adb76-109">O exemplo a seguir gera CS0750:</span><span class="sxs-lookup"><span data-stu-id="adb76-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0750"></a><span data-ttu-id="a0c9b-102">Erro do compilador CS0750</span><span class="sxs-lookup"><span data-stu-id="a0c9b-102">Compiler Error CS0750</span></span>
<span data-ttu-id="a0c9b-103">Um método parcial não pode ter modificadores de acesso ou o virtual, abstract, override, new, sealed ou modificadores extern.</span><span class="sxs-lookup"><span data-stu-id="a0c9b-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="a0c9b-104">Devido a seu comportamento especial, os métodos parciais estão sujeitos às restrições sobre os modificadores que pode ser aceito.</span><span class="sxs-lookup"><span data-stu-id="a0c9b-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

1. <span data-ttu-id="a0c9b-106">Remova o modificador não autorizado da declaração de método.</span><span class="sxs-lookup"><span data-stu-id="a0c9b-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0c9b-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a0c9b-107">Example</span></span>  
 <span data-ttu-id="a0c9b-108">O exemplo a seguir gera CS0750:</span><span class="sxs-lookup"><span data-stu-id="a0c9b-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs1010"></a>Erro do compilador CS1010
Nova linha em constante  
  
 Uma [cadeia de caracteres](../language-reference/keywords/string.md) não foi delimitada corretamente.  
  
 O exemplo a seguir gera CS1010:  
  
```csharp  
// CS1010.cs  
class Sample  
{  
   static void Main()  
   {  
      string a = "Hello World;   // CS1010  
      // Use the following line, with end quote before semicolon:  
      string a = "Hello World"; //  
   }  
}  
```

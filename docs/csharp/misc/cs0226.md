# <a name="compiler-error-cs0226"></a>CS0226 de erro do compilador
Uma expressão arglist só pode aparecer dentro de uma chamada ou uma nova expressão.  
  
 A palavra-chave sem suporte `__arglist` só pode aparecer em uma chamada de método ou uma nova expressão.  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera CS0226:  
  
```  
// cs0226.cs  
using System;  
  
public class C  
    {  
    public static int Main ()  
        {  
        __arglist(1,"This is a string"); // CS0226  
        return 0;  
        }  
    }  
```

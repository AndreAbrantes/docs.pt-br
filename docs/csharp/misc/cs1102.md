# <a name="compiler-error-cs1102"></a><span data-ttu-id="25aca-102">CS1102 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="25aca-102">Compiler Error CS1102</span></span>
<span data-ttu-id="25aca-103">O modificador de parâmetro 'out' não pode ser usado com 'this'.</span><span class="sxs-lookup"><span data-stu-id="25aca-103">The parameter modifier 'out' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="25aca-104">Quando o `this` palavra-chave modifica o primeiro parâmetro de um método estático, ele sinaliza ao compilador que o método é um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="25aca-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="25aca-105">Não há outros modificadores são necessárias ou permitidos no primeiro parâmetro de um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="25aca-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="25aca-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="25aca-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="25aca-107">Remova os modificadores não autorizados do primeiro parâmetro.</span><span class="sxs-lookup"><span data-stu-id="25aca-107">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="25aca-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="25aca-108">Example</span></span>  
 <span data-ttu-id="25aca-109">O exemplo a seguir gera CS1102:</span><span class="sxs-lookup"><span data-stu-id="25aca-109">The following example generates CS1102:</span></span>  
  
```  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```

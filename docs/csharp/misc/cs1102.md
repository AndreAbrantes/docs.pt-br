## <a name="example"></a><span data-ttu-id="c8b28-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="c8b28-109">Example</span></span>  
 <span data-ttu-id="c8b28-110">O exemplo a seguir gera CS1102:</span><span class="sxs-lookup"><span data-stu-id="c8b28-110">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```

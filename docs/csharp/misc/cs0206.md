# <a name="compiler-error-cs0206"></a>CS0206 de erro do compilador
Uma propriedade ou indexador não pode ser passado como ref ou out parâmetro  
  
 Um [propriedade](../../csharp/programming-guide/classes-and-structs/properties.md) não está disponível para ser passado como um [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parâmetro. Para obter mais informações, consulte [Passando Parâmetros](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0206:  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```

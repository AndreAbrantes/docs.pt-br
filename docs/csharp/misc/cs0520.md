# <a name="compiler-error-cs0520"></a>CS0520 de erro do compilador
O tipo predefinido 'name' foi declarado incorretamente  
  
 Um ou mais arquivos necessários para o compilador não foram encontrados. Reinstale o common language runtime do .NET Framework.

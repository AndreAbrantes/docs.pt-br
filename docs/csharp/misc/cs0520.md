# <a name="compiler-error-cs0520"></a>CS0520 de erro do compilador
Tipo predefinido 'name' é declarado incorretamente  
  
 Não foram encontrados um ou mais arquivos necessários pelo compilador. Reinstale o .NET Framework common language runtime.

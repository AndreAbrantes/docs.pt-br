# <a name="compiler-error-cs0520"></a>Erro do Compilador CS0520
O tipo predefinido ' name ' é declarado incorretamente  
  
 Um ou mais arquivos necessários para o compilador não foram encontrados. Reinstale o Common Language Runtime de .NET Framework.

# <a name="compiler-error-cs1719"></a>CS1719 de erro do compilador
Erro ao ler o arquivo de recurso Win32 'Nome do arquivo' - 'razão'  
  
 Uma tentativa de ler o arquivo de recurso Win32 falhou pelo motivo apresentado no erro, normalmente algo como "arquivo não encontrado" ou "acesso negado". Esse erro seja resolvido, corrigindo o problema descrito pelo motivo.

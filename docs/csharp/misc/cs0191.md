## <a name="example"></a><span data-ttu-id="30972-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="30972-107">Example</span></span>

```csharp

class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration

    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

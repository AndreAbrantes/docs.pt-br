## <a name="example"></a><span data-ttu-id="f7517-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f7517-107">Example</span></span>

```csharp

    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration

        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

 <span data-ttu-id="18d7a-104">O padrão para um `join` cláusula é `join`... `in`... `on`... `equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="18d7a-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="18d7a-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="18d7a-107">Example</span></span>  
 <span data-ttu-id="18d7a-108">O código a seguir gera CS0743:</span><span class="sxs-lookup"><span data-stu-id="18d7a-108">The following code generates CS0743:</span></span>  
  
```  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0743"></a><span data-ttu-id="6186e-102">CS0743 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="6186e-102">Compiler Error CS0743</span></span>
<span data-ttu-id="6186e-103">Palavra-chave contextual esperada 'on'</span><span class="sxs-lookup"><span data-stu-id="6186e-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="6186e-104">O padrão para um `join` cláusula é `join`... `in`... `on`... `equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="6186e-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="6186e-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="6186e-107">Example</span></span>

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};

    }  
}

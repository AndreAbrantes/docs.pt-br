# <a name="compiler-error-cs0743"></a><span data-ttu-id="cd502-102">CS0743 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="cd502-102">Compiler Error CS0743</span></span>
<span data-ttu-id="cd502-103">Palavra-chave contextual 'on' esperado</span><span class="sxs-lookup"><span data-stu-id="cd502-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="cd502-104">O padrão para um `join` cláusula é `join`... `in`... `on`... `equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="cd502-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="cd502-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="cd502-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="cd502-106">Adicionar o `on` palavra-chave para o `join` cláusula.</span><span class="sxs-lookup"><span data-stu-id="cd502-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cd502-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="cd502-107">Example</span></span>  
 <span data-ttu-id="cd502-108">O código a seguir gera CS0743:</span><span class="sxs-lookup"><span data-stu-id="cd502-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0743"></a><span data-ttu-id="84541-103">Erro do Compilador CS0743</span><span class="sxs-lookup"><span data-stu-id="84541-103">Compiler Error CS0743</span></span>

<span data-ttu-id="84541-104">Palavra-chave contextual esperada ' on '</span><span class="sxs-lookup"><span data-stu-id="84541-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="84541-105">O padrão para uma `join` cláusula é `join` ... `in` ...`on` ... `equals` , conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="84541-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="84541-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="84541-106">To correct this error</span></span>  
  
1. <span data-ttu-id="84541-107">Adicione a `on` palavra-chave à `join` cláusula.</span><span class="sxs-lookup"><span data-stu-id="84541-107">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="84541-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="84541-108">Example</span></span>  

 <span data-ttu-id="84541-109">O código a seguir gera CS0743:</span><span class="sxs-lookup"><span data-stu-id="84541-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```

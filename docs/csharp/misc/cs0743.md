# <a name="compiler-error-cs0743"></a><span data-ttu-id="2a4c4-102">Erro do compilador CS0743</span><span class="sxs-lookup"><span data-stu-id="2a4c4-102">Compiler Error CS0743</span></span>
<span data-ttu-id="2a4c4-103">Palavra-chave contextual esperada ' on '</span><span class="sxs-lookup"><span data-stu-id="2a4c4-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="2a4c4-104">O padrão para uma `join` cláusula é `join`... `in`... `on`... `equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="2a4c4-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="2a4c4-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="2a4c4-107">Example</span></span>  
 <span data-ttu-id="2a4c4-108">O código a seguir gera CS0743:</span><span class="sxs-lookup"><span data-stu-id="2a4c4-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```

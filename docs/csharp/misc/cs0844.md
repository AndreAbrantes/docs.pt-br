## <a name="example"></a><span data-ttu-id="7b8cb-112">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7b8cb-112">Example</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

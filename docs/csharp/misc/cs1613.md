# <a name="compiler-error-cs1613"></a><span data-ttu-id="5d01c-103">Erro do Compilador CS1613</span><span class="sxs-lookup"><span data-stu-id="5d01c-103">Compiler Error CS1613</span></span>
<span data-ttu-id="5d01c-104">A classe ' class ' do wrapper da coclasse gerenciada para a interface ' interface ' não pode ser encontrada (está faltando uma referência de assembly?)</span><span class="sxs-lookup"><span data-stu-id="5d01c-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="5d01c-105">Foi feita uma tentativa de instanciar um objeto COM de uma interface.</span><span class="sxs-lookup"><span data-stu-id="5d01c-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="5d01c-106">A interface tem os atributos **ComImport** e `CoClass` , mas o compilador não pode localizar o tipo fornecido para o `CoClass` atributo.</span><span class="sxs-lookup"><span data-stu-id="5d01c-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="5d01c-107">Para resolver esse erro, você pode tentar um dos seguintes:</span><span class="sxs-lookup"><span data-stu-id="5d01c-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="5d01c-108">Adicione uma referência ao assembly que tem a coclass (na maioria das vezes, a interface e a coclass devem estar no mesmo assembly).</span><span class="sxs-lookup"><span data-stu-id="5d01c-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="5d01c-109">Consulte [a](../language-reference/compiler-options/reference-compiler-option.md) caixa de [diálogo referência ou Adicionar referência](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) para obter informações.</span><span class="sxs-lookup"><span data-stu-id="5d01c-109">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="5d01c-110">Corrija o `CoClass` atributo na interface.</span><span class="sxs-lookup"><span data-stu-id="5d01c-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="5d01c-111">O exemplo a seguir demonstra o uso correto de **Coclassidattribute**:</span><span class="sxs-lookup"><span data-stu-id="5d01c-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```

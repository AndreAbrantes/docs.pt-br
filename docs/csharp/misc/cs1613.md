# <a name="compiler-error-cs1613"></a><span data-ttu-id="e72a9-102">Erro do compilador CS1613</span><span class="sxs-lookup"><span data-stu-id="e72a9-102">Compiler Error CS1613</span></span>
<span data-ttu-id="e72a9-103">A classe de wrapper gerenciado coclass 'class' para interface 'interface' não pode ser encontrada (uma referência de assembly está faltando?)</span><span class="sxs-lookup"><span data-stu-id="e72a9-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="e72a9-104">Foi feita uma tentativa de instanciar um objeto COM de uma interface.</span><span class="sxs-lookup"><span data-stu-id="e72a9-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="e72a9-105">A interface tem o **ComImport** e `CoClass` atributos, mas o compilador não é possível localizar o tipo de dado para o `CoClass` atributo.</span><span class="sxs-lookup"><span data-stu-id="e72a9-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="e72a9-106">Para resolver esse erro, tente um destes procedimentos:</span><span class="sxs-lookup"><span data-stu-id="e72a9-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="e72a9-107">Adicione uma referência ao assembly que tem a coclass (a maioria das vezes, que a interface e coclass devem estar no mesmo assembly).</span><span class="sxs-lookup"><span data-stu-id="e72a9-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="e72a9-108">Ver [/Reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) ou [caixa de diálogo Adicionar referência](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) para obter informações.</span><span class="sxs-lookup"><span data-stu-id="e72a9-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
-   <span data-ttu-id="e72a9-109">Corrigir a `CoClass` atributo na interface.</span><span class="sxs-lookup"><span data-stu-id="e72a9-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="e72a9-110">O exemplo a seguir demonstra o uso correto dos **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="e72a9-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```

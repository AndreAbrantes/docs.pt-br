# <a name="compiler-error-cs1559"></a>Erro do compilador CS1559
Não é possível usar ' Object ' para o método Main porque ele é importado  
  
 Uma classe inválida foi especificada para a opção de compilador [/Main](../language-reference/compiler-options/main-compiler-option.md) ; a classe não pode ser usada como um local para o método [Main](../programming-guide/main-and-command-args/index.md) .

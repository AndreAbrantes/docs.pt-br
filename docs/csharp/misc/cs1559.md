# <a name="compiler-error-cs1559"></a>Erro do compilador CS1559
Não é possível usar 'object' para o método Main porque ele é importado  
  
 Uma classe inválida foi especificada para o [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador; a classe não pode ser usada como um local para o [Main](../../csharp/programming-guide/main-and-command-args/index.md) método.

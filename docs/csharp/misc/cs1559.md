# <a name="compiler-error-cs1559"></a>CS1559 de erro do compilador
Não é possível usar 'object' para o método Main porque é importado  
  
 Uma classe inválida foi especificada para o [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador; a classe não pode ser usada como um local para o [principal](../../csharp/programming-guide/main-and-command-args/index.md) método.

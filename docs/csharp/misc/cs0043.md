# <a name="compiler-error-cs0043"></a>Erro do compilador CS0043

O arquivo PDB ' file ' tem um formato incorreto ou desatualizado. Exclua-o e recompile.

 Exclua o arquivo. pdb para esta compilação e recompile.

> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

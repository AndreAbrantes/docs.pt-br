# <a name="compiler-error-cs0043"></a>Erro do compilador CS0043
Arquivo PDB 'file' tem um formato incorreto ou desatualizado. Excluí-lo e recompile.  
  
 Exclua o arquivo. PDB para esta compilação e recompilação.

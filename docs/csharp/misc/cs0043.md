# <a name="compiler-error-cs0043"></a>CS0043 de erro do compilador
Arquivo PDB 'arquivo' tem um formato incorreto ou desatualizado. Exclua e recrie.  
  
 Exclua o arquivo. PDB para esta compilação e recompilação.

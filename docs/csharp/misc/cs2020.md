# <a name="compiler-error-cs2020"></a>Erro do Compilador CS2020

Somente o primeiro conjunto de arquivos de entrada pode criar um destino diferente de ' module '  
  
 Em uma compilação de várias saídas, o primeiro arquivo de saída deve ser criado com [-target: exe](../language-reference/compiler-options/target-exe-compiler-option.md), [-target: winexe](../language-reference/compiler-options/target-winexe-compiler-option.md)ou [-target: library](../language-reference/compiler-options/target-library-compiler-option.md). Todos os arquivos de saída subsequentes devem ser compilados com o [módulo-target:](../language-reference/compiler-options/target-module-compiler-option.md).

 <span data-ttu-id="1f6cc-104">Esse erro ocorre se você tentar atribuir ou caso contrário, converta um bloco de método anônimo para um tipo que não é um tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="1f6cc-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>

   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660

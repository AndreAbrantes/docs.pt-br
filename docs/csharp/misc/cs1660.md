# <a name="compiler-error-cs1660"></a><span data-ttu-id="5d92e-103">Erro do Compilador CS1660</span><span class="sxs-lookup"><span data-stu-id="5d92e-103">Compiler Error CS1660</span></span>

<span data-ttu-id="5d92e-104">Não é possível converter o bloco de métodos anônimo no tipo ' type ' porque ele não é um tipo delegado</span><span class="sxs-lookup"><span data-stu-id="5d92e-104">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="5d92e-105">Esse erro ocorrerá se você tentar atribuir ou converter um bloco de método anônimo em um tipo que não seja um tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="5d92e-105">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="5d92e-106">O exemplo a seguir gera CS1660:</span><span class="sxs-lookup"><span data-stu-id="5d92e-106">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```

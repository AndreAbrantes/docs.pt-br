# <a name="compiler-error-cs1660"></a><span data-ttu-id="c514f-102">CS1660 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="c514f-102">Compiler Error CS1660</span></span>
<span data-ttu-id="c514f-103">Não é possível converter bloco de métodos anônimo para o tipo 'type' porque ele não é um tipo delegado</span><span class="sxs-lookup"><span data-stu-id="c514f-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="c514f-104">Esse erro ocorre se você tentar atribuir ou caso contrário, converta um bloco de métodos anônimo para um tipo que não é um tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="c514f-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="c514f-105">O exemplo a seguir gera CS1660:</span><span class="sxs-lookup"><span data-stu-id="c514f-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```

 <span data-ttu-id="db2b4-104">Esse erro ocorre se você tentar atribuir ou caso contrário, converta um bloco de métodos anônimo para um tipo que não é um tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="db2b4-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="db2b4-105">O exemplo a seguir gera CS1660:</span><span class="sxs-lookup"><span data-stu-id="db2b4-105">The following sample generates CS1660:</span></span>  
  
```  
// CS1660.cs

   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```

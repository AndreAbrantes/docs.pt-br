# <a name="compiler-error-cs0005"></a>Erro do compilador CS0005
Opção de compilador 'compiler_option' deve ser seguida por um argumento  
  
 Algumas opções de compilador exigem parâmetros. Se você não passar os argumentos necessários pela opção do compilador, CS0005 será gerado.  
  
 Para obter mais informações, consulte [ C# opções do compilador](../../csharp/language-reference/compiler-options/index.md).

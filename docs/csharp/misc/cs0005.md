# <a name="compiler-error-cs0005"></a>CS0005 de erro do compilador
Opção de compilador 'compiler_option' deve ser seguida por um argumento  
  
 Algumas opções de compilador requerem parâmetros. Se você não passar os argumentos necessários para a opção de compilador, CS0005 será gerado.  
  
 Para obter mais informações, consulte [opções do compilador c#](../../csharp/language-reference/compiler-options/index.md).

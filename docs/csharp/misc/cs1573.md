# <a name="compiler-warning-level-4-cs1573"></a>Aviso do compilador (nível 4) CS1573
O parâmetro ' Parameter ' não tem nenhuma marca param correspondente no comentário XML para ' Parameter ' (mas outros parâmetros têm)  
  
 Ao usar a opção de compilador [/Doc](../language-reference/compiler-options/doc-compiler-option.md) , um comentário foi especificado para alguns, mas não para todos os parâmetros em um método. Você pode ter esquecido de inserir um comentário para esses parâmetros.  
  
 O exemplo a seguir gera CS1573:  
  
```csharp  
// CS1573.cs  
// compile with: /W:4  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   // enter a comment for Char1?  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   public static void Main ()  
   {  
   }  
}  
```

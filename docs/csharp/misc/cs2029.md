# <a name="compiler-warning-level-1-cs2029"></a>Compilador CS2029 de aviso (nível 1)
Valor inválido para '/define'; 'identifier' não é um identificador válido  
  
 Este aviso ocorre se o valor que é usado no [/define](../../csharp/language-reference/compiler-options/define-compiler-option.md) opção tem caracteres inválidos.  
  
 Esse aviso não pode ser suprimido pelo [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opção.

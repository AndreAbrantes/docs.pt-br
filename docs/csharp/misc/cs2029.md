# <a name="compiler-warning-level-1-cs2029"></a>Aviso do compilador (nível 1) CS2029
Valor inválido para '/define'; 'identifier' não é um identificador válido  
  
 Este aviso ocorre se o valor que é usado na [/define](../../csharp/language-reference/compiler-options/define-compiler-option.md) opção tem caracteres inválidos.  
  
 Esse aviso não pode ser suprimido pela [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opção.

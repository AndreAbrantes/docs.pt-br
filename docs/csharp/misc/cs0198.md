 <span data-ttu-id="8da46-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variável deve ter o mesmo [estático](../../csharp/language-reference/keywords/static.md) uso do construtor no qual você deseja inicializar.</span><span class="sxs-lookup"><span data-stu-id="8da46-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="8da46-105">Para obter mais informações, consulte [construtores estáticos](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="8da46-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="8da46-106">O exemplo a seguir gera CS0198:</span><span class="sxs-lookup"><span data-stu-id="8da46-106">The following sample generates CS0198:</span></span>  
  
```  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

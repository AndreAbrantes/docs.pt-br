# <a name="compiler-error-cs0167"></a>Erro do compilador CS0167
O delegado 'delegate' está ausente no método Invoke  
  
 Você importou e usaram um programa gerenciado (um que usa o .NET Framework common language runtime) que foi criado com outro compilador. Que o compilador permitido um mal formado [delegar](../../csharp/language-reference/keywords/delegate.md). Portanto, o `Invoke` método não estava disponível. Para obter mais informações, consulte [Delegados](../../csharp/programming-guide/delegates/index.md).

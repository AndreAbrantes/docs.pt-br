# <a name="compiler-error-cs0167"></a>Erro do Compilador CS0167
O método Invoke está ausente no delegado ' delegate '  
  
 Você importou e usou um programa gerenciado (um que usa o .NET Framework Common Language Runtime) que foi criado com outro compilador. Esse compilador permitia um [delegado](../language-reference/builtin-types/reference-types.md)mal formado. Portanto, o `Invoke` método não estava disponível. Para obter mais informações, consulte [Delegados](../programming-guide/delegates/index.md).

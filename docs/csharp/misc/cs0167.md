# <a name="compiler-error-cs0167"></a>CS0167 de erro do compilador
O representante 'representante' está ausente no método Invoke  
  
 Você importou e usado um programa gerenciado (um que usa o .NET Framework common language runtime) que foi criado com outro compilador. Que o compilador permitido um mal formados [delegar](../../csharp/language-reference/keywords/delegate.md). Portanto, o `Invoke` método não estava disponível. Para obter mais informações, consulte [Delegados](../../csharp/programming-guide/delegates/index.md).

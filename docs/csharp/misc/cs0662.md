 <span data-ttu-id="8c1fb-105">Um método de interface tem um parâmetro que usa [ref](../../csharp/language-reference/keywords/ref.md) com apenas o [Out](xref:System.Runtime.InteropServices.OutAttribute) atributo.</span><span class="sxs-lookup"><span data-stu-id="8c1fb-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="8c1fb-106">A `ref` parâmetro usa o **Out** atributo também deve usar o [na](xref:System.Runtime.InteropServices.InAttribute) atributo.</span><span class="sxs-lookup"><span data-stu-id="8c1fb-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="8c1fb-107">O exemplo a seguir gera CS0662:</span><span class="sxs-lookup"><span data-stu-id="8c1fb-107">The following sample generates CS0662:</span></span>

```csharp
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```

 <span data-ttu-id="78304-104">Esse erro ocorre quando você usar "::" à direita de um identificador que não é um alias.</span><span class="sxs-lookup"><span data-stu-id="78304-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="78304-105">Para resolver o erro, use "." em vez disso.</span><span class="sxs-lookup"><span data-stu-id="78304-105">To resolve the error, use "." instead.</span></span>

```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```

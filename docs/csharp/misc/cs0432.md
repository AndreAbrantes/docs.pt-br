# <a name="compiler-error-cs0432"></a><span data-ttu-id="6cfc7-103">Erro do Compilador CS0432</span><span class="sxs-lookup"><span data-stu-id="6cfc7-103">Compiler Error CS0432</span></span>

 <span data-ttu-id="6cfc7-105">Esse erro ocorre quando você usa "::" à direita de um identificador que não é um alias.</span><span class="sxs-lookup"><span data-stu-id="6cfc7-105">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="6cfc7-106">Para resolver o erro, use "." em vez disso.</span><span class="sxs-lookup"><span data-stu-id="6cfc7-106">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="6cfc7-107">O exemplo a seguir gera CS0432:</span><span class="sxs-lookup"><span data-stu-id="6cfc7-107">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```

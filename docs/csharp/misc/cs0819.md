## <a name="to-correct-this-error"></a><span data-ttu-id="bb675-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="bb675-105">To correct this error</span></span>

<span data-ttu-id="bb675-106">Existem duas opções:</span><span class="sxs-lookup"><span data-stu-id="bb675-106">There are two options:</span></span>
1. <span data-ttu-id="bb675-107">Se as variáveis forem do mesmo tipo, use declarações explícitas.</span><span class="sxs-lookup"><span data-stu-id="bb675-107">If the variables are of the same type, use explicit declarations.</span></span>

## <a name="example-1"></a><span data-ttu-id="bb675-109">Exemplo 1</span><span class="sxs-lookup"><span data-stu-id="bb675-109">Example 1</span></span>
 <span data-ttu-id="bb675-110">O código a seguir gera CS0819:</span><span class="sxs-lookup"><span data-stu-id="bb675-110">The following code generates CS0819:</span></span>
  
```csharp
// cs0819.cs
class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        // Second correction option.
        //var a = 3;
        //var b = 2;
    }
}
```

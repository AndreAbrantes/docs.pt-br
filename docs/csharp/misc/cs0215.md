# <a name="compiler-error-cs0215"></a>Erro do compilador CS0215
O tipo de retorno do operador True ou False deve ser bool  
  
Definido pelo usu√°rio [verdadeiros e falsos](../language-reference/operators/true-false-operators.md) operadores devem ter um tipo de retorno [bool](../language-reference/keywords/bool.md).
  
O exemplo a seguir gera CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

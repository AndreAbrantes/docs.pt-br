# <a name="compiler-error-cs0215"></a>Erro do compilador CS0215
O tipo de retorno do operador true ou false deve ser bool  
  
Os operadores [true e false](../language-reference/operators/true-false-operators.md) definidos pelo usu√°rio devem ter um tipo de retorno de [bool](../language-reference/builtin-types/bool.md).
  
O exemplo a seguir gera CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="2c5a4-102">Compilador aviso (nível 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="2c5a4-102">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="2c5a4-103">'class' substitui Object. Equals (objeto o) mas não substitui Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="2c5a4-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="2c5a4-104">O compilador detectou uma substituição do <xref:System.Object.Equals%2A?displayProperty=nameWithType> substituição de método, mas não do <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> método.</span><span class="sxs-lookup"><span data-stu-id="2c5a4-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="2c5a4-105">Uma substituição de <xref:System.Object.Equals%2A> implica que você também deseja substituir <xref:System.Object.GetHashCode%2A>.</span><span class="sxs-lookup"><span data-stu-id="2c5a4-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="2c5a4-106">O código a seguir gera CS0659:</span><span class="sxs-lookup"><span data-stu-id="2c5a4-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```

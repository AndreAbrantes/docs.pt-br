# <a name="compiler-warning-level-4-cs0649"></a><span data-ttu-id="26950-102">Compilador CS0649 de aviso (nível 4)</span><span class="sxs-lookup"><span data-stu-id="26950-102">Compiler Warning (level 4) CS0649</span></span>
<span data-ttu-id="26950-103">O campo 'field' nunca é atribuído e sempre terá seu valor padrão 'value'</span><span class="sxs-lookup"><span data-stu-id="26950-103">Field 'field' is never assigned to, and will always have its default value 'value'</span></span>  
  
 <span data-ttu-id="26950-104">O compilador detectou uma declaração de campo particular ou interno não inicializada nunca é atribuída um valor.</span><span class="sxs-lookup"><span data-stu-id="26950-104">The compiler detected an uninitialized private or internal field declaration that is never assigned a value.</span></span>  
  
 <span data-ttu-id="26950-105">O exemplo a seguir gera CS0649:</span><span class="sxs-lookup"><span data-stu-id="26950-105">The following sample generates CS0649:</span></span>  
  
```csharp  
// CS0649.cs  
// compile with: /W:4  
using System.Collections;  
  
class MyClass  
{  
   Hashtable table;  // CS0649  
   // You may have intended to initialize the variable to null  
   // Hashtable table = null;  
  
   // Or you may have meant to create an object here  
   // Hashtable table = new Hashtable();  
  
   public void Func(object o, string p)  
   {  
      // Or here  
      // table = new Hashtable();  
      table[p] = o;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

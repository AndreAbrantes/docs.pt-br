## <a name="example"></a><span data-ttu-id="8488f-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8488f-107">Example</span></span>  
 <span data-ttu-id="8488f-108">O exemplo a seguir gera CS0272:</span><span class="sxs-lookup"><span data-stu-id="8488f-108">The following example generates CS0272:</span></span>  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```

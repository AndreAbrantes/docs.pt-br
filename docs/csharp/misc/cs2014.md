# <a name="compiler-warning-level-1-cs2014"></a>Compilador CS2014 de aviso (nível 1)
Opção de compilador 'antigo option' é obsoleta, em vez disso, use 'nova opção'  
  
 O formato da opção de compilador foi preterido. Consulte [opções do compilador c#](../../csharp/language-reference/compiler-options/index.md) para obter mais informações.

# <a name="compiler-warning-level-1-cs2014"></a>Compilador aviso (nível 1) CS2014
Opção de compilador 'option antigo' é obsoleta, use 'nova opção' em vez disso  
  
 A forma da opção do compilador foi preterida. Ver [ C# opções do compilador](../../csharp/language-reference/compiler-options/index.md) para obter mais informações.

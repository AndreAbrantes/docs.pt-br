# <a name="compiler-warning-level-1-cs2014"></a>Aviso do compilador (nível 1) CS2014
A opção de compilador ' Old Option ' é obsoleta, use ' New Option ' em vez disso  
  
 A forma da opção do compilador é preterida. Consulte [ C# opções do compilador](../language-reference/compiler-options/index.md) para obter mais informações.

# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="21c9b-102">Compilador CS0684 de aviso (nível 1)</span><span class="sxs-lookup"><span data-stu-id="21c9b-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="21c9b-103">interface de 'interface' marcada com 'CoClassAttribute' não está marcada com 'ComImportAttribute'</span><span class="sxs-lookup"><span data-stu-id="21c9b-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="21c9b-104">Se você especificar **CoClassAttribute** em uma interface, você também deve especificar **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="21c9b-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="21c9b-105">O exemplo a seguir gera CS0684:</span><span class="sxs-lookup"><span data-stu-id="21c9b-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```

# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="82596-102">Compilador CS0078 de aviso (nível 4)</span><span class="sxs-lookup"><span data-stu-id="82596-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="82596-103">O sufixo 'l'é facilmente confundido com o dígito '1' - use 'l' para maior clareza</span><span class="sxs-lookup"><span data-stu-id="82596-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="82596-104">O compilador avisa quando ele detectar uma conversão de tempo usando um l minúsculo, em vez de um l maiusculo</span><span class="sxs-lookup"><span data-stu-id="82596-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="82596-105">O exemplo a seguir gera CS0078:</span><span class="sxs-lookup"><span data-stu-id="82596-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```

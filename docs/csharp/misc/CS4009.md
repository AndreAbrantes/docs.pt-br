# <a name="compiler-error-cs4009"></a><span data-ttu-id="f1b77-102">CS4009 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="f1b77-102">Compiler Error CS4009</span></span>

<span data-ttu-id="f1b77-103">'Type.Method': um ponto de entrada não pode ser marcado com o `async` modificador.</span><span class="sxs-lookup"><span data-stu-id="f1b77-103">'Type.Method': an entry point cannot be marked with the `async` modifier.</span></span>

<span data-ttu-id="f1b77-104">Não é possível usar o `async` palavra-chave no ponto de entrada do aplicativo (normalmente o `Main` método).</span><span class="sxs-lookup"><span data-stu-id="f1b77-104">You cannot use the `async` keyword in the application entry point (typically the `Main` method).</span></span>  

<span data-ttu-id="f1b77-105">O exemplo a seguir produz CS4009:</span><span class="sxs-lookup"><span data-stu-id="f1b77-105">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```

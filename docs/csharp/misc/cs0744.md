# <a name="compiler-error-cs0744"></a><span data-ttu-id="a4230-102">Erro do compilador CS0744</span><span class="sxs-lookup"><span data-stu-id="a4230-102">Compiler Error CS0744</span></span>
<span data-ttu-id="a4230-103">Palavra-chave contextual esperada ' Equals '</span><span class="sxs-lookup"><span data-stu-id="a4230-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="a4230-104">O padrão para uma cláusula `join` é `join`...`in`...`on`...`equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="a4230-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a4230-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="a4230-105">To correct this error</span></span>  
  
1. <span data-ttu-id="a4230-106">Adicione a palavra-chave `equals` à cláusula `join`.</span><span class="sxs-lookup"><span data-stu-id="a4230-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a4230-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a4230-107">Example</span></span>  
 <span data-ttu-id="a4230-108">O código a seguir gera CS0744:</span><span class="sxs-lookup"><span data-stu-id="a4230-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```

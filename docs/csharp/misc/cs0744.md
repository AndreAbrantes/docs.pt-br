# <a name="compiler-error-cs0744"></a><span data-ttu-id="22dcd-102">CS0744 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="22dcd-102">Compiler Error CS0744</span></span>
<span data-ttu-id="22dcd-103">Palavra-chave contextual esperada 'equals'</span><span class="sxs-lookup"><span data-stu-id="22dcd-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="22dcd-104">O padrão para um `join` cláusula é `join`... `in`... `on`... `equals`, conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="22dcd-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y  
            select x;

## <a name="example"></a><span data-ttu-id="22dcd-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="22dcd-107">Example</span></span>

public class C  
{  
    public static int Main()

# <a name="compiler-error-cs0744"></a><span data-ttu-id="4e439-103">Erro do Compilador CS0744</span><span class="sxs-lookup"><span data-stu-id="4e439-103">Compiler Error CS0744</span></span>

<span data-ttu-id="4e439-104">Palavra-chave contextual esperada ' Equals '</span><span class="sxs-lookup"><span data-stu-id="4e439-104">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="4e439-105">O padrão para uma `join` cláusula é `join` ... `in` ...`on` ... `equals` , conforme mostrado neste exemplo:</span><span class="sxs-lookup"><span data-stu-id="4e439-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4e439-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="4e439-106">To correct this error</span></span>  
  
1. <span data-ttu-id="4e439-107">Adicione a `equals` palavra-chave à `join` cláusula.</span><span class="sxs-lookup"><span data-stu-id="4e439-107">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4e439-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4e439-108">Example</span></span>  

 <span data-ttu-id="4e439-109">O código a seguir gera CS0744:</span><span class="sxs-lookup"><span data-stu-id="4e439-109">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```

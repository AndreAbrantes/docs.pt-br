# <a name="compiler-error-cs1558"></a>Erro do compilador CS1558
' class ' não tem um método Main estático adequado  
  
 A opção de compilador [/Main](../language-reference/compiler-options/main-compiler-option.md) especificou uma classe na qual procurar um método **Main** . No entanto, o método [Main](../programming-guide/main-and-command-args/index.md) não foi definido corretamente.  
  
 O exemplo a seguir gera CS1558 devido a um tipo de retorno inválido.  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```

# <a name="compiler-error-cs1558"></a><span data-ttu-id="2418b-102">Erro do compilador CS1558</span><span class="sxs-lookup"><span data-stu-id="2418b-102">Compiler Error CS1558</span></span>
<span data-ttu-id="2418b-103">' class ' não tem um método Main estático adequado</span><span class="sxs-lookup"><span data-stu-id="2418b-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="2418b-104">A opção de compilador [/Main](../language-reference/compiler-options/main-compiler-option.md) especificou uma classe na qual procurar um método **Main** .</span><span class="sxs-lookup"><span data-stu-id="2418b-104">The [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="2418b-105">No entanto, o método [Main](../programming-guide/main-and-command-args/index.md) não foi definido corretamente.</span><span class="sxs-lookup"><span data-stu-id="2418b-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="2418b-106">O exemplo a seguir gera CS1558 devido a um tipo de retorno inválido.</span><span class="sxs-lookup"><span data-stu-id="2418b-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```

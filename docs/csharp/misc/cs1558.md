# <a name="compiler-error-cs1558"></a><span data-ttu-id="0f2df-102">Erro do Compilador CS1558</span><span class="sxs-lookup"><span data-stu-id="0f2df-102">Compiler Error CS1558</span></span>
<span data-ttu-id="0f2df-103">'classe' não tem um método principal estático adequado</span><span class="sxs-lookup"><span data-stu-id="0f2df-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="0f2df-104">A opção de compilador [principal](../language-reference/compiler-options/main-compiler-option.md) especificou uma classe na qual procurar um método **Principal.**</span><span class="sxs-lookup"><span data-stu-id="0f2df-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="0f2df-105">No entanto, o método [Principal](../programming-guide/main-and-command-args/index.md) não foi definido corretamente.</span><span class="sxs-lookup"><span data-stu-id="0f2df-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="0f2df-106">O exemplo a seguir gera CS1558 devido ao tipo de retorno inválido.</span><span class="sxs-lookup"><span data-stu-id="0f2df-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```

# <a name="compiler-error-cs1558"></a><span data-ttu-id="c9f23-103">Erro do Compilador CS1558</span><span class="sxs-lookup"><span data-stu-id="c9f23-103">Compiler Error CS1558</span></span>

<span data-ttu-id="c9f23-104">' class ' não tem um método Main estático adequado</span><span class="sxs-lookup"><span data-stu-id="c9f23-104">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="c9f23-105">A opção [-Main](../language-reference/compiler-options/main-compiler-option.md) do compilador especificou uma classe na qual procurar um método **Main** .</span><span class="sxs-lookup"><span data-stu-id="c9f23-105">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="c9f23-106">No entanto, o método [Main](../programming-guide/main-and-command-args/index.md) não foi definido corretamente.</span><span class="sxs-lookup"><span data-stu-id="c9f23-106">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="c9f23-107">O exemplo a seguir gera CS1558 devido a um tipo de retorno inválido.</span><span class="sxs-lookup"><span data-stu-id="c9f23-107">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```

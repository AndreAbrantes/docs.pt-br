# <a name="compiler-error-cs1558"></a><span data-ttu-id="a3307-102">CS1558 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="a3307-102">Compiler Error CS1558</span></span>
<span data-ttu-id="a3307-103">'class' não tem um método Main estático adequado</span><span class="sxs-lookup"><span data-stu-id="a3307-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="a3307-104">O [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) opção de compilador especificou uma classe na qual procurar um **principal** método.</span><span class="sxs-lookup"><span data-stu-id="a3307-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="a3307-105">No entanto, o [principal](../../csharp/programming-guide/main-and-command-args/index.md) método não foi definido corretamente.</span><span class="sxs-lookup"><span data-stu-id="a3307-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="a3307-106">O exemplo a seguir gera CS1558 devido ao tipo de retorno inválido.</span><span class="sxs-lookup"><span data-stu-id="a3307-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```

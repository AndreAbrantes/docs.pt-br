# <a name="compiler-error-cs1949"></a>Erro do compilador CS1949
A palavra-chave contextual 'var' não pode ser usado em uma declaração de variável de intervalo.  
  
 Uma variável de intervalo é digitada implicitamente pelo compilador. Não é necessário usar [var](../../csharp/language-reference/keywords/var.md) com uma variável de intervalo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `var` palavra-chave from in front of a variável de intervalo.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1949:  
  
```csharp  
// cs1949.cs  
using System;  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        var x = from var i in Enumerable.Range(1, 100) // CS1949  
        select i;  
    }  
}  
```

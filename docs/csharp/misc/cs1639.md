# <a name="compiler-error-cs1639"></a>CS1639 de erro do compilador
A assinatura de classe de wrapper gerenciado coclass 'assinatura' para interface 'interface' não é uma assinatura de nome de classe válida  
  
 Esse erro ocorre se os metadados para a interface são inválido. Isso nunca deve ocorrer se a interface tiver sido gerada usando c# ou um idioma com suporte do .NET. Entre em contato com o fornecedor da interface ou verifique se o assembly de interface foi gerado com êxito.

# <a name="compiler-error-cs1639"></a>Erro do compilador CS1639
A assinatura de classe de wrapper gerenciado coclass 'assinatura' para interface 'interface' não é uma assinatura de nome de classe válida  
  
 Esse erro ocorre se os metadados para a interface são inválido. Isso nunca deverá ocorrer se a interface foi gerada usando C# ou uma linguagem com suporte do .NET. Entre em contato com o fornecedor da interface ou verificar se o assembly da interface foi gerado com êxito.

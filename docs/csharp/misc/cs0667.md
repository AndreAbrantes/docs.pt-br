# <a name="compiler-error-cs0667"></a>Erro do compilador CS0667
O recurso 'recurso inválido' é preterido. Em vez disso, use 'recurso válido' '.  
  
 O recurso que você está tentando usar foi preterido. Atualize seu código para usar o recurso válido em vez disso.

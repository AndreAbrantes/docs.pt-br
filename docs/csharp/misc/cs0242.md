# <a name="compiler-error-cs0242"></a>Erro do compilador CS0242
A operação em questão não está definida nos ponteiros void  
  
 Não é permitido incrementar um ponteiro nulo. Para obter mais informações, consulte [Código não seguro e ponteiros](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 O exemplo a seguir gera CS0242:  
  
```csharp  
// CS0242.cs  
// compile with: /unsafe  
class TestClass  
{  
   public unsafe void Test()  
   {  
      void * p = null;  
      p++;   // CS0242, incrementing a void pointer not allowed  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

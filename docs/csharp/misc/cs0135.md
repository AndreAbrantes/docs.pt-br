## <a name="example"></a><span data-ttu-id="36e2b-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="36e2b-106">Example</span></span>  
 <span data-ttu-id="36e2b-107">O exemplo a seguir gera CS0135:</span><span class="sxs-lookup"><span data-stu-id="36e2b-107">The following sample generates CS0135:</span></span>  
  
```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```

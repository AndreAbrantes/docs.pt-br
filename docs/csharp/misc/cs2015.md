# <a name="compiler-error-cs2015"></a>CS2015 de erro do compilador
'file' é um arquivo binário em vez de um arquivo de texto  
  
 Um arquivo foi passado para o compilador que era um arquivo binário. O compilador espera um arquivo de código fonte.

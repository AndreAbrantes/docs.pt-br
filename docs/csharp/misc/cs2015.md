# <a name="compiler-error-cs2015"></a>CS2015 de erro do compilador
'arquivo' é um arquivo binário em vez de um arquivo de texto  
  
 Um arquivo foi passado para o compilador que era um arquivo binário. O compilador espera que um arquivo de código fonte.

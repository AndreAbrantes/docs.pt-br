# <a name="compiler-warning-level-1-cs1201"></a>Compilador CS1201 de aviso (nível 1)
O recurso 'recurso inválido' está obsoleto. Use 'recurso válido'.  
  
 O recurso que você está tentando usar agora foi substituído. Atualize seu código para usar o recurso válido em vez disso.

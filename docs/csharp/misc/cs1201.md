# <a name="compiler-warning-level-1-cs1201"></a>Compilador aviso (nível 1) CS1201
O recurso 'recurso inválido' é preterido. Use o 'recurso válido'.  
  
 O recurso que você está tentando usar foi preterido. Atualize seu código para usar o recurso válido em vez disso.

# <a name="compiler-error-cs0528"></a>Erro do compilador CS0528
'interface' já está listado na lista de interfaces  
  
 Uma lista de herança da interface inclui uma duplicata. Uma [interface](../../csharp/language-reference/keywords/interface.md) só pode ser especificada uma vez na lista de herança.  
  
 O exemplo a seguir gera CS0528:  
  
```csharp  
// CS0528.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public class b : a, a   // CS0528  
   {  
      public void Main()  
      {  
      }  
   }  
}  
```

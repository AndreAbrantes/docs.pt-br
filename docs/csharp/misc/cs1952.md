## <a name="example"></a><span data-ttu-id="d7454-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d7454-107">Example</span></span>  

 <span data-ttu-id="d7454-108">O código a seguir produz CS1952:</span><span class="sxs-lookup"><span data-stu-id="d7454-108">The following code produces CS1952:</span></span>  
  
```csharp

using System;  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```

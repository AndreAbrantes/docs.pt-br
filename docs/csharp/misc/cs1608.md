# <a name="compiler-error-cs1608"></a>CS1608 de erro do compilador
O atributo Required não é permitido em tipos c#  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute>não é permitido em tipos definidos no código do c#.

<span data-ttu-id="67a9a-103">Os membros do campo somente leitura 'name' do tipo 'struct name' não podem ser atribuídos com um inicializador de objeto porque ele é de um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="67a9a-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>

-   <span data-ttu-id="67a9a-107">Inicialize a estrutura com um construtor.</span><span class="sxs-lookup"><span data-stu-id="67a9a-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="67a9a-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="67a9a-108">Example</span></span>  
 <span data-ttu-id="67a9a-109">O código a seguir gera CS1917:</span><span class="sxs-lookup"><span data-stu-id="67a9a-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```

# <a name="compiler-error-cs1917"></a><span data-ttu-id="f7fc3-102">Erro do Compilador CS1917</span><span class="sxs-lookup"><span data-stu-id="f7fc3-102">Compiler Error CS1917</span></span>

<span data-ttu-id="f7fc3-103">Os membros do campo somente leitura ' name ' do tipo ' struct Name ' não podem ser atribuídos com um inicializador de objeto porque ele é de um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="f7fc3-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f7fc3-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="f7fc3-105">To correct this error</span></span>  
  
- <span data-ttu-id="f7fc3-106">Altere a estrutura para um tipo de classe.</span><span class="sxs-lookup"><span data-stu-id="f7fc3-106">Change the struct to a class type.</span></span>  
  
- <span data-ttu-id="f7fc3-107">Inicialize o struct com um construtor.</span><span class="sxs-lookup"><span data-stu-id="f7fc3-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f7fc3-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f7fc3-108">Example</span></span>  

 <span data-ttu-id="f7fc3-109">O código a seguir gera CS1917:</span><span class="sxs-lookup"><span data-stu-id="f7fc3-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```

# <a name="compiler-error-cs1917"></a><span data-ttu-id="628dd-102">Erro do compilador CS1917</span><span class="sxs-lookup"><span data-stu-id="628dd-102">Compiler Error CS1917</span></span>
<span data-ttu-id="628dd-103">Membros do campo somente leitura 'name' do tipo 'nome do struct' não podem ser atribuídos com um inicializador de objeto porque ele é de um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="628dd-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="628dd-104">Campos somente leitura que são tipos de valor podem ser atribuídos apenas em um construtor.</span><span class="sxs-lookup"><span data-stu-id="628dd-104">Read-only fields that are value types can only be assigned in a constructor.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="628dd-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="628dd-105">To correct this error</span></span>  
  
- <span data-ttu-id="628dd-106">Altere a estrutura para um tipo de classe.</span><span class="sxs-lookup"><span data-stu-id="628dd-106">Change the struct to a class type.</span></span>  
  
- <span data-ttu-id="628dd-107">Inicialize a estrutura com um construtor.</span><span class="sxs-lookup"><span data-stu-id="628dd-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="628dd-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="628dd-108">Example</span></span>  
 <span data-ttu-id="628dd-109">O código a seguir gera CS1917:</span><span class="sxs-lookup"><span data-stu-id="628dd-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```

<span data-ttu-id="af80d-103">A propriedade ou o indexador 'propriedade/indexador' não pode ser usado neste contexto porque o acessador get é inacessível</span><span class="sxs-lookup"><span data-stu-id="af80d-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="af80d-104">Esse erro ocorre quando você tentar acessar um inacessível `get` acessador.</span><span class="sxs-lookup"><span data-stu-id="af80d-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="af80d-105">Para resolver esse erro, aumente a acessibilidade do acessador ou alterar o local de chamada.</span><span class="sxs-lookup"><span data-stu-id="af80d-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="af80d-106">Para obter mais informações, consulte [acessibilidade ao acessador](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) e [propriedades](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="af80d-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="af80d-107">O exemplo a seguir gera CS0271:</span><span class="sxs-lookup"><span data-stu-id="af80d-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```

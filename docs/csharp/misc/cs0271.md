<span data-ttu-id="e7571-104">A propriedade ou o indexador ' Property/indexer ' não pode ser usado neste contexto porque o acessador get é inacessível</span><span class="sxs-lookup"><span data-stu-id="e7571-104">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="e7571-105">Esse erro ocorre quando você tenta acessar um `get` acessador inacessível.</span><span class="sxs-lookup"><span data-stu-id="e7571-105">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="e7571-106">Para resolver esse erro, aumente a acessibilidade do acessador ou altere o local de chamada.</span><span class="sxs-lookup"><span data-stu-id="e7571-106">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="e7571-107">Para obter mais informações, consulte acessibilidade e [Propriedades](../programming-guide/classes-and-structs/properties.md)do [acessador](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) .</span><span class="sxs-lookup"><span data-stu-id="e7571-107">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="e7571-108">O exemplo a seguir gera CS0271:</span><span class="sxs-lookup"><span data-stu-id="e7571-108">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```

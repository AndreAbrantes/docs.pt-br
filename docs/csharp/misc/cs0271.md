<span data-ttu-id="e2be2-103">A propriedade ou o indexador ' Property/indexer ' não pode ser usado neste contexto porque o acessador get é inacessível</span><span class="sxs-lookup"><span data-stu-id="e2be2-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="e2be2-104">Esse erro ocorre quando você tenta acessar um acessador `get` inacessível.</span><span class="sxs-lookup"><span data-stu-id="e2be2-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="e2be2-105">Para resolver esse erro, aumente a acessibilidade do acessador ou altere o local de chamada.</span><span class="sxs-lookup"><span data-stu-id="e2be2-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="e2be2-106">Para obter mais informações, consulte [acessibilidade](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) e [Propriedades](../programming-guide/classes-and-structs/properties.md)do acessador.</span><span class="sxs-lookup"><span data-stu-id="e2be2-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="e2be2-107">O exemplo a seguir gera CS0271:</span><span class="sxs-lookup"><span data-stu-id="e2be2-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```

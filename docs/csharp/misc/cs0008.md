# <a name="compiler-error-cs0008"></a>CS0008 de erro do compilador
Erro inesperado ao ler metadados do arquivo 'file', 'description'  
  
 Uma DLL foi aberta com êxito para recuperar metadados, mas está corrompida, que não foi possível ler dados dele. Para obter mais informações, consulte [/reference (opções do compilador C#)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).

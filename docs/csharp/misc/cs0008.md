# <a name="compiler-error-cs0008"></a>Erro do compilador CS0008
Erro inesperado ao ler metadados do arquivo 'file' — 'Descrição'  
  
 Uma DLL foi aberta com êxito para recuperar metadados, mas está corrompida, de modo que os dados não foi possível ler a partir dele. Para obter mais informações, consulte [/reference (opções do compilador C#)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).

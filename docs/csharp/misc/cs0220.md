# <a name="compiler-error-cs0220"></a><span data-ttu-id="00fc7-102">CS0220 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="00fc7-102">Compiler Error CS0220</span></span>
<span data-ttu-id="00fc7-103">A operação estoura o tempo de compilação no modo de ativação</span><span class="sxs-lookup"><span data-stu-id="00fc7-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="00fc7-104">Foi detectada uma operação por [check](../../csharp/language-reference/keywords/checked.md), que é o padrão e resultou de uma perda de dados.</span><span class="sxs-lookup"><span data-stu-id="00fc7-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="00fc7-105">O corrija as entradas para a atribuição ou use [desmarcada](../../csharp/language-reference/keywords/unchecked.md) para resolver esse erro.</span><span class="sxs-lookup"><span data-stu-id="00fc7-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="00fc7-106">Para obter mais informações, consulte [marcado e desmarcado](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="00fc7-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="00fc7-107">O exemplo a seguir gera CS0220:</span><span class="sxs-lookup"><span data-stu-id="00fc7-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```

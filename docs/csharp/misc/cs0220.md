# <a name="compiler-error-cs0220"></a><span data-ttu-id="06295-102">Erro do compilador CS0220</span><span class="sxs-lookup"><span data-stu-id="06295-102">Compiler Error CS0220</span></span>
<span data-ttu-id="06295-103">A operação estoura o tempo de compilação no modo de ativação</span><span class="sxs-lookup"><span data-stu-id="06295-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="06295-104">Uma operação foi detectada [marcada](../language-reference/keywords/checked.md), que é o padrão, e uma perda de dados resultou.</span><span class="sxs-lookup"><span data-stu-id="06295-104">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="06295-105">Corrija as entradas para a atribuição ou use [desmarcadas](../language-reference/keywords/unchecked.md) para resolver esse erro.</span><span class="sxs-lookup"><span data-stu-id="06295-105">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="06295-106">Para obter mais informações, consulte [marcada e desmarcada](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="06295-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="06295-107">O exemplo a seguir gera CS0220:</span><span class="sxs-lookup"><span data-stu-id="06295-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```

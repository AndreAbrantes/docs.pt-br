 <span data-ttu-id="0e43f-104">Um evento em uma interface não pode ter um inicializador.</span><span class="sxs-lookup"><span data-stu-id="0e43f-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="0e43f-105">Para obter mais informações, consulte [Interfaces](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="0e43f-105">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>

{  
   event MyDelegate d = new MyDelegate(M.f);

   {  
   }

   {  
   }  
}

 <span data-ttu-id="31dfe-104">Um evento em uma interface não pode ter um inicializador.</span><span class="sxs-lookup"><span data-stu-id="31dfe-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="31dfe-105">Para obter mais informações, consulte [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="31dfe-105">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>

{  
   event MyDelegate d = new MyDelegate(M.f);

   }

   }  
}

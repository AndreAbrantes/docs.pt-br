## <a name="example"></a><span data-ttu-id="edaca-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="edaca-107">Example</span></span>

```csharp  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }

```

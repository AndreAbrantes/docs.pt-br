# <a name="compiler-error-cs0629"></a><span data-ttu-id="bc517-102">Erro do compilador CS0629</span><span class="sxs-lookup"><span data-stu-id="bc517-102">Compiler error CS0629</span></span>

<span data-ttu-id="bc517-103">O membro condicional 'membro' não pode implementar o membro da interface 'membro da classe base' no tipo 'Nome de tipo'</span><span class="sxs-lookup"><span data-stu-id="bc517-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="bc517-104">O [atributo Conditional](../language-reference/attributes/general.md#conditional-attribute) não pode ser usado na implementação de uma interface.</span><span class="sxs-lookup"><span data-stu-id="bc517-104">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="bc517-105">A seguinte amostra gera CS0629:</span><span class="sxs-lookup"><span data-stu-id="bc517-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

# <a name="compiler-error-cs0127"></a><span data-ttu-id="4106f-102">CS0127 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="4106f-102">Compiler Error CS0127</span></span>
<span data-ttu-id="4106f-103">Como 'function' retorna void, uma palavra-chave return não deve ser seguida por uma expressão de objeto</span><span class="sxs-lookup"><span data-stu-id="4106f-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="4106f-104">Um método com um [void](../../csharp/language-reference/keywords/void.md) tipo não pode retornar um valor de retorno.</span><span class="sxs-lookup"><span data-stu-id="4106f-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="4106f-105">Para saber mais, veja [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="4106f-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="4106f-106">O exemplo a seguir gera CS0127:</span><span class="sxs-lookup"><span data-stu-id="4106f-106">The following sample generates CS0127:</span></span>  
  
```  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```

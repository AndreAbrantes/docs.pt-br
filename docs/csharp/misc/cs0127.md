# <a name="compiler-error-cs0127"></a><span data-ttu-id="5582f-103">Erro do Compilador CS0127</span><span class="sxs-lookup"><span data-stu-id="5582f-103">Compiler Error CS0127</span></span>
<span data-ttu-id="5582f-104">Como ' function ' retorna void, uma palavra-chave Return não deve ser seguida por uma expressão de objeto</span><span class="sxs-lookup"><span data-stu-id="5582f-104">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="5582f-105">Um método com um tipo de retorno [void](../language-reference/builtin-types/void.md) não pode retornar um valor.</span><span class="sxs-lookup"><span data-stu-id="5582f-105">A method with a [void](../language-reference/builtin-types/void.md) return type cannot return a value.</span></span> <span data-ttu-id="5582f-106">Para saber mais, veja [Métodos](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="5582f-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="5582f-107">O exemplo a seguir gera CS0127:</span><span class="sxs-lookup"><span data-stu-id="5582f-107">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```

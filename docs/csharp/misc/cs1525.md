# <a name="compiler-error-cs1525"></a><span data-ttu-id="1ccfa-102">CS1525 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="1ccfa-102">Compiler Error CS1525</span></span>
<span data-ttu-id="1ccfa-103">Termo de expressão inválido 'character'</span><span class="sxs-lookup"><span data-stu-id="1ccfa-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="1ccfa-104">O compilador detectou um caractere inválido em uma expressão.</span><span class="sxs-lookup"><span data-stu-id="1ccfa-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="1ccfa-105">O exemplo a seguir gera CS1525:</span><span class="sxs-lookup"><span data-stu-id="1ccfa-105">The following sample generates CS1525:</span></span>  
  
```  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="1ccfa-106">Um rótulo vazio também pode gerar CS1525, como no exemplo a seguir:</span><span class="sxs-lookup"><span data-stu-id="1ccfa-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```

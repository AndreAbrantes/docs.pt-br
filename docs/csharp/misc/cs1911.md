# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="523ee-102">Aviso do compilador (nível 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="523ee-102">Compiler Warning (level 1) CS1911</span></span>

<span data-ttu-id="523ee-103">O acesso ao membro ' name ' por meio de uma palavra-chave ' base ' de um método anônimo, expressão lambda, expressão de consulta ou iterador resulta em código não verificável.</span><span class="sxs-lookup"><span data-stu-id="523ee-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="523ee-104">Considere mover o acesso para um método auxiliar no tipo recipiente.</span><span class="sxs-lookup"><span data-stu-id="523ee-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="523ee-105">Chamar funções virtuais com a `base` palavra-chave dentro do corpo do método de um iterador ou métodos anônimos resultará em código não verificável.</span><span class="sxs-lookup"><span data-stu-id="523ee-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="523ee-106">O código não verificável não será executado em um ambiente de confiança parcial.</span><span class="sxs-lookup"><span data-stu-id="523ee-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="523ee-107">Uma resolução para CS1911 é mover a chamada de função virtual para uma função auxiliar.</span><span class="sxs-lookup"><span data-stu-id="523ee-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="523ee-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="523ee-108">Example</span></span>  

 <span data-ttu-id="523ee-109">O exemplo a seguir gera CS1911.</span><span class="sxs-lookup"><span data-stu-id="523ee-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```

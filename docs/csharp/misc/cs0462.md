# <a name="compiler-error-cs0462"></a><span data-ttu-id="75d9f-103">Erro do Compilador CS0462</span><span class="sxs-lookup"><span data-stu-id="75d9f-103">Compiler Error CS0462</span></span>

<span data-ttu-id="75d9f-104">Os membros herdados ' member1 ' e ' membro2 ' têm a mesma assinatura no tipo ' type ', portanto, eles não podem ser substituídos</span><span class="sxs-lookup"><span data-stu-id="75d9f-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="75d9f-105">Esse erro ocorre com a introdução de genéricos.</span><span class="sxs-lookup"><span data-stu-id="75d9f-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="75d9f-106">Normalmente, você não pode ter duas versões de um método em uma classe com a mesma assinatura.</span><span class="sxs-lookup"><span data-stu-id="75d9f-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="75d9f-107">Mas com os genéricos, você pode especificar um método genérico que poderá duplicar outro método se ele for instanciado com um tipo específico.</span><span class="sxs-lookup"><span data-stu-id="75d9f-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="75d9f-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="75d9f-108">Example</span></span>  

 <span data-ttu-id="75d9f-109">Quando `C<int>` é instanciada, duas versões do método `F` são criadas com a mesma assinatura, de modo que a substituição na classe `D` não pode decidir para qual delas aplicar a substituição.</span><span class="sxs-lookup"><span data-stu-id="75d9f-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="75d9f-110">O exemplo a seguir gera CS0462.</span><span class="sxs-lookup"><span data-stu-id="75d9f-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```

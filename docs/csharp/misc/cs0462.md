# <a name="compiler-error-cs0462"></a><span data-ttu-id="1c430-102">Erro do Compilador CS0462</span><span class="sxs-lookup"><span data-stu-id="1c430-102">Compiler Error CS0462</span></span>
<span data-ttu-id="1c430-103">Os membros herdados 'member1' e 'member2' têm a mesma assinatura no tipo 'tipo', portanto não podem ser substituídos</span><span class="sxs-lookup"><span data-stu-id="1c430-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="1c430-104">Esse erro surge com a introdução de genéricos.</span><span class="sxs-lookup"><span data-stu-id="1c430-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="1c430-105">Normalmente, você não pode ter duas versões de um método em uma classe com a mesma assinatura.</span><span class="sxs-lookup"><span data-stu-id="1c430-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="1c430-106">Mas com genéricos, você pode especificar um método genérico que pode duplicar outro método se ele for instanciado com um tipo específico.</span><span class="sxs-lookup"><span data-stu-id="1c430-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1c430-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="1c430-107">Example</span></span>  
 <span data-ttu-id="1c430-108">Quando `C<int>` é instanciado, duas `F` versões do método são criadas com `D` a mesma assinatura, de modo que a substituição na classe não pode decidir a qual aplicar a substituição.</span><span class="sxs-lookup"><span data-stu-id="1c430-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="1c430-109">A seguinte amostra gera CS0462.</span><span class="sxs-lookup"><span data-stu-id="1c430-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```

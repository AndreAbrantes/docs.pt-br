<span data-ttu-id="955d8-103">O melhor método sobrecarregado correspondência 'method' para o elemento do inicializador de coleção não pode ser usado.</span><span class="sxs-lookup"><span data-stu-id="955d8-103">The best overloaded method match 'method' for the collection initializer element cannot be used.</span></span> <span data-ttu-id="955d8-104">Métodos de 'Add' de inicializador de coleção não podem ter ref ou out parâmetros.</span><span class="sxs-lookup"><span data-stu-id="955d8-104">Collection initializer 'Add' methods cannot have ref or out parameters.</span></span>  
  
 <span data-ttu-id="955d8-105">Para uma classe de coleção seja inicializada usando um inicializador de coleção, a classe deve ter um `public``Add` método que não é um `ref` ou `out` parâmetro.</span><span class="sxs-lookup"><span data-stu-id="955d8-105">For a collection class to be initialized by using a collection initializer, the class must have a `public``Add` method that is not a `ref` or `out` parameter.</span></span>

## <a name="example"></a><span data-ttu-id="955d8-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="955d8-110">Example</span></span>  
 <span data-ttu-id="955d8-111">O exemplo a seguir produz CS1954 porque a sobrecarga disponível somente do `Add` lista `MyList` tem um `ref` parâmetro.</span><span class="sxs-lookup"><span data-stu-id="955d8-111">The following example produces CS1954 because the only available overload of the `Add` list in `MyList` has a `ref` parameter.</span></span>  
  
```csharp  
// cs1954.cs  
using System.Collections.Generic;  
class MyList<T> : IEnumerable<T>  
{  
    List<T> _list;  
    public void Add(ref T item)  
    {  
        _list.Add(item);  
    }  
  
    public System.Collections.Generic.IEnumerator<T> GetEnumerator()  
    {  
        int index = 0;  
        T current = _list[index];  
        while (current != null)  
        {  
            yield return _list[index++];  
        }  
    }  
  
    System.Collections.IEnumerator System.Collections.IEnumerable.GetEnumerator()  
    {  
        return GetEnumerator();  
    }  
}  
  
public class MyClass  
{  
    public string tree { get; set; }  
}  
class Program  
{  
    static void Main(string[] args)  
    {  
        MyList<MyClass> myList = new MyList<MyClass> { new MyClass { tree = "maple" } }; // CS1954  
    }  
}  
```

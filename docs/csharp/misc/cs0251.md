# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="9506c-102">Compilador CS0251 de aviso (nível 2)</span><span class="sxs-lookup"><span data-stu-id="9506c-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="9506c-103">Indexando uma matriz com um índice negativo (matriz de índices sempre começam em zero)</span><span class="sxs-lookup"><span data-stu-id="9506c-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="9506c-104">Não use um número negativo para o índice em uma matriz.</span><span class="sxs-lookup"><span data-stu-id="9506c-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="9506c-105">O exemplo a seguir gera CS0251:</span><span class="sxs-lookup"><span data-stu-id="9506c-105">The following sample generates CS0251:</span></span>  
  
```  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```

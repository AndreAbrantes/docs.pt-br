## <a name="example"></a><span data-ttu-id="8140c-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8140c-109">Example</span></span>  
 <span data-ttu-id="8140c-110">O código a seguir gera CS0314:</span><span class="sxs-lookup"><span data-stu-id="8140c-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```

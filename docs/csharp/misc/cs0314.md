## <a name="example"></a><span data-ttu-id="4fc3c-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="4fc3c-108">Example</span></span>  
 <span data-ttu-id="4fc3c-109">O código a seguir gera CS0314:</span><span class="sxs-lookup"><span data-stu-id="4fc3c-109">The following code generates CS0314:</span></span>  
  
```  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.

{ }  
```

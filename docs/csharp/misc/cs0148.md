# <a name="compiler-error-cs0148"></a>CS0148 de erro do compilador
O representante 'representante' não tem um construtor válido  
  
 Você importou e usado um programa gerenciado (um que usa o .NET Framework common language runtime) que foi criado com outro compilador. Que o compilador permitido um mal formados [delegar](../../csharp/language-reference/keywords/delegate.md) construtor. Para obter mais informações, consulte [Delegados](../../csharp/programming-guide/delegates/index.md).

# <a name="compiler-error-cs0583"></a>Erro do Compilador CS0583

Erro do compilador interno. ocorreu um erro interno no compilador. Para contornar esse problema, tente simplificar ou alterar o programa perto dos locais listados abaixo. Os locais no topo da lista estão mais próximos do ponto em que ocorreu o erro interno. Erros como este podem ser relatados à Microsoft usando a opção /errorreport.

 Crie um arquivo de relatório de bugs com a opção '/bugreport' e envie o relatório para o contato de relatório de problemas atribuído.

 Use a opção [-bugreport](../language-reference/compiler-options/bugreport-compiler-option.md) compilour para capturar o problema.

> [!NOTE]
> Este erro do compilador não é mais usado em Roslyn.

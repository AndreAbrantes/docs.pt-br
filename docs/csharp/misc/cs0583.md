# <a name="compiler-error-cs0583"></a>Erro do compilador CS0583

Erro interno do compilador. Ocorreu um erro interno no compilador. Para contornar esse problema, tente simplificar ou alterar o programa próximo aos locais listados abaixo. Os locais na parte superior da lista estão mais próximos do ponto em que ocorreu o erro interno. Erros como esse podem ser relatados à Microsoft usando a opção/errorreport.

 Crie um arquivo de relatório de bugs com a opção '/bugreport ' e envie o relatório para o contato do relatório de problemas atribuído.

 Use a opção de compilador [-bugreport](../language-reference/compiler-options/bugreport-compiler-option.md) para capturar o problema.
 
> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

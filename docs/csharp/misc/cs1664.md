# <a name="compiler-error-cs1664"></a>Erro do compilador CS1664
Buffer de tamanho fixo de comprimento 'length' e tipo 'type' é muito grande  
  
 O tamanho máximo de um buffer de tamanho fixo (conforme determinado pelo comprimento multiplicado pelo tamanho do elemento) é 2 ^ 31 = 268435455.

# <a name="compiler-error-cs0404"></a>CS0404 de erro do compilador
' <' inesperado: atributos não podem ser genéricos  
  
 Parâmetros de tipo genérico não são permitidos em atributos. Remova o parâmetro de tipo e colchetes angulares.  
  
 O exemplo a seguir gera CS0404:  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```

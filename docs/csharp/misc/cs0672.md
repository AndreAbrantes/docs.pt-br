<span data-ttu-id="1ef24-103">Membro 'member1' substitui o membro obsoleto ' membro2.</span><span class="sxs-lookup"><span data-stu-id="1ef24-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="1ef24-104">Adicionar o obsoleto atributo 'member1'</span><span class="sxs-lookup"><span data-stu-id="1ef24-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="1ef24-105">O compilador encontrou uma `override` para um método marcado como `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="1ef24-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="1ef24-106">No entanto, o método de substituição não foi marcado como obsoleto.</span><span class="sxs-lookup"><span data-stu-id="1ef24-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="1ef24-107">Substituindo o método ainda irá gerar [CS0612](../../csharp/misc/cs0612.md), se a chamada.</span><span class="sxs-lookup"><span data-stu-id="1ef24-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="1ef24-108">Revise suas declarações de método e indicar explicitamente se um método (e todos os seus substituições) devem ser marcados `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="1ef24-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="1ef24-109">O exemplo a seguir gera CS0672:</span><span class="sxs-lookup"><span data-stu-id="1ef24-109">The following sample generates CS0672:</span></span>  
  
```  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```

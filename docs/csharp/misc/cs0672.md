<span data-ttu-id="0be6f-103">O membro ' member1 ' substitui o membro obsoleto ' membro2.</span><span class="sxs-lookup"><span data-stu-id="0be6f-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="0be6f-104">Adicionar o atributo obsoleto a ' member1 '</span><span class="sxs-lookup"><span data-stu-id="0be6f-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="0be6f-105">O compilador encontrou um `override` para um método marcado como `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="0be6f-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="0be6f-106">No entanto, o método de substituição não estava marcado como obsoleto.</span><span class="sxs-lookup"><span data-stu-id="0be6f-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="0be6f-107">O método de substituição ainda gerará [CS0612](./cs0612.md), se chamado.</span><span class="sxs-lookup"><span data-stu-id="0be6f-107">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="0be6f-108">Revise suas declarações de método e indique explicitamente se um método (e todas as suas substituições `obsolete`) deve ser marcado.</span><span class="sxs-lookup"><span data-stu-id="0be6f-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="0be6f-109">O exemplo a seguir gera CS0672:</span><span class="sxs-lookup"><span data-stu-id="0be6f-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```

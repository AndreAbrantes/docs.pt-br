# <a name="compiler-warning-level-1-cs2023"></a>Aviso do compilador (nível 1) CS2023
Ignorando a opção /noconfig porque foi especificada em um arquivo de resposta  
  
 O [/noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) opção de compilador foi especificada em um arquivo de resposta, que não é permitido.  
  
 Este aviso não pode ser suprimido pelo [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opção.

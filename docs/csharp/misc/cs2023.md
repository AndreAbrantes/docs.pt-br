# <a name="compiler-warning-level-1-cs2023"></a>Compilador CS2023 de aviso (nível 1)
Ignorando a opção /noconfig porque ela foi especificada em um arquivo de resposta  
  
 O [/noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) opção de compilador foi especificada em um arquivo de resposta, que não é permitido.  
  
 Esse aviso não pode ser suprimido pelo [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) opção.

# <a name="compiler-error-cs1940"></a><span data-ttu-id="38731-102">CS1940 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="38731-102">Compiler Error CS1940</span></span>
<span data-ttu-id="38731-103">Várias implementações do padrão de consulta foram encontradas para o tipo de origem 'type'.</span><span class="sxs-lookup"><span data-stu-id="38731-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="38731-104">Chamada ambígua para 'method'.</span><span class="sxs-lookup"><span data-stu-id="38731-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="38731-105">Esse erro é gerado quando várias implementações de um método de consulta são definidas e o compilador não pode desfazer a ambiguidade de qual delas é melhor usar para a consulta.</span><span class="sxs-lookup"><span data-stu-id="38731-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="38731-106">No exemplo a seguir, ambas as versões do `Select` tem a mesma assinatura, porque ambos aceitarem uma `int` como um parâmetro de entrada e ter `int` como um valor de retorno.</span><span class="sxs-lookup"><span data-stu-id="38731-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

1.  <span data-ttu-id="38731-108">Forneça somente uma implementação para cada método.</span><span class="sxs-lookup"><span data-stu-id="38731-108">Provide only one implementation for each method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="38731-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="38731-109">Example</span></span>  
 <span data-ttu-id="38731-110">O código a seguir gera CS1940:</span><span class="sxs-lookup"><span data-stu-id="38731-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```

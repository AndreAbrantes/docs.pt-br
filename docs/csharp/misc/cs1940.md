# <a name="compiler-error-cs1940"></a><span data-ttu-id="5ae72-102">Erro do compilador CS1940</span><span class="sxs-lookup"><span data-stu-id="5ae72-102">Compiler Error CS1940</span></span>
<span data-ttu-id="5ae72-103">Várias implementações do padrão de consulta foram encontradas para o tipo de origem ' type '.</span><span class="sxs-lookup"><span data-stu-id="5ae72-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="5ae72-104">Chamada ambígua para ' method '.</span><span class="sxs-lookup"><span data-stu-id="5ae72-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="5ae72-105">Esse erro é gerado quando várias implementações de um método de consulta são definidas e o compilador não pode desambiguar qual é a melhor maneira de usar para a consulta.</span><span class="sxs-lookup"><span data-stu-id="5ae72-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="5ae72-106">No exemplo a seguir, ambas as versões `Select` do têm a mesma assinatura, porque ambas aceitam `int` uma como um parâmetro de entrada `int` e têm como um valor de retorno.</span><span class="sxs-lookup"><span data-stu-id="5ae72-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="5ae72-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5ae72-109">Example</span></span>  
 <span data-ttu-id="5ae72-110">O código a seguir gera CS1940:</span><span class="sxs-lookup"><span data-stu-id="5ae72-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```

# <a name="compiler-error-cs0556"></a>CS0556 de erro do compilador
A conversão definida pelo usuário deve ser convertida a partir de ou em um tipo de delimitador  
  
 Uma rotina de conversão definida pelo usuário deve converter para ou da classe que contém a rotina.  
  
 O exemplo a seguir gera CS0556:  
  
```  
// CS0556.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         public static implicit operator int(byte aa)   // CS0556  
         // try the following line instead  
         // public static implicit operator int(iii aa)  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```

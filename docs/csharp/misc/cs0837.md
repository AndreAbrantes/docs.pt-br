## <a name="example"></a><span data-ttu-id="f326d-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f326d-110">Example</span></span>

<span data-ttu-id="f326d-111">O exemplo a seguir gera CS0837:</span><span class="sxs-lookup"><span data-stu-id="f326d-111">The following example generates CS0837:</span></span>

```csharp
// cs0837.cs
namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837
            return 1;
        }
    }
}
```

# <a name="compiler-error-cs0837"></a><span data-ttu-id="959f1-102">CS0837 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="959f1-102">Compiler Error CS0837</span></span>
<span data-ttu-id="959f1-103">O primeiro operando de um "é" ou "as" operador não pode ser uma expressão lambda ou um método anônimo.</span><span class="sxs-lookup"><span data-stu-id="959f1-103">The first operand of an "is" or "as" operator may not be a lambda expression or anonymous method.</span></span>  
  
 <span data-ttu-id="959f1-104">Métodos anônimos e expressões lambda não podem ser usados no lado esquerdo do [é](../../csharp/language-reference/keywords/is.md) ou [como](../../csharp/language-reference/keywords/as.md).</span><span class="sxs-lookup"><span data-stu-id="959f1-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../../csharp/language-reference/keywords/is.md) or [as](../../csharp/language-reference/keywords/as.md).</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="959f1-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="959f1-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="959f1-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="959f1-109">Example</span></span>  
 <span data-ttu-id="959f1-110">O exemplo a seguir gera CS0837:</span><span class="sxs-lookup"><span data-stu-id="959f1-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```

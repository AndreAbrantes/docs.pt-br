# <a name="compiler-error-cs2016"></a>Erro do compilador CS2016
A página de código ' CodePage ' é inválida ou não está instalada  
  
 A opção de compilador [/CodePage](../language-reference/compiler-options/codepage-compiler-option.md) recebeu um valor inválido.  
  
 O exemplo a seguir gera CS2016:  
  
```csharp  
// CS2016.cs  
// compile with: /codepage:x  
// CS2016 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```

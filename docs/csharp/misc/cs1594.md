# <a name="compiler-error-cs1594"></a><span data-ttu-id="b99c9-102">CS1594 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="b99c9-102">Compiler Error CS1594</span></span>
<span data-ttu-id="b99c9-103">Delegado 'delegado' tem alguns argumentos inválidos</span><span class="sxs-lookup"><span data-stu-id="b99c9-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="b99c9-104">O tipo de um argumento passado para um [delegar](../../csharp/language-reference/keywords/delegate.md) invocação não concordar com o tipo do parâmetro na declaração do delegado.</span><span class="sxs-lookup"><span data-stu-id="b99c9-104">The type of an argument passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="b99c9-105">O exemplo a seguir gera CS1594:</span><span class="sxs-lookup"><span data-stu-id="b99c9-105">The following sample generates CS1594:</span></span>  
  
```  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```

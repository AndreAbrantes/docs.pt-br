# <a name="compiler-error-cs1594"></a><span data-ttu-id="03cf8-103">Erro do Compilador CS1594</span><span class="sxs-lookup"><span data-stu-id="03cf8-103">Compiler Error CS1594</span></span>

<span data-ttu-id="03cf8-104">Delegate "delegate" tem alguns argumentos inválidos</span><span class="sxs-lookup"><span data-stu-id="03cf8-104">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="03cf8-105">O tipo de um argumento passado para uma invocação [delegate](../language-reference/builtin-types/reference-types.md) não concorda com o tipo do parâmetro na declaração delegate.</span><span class="sxs-lookup"><span data-stu-id="03cf8-105">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="03cf8-106">O exemplo a seguir gera CS1594:</span><span class="sxs-lookup"><span data-stu-id="03cf8-106">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```

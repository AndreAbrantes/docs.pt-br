# <a name="compiler-error-cs0557"></a><span data-ttu-id="4077e-103">Erro do Compilador CS0557</span><span class="sxs-lookup"><span data-stu-id="4077e-103">Compiler Error CS0557</span></span>
<span data-ttu-id="4077e-104">Conversão definida pelo usuário duplicada no tipo ' class '</span><span class="sxs-lookup"><span data-stu-id="4077e-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="4077e-105">Rotinas de conversão duplicadas não são permitidas em uma classe.</span><span class="sxs-lookup"><span data-stu-id="4077e-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="4077e-106">O exemplo a seguir gera CS0557:</span><span class="sxs-lookup"><span data-stu-id="4077e-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```

 <span data-ttu-id="c868d-104">Rotinas de conversão duplicados não são permitidas em uma classe.</span><span class="sxs-lookup"><span data-stu-id="c868d-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="c868d-105">O exemplo a seguir gera CS0557:</span><span class="sxs-lookup"><span data-stu-id="c868d-105">The following example generates CS0557:</span></span>  
  
```  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```

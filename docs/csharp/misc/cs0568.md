# <a name="compiler-error-cs0568"></a><span data-ttu-id="da44f-103">Erro do Compilador CS0568</span><span class="sxs-lookup"><span data-stu-id="da44f-103">Compiler Error CS0568</span></span>

<span data-ttu-id="da44f-104">Structs não podem conter construtores explícitos sem parâmetros</span><span class="sxs-lookup"><span data-stu-id="da44f-104">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="da44f-105">Cada [struct](../language-reference/builtin-types/struct.md) já tem um construtor sem parâmetros que inicializa o objeto como zero.</span><span class="sxs-lookup"><span data-stu-id="da44f-105">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="da44f-106">Portanto, os construtores que você pode criar para um struct devem usar um ou mais parâmetros.</span><span class="sxs-lookup"><span data-stu-id="da44f-106">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="da44f-107">O exemplo a seguir gera CS0568:</span><span class="sxs-lookup"><span data-stu-id="da44f-107">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```

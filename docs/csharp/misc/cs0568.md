# <a name="compiler-error-cs0568"></a><span data-ttu-id="e35ea-102">CS0568 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="e35ea-102">Compiler Error CS0568</span></span>
<span data-ttu-id="e35ea-103">Structs não podem conter construtores explícitos sem parâmetro</span><span class="sxs-lookup"><span data-stu-id="e35ea-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="e35ea-104">Cada [struct](../../csharp/language-reference/keywords/struct.md) já tem um construtor padrão que inicializa o objeto como zero.</span><span class="sxs-lookup"><span data-stu-id="e35ea-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="e35ea-105">Portanto, os construtores que você pode criar para um struct devem levar um ou mais parâmetros.</span><span class="sxs-lookup"><span data-stu-id="e35ea-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="e35ea-106">O exemplo a seguir gera CS0568:</span><span class="sxs-lookup"><span data-stu-id="e35ea-106">The following sample generates CS0568:</span></span>  
  
```  
// CS0568.cs

{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```

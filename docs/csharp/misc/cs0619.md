# <a name="compiler-error-cs0619"></a>Erro do compilador CS0619
'member' obsoleto: 'text'  
  
 Um membro de classe foi marcado com [obsoleto] ((... /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) atributo, de modo que um erro será emitido quando o membro de classe é referenciado.  
  
 O exemplo a seguir gera CS0619:  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```

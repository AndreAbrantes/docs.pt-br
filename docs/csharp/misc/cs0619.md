# <a name="compiler-error-cs0619"></a>Erro do Compilador CS0619

'member' obsoleto: 'text'

Um membro de classe foi marcado com o atributo [obsoleto](../language-reference/attributes/general.md#obsolete-attribute) , de modo que um erro ser√° emitido quando o membro da classe for referenciado.

## <a name="example"></a>Exemplo

O exemplo a seguir gera CS0619:  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```

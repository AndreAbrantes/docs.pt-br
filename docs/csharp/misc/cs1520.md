# <a name="compiler-error-cs1520"></a><span data-ttu-id="abbfc-102">Erro do compilador CS1520</span><span class="sxs-lookup"><span data-stu-id="abbfc-102">Compiler Error CS1520</span></span>
<span data-ttu-id="abbfc-103">O método deve ter um tipo de retorno</span><span class="sxs-lookup"><span data-stu-id="abbfc-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="abbfc-104">Um método declarado em uma classe, estrutura ou interface deve ter um tipo de retorno explícito.</span><span class="sxs-lookup"><span data-stu-id="abbfc-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="abbfc-105">No exemplo a seguir, o método Square tem um valor de retorno de [String](../language-reference/builtin-types/reference-types.md):</span><span class="sxs-lookup"><span data-stu-id="abbfc-105">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="abbfc-106">O exemplo a seguir gera CS1520:</span><span class="sxs-lookup"><span data-stu-id="abbfc-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="abbfc-107">Como alternativa, esse erro pode ser encontrado quando o caso do nome de um Construtor difere do da declaração Class ou struct, como no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="abbfc-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="abbfc-108">Como o nome não é exatamente o mesmo que o nome da classe, o compilador o interpreta como um método regular, não um construtor e produz o erro:</span><span class="sxs-lookup"><span data-stu-id="abbfc-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```

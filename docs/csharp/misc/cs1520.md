# <a name="compiler-error-cs1520"></a><span data-ttu-id="61e37-102">Erro do compilador CS1520</span><span class="sxs-lookup"><span data-stu-id="61e37-102">Compiler Error CS1520</span></span>
<span data-ttu-id="61e37-103">Método deve ter um tipo de retorno</span><span class="sxs-lookup"><span data-stu-id="61e37-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="61e37-104">Um método que é declarado em uma classe, struct ou interface deve ter um tipo de retorno explícito.</span><span class="sxs-lookup"><span data-stu-id="61e37-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="61e37-105">No exemplo a seguir, o método quadrado tem um valor de retorno [cadeia de caracteres](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="61e37-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="61e37-106">O exemplo a seguir gera CS1520:</span><span class="sxs-lookup"><span data-stu-id="61e37-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="61e37-107">Como alternativa, esse erro pode ser encontrado quando o caso do nome de um construtor é diferente da declaração de classe ou struct, como no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="61e37-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="61e37-108">Porque o nome não é exatamente o mesmo que o nome de classe, o compilador interpreta como um método regular, e não um construtor e produz o erro:</span><span class="sxs-lookup"><span data-stu-id="61e37-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```

 <span data-ttu-id="8f29c-105">A instrução yield não é permitida de dentro do corpo de uma cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="8f29c-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="8f29c-106">Para evitar esse erro, mova a instrução yield para fora do corpo da cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="8f29c-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="8f29c-107">O exemplo a seguir gera CS1631:</span><span class="sxs-lookup"><span data-stu-id="8f29c-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```

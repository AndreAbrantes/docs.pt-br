<span data-ttu-id="c7203-103">Não pode produzir um valor no corpo de uma cláusula de captura</span><span class="sxs-lookup"><span data-stu-id="c7203-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="c7203-104">A declaração de rendimento não é permitida de dentro do corpo de uma cláusula de captura.</span><span class="sxs-lookup"><span data-stu-id="c7203-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="c7203-105">Para evitar esse erro, mova a declaração de rendimento para fora do corpo da cláusula de captura.</span><span class="sxs-lookup"><span data-stu-id="c7203-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="c7203-106">A seguinte amostra gera CS1631:</span><span class="sxs-lookup"><span data-stu-id="c7203-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```

# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="2b736-102">Aviso do compilador (nível 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="2b736-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="2b736-103">A expressão dada é sempre do tipo fornecido ('tipo')</span><span class="sxs-lookup"><span data-stu-id="2b736-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="2b736-104">Se uma declaração condicional sempre avalia a **verdade,** então você não precisa de uma declaração condicional.</span><span class="sxs-lookup"><span data-stu-id="2b736-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="2b736-105">Este aviso ocorre quando você tenta avaliar um tipo usando o **operador is.**</span><span class="sxs-lookup"><span data-stu-id="2b736-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="2b736-106">Se a avaliação for um tipo de valor, então a verificação é desnecessária.</span><span class="sxs-lookup"><span data-stu-id="2b736-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="2b736-107">A seguinte amostra gera CS0183:</span><span class="sxs-lookup"><span data-stu-id="2b736-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```

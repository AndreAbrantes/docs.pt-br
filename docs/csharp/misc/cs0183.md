# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="a15eb-102">Compilador CS0183 de aviso (nível 1)</span><span class="sxs-lookup"><span data-stu-id="a15eb-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="a15eb-103">A expressão especificada é sempre do tipo fornecido ('type')</span><span class="sxs-lookup"><span data-stu-id="a15eb-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="a15eb-104">Se uma instrução condicional sempre avaliada como **true**, em seguida, você não precisa de uma instrução condicional.</span><span class="sxs-lookup"><span data-stu-id="a15eb-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="a15eb-105">Este aviso ocorre quando você tentar avaliar um tipo usando o **é** operador.</span><span class="sxs-lookup"><span data-stu-id="a15eb-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="a15eb-106">Se a avaliação é um tipo de valor, a verificação é desnecessária.</span><span class="sxs-lookup"><span data-stu-id="a15eb-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="a15eb-107">O exemplo a seguir gera CS0183:</span><span class="sxs-lookup"><span data-stu-id="a15eb-107">The following sample generates CS0183:</span></span>  
  
```  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```

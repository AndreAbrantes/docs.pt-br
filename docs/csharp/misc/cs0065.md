# <a name="compiler-error-cs0065"></a><span data-ttu-id="7f94d-102">Erro do compilador CS0065</span><span class="sxs-lookup"><span data-stu-id="7f94d-102">Compiler Error CS0065</span></span>

<span data-ttu-id="7f94d-103">'event': propriedade de evento deve ter acessadores add e remove</span><span class="sxs-lookup"><span data-stu-id="7f94d-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="7f94d-104">Um evento que não é um campo deve ter os dois métodos de acesso.</span><span class="sxs-lookup"><span data-stu-id="7f94d-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="7f94d-105">O exemplo a seguir gera CS0065:</span><span class="sxs-lookup"><span data-stu-id="7f94d-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```

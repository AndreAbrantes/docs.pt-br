# <a name="compiler-error-cs0564"></a>CS0564 de erro do compilador
O primeiro operando de um operador de deslocamento sobrecarregado deve ser do tipo recipiente, enquanto o do segundo deve ser int  
  
 VocÃª tentou sobrecarregar um operador de deslocamento (\< ou >>) com operandos digitados incorretamente. O primeiro operando deve ser do tipo e o segundo operando deve ser do tipo `int`.  
  
 O exemplo a seguir gera CS0564:  
  
```  
// CS0564.cs  
using System;  
class C  
{  
   public static int operator << (C c1, C c2) // CS0564  
// To correct, change second operand to int, like so:  
// public static int operator << (C c1, int c2)  
   {  
      return 0;  
   }  
   static void Main()   
   {  
   }  
}  
```

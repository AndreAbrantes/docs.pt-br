## <a name="to-correct-this-error"></a><span data-ttu-id="f498b-104">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="f498b-104">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="f498b-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f498b-107">Example</span></span>  
 <span data-ttu-id="f498b-108">O código a seguir gera CS0825 porque `var` é usado em um campo de classe:</span><span class="sxs-lookup"><span data-stu-id="f498b-108">The following code generates CS0825 because `var` is used on a class field:</span></span>  
  
```csharp

class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```

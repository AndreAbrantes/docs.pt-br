## <a name="example"></a><span data-ttu-id="61e45-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="61e45-107">Example</span></span>  
 <span data-ttu-id="61e45-108">O código a seguir gera CS0825 `var` porque é usado em um campo de classe:</span><span class="sxs-lookup"><span data-stu-id="61e45-108">The following code generates CS0825 because `var` is used on a class field:</span></span>  
  
```csharp  
// cs0825.cs  
class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;   
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```

## <a name="example"></a><span data-ttu-id="db63b-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="db63b-108">Example</span></span>  
 <span data-ttu-id="db63b-109">O código a seguir gera CS0825 porque `var` é usado em um campo de classe:</span><span class="sxs-lookup"><span data-stu-id="db63b-109">The following code generates CS0825 because `var` is used on a class field:</span></span>

```csharp  
{  
    // Both of these declarations trigger CS0825
    private var genreName;
    private var bookTitles = new List<string>();

    {  
        var totalBooks = 42; // var is OK here

    }  
}  
```

# <a name="compiler-error-cs0218"></a><span data-ttu-id="d9711-102">Erro do compilador CS0218</span><span class="sxs-lookup"><span data-stu-id="d9711-102">Compiler Error CS0218</span></span>
<span data-ttu-id="d9711-103">O tipo ('type') deve conter declarações de operador true e false</span><span class="sxs-lookup"><span data-stu-id="d9711-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="d9711-104">Se um tipo definido pelo usuário sobrecarregar o [& operador](../language-reference/operators/and-operator.md) ou [ &#124; operador](../language-reference/operators/or-operator.md), ele também deve definir [true e false](../language-reference/keywords/true-false-operators.md) operadores, para fazer o curto-circuito [& & operador](../language-reference/operators/conditional-and-operator.md) ou [ &#124; &#124; operador](../language-reference/operators/conditional-or-operator.md) definidos.</span><span class="sxs-lookup"><span data-stu-id="d9711-104">If a user-defined type overloads the [& operator](../language-reference/operators/and-operator.md) or [&#124; operator](../language-reference/operators/or-operator.md), it must also define [true and false](../language-reference/keywords/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/conditional-and-operator.md) or [&#124;&#124; operator](../language-reference/operators/conditional-or-operator.md) defined.</span></span>
  
 <span data-ttu-id="d9711-105">O exemplo a seguir gera CS0218:</span><span class="sxs-lookup"><span data-stu-id="d9711-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```

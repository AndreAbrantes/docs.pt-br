## <a name="example"></a><span data-ttu-id="0a8a1-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="0a8a1-106">Example</span></span>

```  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```

 <span data-ttu-id="76bb2-104">Os valores passados para o **IndexerName** atributo deve ser o mesmo para todos os indexadores em um tipo.</span><span class="sxs-lookup"><span data-stu-id="76bb2-104">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="76bb2-105">Para obter mais informações sobre o **IndexerName** de atributo, consulte [IndexerNameAttribute classe](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="76bb2-105">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="76bb2-106">O exemplo a seguir gera CS0668:</span><span class="sxs-lookup"><span data-stu-id="76bb2-106">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```

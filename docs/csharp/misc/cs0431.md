# <a name="compiler-error-cs0431"></a>Erro do compilador CS0431
Não é possível usar o alias 'identifier' com ':: ', pois o alias faz referência a um tipo. Use '.' em vez disso.  
  
 Você usou "::" com um alias que faz referência a um tipo. Para resolver esse erro, use o "." operador.  
  
 O exemplo a seguir gera CS0431:  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```

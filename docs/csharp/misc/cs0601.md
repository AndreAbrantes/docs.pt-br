# <a name="compiler-error-cs0601"></a><span data-ttu-id="67c03-102">CS0601 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="67c03-102">Compiler Error CS0601</span></span>
<span data-ttu-id="67c03-103">O atributo DllImport deve ser especificado em um método marcado como 'static' e 'extern'</span><span class="sxs-lookup"><span data-stu-id="67c03-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="67c03-104">O `DllImport` atributo foi usado em um método que não tinha as palavras-chave de acesso correto.</span><span class="sxs-lookup"><span data-stu-id="67c03-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="67c03-105">O exemplo a seguir gera CS0601:</span><span class="sxs-lookup"><span data-stu-id="67c03-105">The following sample generates CS0601:</span></span>  
  
```  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```

# <a name="compiler-error-cs0601"></a><span data-ttu-id="fc8d0-102">CS0601 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="fc8d0-102">Compiler Error CS0601</span></span>
<span data-ttu-id="fc8d0-103">O atributo DllImport deve ser especificado em um método marcado como 'static' e 'extern'</span><span class="sxs-lookup"><span data-stu-id="fc8d0-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="fc8d0-104">O `DllImport` atributo foi usado em um método que não tinha as palavras-chave de acesso corretas.</span><span class="sxs-lookup"><span data-stu-id="fc8d0-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="fc8d0-105">O exemplo a seguir gera CS0601:</span><span class="sxs-lookup"><span data-stu-id="fc8d0-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```

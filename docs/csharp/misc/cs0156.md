# <a name="compiler-error-cs0156"></a><span data-ttu-id="17c8b-102">Erro do compilador CS0156</span><span class="sxs-lookup"><span data-stu-id="17c8b-102">Compiler Error CS0156</span></span>
<span data-ttu-id="17c8b-103">Uma instrução Throw sem argumentos não é permitida em uma cláusula finally que esteja aninhada dentro da cláusula catch de circunscrição mais próxima</span><span class="sxs-lookup"><span data-stu-id="17c8b-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="17c8b-104">Uma instrução [throw](../language-reference/keywords/throw.md) sem parâmetros só pode aparecer em uma cláusula **Catch** que não aceite parâmetros.</span><span class="sxs-lookup"><span data-stu-id="17c8b-104">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="17c8b-105">Para obter mais informações, consulte [exceções e tratamento de exceções](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="17c8b-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="17c8b-106">O exemplo a seguir gera CS0156:</span><span class="sxs-lookup"><span data-stu-id="17c8b-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```

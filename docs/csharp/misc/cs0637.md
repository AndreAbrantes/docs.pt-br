# <a name="compiler-error-cs0637"></a><span data-ttu-id="afa12-102">Erro do compilador CS0637</span><span class="sxs-lookup"><span data-stu-id="afa12-102">Compiler Error CS0637</span></span>

<span data-ttu-id="afa12-103">O atributo FieldOffset não é permitido em campos estáticos ou const.</span><span class="sxs-lookup"><span data-stu-id="afa12-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="afa12-104">O [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atributo não pode ser usado em campos marcados com [estático](../../csharp/language-reference/keywords/static.md) ou [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="afa12-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="afa12-105">O exemplo a seguir gera CS0637:</span><span class="sxs-lookup"><span data-stu-id="afa12-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```

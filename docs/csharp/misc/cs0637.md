 <span data-ttu-id="4f893-104">O [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atributo não pode ser usado em campos marcados com [estático](../../csharp/language-reference/keywords/static.md) ou [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="4f893-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="4f893-105">O exemplo a seguir gera CS0637:</span><span class="sxs-lookup"><span data-stu-id="4f893-105">The following sample generates CS0637:</span></span>  
  
```

using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]

{

   {  
   }  
}  
```

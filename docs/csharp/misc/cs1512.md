# <a name="compiler-error-cs1512"></a><span data-ttu-id="e4fc9-103">Erro do Compilador CS1512</span><span class="sxs-lookup"><span data-stu-id="e4fc9-103">Compiler Error CS1512</span></span>
<span data-ttu-id="e4fc9-104">A palavra-chave ' base ' não está disponível no contexto atual</span><span class="sxs-lookup"><span data-stu-id="e4fc9-104">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="e4fc9-105">A palavra-chave [base](../language-reference/keywords/base.md) foi usada fora de um método, propriedade ou construtor.</span><span class="sxs-lookup"><span data-stu-id="e4fc9-105">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="e4fc9-106">O exemplo a seguir gera CS1512:</span><span class="sxs-lookup"><span data-stu-id="e4fc9-106">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```

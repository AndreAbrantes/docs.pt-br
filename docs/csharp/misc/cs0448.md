## <a name="example"></a><span data-ttu-id="e69c3-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="e69c3-105">Example</span></span>  
 <span data-ttu-id="e69c3-106">O exemplo a seguir gera CS0448.</span><span class="sxs-lookup"><span data-stu-id="e69c3-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```

   public static S? operator ++(S s) { return new S(); }   // CS0448

   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}

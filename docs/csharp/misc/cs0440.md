## <a name="example"></a><span data-ttu-id="da9c6-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="da9c6-106">Example</span></span>  
 <span data-ttu-id="da9c6-107">O exemplo a seguir gera CS0440:</span><span class="sxs-lookup"><span data-stu-id="da9c6-107">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```

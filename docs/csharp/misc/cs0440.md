## <a name="example"></a><span data-ttu-id="1b4c8-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="1b4c8-105">Example</span></span>  
 <span data-ttu-id="1b4c8-106">O exemplo a seguir gera CS0440:</span><span class="sxs-lookup"><span data-stu-id="1b4c8-106">The following example generates CS0440:</span></span>  
  
```  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```

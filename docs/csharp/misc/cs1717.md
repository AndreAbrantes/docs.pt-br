# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="ce9de-102">Compilador CS1717 de aviso (nível 3)</span><span class="sxs-lookup"><span data-stu-id="ce9de-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="ce9de-103">Atribuição feita à mesma variável. Você pretendia atribuir outro elemento?</span><span class="sxs-lookup"><span data-stu-id="ce9de-103">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="ce9de-104">Este aviso ocorre quando você atribuir uma variável a mesma, como `a = a`.</span><span class="sxs-lookup"><span data-stu-id="ce9de-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="ce9de-105">Vários erros comuns podem gerar esse aviso:</span><span class="sxs-lookup"><span data-stu-id="ce9de-105">Several common mistakes can generate this warning:</span></span>

# <a name="compiler-error-cs0764"></a><span data-ttu-id="ded01-103">Erro do Compilador CS0764</span><span class="sxs-lookup"><span data-stu-id="ded01-103">Compiler Error CS0764</span></span>
<span data-ttu-id="ded01-104">As duas declarações de método parcial devem ser inseguras ou nenhuma delas pode não ser segura</span><span class="sxs-lookup"><span data-stu-id="ded01-104">Both partial method declarations must be unsafe or neither may be unsafe</span></span>  
  
 <span data-ttu-id="ded01-105">Um método parcial consiste em uma declaração de definição (assinatura) e uma declaração de implementação opcional (corpo).</span><span class="sxs-lookup"><span data-stu-id="ded01-105">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="ded01-106">Se a declaração de definição tiver o `unsafe` modificador, a declaração de implementação também deverá tê-lo.</span><span class="sxs-lookup"><span data-stu-id="ded01-106">If the defining declaration has the `unsafe` modifier, the implementing declaration must also have it.</span></span> <span data-ttu-id="ded01-107">Por outro lado, se a declaração de implementação tiver o `unsafe` modificador, a declaração de definição também deverá.</span><span class="sxs-lookup"><span data-stu-id="ded01-107">Conversely, if the implementing declaration has the `unsafe` modifier, the defining declaration must also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ded01-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="ded01-108">To correct this error</span></span>

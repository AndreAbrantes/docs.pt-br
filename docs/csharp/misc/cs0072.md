# <a name="compiler-error-cs0072"></a><span data-ttu-id="8f244-102">Erro do compilador CS0072</span><span class="sxs-lookup"><span data-stu-id="8f244-102">Compiler Error CS0072</span></span>
<span data-ttu-id="8f244-103">' Event ': não é possível substituir; ' method ' não é um evento</span><span class="sxs-lookup"><span data-stu-id="8f244-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="8f244-104">Um [evento](../language-reference/keywords/event.md) só pode substituir outro evento.</span><span class="sxs-lookup"><span data-stu-id="8f244-104">An [event](../language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="8f244-105">Para obter mais informações, consulte [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="8f244-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="8f244-106">O exemplo a seguir gera CS0072:</span><span class="sxs-lookup"><span data-stu-id="8f244-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

# <a name="compiler-error-cs0542"></a><span data-ttu-id="a4414-103">Erro do Compilador CS0542</span><span class="sxs-lookup"><span data-stu-id="a4414-103">Compiler Error CS0542</span></span>

<span data-ttu-id="a4414-104">' tipo definido pelo usuário ': os nomes de membro não podem ser iguais aos do tipo delimitador</span><span class="sxs-lookup"><span data-stu-id="a4414-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="a4414-105">Os membros de uma classe ou struct não podem ter o mesmo nome que a classe ou struct, a menos que o membro seja um construtor.</span><span class="sxs-lookup"><span data-stu-id="a4414-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="a4414-106">O exemplo a seguir gera CS0542:</span><span class="sxs-lookup"><span data-stu-id="a4414-106">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="a4414-107">Esse erro pode ser causado se você colocar inadvertidamente um tipo de retorno em um construtor, o que, em vigor, o torna em um método comum.</span><span class="sxs-lookup"><span data-stu-id="a4414-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="a4414-108">O exemplo a seguir gera CS0542 porque `F` é um método, não um construtor, porque ele tem um tipo de retorno:</span><span class="sxs-lookup"><span data-stu-id="a4414-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="a4414-109">Se sua classe for denominada ' item ' e tiver um indexador declarado como `this` , você poderá receber esse erro.</span><span class="sxs-lookup"><span data-stu-id="a4414-109">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="a4414-110">Um indexador padrão recebe o nome ' item ' no código emitido, criando o conflito.</span><span class="sxs-lookup"><span data-stu-id="a4414-110">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>

   {  
   }

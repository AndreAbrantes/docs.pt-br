# <a name="compiler-error-cs0542"></a><span data-ttu-id="0e176-102">Erro do compilador CS0542</span><span class="sxs-lookup"><span data-stu-id="0e176-102">Compiler Error CS0542</span></span>
<span data-ttu-id="0e176-103">'tipo definido pelo usuário': nomes de membro não podem ser o mesmo que seu tipo delimitador</span><span class="sxs-lookup"><span data-stu-id="0e176-103">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="0e176-104">Os membros de uma classe ou struct não podem ter o mesmo nome que a classe ou struct, a menos que o membro é um construtor.</span><span class="sxs-lookup"><span data-stu-id="0e176-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="0e176-105">O exemplo a seguir gera CS0542:</span><span class="sxs-lookup"><span data-stu-id="0e176-105">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="0e176-106">Esse erro pode ser causado se você acidentalmente colocar um tipo de retorno em um construtor, o que torna em vigor em um método comum.</span><span class="sxs-lookup"><span data-stu-id="0e176-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="0e176-107">O exemplo a seguir gera CS0542 porque `F` é um método, não é um construtor, porque ele tem um tipo de retorno:</span><span class="sxs-lookup"><span data-stu-id="0e176-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="0e176-108">Se sua classe é chamada 'Item' e tem um indexador declarados como `this`, você pode obter esse erro.</span><span class="sxs-lookup"><span data-stu-id="0e176-108">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="0e176-109">Um indexador padrão recebe o nome 'Item' no código emitido, criando o conflito.</span><span class="sxs-lookup"><span data-stu-id="0e176-109">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>

      get  
      {  
         return 0;  
      }  
   }

   {  
   }  
}

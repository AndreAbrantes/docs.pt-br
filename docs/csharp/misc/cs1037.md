# <a name="compiler-error-cs1037"></a>CS1037 de erro do compilador
Operador que pode ser sobrecarregado é esperado  
  
 Ao especificar um comentário com [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), o compilador encontrou um link inválido.

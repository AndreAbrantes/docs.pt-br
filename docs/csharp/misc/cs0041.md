# <a name="compiler-error-cs0041"></a>Erro do compilador CS0041
O nome totalmente qualificado para 'type' é muito longo para informações de depuração. Compilar sem opção '/debug'.  
  
 Esse erro pode ocorrer ao usar o [/Debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) opção de compilador. Se você encontrar esse erro, tente excluir os arquivos PDB no diretório bin e recompilar. Se você ainda estiver tendo esse erro, você terá que reparar ou reinstalar o Visual Studio.

# <a name="compiler-error-cs0041"></a>Erro do compilador CS0041
O nome totalmente qualificado para ' type ' é muito longo para informações de depuração. Compilar sem a opção '/Debug '.  
  
 Esse erro pode ocorrer ao usar a opção de compilador [/debug](../language-reference/compiler-options/debug-compiler-option.md) . Se você encontrar esse erro, tente excluir os arquivos PDB no diretório bin e recompilá-los. Se você ainda estiver encontrando esse erro, talvez seja necessário reparar ou reinstalar o Visual Studio.

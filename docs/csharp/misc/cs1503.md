# <a name="compiler-error-cs1503"></a>Erro do compilador CS1503
O argumento 'number' não pode converter de TypeA TypeB  
  
 O tipo de um argumento em um método não coincide com o tipo que foram passado quando a classe foi instanciada. Esse erro normalmente aparece junto com CS1502. Ver [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) para uma discussão sobre como resolver esse erro.

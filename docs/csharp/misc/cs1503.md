# <a name="compiler-error-cs1503"></a>CS1503 de erro do compilador
O argumento 'número' não é possível converter de TypeA TypeB  
  
 O tipo de um argumento em um método não coincide com o tipo passado quando a classe foi instanciada. Este erro normalmente aparece junto com CS1502. Consulte [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) para uma discussão sobre como resolver esse erro.

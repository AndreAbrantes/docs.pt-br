# <a name="compiler-error-cs1673"></a><span data-ttu-id="46104-102">Erro do Compilador CS1673</span><span class="sxs-lookup"><span data-stu-id="46104-102">Compiler Error CS1673</span></span>

<span data-ttu-id="46104-103">Os métodos anônimos, as expressões lambda e as expressões de consulta dentro de structs não podem acessar os membros de instância de ' this '.</span><span class="sxs-lookup"><span data-stu-id="46104-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="46104-104">Considere copiar ' this ' para uma variável local fora do método anônimo, expressão lambda ou expressão de consulta e usando o local em vez disso.</span><span class="sxs-lookup"><span data-stu-id="46104-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="46104-105">O exemplo a seguir gera CS1673:</span><span class="sxs-lookup"><span data-stu-id="46104-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```

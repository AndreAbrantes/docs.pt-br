# <a name="compiler-error-cs1604"></a>CS1604 de erro do compilador
Não é possível atribuir a 'variável' porque ele é somente leitura  
  
 Uma atribuição foi feita em uma variável somente leitura. Para evitar esse erro, não atribua ou tentativa de modificar a essa variável neste contexto.

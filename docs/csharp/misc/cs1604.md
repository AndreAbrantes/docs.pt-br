# <a name="compiler-error-cs1604"></a>Erro do compilador CS1604
Não é possível atribuir à 'variável' porque ele é somente leitura  
  
 Uma atribuição foi feita a uma variável somente leitura. Para evitar esse erro, não atribua ou tentativa de modificar a essa variável neste contexto.

# <a name="compiler-error-cs0690"></a>Erro do compilador CS0690
O arquivo de entrada 'file' contém metadados inválidos.  
  
 Você é capaz de abrir o arquivo de metadados, mas devido a algum problema de localização, os metadados estava corrompido. Isso é semelhante ao erro [CS0009](../../csharp/misc/cs0009.md), exceto que você é capaz de abrir o arquivo de metadados.  
  
 Um PE (executável de processo) é um arquivo executável.

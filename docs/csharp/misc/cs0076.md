# <a name="compiler-error-cs0076"></a>CS0076 de erro do compilador
O Value enumerador nome _' está reservado e não pode ser usado  
  
 Um item em uma [enumeração](../../csharp/language-reference/keywords/enum.md) não pode ter um identificador chamado **value _**.

# <a name="compiler-error-cs0076"></a>Erro do compilador CS0076
O nome do enumerador ' _ value ' é reservado e não pode ser usado  
  
 Um item em uma [Enumeração](../language-reference/keywords/enum.md) não pode ter um identificador chamado **_ Value**.

# <a name="compiler-error-cs0061"></a><span data-ttu-id="1730a-103">Erro do Compilador CS0061</span><span class="sxs-lookup"><span data-stu-id="1730a-103">Compiler Error CS0061</span></span>
<span data-ttu-id="1730a-104">Acessibilidade inconsistente: a interface base ' interface 1 ' é menos acessível que a interface ' interface 2 '</span><span class="sxs-lookup"><span data-stu-id="1730a-104">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="1730a-105">Uma construção [pública](../language-reference/keywords/public.md) deve retornar um objeto publicamente acessível.</span><span class="sxs-lookup"><span data-stu-id="1730a-105">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>  
  
 <span data-ttu-id="1730a-106">A acessibilidade da interface não pode ser limitada em uma interface derivada.</span><span class="sxs-lookup"><span data-stu-id="1730a-106">Interface accessibility cannot be narrowed in a derived interface.</span></span> <span data-ttu-id="1730a-107">Para obter mais informações, consulte [interfaces](../programming-guide/interfaces/index.md) e [modificadores de acesso](../programming-guide/classes-and-structs/access-modifiers.md).</span><span class="sxs-lookup"><span data-stu-id="1730a-107">For more information, see [Interfaces](../programming-guide/interfaces/index.md) and [Access Modifiers](../programming-guide/classes-and-structs/access-modifiers.md).</span></span>  
  
 <span data-ttu-id="1730a-108">O exemplo a seguir gera CS0061.</span><span class="sxs-lookup"><span data-stu-id="1730a-108">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```

 <span data-ttu-id="0c353-104">Um [pública](../../csharp/language-reference/keywords/public.md) construção deve retornar um objeto acessível publicamente.</span><span class="sxs-lookup"><span data-stu-id="0c353-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="0c353-107">O exemplo a seguir gera CS0061.</span><span class="sxs-lookup"><span data-stu-id="0c353-107">The following sample generates CS0061.</span></span>  
  
```  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```

 <span data-ttu-id="0aa84-107">O exemplo a seguir gera CS0061.</span><span class="sxs-lookup"><span data-stu-id="0aa84-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```

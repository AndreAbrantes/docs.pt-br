## <a name="example"></a><span data-ttu-id="20c44-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="20c44-107">Example</span></span>

```csharp  
// CS0704.cs

{  
    public class I { }  
}

{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}

{  
    public static void Main() {}  
}
```

# <a name="compiler-error-cs1504"></a>CS1504 de erro do compilador
Não foi possível abrir o arquivo de origem 'file' ('motivo')  
  
 Um arquivo de origem não foi possível abrir ou ler pelo compilador. O arquivo pode estar bloqueado por outro aplicativo ou pode haver algum outro problema de sistema operacional. A mensagem inclui o motivo do sistema operacional por que o compilador não foi possível abrir ou ler o arquivo.

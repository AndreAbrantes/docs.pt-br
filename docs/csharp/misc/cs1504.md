# <a name="compiler-error-cs1504"></a>Erro do compilador CS1504
Não foi possível abrir o arquivo de origem 'file' ('reason')  
  
 Um arquivo de origem não foi possível abrir ou ler pelo compilador. O arquivo pode ser bloqueado por outro aplicativo ou pode haver algum outro problema do sistema operacional. A mensagem inclui o motivo do sistema operacional por que o compilador não pôde abrir ou ler o arquivo.

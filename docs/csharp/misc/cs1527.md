# <a name="compiler-error-cs1527"></a><span data-ttu-id="6f2a3-102">CS1527 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="6f2a3-102">Compiler Error CS1527</span></span>
<span data-ttu-id="6f2a3-103">Os elementos definidos em um namespace não podem ser declarados explicitamente como privado, protegido ou interno protegido</span><span class="sxs-lookup"><span data-stu-id="6f2a3-103">Elements defined in a namespace cannot be explicitly declared as private, protected, or protected internal</span></span>  
  
 <span data-ttu-id="6f2a3-104">Declarações de tipo em um namespace podem ter um [pública](../../csharp/language-reference/keywords/public.md) ou [interno](../../csharp/language-reference/keywords/internal.md) acesso.</span><span class="sxs-lookup"><span data-stu-id="6f2a3-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="6f2a3-105">Não se for especificada nenhum acessibilidade, **interno** é o padrão.</span><span class="sxs-lookup"><span data-stu-id="6f2a3-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="6f2a3-106">O exemplo a seguir gera CS1527:</span><span class="sxs-lookup"><span data-stu-id="6f2a3-106">The following sample generates CS1527:</span></span>  
  
```  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {};             // CS1527  
   protected class C2 {};           // CS1527  
   protected internal class C3 {};  // CS1527  
}  
```  
  
 <span data-ttu-id="6f2a3-107">O exemplo a seguir gera CS1527 porque quando nenhum espaço para nome for declarado explicitamente no seu código do programa, todas as declarações de tipo estão localizadas implicitamente no namespace global.</span><span class="sxs-lookup"><span data-stu-id="6f2a3-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```  
//cs1527_2.cs  
using System;  
  
protected class C4{}  
private struct S1{}  
```

# <a name="compiler-error-cs1527"></a><span data-ttu-id="9ab3a-102">Erro do compilador CS1527</span><span class="sxs-lookup"><span data-stu-id="9ab3a-102">Compiler Error CS1527</span></span>

<span data-ttu-id="9ab3a-103">Os elementos definidos em um namespace não podem ser declarados explicitamente como privados, protegidos, protegidos internamente ou protegidos de forma privada.</span><span class="sxs-lookup"><span data-stu-id="9ab3a-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="9ab3a-104">Declarações de tipo em um namespace podem ter acesso [público](../language-reference/keywords/public.md) ou [interno](../language-reference/keywords/internal.md) .</span><span class="sxs-lookup"><span data-stu-id="9ab3a-104">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="9ab3a-105">Se nenhuma acessibilidade for especificada, **interna** será o padrão.</span><span class="sxs-lookup"><span data-stu-id="9ab3a-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="9ab3a-106">O exemplo a seguir gera CS1527:</span><span class="sxs-lookup"><span data-stu-id="9ab3a-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="9ab3a-107">O exemplo a seguir gera CS1527 porque quando nenhum namespace é declarado explicitamente no código do programa, todas as declarações de tipo estão localizadas implicitamente no namespace global.</span><span class="sxs-lookup"><span data-stu-id="9ab3a-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```

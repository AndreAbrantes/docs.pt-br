# <a name="compiler-error-cs0569"></a>CS0569 de erro do compilador
'method2': não é possível substituir 'method1' porque ele não é suportado pelo idioma  
  
 Esse erro ocorre quando você derivar de uma classe base que foi escrita em outra linguagem e compilador não suporta o método que você está tentando substituir.

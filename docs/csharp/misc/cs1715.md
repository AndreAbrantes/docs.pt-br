# <a name="compiler-error-cs1715"></a><span data-ttu-id="479fb-102">Erro do compilador CS1715</span><span class="sxs-lookup"><span data-stu-id="479fb-102">Compiler Error CS1715</span></span>
<span data-ttu-id="479fb-103">'Type1': o tipo deve ser 'Type2' para corresponder ao membro substituído 'MemberName'</span><span class="sxs-lookup"><span data-stu-id="479fb-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="479fb-104">Esse erro é o mesmo que [erro do compilador CS0508](../../csharp/misc/cs0508.md), exceto que CS0508 agora se aplica apenas aos métodos que têm tipos de retorno, enquanto CS1715 se aplica às propriedades e indexadores que têm apenas 'tipos' em vez de 'tipos de retorno'.</span><span class="sxs-lookup"><span data-stu-id="479fb-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="479fb-105">Exemplo</span><span class="sxs-lookup"><span data-stu-id="479fb-105">Example</span></span>  
 <span data-ttu-id="479fb-106">O código a seguir gera CS1715.</span><span class="sxs-lookup"><span data-stu-id="479fb-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```

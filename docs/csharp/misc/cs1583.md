# <a name="compiler-error-cs1583"></a>CS1583 de erro do compilador
'file' não é um arquivo de recurso Win32 válido  
  
 Esse erro ocorre quando você especificar um nome de arquivo com o [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opção de compilador que não é um arquivo de recurso Win32 válido ou está formatado corretamente. No Visual Studio, o nome do arquivo é especificado no painel Application do Project Designer.

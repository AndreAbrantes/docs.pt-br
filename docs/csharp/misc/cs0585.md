# <a name="compiler-error-cs0585"></a>CS0585 de erro do compilador
Erro interno no compilador: etapa 'estágio'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se esse não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).

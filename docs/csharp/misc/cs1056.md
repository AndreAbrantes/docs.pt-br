# <a name="compiler-error-cs1056"></a>CS1056 de erro do compilador
Caractere inesperado 'character'  
  
 O compilador c# encontrou um caractere inesperado e não pode identificar o token que está sendo processado no momento. Por exemplo, se o compilador encontra um caractere de Euro no meio de um identificador de processamento, ele será possível classificar o identificador, como um caractere de Euro só será válido dentro de uma cadeia de caracteres, e o compilador sabe que ele não está processando uma cadeia de caracteres.

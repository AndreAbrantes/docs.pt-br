# <a name="compiler-error-cs0648"></a>CS0648 de erro do compilador
'type' é um tipo sem suporte de linguagem  
  
 Metadados que foi gerado de outro idioma, possivelmente C++, continham um tipo que não foi marcado como gerenciado. Informações sobre esse tipo estão incluídas nos metadados, mas o tipo não está disponível para programas escritos em c#.

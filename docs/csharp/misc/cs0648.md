# <a name="compiler-error-cs0648"></a>CS0648 de erro do compilador
'type' é um tipo sem suporte de linguagem  
  
 Metadados que foi gerado a partir de outro idioma, possivelmente C++, continham um tipo que não foi marcado como gerenciado. Informações sobre esse tipo são incluídas nos metadados, mas o tipo não está disponível para os programas escritos em c#.

# <a name="compiler-error-cs0648"></a>Erro do compilador CS0648
'type' é um tipo não suportado pelo idioma  
  
 Metadados que foi gerado a partir de outra linguagem, possivelmente, C++, continha um tipo que não foi marcado como gerenciado. Informações sobre esse tipo estão incluídas nos metadados, mas o tipo não está disponível para os programas escritos em C#.

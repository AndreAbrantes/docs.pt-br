# <a name="compiler-error-cs0502"></a><span data-ttu-id="4e19a-102">CS0502 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="4e19a-102">Compiler Error CS0502</span></span>

 <span data-ttu-id="4e19a-104">Um membro de classe não pode ser [abstrata](../../csharp/language-reference/keywords/abstract.md) e [lacrado](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="4e19a-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>

```csharp  
{  
   abstract public void F();  
}

{  
   abstract sealed override public void F()   // CS0502

}

{  
    public static void Main()  
    { }  
}  
```

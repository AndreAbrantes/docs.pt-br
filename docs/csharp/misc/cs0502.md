 <span data-ttu-id="29ae7-104">Um membro de classe não pode ser ambos [abstrata](../../csharp/language-reference/keywords/abstract.md) e [lacrado](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="29ae7-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>

    { }  
}

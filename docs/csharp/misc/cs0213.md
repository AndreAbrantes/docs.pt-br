## <a name="example"></a><span data-ttu-id="f8f5f-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f8f5f-106">Example</span></span>  
 <span data-ttu-id="f8f5f-107">O exemplo a seguir gera CS0213.</span><span class="sxs-lookup"><span data-stu-id="f8f5f-107">The following sample generates CS0213.</span></span>  
  
```  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```

# <a name="compiler-error-cs1535"></a>Erro do compilador CS1535
Operador unário sobrecarregado 'operator' usa um parâmetro  
  
 A definição de um unário [operador](../../csharp/language-reference/keywords/operator.md) devem ter um parâmetro.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS1535:  
  
```csharp  
// CS1535.cs  
class MyClass  
{  
    // uncomment the method parameter to resolve CS1535  
    public static MyClass operator ++ (/*MyClass MC1*/)   // CS1535  
    {  
        return new MyClass();  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```

# <a name="compiler-error-cs0468"></a>Erro do compilador CS0468
Ambiguidade entre o tipo 'type1' e tipo 'type2'  
  
 Esse erro é gerado quando há dois tipos no assembly que está sendo compilado que têm o mesmo nome totalmente qualificado. Isso pode ocorrer se as duas estarão em módulos adicionados ou um está em um módulo adicionado e outro no código-fonte.

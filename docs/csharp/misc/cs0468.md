# <a name="compiler-error-cs0468"></a>CS0468 de erro do compilador
Ambiguidade entre o tipo 'type1' e 'type2'  
  
 Esse erro é gerado quando há dois tipos no assembly que está sendo compilado que têm o mesmo nome totalmente qualificado. Isso pode ocorrer se eles estiverem no módulos adicionados ou um está em um módulo adicionado e outro na fonte.

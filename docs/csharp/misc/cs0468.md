# <a name="compiler-error-cs0468"></a>Erro do Compilador CS0468

Ambiguidade entre o tipo ' type1 ' e o tipo ' type2 '

 Esse erro é gerado quando há dois tipos no assembly que estão sendo compilados que têm o mesmo nome totalmente qualificado. Isso pode ocorrer se ambos estiverem em módulos adicionados ou se um estiver em um módulo adicionado e outro na origem.

> [!NOTE]
> Este erro de compilador não é mais usado em Roslyn.

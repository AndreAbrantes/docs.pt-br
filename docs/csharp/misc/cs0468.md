# <a name="compiler-error-cs0468"></a>Erro do Compilador CS0468

Ambiguidade entre o tipo 'type1' e o tipo 'type2'

 Este erro é gerado quando há dois tipos no conjunto sendo compilados que têm o mesmo nome totalmente qualificado. Isso pode ocorrer se ambos estiverem em módulos adicionados ou um em um módulo adicionado e outro na fonte.

> [!NOTE]
> Este erro do compilador não é mais usado em Roslyn.

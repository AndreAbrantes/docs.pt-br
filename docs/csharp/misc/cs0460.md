## <a name="example"></a><span data-ttu-id="d43f1-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d43f1-106">Example</span></span>  
 <span data-ttu-id="d43f1-107">O exemplo a seguir gera CS0460.</span><span class="sxs-lookup"><span data-stu-id="d43f1-107">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass   
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```

<span data-ttu-id="1161f-103">'rotina de conversão': usuário definido a conversão em/de classe derivada</span><span class="sxs-lookup"><span data-stu-id="1161f-103">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="1161f-104">Conversões definidas pelo usuário para os valores de uma classe derivada não são permitidos; não é necessário um operador.</span><span class="sxs-lookup"><span data-stu-id="1161f-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="1161f-105">Consulte o capítulo 6 a especificação da linguagem c# para obter mais informações sobre conversões definidas pelo usuário.</span><span class="sxs-lookup"><span data-stu-id="1161f-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="1161f-106">O exemplo a seguir gera CS0554:</span><span class="sxs-lookup"><span data-stu-id="1161f-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```

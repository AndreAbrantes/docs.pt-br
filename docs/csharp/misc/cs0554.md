 <span data-ttu-id="06b23-104">Conversões definidas pelo usuário para os valores de uma classe derivada não são permitidos; não é necessário um operador.</span><span class="sxs-lookup"><span data-stu-id="06b23-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>

 <span data-ttu-id="06b23-106">O exemplo a seguir gera CS0554:</span><span class="sxs-lookup"><span data-stu-id="06b23-106">The following sample generates CS0554:</span></span>  
  
```  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```

# <a name="compiler-error-cs0554"></a><span data-ttu-id="9f533-103">Erro do Compilador CS0554</span><span class="sxs-lookup"><span data-stu-id="9f533-103">Compiler Error CS0554</span></span>

<span data-ttu-id="9f533-104">' rotina de conversão ': conversão definida pelo usuário de/para classe derivada</span><span class="sxs-lookup"><span data-stu-id="9f533-104">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="9f533-105">Conversões definidas pelo usuário para valores de uma classe derivada não são permitidas; Você não precisa desse operador.</span><span class="sxs-lookup"><span data-stu-id="9f533-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="9f533-106">Consulte o capítulo 6 na especificação da linguagem C# para obter mais informações sobre conversões definidas pelo usuário.</span><span class="sxs-lookup"><span data-stu-id="9f533-106">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="9f533-107">O exemplo a seguir gera CS0554:</span><span class="sxs-lookup"><span data-stu-id="9f533-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```

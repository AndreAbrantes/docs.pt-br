# <a name="compiler-error-cs0682"></a>CS0682 de erro do compilador
'type1' não pode implementar 'type2' porque não há suporte para o idioma  
  
 Esse erro ocorre quando você tenta implementar uma interface escrita em outra linguagem e o compilador não oferece suporte a interface.

# <a name="compiler-error-cs2035"></a>CS2035 de erro do compilador
Erro de sintaxe de linha de comando: ausente ':\<número >' para a opção 'compiler_option'  
  
 Algumas opções de compilador exigem um valor.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS2035.  
  
```  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```

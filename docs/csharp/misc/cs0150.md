# <a name="compiler-error-cs0150"></a><span data-ttu-id="c6dce-103">Erro do Compilador CS0150</span><span class="sxs-lookup"><span data-stu-id="c6dce-103">Compiler Error CS0150</span></span>
<span data-ttu-id="c6dce-104">Um valor constante é esperado</span><span class="sxs-lookup"><span data-stu-id="c6dce-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="c6dce-105">Uma variável foi encontrada onde era esperada uma constante.</span><span class="sxs-lookup"><span data-stu-id="c6dce-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="c6dce-106">Para obter mais informações, consulte [switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="c6dce-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="c6dce-107">O exemplo a seguir gera CS0150:</span><span class="sxs-lookup"><span data-stu-id="c6dce-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="c6dce-108">Esse erro também é produzido quando um tamanho de matriz é especificado com um valor de variável e inicializado com um inicializador de matriz.</span><span class="sxs-lookup"><span data-stu-id="c6dce-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="c6dce-109">Para remover o erro, inicialize a matriz em uma ou mais instruções separadas.</span><span class="sxs-lookup"><span data-stu-id="c6dce-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```

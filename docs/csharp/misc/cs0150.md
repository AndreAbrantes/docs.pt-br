# <a name="compiler-error-cs0150"></a><span data-ttu-id="cd938-102">Erro do Compilador CS0150</span><span class="sxs-lookup"><span data-stu-id="cd938-102">Compiler Error CS0150</span></span>
<span data-ttu-id="cd938-103">Um valor constante é esperado</span><span class="sxs-lookup"><span data-stu-id="cd938-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="cd938-104">Uma variável foi encontrada onde se esperava uma constante.</span><span class="sxs-lookup"><span data-stu-id="cd938-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="cd938-105">Para obter mais informações, consulte [switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="cd938-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="cd938-106">A seguinte amostra gera CS0150:</span><span class="sxs-lookup"><span data-stu-id="cd938-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="cd938-107">Este erro também é produzido quando um tamanho de matriz é especificado com um valor variável e inicializado com um inicializador de matriz.</span><span class="sxs-lookup"><span data-stu-id="cd938-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="cd938-108">Para remover o erro, inicialize a matriz em uma instrução ou instruções separadas.</span><span class="sxs-lookup"><span data-stu-id="cd938-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```

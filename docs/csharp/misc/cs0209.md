# <a name="compiler-error-cs0209"></a>Erro do compilador CS0209
O tipo de local declarado em uma instrução fixed deve ser um tipo de ponteiro  
  
 A variável que você declara em um [instrução fixed](../../csharp/language-reference/keywords/fixed-statement.md) deve ser um ponteiro. Para obter mais informações, consulte [Código não seguro e ponteiros](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 O exemplo a seguir gera CS0209:  
  
```csharp  
// CS0209.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0209  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```

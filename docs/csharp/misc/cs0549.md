# <a name="compiler-error-cs0549"></a>Erro do compilador CS0549
'function' é um novo membro virtual em classe sealed 'class'  
  
 Um [lacrado](../../csharp/language-reference/keywords/sealed.md)[classe](../../csharp/language-reference/keywords/class.md) não pode ser usado como uma classe base.  Portanto, é inútil ter um método virtual na classe selada.  
  
 O exemplo a seguir gera CS0549:  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```

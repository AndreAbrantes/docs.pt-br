# <a name="compiler-error-cs0549"></a><span data-ttu-id="8abe1-102">CS0549 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="8abe1-102">Compiler Error CS0549</span></span>

 <span data-ttu-id="8abe1-104">Um [lacrado](../../csharp/language-reference/keywords/sealed.md)[classe](../../csharp/language-reference/keywords/class.md) não pode ser usado como uma classe base.</span><span class="sxs-lookup"><span data-stu-id="8abe1-104">A [sealed](../../csharp/language-reference/keywords/sealed.md)[class](../../csharp/language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="8abe1-105">Portanto, é inútil para ter um método virtual na classe sealed.</span><span class="sxs-lookup"><span data-stu-id="8abe1-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>

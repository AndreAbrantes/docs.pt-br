## <a name="example"></a><span data-ttu-id="24f8f-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="24f8f-107">Example</span></span>  
 <span data-ttu-id="24f8f-108">O exemplo a seguir gera CS1657:</span><span class="sxs-lookup"><span data-stu-id="24f8f-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```

 <span data-ttu-id="846dc-105">Esse erro ocorre quando uma variável é passada como um argumento [ref](../language-reference/keywords/ref.md) ou [out](../language-reference/keywords/out-parameter-modifier.md) em um contexto no qual essa variável é ReadOnly.</span><span class="sxs-lookup"><span data-stu-id="846dc-105">This error occurs when a variable is passed as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="846dc-106">Contextos ReadOnly incluem variáveis de iteração [foreach](../language-reference/keywords/foreach-in.md) , [usando](../language-reference/keywords/using-statement.md) variáveis e `fixed` variáveis.</span><span class="sxs-lookup"><span data-stu-id="846dc-106">Readonly contexts include [foreach](../language-reference/keywords/foreach-in.md) iteration variables, [using](../language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="846dc-107">Para resolver esse erro, não chame funções que usam a `foreach` variável, `using` ou `fixed` como um `ref` parâmetro ou `out` em `using` blocos, `foreach` instruções e `fixed` instruções.</span><span class="sxs-lookup"><span data-stu-id="846dc-107">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="846dc-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="846dc-108">Example</span></span>  

 <span data-ttu-id="846dc-109">O exemplo a seguir gera CS1657:</span><span class="sxs-lookup"><span data-stu-id="846dc-109">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```

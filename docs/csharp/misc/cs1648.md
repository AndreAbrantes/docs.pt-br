## <a name="example"></a><span data-ttu-id="5dec0-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5dec0-107">Example</span></span>

 <span data-ttu-id="5dec0-108">O exemplo a seguir gera CS1648:</span><span class="sxs-lookup"><span data-stu-id="5dec0-108">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```

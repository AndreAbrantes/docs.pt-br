## <a name="example"></a><span data-ttu-id="ffca0-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ffca0-106">Example</span></span>

 <span data-ttu-id="ffca0-107">O exemplo a seguir gera CS1648:</span><span class="sxs-lookup"><span data-stu-id="ffca0-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```

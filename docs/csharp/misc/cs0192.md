 <span data-ttu-id="c532e-104">Um campo (variável) é marcado com o [readonly](../../csharp/language-reference/keywords/readonly.md) palavra-chave não pode ser passado para um [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out.md) parâmetro exceto dentro de um construtor.</span><span class="sxs-lookup"><span data-stu-id="c532e-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameter except inside a constructor.</span></span> <span data-ttu-id="c532e-105">Para obter mais informações, consulte [campos](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="c532e-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="c532e-106">CS0192 também ocorrerá se o `readonly` campo é [estático](../../csharp/language-reference/keywords/static.md) e não está marcado como o construtor `static`.</span><span class="sxs-lookup"><span data-stu-id="c532e-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c532e-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="c532e-107">Example</span></span>  
 <span data-ttu-id="c532e-108">O exemplo a seguir gera CS0192.</span><span class="sxs-lookup"><span data-stu-id="c532e-108">The following sample generates CS0192.</span></span>  
  
```  
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

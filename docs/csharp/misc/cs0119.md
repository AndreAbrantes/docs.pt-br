B () ' é um método, que não é válido no contexto fornecido.</span><span class="sxs-lookup"><span data-stu-id="f5e72-110">The following sample generates CS0119: 'C.B()' is a method, which is not valid in the given context.</span></span> <span data-ttu-id="f5e72-111">Você pode corrigir esse erro alterando o nome do método `C.B` ou usando o nome totalmente qualificado para a classe `B` como `N2.B` .</span><span class="sxs-lookup"><span data-stu-id="f5e72-111">You can fix this error by changing the name of the method `C.B`, or using the fully qualified name for the class `B` like `N2.B`.</span></span>

{
    public static class B
    {
        public static void X() {}
    }
}

    public class C
    {
        void B() {}
        void M() => B.X();   // CS0119
    }

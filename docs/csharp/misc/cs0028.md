# <a name="compiler-warning-level-4-cs0028"></a>Compilador aviso (nível 4) CS0028
'declaração de função ' tem assinatura incorreta para um ponto de entrada  
  
 A declaração de método para `Main` era inválido: foi declarado com uma assinatura inválida. `Main` deve ser declarado como estático e ele devem retornar [int](../../csharp/language-reference/builtin-types/integral-numeric-types.md) ou [void](../../csharp/language-reference/keywords/void.md). Para obter mais informações, consulte [Main () e argumentos de linha de comando](../../csharp/programming-guide/main-and-command-args/index.md).  
  
 O exemplo a seguir gera CS0028:  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```

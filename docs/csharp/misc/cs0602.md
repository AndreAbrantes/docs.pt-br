# <a name="compiler-warning-level-1-cs0602"></a>Compilador CS0602 de aviso (nível 1)
O recurso 'old_feature' está obsoleto. Em vez disso, use 'new_feature'  
  
 Um recurso de linguagem usado em seu código (*old_feature*) ainda é suportado, mas esse suporte pode ser removido em uma versão futura. Em vez disso, você deve usar a sintaxe recomendada (*new_feature*).

# <a name="compiler-warning-level-1-cs0602"></a>Aviso do compilador (nível 1) CS0602

O recurso 'old_feature' é preterido. Por favor, use 'new_feature' em vez disso

 Um recurso de idioma usado em seu código *(old_feature)* ainda é suportado, mas esse suporte pode ser removido em uma versão futura. Em vez disso, você deve usar a sintaxe recomendada *(new_feature).*

> [!NOTE]
> Este aviso de compilador não é mais usado em Roslyn.

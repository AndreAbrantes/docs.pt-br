# <a name="compiler-warning-level-1-cs0602"></a>Compilador CS0602 de aviso (nível 1)
O recurso 'old_feature' é preterido. Em vez disso, use 'new_feature'  
  
 Um recurso de idioma usado no seu código (*old_feature*) ainda é suportada, mas esse suporte pode ser removido em uma versão futura. Em vez disso, você deve usar a sintaxe recomendada (*new_feature*).

# <a name="compiler-warning-level-1-cs0602"></a>Aviso do compilador (nível 1) CS0602

O recurso ' old_feature ' foi preterido. Em vez disso, use ' new_feature '

 Um recurso de linguagem usado em seu código (*old_feature*) ainda tem suporte, mas esse suporte pode ser removido em uma versão futura. Em vez disso, você deve usar a sintaxe recomendada (*new_feature*).

> [!NOTE]
> Este aviso do compilador não é mais usado no Roslyn.

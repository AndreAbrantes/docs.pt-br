# <a name="compiler-warning-level-3-cs0661"></a>Compilador aviso (nível 3) CS0661
'class' define o operador = = ou operador! = mas não substitui Object.GetHashCode()  
  
 O compilador detectou a igualdade definido pelo usuário ou o operador de desigualdade, mas nenhuma substituição para o **GetHashCode** função. Um operador de igualdade ou desigualdade definido pelo usuário implica que você também deseja substituir a **GetHashCode** função.  
  
 O exemplo a seguir gera CS0661:  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```

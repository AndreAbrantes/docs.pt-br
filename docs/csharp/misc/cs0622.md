# <a name="compiler-error-cs0622"></a>Erro do compilador CS0622
Só pode usar expressões de inicializador de matriz para atribuir a tipos de matriz. Tente usar uma nova expressão.  
  
 A sintaxe apropriada para inicializar uma matriz foi usada na declaração de não matriz.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0622:  
  
```csharp  
// CS0622.cs  
using System;  
  
public class Test  
{  
    public static void Main ()  
    {  
        Test t = { new Test() };   // CS0622  
        // Try the following instead:  
        // Test[] t = { new Test() };  
    }  
}  
```

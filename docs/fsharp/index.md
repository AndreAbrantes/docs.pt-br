# <a name="f-guide"></a>Guia de F#

O guia de F # fornece muitos recursos para aprender a linguagem F #.

## <a name="learning-f"></a>Aprender a F # #

[O que é o F #](what-is-fsharp.md) descreve o que é a linguagem F # e programação nele é semelhante, com exemplos de código curtos. Isso é recomendado se você for novo no F #.

[Tour do F #](tour.md) fornece uma visão geral dos recursos de linguagem principal com muitos dos exemplos de código. Isso é recomendado se você estiver interessado em ver os recursos de núcleo do F # em ação.

[Introdução ao F # no Visual Studio](get-started/get-started-visual-studio.md) se você está no Windows e deseja que a experiência completa de IDE (ambiente de desenvolvimento integrado) do Visual Studio.

[Introdução ao F # no Visual Studio para Mac](get-started/get-started-with-visual-studio-for-mac.md) se você estiver no macOS e quiser usar um IDE do Visual Studio.

[Introdução ao F # no Visual Studio Code](get-started/get-started-vscode.md) se desejar que uma leve, plataforma cruzada, e a experiência de IDE repleto de recursos.

[Introdução ao F # com a CLI do .NET Core](get-started/get-started-command-line.md) se você quiser usar as ferramentas de linha de comando.

[Introdução ao F # e Xamarin](https://docs.microsoft.com/xamarin/cross-platform/platform/fsharp/) para programação móvel com o F #.

[F # para Azure Notebooks](https://notebooks.azure.com/Microsoft/libraries/samples/html/FSharp%20for%20Azure%20Notebooks.ipynb) é um tutorial para aprender a F # em um Jupyter Notebook hospedado, livre.

## <a name="other-useful-resources"></a>Outros recursos úteis

O [site de trechos de código F #](http://www.fssnip.net) contém um conjunto grande de trechos de código mostrando como fazer qualquer coisa no F #, desde iniciantes até trechos de código altamente avançados.

O [F # Software Foundation Slack](https://fsharp.org/guides/slack/) é um ótimo lugar para iniciantes e especialistas da mesma maneira, é altamente ativos, e tem algumas das melhores programadores do mundo em F # disponíveis para um bate-papo. É altamente recomendável ingressar.

## <a name="the-f-software-foundation"></a>O F# Software Foundation

Embora a Microsoft seja o desenvolvedor principal da linguagem F # e suas ferramentas no Visual Studio, F # também recebe suporte de uma base independente, o F # Software Foundation (FSSF).

A missão do F# Software Foundation é promover, proteger e evoluir a linguagem de programação F#, além de dar suporte e facilitar o crescimento de uma comunidade internacional e diversa de programadores em F#.

Para saber mais e se envolver, confira [fsharp.org](https://fsharp.org). A junção é gratuita e a rede de desenvolvedores do F # no foundation é algo que não deseja perder!

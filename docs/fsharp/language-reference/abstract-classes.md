# <a name="abstract-classes"></a><span data-ttu-id="f28e7-103">Classes abstratas</span><span class="sxs-lookup"><span data-stu-id="f28e7-103">Abstract Classes</span></span>

<span data-ttu-id="f28e7-104">*Classes abstratas* são classes que deixam alguns ou todos os membros não implementados, para que as implementações podem ser fornecidas por classes derivadas.</span><span class="sxs-lookup"><span data-stu-id="f28e7-104">*Abstract classes* are classes that leave some or all members unimplemented, so that implementations can be provided by derived classes.</span></span>

## <a name="syntax"></a><span data-ttu-id="f28e7-105">Sintaxe</span><span class="sxs-lookup"><span data-stu-id="f28e7-105">Syntax</span></span>

```fsharp
// Abstract class syntax.
[<AbstractClass>]
type [ accessibility-modifier ] abstract-class-name =
[ inherit base-class-or-interface-name ]
[ abstract-member-declarations-and-member-definitions ]

// Abstract member syntax.
abstract member member-name : type-signature
```

# <a name="type-providers"></a><span data-ttu-id="08d17-103">Provedores de tipos</span><span class="sxs-lookup"><span data-stu-id="08d17-103">Type Providers</span></span>

<span data-ttu-id="08d17-104">Um provedor de tipos F# é um componente que fornece tipos, propriedades e métodos para uso em seu programa.</span><span class="sxs-lookup"><span data-stu-id="08d17-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="08d17-105">Provedores de tipos geram o que é conhecido como **fornecidos tipos**, que é gerado pelo compilador do F # e baseiam-se em uma fonte de dados externa.</span><span class="sxs-lookup"><span data-stu-id="08d17-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="08d17-106">Por exemplo, um provedor de tipos F # para o SQL pode gerar tipos que representam tabelas e colunas em um banco de dados relacional.</span><span class="sxs-lookup"><span data-stu-id="08d17-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="08d17-107">Na verdade, isso é o que o [SQLProvider](https://fsprojects.github.io/SQLProvider/) tipo de provedor não.</span><span class="sxs-lookup"><span data-stu-id="08d17-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="08d17-108">Fornecidos que tipos dependem de parâmetros de entrada para um provedor de tipo.</span><span class="sxs-lookup"><span data-stu-id="08d17-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="08d17-109">Essa entrada pode ser uma fonte de dados de exemplo (como um arquivo de esquema JSON), uma URL que aponta diretamente para um serviço externo ou uma cadeia de caracteres de conexão a uma fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="08d17-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="08d17-110">Um provedor de tipos também pode garantir que grupos de tipos sejam expandidos sob demanda; ou seja, eles serão expandidos se os tipos forem realmente referenciados pelo seu programa.</span><span class="sxs-lookup"><span data-stu-id="08d17-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="08d17-111">Isso permite a integração direta e sob demanda de espaços de informações em grande escala como mercados de dados online de uma forma fortemente tipada.</span><span class="sxs-lookup"><span data-stu-id="08d17-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="08d17-112">Provedores de tipos produtivas e apagado</span><span class="sxs-lookup"><span data-stu-id="08d17-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="08d17-113">Provedores de tipos têm duas formas: apagados e produtivas.</span><span class="sxs-lookup"><span data-stu-id="08d17-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="08d17-114">Provedores de tipos produtivas produzir tipos que podem ser gravados como tipos do .NET no assembly no qual eles são produzidos.</span><span class="sxs-lookup"><span data-stu-id="08d17-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="08d17-115">Isso permite que eles para serem consumidos pelo código em outros assemblies.</span><span class="sxs-lookup"><span data-stu-id="08d17-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="08d17-116">Isso significa que a representação de tipo da fonte de dados geralmente deve ser um que seja viável para representar com tipos .NET.</span><span class="sxs-lookup"><span data-stu-id="08d17-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="08d17-117">Apagar provedores de tipos produzir tipos que só podem ser consumidos no assembly ou eles são gerados a partir do projeto.</span><span class="sxs-lookup"><span data-stu-id="08d17-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="08d17-118">Os tipos são efêmeros; ou seja, eles não são gravados em um assembly e não podem ser consumidos pelo código em outros assemblies.</span><span class="sxs-lookup"><span data-stu-id="08d17-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="08d17-119">Eles podem conter *atrasada* membros, permitindo que os tipos de uso fornecido de um espaço de informações potencialmente infinito.</span><span class="sxs-lookup"><span data-stu-id="08d17-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="08d17-120">Eles são úteis para o uso de um pequeno subconjunto de uma fonte de dados grandes e interconectadas.</span><span class="sxs-lookup"><span data-stu-id="08d17-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

- <span data-ttu-id="08d17-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/) tem um conjunto de provedores de tipo para o tempo de compilação verificada incorporação do T-SQL em F #.</span><span class="sxs-lookup"><span data-stu-id="08d17-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/) has a set of type providers for compile-time checked embedding of T-SQL in F#.</span></span>

- <span data-ttu-id="08d17-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html) contém o **GraphQLProvider**, que fornece tipos com base em um servidor GraphQL especificado pela URL.</span><span class="sxs-lookup"><span data-stu-id="08d17-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html) contains the **GraphQLProvider**, which provides types based on a GraphQL server specified by URL.</span></span>

# <a name="type-providers"></a><span data-ttu-id="a44fd-103">Provedores de tipos</span><span class="sxs-lookup"><span data-stu-id="a44fd-103">Type Providers</span></span>

<span data-ttu-id="a44fd-104">Um provedor de tipos F# é um componente que fornece tipos, propriedades e métodos para uso em seu programa.</span><span class="sxs-lookup"><span data-stu-id="a44fd-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="a44fd-105">Os provedores de tipos geram o que são conhecidos como **tipos fornecidos**, que são gerados pelo compilador F # e são baseados em uma fonte de dados externa.</span><span class="sxs-lookup"><span data-stu-id="a44fd-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="a44fd-106">Por exemplo, um provedor de tipo F # para SQL pode gerar tipos que representam tabelas e colunas em um banco de dados relacional.</span><span class="sxs-lookup"><span data-stu-id="a44fd-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="a44fd-107">Na verdade, é isso que o provedor de tipo [Sqlfornecetor](https://fsprojects.github.io/SQLProvider/) faz.</span><span class="sxs-lookup"><span data-stu-id="a44fd-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="a44fd-108">Os tipos fornecidos dependem de parâmetros de entrada para um provedor de tipo.</span><span class="sxs-lookup"><span data-stu-id="a44fd-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="a44fd-109">Essa entrada pode ser uma fonte de dados de exemplo (como um arquivo de esquema JSON), uma URL que aponta diretamente para um serviço externo ou uma cadeia de conexão para uma fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="a44fd-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="a44fd-110">Um provedor de tipos também pode garantir que os grupos de tipos sejam expandidos apenas sob demanda; ou seja, eles serão expandidos se os tipos forem realmente referenciados por seu programa.</span><span class="sxs-lookup"><span data-stu-id="a44fd-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="a44fd-111">Isso permite a integração direta e sob demanda de espaços de informações em grande escala como mercados de dados online de uma forma fortemente tipada.</span><span class="sxs-lookup"><span data-stu-id="a44fd-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="a44fd-112">Geração e provedores de tipo apagados</span><span class="sxs-lookup"><span data-stu-id="a44fd-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="a44fd-113">Os provedores de tipos são fornecidos em duas formas: geração e Erase.</span><span class="sxs-lookup"><span data-stu-id="a44fd-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="a44fd-114">Os provedores de tipo geração produzem tipos que podem ser gravados como tipos .NET no assembly no qual são produzidos.</span><span class="sxs-lookup"><span data-stu-id="a44fd-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="a44fd-115">Isso permite que eles sejam consumidos do código em outros assemblies.</span><span class="sxs-lookup"><span data-stu-id="a44fd-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="a44fd-116">Isso significa que a representação tipada da fonte de dados deve ser geralmente uma que seja viável para representar com tipos .NET.</span><span class="sxs-lookup"><span data-stu-id="a44fd-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="a44fd-117">Apagar provedores de tipo produz tipos que só podem ser consumidos no assembly ou projeto do qual são gerados.</span><span class="sxs-lookup"><span data-stu-id="a44fd-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="a44fd-118">Os tipos são efêmeros; ou seja, eles não são gravados em um assembly e não podem ser consumidos pelo código em outros assemblies.</span><span class="sxs-lookup"><span data-stu-id="a44fd-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="a44fd-119">Eles podem conter membros *atrasados* , permitindo que você use tipos fornecidos de um espaço de informações potencialmente infinito.</span><span class="sxs-lookup"><span data-stu-id="a44fd-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="a44fd-120">Eles são úteis para usar um pequeno subconjunto de uma fonte de dados grande e interconectada.</span><span class="sxs-lookup"><span data-stu-id="a44fd-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

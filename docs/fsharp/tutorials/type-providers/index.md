<span data-ttu-id="6d3ce-107">Um provedor de tipos F# é um componente que fornece tipos, propriedades e métodos para uso em seu programa.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-107">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="6d3ce-108">Provedores de tipos são uma parte significativa do suporte de F# 3.0 para programação rica de informações.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-108">Type providers are a significant part of F# 3.0 support for information-rich programming.</span></span> <span data-ttu-id="6d3ce-109">A chave para programação rica em informações é eliminar barreiras para trabalhar com várias fontes de informação encontradas na Internet e em ambientes empresariais modernos.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-109">The key to information-rich programming is to eliminate barriers to working with diverse information sources found on the Internet and in modern enterprise environments.</span></span> <span data-ttu-id="6d3ce-110">Uma barreira significativa para incluir uma fonte de informação em um programa é a necessidade de representar essas informações como tipos, propriedades e métodos para uso em um ambiente da linguagem de programação.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-110">One significant barrier to including a source of information into a program is the need to represent that information as types, properties, and methods for use in a programming language environment.</span></span> <span data-ttu-id="6d3ce-111">Escrever esses tipos manualmente é muito demorado e difícil de manter.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-111">Writing these types manually is very time-consuming and difficult to maintain.</span></span> <span data-ttu-id="6d3ce-112">Uma alternativa comum é usar um gerador de código que adicione arquivos para seu projeto; no entanto, os tipos convencionais de geração de código não se integram bem nos modos exploratórios de programação com suporte em F# porque o código gerado deve ser substituído sempre que uma referência de serviço é definida.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-112">A common alternative is to use a code generator which adds files to your project; however, the conventional types of code generation do not integrate well into exploratory modes of programming supported by F# because the generated code must be replaced each time a service reference is adjusted.</span></span>

<span data-ttu-id="6d3ce-113">Os tipos fornecidos pelos provedores de tipo F# geralmente são baseados em fontes de informações externas.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-113">The types provided by F# type providers are usually based on external information sources.</span></span> <span data-ttu-id="6d3ce-114">Por exemplo, um provedor de tipos F# para SQL fornecerá os tipos, as propriedades e os métodos necessários para trabalhar diretamente com as tabelas de qualquer banco de dados SQL ao qual você tenha acesso.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-114">For example, an F# type provider for SQL will provide the types, properties, and methods you need to work directly with the tables of any SQL database you have access to.</span></span> <span data-ttu-id="6d3ce-115">Da mesma forma, um provedor de tipos para serviços Web WSDL fornecerá os tipos, as propriedades e os métodos necessários para trabalhar diretamente com qualquer serviço Web WSDL.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-115">Similarly, a type provider for WSDL web services will provide the types, properties, and methods you need to work directly with any WSDL web service.</span></span>

<span data-ttu-id="6d3ce-116">O conjunto de tipos, propriedades e métodos fornecidos por um provedor de tipos F# pode depender de parâmetros fornecidos no código de programa.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-116">The set of types, properties, and methods provided by an F# type provider can depend on parameters given in program code.</span></span> <span data-ttu-id="6d3ce-117">Por exemplo, um provedor de tipos pode fornecer tipos diferentes dependendo de uma cadeia de conexão ou de uma URL de serviço.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-117">For example, a type provider can provide different types depending on a connection string or a service URL.</span></span> <span data-ttu-id="6d3ce-118">Dessa forma, o espaço de informações disponível por meio de uma cadeia de conexão ou URL é integrado diretamente em seu programa.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-118">In this way, the information space available by means of a connection string or URL is directly integrated into your program.</span></span> <span data-ttu-id="6d3ce-119">Um provedor de tipos também pode garantir que os grupos de tipos sejam expandidos sob demanda; isto é, eles serão expandidos se os tipos forem realmente referenciados pelo seu programa.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-119">A type provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="6d3ce-120">Isso permite a integração direta e sob demanda de espaços de informações em grande escala como mercados de dados online de uma forma fortemente tipada.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-120">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="6d3ce-121">O F# contém vários provedores de tipos internos para serviços de dados corporativos e da Internet comumente usados.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-121">F# contains several built-in type providers for commonly used Internet and enterprise data services.</span></span> <span data-ttu-id="6d3ce-122">Esses provedores de tipos fornecem acesso simples e normal a bancos de dados SQL relacionais e a serviços OData e WSDL baseados em rede e oferecem suporte auso de consultas F# LINQ com base nessas fontes de dados.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-122">These type providers give simple and regular access to SQL relational databases and network-based OData and WSDL services and support the use of F# LINQ queries against these data sources.</span></span>

<span data-ttu-id="6d3ce-123">Onde necessário, você poderá criar seus próprios provedores de tipos personalizados ou referenciar provedores de tipos que foram criados por outros.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-123">Where necessary, you can create your own custom type providers, or reference type providers that have been created by others.</span></span> <span data-ttu-id="6d3ce-124">Por exemplo, suponha que sua organização tenha um serviço de dados que forneça um grande e crescente número de conjuntos de dados nomeados, cada um com seu próprio esquema de dados estáveis.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-124">For example, assume your organization has a data service providing a large and growing number of named data sets, each with its own stable data schema.</span></span> <span data-ttu-id="6d3ce-125">Você pode optar por criar um provedor de tipos que leia os esquemas e apresente os conjuntos de dados mais recentes disponíveis para o programador de uma forma fortemente tipada.</span><span class="sxs-lookup"><span data-stu-id="6d3ce-125">You may choose to create a type provider that reads the schemas and presents the latest available data sets to the programmer in a strongly typed way.</span></span>

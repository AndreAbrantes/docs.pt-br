 <span data-ttu-id="8e611-145">O elemento `webMessageEncoding` funciona delegando a uma série de codificadores internos para manipular as codificações XML e JSON de texto sem formatação e dados binários "brutos".</span><span class="sxs-lookup"><span data-stu-id="8e611-145">The `webMessageEncoding` element works by delegating to a series of inner encoders to handle the plain-text XML and JSON encodings, and "raw" binary data.</span></span> <span data-ttu-id="8e611-146">Essa delegação é feita por um codificador de mensagem composta.</span><span class="sxs-lookup"><span data-stu-id="8e611-146">This delegation is done by a composite message encoder.</span></span>

# <a name="headers-element"></a>\<cabeçalhos > elemento
Isto contém uma coleção de cabeçalhos de endereço que são necessários para tratar corretamente o emissor local. Você pode usar o `add` palavra-chave para adicionar um cabeçalho a esta coleção.

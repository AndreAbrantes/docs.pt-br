 <span data-ttu-id="230ac-207">Essa associação pode ser usada para se comunicar de forma confiável por meio de um ou mais intermediários do SOAP.</span><span class="sxs-lookup"><span data-stu-id="230ac-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>  
  
 <span data-ttu-id="230ac-208">Por padrão, essa associação gera uma pilha de tempo de execução com WS-ReliableMessaging para confiabilidade, WS-Security para segurança de mensagem e autenticação, o HTTP para a entrega de mensagem e uma codificação de mensagem de texto/XML.</span><span class="sxs-lookup"><span data-stu-id="230ac-208">By default, this binding generates a runtime stack with WS-ReliableMessaging for reliability, WS-Security for message security and authentication, HTTP for message delivery, and a Text/XML message encoding.</span></span>  
  
## <a name="example"></a><span data-ttu-id="230ac-209">Exemplo</span><span class="sxs-lookup"><span data-stu-id="230ac-209">Example</span></span>  
  
```xml  
<configuration>  
<system.ServiceModel>  
<bindings>  
<wsDualHttpBinding>  
    <binding   
        closeTimeout="00:00:10"  
        openTimeout="00:00:20"   
        receiveTimeout="00:00:30"  
        sendTimeout="00:00:40"  
        bypassProxyOnLocal="false"   
        clientBaseAddress="http://localhost:8001/client/"  
        transactionFlow="true"   
        hostNameComparisonMode="WeakWildcard"  
        maxReceivedMessageSize="1000"  
        messageEncoding="Mtom"   
        proxyAddress="http://foo/bar"   
        textEncoding="utf-16"  
        useDefaultWebProxy="false">  
        <reliableSession ordered="false"  
            inactivityTimeout="00:02:00" />  
        <security mode="None">  
            <message clientCredentialType="None"  
                negotiateServiceCredential="false"  
                algorithmSuite="Aes128" />  
        </security>  
    </binding>  
</wsDualHttpBinding>  
</bindings>  
</system.ServiceModel>  
</configuration>  
```

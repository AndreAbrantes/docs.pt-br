# <a name="mitigate-new-behaviors-in-net-framework-46-and-later"></a>Mitigar novos comportamentos no Framework .NET 4.6 e posterior

Esta seção contém artigos que descrevem mitigações para comportamentos que foram introduzidos no .NET Framework 4.6 e posteriores.

Considere mitigar um novo comportamento se ele causar problemas para o seu aplicativo ou for indesejável. Em alguns casos, você também pode *habilitar* um novo comportamento em aplicativos que visam uma versão mais antiga do .NET Framework, mas estão sendo executados no .NET Framework 4.6 ou posterior.

# <a name="about-gdi-managed-code"></a>Sobre código gerenciado no GDI+

GDI+ é a parte do sistema operacional Windows que fornece gráficos de vetor bidimensionais, imagens e tipografia. O GDI+ melhora o GDI (o Graphics Device Interface incluído nas versões anteriores do Windows) adicionando novos recursos e otimizando os recursos existentes.

A interface de classe gerenciada GDI+ (um conjunto de wrappers) faz parte do .NET Framework, um ambiente para compilar, implantar e executar Web Services XML e outros aplicativos.

Esta seção fornece informações sobre a API GDI+ para programadores usando código gerenciado.

# <a name="coordinate-systems-and-transformations"></a>Sistemas de coordenadas e transformações
GDI+ fornece uma transformação global e uma transformação de página para que você pode transformar (girar, dimensionar, traduzir e assim por diante) dos itens que você desenhar. As duas transformações também permitem que você trabalhe em uma variedade de sistemas de coordenadas.

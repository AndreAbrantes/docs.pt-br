# <a name="using-regions"></a>Usando regiões
O [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> classe permite que você defina uma forma personalizada. A forma pode ser composta por linhas, polígonos e curvas.  
  
 Dois usos comuns para as regiões são atingidos, teste e recorte. Teste de clique é determinar se o mouse foi clicado em uma determinada região da tela. Recorte está restringindo o desenho a uma determinada região.

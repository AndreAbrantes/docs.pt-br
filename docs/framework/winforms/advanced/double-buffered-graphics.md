# <a name="double-buffered-graphics"></a><span data-ttu-id="31df0-102">Elementos gráficos em buffer duplo</span><span class="sxs-lookup"><span data-stu-id="31df0-102">Double Buffered Graphics</span></span>
<span data-ttu-id="31df0-103">A cintilação é um problema comum na programação de gráficos.</span><span class="sxs-lookup"><span data-stu-id="31df0-103">Flicker is a common problem when programming graphics.</span></span> <span data-ttu-id="31df0-104">Operações gráficas que requerem várias operações de pintura complexas podem fazer as imagens renderizadas parecerem cintilar ou ter uma aparência inaceitável por outro motivo.</span><span class="sxs-lookup"><span data-stu-id="31df0-104">Graphics operations that require multiple complex painting operations can cause the rendered images to appear to flicker or have an otherwise unacceptable appearance.</span></span> <span data-ttu-id="31df0-105">Para resolver esses problemas, o .NET Framework dá acesso a buffer duplo.</span><span class="sxs-lookup"><span data-stu-id="31df0-105">To address these problems, the .NET Framework provides access to double buffering.</span></span>

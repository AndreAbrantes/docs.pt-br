     <span data-ttu-id="788cb-112">Isso designa o formulário como um recipiente MDI para janelas filho.</span><span class="sxs-lookup"><span data-stu-id="788cb-112">This designates the form as an MDI container for child windows.</span></span>

3.  <span data-ttu-id="788cb-115">Na **Caixa de Ferramentas**, arraste um controle **MenuStrip** para o formulário.</span><span class="sxs-lookup"><span data-stu-id="788cb-115">From the **Toolbox**, drag a **MenuStrip** control to the form.</span></span> <span data-ttu-id="788cb-116">Crie um item de menu de nível superior com a propriedade **Text** definida para **&File** com itens de submenu chamados **&New** e **&Close**.</span><span class="sxs-lookup"><span data-stu-id="788cb-116">Create a top-level menu item with the **Text** property set to **&File** with submenu items called **&New** and **&Close**.</span></span> <span data-ttu-id="788cb-117">Crie também um item de menu de nível superior chamado **&Window**.</span><span class="sxs-lookup"><span data-stu-id="788cb-117">Also create a top-level menu item called **&Window**.</span></span>

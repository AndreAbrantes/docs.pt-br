# <a name="using-transformations-in-managed-gdi"></a>Usando transformações no GDI+ gerenciado
Transformações afins incluem rotação, dimensionamento, refletindo, distorção e conversão. Em [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)], o <xref:System.Drawing.Drawing2D.Matrix> classe fornece a base para executar transformações afins desenhos de vetor, imagens e texto.

# <a name="using-transformations-in-managed-gdi"></a>Usando transformações no GDI+ gerenciado
Transformações afins incluem rotação, dimensionamento, refletindo, distorção e tradução. Na [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)], o <xref:System.Drawing.Drawing2D.Matrix> classe fornece a base para a execução de transformações afins em desenhos vetoriais, imagens e texto.

## <a name="what-are-application-settings"></a><span data-ttu-id="cbac5-110">O que são as configurações de aplicativo?</span><span class="sxs-lookup"><span data-stu-id="cbac5-110">What Are Application Settings?</span></span>  
 <span data-ttu-id="cbac5-111">Seus aplicativos dos Windows Forms frequentemente exigirão dados que são críticos para a execução do aplicativo, mas que você não quer incluir diretamente no código do aplicativo.</span><span class="sxs-lookup"><span data-stu-id="cbac5-111">Your Windows Forms applications will often require data that is critical to running the application, but which you do not want to include directly in the application's code.</span></span> <span data-ttu-id="cbac5-112">Se seu aplicativo usar um serviço Web ou um servidor de banco de dados, convém armazenar essas informações em um arquivo separado para que você possa alterá-las no futuro sem recompilação.</span><span class="sxs-lookup"><span data-stu-id="cbac5-112">If your application uses a Web Service or a database server, you may want to store this information in a separate file, so that you can change it in the future without re-compiling.</span></span> <span data-ttu-id="cbac5-113">Da mesma forma, seus aplicativos podem exigir o armazenamento de dados que são específicos ao usuário atual.</span><span class="sxs-lookup"><span data-stu-id="cbac5-113">Similarly, your applications may require storing data that is specific to the current user.</span></span> <span data-ttu-id="cbac5-114">A maioria dos aplicativos, por exemplo, tem preferências do usuário que personalizam a aparência e o comportamento do aplicativo.</span><span class="sxs-lookup"><span data-stu-id="cbac5-114">Most applications, for example, have user preferences that customize the application's appearance and behavior.</span></span>  
  
 <span data-ttu-id="cbac5-115">As configurações de aplicativo abordam essas duas necessidades fornecendo uma maneira fácil de armazenar configurações de escopo do aplicativo e de escopo do usuário no computador cliente.</span><span class="sxs-lookup"><span data-stu-id="cbac5-115">Application settings addresses both needs by providing an easy way to store both application-scoped and user-scoped settings on the client computer.</span></span> <span data-ttu-id="cbac5-116">Usando o Visual Studio ou um editor de código, você define uma configuração para uma determinada propriedade especificando seu nome, tipo de dados e escopo (aplicativo ou usuário).</span><span class="sxs-lookup"><span data-stu-id="cbac5-116">Using Visual Studio or a code editor, you define a setting for a given property by specifying its name, data type, and scope (application or user).</span></span> <span data-ttu-id="cbac5-117">Você pode até mesmo colocar configurações relacionadas em grupos nomeados para facilitar o uso e a legibilidade.</span><span class="sxs-lookup"><span data-stu-id="cbac5-117">You can even place related settings into named groups for easier use and readability.</span></span> <span data-ttu-id="cbac5-118">Uma vez definidas, essas configurações são mantidas e lidas na memória automaticamente em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="cbac5-118">Once defined, these settings are persisted and read back into memory automatically at run time.</span></span> <span data-ttu-id="cbac5-119">Uma arquitetura conectável permite que o mecanismo de persistência seja alterado, mas, por padrão, o sistema de arquivos local é usado.</span><span class="sxs-lookup"><span data-stu-id="cbac5-119">A pluggable architecture enables the persistence mechanism to be changed, but by default, the local file system is used.</span></span>  
  
 <span data-ttu-id="cbac5-120">As configurações de aplicativo funcionam gravando dados como XML em arquivos de configuração (.config) diferentes, que indicam se a configuração tem escopo do aplicativo ou escopo do usuário.</span><span class="sxs-lookup"><span data-stu-id="cbac5-120">Application settings works by persisting data as XML to different configuration (.config) files, corresponding to whether the setting is application-scoped or user-scoped.</span></span> <span data-ttu-id="cbac5-121">Na maioria dos casos, as configurações de escopo do aplicativo são somente leitura.  <span data-ttu-id="cbac5-124">As configurações são armazenadas como fragmentos XML em arquivos de configuração.</span><span class="sxs-lookup"><span data-stu-id="cbac5-124">Settings are stored as XML fragments in configuration files.</span></span> <span data-ttu-id="cbac5-125">Configurações com escopo do aplicativo são representadas pelo elemento `<application.Settings>` e geralmente são colocadas em *app*.exe.config, em que *app* é o nome do seu arquivo executável principal.</span><span class="sxs-lookup"><span data-stu-id="cbac5-125">Application-scoped settings are represented by the `<application.Settings>` element, and generally are placed in *app*.exe.config, where *app* is the name of your main executable file.</span></span> <span data-ttu-id="cbac5-126">As configurações com escopo do usuário são representadas pelo elemento `<userSettings>` e são colocadas em *usuário*.config, em que *usuário* é o nome de usuário da pessoa que está executando o aplicativo no momento.</span><span class="sxs-lookup"><span data-stu-id="cbac5-126">User-scoped settings are represented by the `<userSettings>` element and are placed in *user*.config, where *user* is the user name of the person currently running the application.</span></span> <span data-ttu-id="cbac5-127">Você precisa implantar o arquivo *app*.exe.config com seu aplicativo.  <span data-ttu-id="cbac5-129">Controles personalizados também podem salvar suas próprias configurações implementando a <xref:System.Configuration.IPersistComponentSettings> da interface, que expõe o <xref:System.Configuration.IPersistComponentSettings.SaveSettings%2A> método.</span><span class="sxs-lookup"><span data-stu-id="cbac5-129">Custom controls can also save their own settings by implementing the <xref:System.Configuration.IPersistComponentSettings> interface, which exposes the <xref:System.Configuration.IPersistComponentSettings.SaveSettings%2A> method.</span></span> <span data-ttu-id="cbac5-130">Os formulários do Windows <xref:System.Windows.Forms.ToolStrip> controle implementa essa interface para salvar a posição das barras de ferramentas e itens de barra de ferramentas entre as sessões do aplicativo.</span><span class="sxs-lookup"><span data-stu-id="cbac5-130">The Windows Forms <xref:System.Windows.Forms.ToolStrip> control implements this interface to save the position of toolbars and toolbar items between application sessions.</span></span> <span data-ttu-id="cbac5-131">Para obter mais informações sobre controles personalizados e configurações do aplicativo, consulte [Configurações do aplicativo para controles personalizados](../../../../docs/framework/winforms/advanced/application-settings-for-custom-controls.md).</span><span class="sxs-lookup"><span data-stu-id="cbac5-131">For more information about custom controls and application settings, see [Application Settings for Custom Controls](../../../../docs/framework/winforms/advanced/application-settings-for-custom-controls.md).</span></span>  
  
## <a name="limitations-of-application-settings"></a><span data-ttu-id="cbac5-132">Limitações das configurações de aplicativo</span><span class="sxs-lookup"><span data-stu-id="cbac5-132">Limitations of Application Settings</span></span>

 <span data-ttu-id="cbac5-135">Atualmente, não é possível associar a algumas propriedades nos Windows Forms.</span><span class="sxs-lookup"><span data-stu-id="cbac5-135">You currently cannot bind to some properties in Windows Forms.</span></span> <span data-ttu-id="cbac5-136">O exemplo mais notável é o <xref:System.Windows.Forms.Form.ClientSize%2A> propriedade, como associação a essa propriedade causaria um comportamento imprevisível em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="cbac5-136">The most notable example is the <xref:System.Windows.Forms.Form.ClientSize%2A> property, as binding to this property would cause unpredictable behavior at run time.</span></span> <span data-ttu-id="cbac5-137">Geralmente, você pode contornar esses problemas salvando e carregando essas configurações programaticamente.</span><span class="sxs-lookup"><span data-stu-id="cbac5-137">You can usually work around these issues by saving and loading these settings programmatically.</span></span>

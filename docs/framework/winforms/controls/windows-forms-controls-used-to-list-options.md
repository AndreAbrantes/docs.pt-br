# <a name="windows-forms-controls-used-to-list-options"></a>Controles dos Windows Forms usados para listar opções
Você pode adicionar uma variedade de controles a um Windows Form para fornecer aos usuários uma lista de opções de escolha. Dependendo de quanto você deseja restringir a entrada de seus usuários, você pode adicionar um controle de <xref:System.Windows.Forms.ListBox>, um controle de <xref:System.Windows.Forms.ComboBox> ou um controle de <xref:System.Windows.Forms.CheckedListBox>. Use os links a seguir para determinar qual controle atende mais bem às suas necessidades.

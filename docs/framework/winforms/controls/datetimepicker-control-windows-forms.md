# <a name="datetimepicker-control-windows-forms"></a>Controle DateTimePicker (Windows Forms)
O controle de `DateTimePicker` de Windows Forms permite que o usuário selecione um único item de uma lista de datas ou horas. Quando usado para representar uma data, ele aparece em duas partes: uma lista suspensa com uma data representada em texto e uma grade exibida ao clicar na seta para baixo ao lado da lista.

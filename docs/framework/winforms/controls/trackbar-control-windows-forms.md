# <a name="trackbar-control-windows-forms"></a>Controle TrackBar (Windows Forms)
O controle de `TrackBar` de Windows Forms (também chamado de controle "slider") é usado para navegar por uma grande quantidade de informações ou para ajustar visualmente uma configuração numérica. O controle de `TrackBar` tem duas partes: o Thumb, também conhecido como um controle deslizante e as marcas de escala. O elevador é parte que pode ser ajustada. Sua posição corresponde à propriedade <xref:System.Windows.Forms.TrackBar.Value%2A>. As marcas de escala são indicadores visuais espaçados em intervalos regulares. A barra de controle é movida em incrementos que você especifica e pode ser alinhada horizontal ou verticalmente. Um exemplo de uso de uma barra de faixa seria para definir a taxa de intermitência do cursor ou a velocidade do mouse.

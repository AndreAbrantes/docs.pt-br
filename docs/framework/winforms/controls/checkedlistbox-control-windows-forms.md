# <a name="checkedlistbox-control-windows-forms"></a>Controle CheckedListBox (Windows Forms)
O controle de `CheckedListBox` de Windows Forms exibe uma lista de itens, como o controle de <xref:System.Windows.Forms.ListBox>, e também pode exibir uma marca de seleção ao lado de itens na lista.

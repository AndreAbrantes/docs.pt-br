# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>Como posicionar um ToolStripItem em um ToolStrip
Você pode mover ou adicionar uma <xref:System.Windows.Forms.ToolStripItem> à esquerda ou à direita de um <xref:System.Windows.Forms.ToolStrip>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>Para mover ou adicionar um ToolStripItem para o lado esquerdo de um ToolStrip  
  
1.  Definir o <xref:System.Windows.Forms.ToolStripItemAlignment> propriedade o <xref:System.Windows.Forms.ToolStripItem> para <xref:System.Windows.Forms.ToolStripItemAlignment.Left>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>Para mover ou adicionar um ToolStripItem à direita de um ToolStrip  
  
1.  Definir o <xref:System.Windows.Forms.ToolStripItemAlignment> propriedade o <xref:System.Windows.Forms.ToolStripItem> para <xref:System.Windows.Forms.ToolStripItemAlignment.Right>.

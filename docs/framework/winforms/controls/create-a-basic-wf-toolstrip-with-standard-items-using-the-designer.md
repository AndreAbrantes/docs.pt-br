# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Como: Criar um ToolStrip de formulários do Windows básico com itens padrão usando o Designer
O procedimento a seguir demonstra como criar uma <xref:System.Windows.Forms.ToolStrip> e adicionar sete <xref:System.Windows.Forms.ToolStripButton> controles que representam tarefas comuns.  
  
> [!NOTE]
>  As caixas de diálogo e os comandos de menu que você vê podem ser diferentes dos descritos na Ajuda, dependendo da sua edição ou das configurações ativas. Para alterar as configurações, escolha **Importar e Exportar Configurações** no menu **Ferramentas**. Para obter mais informações, confira [Personalizar o IDE do Visual Studio](/visualstudio/ide/personalizing-the-visual-studio-ide).  
  
### <a name="to-add-standard-items-in-the-designer"></a>Para adicionar itens padrão no designer  
  
1.  criar um <xref:System.Windows.Forms.ToolStrip> controle.  
  
2.  No canto superior direito do <xref:System.Windows.Forms.ToolStrip>, clique na seta de tarefa inteligente para exibir o **tarefas do ToolStrip** painel.  
  
3.  No **tarefas do ToolStrip** painel, escolha **inserir itens padrão**.

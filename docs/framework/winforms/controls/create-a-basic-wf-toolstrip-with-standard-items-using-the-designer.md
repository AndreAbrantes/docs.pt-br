# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Como: Criar uma faixa de Windows Forms básica com itens padrão usando o designer
O procedimento a seguir demonstra como criar um <xref:System.Windows.Forms.ToolStrip> e adicionar sete <xref:System.Windows.Forms.ToolStripButton> controles que representam tarefas típicas.

## <a name="to-add-standard-items-in-the-designer"></a>Para adicionar itens padrão no designer

1. Criar um <xref:System.Windows.Forms.ToolStrip> controle.

2. No canto superior direito do <xref:System.Windows.Forms.ToolStrip>, clique na seta de tarefa inteligente para exibir o painel **tarefas ToolStrip** .

3. No painel **tarefas de ToolStrip** , escolha **Inserir itens padrão**.

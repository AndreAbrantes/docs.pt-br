# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Como criar um ToolStrip básico dos Windows Forms com itens padrão usando o designer
O procedimento a seguir demonstra como criar um <xref:System.Windows.Forms.ToolStrip> e adicionar sete <xref:System.Windows.Forms.ToolStripButton> controles que representam tarefas típicas.

## <a name="to-add-standard-items-in-the-designer"></a>Para adicionar itens padrão no designer

1. Crie um controle de <xref:System.Windows.Forms.ToolStrip>.

2. No canto superior direito da <xref:System.Windows.Forms.ToolStrip>, clique na seta de tarefa inteligente para exibir o painel **tarefas ToolStrip** .

3. No painel **tarefas de ToolStrip** , escolha **Inserir itens padrão**.

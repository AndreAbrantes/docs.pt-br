# <a name="flowlayoutpanel-control-windows-forms"></a>Controle FlowLayoutPanel (Windows Forms)
O controle de `FlowLayoutPanel` organiza seu conteúdo em uma direção de fluxo horizontal ou vertical. Seu conteúdo pode ser encapsulado de uma linha à outra ou de uma coluna à próxima. Como alternativa, seu conteúdo pode ser recortado, em vez de encapsulado.  
  
 Os tópicos nesta seção descrevem os conceitos e as técnicas que permitem que você crie `FlowLayoutPanel` recursos em seus aplicativos.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Visão geral do controle FlowLayoutPanel](flowlayoutpanel-control-overview.md)  
 Apresenta os conceitos gerais do controle de `FlowLayoutPanel`, que permite criar um layout que flui horizontal ou verticalmente.  
  
 [Como ancorar e encaixar controles filho em um controle FlowLayoutPanel](how-to-anchor-and-dock-child-controls-in-a-flowlayoutpanel-control.md)  
 Explica como usar as propriedades <xref:System.Windows.Forms.Control.Anchor%2A> e <xref:System.Windows.Forms.Control.Dock%2A> para ancorar e encaixar controles filho em um controle de <xref:System.Windows.Forms.FlowLayoutPanel>.  
  
 Consulte também [Explicação passo a passo: organizando controles nos Windows Forms utilizando um FlowLayoutPanel](walkthrough-arranging-controls-on-windows-forms-using-a-flowlayoutpanel.md).

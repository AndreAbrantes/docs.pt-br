# <a name="folderbrowserdialog-component-windows-forms"></a>Componente FolderBrowserDialog (Windows Forms)
Os formulários do Windows `FolderBrowserDialog` componente exibe uma interface com a qual os usuários podem procurar e selecionar uma pasta ou criar um novo. É um complemento para o [componente OpenFileDialog](openfiledialog-component-windows-forms.md) componente, que é usado para procurar e selecionar os arquivos.

# <a name="folderbrowserdialog-component-windows-forms"></a>Componente FolderBrowserDialog (Windows Forms)
O componente Windows Forms `FolderBrowserDialog` exibe uma interface com a qual os usuários podem procurar e selecionar uma pasta ou criar uma nova. É um complemento do componente de [componente OpenFileDialog](openfiledialog-component-windows-forms.md) , que é usado para procurar e selecionar arquivos.

# <a name="toolstrippanel-control-overview"></a>Visão geral do controle ToolStripPanel
Um <xref:System.Windows.Forms.ToolStripPanel> fornece uma única área de posicionamento e rafting <xref:System.Windows.Forms.ToolStrip>, <xref:System.Windows.Forms.MenuStrip>, e <xref:System.Windows.Forms.StatusStrip> controles. Vários <xref:System.Windows.Forms.ToolStrip> controles pilha vertical ou horizontalmente, dependendo do <xref:System.Windows.Forms.ToolStripPanelRow.Orientation%2A> do <xref:System.Windows.Forms.ToolStripPanel>.  
  
### <a name="important-toolstrippanel-members"></a>Membros importantes ToolStripPanel  
  
|Nome|Descrição|  
|----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripPanel.Orientation%2A>|Obtém ou define um valor que indica a orientação horizontal ou vertical do <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Renderer%2A>|Obtém ou define um <xref:System.Windows.Forms.ToolStripRenderer> usado para personalizar a aparência de um <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.RenderMode%2A>|Obtém ou define os estilos de pintura a serem aplicadas para o <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.RowMargin%2A>|Obtém ou define o espaçamento, em pixels, entre a <xref:System.Windows.Forms.ToolStripPanelRow> e <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Rows%2A>|Obtém o <xref:System.Windows.Forms.ToolStripPanelRow> neste <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Join%2A>|Adiciona um <xref:System.Windows.Forms.ToolStrip> a um <xref:System.Windows.Forms.ToolStripPanel>.|

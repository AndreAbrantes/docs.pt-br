<span data-ttu-id="1377a-103">Alguns documentos HTML são compostos de *quadros* ou janelas que podem manter seus próprios documentos HTML distintos.</span><span class="sxs-lookup"><span data-stu-id="1377a-103">Some HTML documents are composed out of *frames*, or windows that can hold their own distinct HTML documents.</span></span> <span data-ttu-id="1377a-104">Usar quadros facilita a criação de páginas HTML na qual uma ou mais partes da página permanecem estáticas, como uma barra de navegação, enquanto outros quadros alterar seu conteúdo constantemente.</span><span class="sxs-lookup"><span data-stu-id="1377a-104">Using frames makes it easy to create HTML pages in which one or more pieces of the page remain static, such as a navigation bar, while other frames constantly change their content.</span></span>  
  
 <span data-ttu-id="1377a-105">Criadores de HTML podem criar quadros de duas maneiras:</span><span class="sxs-lookup"><span data-stu-id="1377a-105">HTML authors can create frames in one of two ways:</span></span>  
  
-   <span data-ttu-id="1377a-106">Usando as marcas `FRAMESET` e `FRAME`, que criam janelas fixas.</span><span class="sxs-lookup"><span data-stu-id="1377a-106">Using the `FRAMESET` and `FRAME` tags, which create fixed windows.</span></span>  
  
 <span data-ttu-id="1377a-107">- ou -</span><span class="sxs-lookup"><span data-stu-id="1377a-107">-or-</span></span>  
  
-   <span data-ttu-id="1377a-108">Usando a marca `IFRAME`, que cria uma janela flutuante que pode ser reposicionada em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="1377a-108">Using the `IFRAME` tag, which creates a floating window that can be repositioned at run time.</span></span>  
  
1. <span data-ttu-id="1377a-109">Como os quadros contêm documentos HTML, eles são representados no Modelo de Objeto do Documento (DOM) como elementos de janela e de quadro.</span><span class="sxs-lookup"><span data-stu-id="1377a-109">Because frames contain HTML documents, they are represented in the Document Object Model (DOM) as both window elements and frame elements.</span></span>  
  
2. <span data-ttu-id="1377a-110">Quando você acessa uma `FRAME` ou `IFRAME` marca usando a coleção de quadros de <xref:System.Windows.Forms.HtmlWindow>, o elemento de janela correspondente ao quadro.</span><span class="sxs-lookup"><span data-stu-id="1377a-110">When you access a `FRAME` or `IFRAME` tag by using the Frames collection of <xref:System.Windows.Forms.HtmlWindow>, you are retrieving the window element corresponding to the frame.</span></span> <span data-ttu-id="1377a-111">Isso representa todas as propriedades dinâmicas do quadro, como sua URL, documento e tamanho atual.</span><span class="sxs-lookup"><span data-stu-id="1377a-111">This represents all of the frame's dynamic properties, such as its current URL, document, and size.</span></span>  
  
3. <span data-ttu-id="1377a-112">Quando você acessa uma `FRAME` ou `IFRAME` marca usando o <xref:System.Windows.Forms.HtmlWindow.WindowFrameElement%2A> propriedade do <xref:System.Windows.Forms.HtmlWindow>, o <xref:System.Windows.Forms.HtmlElement.Children%2A> coleção ou métodos como <xref:System.Windows.Forms.HtmlElementCollection.GetElementsByName%2A> ou <xref:System.Windows.Forms.HtmlDocument.GetElementById%2A>, você está recuperando o elemento de quadro.</span><span class="sxs-lookup"><span data-stu-id="1377a-112">When you access a `FRAME` or `IFRAME` tag by using the <xref:System.Windows.Forms.HtmlWindow.WindowFrameElement%2A> property of <xref:System.Windows.Forms.HtmlWindow>, the <xref:System.Windows.Forms.HtmlElement.Children%2A> collection, or methods such as <xref:System.Windows.Forms.HtmlElementCollection.GetElementsByName%2A> or <xref:System.Windows.Forms.HtmlDocument.GetElementById%2A>, you are retrieving the frame element.</span></span> <span data-ttu-id="1377a-113">Isso representa as propriedades estáticas do quadro, incluindo a URL especificada no arquivo HTML original.</span><span class="sxs-lookup"><span data-stu-id="1377a-113">This represents the static properties of the frame, including the URL specified in the original HTML file.</span></span>

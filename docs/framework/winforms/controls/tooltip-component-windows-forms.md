# <a name="tooltip-component-windows-forms"></a>Componente ToolTip (Windows Forms)
Windows Forms <xref:System.Windows.Forms.ToolTip> componente exibe texto quando o usuário aponta para controles. Uma dica de ferramenta pode ser associada a qualquer controle. Um exemplo de uso deste controle: para economizar espaço em um formulário, você pode exibir um pequeno ícone em um botão e use uma dica de ferramenta para explicar a função do botão.

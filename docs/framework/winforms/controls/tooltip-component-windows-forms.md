# <a name="tooltip-component-windows-forms"></a>Componente ToolTip (Windows Forms)
O componente Windows Forms <xref:System.Windows.Forms.ToolTip> exibe texto quando o usuário aponta para os controles. Uma dica de ferramenta pode ser associada a qualquer controle. Um exemplo de uso deste controle: para economizar espaço em um formulário, você pode exibir um ícone pequeno em um botão e usar uma dica de ferramenta para explicar a função do botão.

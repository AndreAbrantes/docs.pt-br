# <a name="tabcontrol-control-windows-forms"></a>Controle TabControl (Windows Forms)
O Windows Forms `TabControl` exibe várias guias, como os divisores em um bloco de anotações ou rótulos em um conjunto de pastas em um gabinete de arquivamento. As guias podem conter imagens e outros controles. Use o `TabControl` para criar páginas de propriedades.

# <a name="checkbox-control-windows-forms"></a>Controle CheckBox (Windows Forms)
Windows Forms `CheckBox` controle indica se uma determinada condição está ativado ou desativado. É usado normalmente para apresentar uma seleção Sim/Não ou Verdadeiro/Falso ao usuário. Use os controles de caixa de seleção em grupos para exibir múltiplas opções entre as quais o usuário pode escolher uma ou mais. É semelhante do <xref:System.Windows.Forms.RadioButton> agrupados de controle, mas qualquer número de `CheckBox` controles podem ser selecionados.

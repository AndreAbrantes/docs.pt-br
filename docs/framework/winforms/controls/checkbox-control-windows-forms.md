# <a name="checkbox-control-windows-forms"></a>Controle CheckBox (Windows Forms)
Os formulários do Windows `CheckBox` controle indica se uma determinada condição é ativada ou desativada. É usado normalmente para apresentar uma seleção Sim/Não ou Verdadeiro/Falso ao usuário. Use os controles de caixa de seleção em grupos para exibir múltiplas opções entre as quais o usuário pode escolher uma ou mais. Ele é semelhante ao <xref:System.Windows.Forms.RadioButton> agrupados de controle, mas qualquer número de `CheckBox` controles podem ser selecionados.

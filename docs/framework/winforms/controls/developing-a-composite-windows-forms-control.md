# <a name="develop-a-composite-windows-forms-control"></a>Desenvolver um controle de Windows Forms de composição

Você pode desenvolver um controle de Windows Forms de composição combinando outros controles de Windows Forms. Os controles compostos que derivam de são chamados de controles de <xref:System.Web.UI.UserControl> usuário. A classe base, <xref:System.Windows.Forms.UserControl>, fornece roteamento de teclado para os controles filho, garantindo assim que os controles filho possam receber o foco. Para obter um exemplo de controle de usuário, consulte <xref:System.Windows.Forms.UserControl> o exemplo [em como: Aplicar atributos em controles](how-to-apply-attributes-in-windows-forms-controls.md)de Windows Forms.

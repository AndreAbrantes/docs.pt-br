# <a name="monthcalendar-control-windows-forms"></a>Controle MonthCalendar (Windows Forms)
O controle `MonthCalendar` dos Windows Forms apresenta uma interface gráfica intuitiva para os usuários exibirem e definirem as informações de data. O controle exibe uma grade que contém os dias numerados do mês, organizados em colunas abaixo dos dias da semana. Você pode selecionar um outro mês clicando nos botões de seta em um dos lados da legenda do mês. Ao contrário do controle de <xref:System.Windows.Forms.DateTimePicker> semelhante, você pode selecionar um intervalo de datas com este controle; no entanto, o controle <xref:System.Windows.Forms.DateTimePicker> permite definir horários e datas.

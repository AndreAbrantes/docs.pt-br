# <a name="statusbar-control-windows-forms"></a>Controle StatusBar (Windows Forms)
> [!NOTE]
> O controle <xref:System.Windows.Forms.ToolStripStatusLabel> substitui e adiciona funcionalidade ao controle <xref:System.Windows.Forms.StatusBar>, no entanto, o controle <xref:System.Windows.Forms.StatusBar> é mantido para compatibilidade com versões anteriores e para uso futuro, se desejado.  
  
 O controle de <xref:System.Windows.Forms.StatusBar> de Windows Forms é usado em formulários como uma área, geralmente exibida na parte inferior de uma janela, na qual um aplicativo pode exibir vários tipos de informações de status. os controles de <xref:System.Windows.Forms.StatusBar> podem ter painéis de barra de status que exibem ícones para indicar o estado ou uma série de ícones em uma animação que indicam que um processo está funcionando; por exemplo, o Microsoft Word indica que o documento está sendo salvo.

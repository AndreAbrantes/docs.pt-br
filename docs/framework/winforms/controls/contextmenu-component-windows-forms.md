# <a name="contextmenu-component-windows-forms"></a>Componente ContextMenu (Windows Forms)
> [!NOTE]
> Embora `MenuStrip` e <xref:System.Windows.Forms.ContextMenu> <xref:System.Windows.Forms.ContextMenu> <xref:System.Windows.Forms.MainMenu> <xref:System.Windows.Forms.MainMenu> substituam e adicionem funcionalidade aos controles e de versões anteriores, e são mantidos para compatibilidade com versões anteriores e uso futuro, se você escolher. `ContextMenuStrip`  
  
 O componente `ContextMenu` Windows Forms é usado para fornecer aos usuários um menu de atalho facilmente acessível dos comandos usados com frequência que estão associados ao objeto selecionado. Os itens em um menu de atalho frequentemente são um subconjunto dos itens de menus principais que aparecem em outro lugar no aplicativo. Os menus de atalho estão geralmente disponíveis clicando com o botão direito do mouse. Em Windows Forms eles estão associados a outros controles.

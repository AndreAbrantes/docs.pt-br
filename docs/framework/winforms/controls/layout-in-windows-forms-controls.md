# <a name="layout-in-windows-forms-controls"></a>Layout em controles dos Windows Forms

O posicionamento exato dos controles no formulário é uma prioridade alta para muitos aplicativos. O namespace <xref:System.Windows.Forms?displayProperty=nameWithType> oferece muitas ferramentas de layout para fazer isso.

# <a name="fontdialog-component-windows-forms"></a>Componente FontDialog (Windows Forms)
Os formulários do Windows <xref:System.Windows.Forms.FontDialog> componente é uma caixa de diálogo pré-configurada. É o mesmo **fonte** caixa de diálogo exposta pelo sistema operacional Windows. O componente herda o <xref:System.Windows.Forms.CommonDialog> classe.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Visão geral do componente FontDialog](fontdialog-component-overview-windows-forms.md)  
 Apresenta os conceitos gerais do <xref:System.Windows.Forms.FontDialog> componente, que você pode usar para exibir uma caixa de diálogo pré-configurada. Os usuários podem usar a caixa de diálogo para manipular fontes e suas configurações.

 Explica como escolher uma fonte em tempo de execução por meio de uma instância da <xref:System.Windows.Forms.FontDialog> componente.

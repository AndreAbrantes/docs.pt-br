# <a name="mainmenu-component-windows-forms"></a>Componente MainMenu (Windows Forms)
> [!NOTE]
>  Embora `MenuStrip` e `ContextMenuStrip` substituir e adiciona a funcionalidade para o `MainMenu` e <xref:System.Windows.Forms.ContextMenu> controles de versões anteriores, `MainMenu` e <xref:System.Windows.Forms.ContextMenu> são mantidos para uso futuro e compatibilidade com versões anteriores, se você escolher.  
  
 Windows Forms `MainMenu` componente exibe um menu no tempo de execução.

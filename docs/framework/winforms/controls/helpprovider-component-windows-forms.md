# <a name="helpprovider-component-windows-forms"></a>Componente HelpProvider (Windows Forms)
O componente Windows Forms `HelpProvider` é usado para associar um arquivo de ajuda HTML ajuda 1. x (um arquivo. chm, produzido com o HTML Help Workshop ou um arquivo. htm) com seu aplicativo baseado no Windows.

 Apresenta os conceitos gerais do componente `HelpProvider`, que permite associar um arquivo de ajuda HTML a um aplicativo baseado no Windows.

# <a name="toolstripcontainer-control"></a>Controle ToolStripContainer
<xref:System.Windows.Forms.ToolStrip> controles de recurso rafting interno (compartilhamento de espaço horizontal ou vertical dentro da área de ferramenta quando encaixado) usando o <xref:System.Windows.Forms.ToolStripContainer>.  
  
 Os tópicos nesta seção descrevem os conceitos e técnicas que você pode usar para compilar recursos <xref:System.Windows.Forms.ToolStripContainer> em seus aplicativos.

 Fornece tópicos que descrevem a finalidade e os principais conceitos de formulários do Windows <xref:System.Windows.Forms.ToolStripContainer> controle.  
  
 [Como adicionar um ToolStripContainer a um formulário](../../../../docs/framework/winforms/controls/how-to-add-a-toolstripcontainer-to-a-form.md)

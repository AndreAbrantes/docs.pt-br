# <a name="timer-component-windows-forms"></a>Componente Temporizador (Windows Forms)
O <xref:System.Windows.Forms.Timer> de Windows Forms é um componente que gera um evento em intervalos regulares. Esse componente foi criado para um ambiente dos Windows Forms.

# <a name="bindingnavigator-control-windows-forms"></a>Controle BindingNavigator (Windows Forms)
O `BindingNavigator` controle é a navegação e manipulação de interface do usuário (UI) para controles associados a dados. O `BindingNavigator` controle permite aos usuários navegar e manipular dados em um Windows Form.  
  
 Os tópicos nesta seção fornecem uma visão geral do `BindingNavigator` controlar e oferecem instruções passo a passo sobre como usar o controle navegar em dados e percorrer um <xref:System.Data.DataSet>.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Visão geral do controle BindingNavigator](../../../../docs/framework/winforms/controls/bindingnavigator-control-overview-windows-forms.md)  
 Apresenta os conceitos gerais do `BindingNavigator` controle, que permite aos usuários mover os itens de uma fonte de dados.  
  
 [Como navegar em dados com o controle BindingNavigator dos Windows Forms](../../../../docs/framework/winforms/controls/how-to-navigate-data-with-the-windows-forms-bindingnavigator-control.md)  
 Fornece etapas para associar um `BindingNavigator` controle a uma fonte de dados.  
  
 [Como navegar em um DataSet com o controle BindingNavigator dos Windows Forms](../../../../docs/framework/winforms/controls/move-through-a-dataset-with-wf-bindingnavigator-control.md)  
 Demonstra como usar um `BindingNavigator` controle para percorrer registros em um <xref:System.Data.DataSet>.  
  
 Consulte também [Como adicionar botões Carregar, Salvar e Cancelar ao controle BindingNavigator do Windows Forms](../../../../docs/framework/winforms/controls/load-save-and-cancel-bindingnavigator.md).

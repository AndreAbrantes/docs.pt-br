# <a name="linklabel-control-windows-forms"></a>Controle LinkLabel (Windows Forms)
O controle de `LinkLabel` de Windows Forms permite que você adicione links de estilo da Web a aplicativos Windows Forms. Você pode usar o controle de `LinkLabel` para tudo o que você pode usar para o controle de <xref:System.Windows.Forms.Label>; Você também pode definir parte do texto como um link para um objeto ou página da Web.

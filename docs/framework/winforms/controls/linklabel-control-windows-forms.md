# <a name="linklabel-control-windows-forms"></a>Controle LinkLabel (Windows Forms)
Os formulários do Windows `LinkLabel` controle permite que você adicione links no estilo Web para aplicativos do Windows Forms. Você pode usar o `LinkLabel` controle de tudo o que você pode usar o <xref:System.Windows.Forms.Label> de controle para; você também pode definir a parte do texto como um link para um objeto ou página da Web.

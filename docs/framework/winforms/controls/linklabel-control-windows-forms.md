# <a name="linklabel-control-windows-forms"></a>Controle LinkLabel (Windows Forms)
O controle `LinkLabel` do Windows Forms permite adicionar links no estilo Web aos aplicativos do Windows Forms. Você pode `LinkLabel` usar o controle para <xref:System.Windows.Forms.Label> tudo o que você pode usar o controle para; você também pode definir parte do texto como um link para um objeto ou página da Web.

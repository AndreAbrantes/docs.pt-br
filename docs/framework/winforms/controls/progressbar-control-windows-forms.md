# <a name="progressbar-control-windows-forms"></a>Controle ProgressBar (Windows Forms)
> [!IMPORTANT]
>  O controle <xref:System.Windows.Forms.ToolStripProgressBar> substitui e adiciona funcionalidade ao controle <xref:System.Windows.Forms.ProgressBar>, no entanto, o controle <xref:System.Windows.Forms.ProgressBar> é mantido para compatibilidade com versões anteriores e para uso futuro, se desejado.  
  
 Os formulários do Windows <xref:System.Windows.Forms.ProgressBar> controle indica o progresso de uma ação, exibindo um número apropriado de retângulos organizados em uma barra horizontal. Quando a ação for concluída, a barra será preenchida. Barras de progresso são comumente usadas para dar ao usuário uma indicação de como longa espera para concluir uma ação demorada — por exemplo, quando um arquivo grande está sendo carregado.

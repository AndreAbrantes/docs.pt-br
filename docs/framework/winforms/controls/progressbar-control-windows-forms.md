# <a name="progressbar-control-windows-forms"></a>Controle ProgressBar (Windows Forms)
> [!IMPORTANT]
>  O controle <xref:System.Windows.Forms.ToolStripProgressBar> substitui e adiciona funcionalidade ao controle <xref:System.Windows.Forms.ProgressBar>, no entanto, o controle <xref:System.Windows.Forms.ProgressBar> é mantido para compatibilidade com versões anteriores e para uso futuro, se desejado.  
  
 Windows Forms <xref:System.Windows.Forms.ProgressBar> controle indica o progresso de uma ação, exibindo um número apropriado de retângulos organizados em uma barra horizontal. Quando a ação for concluída, a barra é preenchida. Barras de progresso são usadas para fornecer ao usuário uma indicação de como longos de espera para concluir uma ação demorada — por exemplo, quando um arquivo grande está sendo carregado.

# <a name="progressbar-control-windows-forms"></a>Controle ProgressBar (Windows Forms)
> [!IMPORTANT]
> O controle <xref:System.Windows.Forms.ToolStripProgressBar> substitui e adiciona funcionalidade ao controle <xref:System.Windows.Forms.ProgressBar>, no entanto, o controle <xref:System.Windows.Forms.ProgressBar> é mantido para compatibilidade com versões anteriores e para uso futuro, se desejado.  
  
 O controle de <xref:System.Windows.Forms.ProgressBar> de Windows Forms indica o progresso de uma ação exibindo um número apropriado de retângulos organizados em uma barra horizontal. Quando a ação for concluída, a barra será preenchida. As barras de progresso são comumente usadas para dar ao usuário uma indicação de quanto tempo esperar que uma ação prolongada seja concluída — por exemplo, quando um arquivo grande está sendo carregado.

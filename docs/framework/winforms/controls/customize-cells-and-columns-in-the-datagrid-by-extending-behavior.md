<span data-ttu-id="5b8dc-103">O <xref:System.Windows.Forms.DataGridView> controle fornece várias maneiras de personalizar sua aparência e comportamento usando propriedades, eventos e classe complementares.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-103">The <xref:System.Windows.Forms.DataGridView> control provides a number of ways to customize its appearance and behavior using properties, events, and companion classes.</span></span> <span data-ttu-id="5b8dc-104">Ocasionalmente, você pode ter requisitos para as suas células que ultrapassem o que esses recursos podem fornecer.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-104">Occasionally, you may have requirements for your cells that go beyond what these features can provide.</span></span> <span data-ttu-id="5b8dc-105">Você pode criar seu próprio <xref:System.Windows.Forms.DataGridViewCell> classe para fornecer funcionalidade estendida.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-105">You can create your own custom <xref:System.Windows.Forms.DataGridViewCell> class to provide extended functionality.</span></span>  
  
 <span data-ttu-id="5b8dc-106">Criar um personalizado <xref:System.Windows.Forms.DataGridViewCell> classe derivando de <xref:System.Windows.Forms.DataGridViewCell> classe base ou uma de suas classes derivadas.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-106">You create a custom <xref:System.Windows.Forms.DataGridViewCell> class by deriving from the <xref:System.Windows.Forms.DataGridViewCell> base class or one of its derived classes.</span></span> <span data-ttu-id="5b8dc-107">Embora você possa exibir qualquer tipo de célula em qualquer tipo de coluna, você normalmente também criará um personalizado <xref:System.Windows.Forms.DataGridViewColumn> classe especializada para exibir seu tipo de célula.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-107">Although you can display any type of cell in any type of column, you will typically also create a custom <xref:System.Windows.Forms.DataGridViewColumn> class specialized for displaying your cell type.</span></span> <span data-ttu-id="5b8dc-108">Classes de coluna derivam de <xref:System.Windows.Forms.DataGridViewColumn> ou um de seus tipos derivados.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-108">Column classes derive from <xref:System.Windows.Forms.DataGridViewColumn> or one of its derived types.</span></span>  
  
 <span data-ttu-id="5b8dc-109">No exemplo de código a seguir, você criará uma classe de célula personalizada chamada `DataGridViewRolloverCell` que detecta quando o mouse entra e sai dos limites da célula.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-109">In the following code example, you will create a custom cell class called `DataGridViewRolloverCell` that detects when the mouse enters and leaves the cell boundaries.</span></span> <span data-ttu-id="5b8dc-110">Enquanto o mouse está dentro dos limites da célula, um retângulo de inserção é desenhado.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-110">While the mouse is within the cell's bounds, an inset rectangle is drawn.</span></span> <span data-ttu-id="5b8dc-111">Esse novo tipo deriva de <xref:System.Windows.Forms.DataGridViewTextBoxCell> e se comporta em todos os outros aspectos como sua classe base.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-111">This new type derives from <xref:System.Windows.Forms.DataGridViewTextBoxCell> and behaves in all other respects as its base class.</span></span> <span data-ttu-id="5b8dc-112">A classe de coluna correspondente chama-se `DataGridViewRolloverColumn`.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-112">The companion column class is called `DataGridViewRolloverColumn`.</span></span>

> [!NOTE]
>  <span data-ttu-id="5b8dc-119">Quando você deriva <xref:System.Windows.Forms.DataGridViewCell> ou <xref:System.Windows.Forms.DataGridViewColumn> e adicionar novas propriedades à classe derivada, certifique-se de substituir o `Clone` método para copiar as novas propriedades durante operações de clonagem.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-119">When you derive from <xref:System.Windows.Forms.DataGridViewCell> or <xref:System.Windows.Forms.DataGridViewColumn> and add new properties to the derived class, be sure to override the `Clone` method to copy the new properties during cloning operations.</span></span> <span data-ttu-id="5b8dc-120">Você também deve chamar o método `Clone` da classe base para que as propriedades da classe base sejam copiadas para a nova célula ou coluna.</span><span class="sxs-lookup"><span data-stu-id="5b8dc-120">You should also call the base class's `Clone` method so that the properties of the base class are copied to the new cell or column.</span></span>

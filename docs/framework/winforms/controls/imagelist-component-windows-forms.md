# <a name="imagelist-component-windows-forms"></a>Componente ImageList (Windows Forms)
O componente Windows Forms `ImageList` é usado para armazenar imagens, que podem ser exibidas por controles. Uma lista de imagens permite escrever código para um catálogo de imagens único e consistente.

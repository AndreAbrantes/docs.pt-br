# <a name="errorprovider-component-windows-forms"></a>Componente ErrorProvider (Windows Forms)
Os formulários do Windows `ErrorProvider` componente é usado para mostrar ao usuário de forma não intrusiva que algo está errado. Ele é normalmente usado junto com a validação de entrada do usuário em um formulário ou para exibir erros dentro de um conjunto de dados.

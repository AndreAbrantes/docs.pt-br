# <a name="errorprovider-component-windows-forms"></a>Componente ErrorProvider (Windows Forms)
O componente Windows Forms `ErrorProvider` é usado para mostrar o usuário de forma não intrusiva de que algo está errado. Ele é normalmente usado junto com a validação de entrada do usuário em um formulário ou para exibir erros dentro de um conjunto de dados.

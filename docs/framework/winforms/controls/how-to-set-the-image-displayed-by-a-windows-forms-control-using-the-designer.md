# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Como: Definir a imagem exibida por um controle de Windows Forms usando o designer

Vários controles de Windows Forms podem exibir imagens. A imagem pode ser um ícone que esclarece a finalidade do controle, como um ícone de disco em um botão que indica o comando **salvar** . Ou, o ícone pode ser uma imagem de plano de fundo para dar ao controle a aparência desejada.

## <a name="to-set-the-image-displayed-by-a-control"></a>Para definir a imagem exibida por um controle

1. Na janela **Propriedades** , selecione a propriedade **Image** ou **BackgroundImage** do controle e, em seguida, selecione as reticências![(o botão de reticências](./media/visual-studio-ellipsis-button.png)no Visual Studio) para exibir o **recurso de seleção** caixa de diálogo.

2. Selecione a imagem que você deseja exibir.

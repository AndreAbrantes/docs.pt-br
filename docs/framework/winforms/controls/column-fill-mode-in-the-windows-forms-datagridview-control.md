<span data-ttu-id="5e24c-103">No modo de preenchimento de coluna, o <xref:System.Windows.Forms.DataGridView> controle redimensiona as colunas automaticamente para que eles preenchem a largura da área de exibição disponíveis.</span><span class="sxs-lookup"><span data-stu-id="5e24c-103">In column fill mode, the <xref:System.Windows.Forms.DataGridView> control resizes its columns automatically so that they fill the width of the available display area.</span></span> <span data-ttu-id="5e24c-104">O controle não exibir a barra de rolagem horizontal, exceto quando é necessário para manter a largura de cada coluna igual a ou maior que seu <xref:System.Windows.Forms.DataGridViewColumn.MinimumWidth%2A> o valor da propriedade.</span><span class="sxs-lookup"><span data-stu-id="5e24c-104">The control does not display the horizontal scroll bar except when it is necessary to keep the width of every column equal to or greater than its <xref:System.Windows.Forms.DataGridViewColumn.MinimumWidth%2A> property value.</span></span>  
  
 <span data-ttu-id="5e24c-105">O comportamento de dimensionamento de cada coluna depende de seu <xref:System.Windows.Forms.DataGridViewColumn.InheritedAutoSizeMode%2A> propriedade.</span><span class="sxs-lookup"><span data-stu-id="5e24c-105">The sizing behavior of each column depends on its <xref:System.Windows.Forms.DataGridViewColumn.InheritedAutoSizeMode%2A> property.</span></span> <span data-ttu-id="5e24c-106">O valor dessa propriedade é herdado da coluna de <xref:System.Windows.Forms.DataGridViewColumn.AutoSizeMode%2A> propriedade ou o controle <xref:System.Windows.Forms.DataGridView.AutoSizeColumnsMode%2A> propriedade se o valor da coluna é <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.NotSet> (o valor padrão).</span><span class="sxs-lookup"><span data-stu-id="5e24c-106">The value of this property is inherited from the column's <xref:System.Windows.Forms.DataGridViewColumn.AutoSizeMode%2A> property or the control's <xref:System.Windows.Forms.DataGridView.AutoSizeColumnsMode%2A> property if the column value is <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.NotSet> (the default value).</span></span>  
  
 <span data-ttu-id="5e24c-107">Cada coluna pode ter um modo de tamanho diferente, mas todas as colunas com um modo de tamanho de <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.Fill> compartilharão a largura da área de exibição que não é usada por outras colunas.</span><span class="sxs-lookup"><span data-stu-id="5e24c-107">Each column can have a different size mode, but any columns with a size mode of <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.Fill> will share the display-area width that is not used by the other columns.</span></span> <span data-ttu-id="5e24c-108">Essa largura é dividida entre as colunas de modo de preenchimento em proporções relativo a seu <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores de propriedade.</span><span class="sxs-lookup"><span data-stu-id="5e24c-108">This width is divided among the fill-mode columns in proportions relative to their <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> property values.</span></span> <span data-ttu-id="5e24c-109">Por exemplo, se tem duas colunas <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores de 100 e 200, a primeira coluna será metade da largura da segunda coluna.</span><span class="sxs-lookup"><span data-stu-id="5e24c-109">For example, if two columns have <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100 and 200, the first column will be half as wide as the second column.</span></span>  
  
## <a name="user-resizing-in-fill-mode"></a><span data-ttu-id="5e24c-110">Redimensionamento de usuário no modo de preenchimento</span><span class="sxs-lookup"><span data-stu-id="5e24c-110">User Resizing in Fill Mode</span></span>  
 <span data-ttu-id="5e24c-111">Ao contrário de dimensionamento modos redimensionar com base no conteúdo da célula, modo de preenchimento não impedir que usuários redimensionamento de colunas que tenham <xref:System.Windows.Forms.DataGridViewColumn.Resizable%2A> valores de propriedade de `true`.</span><span class="sxs-lookup"><span data-stu-id="5e24c-111">Unlike sizing modes that resize based on cell contents, fill mode does not prevent users from resizing columns that have <xref:System.Windows.Forms.DataGridViewColumn.Resizable%2A> property values of `true`.</span></span> <span data-ttu-id="5e24c-112">Quando um usuário redimensionar uma coluna do modo de preenchimento, as colunas de modo de preenchimento depois da coluna redimensionada (à direita se <xref:System.Windows.Forms.Control.RightToLeft%2A> é `false`;  <span data-ttu-id="5e24c-116">Depois de redimensionar uma coluna de modo de preenchimento, o <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores para todas as colunas que foram alteradas são ajustadas proporcionalmente.</span><span class="sxs-lookup"><span data-stu-id="5e24c-116">After resizing a fill-mode column, the <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values for all columns that changed are adjusted proportionally.</span></span> <span data-ttu-id="5e24c-117">Por exemplo, se tem quatro colunas de modo de preenchimento <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores de 100, redimensionando a segunda coluna meia largura original resultará em <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores de 50, 100, 125 e 125.</span><span class="sxs-lookup"><span data-stu-id="5e24c-117">For example, if four fill-mode columns have <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100, resizing the second column to half its original width will result in <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100, 50, 125, and 125.</span></span> <span data-ttu-id="5e24c-118">Redimensionamento de uma coluna que não está no modo de preenchimento não alterará qualquer <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> valores porque as colunas de modo de preenchimento simplesmente serão redimensionado para compensar enquanto retém as mesmas proporções.</span><span class="sxs-lookup"><span data-stu-id="5e24c-118">Resizing a column that is not in fill mode will not change any <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values because the fill-mode columns will simply resize to compensate while retaining the same proportions.</span></span>

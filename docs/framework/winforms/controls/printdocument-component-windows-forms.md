# <a name="printdocument-component-windows-forms"></a>Componente PrintDocument (Windows Forms)
O componente `PrintDocument` dos Windows Forms é usado para definir as propriedades que descrevem o que imprimir e, em seguida, imprime o documento em aplicativos baseados no Windows. Ele pode ser usado em conjunto com o componente <xref:System.Windows.Forms.PrintDialog> para ser o comando de todos os aspectos da impressão de documentos.

 Apresenta os conceitos gerais do componente `PrintDocument`, que permite definir propriedades que descrevem o que imprimir e inicia a impressão em um aplicativo baseado em Windows.

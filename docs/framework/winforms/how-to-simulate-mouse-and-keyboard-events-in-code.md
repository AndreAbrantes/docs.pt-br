 <span data-ttu-id="3b724-113">Existem outras maneiras de simular a entrada do mouse.</span><span class="sxs-lookup"><span data-stu-id="3b724-113">There are other ways to simulate mouse input.</span></span> <span data-ttu-id="3b724-114">Por exemplo, você pode definir uma propriedade de controle que representa um estado que normalmente é definido por meio de entrada do mouse por meio de programação (como o <xref:System.Windows.Forms.CheckBox.Checked%2A> propriedade do <xref:System.Windows.Forms.CheckBox> controle), ou você pode chamar diretamente o representante que é anexado ao evento você você deseja simular.</span><span class="sxs-lookup"><span data-stu-id="3b724-114">For example, you can programmatically set a control property that represents a state that is typically set through mouse input (such as the <xref:System.Windows.Forms.CheckBox.Checked%2A> property of the <xref:System.Windows.Forms.CheckBox> control), or you can directly call the delegate that is attached to the event you want to simulate.</span></span>

# <a name="dialog-boxes-in-windows-forms"></a>Caixas de diálogo no Windows Forms
Caixas de diálogo são usadas para interagir com o usuário e recuperar informações. Em termos simples, uma caixa de diálogo é um formulário com sua propriedade de enumeração <xref:System.Windows.Forms.FormBorderStyle> definida como `FixedDialog`. Você pode construir suas próprias caixas de diálogo personalizadas usando o Designer de Formulários do Windows no Visual Studio. Adicione controles como `Label`, `Textbox` e `Button` para personalizar caixas de diálogo conforme suas necessidades específicas. O .NET Framework também inclui caixas de diálogo predefinidas, como caixas de mensagem e **abertura de arquivo** , que você pode adaptar para seus próprios aplicativos. Para obter mais informações, confira [Controles e componentes da caixa de diálogo](./controls/dialog-box-controls-and-components-windows-forms.md).

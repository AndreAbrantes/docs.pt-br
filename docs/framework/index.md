# <a name="net-framework-guide"></a>Guia do .NET Framework

> [!NOTE]
> Este conjunto de conteúdo do .NET Framework inclui informações para versões do .NET Framework 4.5 e 4.8. Para baixar o .NET Framework, consulte [Install .NET Framework](./install/guide-for-developers.md). Para obter uma lista de novos recursos e alterações no .NET Framework, consulte [O que há de novo no .NET Framework](./whats-new/index.md). Para obter uma lista de plataformas com suporte, confira [Requisitos do sistema .NET Framework](./get-started/system-requirements.md). Para obter a documentação sobre versões mais antigas do .NET Framework, consulte [a documentação das versões anteriores .NET](https://docs.microsoft.com/previous-versions/dotnet/).

.NET Framework é uma plataforma de desenvolvimento para a construção de aplicativos para Web, Windows, Windows Server e Microsoft Azure. Ele consiste no Common Language Runtime (CLR) e na biblioteca de classes do .NET Framework, que inclui uma ampla gama de recursos e suporte para muitos padrões do setor.

O .NET Framework fornece muitos serviços, incluindo gerenciamento de memória, segurança de tipo e memória, segurança, rede e implantação de aplicativos. Ele fornece APIs e estruturas de dados fáceis de usam que abstraem o sistema operacional do Windows. Você pode usar diferentes linguagens de programação com .NET Framework, incluindo C#, F#e Visual Basic.

Para obter uma introdução geral ao .NET Framework para usuários e desenvolvedores, consulte [Getting Started](./get-started/index.md). Para obter uma introdução à arquitetura e aos principais recursos do .NET Framework, consulte a [visão geral](./get-started/overview.md).

.NET Framework pode ser usado com Docker e com [contêineres do Windows](/virtualization/windowscontainers/about/).

## <a name="installation"></a>Instalação

.NET Framework vem com o Windows, que permite executar aplicativos .NET Framework. Você pode precisar de uma versão posterior do .NET Framework do que a que vem com a sua versão do Windows. Para obter mais informações, consulte [Instalar .NET Framework no Windows](./install/index.md).

Para saber como reparar sua instalação do .NET Framework se estiver com erros durante a instalação, consulte [Reparar .NET Framework](./install/repair.md).

Para obter informações mais detalhadas sobre como baixar o .NET Framework, consulte [Instalar o .NET Framework para desenvolvedores](./install/guide-for-developers.md).

# <a name="wcf-and-net-framework-client-profile"></a>Perfil de cliente WCF e .NET Framework
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] é uma versão leve do completo .net Framework criado para clientes que não é necessário a estrutura inteira. Nem todos os Windows Communication Foundation é suportado pela estrutura do cliente.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>WCF recursos com suporte do .net Framework Client Profile  
 Os seguintes recursos do Windows Communication Foundation são suportados por [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]:  
  
-   Todos WCF é têm suporte, exceto o Cardspace e hospedagem na web.  
  
-   Há suporte para os canais de comunicação remota TCP/IP.  
  
-   Não há suporte para ASMX (Web Services).

# <a name="data-contract-serializer"></a>Serializador de contrato de dados
A infraestrutura do Windows Communication Foundation (WCF) usa o <xref:System.Runtime.Serialization.DataContractSerializer> classe como o serializador padrão. Os tópicos nesta seção fornecem mais detalhes para usá-lo como um serializador autônomo.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Serialização e desserialização](../../../../docs/framework/wcf/feature-details/serialization-and-deserialization.md)  
 Fornece detalhes sobre como usar o <xref:System.Runtime.Serialization.DataContractSerializer> documentos de classe para serializar grafos de objeto e desserializar XML em objetos.  
  
 [Exportação e importação de esquema](../../../../docs/framework/wcf/feature-details/schema-import-and-export.md)  
 Explica como gerar um esquema de (XSD) de linguagem de definição de esquema XML de um [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] classe ou como gerar o código de um documento de esquema XSD.

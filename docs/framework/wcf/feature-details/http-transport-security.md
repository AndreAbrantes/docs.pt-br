 <span data-ttu-id="727ea-118">Para iniciar uma sessão segura, o cliente envia o equivalente de uma "saudação" para o servidor junto com uma lista de algoritmos criptográficos que ele pode usar para assinar, gerar hashes e criptografar e descriptografar com o.</span><span class="sxs-lookup"><span data-stu-id="727ea-118">To initiate a secure session, the client sends the equivalent of a "hello" to the server along with a list of cryptographic algorithms it can use to sign, generate hashes, and encrypt and decrypt with.</span></span> <span data-ttu-id="727ea-119">Em resposta, o site envia de volta uma confirmação e sua escolha de um dos conjuntos de algoritmos.</span><span class="sxs-lookup"><span data-stu-id="727ea-119">In response, the site sends back an acknowledgment and its choice of one of the algorithms suites.</span></span> <span data-ttu-id="727ea-120">Durante esse handshake inicial, ambas as partes enviam e recebem nonces.</span><span class="sxs-lookup"><span data-stu-id="727ea-120">During this initial handshake, both parties send and receive nonces.</span></span> <span data-ttu-id="727ea-121">Um *nonce* é um trecho de dados gerado aleatoriamente que é usado, em combinação com a chave pública do site, para criar um hash.</span><span class="sxs-lookup"><span data-stu-id="727ea-121">A *nonce* is a randomly generated piece of data that is used, in combination with the site's public key, to create a hash.</span></span> <span data-ttu-id="727ea-122">Um *hash* é um novo número que é derivado dos dois números usando um algoritmo padrão, como SHA1.</span><span class="sxs-lookup"><span data-stu-id="727ea-122">A *hash* is a new number that is derived from the two numbers using a standard algorithm, such as SHA1.</span></span> <span data-ttu-id="727ea-123">(O cliente e o site também trocam mensagens para concordar com o algoritmo de hash a ser usado.)

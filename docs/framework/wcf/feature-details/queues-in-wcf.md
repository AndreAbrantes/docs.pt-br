- <span data-ttu-id="352ed-105">Aplicativos menos rígidos.</span><span class="sxs-lookup"><span data-stu-id="352ed-105">Loosely coupled applications.</span></span> <span data-ttu-id="352ed-106">O envio de aplicativos pode enviar mensagens para filas sem a necessidade de saber se o aplicativo receptor está disponível para processar a mensagem.</span><span class="sxs-lookup"><span data-stu-id="352ed-106">Sending applications can send messages to queues without needing to know whether the receiving application is available to process the message.</span></span> <span data-ttu-id="352ed-107">A fila fornece independência de processamento que permite que um aplicativo de envio envie mensagens para a fila em uma taxa que não depende de quão rápido os aplicativos de recebimento podem processar as mensagens.</span><span class="sxs-lookup"><span data-stu-id="352ed-107">The queue provides processing independence that allows a sending application to send messages to the queue at a rate that does not depend on how fast the receiving applications can process the messages.</span></span> <span data-ttu-id="352ed-108">A disponibilidade geral do sistema aumenta quando o envio de mensagens para uma fila não está rigidamente acoplado ao processamento de mensagens.</span><span class="sxs-lookup"><span data-stu-id="352ed-108">Overall system availability increases when sending messages to a queue is not tightly coupled to message processing.</span></span>

- <span data-ttu-id="352ed-114">Nivelamento de carga.</span><span class="sxs-lookup"><span data-stu-id="352ed-114">Load leveling.</span></span> <span data-ttu-id="352ed-115">O envio de aplicativos pode saturar o recebimento de aplicativos com mensagens.</span><span class="sxs-lookup"><span data-stu-id="352ed-115">Sending applications can overwhelm receiving applications with messages.</span></span> <span data-ttu-id="352ed-116">As filas podem gerenciar taxas de produção e consumo de mensagens incompatíveis para que um receptor não fique sobrecarregado.</span><span class="sxs-lookup"><span data-stu-id="352ed-116">Queues can manage mismatched message production and consumption rates so that a receiver is not overwhelmed.</span></span>  
  
- <span data-ttu-id="352ed-117">Operações desconectadas.</span><span class="sxs-lookup"><span data-stu-id="352ed-117">Disconnected operations.</span></span> <span data-ttu-id="352ed-118">O envio, recebimento e processamento de operações podem ser desconectados ao se comunicar por redes de alta latência ou redes de disponibilidade limitada, como no caso de dispositivos móveis.</span><span class="sxs-lookup"><span data-stu-id="352ed-118">Sending, receiving, and processing operations can become disconnected when communicating over high-latency networks or limited-availability networks, such as in the case of mobile devices.</span></span> <span data-ttu-id="352ed-119">As filas permitem que essas operações continuem, mesmo quando os pontos de extremidade são desconectados.</span><span class="sxs-lookup"><span data-stu-id="352ed-119">Queues allow these operations to continue, even when the endpoints are disconnected.</span></span> <span data-ttu-id="352ed-120">Quando a conexão é restabelecida, a fila encaminha mensagens para o aplicativo de recebimento.</span><span class="sxs-lookup"><span data-stu-id="352ed-120">When the connection is reestablished, the queue forwards messages to the receiving application.</span></span>  
  
 <span data-ttu-id="352ed-121">Para usar o recurso filas em um aplicativo WCF, você pode usar uma das associações padrão ou pode criar uma associação personalizada se uma das associações padrão não atender às suas necessidades.</span><span class="sxs-lookup"><span data-stu-id="352ed-121">To use the queues feature in a WCF application, you can use one of the standard bindings, or you can create a custom binding if one of the standard bindings does not satisfy your requirements.</span></span> <span data-ttu-id="352ed-122">Para obter mais informações sobre associações padrão relevantes e como escolher uma, consulte [como: trocar mensagens com pontos de extremidade WCF e aplicativos de enfileiramento de mensagens](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span><span class="sxs-lookup"><span data-stu-id="352ed-122">For more information about relevant standard bindings and how to choose one, see [How to: Exchange Messages with WCF Endpoints and Message Queuing Applications](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span></span> <span data-ttu-id="352ed-123">Para obter mais informações sobre como criar associações personalizadas, confira [Associações personalizadas](../../../../docs/framework/wcf/extending/custom-bindings.md).</span><span class="sxs-lookup"><span data-stu-id="352ed-123">For more information about creating custom bindings, see [Custom Bindings](../../../../docs/framework/wcf/extending/custom-bindings.md).</span></span>

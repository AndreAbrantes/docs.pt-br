- <span data-ttu-id="c5417-114">Nivelamento de carga.</span><span class="sxs-lookup"><span data-stu-id="c5417-114">Load leveling.</span></span> <span data-ttu-id="c5417-115">Aplicativos de envio podem sobrecarregar a aplicativos de recebimento com mensagens.</span><span class="sxs-lookup"><span data-stu-id="c5417-115">Sending applications can overwhelm receiving applications with messages.</span></span> <span data-ttu-id="c5417-116">As filas podem gerenciar taxas de produção e consumo de mensagens incompatíveis para que um destinatário não seja sobrecarregado.</span><span class="sxs-lookup"><span data-stu-id="c5417-116">Queues can manage mismatched message production and consumption rates so that a receiver is not overwhelmed.</span></span>  
  
- <span data-ttu-id="c5417-117">Operações desconectadas.</span><span class="sxs-lookup"><span data-stu-id="c5417-117">Disconnected operations.</span></span> <span data-ttu-id="c5417-118">Envio, recebimento e operações de processamento podem ser desconectados ao se comunicar por redes de alta latência ou disponibilidade limitada, como no caso de dispositivos móveis.</span><span class="sxs-lookup"><span data-stu-id="c5417-118">Sending, receiving, and processing operations can become disconnected when communicating over high-latency networks or limited-availability networks, such as in the case of mobile devices.</span></span> <span data-ttu-id="c5417-119">As filas permitem que essas operações continuar, mesmo quando os pontos de extremidade são desconectados.</span><span class="sxs-lookup"><span data-stu-id="c5417-119">Queues allow these operations to continue, even when the endpoints are disconnected.</span></span> <span data-ttu-id="c5417-120">Quando a conexão for restabelecida, a fila encaminha mensagens para o aplicativo de recebimento.</span><span class="sxs-lookup"><span data-stu-id="c5417-120">When the connection is reestablished, the queue forwards messages to the receiving application.</span></span>

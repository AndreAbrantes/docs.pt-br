## <a name="to-call-the-service-passing-username-and-password"></a><span data-ttu-id="cdbad-126">Para chamar o nome de usuário e a senha do serviço</span><span class="sxs-lookup"><span data-stu-id="cdbad-126">To call the service passing username and password</span></span>

1. <span data-ttu-id="cdbad-127">O aplicativo cliente deve solicitar seu nome de usuário e senha.</span><span class="sxs-lookup"><span data-stu-id="cdbad-127">The client application must prompt the user for their username and password.</span></span> <span data-ttu-id="cdbad-128">O código a seguir solicita o nome de usuário e a senha:</span><span class="sxs-lookup"><span data-stu-id="cdbad-128">The following code asks the user for username and password:</span></span>

    > [!WARNING]
    > <span data-ttu-id="cdbad-129">Esse código não deve ser usado em produção, pois a senha é exibida durante a inserção.</span><span class="sxs-lookup"><span data-stu-id="cdbad-129">This code should not be used in production as the password is displayed while being entered.</span></span>

    ```csharp
    public static void GetPassword(out string username, out string password)
    {
        Console.WriteLine("Provide a valid machine or domain account. [domain\\user]");
        Console.WriteLine("   Enter username:");
        username = Console.ReadLine();
        Console.WriteLine("   Enter password:");
        password = Console.ReadLine();
    }
    ```

2. <span data-ttu-id="cdbad-130">Crie uma instância do proxy do cliente especificando as credenciais do cliente, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="cdbad-130">Create an instance of the client proxy specifying the client's credentials as shown in the following code:</span></span>

    ```csharp
    string password;

    var proxy = new Service1Client();

    // Prompt the user for username & password.
    GetPassword(out username, out password);

    // Set the user's credentials on the proxy.
    proxy.ClientCredentials.UserName.UserName = username;
    proxy.ClientCredentials.UserName.Password = password;

    // Treat the test certificate as trusted.
    proxy.ClientCredentials.ServiceCertificate.Authentication.CertificateValidationMode = System.ServiceModel.Security.X509CertificateValidationMode.PeerOrChainTrust;
    // Call the service operation using the proxy
    ```

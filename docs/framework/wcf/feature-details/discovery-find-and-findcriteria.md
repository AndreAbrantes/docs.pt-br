## <a name="findresponse"></a><span data-ttu-id="84d63-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="84d63-140">FindResponse</span></span>

<span data-ttu-id="84d63-141"><xref:System.ServiceModel.Discovery.FindResponse> tem uma propriedade de coleção de <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> que contém as respostas enviadas por serviços correspondentes na rede.</span><span class="sxs-lookup"><span data-stu-id="84d63-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="84d63-142">Se nenhum serviço tiver respondido, a coleção estará vazia.</span><span class="sxs-lookup"><span data-stu-id="84d63-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="84d63-143">Se um ou mais serviços tiverem respondido, cada resposta será armazenada em um objeto <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata>, que contém o endereço, o contrato e algumas informações adicionais sobre o serviço.</span><span class="sxs-lookup"><span data-stu-id="84d63-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>

<span data-ttu-id="84d63-144">O exemplo a seguir mostra como executar uma operação Localizar no código.</span><span class="sxs-lookup"><span data-stu-id="84d63-144">The following example shows how to perform a find operation in code.</span></span>

```csharp
// Create DiscoveryClient
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());

// Create FindCriteria
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));
findCriteria.Duration = TimeSpan.FromSeconds(10);

// Find ICalculatorService endpoints
FindResponse findResponse = discoveryClient.Find(findCriteria);

Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)
```

## <a name="findresponse"></a><span data-ttu-id="b57b9-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="b57b9-140">FindResponse</span></span>

<span data-ttu-id="b57b9-141"><xref:System.ServiceModel.Discovery.FindResponse>tem uma <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> propriedade de coleção que contém todas as respostas enviadas por serviços correspondentes na rede.</span><span class="sxs-lookup"><span data-stu-id="b57b9-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="b57b9-142">Se nenhum serviço tiver respondido, a coleção estará vazia.</span><span class="sxs-lookup"><span data-stu-id="b57b9-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="b57b9-143">Se um ou mais serviços tiverem respondido, cada resposta será armazenada em um <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> objeto, que contém o endereço, o contrato e algumas informações adicionais sobre o serviço.</span><span class="sxs-lookup"><span data-stu-id="b57b9-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>

<span data-ttu-id="b57b9-144">O exemplo a seguir mostra como executar uma operação Localizar no código.</span><span class="sxs-lookup"><span data-stu-id="b57b9-144">The following example shows how to perform a find operation in code.</span></span>

```csharp
// Create DiscoveryClient
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());

// Create FindCriteria
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));
findCriteria.Duration = TimeSpan.FromSeconds(10);

// Find ICalculatorService endpoints
FindResponse findResponse = discoveryClient.Find(findCriteria);

Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)
```

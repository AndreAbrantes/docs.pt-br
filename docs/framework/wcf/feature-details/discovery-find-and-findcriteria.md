## <a name="findresponse"></a><span data-ttu-id="d74a0-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="d74a0-140">FindResponse</span></span>  
 <span data-ttu-id="d74a0-141"><xref:System.ServiceModel.Discovery.FindResponse>tem um <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> propriedade da coleção que contém as respostas enviadas pela correspondência de serviços na rede.</span><span class="sxs-lookup"><span data-stu-id="d74a0-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="d74a0-142">Se nenhum serviço respondeu, a coleção está vazia.</span><span class="sxs-lookup"><span data-stu-id="d74a0-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="d74a0-143">Se um ou mais serviços respondeu, cada resposta é armazenada em um <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> objeto, que contém o endereço, contrato e algumas informações adicionais sobre o serviço.</span><span class="sxs-lookup"><span data-stu-id="d74a0-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="d74a0-144">O exemplo a seguir mostra como executar uma operação de localização no código.</span><span class="sxs-lookup"><span data-stu-id="d74a0-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```

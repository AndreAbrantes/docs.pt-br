### <a name="importing-policies"></a><span data-ttu-id="394fc-124">Importando políticas</span><span class="sxs-lookup"><span data-stu-id="394fc-124">Importing Policies</span></span>

 <span data-ttu-id="394fc-126">A lógica de importação de política manipula automaticamente as referências de política para expressões de política no mesmo documento WSDL e é identificada com um `wsu:Id` `xml:id` atributo ou.</span><span class="sxs-lookup"><span data-stu-id="394fc-126">The policy import logic automatically handles policy references to policy expressions in the same WSDL document and is identified with a `wsu:Id` or `xml:id` attribute.</span></span> <span data-ttu-id="394fc-127">A lógica de importação de política protege aplicativos contra referências de política circulares limitando o tamanho de uma expressão de política a 4096 nós, em que um nó é um dos seguintes elementos: `wsp:Policy` ,, `wsp:All` `wsp:ExactlyOne` , `wsp:policyReference` .</span><span class="sxs-lookup"><span data-stu-id="394fc-127">The policy import logic protects applications against circular policy references by limiting the size of a policy expression to 4096 nodes, where a node is a one of the following elements: `wsp:Policy`, `wsp:All`, `wsp:ExactlyOne`, `wsp:policyReference`.</span></span>

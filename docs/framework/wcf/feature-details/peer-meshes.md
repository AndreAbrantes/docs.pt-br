<span data-ttu-id="bbd4f-103">Uma *malha* é uma coleção nomeada (um gráfico interconectado) de nós pares que podem se comunicar entre si e que são identificados por uma ID de malha exclusiva.</span><span class="sxs-lookup"><span data-stu-id="bbd4f-103">A *mesh* is a named collection (an interconnected graph) of peer nodes that can communicate among themselves and that are identified by a unique mesh ID.</span></span> <span data-ttu-id="bbd4f-104">Cada nó é conectado a vários outros nós.</span><span class="sxs-lookup"><span data-stu-id="bbd4f-104">Each node is connected to multiple other nodes.</span></span> <span data-ttu-id="bbd4f-105">Em uma malha bem conectada, há um caminho entre dois nós, com relativamente poucos saltos entre os nós nas bordas mais distantes da malha, e a malha permanecerá conectada mesmo que alguns nós ou conexões sejam cancelados. Os nós ativos na malha publicam suas informações de ponto de extremidade com uma ID de malha correspondente para que outros pontos possam encontrá-las.</span><span class="sxs-lookup"><span data-stu-id="bbd4f-105">In a well-connected mesh, there is a path between any two nodes, with relatively few hops between the nodes on the furthest edges of the mesh, and the mesh will remain connected even if some nodes or connections drop out.

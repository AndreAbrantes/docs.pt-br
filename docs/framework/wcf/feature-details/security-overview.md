 <span data-ttu-id="f07d7-179">Dois mecanismos principais são usados para implementar a segurança de transferência no WCF: modo de segurança de *transporte* e modo de segurança de *mensagem* .</span><span class="sxs-lookup"><span data-stu-id="f07d7-179">Two main mechanisms are used to implement transfer security in WCF: *transport* security mode and *message* security mode.</span></span>

 <span data-ttu-id="f07d7-185">Para obter mais informações sobre essas diferenças, consulte [protegendo serviços e clientes](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span><span class="sxs-lookup"><span data-stu-id="f07d7-185">For more information about these differences, see [Securing Services and Clients](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span></span>  
  
 <span data-ttu-id="f07d7-186">Um terceiro modo de segurança usa ambos os modos anteriores e traz as vantagens de ambos.</span><span class="sxs-lookup"><span data-stu-id="f07d7-186">A third security mode uses both previous modes and brings advantages of both.</span></span> <span data-ttu-id="f07d7-187">Esse modo é chamado `TransportWithMessageCredential`.</span><span class="sxs-lookup"><span data-stu-id="f07d7-187">This mode is called `TransportWithMessageCredential`.</span></span> <span data-ttu-id="f07d7-188">Nesse modo, a segurança da mensagem é usada para autenticar o cliente e a segurança de transporte é usada para autenticar o servidor e fornecer confidencialidade e integridade da mensagem.</span><span class="sxs-lookup"><span data-stu-id="f07d7-188">In this mode, message security is used to authenticate the client and transport security is used to authenticate the server and provide message confidentiality and integrity.</span></span> <span data-ttu-id="f07d7-189">Graças a isso, o `TransportWithMessageCredential` modo de segurança é quase tão rápido quanto o modo de segurança de transporte e fornece extensibilidade de autenticação de cliente da mesma maneira que a segurança de mensagem.</span><span class="sxs-lookup"><span data-stu-id="f07d7-189">Thanks to this, the `TransportWithMessageCredential` security mode is almost as fast as transport security mode and provides client authentication extensibility in the same way as message security.</span></span> <span data-ttu-id="f07d7-190">No entanto, ao contrário do modo de segurança da mensagem, ele não fornece segurança completa de ponta a ponta.</span><span class="sxs-lookup"><span data-stu-id="f07d7-190">However, unlike message security mode, it does not provide complete end-to-end security.</span></span>

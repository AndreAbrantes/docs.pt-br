-   <span data-ttu-id="fd263-180">*Modo de segurança de transporte* usa um protocolo de nível de transporte, como HTTPS, para fins de segurança de transferência.</span><span class="sxs-lookup"><span data-stu-id="fd263-180">*Transport security mode* uses a transport-level protocol, such as HTTPS, to achieve transfer security.</span></span> <span data-ttu-id="fd263-181">O modo de transporte tem a vantagem de ser amplamente adotada, disponível em muitas plataformas e computacionalmente menos complexo.</span><span class="sxs-lookup"><span data-stu-id="fd263-181">Transport mode has the advantage of being widely adopted, available on many platforms, and less computationally complex.</span></span> <span data-ttu-id="fd263-182">No entanto, ele tem a desvantagem de proteger mensagens apenas de ponto a ponto.</span><span class="sxs-lookup"><span data-stu-id="fd263-182">However, it has the disadvantage of securing messages only from point-to-point.</span></span>

 <span data-ttu-id="fd263-185">Para obter mais informações sobre essas diferenças, consulte [protegendo serviços e clientes](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span><span class="sxs-lookup"><span data-stu-id="fd263-185">For more information about these differences, see [Securing Services and Clients](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span></span>

-   <span data-ttu-id="8a603-180">*O modo de segurança de transporte* usa um protocolo de nível de transporte, como HTTPS, para obter a segurança de transferência.</span><span class="sxs-lookup"><span data-stu-id="8a603-180">*Transport security mode* uses a transport-level protocol, such as HTTPS, to achieve transfer security.</span></span> <span data-ttu-id="8a603-181">O modo de transporte tem a vantagem de ser amplamente adotada, disponível em muitas plataformas e computacionalmente menos complexas.</span><span class="sxs-lookup"><span data-stu-id="8a603-181">Transport mode has the advantage of being widely adopted, available on many platforms, and less computationally complex.</span></span> <span data-ttu-id="8a603-182">No entanto, ele tem a desvantagem de proteger as mensagens somente de ponto a ponto.</span><span class="sxs-lookup"><span data-stu-id="8a603-182">However, it has the disadvantage of securing messages only from point-to-point.</span></span>  
  
-   <span data-ttu-id="8a603-183">*Modo de segurança de mensagem*, no outro lado, usa WS-Security (e outras especificações) para implementar a segurança de transferência.</span><span class="sxs-lookup"><span data-stu-id="8a603-183">*Message security mode*, on the other hand, uses WS-Security (and other specifications) to implement transfer security.</span></span> <span data-ttu-id="8a603-184">Como a segurança da mensagem é aplicada diretamente para as mensagens SOAP e está contida dentro de envelopes SOAP, junto com os dados de aplicativo, ele tem a vantagem de ser transporte independente de protocolo, mais extensível e garantir segurança de ponta a ponta (em vez de ponto a ponto); ele tem a desvantagem de sendo várias vezes mais lento do que o modo de segurança de transporte porque ele tem que lidar com a natureza XML das mensagens de SOAP.</span><span class="sxs-lookup"><span data-stu-id="8a603-184">Because the message security is applied directly to the SOAP messages and is contained inside the SOAP envelopes, together with the application data, it has the advantage of being transport protocol-independent, more extensible, and ensuring end-to-end security (versus point-to-point); it has the disadvantage of being several times slower than transport security mode because it has to deal with the XML nature of the SOAP messages.</span></span>  
  
 <span data-ttu-id="8a603-185">Para obter mais informações sobre essas diferenças, consulte [protegendo serviços e clientes](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span><span class="sxs-lookup"><span data-stu-id="8a603-185">For more information about these differences, see [Securing Services and Clients](../../../../docs/framework/wcf/feature-details/securing-services-and-clients.md).</span></span>  
  
 <span data-ttu-id="8a603-186">Um modo de segurança terceiro usa ambos os modos anteriores e traz as vantagens de ambos.</span><span class="sxs-lookup"><span data-stu-id="8a603-186">A third security mode uses both previous modes and brings advantages of both.</span></span> <span data-ttu-id="8a603-187">Esse modo é chamado `TransportWithMessageCredential`.</span><span class="sxs-lookup"><span data-stu-id="8a603-187">This mode is called `TransportWithMessageCredential`.</span></span> <span data-ttu-id="8a603-188">Nesse modo, segurança de mensagem é usada para autenticar o cliente e a segurança de transporte é usada para autenticar o servidor e fornecer a integridade e confidencialidade da mensagem.</span><span class="sxs-lookup"><span data-stu-id="8a603-188">In this mode, message security is used to authenticate the client and transport security is used to authenticate the server and provide message confidentiality and integrity.</span></span> <span data-ttu-id="8a603-189">Graças isso, o `TransportWithMessageCredential` modo de segurança é quase tão rápido quanto o modo de segurança de transporte e fornece extensibilidade de autenticação de cliente da mesma maneira como a segurança de mensagem.</span><span class="sxs-lookup"><span data-stu-id="8a603-189">Thanks to this, the `TransportWithMessageCredential` security mode is almost as fast as transport security mode and provides client authentication extensibility in the same way as message security.</span></span> <span data-ttu-id="8a603-190">No entanto, ao contrário do modo de segurança de mensagem, ele não fornece segurança de ponta a ponta completa.</span><span class="sxs-lookup"><span data-stu-id="8a603-190">However, unlike message security mode, it does not provide complete end-to-end security.</span></span>

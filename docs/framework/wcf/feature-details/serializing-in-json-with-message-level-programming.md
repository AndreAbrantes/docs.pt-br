## <a name="untyped-message-programming"></a><span data-ttu-id="94f77-116">Programação de mensagem não tipada</span><span class="sxs-lookup"><span data-stu-id="94f77-116">Untyped Message Programming</span></span>  
 <span data-ttu-id="94f77-117">Ao trabalhar diretamente com objetos de mensagem sem tipo, você deve definir explicitamente as propriedades na mensagem não tipada serializá-lo como JSON.</span><span class="sxs-lookup"><span data-stu-id="94f77-117">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="94f77-118">O trecho de código a seguir mostra como fazer isso.</span><span class="sxs-lookup"><span data-stu-id="94f77-118">The following code snippet shows how to do this.</span></span>  
  
```  
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```

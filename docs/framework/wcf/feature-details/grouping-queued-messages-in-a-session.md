<span data-ttu-id="a6989-103">O Windows Communication Foundation (WCF) fornece uma sessão que permite agrupar um conjunto de mensagens relacionadas para processamento por um único aplicativo de recebimento.</span><span class="sxs-lookup"><span data-stu-id="a6989-103">Windows Communication Foundation (WCF) provides a session that allows you to group a set of related messages together for data-stu-id="a6989-109">Similarly, when messages in a session fail to be sent to an application from the application queue, the entire session is placed in the poison queue (if ## <a name="procedures"></a><span data-ttu-id="a6989-119">Procedimentos</span><span class="sxs-lookup"><span data-stu-id="a6989-119">Procedures</span></span>  
  
#### <a name="to-set-up-a-service-contract-to-use-sessions"></a><span data-ttu-id="a6989-120">Para configurar um contrato de serviço para usar sessões</span><span class="sxs-lookup"><span data-stu-id="a6989-120">To set up a service contract to use sessions</span></span>  
  
1. <span data-ttu-id="a6989-121">Defina um contrato de serviço que exija uma sessão.</span><span class="sxs-lookup"><span data-stu-id="a6989-121">Define a service contract that requires a session.</span></span> <span data-ttu-id="a6989-122">Faça isso com o <xref:System.ServiceModel.ServiceContractAttribute> atributo especificando:</span><span class="sxs-lookup"><span data-stu-id="a6989-122">Do this with the <xref:System.ServiceModel.ServiceContractAttribute> attribute by specifying:</span></span>  
  
    ```csharp
    SessionMode=SessionMode.Required  
    ```  
  
2. <span data-ttu-id="a6989-123">Marque as operações no contrato como unidirecional, pois esses métodos não retornam nada.</span><span class="sxs-lookup"><span data-stu-id="a6989-123">Mark the operations in the contract as one-way, because these methods do not return anything.</span></span> <span data-ttu-id="a6989-124">Isso é feito com o <xref:System.ServiceModel.OperationContractAttribute> atributo especificando:</span><span class="sxs-lookup"><span data-stu-id="a6989-124">This is done with the <xref:System.ServiceModel.OperationContractAttribute> attribute by specifying:</span></span>  
  
    ```csharp  
    [OperationContract(IsOneWay = true)]  
    ```  
  
3. <span data-ttu-id="a6989-125">Implemente o contrato de serviço e especifique um <xref:System.ServiceModel.ServiceBehaviorAttribute.InstanceContextMode> de <xref:System.ServiceModel.InstanceContextMode.PerSession?displayProperty=nameWithType> .</span><span class="sxs-lookup"><span data-stu-id="a6989-125">Implement the service contract and specify an <xref:System.ServiceModel.ServiceBehaviorAttribute.InstanceContextMode> of <xref:System.ServiceModel.InstanceContextMode.PerSession?displayProperty=nameWithType>.</span></span> <span data-ttu-id="a6989-126">Isso instancia o serviço apenas uma vez para cada sessão.</span><span class="sxs-lookup"><span data-stu-id="a6989-126">This instantiates the service only once for each session.</span></span>  
  
    ```csharp  
    [ServiceBehavior(InstanceContextMode=InstanceContextMode.PerSession)]  
    ```  
  
4. <span data-ttu-id="a6989-127">Cada operação de serviço requer uma transação.</span><span class="sxs-lookup"><span data-stu-id="a6989-127">Each service operation requires a transaction.</span></span> <span data-ttu-id="a6989-128">Especifique isso com o <xref:System.ServiceModel.OperationBehaviorAttribute> atributo.</span><span class="sxs-lookup"><span data-stu-id="a6989-128">Specify this with the <xref:System.ServiceModel.OperationBehaviorAttribute> attribute.</span></span> <span data-ttu-id="a6989-129">A operação que conclui a transação também deve ser definida <xref:System.ServiceModel.OperationBehaviorAttribute.TransactionAutoComplete> como `true` .</span><span class="sxs-lookup"><span data-stu-id="a6989-129">The operation that completes the transaction should also set <xref:System.ServiceModel.OperationBehaviorAttribute.TransactionAutoComplete> to `true`.</span></span>  
  
    ```csharp  
    [OperationBehavior(TransactionScopeRequired = true, TransactionAutoComplete = true)]
    ```  
  
5. <span data-ttu-id="a6989-130">Configure um ponto de extremidade que usa a associação fornecida pelo sistema `NetMsmqBinding` .</span><span class="sxs-lookup"><span data-stu-id="a6989-130">Configure an endpoint that uses the system-provided `NetMsmqBinding` binding.</span></span>  
  
6. <span data-ttu-id="a6989-131">Crie uma fila transacional usando <xref:System.Messaging> .</span><span class="sxs-lookup"><span data-stu-id="a6989-131">Create a transactional queue using <xref:System.Messaging>.</span></span> <span data-ttu-id="a6989-132">Você também pode criar a fila usando o MSMQ (enfileiramento de mensagens) ou o MMC.</span><span class="sxs-lookup"><span data-stu-id="a6989-132">You can also create the queue by using Message Queuing (MSMQ) or MMC.</span></span> <span data-ttu-id="a6989-133">Se você fizer isso, crie uma fila transacional.</span><span class="sxs-lookup"><span data-stu-id="a6989-133">If you do, create a transactional queue.</span></span>  
  
7. <span data-ttu-id="a6989-134">Crie um host de serviço para o serviço usando o <xref:System.ServiceModel.ServiceHost> .</span><span class="sxs-lookup"><span data-stu-id="a6989-134">Create a service host for the service by using <xref:System.ServiceModel.ServiceHost>.</span></span>  
  
8. <span data-ttu-id="a6989-135">Abra o host de serviço para disponibilizar o serviço.</span><span class="sxs-lookup"><span data-stu-id="a6989-135">Open the service host to make the service available.</span></span>  
  
9. <span data-ttu-id="a6989-136">Feche o host de serviço.</span><span class="sxs-lookup"><span data-stu-id="a6989-136">Close the service host.</span></span>

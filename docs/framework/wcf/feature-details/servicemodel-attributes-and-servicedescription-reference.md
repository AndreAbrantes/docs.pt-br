<span data-ttu-id="13121-103">O *árvore descrição* é a hierarquia de tipos (começando com o <xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType> classe) que juntas descrevem todos os aspectos de um serviço.</span><span class="sxs-lookup"><span data-stu-id="13121-103">The *description tree* is the hierarchy of types (starting with the <xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType> class) that together describe every aspect of a service.</span></span> [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)]<span data-ttu-id="13121-104">usa uma descrição de árvore para criar um tempo de execução de serviço válido, para publicar Web Services Description Language (WSDL), linguagem de definição de esquema XML (XSD) e declarações de política (metadados) sobre o serviço que os clientes podem usar para conectar e usar o serviço e Gere diversas representações de arquivo de código e configuração dos valores de árvore de descrição.</span><span class="sxs-lookup"><span data-stu-id="13121-104"> uses a description tree to build a valid service runtime, to publish Web Services Description Language (WSDL), XML Schema definition language (XSD), and policy assertions (metadata) about the service that clients can use to connect to and use the service, and to generate various code and configuration file representations of the description tree values.</span></span>  
  
 <span data-ttu-id="13121-105">Este tópico descreve as propriedades como relacionadas ao contrato são obtidos do contrato de serviço e como eles são implementados e adicionados à árvore de descrição.</span><span class="sxs-lookup"><span data-stu-id="13121-105">This topic describes how contract-related properties are obtained from the service contract, and how they are implemented and added to the description tree.</span></span> <span data-ttu-id="13121-106">Em alguns casos, os valores de atributo são convertidos em Propriedades de comportamento e comportamento é inserido na árvore de descrição.</span><span class="sxs-lookup"><span data-stu-id="13121-106">In some cases, attribute values are converted into behavior properties and behavior is then inserted into the description tree.</span></span> [!INCLUDE[crabout](../../../../includes/crabout-md.md)]<span data-ttu-id="13121-107">como os valores de árvore de descrição são convertidos em metadados, consulte [ServiceDescription e referência de WSDL](../../../../docs/framework/wcf/feature-details/servicedescription-and-wsdl-reference.md).</span><span class="sxs-lookup"><span data-stu-id="13121-107"> how the description tree values are converted into metadata, see [ServiceDescription and WSDL Reference](../../../../docs/framework/wcf/feature-details/servicedescription-and-wsdl-reference.md).</span></span>

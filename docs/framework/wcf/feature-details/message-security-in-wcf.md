## <a name="what-is-message-security"></a><span data-ttu-id="b1492-105">O que é a segurança de mensagem?</span><span class="sxs-lookup"><span data-stu-id="b1492-105">What Is Message Security?</span></span>

 <span data-ttu-id="b1492-108">Em suma, segurança de mensagem é diferente de segurança de transporte, encapsulando as credenciais de segurança e declarações com todas as mensagens, juntamente com qualquer proteção de mensagem (assinatura ou criptografia).</span><span class="sxs-lookup"><span data-stu-id="b1492-108">In brief, message security differs from transport security by encapsulating the security credentials and claims with every message along with any message protection (signing or encryption).</span></span> <span data-ttu-id="b1492-109">Aplicando a segurança diretamente para a mensagem modificando seu conteúdo permite que a mensagem segura seja automaticamente recipiente em relação aos aspectos de segurança.</span><span class="sxs-lookup"><span data-stu-id="b1492-109">Applying the security directly to the message by modifying its content allows the secured message to be self-containing with respect to the security aspects.</span></span> <span data-ttu-id="b1492-110">Isso permite alguns cenários que não são possíveis quando a segurança de transporte é usada.</span><span class="sxs-lookup"><span data-stu-id="b1492-110">This enables some scenarios that are not possible when transport security is used.</span></span>

     <span data-ttu-id="45457-219">Rastreamento de WS-AtomicTransaction pode ser habilitado usando o [o utilitário de configuração do WS-AtomicTransaction (wsatConfig.exe)](../../../../docs/framework/wcf/ws-atomictransaction-configuration-utility-wsatconfig-exe.md).</span><span class="sxs-lookup"><span data-stu-id="45457-219">WS-AtomicTransaction tracing can be enabled by using the [WS-AtomicTransaction Configuration Utility (wsatConfig.exe)](../../../../docs/framework/wcf/ws-atomictransaction-configuration-utility-wsatconfig-exe.md).</span></span> <span data-ttu-id="45457-220">Tal rastreamento fornece informações sobre o estado de transações e participantes dentro de um sistema.</span><span class="sxs-lookup"><span data-stu-id="45457-220">Such tracing provides insight into the state of transactions and participants within a system.</span></span> <span data-ttu-id="45457-221">Para habilitar o rastreamento de modelo de serviço interno também, você pode definir o `HKLM\SOFTWARE\Microsoft\WSAT\3.0\ServiceModelDiagnosticTracing` chave do registro para um valor válido do <xref:System.Diagnostics.SourceLevels> enumeração.</span><span class="sxs-lookup"><span data-stu-id="45457-221">To also enable internal Service Model tracing, you can set the `HKLM\SOFTWARE\Microsoft\WSAT\3.0\ServiceModelDiagnosticTracing` registry key to a valid value of the <xref:System.Diagnostics.SourceLevels> enumeration.</span></span> <span data-ttu-id="45457-222">Você pode habilitar a mensagem de log da mesma maneira como outros aplicativos do WCF.</span><span class="sxs-lookup"><span data-stu-id="45457-222">You can enable message logging in the same way as other WCF applications.</span></span>  
  
-   <span data-ttu-id="45457-223">`System.Transactions` Rastreamento</span><span class="sxs-lookup"><span data-stu-id="45457-223">`System.Transactions` tracing</span></span>  
  
     <span data-ttu-id="45457-224">Ao usar o protocolo OleTransactions, mensagens de protocolo não podem ser rastreadas.</span><span class="sxs-lookup"><span data-stu-id="45457-224">When using the OleTransactions protocol, protocol messages cannot be traced.</span></span> <span data-ttu-id="45457-225">O suporte de rastreamento de <xref:System.Transactions> fornece infraestrutura (que usa OleTransactions) permite aos usuários visualizar eventos que ocorreram nas transações.</span><span class="sxs-lookup"><span data-stu-id="45457-225">The tracing support the <xref:System.Transactions> infrastructure provides (which uses OleTransactions) allows users to view events that occurred to the transactions.</span></span> <span data-ttu-id="45457-226">Para habilitar o rastreamento para uma <xref:System.Transactions> aplicativo, incluem o seguinte código no `App.config` arquivo de configuração.</span><span class="sxs-lookup"><span data-stu-id="45457-226">To enable tracing for a <xref:System.Transactions> application, include the following code in the `App.config` configuration file.</span></span>  
  
    ```xml  
    <configuration>  
      <system.diagnostics>  
         <sources>  
            <source name="System.Transactions" switchValue="Verbose, ActivityTracing">  
               <listeners>  
                  <add name="Text"  
                     type="System.Diagnostics.XmlWriterTraceListener"  
                     initializeData="SysTx.log"  
                     traceOutputOptions="Callstack" />  
               </listeners>  
            </source>  
         </sources>  
         <trace autoflush="true" indentsize="4">  
         </trace>  
      </system.diagnostics>  
    </configuration>  
    ```

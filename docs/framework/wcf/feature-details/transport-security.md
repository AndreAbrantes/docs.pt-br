# <a name="transport-security"></a>Segurança de transporte
Segurança de transporte no [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] depende da associação selecionada. O transporte que implementa a associação determina o mecanismo de segurança real. Os tópicos nesta seção explicam os mecanismos que são implementados e suas opções.

# <a name="peer-nodes"></a>Nós pares
Uzel PeerNode se é um ponto de extremidade em uma malha de canal par. Um aplicativo pode ter vários nós pares que participam de malhas diferentes. Por exemplo, um aplicativo de chat peer-to-peer pode fazer parte de duas malhas separadas: uma malha de bate-papo com amigos de ensino; e outro para bate-papo com colegas de trabalho.

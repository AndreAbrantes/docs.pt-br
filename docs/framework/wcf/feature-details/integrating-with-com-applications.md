# <a name="integrating-with-com-applications"></a>Integração com aplicativos COM
Serviços Windows Communication Foundation (WCF) podem ser integrados diretamente em seu código existente usando o moniker de serviço do WCF. O moniker de serviço pode ser usado em ambientes de desenvolvimento todo com base no intervalo de COM, como Office VBA, Visual Basic 6.0 ou Visual C++ 6.0.

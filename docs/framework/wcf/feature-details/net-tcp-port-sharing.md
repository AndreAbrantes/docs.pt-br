## <a name="port-sharing-architecture"></a><span data-ttu-id="eac7a-122">Arquitetura de compartilhamento de porta</span><span class="sxs-lookup"><span data-stu-id="eac7a-122">Port Sharing Architecture</span></span>

 <span data-ttu-id="eac7a-127">O serviço de compartilhamento de porta Net. TCP é um serviço do Windows no modo de usuário que aceita net.

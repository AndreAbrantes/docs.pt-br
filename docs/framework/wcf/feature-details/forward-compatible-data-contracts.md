# <a name="forward-compatible-data-contracts"></a><span data-ttu-id="aa134-102">Contratos de dados compatíveis por encaminhamento</span><span class="sxs-lookup"><span data-stu-id="aa134-102">Forward-Compatible Data Contracts</span></span>
<span data-ttu-id="aa134-103">Um recurso do Windows Communication Foundation (WCF) é o sistema de contrato de dados que contratos pode evoluir ao longo do tempo de maneiras incondicionais.</span><span class="sxs-lookup"><span data-stu-id="aa134-103">A feature of the Windows Communication Foundation (WCF) data contract system is that contracts can evolve over time in nonbreaking ways.</span></span> <span data-ttu-id="aa134-104">Ou seja, um cliente com uma versão mais antiga de um contrato de dados pode se comunicar com um serviço com uma versão mais recente do mesmo contrato de dados, ou um cliente com uma versão mais recente de um contrato de dados pode se comunicar com uma versão mais antiga do mesmo contrato de dados.</span><span class="sxs-lookup"><span data-stu-id="aa134-104">That is, a client with an older version of a data contract can communicate with a service with a newer version of the same data contract, or a client with a newer version of a data contract can communicate with an older version of the same data contract.</span></span> <span data-ttu-id="aa134-105">Para obter mais informações, consulte [práticas recomendadas: controle de versão de contrato de dados](../../../../docs/framework/wcf/best-practices-data-contract-versioning.md).</span><span class="sxs-lookup"><span data-stu-id="aa134-105">For more information, see [Best Practices: Data Contract Versioning](../../../../docs/framework/wcf/best-practices-data-contract-versioning.md).</span></span>  
  
 <span data-ttu-id="aa134-106">Você pode aplicar a maioria dos recursos do controle de versão como necessário quando são criadas novas versões de um contrato de dados existente.</span><span class="sxs-lookup"><span data-stu-id="aa134-106">You can apply most of the versioning features on an as-needed basis when new versions of an existing data contract are created.</span></span> <span data-ttu-id="aa134-107">No entanto, um recurso de controle de versão, *ciclo*, devem ser criados para o tipo da primeira versão para funcionar adequadamente.</span><span class="sxs-lookup"><span data-stu-id="aa134-107">However, one versioning feature, *round-tripping*, must be built into the type from the first version in order to work properly.</span></span>  
  
## <a name="round-tripping"></a><span data-ttu-id="aa134-108">Ciclo</span><span class="sxs-lookup"><span data-stu-id="aa134-108">Round-Tripping</span></span>  
 <span data-ttu-id="aa134-109">Ciclo ocorre quando dados passam de uma nova versão para uma versão antiga e voltar para a nova versão de um contrato de dados.</span><span class="sxs-lookup"><span data-stu-id="aa134-109">Round-tripping occurs when data passes from a new version to an old version and back to the new version of a data contract.</span></span> <span data-ttu-id="aa134-110">Ciclo garante que nenhum dado será perdido.</span><span class="sxs-lookup"><span data-stu-id="aa134-110">Round-tripping guarantees that no data is lost.</span></span> <span data-ttu-id="aa134-111">Habilitar o ciclo faz com o tipo que compatíveis por encaminhamento com alterações futuras com suporte pelo modelo de controle de versão do contrato de dados.</span><span class="sxs-lookup"><span data-stu-id="aa134-111">Enabling round-tripping makes the type forward-compatible with any future changes supported by the data contract versioning model.</span></span>  
  
 <span data-ttu-id="aa134-112">Para habilitar o ciclo de um determinado tipo, o tipo deve implementar o <xref:System.Runtime.Serialization.IExtensibleDataObject> interface.</span><span class="sxs-lookup"><span data-stu-id="aa134-112">To enable round-tripping for a particular type, the type must implement the <xref:System.Runtime.Serialization.IExtensibleDataObject> interface.</span></span> <span data-ttu-id="aa134-113">A interface contém uma propriedade, <xref:System.Runtime.Serialization.IExtensibleDataObject.ExtensionData%2A> (retornando o <xref:System.Runtime.Serialization.ExtensionDataObject> tipo).</span><span class="sxs-lookup"><span data-stu-id="aa134-113">The interface contains one property, <xref:System.Runtime.Serialization.IExtensibleDataObject.ExtensionData%2A> (returning the <xref:System.Runtime.Serialization.ExtensionDataObject> type).</span></span> <span data-ttu-id="aa134-114">A propriedade armazena todos os dados das versões futuras do contrato de dados é desconhecido para a versão atual.</span><span class="sxs-lookup"><span data-stu-id="aa134-114">The property stores any data from future versions of the data contract that is unknown to the current version.</span></span>

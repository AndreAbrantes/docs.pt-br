# <a name="reliable-messaging"></a>Mensagens confiáveis
Este tópico lista todas as exceções geradas pelo sistema de mensagens confiável.  
  
## <a name="exception-list"></a>Lista de exceções  
  
|Código do recurso|Cadeia de caracteres de recurso|  
|-------------------|---------------------|  
|InvalidBufferRemaining|Uma mensagem foi recebida pelo buffer com o tamanho restante especificado. O buffer que permanece não pode ser menor que zero.|

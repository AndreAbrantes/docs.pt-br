# <a name="core-communications-transport-framework"></a>Comunicações principais: estrutura de transporte

## <a name="exception-list"></a>Lista de exceções  
  
|Código do recurso|Cadeia de caracteres de recurso|  
|-------------------|---------------------|  
|Hosting_TransportBindingNotFound|Nenhuma associação de protocolo corresponde ao endereço especificado. Associações de protocolo são configuradas no nível do site na configuração de serviços de informações da Internet ou serviços de ativação de processo do Windows.|

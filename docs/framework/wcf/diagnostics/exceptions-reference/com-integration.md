# <a name="com-integration"></a>Integração de COM+
Este tópico lista todas as exceções geradas pela integração de COM+.  
  
## <a name="exception-list"></a>Lista de exceções  
  
|Código do recurso|Cadeia de caracteres de recurso|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|O tipo para o parâmetro especificado dentro do namespace especificado não pode ser resolvido.|  
|NoEqualSignFound|A palavra-chave especificada não tem nenhum sinal de igual após ele. Certifique-se de que cada palavra-chave é seguido por um sinal de igual e um valor.|  
|UdtNotFoundInAssembly|Tipo definido pelo usuário especificado não foi encontrado. Certifique-se de que o tipo correto e a biblioteca de tipos são registrados e especificados.|

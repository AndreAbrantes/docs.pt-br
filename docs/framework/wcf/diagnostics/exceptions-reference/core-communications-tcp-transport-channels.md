# <a name="core-communications-tcp-transport-channels"></a>Comunicações principais: Canais de transporte TCP

Este tópico lista todas as exceções geradas pelos canais de transporte TCP.  
  
## <a name="exception-list"></a>Lista de exceções  
  
|Código do recurso|Cadeia de caracteres de recurso|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|O ponto de extremidade remoto do soquete especificado não respondeu a uma solicitação de fechamento dentro do tempo limite alocado especificado. É possível que o ponto de extremidade remoto não chame Close depois de receber o sinal EOF (NULL) da operação Receive. O tempo determinado para essa operação pode ter sido uma parte de um tempo limite maior.|

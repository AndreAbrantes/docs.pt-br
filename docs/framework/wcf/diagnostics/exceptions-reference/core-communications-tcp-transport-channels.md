# <a name="core-communications-tcp-transport-channels"></a>Comunicações centrais: canais de transporte TCP
Este tópico lista todas as exceções geradas por canais de transporte TCP.  
  
## <a name="exception-list"></a>Lista de exceções  
  
|Código do recurso|Cadeia de caracteres de recurso|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|O ponto de extremidade remoto do soquete especificado não respondeu a uma solicitação de fechamento dentro do tempo limite alocado especificado. É possível que o ponto de extremidade remoto não esteja chamando Close após receber o sinal EOF (nulo) da operação de recebimento. O tempo determinado para essa operação pode ter sido uma parte de um tempo limite maior.|

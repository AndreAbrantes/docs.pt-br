# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
A transação específica foi concluída quando a sessão foi fechada.  
  
## <a name="description"></a>Descrição  
 Rastreada quando a sessão ativa atual foi fechada e `TransactionAutoCompleteOnSessionClose` é definido como `true`.

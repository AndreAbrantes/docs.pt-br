# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Falha ao enviar uma mensagem de solicitação por HTTP.  
  
## <a name="description"></a>Descrição  
 Rastreamento de nível de aviso indica que uma solicitação de canal HTTP foi anulada. Informações adicionais neste rastreamento revelará a razão ou a mensagem passada para o método de anulação.

# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Falha ao enviar uma mensagem de solicitação por HTTP.  
  
## <a name="description"></a>Descrição  
 Este rastreamento de nível de aviso indica que uma solicitação de canal HTTP anulada. Informações adicionais neste rastreamento revelará a razão ou a mensagem transmitida para o método de anulação.

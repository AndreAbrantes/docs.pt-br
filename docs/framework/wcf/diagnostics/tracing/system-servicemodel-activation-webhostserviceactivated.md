# <a name="systemservicemodelactivationwebhostserviceactivated"></a>System.ServiceModel.Activation.WebHostServiceActivated
System.ServiceModel.Activation.WebHostServiceActivated  
  
## <a name="description"></a>Descrição  
 Ocorre quando um serviço que está hospedado no Internet Information Services (IIS) ou o serviço de ativação de processos do Windows (WAS) está totalmente ativado e começa a escutar as mensagens.

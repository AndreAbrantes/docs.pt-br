# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Falha ao tentar abrir uma conexão de vizinho.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre após a tentativa de estabelecer uma conexão de vizinho. Possíveis razões podem incluir um resolvedor não responder, uma configuração incorreta de associação ou falhas de rede temporárias.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Certifique-se de que ambos os nós tenham endereços válidos e se estão definidos corretamente.

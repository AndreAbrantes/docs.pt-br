# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Falha ao tentar abrir uma conexão vizinha.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre após a tentativa de estabelecer uma conexão vizinha. Possíveis razões podem incluir um resolvedor não responder, uma configuração incorreta de associação ou falhas de rede temporárias.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Certifique-se de que ambos os nós têm endereços válidos e estejam configurados adequadamente.

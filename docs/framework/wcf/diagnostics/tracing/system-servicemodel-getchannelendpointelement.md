# <a name="systemservicemodelgetchannelendpointelement"></a>System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## <a name="description"></a>Descrição  
 Este rastreamento é emitido ao construir uma fábrica de canais, no nível de informações. Ele fornece uma descrição do ponto de extremidade que do cliente está conversando (por exemplo, endereço remoto, associação, nome do contrato).

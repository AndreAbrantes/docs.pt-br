# <a name="systemservicemodelgetchannelendpointelement"></a>System.ServiceModel.GetChannelEndpointElement

System.ServiceModel.GetChannelEndpointElement  
  
## <a name="description"></a>Descrição  

 Esse rastreamento é emitido durante a construção de uma fábrica de canais, no nível de informações. Ele fornece uma descrição do ponto de extremidade ao qual o cliente está se comunicando (por exemplo, endereço remoto, associação, nome do contrato).

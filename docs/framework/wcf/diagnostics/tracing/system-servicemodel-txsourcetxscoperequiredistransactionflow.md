# <a name="systemservicemodeltxsourcetxscoperequiredistransactionflow"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactionFlow
A transação especificada foi colocada em fluxo para a operação especificada.  
  
## <a name="description"></a>Descrição  
 Rastreados se a transação disponibilizada para código do usuário foi colocada em fluxo em de uma fonte externa.

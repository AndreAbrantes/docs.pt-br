# <a name="microsofttransactionstransactionbridgeregisterparticipant"></a>Microsoft.Transactions.TransactionBridge.RegisterParticipant
O serviço do protocolo WS-AT registrou com êxito um participante para um protocolo de controle.  
  
## <a name="description"></a>Descrição  
 Rastreada quando o Gerenciador de transações local confirmou com êxito uma solicitação de registro de um participante para um protocolo 2pc.

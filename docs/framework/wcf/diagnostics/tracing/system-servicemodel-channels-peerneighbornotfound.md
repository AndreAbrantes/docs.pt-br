# <a name="systemservicemodelchannelspeerneighbornotfound"></a>System.ServiceModel.Channels.PeerNeighborNotFound
Uma operação em um vizinho não pôde ser executada porque vizinho não pôde ser encontrado na lista de vizinho.  
  
## <a name="description"></a>Descrição  
 Esse erro ocorre quando se conectar a ou fechar um vizinho.

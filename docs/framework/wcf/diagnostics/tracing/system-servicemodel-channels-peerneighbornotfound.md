# <a name="systemservicemodelchannelspeerneighbornotfound"></a>System.ServiceModel.Channels.PeerNeighborNotFound
Uma operação em um vizinho não pôde ser executada porque esse vizinho não pôde ser encontrado na lista de vizinho.  
  
## <a name="description"></a>Descrição  
 Esse erro ocorre durante a conexão ou fechar um vizinho.

# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Um objeto de comunicação falha ao fechar.  
  
## <a name="description"></a>Descrição  
 Um objeto de comunicação não conseguiu fechar porque ocorreu uma exceção durante a execução de seu método de fechar. Outra mais próxima de rastreamento pode conter detalhes sobre a exceção.

# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Um objeto de comunicação falha ao fechar.  
  
## <a name="description"></a>Descrição  
 Um objeto de comunicação falha ao fechar porque uma exceção foi lançada durante a execução do método Close. Outra mais próxima de rastreamento pode conter detalhes sobre a exceção.

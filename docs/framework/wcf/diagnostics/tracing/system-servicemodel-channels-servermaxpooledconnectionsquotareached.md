System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## <a name="description"></a>Descrição  
 A cota de MaxOutboundConnectionsPerEndpoint foi atingida, portanto, a conexão foi fechada e não é reutilizado pelo ouvinte.

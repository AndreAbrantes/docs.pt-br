# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
A instância do serviço foi lançada após a conclusão da transação '{0}' porque o ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute foi definido como true.  
  
## <a name="description"></a>Descrição  
 Rastreada quando a instância atual do serviço é liberada ou descartada devido à conclusão da transação ativa atual e ReleaseServiceInstanceOnTransactionComplete estiver definido como `true`.

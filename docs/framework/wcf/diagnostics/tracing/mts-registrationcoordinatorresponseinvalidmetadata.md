# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorresponseinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorResponseInvalidMetadata
O serviço de protocolo WS-AT recebeu uma mensagem RegisterResponse de seu coordenador que contém uma referência de ponto de extremidade com metadados inválidos ou incompatíveis.  
  
## <a name="description"></a>Descrição  
 Rastreada quando o Gerenciador de transações local tenta se registrar com seu Gerenciador de transações superior e a excelente retorna um endereço inválido dentro da mensagem RegisterResponse.

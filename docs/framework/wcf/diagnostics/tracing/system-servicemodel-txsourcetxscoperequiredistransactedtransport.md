# <a name="systemservicemodeltxsourcetxscoperequiredistransactedtransport"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
A transação especificada foi recebida para a operação especificada de um transporte transacionado, como MSMQ.  
  
## <a name="description"></a>Descrição  
 Rastreadas se a transação disponibilizada ao código do usuário foi criada pelo transporte em si.

# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
A referência de ponto de extremidade encontrada em uma entrada de log de recuperação participante continha metadados inválidos ou incompatíveis.  
  
## <a name="description"></a>Descrição  
 O sistema não é possível criar um proxy para uma inscrição de participante recuperado.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Se o problema persistir, entre em contato com Microsoft.

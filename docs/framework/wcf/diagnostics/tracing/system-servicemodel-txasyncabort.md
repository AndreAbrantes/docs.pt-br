# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
A transação especificada foi anulada assincronamente.  
  
## <a name="description"></a>Descrição  
 A transação atual foi anulada devido a outro participante de votação para abortar, tempos limite que ocorrem ou outro erro dentro do que o participante de uma transação.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Verifique todos os logs de sistema se essa anulação é esperada para determinar o motivo real para a anulação.

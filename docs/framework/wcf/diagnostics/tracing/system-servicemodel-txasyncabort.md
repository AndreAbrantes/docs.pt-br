# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
A transação especificada foi anulada assincronamente.  
  
## <a name="description"></a>Descrição  
 A transação atual foi anulada devido a outro participante votar anulação, tempos limite ocorrendo ou outro erro dentro de participante de uma transação.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Verifique todos os logs de sistema se essa anulação é inesperada para determinar a razão para a anulação.

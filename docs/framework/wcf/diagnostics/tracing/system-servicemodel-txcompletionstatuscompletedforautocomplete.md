# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
A transação especificada para a operação especificada foi concluída devido ao `OperationBehaviorAttribute.TransactionAutoComplete` membro que está sendo definido como `true`.  
  
## <a name="description"></a>Descrição  
 Rastreada quando a infraestrutura preenche automaticamente a transação em nome do usuário.

# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
A transação especificada para a operação especificada foi concluída devido à `OperationBehaviorAttribute.TransactionAutoComplete` membro que está sendo definido como `true`.  
  
## <a name="description"></a>Descrição  
 Rastreada quando a infra-estrutura preenche automaticamente a transação em nome do usuário.

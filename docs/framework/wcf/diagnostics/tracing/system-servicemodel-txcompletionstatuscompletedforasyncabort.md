# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
A transação especificada para a operação especificada foi concluída devido a anulação assíncrona.  
  
## <a name="description"></a>Descrição  
 A transação atual foi anulada devido a outro participante votar anulação, tempos limite ocorrendo ou outro erro dentro de participante de uma transação.  
  
## <a name="troubleshooting"></a>Solução de problemas  
 Se este abort for inesperado, verifique todos os logs de sistema para determinar a razão para a anulação.

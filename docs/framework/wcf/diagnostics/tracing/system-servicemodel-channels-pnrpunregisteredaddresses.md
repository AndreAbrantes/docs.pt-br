# <a name="systemservicemodelchannelspnrpunregisteredaddresses"></a>System.ServiceModel.Channels.PnrpUnregisteredAddresses
O resolvedor Pnrp foi cancelado endereços do nó local.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre depois que o serviço de PNRP com êxito cancela o registro de endereços do nó local.

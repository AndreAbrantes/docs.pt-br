# <a name="systemservicemodeltxsourcetxscoperequiredisattachedtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction

A transação especificada foi recebida para a operação especificada a partir de uma transação InstanceContext.  
  
## <a name="description"></a>Descrição  

 Rastreado se a transação disponibilizada para o código do usuário foi deixada por trás por um método AutoComplete = false chamado antes desta invocação.

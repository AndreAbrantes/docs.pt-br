# <a name="systemservicemodeltxsourcetxscoperequiredisattachedtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
A transação especificada foi recebida para a operação especificada de uma transação InstanceContext.  
  
## <a name="description"></a>Descrição  
 Rastreado se a transação disponibilizada ao código do usuário foi deixada para trás por um recurso de preenchimento automático = false método chamado antes dessa invocação.

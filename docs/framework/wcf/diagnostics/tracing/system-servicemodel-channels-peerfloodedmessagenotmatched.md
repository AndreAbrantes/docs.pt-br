# <a name="systemservicemodelchannelspeerfloodedmessagenotmatched"></a>System.ServiceModel.Channels.PeerFloodedMessageNotMatched
Um PeerNode recebeu uma mensagem que não corresponde a nenhum dos canais locais.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre durante o processamento de uma mensagem de fluxo de entrada.

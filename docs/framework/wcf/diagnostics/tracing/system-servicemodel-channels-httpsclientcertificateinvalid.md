# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certificado de cliente é inválido.  
  
## <a name="description"></a>Descrição  
 Este rastreamento indica que o certificado fornecido pelo cliente foi considerado inválida ao ser pelo Ouvinte HTTPS. O ouvinte HTTPS foi ao tentar validar a autenticidade do cliente usando este certificado. Um certificado pode ser inválido, se sua autoridade de certificação não é reconhecida pelo servidor que hospeda o serviço.

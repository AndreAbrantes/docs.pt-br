# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certificado de cliente é inválido.  
  
## <a name="description"></a>Descrição  
 Este rastreamento indica que o certificado fornecido pelo cliente foi considerado inválida ao ser pelo ouvinte de HTTPS. O ouvinte HTTPS estava tentando validar a autenticidade do cliente usando esse certificado. Um certificado pode ser inválido, se sua autoridade de certificação não for reconhecida pelo servidor que hospeda o serviço.

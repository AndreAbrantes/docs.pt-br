# <a name="microsofttransactionstransactionbridgecoordinatorstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.CoordinatorStateMachineFinished
A máquina de estado para uma inscrição de coordenador entrou no estado terminado.  
  
## <a name="description"></a>Descrição  
 Rastreada quando o Gerenciador de transações local reconhece que uma inscrição de coordenador superior concluiu o processamento de 2pc. O resultado para a inscrição pode ser confirmado ou anulado ou Forgotten. Se o Gerenciador de transações locais votos somente leitura durante a preparação também são rastreados.

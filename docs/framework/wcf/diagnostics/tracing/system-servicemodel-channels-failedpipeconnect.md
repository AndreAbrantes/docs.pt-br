# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Falha ao tentar se conectar ao ponto de extremidade de pipe nomeado. Outra tentativa será feita dentro do período de tempo limite especificado.  
  
## <a name="description"></a>Descrição  
 Este rastreamento informativo indica uma falha ao se conectar a um ponto de extremidade de pipe nomeado. Isso pode acontecer se o ponto de extremidade de pipe nomeado não foi encontrado ou está ocupado. São feitas tentativas adicionais, cada um separado por um curto período de tempo, até obter êxito ou o OpenTimeout expira.

# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Falha ao tentar se conectar ao ponto de extremidade do pipe nomeado. Outra tentativa é feita dentro do período de tempo limite especificado.  
  
## <a name="description"></a>Descrição  
 Esse rastreamento informativo indica uma falha ao se conectar a um ponto de extremidade do pipe nomeado. Isso pode ocorrer se o ponto de extremidade do pipe nomeado não foi encontrado ou está ocupado. São feitas tentativas adicionais, cada um separado por um curto período de tempo, até obter êxito ou o OpenTimeout expira.

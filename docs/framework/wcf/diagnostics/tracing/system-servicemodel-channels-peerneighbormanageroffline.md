# <a name="systemservicemodelchannelspeerneighbormanageroffline"></a>System.ServiceModel.Channels.PeerNeighborManagerOffline
O PeerNode é agora off-line, indicando que há não atualmente conectado vizinhos.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre após a remoção de uma aproximação da lista de vizinho do nó local.

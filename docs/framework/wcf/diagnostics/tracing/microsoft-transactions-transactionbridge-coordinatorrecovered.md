# <a name="microsofttransactionstransactionbridgecoordinatorrecovered"></a>Microsoft.Transactions.TransactionBridge.CoordinatorRecovered
Uma inscrição de coordenador foi recuperada do log de Gerenciador de transações.  
  
## <a name="description"></a>Descrição  
 Rastreada quando o Gerenciador de transações local tiver realizado com sucesso recuperação para uma inscrição de determinado coordenador superior.

# <a name="systemservicemodeltxsourcetxscoperequirediscreatenewtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsCreateNewTransaction
A transação especificada para a operação especificada foi criada recentemente.  
  
## <a name="description"></a>Descrição  
 Rastreados se a transação disponibilizada para código do usuário foi criada recentemente pela infraestrutura.

# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Certificado de cliente é necessário. Nenhum certificado foi encontrado na solicitação.  
  
## <a name="description"></a>Descrição  
 Este rastreamento indica que o ouvinte HTTPS recebeu uma solicitação HTTPS que não foi associada um certificado de cliente. Desde que o ouvinte foi configurado para exigir certificados de cliente em todas as solicitações HTTPS, o ouvinte não pôde validar a autenticidade do cliente.

# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Descrição  
 A operação de fechamento de um ServiceHost atingiu o tempo limite. Isso indica que alguns canais de sessão não foram corretamente desligamento pelo cliente e pode ter sido anulado. Modificar a hora padrão para ajustar o tempo limite de fechamento.

# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Descrição  
 Uma mensagem de resposta foi recebida, mas não há nenhum solicitações pendentes aguardando a mensagem. A mensagem está sendo descartada pelo tempo de execução do Windows Communication Foundation (WCF).  
  
 Este rastreamento pode ser usado para depuração de mensagens perdidas.

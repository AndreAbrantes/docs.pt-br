# <a name="systemservicemodelchannelspeerneighborstatechangefailed"></a>System.ServiceModel.Channels.PeerNeighborStateChangeFailed
Estado do vizinho foi corrompido.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre após uma falha ao tentar alterar o estado do vizinho. Isso indica um erro interno inesperado ou memória corrompida.

# <a name="systemservicemodelchannelspeerneighborstatechangefailed"></a>System.ServiceModel.Channels.PeerNeighborStateChangeFailed
Um estado do seu vizinho estava corrompido.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre após uma falha na tentativa de alterar o estado de um vizinho. Isso indica um erro interno inesperado ou memória corrompida.

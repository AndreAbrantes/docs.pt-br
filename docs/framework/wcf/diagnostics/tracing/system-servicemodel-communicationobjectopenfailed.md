# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Um objeto de comunicação falha ao abrir.  
  
## <a name="description"></a>Descrição  
 Um objeto de comunicação não conseguiu abrir porque ocorreu uma exceção durante a execução do seu método Open. Outra mais próxima de rastreamento pode conter detalhes sobre a exceção.

# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Um objeto de comunicação falha ao abrir.  
  
## <a name="description"></a>Descrição  
 Um objeto de comunicação não conseguiu abrir porque uma exceção foi gerada durante a execução de seu método aberto. Outra mais próxima de rastreamento pode conter detalhes sobre a exceção.

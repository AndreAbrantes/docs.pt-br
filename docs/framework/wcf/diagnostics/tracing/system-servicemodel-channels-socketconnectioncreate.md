# <a name="systemservicemodelchannelssocketconnectioncreate"></a>System.ServiceModel.Channels.SocketConnectionCreate
System.ServiceModel.Channels.SocketConnectionCreate  
  
## <a name="description"></a>Descrição  
 Esse rastreamento é emitido na primeira atividade de ação de processo realizado pelo cliente e na atividade de recebimento Bytes no serviço. Ele fornece os endereços IP locais e remotos. Ele é emitido no nível de informações.

# <a name="systemservicemodelchannelspnrpresolveexception"></a>System.ServiceModel.Channels.PnrpResolveException
Uma exceção ocorreu enquanto o PnrpResolverService tenta executar uma solução.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre durante uma operação de resolver pelo resolvedor PNRP.

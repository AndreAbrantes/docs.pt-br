# <a name="debugging-on-the-client"></a>Depuração do cliente
Para tornar mais fácil para os usuários escrever aplicativos de cliente para o [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] serviço, você pode adicionar o [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportamento para o arquivo de configuração do serviço de serviço. Esse comportamento pode ser usado para publicar as páginas de Ajuda e retornar informações de exceção gerenciada em detalhes das falhas SOAP retornadas ao cliente.

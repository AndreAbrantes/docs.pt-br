# <a name="debugging-on-the-client"></a>Depuração do cliente
Para tornar mais fácil para os usuários escrevam aplicativos cliente para seu serviço WCF, você pode adicionar o [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportamento ao arquivo de configuração do seu serviço de serviço. Esse comportamento pode ser usado para publicar páginas de Ajuda e retornar informações de exceção gerenciada nos detalhes de falhas SOAP retornadas para o cliente.

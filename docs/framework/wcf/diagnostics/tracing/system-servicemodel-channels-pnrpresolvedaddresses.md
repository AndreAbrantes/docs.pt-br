# <a name="systemservicemodelchannelspnrpresolvedaddresses"></a>System.ServiceModel.Channels.PnrpResolvedAddresses
O resolvedor Pnrp concluído uma operação de resolução.  
  
## <a name="description"></a>Descrição  
 Esse rastreamento de aviso indica que o resolvedor Pnrp não pôde localizar nenhum endereço. Este rastreamento ocorre durante uma operação de resolver pelo resolvedor PNRP.

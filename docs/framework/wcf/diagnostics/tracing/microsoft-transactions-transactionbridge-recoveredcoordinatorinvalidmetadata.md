# <a name="microsofttransactionstransactionbridgerecoveredcoordinatorinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata

A referência de ponto de extremidade encontrada em uma entrada de log de recuperação de coordenador continha metadados inválidos ou incompatíveis.  
  
## <a name="description"></a>Descrição  

 O sistema não pode criar um proxy para uma inscrição de coordenador recuperada.  
  
## <a name="troubleshooting"></a>Solução de problemas  

 Contate a Microsoft se o problema persistir.

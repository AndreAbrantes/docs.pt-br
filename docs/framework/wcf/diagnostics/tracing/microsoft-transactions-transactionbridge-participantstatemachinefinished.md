# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
A máquina de estado para uma inscrição de participante entrou no estado concluído.  
  
## <a name="description"></a>Descrição  
 Rastreada quando uma inscrição de participante subordinada concluiu o processamento de 2pc. O resultado para a inscrição pode ser confirmadas ou anuladas. Ele também é rastreado se qualquer participante votos somente leitura durante a preparação.

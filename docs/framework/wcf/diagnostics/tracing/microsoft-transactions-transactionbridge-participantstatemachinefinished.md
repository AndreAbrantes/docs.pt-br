# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
A máquina de estado para uma inscrição de participante entrou no estado terminado.  
  
## <a name="description"></a>Descrição  
 Rastreada quando uma inscrição de participante subordinada concluiu o processamento 2pc. O resultado para a inscrição pode ser confirmadas ou anuladas. Se qualquer participante votos somente leitura durante a preparação também são rastreados.

# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Descrição  
 Um do acelerador de instâncias/MaxConcurrentCalls/sessões foi atingido e uma mensagem está sendo enfileirada.  
  
 Você pode usar este rastreamento para ajustar os valores de limitação com base na carga de produção esperados e reais.

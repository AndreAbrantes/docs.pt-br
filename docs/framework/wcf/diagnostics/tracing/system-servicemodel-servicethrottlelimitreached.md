# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached

System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Descrição  

 Um dos aceleradores de MaxConcurrentCalls/instâncias/sessões foi atingido e uma mensagem está sendo enfileirada.  
  
 Você pode usar esse rastreamento para ajustar os valores de limitação com base na carga de produção esperada e real.

# <a name="systemservicemodelchannelsprematuredatagrameof"></a>System.ServiceModel.Channels.PrematureDatagramEof

System.ServiceModel.Channels.PrematureDatagramEof  
  
## <a name="description"></a>Descrição  

 Uma mensagem nula (sinalização final do canal) foi recebida de um canal de datagrama, mas o canal ainda está no estado aberto. Isso indica um erro no canal do datagrama e o loop de recebimento do Desmultiplexador foi encerrado prematuramente.

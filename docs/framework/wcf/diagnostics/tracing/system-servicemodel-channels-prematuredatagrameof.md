# <a name="systemservicemodelchannelsprematuredatagrameof"></a>System.ServiceModel.Channels.PrematureDatagramEof
System.ServiceModel.Channels.PrematureDatagramEof  
  
## <a name="description"></a>Descrição  
 Uma mensagem nula (final de sinalização de canal) foi recebida de um canal de datagrama, mas o canal ainda está no estado aberto. Isso indica um erro no canal de datagrama e a eliminação multiplexer receber loop foi encerrado prematuramente.

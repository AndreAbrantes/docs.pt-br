# <a name="systemservicemodelportsharingreadnettcpconfig"></a>System.ServiceModel.PortSharing.ReadNetTcpConfig
System.ServiceModel.PortSharing.ReadNetTcpConfig  
  
## <a name="description"></a>Descrição  
 Dados de configuração para o NetTcpSection foi lida com os dados especificados: listenBacklog, maxConnectionsPendingDispatch, maxPendingAccepts, receiveTimeout e teredoEnabled.

# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound

System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Descrição  

 O desserializador não tem conhecimento do tipo a ser usado durante a desserialização. Considere adicionar o tipo especificado como o tipo de fábrica no XML à lista de tipos conhecidos, por exemplo, usando o atributo KnownTypeattribute.

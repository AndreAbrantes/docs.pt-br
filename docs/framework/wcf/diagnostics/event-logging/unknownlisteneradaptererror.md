# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Id: 161  
  
 Gravidade: Erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro inesperado no adaptador do ouvinte ao manipular uma notificação de serviço de ativação de processos do Windows (WAS). O processo é encerrado. O evento de lista, o protocolo, a exceção, o nome do processo e a ID de processo.

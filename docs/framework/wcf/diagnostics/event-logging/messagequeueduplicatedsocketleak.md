# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
Id: 165  
  
 Gravidade: Erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro durante o envio de um soquete duplicado. Esse identificador é perdido no processo. O evento lista o código-fonte, a exceção, o nome do processo e a ID de processo.

# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
Id: 157  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com o nome de assunto específico e a impressão digital não fornece os dados específicos entre seu EnhancedKeyUsages. O evento lista o nome de processo e a ID de processo.

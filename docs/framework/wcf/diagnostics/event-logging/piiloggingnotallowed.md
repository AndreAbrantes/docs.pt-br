# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed
Id: 108  
  
 Gravidade: Erro  
  
 Categoria: Rastreamento  
  
## <a name="description"></a>Descrição  
 Esse evento indica que nenhuma PII conhecido é que está sendo registrado. Registro em log de PII conhecido não é permitido. Para permitir o registro em log de PII conhecido, defina "enableLoggingKnownPii" `true` em Machine. config. O evento lista o nome de processo e a ID de processo.

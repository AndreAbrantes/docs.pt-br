# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed
Sessão 108  
  
 Severity Erro  
  
 Categorias Rastreamento  
  
## <a name="description"></a>Descrição  
 Esse evento indica que nenhuma PII conhecida está sendo registrada em log. O registro em log de PII conhecido não é permitido. Para permitir o log de PII conhecido, defina "enableLoggingKnownPii" `true` como em Machine. config. O evento lista o nome do processo e a ID do processo.

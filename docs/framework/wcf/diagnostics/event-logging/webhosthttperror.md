# <a name="webhosthttperror"></a>WebHostHttpError
ID: 110  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu uma exceção de WebHost HTTP. O evento lista exceção HtmlErrorMessage, informações de remetente, nome do processo e ID de processo.

# <a name="webhosthttperror"></a>WebHostHttpError
Id: 110  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu uma exceção de HTTP do WebHost. O evento lista exceção HtmlErrorMessage, informações de remetente, nome do processo e ID de processo.

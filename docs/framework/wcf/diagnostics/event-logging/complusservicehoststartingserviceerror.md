# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError

ID: 130  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  

 Esse evento indica que ocorreu um erro ao iniciar o serviço COM+. O evento lista o AppID, ClsID, substituto, exceção, nome do processo e ID do processo.

# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
ID: 130  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao iniciar o serviço COM+. O evento lista AppID, ClsID, substitutos, exceção, o nome do processo e ID de processo.

# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
Id: 130  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao iniciar o serviço COM+. O evento lista AppID, ClsID, substitutos, exceção, nome do processo e ID de processo.

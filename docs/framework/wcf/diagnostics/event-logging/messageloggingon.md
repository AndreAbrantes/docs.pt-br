# <a name="messageloggingon"></a>MessageLoggingOn
ID: 116  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Log de mensagens foi ativado. Podem ser registradas informações confidenciais, mesmo se ele foi criptografado durante a transmissão. Por exemplo, corpos de mensagens. O evento lista o AppID, a exceção, o nome do processo e a ID de processo.

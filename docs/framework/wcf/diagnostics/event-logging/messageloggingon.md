# <a name="messageloggingon"></a>MessageLoggingOn
Id: 116  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Log de mensagens foi ativado. Informações confidenciais podem ser registradas, mesmo se ele tiver sido criptografado durante a transmissão. Por exemplo, corpos de mensagem. O evento lista o AppID, a exceção, o nome do processo e a ID de processo.

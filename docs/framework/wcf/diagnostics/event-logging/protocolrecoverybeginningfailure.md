# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Sessão 144  
  
 Severity Erro  
  
 Categorias TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o protocolo de transação WS-Atomic do MSDTC falhou no início da recuperação. Como resultado, a funcionalidade de transação WS-Atomic está desabilitada. O evento lista a ID do protocolo, o nome do protocolo, a exceção, o nome do processo e a ID do processo.

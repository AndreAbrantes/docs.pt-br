# <a name="protocolstopped"></a>ProtocolStopped
ID: 151  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo WS-AT foi interrompido. O evento lista o ID do protocolo, nome do protocolo, nome do processo e ID de processo. .

# <a name="protocolstartfailure"></a>ProtocolStartFailure
Id: 143  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo de transação WS-Atomic falhou ao iniciar. Como resultado, a funcionalidade de transação WS-Atomic está desabilitada. O evento lista o ID de protocolo, nome do protocolo, exceção, nome do processo e ID de processo. .

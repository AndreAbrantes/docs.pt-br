# <a name="protocolstartfailure"></a>ProtocolStartFailure
Sessão 143  
  
 Severity Erro  
  
 Categorias TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo WS-Atomic Transaction falhou ao iniciar. Como resultado, a funcionalidade de transação WS-Atomic está desabilitada. O evento lista a ID do protocolo, o nome do protocolo, a exceção, o nome do processo e a ID do processo. .

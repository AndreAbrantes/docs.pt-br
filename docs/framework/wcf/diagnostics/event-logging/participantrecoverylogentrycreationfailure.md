# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
ID: 141  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a entrada de log arecovery não pôde ser gerada para uma inscrição de participante. A transação será anulada. A lista de eventos a ID da transação, ID de inscrição, exceção, o nome do processo e ID de processo.

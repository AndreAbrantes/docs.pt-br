# <a name="messageauthenticationsuccess"></a>MessageAuthenticationSuccess
ID: 169  
  
 Severidade: informativa  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que autenticação de mensagem foi bem-sucedida. O evento lista o serviço, a ação, a identidade do cliente e o ID de atividade.

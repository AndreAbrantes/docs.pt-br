# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess
ID: 171  
  
 Gravidade: informativo  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica a negociação de segurança tiver sido bem-sucedido, listando o nome do serviço, ação, identidade do cliente, ActivityId e negociação.

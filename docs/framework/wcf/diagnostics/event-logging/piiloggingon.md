# <a name="piiloggingon"></a>PiiLoggingOn
ID: 107  
  
 Severidade: erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o log de PII foi ativado. As informações confidenciais serão registradas. O evento lista o nome de processo e a ID de processo.

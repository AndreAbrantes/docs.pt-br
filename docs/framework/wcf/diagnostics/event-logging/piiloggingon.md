# <a name="piiloggingon"></a>PiiLoggingOn
Id: 107  
  
 Gravidade: Erro  
  
 Categoria: Rastreamento  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o registro em log o PII foi ativado. As informações confidenciais serão registradas. O evento lista o nome de processo e a ID de processo.

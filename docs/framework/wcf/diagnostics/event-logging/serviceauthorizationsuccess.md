# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
Id: 176  
  
 Gravidade: Informativo  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a autorização do serviço foi bem-sucedida. O evento lista o serviço, ação, identidade do cliente, contexto de autorização, ID da atividade e ServiceAuthorizationManager.

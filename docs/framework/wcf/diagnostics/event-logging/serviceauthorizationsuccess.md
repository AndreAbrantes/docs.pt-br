# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
ID: 176  
  
 Severidade: informativa  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a autorização do serviço foi bem-sucedida. O evento lista o serviço, ação, identidade do cliente, o contexto de autorização, ID da atividade e ServiceAuthorizationManager.

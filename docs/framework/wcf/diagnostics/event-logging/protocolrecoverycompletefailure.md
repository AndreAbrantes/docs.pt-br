# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
Id: 145  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço do protocolo WS-AT não conseguiu concluir a inicialização e recuperação. Como resultado, a funcionalidade de transação WS-Atomic está desabilitada. O evento lista o ID de protocolo, nome do protocolo, exceção, nome do processo e ID de processo.

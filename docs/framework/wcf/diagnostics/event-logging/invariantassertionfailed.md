# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
ID: 106  
  
 Severidade: erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  
 Esse evento indica que uma declaração invariável é falsa. O evento lista a mensagem, o nome do processo e a ID de processo.

# <a name="failfastexception"></a>FailFastException
ID: 103  
  
 Severidade: erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  
 Uma exceção foi lançada durante FailFast. O evento lista a exceção, o nome do processo e a ID de processo.

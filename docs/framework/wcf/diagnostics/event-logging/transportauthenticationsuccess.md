# <a name="transportauthenticationsuccess"></a>TransportAuthenticationSuccess
ID: 173  
  
 Severidade: informativa  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a autenticação de transporte foi bem-sucedida. O evento lista o serviço, a identidade do cliente e a atividade de ID.

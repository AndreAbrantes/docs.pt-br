# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
ID: 122  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a solicitação DeleteInstance WMI não foi processada. O evento lista a exceção, o nome do processo e a ID de processo.

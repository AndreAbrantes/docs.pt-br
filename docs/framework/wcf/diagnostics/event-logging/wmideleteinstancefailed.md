# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
Sessão 122  
  
 Severity Erro  
  
 Categorias ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a solicitação de DeleteInstance do WMI não foi processada. O evento lista a exceção, o nome do processo e a ID do processo.

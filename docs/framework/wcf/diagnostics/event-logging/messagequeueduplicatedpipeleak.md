# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak
Sessão 166  
  
 Severity Erro  
  
 Categorias SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro durante a expedição de um pipe nomeado duplicado. Esse identificador agora está vazado no processo. O evento lista a origem, a exceção, o nome do processo e a ID do processo.

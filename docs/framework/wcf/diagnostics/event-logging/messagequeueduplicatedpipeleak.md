# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak
ID: 166  
  
 Severidade: erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro durante o envio de um pipe nomeado duplicado. Essa alça agora está perdida no processo. O evento lista a origem, a exceção, o nome do processo e o ID de processo.

# <a name="protocolstopfailure"></a>ProtocolStopFailure
ID: 147  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo WS-AT não pôde parar. O evento lista o ID do protocolo, nome do protocolo, exceção, o nome do processo e ID de processo.

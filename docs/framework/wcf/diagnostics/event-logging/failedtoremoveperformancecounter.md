# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
ID: 119  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um contador de desempenho não foi removido. O evento lista o nome da categoria, nome do contador, nome da instância, exceção, o nome do processo e ID de processo.

# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
ID: 111  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que falha do WebHost ao processar uma solicitação. O evento lista as informações do remetente, a exceção, o nome do processo e o processo ID.

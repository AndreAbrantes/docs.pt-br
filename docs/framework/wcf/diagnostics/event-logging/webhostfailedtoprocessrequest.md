# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
Id: 111  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o WebHost falhou ao processar uma solicitação. O evento listas as informações do remetente, a exceção, o nome do processo e o processo de identificação.

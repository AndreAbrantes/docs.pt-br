# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Id: 160  
  
 Gravidade: Erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro no serviço de ativação especificado do protocolo especificado, ao tentar escutar para o site especificado, que assim, o protocolo é desabilitado para o site temporariamente... O evento de lista a URL, status, exceção, o nome do processo e ID de processo.

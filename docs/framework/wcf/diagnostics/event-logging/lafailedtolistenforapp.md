# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
ID: 160  
  
 Severidade: erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro no serviço de ativação especificado do protocolo especificado, ao tentar ouvir o site especificado, que assim, o protocolo está desabilitado para o site temporariamente. O evento lista a URL, status, exceção, o nome do processo e ID de processo.

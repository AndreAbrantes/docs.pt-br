# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
ID: 114  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um filtro de log de mensagens incorretas que estava acessando o corpo da mensagem foi removido. O evento lista o XPath, o nome do processo e a ID de processo.

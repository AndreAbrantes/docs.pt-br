# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
Id: 114  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um filtro de log de mensagens defeituosos que estava acessando o corpo da mensagem foi removido. O evento de lista, o XPath, o nome do processo e a ID de processo.

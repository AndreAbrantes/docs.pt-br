# <a name="wasdisconnected"></a>WasDisconnected
Id: 162  
  
 Gravidade: Erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a ativação de processos do Windows (WAS) foi desconectado. O evento lista o HRESULT, o nome do processo e a ID de processo.

# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
Id: 152  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com a impressão digital específico não pôde ser encontrado. O evento lista o nome do processo e a ID de processo.

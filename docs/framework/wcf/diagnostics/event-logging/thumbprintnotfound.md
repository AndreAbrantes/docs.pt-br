# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
ID: 152  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com a impressão digital específico não pôde ser encontrado. O evento lista o nome do processo e o ID de processo.

# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
ID: 150  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo WS-AT foi concluída com êxito inicialização e recuperação. O evento lista o ID do protocolo, nome do protocolo, nome do processo e ID de processo.

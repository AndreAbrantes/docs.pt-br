# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
Sessão 150  
  
 Severity Erro  
  
 Categorias TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo de transação WS-Atomic concluiu com êxito a inicialização e a recuperação. O evento lista a ID do protocolo, o nome do protocolo, o nome do processo e a ID do processo.

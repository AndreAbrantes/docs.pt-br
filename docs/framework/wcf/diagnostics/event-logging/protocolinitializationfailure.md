# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
Sessão 142  
  
 Severity Erro  
  
 Categorias TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço de protocolo de transação WS-Atomic falhou ao inicializar. Como resultado, a funcionalidade de transação WS-Atomic está desabilitada. O evento lista a ID do protocolo, o nome do protocolo, a exceção, o nome do processo e a ID do processo. .

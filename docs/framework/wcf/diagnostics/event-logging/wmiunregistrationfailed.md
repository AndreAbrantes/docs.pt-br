# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
ID: 127  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o provedor WMI não foi cancelado. O evento lista o objeto WMI, o erro, o nome do processo e a ID de processo.

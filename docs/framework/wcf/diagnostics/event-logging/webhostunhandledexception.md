# <a name="webhostunhandledexception"></a>WebHostUnhandledException
ID: 109  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que uma WebHost exceção não tratada ocorreu. O evento lista as informações do remetente, a exceção, o nome do processo e o processo ID.

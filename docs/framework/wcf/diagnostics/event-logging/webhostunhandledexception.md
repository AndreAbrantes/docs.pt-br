# <a name="webhostunhandledexception"></a>WebHostUnhandledException
Id: 109  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um WebHost de exceção sem tratamento ocorreu. O evento listas as informações do remetente, a exceção, o nome do processo e o processo de identificação.

# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
Id: 153  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com a impressão digital especificada não pôde ser validado. O evento lista o nome de processo e a ID de processo.

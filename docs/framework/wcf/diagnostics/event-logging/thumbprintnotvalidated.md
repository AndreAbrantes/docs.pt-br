# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated
ID: 153  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que não foi possível validar um certificado de identidade com a impressão digital especificada. O evento lista o nome de processo e a ID de processo.

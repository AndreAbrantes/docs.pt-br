# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure
ID: 174  
  
 Severidade: erro  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a autenticação do transporte falhou, listando a identidade do cliente e a ID de atividade.

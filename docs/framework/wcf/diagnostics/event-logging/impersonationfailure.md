# <a name="impersonationfailure"></a>ImpersonationFailure
ID: 176  
  
 Severidade: erro  
  
 Categoria: SecurityAudit  
  
## <a name="description"></a>Descrição  
 Esse evento indica que representação falhou, listando o nome do método, a identidade do cliente e a ID de atividade.

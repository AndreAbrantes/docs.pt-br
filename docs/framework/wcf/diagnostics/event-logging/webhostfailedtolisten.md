# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id: 112  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao tentar escutar a URL específica. Esse processo de trabalho é encerrado. O evento de lista de exceção, o nome do processo e a ID de processo.

# <a name="sslnoprivatekey"></a>SslNoPrivateKey
Id: 154  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com o nome de assunto específico e a impressão digital não tem uma chave privada. O evento lista o nome de processo e a ID de processo.

# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
ID: 131  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao iniciar o inicializador DllHost. O evento lista o AppID, a exceção, o nome do processo e a ID de processo.

# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
ID: 126  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o provedor WMI não foi registrado. O evento lista o objeto WMI, o erro, o nome do processo e a ID de processo.

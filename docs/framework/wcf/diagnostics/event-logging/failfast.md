# <a name="failfast"></a>FailFast
ID: 102  
  
 Severidade: erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  
 FailFast foi invocado. O evento lista a mensagem, o rastreamento de pilha, o nome do processo e a ID de processo.

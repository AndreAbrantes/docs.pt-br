# <a name="starterrorpublish"></a>StartErrorPublish
Id: 158  
  
 Gravidade: Erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o serviço NT falhou ao iniciar. Não foi possível publicar o ponto de extremidade de escutando. O evento lista o nome de processo e a ID de processo.

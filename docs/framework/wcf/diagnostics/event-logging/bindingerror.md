# <a name="bindingerror"></a>BindingError
ID: 159  
  
 Severidade: erro  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao analisar a associação específica do site específico, portanto, o protocolo está desabilitado para o site temporariamente. O evento listas de associação, fonte, exceção, o nome do processo e ID de processo.

# <a name="failedtotraceeventwithexception"></a>FailedToTraceEventWithException
ID: 105  
  
 Severidade: erro  
  
 Categoria: rastreamento  
  
## <a name="description"></a>Descrição  
 Eventos não foram rastreados.  
  
 O evento lista a cadeia de caracteres do evento original, a exceção, o nome do processo e a ID de processo.

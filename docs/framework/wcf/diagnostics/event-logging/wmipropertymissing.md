# <a name="wmipropertymissing"></a>WmiPropertyMissing
ID: 129  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu uma incompatibilidade de tipo. A classe CIM especificada não tem a propriedade especificada do tipo específico. O evento lista o nome de processo e a ID de processo.

# <a name="wmipropertymissing"></a>WmiPropertyMissing
Id: 129  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu uma incompatibilidade de tipo. A classe do CIM especificada não tem a propriedade especificada do tipo específico. O evento lista o nome de processo e a ID de processo.

# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
ID: 148  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que uma máquina de estado tentou processar um evento inesperado. O evento não é considerado fatal. O evento lista o ID da transação, estado máquina, estado atual, histórico, evento nome, evento detalhes, o nome do processo e ID de processo.

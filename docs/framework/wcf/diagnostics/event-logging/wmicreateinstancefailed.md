# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
ID: 123  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a requisição CreateInstance WMI não foi processada. O evento lista o nome da classe, a exceção, o nome do processo e o ID de processo.

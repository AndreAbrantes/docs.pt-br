# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Id: 123  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que a solicitação de WMI CreateInstance não foi processada. O evento lista o nome de classe, a exceção, o nome do processo e a ID de processo.

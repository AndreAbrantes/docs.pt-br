# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey
ID: 155  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que um certificado de identidade com o nome de assunto específico e a impressão digital não tem uma chave privada acessível. O evento lista o nome de processo e a ID de processo.

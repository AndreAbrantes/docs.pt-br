# <a name="complustlbimporterror"></a>ComPlusTLBImportError
Id: 132  
  
 Gravidade: Erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao importar a biblioteca de tipos. O evento listas IID, ID do tipo de biblioteca, exceção, nome do processo e ID de processo.

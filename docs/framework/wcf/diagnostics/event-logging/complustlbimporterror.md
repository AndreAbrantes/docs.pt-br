# <a name="complustlbimporterror"></a>ComPlusTLBImportError
ID: 132  
  
 Severidade: erro  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>Descrição  
 Esse evento indica que ocorreu um erro ao importar a biblioteca de tipos. O evento lista IID, ID do tipo de biblioteca, exceção, o nome do processo e ID de processo.

# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
ID: 146  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que TransactionBridge MSDTC falhou durante a recuperação. Isso é uma condição fatal, para que o serviço MSDTC foi encerrado. O evento lista a exceção, o nome do processo e a ID de processo. .

# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Id: 146  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o MSDTC TransactionBridge falhou durante a recuperação. Essa é uma condição fatal, portanto, o serviço MSDTC foi encerrada. O evento de lista de exceção, o nome do processo e a ID de processo. .

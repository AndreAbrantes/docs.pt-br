# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
ID: 156  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o certificado de identidade com o nome de entidade e impressão digital não fornece dados específicos entre seus KeyUsages. Ela também lista o nome do processo e a ID de processo.

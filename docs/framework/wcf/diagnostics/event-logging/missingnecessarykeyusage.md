# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage
Id: 156  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que o certificado de identidade com o nome do assunto especificado e a impressão digital não fornece um dado específico entre seu KeyUsages. Ela também lista o nome do processo e a ID de processo.

# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Id: 137  
  
 Gravidade: Erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Esse evento indica que uma máquina de estado tentou processar um evento inesperado. O evento foi considerado fatal. O evento lista o ID da transação, estado máquina, o estado atual, histórico, evento nome, evento detalhes, nome do processo e ID de processo.

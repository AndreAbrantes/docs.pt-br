# <a name="queue-dropped-messages"></a>Mensagens de fila ignoradas
Nome do contador: Na fila de mensagens descartadas.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que são ignoradas pelo transporte em fila neste serviço.  
  
 Consulte [manipulação de mensagens suspeitas](http://go.microsoft.com/fwlink/?LinkId=96233) para obter mais informações sobre quando as mensagens são descartadas.

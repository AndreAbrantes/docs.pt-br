# <a name="queue-dropped-messages"></a>Mensagens de fila ignoradas
Nome do contador: Mensagens em fila é descartada.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que serão descartadas pelo transporte enfileirado neste serviço.  
  
 Ver [tratamento de mensagens suspeitas](https://go.microsoft.com/fwlink/?LinkId=96233) para obter mais informações sobre quando as mensagens são descartadas.

# <a name="queued-rejected-messages"></a>Mensagens rejeitadas em fila
Nome do contador: Na fila de mensagens rejeitadas.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que foram rejeitadas pelo transporte em fila neste serviço.  
  
 Consulte [manipulação de mensagens suspeitas](http://go.microsoft.com/fwlink/?LinkID=96233) para obter mais informações sobre quando as mensagens são rejeitadas.

# <a name="calls-failed"></a>Chamadas com falha
Nome do contador: Chamadas com falha  
  
## <a name="description"></a>Descrição  
 Número de chamadas com exceções não tratadas nesta operação. Esse contador é incrementado toda vez que há uma exceção sem tratamento nesta operação.

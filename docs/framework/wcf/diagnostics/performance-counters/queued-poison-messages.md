# <a name="queued-poison-messages"></a>Mensagens Suspeitas em Fila
Nome do contador: Mensagens suspeitas na fila.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que são marcados como inviabilizado pelo transporte neste serviço em fila.  
  
 Ver [tratamento de mensagens suspeitas](https://go.microsoft.com/fwlink/?LinkID=96233) para obter mais informações sobre quando as mensagens são marcados como inviabilizadas.

# <a name="queued-poison-messages"></a>Mensagens Suspeitas em Fila
Nome do contador: Mensagens suspeitas em fila.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que são marcadas como suspeitas pelo transporte em fila neste serviço.  
  
 Consulte [manipulação de mensagens suspeitas](http://go.microsoft.com/fwlink/?LinkID=96233) para obter mais informações sobre quando as mensagens são marcadas como suspeitas.

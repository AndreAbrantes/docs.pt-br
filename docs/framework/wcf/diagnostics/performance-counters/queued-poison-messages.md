# <a name="queued-poison-messages"></a>Mensagens Suspeitas em Fila
Contador Nome: Mensagens venenosas enfileiradas.  
  
## <a name="description"></a>Descrição  
 Número de mensagens que são marcadas envenenadas pelo transporte enfileirado neste serviço.  
  
 Para obter mais informações sobre quando as mensagens são marcadas envenenadas, consulte [Manipulação de mensagens venenosas](../../feature-details/poison-message-handling.md).

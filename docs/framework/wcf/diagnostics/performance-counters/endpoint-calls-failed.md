# <a name="endpoint-calls-failed"></a>Ponto de extremidade: Chamadas com falha
Nome do contador: Chamadas com falha.  
  
## <a name="description"></a>Descrição  
 Número de chamadas com exceções não tratadas neste ponto de extremidade. Esse contador é incrementado toda vez que há uma exceção sem tratamento nesse ponto de extremidade.

# <a name="service-calls-duration"></a>Serviço: Duração das chamadas
Nome do contador: Duração das chamadas  
  
## <a name="description"></a>Descrição  
 A duração média das chamadas para este serviço. A duração média é calculada com base nesta equação: (N1-N0)/(D1-D0).

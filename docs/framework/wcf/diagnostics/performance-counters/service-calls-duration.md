# <a name="service-calls-duration"></a>Serviço: Duração de chamadas
Nome do contador: Duração de chamadas  
  
## <a name="description"></a>Descrição  
 A duração média das chamadas para esse serviço. A duração média é calculada com base nesta equação: (N1-N0)/(D1-D0).

# <a name="service-calls-duration"></a>Serviço: Duração de chamada
Nome do contador: Duração de chamada  
  
## <a name="description"></a>Descrição  
 A duração média das chamadas para esse serviço. A duração média é calculada com base na equação: (N1-N0)/(D1-D0).

# <a name="transactions-flowed"></a>Fluxo de transações
Nome do contador: O fluxo de transações  
  
## <a name="description"></a>Descrição  
 Número de transações fluíram para essa operação. Esse contador é incrementado para qualquer mensagem que é enviada para a operação e contém uma ID da transação.

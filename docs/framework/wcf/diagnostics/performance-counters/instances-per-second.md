# <a name="instances-per-second"></a>Instâncias por segundo
Nome do contador: Instâncias criadas por segundo.  
  
## <a name="description"></a>Descrição  
 Número total de instâncias de serviço criado em um segundo.  
  
 Esse contador é do tipo de contador de desempenho [PERF_COUNTER_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), cujo valor é calculado usando a fórmula a seguir.  
  
 (1 - N 0 N) / ((D - 1D 0) / F)

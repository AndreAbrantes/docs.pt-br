# <a name="service-calls-failed"></a>Serviço: Chamadas com falha
Nome do contador: Chamadas com falha.  
  
## <a name="description"></a>Descrição  
 Número de chamadas com exceções não tratadas neste serviço. Esse contador é incrementado sempre que houver uma exceção não tratada neste serviço.

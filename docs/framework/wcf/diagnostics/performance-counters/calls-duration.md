# <a name="calls-duration"></a>Duração de chamada
Nome do contador: Duração de chamadas  
  
## <a name="description"></a>Descrição  
 A duração média das chamadas para esta operação. A duração média é calculada com base nesta equação: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Quando usado em um serviço WCF assíncrono, o contador de duração de chamadas sempre retornará -1.

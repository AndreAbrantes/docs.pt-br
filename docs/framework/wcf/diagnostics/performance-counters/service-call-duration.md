# <a name="service-call-duration"></a>Duração de chamada de serviço
Nome do contador: Duração da chamada  
  
## <a name="description"></a>Descrição  
 A duração média das chamadas para esse serviço. A duração média é calculada com base nesta equação: (N1-N0)/(D1-D0).

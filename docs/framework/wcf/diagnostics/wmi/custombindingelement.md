# <a name="custombindingelement"></a>CustomBindingElement
CustomBindingElement  
  
## <a name="syntax"></a>Sintaxe  
  
```  
class CustomBindingElement : BindingElement  
{  
  string Name;  
};  
```  
  
## <a name="methods"></a>Métodos  
 A classe CustomBindingElement não define nenhum método.  
  
## <a name="properties"></a>Propriedades  
 A classe CustomBindingElement tem a seguinte propriedade:  
  
### <a name="name"></a>Nome  
 Tipo de dados: cadeia de caracteres  
  
 Tipo de acesso: somente leitura  
  
 Uma cadeia de caracteres que contém o nome da configuração da associação. Esse valor é uma cadeia de caracteres definida pelo usuário que age como a cadeia de caracteres de identificação para a associação personalizada.

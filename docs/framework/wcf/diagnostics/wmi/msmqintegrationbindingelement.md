# <a name="msmqintegrationbindingelement"></a>MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## <a name="syntax"></a>Sintaxe  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## <a name="methods"></a>Métodos  
 A classe MsmqIntegrationBindingElement não define nenhum método.  
  
## <a name="properties"></a>Propriedades  
 A classe MsmqIntegrationBindingElement tem a seguinte propriedade:  
  
### <a name="serializationformat"></a>serializationFormat  
 Tipo de dados: cadeia de caracteres  
  
 Tipo de acesso: somente leitura  
  
 O formato que a associação usa para serializar mensagens.

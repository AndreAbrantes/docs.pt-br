## <a name="description"></a>Descrição  
 Esse evento é emitido quando o DiscoveryClient criado em DiscoveryClientChannel falhou ao fechar e, portanto, foi anulado.  
  
## <a name="message"></a>Mensagem  
 O DiscoveryClient criado em DiscoveryClientChannel falhou ao fechar e portanto foi cancelado.

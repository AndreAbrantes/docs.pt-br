# <a name="4804---discoverymessagereceivedafteroperationcompleted"></a>4804 - DiscoveryMessageReceivedAfterOperationCompleted
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4804|  
|Palavras-chave|Descoberta|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem de descoberta foi recebida depois que a operação foi concluída.  
  
## <a name="message"></a>Mensagem  
 Uma mensagem %1 com messageId = '%2' foi descartada pelo DiscoveryClient porque a operação de %3 correspondente foi concluída.

## <a name="description"></a>Descrição  
 Esse evento é emitido quando o SynchronizationContext foi definido como nulo pelo DiscoveryClientChannel antes de iniciar a operação de localização.  
  
## <a name="message"></a>Mensagem  
 O SynchronizationContext foi definido como nulo pelo DiscoveryClientChannel antes de iniciar a operação de localização.

## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4821|  
|Palavras-chave|Descoberta|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando o SynchronizationContext foi definido como null pelo DiscoveryClientChannel antes de iniciar a operação de localização.  
  
## <a name="message"></a>Mensagem  
 O SynchronizationContext foi definido como null por DiscoveryClientChannel antes de iniciar a operação de localização.

# <a name="4812---discoverymessagewithnullreplyto"></a>4812 - DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4812|  
|Palavras-chave|Descoberta|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem de descoberta foi descartada pelo DiscoveryClient porque ela não tinha um endereço ReplyTo.  
  
## <a name="message"></a>Mensagem  
 Uma mensagem de solicitação de descoberta com messageId = '%1' foi descartada porque ela não tinha um endereço ReplyTo.

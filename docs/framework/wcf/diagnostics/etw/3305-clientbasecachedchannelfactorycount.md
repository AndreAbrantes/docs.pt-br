# <a name="3305---clientbasecachedchannelfactorycount"></a>3305 - ClientBaseCachedChannelFactoryCount
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|3305|  
|Palavras-chave|ServiceModel|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido para relatar a contagem de fábrica de canais de base de clientes em cache.  
  
## <a name="message"></a>Mensagem  
 Número de fábricas de canais armazenadas em cache é: '%1'.  No máximo '%2' fábricas de canais podem ser armazenado em cache.

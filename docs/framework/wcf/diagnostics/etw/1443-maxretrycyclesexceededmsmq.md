## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|1443|  
|Palavras-chave|Cota|  
|Nível|Erro|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/analítico|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando ciclos de máximo de repetições são excedidos em uma mensagem do MSMQ.  
  
## <a name="message"></a>Mensagem  
 Ciclos máximos de repetições excedidos na mensagem MSMQ com a id '%1'

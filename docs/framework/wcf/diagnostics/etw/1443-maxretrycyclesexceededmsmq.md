|||  
|-|-|

|Palavras-chave|Quota|

## <a name="description"></a>Descrição  
 Esse evento é emitido quando ciclos de tentativa máximo for excedidos em uma mensagem MSMQ.  
  
## <a name="message"></a>Mensagem  
 Ciclos de repetição máximo excedidos na mensagem com id '%1' do MSMQ

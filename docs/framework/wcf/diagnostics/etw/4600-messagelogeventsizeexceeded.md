# <a name="4600---messagelogeventsizeexceeded"></a>4600 - MessageLogEventSizeExceeded
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4600|  
|Palavras-chave|WCFMessageLogging|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem não pôde ser registrada porque ele excede o tamanho de eventos ETW.  
  
## <a name="message"></a>Mensagem  
 Mensagem não pôde ser registrada, pois ele excede o tamanho do evento ETW

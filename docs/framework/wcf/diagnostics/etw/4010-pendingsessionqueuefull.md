## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4010|  
|Palavras-chave|ActivationServices|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/analítico|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a distribuição de sessão falhou porque a fila de sessão pendentes está cheia.  
  
## <a name="message"></a>Mensagem  
 Distribuição de sessão para '%1' falhou, pois a fila pendentes sessão está cheia com '%2' itens pendentes.

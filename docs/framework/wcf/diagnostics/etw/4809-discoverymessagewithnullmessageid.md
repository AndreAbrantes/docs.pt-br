# <a name="4809---discoverymessagewithnullmessageid"></a>4809 - DiscoveryMessageWithNullMessageId
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4809|  
|Palavras-chave|Descoberta|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem de descoberta foi descartada porque o cabeçalho da mensagem não contém a propriedade MessageId necessária.  
  
## <a name="message"></a>Mensagem  
 Uma mensagem de %1 foi descartada porque o cabeçalho da mensagem não contém a propriedade MessageId necessária.

|||  
|-|-|  
|ID|3823|  
|Palavras-chave|RoutingServices|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando o serviço de roteamento é enviar de volta uma mensagem de resposta de falha.  
  
## <a name="message"></a>Mensagem  
 O serviço de roteamento é enviar de volta uma mensagem de resposta de falha com ação '%1'.

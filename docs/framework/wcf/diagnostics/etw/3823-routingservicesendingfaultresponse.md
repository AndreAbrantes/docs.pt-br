## <a name="description"></a>Descrição  
 Esse evento é emitido quando o serviço de roteamento está retornando uma mensagem de resposta de falha.  
  
## <a name="message"></a>Mensagem  
 O serviço de roteamento está retornando uma mensagem de resposta Fault com Action '%1'.

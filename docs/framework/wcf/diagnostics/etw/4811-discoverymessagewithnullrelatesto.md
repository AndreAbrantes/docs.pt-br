# <a name="4811---discoverymessagewithnullrelatesto"></a>4811 - DiscoveryMessageWithNullRelatesTo
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4811|  
|Palavras-chave|Descoberta|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem de descoberta foi descartada pelo DiscoveryClient porque o cabeçalho da mensagem não contém a propriedade RelatesTo necessária.  
  
## <a name="message"></a>Mensagem  
 Uma mensagem %1 com messageId = '%2' foi descartado pelo DiscoveryClient porque o cabeçalho da mensagem não contém a propriedade RelatesTo necessária.

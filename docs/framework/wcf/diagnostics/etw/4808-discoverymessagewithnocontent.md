## <a name="description"></a>Descrição  
 Esse evento é emitido quando a mensagem de descoberta foi descartada porque ela tinha um endereço ReplyTo inválido.  
  
## <a name="message"></a>Mensagem  
 Uma mensagem %1 foi descartada porque ela não tinha nenhum conteúdo.

# <a name="4815---endpointdiscoverabilityenabled"></a>4815 - EndpointDiscoverabilityEnabled
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4815|  
|Palavras-chave|Descoberta|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando a descoberta do ponto de extremidade foi habilitada.  
  
## <a name="message"></a>Mensagem  
 A capacidade de descoberta do ponto de extremidade com EndpointAddress = '%1' e ListenUri = '%2' foi ativado.

 <span data-ttu-id="1e654-107">O exemplo a seguir mostra como habilitar o registro em log e especificar opções adicionais.</span><span class="sxs-lookup"><span data-stu-id="1e654-107">The following example shows how to enable logging and specify additional options.</span></span>  
  
```xml

  <sources>  
    <source name="System.ServiceModel.MessageLogging">  
      <listeners>  
         <add name="messages"  
              type="System.Diagnostics.XmlWriterTraceListener"  
              initializeData="c:\logs\messages.svclog" />  
        </listeners>  
    </source>  
  </sources>

  <diagnostics>

         logMalformedMessages="false"  
         logMessagesAtServiceLevel="true"

         maxMessagesToLog="3000"  
         maxSizeOfMessageToLog="2000"/>  
  </diagnostics>  
</system.serviceModel>  
```

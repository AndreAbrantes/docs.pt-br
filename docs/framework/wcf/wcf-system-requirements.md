# <a name="wcf-system-requirements"></a>Requisitos do sistema do WCF

O Windows Communication Foundation (WCF) é uma infraestrutura de comunicação usada para criar aplicativos distribuídos. O a seguir lista os requisitos que permitem que o WCF seja executado.

## <a name="system-requirements"></a>Requisitos de sistema

O WCF é instalado por padrão no [!INCLUDE[wv](../../../includes/wv-md.md)].

O WCF também pode ser instalado em [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], Windows Server 2003 R2 ou [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> A funcionalidade do MSMQ (enfileiramento de mensagens) do WCF tem suporte apenas em [!INCLUDE[wv](../../../includes/wv-md.md)], Windows Server 2003 R2, [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] e Windows XP Professional.

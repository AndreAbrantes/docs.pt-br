# <a name="windows-communication-foundation-tools"></a>Ferramentas do Windows Communication Foundation
Microsoft [!INCLUDE[indigo1](../../../includes/indigo1-md.md)] ferramentas foram projetadas para tornar mais fácil para você criar, implantar e gerenciar [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] aplicativos. Esta seção contém informações detalhadas sobre as ferramentas. Observe que as ferramentas não são suportadas.  
  
 Você pode executar todas as ferramentas da linha de comando.  
  
 A tabela a seguir lista essas ferramentas e fornece uma breve descrição.  
  
|Ferramenta|Descrição|  
|----------|-----------------|  
|[ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md) [Ferramenta de utilitário de metadados ServiceModel (Svcutil.exe)]|Gera o código de modelo de serviço de documentos de metadados e metadados no modelo de código de serviço.|  
|[Find Private Key Tool (FindPrivateKey.exe)](../../../docs/framework/wcf/find-private-key-tool-findprivatekey-exe.md) [Ferramenta par encontrar chave privada (FindPrivateKey.exe)]|Recupera a chave privada de uma loja especificada.|  
|[ServiceModel Registration Tool (ServiceModelReg.exe)](../../../docs/framework/wcf/servicemodelreg-exe.md) [Ferramenta de registro de ServiceModel (ServiceModelReg.exe)]|Gerencia o registro e cancelamento de registro de ServiceModel está em um único computador.|  
|[COM+ Service Model Configuration Tool (ComSvcConfig.exe)](../../../docs/framework/wcf/com-service-model-configuration-tool-comsvcconfig-exe.md) [Ferramenta de configuração de modelo de serviço COM+ (ComSvcConfig.exe)]|Configura interfaces COM+ deve ser exposta como serviços da Web.|  
|[Configuration Editor Tool (SvcConfigEditor.exe)](../../../docs/framework/wcf/configuration-editor-tool-svcconfigeditor-exe.md) (Ferramenta Editor de configuração (SvcConfigEditor.exe))|Cria e modifica as definições de configuração para serviços WCF.|  
|[Service Trace Viewer Tool (SvcTraceViewer.exe)](../../../docs/framework/wcf/service-trace-viewer-tool-svctraceviewer-exe.md) (Ferramenta Visualizador de rastreamento de serviço (SvcTraceViewer.exe))|Ajuda a exibição, o grupo e o rastreamento de filtro mensagens para que você pode diagnosticar, reparar e verificar os problemas com os serviços do WCF.|  
|[WS-AtomicTransaction Configuration Utility (wsatConfig.exe)](../../../docs/framework/wcf/ws-atomictransaction-configuration-utility-wsatconfig-exe.md) [Utilitário de configuração de WS-AtomicTransaction (wsatConfig.exe)]|Define configurações de suporte básico do WS-AtomicTransaction usando uma ferramenta de linha de comando.|  
|[WS-AtomicTransaction Configuration MMC Snap-in](../../../docs/framework/wcf/ws-atomictransaction-configuration-mmc-snap-in.md) (Snap-in do MMC de configuração de WS-AtomicTransaction)|Configura definições de suporte de WS-AtomicTransaction básicas usando um snap-in do MMC.|  
|[WorkFlow Service Registration Tool (WFServicesReg.exe)](../../../docs/framework/wcf/workflow-service-registration-tool-wfservicesreg-exe.md) [Ferramenta de registro de serviço de fluxo de trabalho (WFServicesReg.exe)]|Registra um serviço de fluxo de trabalho do Windows.|  
|[WCF Service Host (WcfSvcHost.exe)](../../../docs/framework/wcf/wcf-service-host-wcfsvchost-exe.md) [Host de serviço do WCF (WcfSvcHost.exe)]|Hosts [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] serviços contidos nos arquivos de bibliotecas (*. dll)|  
|[WCF Test Client (WcfTestClient.exe)](../../../docs/framework/wcf/wcf-test-client-wcftestclient-exe.md) [Cliente de teste do WCF (WcfTestClient.exe)]|Uma ferramenta de interface gráfica do usuário que permite que os parâmetros de entrada de tipos arbitrários, envio de entrada para o serviço e o modo de exibição envia a resposta do serviço.|  
|[Contract-First Tool](../../../docs/framework/wcf/contract-first-tool.md) (Ferramenta de primeiro contrato)|Tarefa que cria classes de código de contratos de dados XSD de compilação de um Visual Studio.|  
  
 Todas as ferramentas anteriores exceto ServiceModelReg.exe, WsatConfig.exe e ComSvcConfig.exe fornecidas com o SDK do Windows e podem ser encontradas na pasta C:\Program Files\Microsoft SDKs\Windows\v6.0\Bin.  As ferramentas específicas de 3 podem ser encontradas em C:\Windows\Microsoft.NET\Framework\v3.0\Windows Communication Foundation.

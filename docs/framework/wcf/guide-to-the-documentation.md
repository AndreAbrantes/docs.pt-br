# <a name="guide-to-the-documentation"></a>Guia para a documentação
Desde que este é um guia sobre o [!INCLUDE[indigo1](../../../includes/indigo1-md.md)] documentação. Documentos associados são pontos de partida recomendados agrupados de acordo com interesses e níveis específicos de experiência.  
  
 Para instalar uma versão autônoma de um visualizador de Ajuda e a documentação, baixe o [v do Microsoft Windows SDK 7.1](http://go.microsoft.com/fwlink/?LinkID=194146&clcid=0x409).  
  
## <a name="new-to-windows-communication-foundation-programming"></a>Novo para programação do Windows Communication Foundation  
  
-   Se você for novo para programação com [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] e você quiser apenas ver aplicativos de exemplo que funcionam, consulte os tópicos listados em [exemplos do Windows Communication Foundation](../../../docs/framework/wcf/samples/index.md).  
  
-   Para obter um tutorial que percorra as etapas básicas de criação de um [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] serviço e cliente, consulte [Tutorial de Introdução](../../../docs/framework/wcf/getting-started-tutorial.md).  
  
-   Se você estiver interessado em conceitos por trás de [!INCLUDE[indigo2](../../../includes/indigo2-md.md)], consulte os tópicos de [visão geral conceitual](../../../docs/framework/wcf/conceptual-overview.md) seção.  
  
-   Para ver exemplos de gráficos de configurações de segurança de cliente/servidor, consulte [cenários comuns de segurança](../../../docs/framework/wcf/feature-details/common-security-scenarios.md).

-   Se você estiver procurando orientação sobre um determinado recurso ou funcionalidade de [!INCLUDE[indigo2](../../../includes/indigo2-md.md)], consulte os tópicos em [detalhes do recurso WCF](../../../docs/framework/wcf/feature-details/index.md).

-   Para obter informações sobre as ferramentas que ajudam a criar e depurar [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] aplicativos, consulte [ferramentas do Windows Communication Foundation](../../../docs/framework/wcf/tools.md).

## <a name="using-windows-communication-foundation-with-other-technologies"></a>O Windows Communication Foundation com outras tecnologias  
  
-   Para criar um serviço que se comunica com clientes do ASP.NET, consulte [como: configurar o serviço do WCF para interoperar com clientes de serviço Web ASP.NET](../../../docs/framework/wcf/feature-details/config-wcf-service-with-aspnet-web-service.md).  
  
-   Integração com [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] remoting é explicado em [Migrando aplicativos de comunicação remota do .NET para WCF](../../../docs/framework/wcf/feature-details/migrating-net-remoting-applications-to-wcf.md).  
  
-   Para integrar um aplicativo COM+ existente com um [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] serviço ou cliente, consulte [integração com visão geral de aplicativos COM+](../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md).  
  
-   Para integrar um aplicativo COM existente com um [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] serviço ou cliente, consulte [integrando aplicativos COM](../../../docs/framework/wcf/feature-details/integrating-with-com-applications.md).  
  
-   Para integrar um aplicativo de MSMQ existente com um [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] serviço ou cliente, consulte [como: mensagens na fila do Exchange com pontos de extremidade do WCF](../../../docs/framework/wcf/feature-details/how-to-exchange-queued-messages-with-wcf-endpoints.md) e [como: com pontos de extremidade do WCF e enfileiramento de mensagens do Exchange Aplicativos](../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).  
  
-   Para usar os serviços de informações da Internet (IIS) para hospedar um serviço, consulte [serviços de hospedagem](../../../docs/framework/wcf/hosting-services.md).  
  
-   Para usar [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] para consumir um serviço extensões WSE (Web Services) 3.0, consulte [como: acessar um serviço do WSE 3.0](../../../docs/framework/wcf/feature-details/how-to-access-a-wse-3-0-service-with-a-wcf-client.md).

 [Provedor de Token emitido durável](../../../../docs/framework/wcf/samples/durable-issued-token-provider.md)

 [Provedor de Token SAML](../../../../docs/framework/wcf/samples/saml-token-provider.md)

 [Tokens de suporte](../../../../docs/framework/wcf/samples/supporting-tokens.md)

 [Provedor de token](../../../../docs/framework/wcf/samples/token-provider.md)

 [Validador de senha do nome de usuário](../../../../docs/framework/wcf/samples/user-name-password-validator.md)

 [Validador de certificado X.509](../../../../docs/framework/wcf/samples/x-509-certificate-validator.md)

 [Validação do cliente](../../../../docs/framework/wcf/samples/client-validation.md)

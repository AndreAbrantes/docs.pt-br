# <a name="ajax"></a>AJAX
Esta seção inclui exemplos que demonstram a funcionalidade de Asynchronous JavaScript and XML.  
  
## <a name="in-this-section"></a>Nesta seção  
 [JSONP](../../../../docs/framework/wcf/samples/jsonp.md)  
 Demonstra como dar suporte a JSON com preenchimento (JSONP) nos serviços REST do WCF.  
  
 [Serialização JSON](../../../../docs/framework/wcf/samples/json-serialization.md)  
 Demonstra como usar o <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> para serializar e desserializar dados no formato de notação JSON (JavaScript Object).  
  
 [Serviço AJAX básico](../../../../docs/framework/wcf/samples/basic-ajax-service.md)  
 Demonstra como usar o WCF para criar um serviço básico do ASP.NET Asynchronous JavaScript and XML (AJAX).  
  
 [Serviço AJAX utilizando HTTP POST](../../../../docs/framework/wcf/samples/ajax-service-using-http-post.md)  
 Demonstra como usar o WCF para criar um serviço ASP.NET Asynchronous JavaScript and XML (AJAX) que usa o HTTP POST.  
  
 [Serviço AJAX sem configuração](../../../../docs/framework/wcf/samples/ajax-service-without-configuration.md)  
 Demonstra como usar o WCF para criar um serviço básico do ASP.NET Asynchronous JavaScript and XML (AJAX).  
  
 [Serviço AJAX utilizando tipos complexos](../../../../docs/framework/wcf/samples/ajax-service-using-complex-types-sample.md)  
 Demonstra como usar o WCF para criar um serviço ASP.NET Asynchronous JavaScript and XML (AJAX) que cria instâncias de tipos complexos.  
  
 [Serviço AJAX com JSON e XML](../../../../docs/framework/wcf/samples/ajax-service-with-json-and-xml-sample.md)  
 Demonstra como usar o WCF para criar um serviço de Asynchronous JavaScript and XML (AJAX) que retorna dados de objeto notação JSON (JavaScript) ou XML.

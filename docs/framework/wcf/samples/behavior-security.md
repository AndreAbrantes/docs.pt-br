# <a name="behavior-security"></a>Segurança de comportamento
Esta seção inclui exemplos que demonstram como configurar segurança de comportamentos de serviço.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Comportamento de auditoria de serviço](../../../../docs/framework/wcf/samples/service-auditing-behavior.md)  
 Este exemplo demonstra como usar o <xref:System.ServiceModel.Description.ServiceSecurityAuditBehavior> para habilitar a auditoria de eventos de segurança durante operações de serviço.  
  
 [Provedor de função e associação](../../../../docs/framework/wcf/samples/membership-and-role-provider.md)  
 Este exemplo demonstra como um serviço pode usar o [!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)] provedores de associação e funções para autenticar e autorizar clientes.  
  
 [Autorizando o acesso a operações de serviço](../../../../docs/framework/wcf/samples/authorizing-access-to-service-operations.md)  
 Este exemplo demonstra como usar o [ \<serviceAuthorization >](../../../../docs/framework/configure-apps/file-schema/wcf/serviceauthorization-element.md) para habilitar o uso do <xref:System.Security.Permissions.PrincipalPermissionAttribute> atributo para autorizar o acesso a operações de serviço.  
  
 [Representar o cliente](../../../../docs/framework/wcf/samples/impersonating-the-client.md)  
 Este exemplo demonstra como representar o aplicativo chamador no serviço de forma que o serviço pode acessar recursos do sistema em nome do chamador.

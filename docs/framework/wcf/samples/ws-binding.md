# <a name="ws-binding"></a>Associação de WS
Esta seção contém exemplos que demonstram a associação de serviço do Windows no Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Fluxo de transação WS](../../../../docs/framework/wcf/samples/ws-transaction-flow.md)  
 Demonstra o uso de uma transação de coordenadas de cliente e as opções de cliente e servidor para o fluxo de transações usando o protocolo WS-AT ou OleTransactions.  
  
 [Associação de HTTP de Federação do WS 2007](../../../../docs/framework/wcf/samples/ws-2007-federation-http-binding.md)  
 Demonstra o uso de <xref:System.ServiceModel.WS2007FederationHttpBinding>.  
  
 [Http duplo de WS](../../../../docs/framework/wcf/samples/ws-dual-http.md)  
 Demonstra como configurar o `WSDualHttpBinding` associação.  
  
 [Codificação de MTOM](../../../../docs/framework/wcf/samples/mtom-encoding.md)  
 Demonstra o uso da mensagem MTOM Message Transmission Optimization Mechanism () codificação com WSHttpBinding.  
  
 [WSHttpBinding](../../../../docs/framework/wcf/samples/wshttpbinding.md)  
 Demonstra como implementar um serviço típico e um cliente típico usando o WCF.  
  
 [Sessão confiável de WS](../../../../docs/framework/wcf/samples/ws-reliable-session.md)  
 Demonstra o uso de sessões confiáveis.  
  
 [Segurança de transporte de WS](../../../../docs/framework/wcf/samples/ws-transport-security.md)  
 Demonstra o uso da segurança de transporte SSL com o <xref:System.ServiceModel.WSHttpBinding> associação.  
  
 [Associação de segurança de mensagem](../../../../docs/framework/wcf/samples/message-security-binding.md)  
 Demonstra como usar a segurança de mensagem no WCF.  
  
 [Transporte de WS com credencial de mensagem](../../../../docs/framework/wcf/samples/ws-transport-with-message-credential.md)  
 Este exemplo demonstra o uso da segurança de transporte do SSL em combinação com a credencial do cliente que está sendo executada na mensagem.

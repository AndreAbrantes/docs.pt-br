## <a name="discussion"></a><span data-ttu-id="57def-108">Discussão</span><span class="sxs-lookup"><span data-stu-id="57def-108">Discussion</span></span>  
 <span data-ttu-id="57def-109">Este exemplo consiste em dois componentes: um projeto de aplicativo Web (serviço) que contém um serviço WCF e um aplicativo de console (cliente) que invoca operações de serviço usando associações SOAP e HTTP.</span><span class="sxs-lookup"><span data-stu-id="57def-109">This sample consists of two components: a Web Application project (Service) that contains a WCF service and a console application (Client) that invokes service operations using SOAP and HTTP bindings.</span></span>  
  
 <span data-ttu-id="57def-110">O serviço WCF expõe 2 operações –`GetData` e `PutData` – que ecoam a cadeia de caracteres que foi passada como entrada.</span><span class="sxs-lookup"><span data-stu-id="57def-110">The WCF service exposes 2 operations –`GetData` and `PutData` – that echo the string that was passed as input.</span></span> <span data-ttu-id="57def-111">As operações de serviço são anotadas com <xref:System.ServiceModel.Web.WebGetAttribute> e <xref:System.ServiceModel.Web.WebInvokeAttribute>.</span><span class="sxs-lookup"><span data-stu-id="57def-111">The service operations are annotated with <xref:System.ServiceModel.Web.WebGetAttribute> and <xref:System.ServiceModel.Web.WebInvokeAttribute>.</span></span> <span data-ttu-id="57def-112">Esses atributos controlam a projeção HTTP dessas operações.</span><span class="sxs-lookup"><span data-stu-id="57def-112">These attributes control the HTTP projection of these operations.</span></span> <span data-ttu-id="57def-113">Além disso, eles são anotados com <xref:System.ServiceModel.OperationContractAttribute>, o que permite que eles sejam expostos em associações SOAP.</span><span class="sxs-lookup"><span data-stu-id="57def-113">In addition, they are annotated with <xref:System.ServiceModel.OperationContractAttribute>, which enables them to be exposed over SOAP bindings.</span></span> <span data-ttu-id="57def-114">O método de `PutData` do serviço gera uma <xref:System.ServiceModel.Web.WebFaultException>, que é enviada de volta por HTTP usando o código de status HTTP e é enviada de volta por SOAP como uma falha de SOAP.</span><span class="sxs-lookup"><span data-stu-id="57def-114">The service’s `PutData` method throws a <xref:System.ServiceModel.Web.WebFaultException>, which gets sent back over HTTP using the HTTP status code and gets sent back over SOAP as a SOAP fault.</span></span>  
  
 <span data-ttu-id="57def-115">O arquivo Web. config configura o serviço WCF com 3 pontos de extremidade:</span><span class="sxs-lookup"><span data-stu-id="57def-115">The Web.config file configures the WCF service with 3 endpoints:</span></span>

- <span data-ttu-id="57def-118">O ponto de extremidade ~/Service.svc/SOAP que permite que os clientes acessem o serviço usando a associação SOAP sobre HTTP.</span><span class="sxs-lookup"><span data-stu-id="57def-118">The ~/service.svc/soap endpoint that allows the clients to access the service using the SOAP over HTTP binding.</span></span>

 <span data-ttu-id="57def-121">O projeto cliente demonstra o acesso ao serviço usando um proxy SOAP (gerado por meio de **Adicionar referência de serviço**) e o acesso ao serviço usando <xref:System.Net.WebClient>.</span><span class="sxs-lookup"><span data-stu-id="57def-121">The client project demonstrates accessing the service using a SOAP proxy (generated through **Add Service Reference**) and accessing the service using <xref:System.Net.WebClient>.</span></span>  
  
 <span data-ttu-id="57def-122">O exemplo consiste em um serviço hospedado na Web e um aplicativo de console.</span><span class="sxs-lookup"><span data-stu-id="57def-122">The sample consists of a Web-hosted service and a console application.</span></span> <span data-ttu-id="57def-123">À medida que o aplicativo de console é executado, o cliente faz solicitações ao serviço e grava as informações pertinentes das respostas na janela do console.</span><span class="sxs-lookup"><span data-stu-id="57def-123">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>

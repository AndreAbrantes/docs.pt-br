## <a name="in-this-section"></a>Nesta seção  
 [Duplex](../../../../docs/framework/wcf/samples/duplex.md)  
 Demonstra como definir e implementar um contrato duplex.  
  
 [Contrato de falha](../../../../docs/framework/wcf/samples/fault-contract.md)

 [Unidirecional](../../../../docs/framework/wcf/samples/one-way.md)

 [Sessão](../../../../docs/framework/wcf/samples/session.md)

 [Fluxo](../../../../docs/framework/wcf/samples/stream.md)

 [Falhas de XmlSerializer](../../../../docs/framework/wcf/samples/xmlserializer-faults.md)

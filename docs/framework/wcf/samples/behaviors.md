# <a name="behaviors"></a>Comportamentos
Esta seção contém exemplos que demonstram os comportamentos de serviço do Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Simultaneidade](../../../../docs/framework/wcf/samples/concurrency.md)  
 Demonstra como usar o <xref:System.ServiceModel.ServiceBehaviorAttribute> com o <xref:System.ServiceModel.ConcurrencyMode> enumeração, que controla se uma instância de um serviço processa mensagens consecutivamente ou simultaneamente.  
  
 [Comportamento de serviço padrão](../../../../docs/framework/wcf/samples/default-service-behavior.md)  
 Demonstra como configurações de comportamento de serviço podem ser configuradas.  
  
 [Instanciação](../../../../docs/framework/wcf/samples/instancing.md)  
 Demonstra a configuração de comportamento de instância, que controla como as instâncias de uma classe de serviço são criadas em resposta a solicitações de cliente.  
  
 [Comportamento de publicação de metadados](../../../../docs/framework/wcf/samples/metadata-publishing-behavior.md)  
 Demonstra como controlar os recursos de um serviço de publicação de metadados.  
  
 [Comportamento de transação de serviço](../../../../docs/framework/wcf/samples/service-transaction-behavior.md)  
 Demonstra o uso de uma transação de coordenada de cliente e as configurações de <xref:System.ServiceModel.ServiceBehaviorAttribute> e <xref:System.ServiceModel.OperationBehaviorAttribute> para controlar o comportamento do serviço transações.  
  
 [Comportamento de depuração de serviço](../../../../docs/framework/wcf/samples/service-debug-behavior.md)  
 Demonstra como as configurações de comportamento de serviço podem ser configuradas.  
  
 [Limitação](../../../../docs/framework/wcf/samples/throttling.md)  
 Demonstra o uso de controles de limitação.  
  
 [Segurança de comportamento](../../../../docs/framework/wcf/samples/behavior-security.md)  
 Demonstra como configurar segurança de comportamento de serviço.

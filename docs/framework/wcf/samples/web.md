# <a name="web"></a>Web
Esta seção contém exemplos hospedado na Web.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Seleção de formato avançada](../../../../docs/framework/wcf/samples/advanced-format-selection.md)  
 Demonstra como estender o modelo de programação WCF Web HTTP para dar suporte a novos formatos de resposta de saída.  
  
 [Seleção automática de formato](../../../../docs/framework/wcf/samples/automatic-format-selection.md)  
 Demonstra como habilitar a seleção automática de formato (XML ou JSON) com o WCF programação HTTP Web do modelo, bem como definir o formato explicitamente no código da operação.  
  
 [Serviço HTTP básico](../../../../docs/framework/wcf/samples/basic-http-service.md)  
 Demonstra como implementar um serviço baseado em HTTP, baseados em RPC - popularmente conhecido como serviço "POX" (Plain Old XML) – usando o modelo de programação WCF Web HTTP.  
  
 [Serviço de recursos básicos](../../../../docs/framework/wcf/samples/basic-resource-service.md)  
 Demonstra como implementar um serviço baseado em HTTP, usando o modelo de programação HTTP Web do WCF que expõe uma coleção de clientes que dá suporte a recuperar, adicionar, excluir e substituir operações.  
  
 [AspNetRouteIntegration](../../../../docs/framework/wcf/samples/aspnetrouteintegration.md)  
 Demonstra como hospedar um serviço WCF Web HTTP usando ASP.NET rotas.  
  
 [Get e Put condicionais](../../../../docs/framework/wcf/samples/conditional-get-and-put.md)  
 Demonstra como usar o novo recuperar condicional e APIs de atualização para o modelo de programação WCF Web HTTP.  
  
 [SOAP e pontos de extremidade HTTP](../../../../docs/framework/wcf/samples/soap-and-http-endpoints.md)  
 Demonstra como implementar um serviço baseado no RPC e expô-lo no formato SOAP e o formato de "Plain Old XML" (POX) usando o modelo de programação WCF Web HTTP.  
  
 [Integração de cache ASP.NET](../../../../docs/framework/wcf/samples/aspnet-caching-integration.md)  
 Demonstra como utilizar o cache de saída do ASP.NET com o modelo de programação WCF Web HTTP.  
  
 [UriTemplate](../../../../docs/framework/wcf/samples/uritemplate-sample.md)  
 Demonstra os principais conceitos relacionados a <xref:System.UriTemplate>.  
  
 [Tabela de UriTemplate](../../../../docs/framework/wcf/samples/uritemplate-table-sample.md)  
 Demonstra como o <xref:System.UriTemplateTable> classe fornece uma estrutura de tabela associativa dicionário semelhante para trabalhar com um conjunto de <xref:System.UriTemplate> instâncias.  
  
 [Dispatcher de Tabela de UriTemplate](../../../../docs/framework/wcf/samples/uritemplate-table-dispatcher-sample.md)  
 Demonstra um mecanismo básico de expedição criado usando <xref:System.UriTemplateTable>, um cenário de uso comum para o <xref:System.UriTemplateTable> classe.

Esta seção contém exemplos que demonstram a interação entre o Windows Communication Foundation (WCF) e outras tecnologias de serviço.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Usando o WCF Moniker com clientes COM](../../../../docs/framework/wcf/samples/using-the-wcf-moniker-with-com-clients.md)  
 Demonstra como usar o [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] moniker de serviço para integrar serviços Web em ambientes de desenvolvimento baseado em COM, como o Microsoft Office Visual Basic for Applications (VBA Office) ou Visual Basic 6.0.  
  
 [Cliente ASMX com um serviço WCF](../../../../docs/framework/wcf/samples/asmx-client-with-a-wcf-service.md)  
 Demonstra como criar um serviço usando [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] e, em seguida, acessar o serviço de não[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] cliente, como um cliente ASMX.

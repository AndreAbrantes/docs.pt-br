# <a name="routing-services"></a>Serviços de roteamento
Esta seção inclui exemplos que demonstram como usar o roteador do Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Olá, Mundo com o serviço de roteamento](../../../../docs/framework/wcf/samples/hello-world-with-the-routing-service.md)  
 Demonstra como usar o roteador de WCF.  
  
 [Ponte e o tratamento de erro](../../../../docs/framework/wcf/samples/bridging-and-error-handling.md)  
 Demonstra como o serviço de roteamento do WCF é usado para a ponte de comunicação entre um cliente e um serviço que usam ligações diferentes.  
  
 [Filtros avançados](../../../../docs/framework/wcf/samples/advanced-filters.md)  
 Demonstra como adaptar o exemplo de cálculo de WCF padrão para se comunicar usando o serviço de roteamento.  
  
 [Reconfiguração dinâmica](../../../../docs/framework/wcf/samples/dynamic-reconfiguration.md)  
 Demonstra como o serviço de roteamento pode ser reconfigurado dinamicamente em tempo de execução.  
  
 [Tratamento avançado de erro](../../../../docs/framework/wcf/samples/advanced-error-handling.md)  
 Demonstra como o serviço de roteamento de maneira inteligente recupera de erros, usando transações e outros conceitos mensagens mais complexos, como multicast.

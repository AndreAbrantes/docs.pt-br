 <span data-ttu-id="3a9a5-116">A página da Web pode chamar o serviço REST do WCF porque o serviço está usando o <xref:System.ServiceModel.Description.WebScriptEndpoint> com `crossDomainScriptAccessEnabled` definido como `true` .</span><span class="sxs-lookup"><span data-stu-id="3a9a5-116">The Web page can call the WCF REST service because the service is using the <xref:System.ServiceModel.Description.WebScriptEndpoint> with `crossDomainScriptAccessEnabled` set to `true`.</span></span> <span data-ttu-id="3a9a5-117">Essas duas configurações são feitas no arquivo Web.config sob o \<system.serviceModel> elemento.</span><span class="sxs-lookup"><span data-stu-id="3a9a5-117">Both of these configurations are done in the Web.config file under the \<system.serviceModel> element.</span></span>

  </standardEndpoints>

 <span data-ttu-id="3a9a5-118">O ScriptManager gerencia a interação com o serviço e oculta a complexidade de implementar manualmente o acesso JSONP.</span><span class="sxs-lookup"><span data-stu-id="3a9a5-118">ScriptManager manages the interaction with the service and hides away the complexity of manually implementing JSONP access.</span></span> <span data-ttu-id="3a9a5-119">Quando `crossDomainScriptAccessEnabled` é definido como `true` e o formato de resposta para uma operação é JSON, a infraestrutura do WCF INSPECIONA o URI da solicitação de um parâmetro de cadeia de caracteres de consulta de retorno de chamada e encapsula a resposta JSON com o valor do parâmetro de cadeia de caracteres de consulta de retorno de chamada.</span><span class="sxs-lookup"><span data-stu-id="3a9a5-119">When `crossDomainScriptAccessEnabled` is set to `true` and the response format for an operation is JSON, the WCF infrastructure inspects the URI of the request for a callback query string parameter and wraps the JSON response with the value of the callback query string parameter.</span></span> <span data-ttu-id="3a9a5-120">No exemplo, a página da Web chama o serviço REST do WCF com o URI a seguir.</span><span class="sxs-lookup"><span data-stu-id="3a9a5-120">In the sample, the Web page calls the WCF REST service with the following URI.</span></span>

 <span data-ttu-id="3a9a5-121">Como o parâmetro de cadeia de caracteres de consulta de retorno de chamada tem um valor de `JsonPCallback` , o serviço WCF retorna uma resposta JSONP mostrada no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="3a9a5-121">Because the callback query string parameter has a value of `JsonPCallback`, the WCF service returns a JSONP response shown in the following example.</span></span>

# <a name="client"></a>Cliente
Esta seção contém exemplos que demonstram os aplicativos de cliente do Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Interoperabilidade do cliente](../../../../docs/framework/wcf/samples/client-interoperability.md)  
 Demonstra a interoperabilidade entre o WCF e outras tecnologias.  
  
 [Cabeçalhos de endereço](../../../../docs/framework/wcf/samples/address-headers.md)  
 Demonstra como os clientes podem passar parâmetros de referência a um serviço usando o WCF.  
  
 [Fábrica de canais](../../../../docs/framework/wcf/samples/channel-factory.md)  
 Demonstra como um aplicativo cliente pode criar um canal com o <xref:System.ServiceModel.ChannelFactory> classe em vez de um cliente gerado.  
  
 [Exceções esperadas](../../../../docs/framework/wcf/samples/expected-exceptions.md)  
 Demonstra como capturar exceções esperadas, ao usar um cliente com tipo.  
  
 [Recuperar metadados](../../../../docs/framework/wcf/samples/retrieve-metadata.md)  
 Demonstra como implementar um cliente que recupera os metadados de um serviço para escolher a comunicação com um ponto de extremidade dinamicamente.  
  
 [Use fechar e anular para liberar recursos de cliente do WCF](../../../../docs/framework/wcf/samples/use-close-abort-release-wcf-client-resources.md)  
 Demonstra como você não deve usar o C# "using" de instrução para limpar automaticamente os recursos ao usar um cliente com tipo.  
  
 [Cliente tipado](../../../../docs/framework/wcf/samples/typed-client.md)  
 Demonstra como obter informações de um cliente com tipo gerado pelo [ferramenta de utilitário de metadados ServiceModel (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).

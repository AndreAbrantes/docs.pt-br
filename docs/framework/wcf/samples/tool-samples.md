# <a name="tool-samples"></a>Exemplos de ferramentas
Esta seção contém as ferramentas do Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [ConfigurationCodeGenerator](../../../../docs/framework/wcf/samples/configurationcodegenerator.md)  
 Descreve como usar a ferramenta ConfigurationCodeGenerator.exe para expor as implementações de canal personalizado para o sistema de configuração.  
  
 [CustomChannelsTester](../../../../docs/framework/wcf/samples/customchannelstester.md)  
 Descreve como usar a ferramenta CustomChannelsTester.exe para testar suas implementações de canal personalizado com um conjunto de contratos de serviço predefinidos.  
  
 [FindPrivateKey](../../../../docs/framework/wcf/samples/findprivatekey.md)  
 Descreve como usar a ferramenta FindPrivateKey.exe para facilitar o processo de encontrar o local e o nome do arquivo da chave privado associado com um certificado x. 509 específico no repositório de certificados.

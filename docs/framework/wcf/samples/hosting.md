# <a name="hosting"></a>Hospedagem
Esta seção contém exemplos que demonstram os serviços de hospedagem de Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Ativação de Processo do Windows](../../../../docs/framework/wcf/samples/windows-process-activation.md)  
 Demonstra a ativação do serviço por meio de protocolos de rede.  
  
 [Ativação baseada em configuração](../../../../docs/framework/wcf/samples/configuration-based-activation.md)  
 Demonstra a ativação do [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] serviços sem a necessidade de um arquivo. svc.  
  
 [SystemWebRouting Integration Sample](../../../../docs/framework/wcf/samples/systemwebrouting-integration-sample.md)  
 Demonstra a integração da camada de hospedagem com as classes de <xref:System.Web.Routing> namespace.  
  
 [Compatibilidade de ASP.NET](../../../../docs/framework/wcf/samples/aspnet-compatibility.md)  
 Demonstra como habilitar [!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)] modo de compatibilidade em [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].  
  
 [Hospedagem do IIS utilizando código embutido](../../../../docs/framework/wcf/samples/iis-hosting-using-inline-code.md)  
 Demonstra como implementar um serviço hospedado pelo Internet Information Services (IIS), em que o código de serviço está contido na linha em um arquivo. svc e compilado por demanda.  
  
 [Host de serviço Windows](../../../../docs/framework/wcf/samples/windows-service-host.md)  
 Demonstra um [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] serviço hospedado em um serviço Windows gerenciado.  
  
 [Auto-hospedagem](../../../../docs/framework/wcf/samples/self-host.md)  
 Demonstra como implementar um serviço hospedado automaticamente em um aplicativo de console.

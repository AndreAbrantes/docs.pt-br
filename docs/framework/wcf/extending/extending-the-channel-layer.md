# <a name="extending-the-channel-layer"></a>Estendendo a camada do canal
A camada de canais é responsável para a troca de mensagens entre clientes e serviços. Extensões de canal podem implementar a nova funcionalidade de protocolo, como segurança ou funcionalidade de transporte, como implementar um novo transporte de rede para transportar mensagens SOAP.

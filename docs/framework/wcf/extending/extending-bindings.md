# <a name="extending-bindings"></a>Estendendo associações
Associações de especificam o transporte, codificação e protocolo necessárias para se conectar a um ponto de extremidade. As extensões de associação e associações personalizadas implementam a funcionalidade de comunicação personalizados necessária para dar suporte a recursos do aplicativo.

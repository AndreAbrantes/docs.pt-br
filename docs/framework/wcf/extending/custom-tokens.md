# <a name="custom-tokens"></a>Tokens personalizados
Embora o Windows Communication Foundation (WCF) nativamente dá suporte a tokens de nome de usuário como credenciais, Tokens de contexto de segurança, os tíquetes Kerberos e certificados x. 509, é flexível o suficiente para permitir que você use suas próprias credenciais personalizadas.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Como: Criar um Token personalizado](../../../../docs/framework/wcf/extending/how-to-create-a-custom-token.md)  
 Descreve como criar um token de segurança personalizada usando o <xref:System.IdentityModel.Tokens.SecurityToken> classe e como integrá-lo com um provedor de token de segurança personalizada e o autenticador.

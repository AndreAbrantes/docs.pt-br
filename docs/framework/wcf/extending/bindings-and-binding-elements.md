 <span data-ttu-id="a7bff-126">Associações e elementos de associação são a conexão entre o modelo de programação de aplicativo, que inclui os atributos e comportamentos, e o modelo de canal, que inclui fábricas e ouvintes, codificadores de mensagem e transporte e protocolo implementações.</span><span class="sxs-lookup"><span data-stu-id="a7bff-126">Bindings and binding elements are the connection between the application programming model, which includes the attributes and behaviors, and the channel model, which includes the factories and listeners, message encoders, and transport and protocol implementations.</span></span> <span data-ttu-id="a7bff-127">Normalmente, as associações e elementos de associação são implementadas para habilitar os canais a ser usado pela camada de aplicativo.</span><span class="sxs-lookup"><span data-stu-id="a7bff-127">Typically, binding elements and bindings are implemented to enable channels to be used by the application layer.</span></span>

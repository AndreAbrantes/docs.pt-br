 <span data-ttu-id="42f11-176">Listas grandes geralmente são virtualizadas dentro [!INCLUDE[TLA2#tla_ui](../../../includes/tla2sharptla-ui-md.md)] estruturas para ajudar no desempenho.</span><span class="sxs-lookup"><span data-stu-id="42f11-176">Large lists are often data virtualized within [!INCLUDE[TLA2#tla_ui](../../../includes/tla2sharptla-ui-md.md)] frameworks to assist in performance.</span></span> <span data-ttu-id="42f11-177">Devido a isso, um cliente de automação de interface do usuário não é possível usar o [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] recurso de consulta para extrair o conteúdo da árvore inteira da mesma forma que ele pode em outros contêineres de itens.</span><span class="sxs-lookup"><span data-stu-id="42f11-177">Due to this, a UI Automation client cannot use the [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] query feature to scrape the contents of the full tree in the same way that it can in other item containers.</span></span> <span data-ttu-id="42f11-178">Um cliente deve rolar o item na exibição (ou expanda o controle para mostrar todas as opções de valiosas) antes de acessar o conjunto completo de informações do item de dados.</span><span class="sxs-lookup"><span data-stu-id="42f11-178">A client should scroll the item into view (or expand the control to show all valuable options)prior to accessing the full set of information from the data item.</span></span>

## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="fecd6-110">Diretrizes e convenções de implementação</span><span class="sxs-lookup"><span data-stu-id="fecd6-110">Implementation Guidelines and Conventions</span></span>

  - <span data-ttu-id="fecd6-119">A ação requer interação do usuário.</span><span class="sxs-lookup"><span data-stu-id="fecd6-119">The action requires user interaction.</span></span>

  - <span data-ttu-id="fecd6-120">A ação é demorada e fará com que o cliente de chamada seja bloqueado por um período significativo.</span><span class="sxs-lookup"><span data-stu-id="fecd6-120">The action is time-consuming and will cause the calling client to block for a significant amount of time.</span></span>

- <span data-ttu-id="fecd6-121">Se invocar o controle tiver efeitos colaterais significativos, esses efeitos colaterais devem ser expostos por meio <xref:System.Windows.Automation.AutomationElement.AutomationElementInformation.HelpText%2A> da propriedade.</span><span class="sxs-lookup"><span data-stu-id="fecd6-121">If invoking the control has significant side-effects, those side-effects should be exposed through the <xref:System.Windows.Automation.AutomationElement.AutomationElementInformation.HelpText%2A> property.</span></span> <span data-ttu-id="fecd6-122">Por exemplo, embora <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> não esteja associado à seleção, <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> o pode fazer com que outro controle se torne selecionado.</span><span class="sxs-lookup"><span data-stu-id="fecd6-122">For example, even though <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> is not associated with selection, <xref:System.Windows.Automation.Provider.IInvokeProvider.Invoke%2A> may cause another control to become selected.</span></span>

> [!NOTE]

- <span data-ttu-id="fecd6-129">Um elemento pode desaparecer da [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] árvore imediatamente após ser invocado.</span><span class="sxs-lookup"><span data-stu-id="fecd6-129">An element can disappear from the [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] tree immediately upon being invoked.</span></span> <span data-ttu-id="fecd6-130">A solicitação de informações do elemento fornecido pelo retorno de chamada do evento pode falhar como resultado.</span><span class="sxs-lookup"><span data-stu-id="fecd6-130">Requesting information from the element provided by the event callback may fail as a result.</span></span> <span data-ttu-id="fecd6-131">A solução prévia de informações em cache é a alternativa recomendada.</span><span class="sxs-lookup"><span data-stu-id="fecd6-131">Pre-fetching cached information is the recommended workaround.</span></span>

- <span data-ttu-id="fecd6-132">Os controles podem implementar vários padrões de controle.</span><span class="sxs-lookup"><span data-stu-id="fecd6-132">Controls can implement multiple control patterns.</span></span> <span data-ttu-id="fecd6-133">Por exemplo, o controle cor de preenchimento na [!INCLUDE[TLA#tla_xl](../../../includes/tlasharptla-xl-md.md)] barra de ferramentas implementa <xref:System.Windows.Automation.InvokePattern> o e <xref:System.Windows.Automation.ExpandCollapsePattern> os padrões de controle.</span><span class="sxs-lookup"><span data-stu-id="fecd6-133">For example, the Fill Color control on the [!INCLUDE[TLA#tla_xl](../../../includes/tlasharptla-xl-md.md)] toolbar implements both the <xref:System.Windows.Automation.InvokePattern> and the <xref:System.Windows.Automation.ExpandCollapsePattern> control patterns.</span></span> <span data-ttu-id="fecd6-134"><xref:System.Windows.Automation.ExpandCollapsePattern>expõe o menu e <xref:System.Windows.Automation.InvokePattern> preenche a seleção ativa com a cor escolhida.</span><span class="sxs-lookup"><span data-stu-id="fecd6-134"><xref:System.Windows.Automation.ExpandCollapsePattern> exposes the menu and the <xref:System.Windows.Automation.InvokePattern> fills the active selection with the chosen color.</span></span>

# <a name="etw-events-in-the-net-framework"></a>Eventos ETW no .NET Framework
O rastreamento de eventos para Windows (ETW) é um sistema de rastreamento de alto desempenho, baixa sobrecarga e escalonável fornecido pelos sistemas operacionais Windows. Ele complementa o suporte de depuração e criação de perfil fornecido pelo .NET Framework e pode ser usado para solucionar problemas de uma variedade de cenários. Para obter informações gerais sobre o ETW, consulte [Melhorar o ajuste de depuração e desempenho com o ETW](http://go.microsoft.com/fwlink/?LinkID=161142) na Biblioteca MSDN.  
  
 No .NET Framework, o rastreamento de eventos ETW está disponível para o CLR (Common Language Runtime), a [Biblioteca de Paralelismo de Tarefas](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) e o [PLINQ (LINQ Paralelo)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).

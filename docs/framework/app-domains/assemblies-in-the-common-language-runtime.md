 <span data-ttu-id="898fa-107">Um assembly executa as seguintes funções:</span><span class="sxs-lookup"><span data-stu-id="898fa-107">An assembly performs the following functions:</span></span>

 <span data-ttu-id="898fa-132">Assemblies podem ser estáticos ou dinâmicos.</span><span class="sxs-lookup"><span data-stu-id="898fa-132">Assemblies can be static or dynamic.</span></span> <span data-ttu-id="898fa-133">Assemblies estáticos podem incluir tipos do .NET Framework (interfaces e classes), bem como recursos para o assembly (bitmaps, arquivos JPEG, arquivos de recurso etc.).</span><span class="sxs-lookup"><span data-stu-id="898fa-133">Static assemblies can include .NET Framework types (interfaces and classes), as well as resources for the assembly (bitmaps, JPEG files, resource files, and so on).</span></span> <span data-ttu-id="898fa-134">Assemblies estáticos são armazenados em disco em arquivos PE.</span><span class="sxs-lookup"><span data-stu-id="898fa-134">Static assemblies are stored on disk in portable executable (PE) files.</span></span> <span data-ttu-id="898fa-135">Você também pode usar o .NET Framework para criar assemblies dinâmicos, executados diretamente da memória e não são salvos em disco antes da execução.</span><span class="sxs-lookup"><span data-stu-id="898fa-135">You can also use the .NET Framework to create dynamic assemblies, which are run directly from memory and are not saved to disk before execution.</span></span> <span data-ttu-id="898fa-136">Você pode salvar assemblies dinâmicos em disco após sua execução.</span><span class="sxs-lookup"><span data-stu-id="898fa-136">You can save dynamic assemblies to disk after they have executed.</span></span>

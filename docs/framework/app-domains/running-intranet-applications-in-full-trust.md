<span data-ttu-id="32072-103">A partir do .NET Framework versão 3.5 SP1 (Service Pack 1), os aplicativos e seus assemblies de bibliotecas podem ser executados como assemblies de confiança total de um compartilhamento de rede.</span><span class="sxs-lookup"><span data-stu-id="32072-103">Starting with the .NET Framework version 3.5 Service Pack 1 (SP1), applications and their library assemblies can be run as full-trust assemblies from a network share.</span></span> <span data-ttu-id="32072-104">A evidência de zona <xref:System.Security.SecurityZone.MyComputer> é automaticamente adicionada a assemblies que são carregados de um compartilhamento na intranet.</span><span class="sxs-lookup"><span data-stu-id="32072-104"><xref:System.Security.SecurityZone.MyComputer> zone evidence is automatically added to assemblies that are loaded from a share on the intranet.</span></span> <span data-ttu-id="32072-105">Essa evidência fornece a esses assemblies o mesmo conjunto de concessões (que geralmente é de confiança total) que os assemblies que residem no computador.</span><span class="sxs-lookup"><span data-stu-id="32072-105">This evidence gives those assemblies the same grant set (which is typically full trust) as the assemblies that reside on the computer.</span></span> <span data-ttu-id="32072-106">Essa funcionalidade não se aplica aos aplicativos ClickOnce ou que são projetados para serem executados em um host.</span><span class="sxs-lookup"><span data-stu-id="32072-106">This functionality does not apply to ClickOnce applications or to applications that are designed to run on a host.</span></span>  
  
## <a name="rules-for-library-assemblies"></a><span data-ttu-id="32072-107">Regras para assemblies de bibliotecas</span><span class="sxs-lookup"><span data-stu-id="32072-107">Rules for Library Assemblies</span></span>  
 <span data-ttu-id="32072-108">As seguintes regras se aplicam a assemblies que são carregados por um executável em um compartilhamento de rede:</span><span class="sxs-lookup"><span data-stu-id="32072-108">The following rules apply to assemblies that are loaded by an executable on a network share:</span></span>

## <a name="query-methods"></a><span data-ttu-id="4cf5d-115">Métodos de consulta</span><span class="sxs-lookup"><span data-stu-id="4cf5d-115">Query Methods</span></span>

 <span data-ttu-id="4cf5d-118">As propriedades de reflexão retornam apenas os métodos declarados em um objeto específico em vez de percorrer a árvore de herança.</span><span class="sxs-lookup"><span data-stu-id="4cf5d-118">The reflection properties return only the declared methods on a particular object instead of traversing the inheritance tree.</span></span> <span data-ttu-id="4cf5d-119">Além disso, elas não usam parâmetros <xref:System.Reflection.BindingFlags> para filtragem.</span><span class="sxs-lookup"><span data-stu-id="4cf5d-119">Moreover, they do not use <xref:System.Reflection.BindingFlags> parameters for filtering.</span></span> <span data-ttu-id="4cf5d-120">Em vez disso, a filtragem ocorre no código do usuário, usando consultas LINQ nas coleções retornadas.</span><span class="sxs-lookup"><span data-stu-id="4cf5d-120">Instead, filtering takes place in user code, by using LINQ queries on the returned collections.</span></span> <span data-ttu-id="4cf5d-121">Para objetos de reflexão que se originam com o tempo de execução (por exemplo, como resultado de `typeof(Object)`), a passagem pela árvore de herança é melhor realizada usando os métodos auxiliares da classe <xref:System.Reflection.RuntimeReflectionExtensions>.</span><span class="sxs-lookup"><span data-stu-id="4cf5d-121">For reflection objects that originate with the runtime (for example, as the result of `typeof(Object)`), traversing the inheritance tree is best accomplished by using the helper methods of the <xref:System.Reflection.RuntimeReflectionExtensions> class.</span></span> <span data-ttu-id="4cf5d-122">Os consumidores de objetos de contextos de reflexão personalizados não podem usar esses métodos e devem percorrer a árvore de herança sozinhos.</span><span class="sxs-lookup"><span data-stu-id="4cf5d-122">Consumers of objects from customized reflection contexts cannot use these methods, and must traverse the inheritance tree themselves.</span></span>

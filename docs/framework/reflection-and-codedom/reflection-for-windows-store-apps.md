## <a name="typeinfo-and-assembly-loading"></a><span data-ttu-id="b6e4c-108">TypeInfo e Carregamento do Assembly</span><span class="sxs-lookup"><span data-stu-id="b6e4c-108">TypeInfo and Assembly Loading</span></span>

## <a name="query-methods"></a><span data-ttu-id="b6e4c-115">Métodos de consulta</span><span class="sxs-lookup"><span data-stu-id="b6e4c-115">Query Methods</span></span>

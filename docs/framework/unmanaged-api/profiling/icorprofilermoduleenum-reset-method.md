# <a name="icorprofilermoduleenumreset-method"></a>Método ICorProfilerModuleEnum::Reset
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Reset();  
```

# <a name="icorprofilerinfo10suspendruntime-method"></a>Método ICorProfilerInfo10:: SuspendRuntime
  
 Suspende o tempo de execução sem executar um GC.   
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT SuspendRuntime();
```

# <a name="cor_prf_finalizer_flags-enumeration"></a>Enumeração COR_PRF_FINALIZER_FLAGS
Descreve o finalizador de um objeto.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
typedef enum {  
    COR_PRF_FINALIZER_CRITICAL = 0x1  
} COR_PRF_FINALIZER_FLAGS;  
```  
  
## <a name="members"></a>Membros  
  
|{1&gt;Membro&lt;1}|Descrição|  
|------------|-----------------|  
|`COR_PRF_FINALIZER_CRITICAL`|O finalizador é crítico.|  
  
## <a name="remarks"></a>Comentários  
 A enumeração `COR_PRF_FINALIZER_FLAGS` é usada pelo método [ICorProfilerCallback2:: FinalizeableObjectQueued](icorprofilercallback2-finalizeableobjectqueued-method.md) para descrever o finalizador de um objeto.

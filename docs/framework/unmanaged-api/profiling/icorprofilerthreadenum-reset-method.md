# <a name="icorprofilerthreadenumreset-method"></a>Método ICorProfilerThreadEnum::Reset
Moves the enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Reset();  
```

# <a name="icorprofilerthreadenumreset-method"></a>Método ICorProfilerThreadEnum::Reset
Move o cursor do enumerador para a posição inicial da sequência.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Reset();  
```

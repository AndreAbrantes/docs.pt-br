 <span data-ttu-id="ac406-248">O CLR chama um método na interface `ICorProfilerCallback` (ou [ICorProfilerCallback2](icorprofilercallback2-interface.md)) para notificar o criador de perfil quando um evento, ao qual o criador de perfil assinou, ocorre.</span><span class="sxs-lookup"><span data-stu-id="ac406-248">The CLR calls a method in the `ICorProfilerCallback` (or [ICorProfilerCallback2](icorprofilercallback2-interface.md)) interface to notify the profiler when an event, to which the profiler has subscribed, occurs.</span></span> <span data-ttu-id="ac406-249">Essa é a interface de retorno de chamada principal por meio da qual o CLR se comunica com o criador de perfil de código.</span><span class="sxs-lookup"><span data-stu-id="ac406-249">This is the primary callback interface through which the CLR communicates with the code profiler.</span></span>

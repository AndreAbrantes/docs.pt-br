# <a name="isymunmanagedmethodgetnamespace-method"></a>Método ISymUnmanagedMethod::GetNamespace
Obtém o namespace no qual esse método é definido.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT GetNamespace(  
   [out] ISymUnmanagedNamespace  **pRetVal);  
```  
  
## <a name="parameters"></a>Parâmetros  
 `pRetVal`  
 fora Um ponteiro que é definido para a interface [ISymUnmanagedNamespace](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagednamespace-interface.md) retornada.  
  
## <a name="return-value"></a>Valor retornado  
 S_OK se o método tiver sucesso; caso contrário, E_FAIL ou algum outro código de erro.

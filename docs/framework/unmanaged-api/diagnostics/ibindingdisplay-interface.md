# <a name="ibindingdisplay-interface"></a>Interface IBindingDisplay
Fornece métodos que exibem as informações de associação atuais sobre o aplicativo em execução.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método GetCurrentDisplay](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-getcurrentdisplay-method.md)|Retorna as informações de exibição de associação atual.|  
|[Método InitializeForProcess](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-initializeforprocess-method.md)|Inicializa o objeto `IBindingDisplay`.|

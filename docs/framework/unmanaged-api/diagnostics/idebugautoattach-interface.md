|Método|Descrição|  
|------------|-----------------|  
|[Método AutoAttach](../../../../docs/framework/unmanaged-api/diagnostics/idebugautoattach-autoattach-method.md)|Executa a anexação automática do depurador invocado pelo servidor.|

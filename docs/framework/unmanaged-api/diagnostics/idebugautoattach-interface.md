# <a name="idebugautoattach-interface"></a>Interface IDebugAutoAttach
Define a interface para a anexação automática de servidor chamado depurador.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método AutoAttach](../../../../docs/framework/unmanaged-api/diagnostics/idebugautoattach-autoattach-method.md)|Executa automaticamente o depurador de servidor chamado attach.|  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** DbgAutoAttach.idl

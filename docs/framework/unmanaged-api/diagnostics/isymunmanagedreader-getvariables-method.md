# <a name="isymunmanagedreadergetvariables-method"></a>Método ISymUnmanagedReader::GetVariables
Retorna uma variável não local, dado seu pai e o nome.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetVariables (  
    [in]  mdToken  parent,  
    [in]  ULONG32  cVars,  
    [out] ULONG32  *pcVars,  
    [out, size_is (cVars),  
        length_is (*pcVars)] ISymUnmanagedVariable *pVars[]);  
```  
  
## <a name="parameters"></a>Parâmetros  
 `parent`  
 [in] O pai da variável.  
  
 `cVars`  
 [in] O tamanho do `pVars` matriz.  
  
 `pcVars`  
 [out] Um ponteiro para a variável que recebe o número de variáveis retornadas em `pVars`.  
  
 `pVars`  
 [out] Um ponteiro para a variável que recebe as variáveis.  
  
## <a name="return-value"></a>Valor de retorno  
 S_OK se o método for bem-sucedido; Caso contrário, E_FAIL ou algum outro código de erro.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** CorSym.idl, CorSym.h

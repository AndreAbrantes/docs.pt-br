# <a name="isymunmanageddispose-interface"></a>Interface ISymUnmanagedDispose
Libera recursos não gerenciados.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Destroy](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddispose-destroy-method.md)|Faz com que o objeto subjacente liberar todas as referências internas e retornar falha em todas as chamadas método subsequentes.|

 **Cabeçalho:** CorSym.idl, CorSym.h

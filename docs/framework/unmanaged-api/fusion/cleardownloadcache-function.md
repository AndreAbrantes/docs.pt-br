# <a name="cleardownloadcache-function"></a>Função ClearDownloadCache
Limpa o cache de assembly global de assemblies baixados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT ClearDownloadCache ();  
```

 **Plataformas:** consulte [requisitos de sistema](../../../../docs/framework/get-started/system-requirements.md).  
  
 **Cabeçalho:** Fusion.h  
  
 **Biblioteca:** Fusion e arquivo Mscorwks.dll. Use Fusion em vez do arquivo Mscorwks.dll para garantir que você a versão correta do .NET Framework de destino.

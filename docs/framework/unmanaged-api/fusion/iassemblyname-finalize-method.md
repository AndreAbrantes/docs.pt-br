# <a name="iassemblynamefinalize-method"></a>Método IAssemblyName::Finalize
Permite que este [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) objeto para liberar recursos e executar outras operações de limpeza antes do destruidor é chamado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Finalize ();  
```

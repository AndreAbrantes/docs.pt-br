# <a name="ehostapplicationpolicy-enumeration"></a>Enumeração EHostApplicationPolicy
Indica como modificar um objeto de interface [IHostAssemblyManager](../../../../docs/framework/unmanaged-api/hosting/ihostassemblymanager-interface.md) . Esta enumeração foi preterida.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
typedef enum _HostApplicationPolicy {  
    HOST_APPLICATION_BINDING_POLICY = 1  
} EHostApplicationPolicy;  
```

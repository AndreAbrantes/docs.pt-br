# <a name="hosting-coclasses"></a>Hospedando coclasses
Esta seção descreve as coclasses não gerenciadas de API que usa a API de hospedagem.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Coclass CLRRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Fornece interfaces para gerenciar a execução de código em tempo de execução.  
  
 [Coclass ComCallUnmarshal](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Fornece interfaces para gerenciar o marshaling de ponteiros de interface.  
  
 [Coclass CorRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Fornece interfaces para gerenciar aplicativos que estão sendo executados pelo common language runtime.  
  
 [Coclass TypeNameFactory](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Fornece uma interface para gerenciar a desconstrução de um nome de tipo.

# <a name="fexecuteinappdomaincallback-function-pointer"></a>Ponteiro de função FExecuteInAppDomainCallback
Aponta para uma função que é chamado pelo common language runtime (CLR) para executar código gerenciado.  
  
 Este ponteiro de função foi preterido no [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)].  
  
## <a name="syntax"></a>Sintaxe  
  
```  
typedef HRESULT (__stdcall *FExecuteInAppDomainCallback) (  
    [in] void  *cookie  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `cookie`  
 [in] Um ponteiro para a memória alocada pelo chamador opaco que contém o código gerenciado para ser executado.  
  
 A alocação e a vida útil da memória são controladas pelo chamador (ou seja, o CLR). Isso não é memória de heap gerenciado do CLR.

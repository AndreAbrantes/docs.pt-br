# <a name="iobjecthandle-interface"></a>Interface IObjectHandle
Fornece um método para descodificar objetos marshal-by-value de indireção.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método unwrap](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|Ou não um objeto marshal-by-value de indireção.|

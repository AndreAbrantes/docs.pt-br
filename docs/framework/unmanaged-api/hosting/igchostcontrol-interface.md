# <a name="igchostcontrol-interface"></a>Interface IGCHostControl
Fornece um método que permite que o coletor de lixo solicitar o host para alterar os limites de memória virtual.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método RequestVirtualMemLimit](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Solicitações do host para alterar os limites de memória virtual.|

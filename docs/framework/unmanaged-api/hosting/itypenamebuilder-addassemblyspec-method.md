Esse método oferece suporte a infraestrutura do .NET Framework e não se destina a ser usado diretamente do seu código.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT AddAssemblySpec (  
    [in] LPCWSTR szAssemblySpec  
);
```  

# <a name="ihostautoevent-interface"></a>Interface IHostAutoEvent
Fornece uma representação de implementação do host de um evento de redefinição automática.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Set](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-set-method.md)|Define o atual `IHostAutoEvent` instância para um estado sinalizado.|  
|[Método Wait](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-wait-method.md)|Faz com que o atual `IHostAutoEvent` instância para aguardar até que o evento é de propriedade ou um período de tempo especificado.|

# <a name="couninitializeee-function"></a>Função CoUninitializeEE
`CoUninitializeEE`está obsoleto e não fornece nenhuma funcionalidade.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
void CoUninitializeEE (  
    BOOL fFlags  
);  
```

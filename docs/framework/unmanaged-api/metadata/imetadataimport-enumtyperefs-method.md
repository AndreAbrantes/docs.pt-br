# <a name="imetadataimportenumtyperefs-method"></a>Método IMetaDataImport::EnumTypeRefs
Enumera os tokens de TypeRef definidos no escopo atual de metadados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT EnumTypeRefs (  
   [in, out] HCORENUM    *phEnum,   
   [out] mdTypeRef       rTypeRefs[],  
   [in]  ULONG           cMax,   
   [out] ULONG           *pcTypeRefs  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `phEnum`  
 [out no] Um ponteiro para o enumerador. Isso deve ser NULL para a primeira chamada do método.  
  
 `rTypeRefs`  
 [out] A matriz usada para armazenar os tokens de TypeRef.  
  
 `cMax`  
 [in] O tamanho máximo da `rTypeRefs` matriz.  
  
 `pcTypeRefs`  
 [out] Um ponteiro para o número de tokens de TypeRef retornados em `rTypeRefs`.  
  
## <a name="return-value"></a>Valor de retorno  
  
|HRESULT|Descrição|  
|-------------|-----------------|  
|`S_OK`|`EnumTypeRefs` retornou com êxito.|  
|`S_FALSE`|Não há nenhum tokens para enumerar. Nesse caso, `pcTypeRefs` é zero.|  
  
## <a name="remarks"></a>Comentários  
 Um token TypeRef representa uma referência a um tipo.

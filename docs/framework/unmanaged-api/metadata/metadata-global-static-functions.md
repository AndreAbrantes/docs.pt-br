# <a name="metadata-global-static-functions"></a>Funções estáticas globais de metadados
Começando com o [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)], a documentação para o `_Cor*`, `CoInitialize*`, `CoUninitialize*`, e `CoEEShutDownCOM` funções, que pertencem à hospedagem, está localizado na [funções de hospedagem de CLR preterido](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Não há nenhum funções estáticas globais que pertencem aos metadados.

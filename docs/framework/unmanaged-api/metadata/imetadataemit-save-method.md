# <a name="imetadataemitsave-method"></a>Método IMetaDataEmit::Save
Salva todos os metadados no escopo atual para o arquivo no endereço especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `wzFile`  
 [em] O nome do arquivo para salvar. Se esse valor for nulo, a cópia na memória será salva no último local usado.  
  
 `dwSaveFlags`  
 [in] Reservado. Deve ser zero.

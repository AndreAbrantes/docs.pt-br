# <a name="imetadataemit2savedelta-method"></a>Método IMetaDataEmit2::SaveDelta
Salva as alterações da sessão atual edit-and-continue para o arquivo especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `szFile`  
 [in] O nome do arquivo no qual salvar as alterações.  
  
 `dwSaveFlags`  
 [in] Reservado. Deve ser zero.

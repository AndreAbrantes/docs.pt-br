# <a name="imetadataemit2savedelta-method"></a>Método IMetaDataEmit2::SaveDelta
Salva alterações da sessão de edição e continuação atual para o arquivo especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `szFile`  
 [em] O nome do arquivo o qual salvar alterações.  
  
 `dwSaveFlags`  
 [in] Reservado. Deve ser zero.

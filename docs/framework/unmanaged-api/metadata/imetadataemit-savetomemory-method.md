# <a name="imetadataemitsavetomemory-method"></a>Método IMetaDataEmit::SaveToMemory
Salva todos os metadados no escopo atual para a área de memória especificada.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `pbData`  
 [fora] O endereço para começar a escrever metadados.  
  
 `cbData`  
 [em] O tamanho, em bytes, da memória alocada.

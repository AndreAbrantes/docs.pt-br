# <a name="imetadataimportgetsigfromtoken-method"></a>Método IMetaDataImport::GetSigFromToken
Obtém a assinatura binária metadados associada com o token especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetSigFromToken (   
   [in]   mdSignature        mdSig,   
   [out]  PCCOR_SIGNATURE    *ppvSig,   
   [out]  ULONG              *pcbSig   
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `mdSig`  
 [in] O token para retornar a assinatura de metadados de binários para.  
  
 `ppvSig`  
 [out] Um ponteiro para a assinatura de metadados retornados.  
  
 `pcbSig`  
 [out] O tamanho em bytes da assinatura de metadados de binário.

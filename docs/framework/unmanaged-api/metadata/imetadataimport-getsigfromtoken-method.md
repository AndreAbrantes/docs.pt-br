# <a name="imetadataimportgetsigfromtoken-method"></a>Método IMetaDataImport::GetSigFromToken
Obtém a assinatura binária de metadados associada ao token especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT GetSigFromToken (
   [in]   mdSignature        mdSig,
   [out]  PCCOR_SIGNATURE    *ppvSig,
   [out]  ULONG              *pcbSig
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `mdSig`  
 [em] O token para retornar a assinatura binária de metadados para.  
  
 `ppvSig`  
 [fora] Um ponteiro para a assinatura de metadados retornado.  
  
 `pcbSig`  
 [fora] O tamanho em bytes da assinatura binária de metadados.

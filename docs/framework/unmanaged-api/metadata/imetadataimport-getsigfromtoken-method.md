# <a name="imetadataimportgetsigfromtoken-method"></a>Método IMetaDataImport::GetSigFromToken
Obtém a assinatura de binários de metadados associada ao token especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetSigFromToken (   
   [in]   mdSignature        mdSig,   
   [out]  PCCOR_SIGNATURE    *ppvSig,   
   [out]  ULONG              *pcbSig   
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `mdSig`  
 [in] O token para retornar a assinatura de metadados binário para.  
  
 `ppvSig`  
 [out] Um ponteiro para a assinatura de metadados retornados.  
  
 `pcbSig`  
 [out] O tamanho em bytes da assinatura de metadados binário.

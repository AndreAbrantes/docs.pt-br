# <a name="iceegengetimaptokeniface-method"></a>Método ICeeGen::GetIMapTokenIface
Obtém a interface referenciada pelo token especificado.  
  
 Esse método está obsoleto e não deve ser usado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetIMapTokenIface (  
    [in, out] IUnknown   **pIMapToken  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pIMapToken`  
 [out no] O token de metadados para a interface a ser retornado.

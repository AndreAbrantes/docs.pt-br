# <a name="imetadatafiltermarktoken-method"></a>Método IMetaDataFilter::MarkToken
Define um valor que indica se o token de metadados especificado foi processado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `tk`  
 [in] O token marcar como processado.

# <a name="imetadataemitdefineuserstring-method"></a>Método IMetaDataEmit::DefineUserString
Obtém um token de metadados para a seqüência literal especificada.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `szString`  
 [em] A seqüência de usuário para armazenar.  
  
 `cchString`  
 [em] A contagem de `szString`personagens largos em .  
  
 `pstk`  
 [fora] O token de string atribuído.

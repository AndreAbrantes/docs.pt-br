# <a name="corsavesize-enumeration"></a>Enumeração CorSaveSize
Contém valores que indica o nível de precisão necessária ao consultar o tamanho de um salvamento operação.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
typedef enum CorSaveSize {  
    cssAccurate                = 0x0000,   
    cssQuick                   = 0x0001,   
    cssDiscardTransientCAs     = 0x0002  
} CorSaveSize;  
```  
  
## <a name="members"></a>Membros  
  
|Membro|Descrição|  
|------------|-----------------|  
|`cssAccurate`|Especifica que o valor de retorno deve ser exato.|  
|`cssQuick`|Especifica que o valor de retorno deve ser estimado.|  
|`cssDiscardTransientCAs`|Especifica que tipos descartáveis devem ser removidos.|

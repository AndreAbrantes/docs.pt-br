# <a name="imetadataimportgetuserstring-method"></a>Método IMetaDataImport::GetUserString
Obtém a cadeia de caracteres literal, representada pelo token de metadados especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `stk`  
 [in] O token de cadeia de caracteres para retornar a cadeia de caracteres associada para.  
  
 `szString`  
 [out] Uma cópia da cadeia de caracteres solicitada.  
  
 `cchString`  
 [in] O tamanho máximo em caracteres largos de solicitados `szString`.  
  
 `pchString`  
 [out] O tamanho em caracteres largos de retornado `szString`.

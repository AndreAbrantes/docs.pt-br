# <a name="imetadataemitsetmethodimplflags-method"></a>Método IMetaDataEmit::SetMethodImplFlags
Define ou atualiza a assinatura de metadados da implementação do método herdado que é referenciada pelo token especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT SetMethodImplFlags (
    [in]  mdMethodDef   md,
    [in]  DWORD         dwImplFlags
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `md`  
 [em] O token para que o método seja alterado.  
  
 `dwImplFlags`  
 [em] Uma combinação dos valores da enumeração [CorMethodImpl](../../../../docs/framework/unmanaged-api/metadata/cormethodimpl-enumeration.md) que especifica os recursos de implementação do método.

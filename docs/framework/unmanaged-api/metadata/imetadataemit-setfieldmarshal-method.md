# <a name="imetadataemitsetfieldmarshal-method"></a>Método IMetaDataEmit::SetFieldMarshal
Define as informações de empacotamento do PInvoke para o campo, o retorno do método ou o parâmetro do método referenciado pelo token especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT SetFieldMarshal (  
    [in]  mdToken          tk,
    [in]  PCCOR_SIGNATURE  pvNativeType,
    [in]  ULONG            cbNativeType
);  
```  
  
## <a name="parameters"></a>parâmetros  
 `tk`  
 [em] O token para o item de dados de destino. Isso é `mdFieldDef` um `mdParamDef` ou um símbolo.  
  
 `pvNativeType`  
 [em] A assinatura para o tipo não gerenciado.  
  
 `cbNativeType`  
 [em] A contagem de `pvNativeType`bytes em .

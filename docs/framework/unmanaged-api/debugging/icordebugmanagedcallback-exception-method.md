# <a name="icordebugmanagedcallbackexception-method"></a>Método ICorDebugManagedCallback::Exception
Notifica o depurador que tenha sido lançada uma exceção do código gerenciado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Exception (  
    [in] ICorDebugAppDomain *pAppDomain,  
    [in] ICorDebugThread    *pThread,  
    [in] BOOL                unhandled  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pAppDomain`  
 [in] Um ponteiro para um objeto ICorDebugAppDomain que representa o domínio de aplicativo no qual a exceção foi gerada.  
  
 `pThread`  
 [in] Um ponteiro para um objeto ICorDebugThread que representa o thread em que a exceção foi lançada.  
  
 `unhandled`  
 [in] Se esse valor for `false`, a exceção ainda não foi processada pelo aplicativo; caso contrário, a exceção é sem tratamento e o processo será encerrado.  
  
## <a name="remarks"></a>Comentários  
 A exceção específica pode ser recuperada do objeto de thread.

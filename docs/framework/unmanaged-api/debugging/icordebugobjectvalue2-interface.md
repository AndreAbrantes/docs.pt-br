# <a name="icordebugobjectvalue2-interface"></a>Interface ICorDebugObjectValue2

Estende a interface "ICorDebugObjectValue" para dar suporte à herança e às substituições.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método GetVirtualMethodAndType](../../../../docs/framework/unmanaged-api/debugging/icordebugobjectvalue2-getvirtualmethodandtype-method.md)|`GetVirtualMethodAndType` ainda não foi implementado.|  
  
## <a name="remarks"></a>Comentários

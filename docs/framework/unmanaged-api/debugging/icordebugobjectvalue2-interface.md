# <a name="icordebugobjectvalue2-interface1"></a>Interface1 ICorDebugObjectValue2
Estende a interface "ICorDebugObjectValue" para dar suporte a herança e o substitui.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método GetVirtualMethodAndType](../../../../docs/framework/unmanaged-api/debugging/icordebugobjectvalue2-getvirtualmethodandtype-method.md)|`GetVirtualMethodAndType` ainda não implementado.|  
  
## <a name="remarks"></a>Comentários

>  Essa interface não dá suporte a ser chamada remotamente, entre computadores ou entre processos.

# <a name="iclrdataenummemoryregions-interface"></a>Interface ICLRDataEnumMemoryRegions
Fornece um método para enumerar as regiões da memória que são especificadas por chamadores.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método EnumMemoryRegions](../../../../docs/framework/unmanaged-api/debugging/iclrdataenummemoryregions-enummemoryregions-method.md)|Enumera a áreas específicas de memória.|

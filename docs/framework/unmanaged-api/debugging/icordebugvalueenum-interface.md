# <a name="icordebugvalueenum-interface"></a>Interface ICorDebugValueEnum
Implementa os métodos "ICorDebugEnum" e enumera as matrizes "ICorDebugValue".  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugvalueenum-next-method.md)|Obtém o número especificado de `ICorDebugValue` instâncias da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

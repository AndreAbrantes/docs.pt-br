# <a name="icordebugappdomainenumeratesteppers-method"></a>Método ICorDebugAppDomain::EnumerateSteppers
Obtém um enumerador para todos os steppers ativos no domínio do aplicativo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT EnumerateSteppers (  
    [out] ICorDebugStepperEnum   **ppSteppers  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `ppSteppers`  
 [out] Um ponteiro para o endereço de um objeto de ICorDebugStepperEnum que é o enumerador para todos os steppers ativos no domínio do aplicativo.

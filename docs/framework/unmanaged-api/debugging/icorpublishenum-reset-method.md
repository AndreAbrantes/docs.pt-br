# <a name="icorpublishenumreset-method"></a>Método ICorPublishEnum::Reset

## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Parâmetros  
 `Reset` não usa parâmetros.

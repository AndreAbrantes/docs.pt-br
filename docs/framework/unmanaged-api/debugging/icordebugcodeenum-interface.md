# <a name="icordebugcodeenum-interface"></a>Interface ICorDebugCodeEnum

Implementa os métodos "ICorDebugEnum" e enumera as matrizes "ICorDebugCode".  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugcodeenum-next-method.md)|Obtém o número especificado de `ICorDebugCode` instâncias da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

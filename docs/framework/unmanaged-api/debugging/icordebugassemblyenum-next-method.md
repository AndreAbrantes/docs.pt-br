# <a name="icordebugassemblyenumnext-method"></a>Método ICorDebugAssemblyEnum::Next
Obtém o número especificado de assemblies da coleção, começando na posição atual do cursor.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Next (  
    [in] ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugAssembly *values[],  
    [out] ULONG *pceltFetched  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `celt`  
 [in] O número de módulos (assemblies) a ser recuperado.  
  
 `values`  
 [out] Uma matriz de ponteiros, cada um deles aponta para um objeto ICorDebugAssembly que representa um assembly.  
  
 `pceltFetched`  
 [out] Um ponteiro para o número de assemblies de fato retornadas. Esse valor pode ser null se `celt` é um.

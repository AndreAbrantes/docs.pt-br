# <a name="icordebugenumclone-method"></a>Método ICorDebugEnum::Clone
Cria uma cópia deste objeto ICorDebugEnum.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Clone (  
    [out] ICorDebugEnum **ppEnum  
);  
```  
  
## <a name="parameters"></a>Parâmetros  
 `ppEnum`  
 fora Um ponteiro para o endereço de um objeto de `ICorDebugEnum` que é uma cópia desse objeto `ICorDebugEnum`.

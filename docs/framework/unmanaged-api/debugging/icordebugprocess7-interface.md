# <a name="icordebugprocess7-interface"></a>Interface ICorDebugProcess7
[Com suporte no .NET Framework 4.5.2 e versões posteriores]  
  
 Fornece um método que configura o depurador para manipular atualizações de metadados na memória no processo de destino.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método SetWriteableMetadataUpdateMode](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess7-setwriteablemetadataupdatemode-method.md)|Define um valor que determina como o depurador manipula atualizações de metadados na memória no processo de destino.|

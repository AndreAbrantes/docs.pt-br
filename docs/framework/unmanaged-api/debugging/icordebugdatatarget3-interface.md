# <a name="icordebugdatatarget3-interface"></a>Interface ICorDebugDataTarget3
Estende logicamente a interface [ICorDebugDataTarget](../../../../docs/framework/unmanaged-api/debugging/icordebugdatatarget-interface.md) para fornecer informações sobre os módulos carregados.  
  
## <a name="method"></a>Método  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método GetLoadedModules](../../../../docs/framework/unmanaged-api/debugging/icordebugdatatarget3-getloadedmodules-method.md)|Obtém uma lista dos módulos que foram carregados até agora.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Essa interface está disponível somente com .NET Native. Se você implementar essa interface para cenários ICorDebug fora do .NET Native, o Common Language Runtime ignorará essa interface.

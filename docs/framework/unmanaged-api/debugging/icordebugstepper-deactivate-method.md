# <a name="icordebugstepperdeactivate-method"></a>Método ICorDebugStepper::Deactivate
Faz com que esse ICorDebugStepper cancelar o último comando etapa que ele recebeu.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>Comentários  
 Um novo comando passo a passo pode ser emitido após o comando de etapa recebido recentemente foi cancelado.

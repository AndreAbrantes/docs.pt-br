# <a name="icordebughandlevaluedispose-method"></a>Método ICorDebugHandleValue::Dispose
Libera o identificador referenciado pelo objeto ICorDebugHandleValue sem liberar explicitamente o ponteiro de interface.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Dispose ();  
```

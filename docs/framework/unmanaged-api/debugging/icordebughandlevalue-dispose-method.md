# <a name="icordebughandlevaluedispose-method"></a>Método ICorDebugHandleValue::Dispose
Libera o identificador referenciado por este objeto ICorDebugHandleValue sem liberar explicitamente o ponteiro de interface.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT Dispose ();  
```

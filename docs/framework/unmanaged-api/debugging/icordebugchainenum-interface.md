# <a name="icordebugchainenum-interface"></a>Interface ICorDebugChainEnum

Implementa métodos ICorDebugEnum e enumera matrizes de ICorDebugChain.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Obtém o número especificado de `ICorDebugChain` instâncias de enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
>  Essa interface não dá suporte a ser chamada remotamente, entre computadores ou entre processos.

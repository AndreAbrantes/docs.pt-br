# <a name="icordebugchainenum-interface"></a>Interface ICorDebugChainEnum

Implementa métodos ICorDebugEnum e enumera matrizes ICorDebugChain.  
  
## <a name="methods"></a>{1&gt;Métodos&lt;1}  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](icordebugchainenum-next-method.md)|Obtém o número especificado de instâncias de `ICorDebugChain` da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

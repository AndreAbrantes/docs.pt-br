# <a name="icordebugchainenum-interface1"></a>ICorDebugChainEnum Interface1
Implementa métodos ICorDebugEnum e enumera ICorDebugChain matrizes.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Obtém o número especificado de `ICorDebugChain` instâncias de enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
>  Esta interface não dá suporte a que está sendo chamado remotamente, entre computadores ou entre processos.

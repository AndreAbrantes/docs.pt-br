# <a name="icordebugchainenum-interface"></a>Interface ICorDebugChainEnum

Implementa métodos ICorDebugEnum e enumera matrizes ICorDebugChain.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Obtém o número especificado de `ICorDebugChain` instâncias da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

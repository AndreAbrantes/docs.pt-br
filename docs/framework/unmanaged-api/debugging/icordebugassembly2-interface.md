# <a name="icordebugassembly2-interface"></a>Interface ICorDebugAssembly2

Representa um assembly. Essa interface é uma extensão da interface ICorDebugAssembly.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método IsFullyTrusted](../../../../docs/framework/unmanaged-api/debugging/icordebugassembly2-isfullytrusted-method.md)|Obtém um valor que indica se o assembly recebeu confiança total pelo sistema de segurança de tempo de execução.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

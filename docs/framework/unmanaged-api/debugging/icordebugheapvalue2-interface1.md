# <a name="icordebugheapvalue2-interface1"></a>Interface1 ICorDebugHeapValue2
Uma extensão de ICorDebugHeapValue que fornece suporte para o common language runtime (CLR) trata.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método CreateHandle](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|Cria um identificador do tipo especificado para este `ICorDebugHeapValue2` objeto.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
>  Essa interface não dá suporte a ser chamada remotamente, entre computadores ou entre processos.

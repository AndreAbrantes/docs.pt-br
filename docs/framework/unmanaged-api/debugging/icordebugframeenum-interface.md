# <a name="icordebugframeenum-interface"></a>Interface ICorDebugFrameEnum

Implementa métodos ICorDebugEnum e enumera matrizes ICorDebugFrame.  
  
## <a name="methods"></a>{1&gt;Métodos&lt;1}  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](icordebugframeenum-next-method.md)|Obtém o número especificado de instâncias de `ICorDebugFrame` da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

# <a name="icordebugtypeenum-interface"></a>Interface ICorDebugTypeEnum
Implementa os métodos "ICorDebugEnum" e enumera as matrizes "ICorDebugType".  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](../../../../docs/framework/unmanaged-api/debugging/icordebugtypeenum-next-method.md)|Obtém o número especificado de `ICorDebugType` instâncias da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

# <a name="icordebugtypeenum-interface"></a>Interface ICorDebugTypeEnum
Implementa os métodos "ICorDebugEnum" e enumera as matrizes "ICorDebugType".  
  
## <a name="methods"></a>{1&gt;Métodos&lt;1}  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método Next](icordebugtypeenum-next-method.md)|Obtém o número especificado de instâncias de `ICorDebugType` da enumeração, começando na posição atual.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.

# <a name="icordebugfunctiongetclass-method"></a>Método ICorDebugFunction::GetClass
Obtém um objeto ICorDebugClass que representa a classe da qual essa função é membro.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT GetClass (  
    [out] ICorDebugClass **ppClass  
);  
```  
  
## <a name="parameters"></a>Parâmetros  
 `ppClass`  
 fora Um ponteiro para o endereço do objeto `ICorDebugClass` que representa a classe, ou NULL, se essa função não for um membro de uma classe.

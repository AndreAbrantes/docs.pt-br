# <a name="install-the-net-framework-on-windows-8-and-windows-server-2012"></a>Instalar o .NET Framework no Windows 8 e Windows Server 2012

O .NET Framework é necessário para executar muitos aplicativos no Windows. Você pode usar as instruções a seguir para instalá-lo. Você pode ter chegado nesta página depois de tentar executar um aplicativo e ver a caixa de diálogo a seguir em seu computador.

![Este aplicativo não pôde ser iniciado](./media/this-application-could-not-be-started.png)

Essas instruções ajudará você a instalar as versões do .NET Framework que você precisa. O [do .NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) é a versão mais recente. Não há suporte no Windows 8. Ele está incluído com o [criadores de atualização do Windows 10 estão](https://www.microsoft.com/software-download/windows10) e [1709 de versão do Windows Server 2016](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709).

## <a name="net-framework-471"></a>.NET framework 4.7.1

* [Baixar o .NET Framework 4.7.1](https://www.microsoft.com/net/framework/versions/net471?utm_source=ms-docs&utm_medium=referral)

Há suporte para o .NET Framework 4.7.1 no Windows Server 2012. Não há suporte no Windows 8.

O [do .NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) pode ser usado para executar aplicativos criados para o .NET Framework 4.0 ou posterior.

## <a name="net-framework-46"></a>.NET Framework 4.6

O [do .NET Framework 4.6](https://www.microsoft.com/en-us/download/details.aspx?id=48130) é a versão mais recente com suporte do .NET Framework no Windows 8.

O .NET Framework 4.6 dá suporte a aplicativos criados para o .NET Framework 4.0 a 4.6.

## <a name="net-framework-35"></a>.NET Framework 3,5

Siga as instruções para instalar o [.NET Framework 3.5 no Windows 8 e 10](dotnet-35-windows-10.md).

O .NET Framework 3.5 dá suporte a aplicativos criados para o .NET Framework 1.0 a 3.5.

## <a name="help"></a>Ajuda

Você pode [entrar em contato com a Microsoft para obter ajuda](mailto:dotnet-install-help@service.microsoft.com?subject=Install-Help) se não for possível instalar a versão correta do .NET Framework.

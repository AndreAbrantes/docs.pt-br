# <a name="install-the-net-framework-on-windows-vista-and-windows-server-2008"></a>Instalar o .NET Framework no Windows Vista e Windows Server 2008

O .NET Framework é necessário para executar muitos aplicativos no Windows. Você pode usar as instruções a seguir para instalá-lo. Você pode ter chegado nesta página depois de tentar executar um aplicativo e ver a caixa de diálogo a seguir em seu computador.

![Este aplicativo não pôde ser iniciado](./media/this-application-could-not-be-started.png)

Essas instruções ajudará você a instalar as versões do .NET Framework que você precisa. O [do .NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) é a versão mais recente. Não há suporte no Windows Vista e Windows Server 2008. Ele está incluído com o [criadores de atualização do Windows 10 estão](https://www.microsoft.com/software-download/windows10) e [1709 de versão do Windows Server 2016](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709).

## <a name="net-framework-46"></a>.NET Framework 4.6

O [do .NET Framework 4.6](https://www.microsoft.com/en-us/download/details.aspx?id=48130&e6b34bbe-475b-1abd-2c51-b5034bcdd6d2=True) é a versão mais recente com suporte do .NET Framework no Windows Vista e Windows Server 2008.

O [do .NET Framework 4.6](https://www.microsoft.com/en-us/download/details.aspx?id=48130&e6b34bbe-475b-1abd-2c51-b5034bcdd6d2=True) oferece suporte a aplicativos criados para o .NET Framework 4.0 a 4.6.

## <a name="net-framework-35"></a>.NET Framework 3,5

Você pode instalar o [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) no Windows Vista.

O .NET Framework 3.5 dá suporte a aplicativos criados para o .NET Framework 1.0 a 3.5.

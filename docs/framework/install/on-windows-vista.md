# <a name="install-the-net-framework-on-windows-vista"></a>Instalar o .NET Framework no Windows Vista

Talvez você precise do .NET Framework para executar um aplicativo no Windows Vista. Você pode usar as instruções a seguir para ajudar. As instruções dependem de qual versão do .NET Framework que você precisa.

## <a name="net-framework-461-and-later"></a>.NET Framework 4.6.1 e posterior

Você não pode instalar o .NET Framework 4.6.1 ou posterior no Windows Vista. Em vez disso, você deve atualizar para uma versão posterior do Windows para usar o .NET Framework 4.6.1 ou posterior.

O .NET Framework 4.6.2 está incluído na [Atualização de Aniversário do Windows 10](https://www.microsoft.com/software-download/windows10).

## <a name="net-framework-46"></a>.NET Framework 4.6

Você pode instalar o [.NET Framework 4.6](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) no Windows Vista.

## <a name="net-framework-35"></a>.NET Framework 3,5

Você pode instalar o [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) no Windows Vista.

O .NET Framework 3.5 pode ser usado para executar aplicativos compilados para o .NET Framework 1.0 a 3.5.

## <a name="help"></a>Ajuda

Você pode [entrar em contato com a Microsoft para obter ajuda](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework) se não for possível instalar a versão correta do .NET Framework.

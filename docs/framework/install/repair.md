# <a name="repair-the-net-framework"></a>Reparar o .NET Framework

Em algumas situações, a instalação do .NET Framework pode ser danificada e precisar ser reparada. Isso pode ser o caso se seu aplicativo falhar logo após você tentar iniciá-lo ou se não for possível instalar versões mais recentes do .NET Framework.

Você pode reparar sua instalação do .NET Framework utilizando a [Ferramenta de Reparo do .NET Framework](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe).

Se seu aplicativo ainda não estiver iniciando após reparar o .NET Framework, o aplicativo poderá ter um problema. Nesse caso, você deverá contatar o editor do aplicativo.

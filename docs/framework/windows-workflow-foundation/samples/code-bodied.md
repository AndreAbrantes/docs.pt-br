# <a name="code-bodied"></a>o corpóreo
Esta seção contém exemplos de atividade personalizados criados declarativamente no código.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Indicadores](../../../../docs/framework/windows-workflow-foundation/samples/bookmarks.md)  
 Este exemplo demonstra como escrever uma atividade personalizado que cria um indicador para receber entrada externo.  
  
 [Composição personalizada usando atividade nativa](../../../../docs/framework/windows-workflow-foundation/samples/custom-composite-using-native-activity.md)  
 Este exemplo demonstra como escrever <xref:System.Activities.NativeActivity> que agenda outros objetos de <xref:System.Activities.Activity> para controlar o fluxo de execução de um fluxo de trabalho.  
  
 [Propriedades de execução](../../../../docs/framework/windows-workflow-foundation/samples/execution-properties.md)  
 Este exemplo mostra como definir e usar uma propriedade de execução em uma atividade personalizado.

 Este exemplo demonstra como desenvolver uma atividade personalizado que tem <xref:System.Activities.ActivityAction>.  
  
 [Usando AsyncCodeActivityContext em uma atividade](../../../../docs/framework/windows-workflow-foundation/samples/using-asyncoperationcontext-in-an-activity-sample.md)  
 Este exemplo demonstra como desenvolver um personalizado <xref:System.Activities.CodeActivity> que usa <xref:System.Activities.AsyncCodeActivityContext> para executar o trabalho de forma assíncrona fora de fluxo de trabalho. 
  
 [Atividade personalizada Olá, Mundo](../../../../docs/framework/windows-workflow-foundation/samples/hello-world-custom-activity.md)  
 Este exemplo demonstra vários recursos importantes do Windows Workflow Foundation (WF), incluindo como criar uma atividade personalizada simple.  
  
 [Argumentos dinâmicos](../../../../docs/framework/windows-workflow-foundation/samples/dynamic-arguments.md)  
 Este exemplo demonstra como implementar uma atividade para os argumentos são definidos pelo consumidor da atividade em vez do autor de atividade.

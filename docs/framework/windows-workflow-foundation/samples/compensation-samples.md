# <a name="compensation-samples"></a>Compensação (Exemplos)
Esta seção contém exemplos relacionados a compensação no Windows Workflow Foundation (WF).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Atividade compensável](../../../../docs/framework/windows-workflow-foundation/samples/compensable-activity-sample.md)  
 Demonstra como usar a atividade de <xref:System.Activities.Statements.CompensableActivity> para definir o trabalho a ser feitos se necessário para uma determinada ação durante a execução normal e o trabalho que é necessário para ser feito para compensar essa ação, mais tarde.  
  
 [Compensação personalizada](../../../../docs/framework/windows-workflow-foundation/samples/custom-compensation-sample.md)  
 Demonstra como usar <xref:System.Activities.Statements.CompensableActivity> e seu manipulador de compensação para definir a lógica personalizada de compensação.  
  
 [Confirmação](../../../../docs/framework/windows-workflow-foundation/samples/confirmation.md)  
 Demonstra quatro cenários comuns envolvendo o uso de <xref:System.Activities.Statements.CompensableActivity> e de confirmação.  
  
 [Manipulador de cancelamento atividade compensável](../../../../docs/framework/windows-workflow-foundation/samples/cancellation-handler-on-compensable-activity.md)  
 Demonstra o uso de um manipulador cancelar em <xref:System.Activities.Statements.CompensableActivity>.

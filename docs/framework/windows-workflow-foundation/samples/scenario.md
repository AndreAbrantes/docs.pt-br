Esta seção contém exemplos de cenários do Windows Workflow Foundation (WF).

 [Biblioteca de atividades](../../../../docs/framework/windows-workflow-foundation/samples/activity-library.md)  
 Contém exemplos que demonstram cenários usando a biblioteca interno de atividade.  
  
 [Serviços](../../../../docs/framework/windows-workflow-foundation/samples/services.md)

 [Transações](../../../../docs/framework/windows-workflow-foundation/samples/transactions.md)

 [Padrão de confirmação automática](../../../../docs/framework/windows-workflow-foundation/samples/auto-confirm-pattern.md)

 [Integração de WF e WPF em XAML](../../../../docs/framework/windows-workflow-foundation/samples/wpf-and-wf-integration-in-xaml.md)

 [Kit de ferramentas de Ruleset externo](../../../../docs/framework/windows-workflow-foundation/samples/external-ruleset-toolkit.md)

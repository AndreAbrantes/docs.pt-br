# <a name="validation"></a>Validação
Esta seção contém exemplos que demonstram validação de atividade.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Validação externo de atividades](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 Este exemplo mostra como adicionar lógica de validação para uma atividade interno que não é do autor.  
  
 [Validação básica](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 Demonstra uma atividade, `CreateProduct`, que valida que o argumento de `Cost` é menor que ou igual ao seu argumento de `Price` .  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 Demonstra uma atividade, `CreateLocation`, que requer argumentos e argumentos opcionais e permite que o usuário escolha para fornecer um dos dois diferentes conjuntos de argumentos.  
  
 [Validação de relações de atividades](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 Demonstra como validar relações entre atividades.

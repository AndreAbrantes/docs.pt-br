# <a name="how-to-create-a-custom-persistence-participant"></a><span data-ttu-id="a1801-102">Como: Crie um participante personalizado de persistência</span><span class="sxs-lookup"><span data-stu-id="a1801-102">How to: Create a Custom Persistence Participant</span></span>
<span data-ttu-id="a1801-103">O seguinte procedimento tem as etapas para criar um participante de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-103">The following procedure has steps to create a persistence participant.</span></span> <span data-ttu-id="a1801-104">Consulte a [participação na persistência](https://go.microsoft.com/fwlink/?LinkID=177735) exemplo e [extensibilidade de Store](../../../docs/framework/windows-workflow-foundation/store-extensibility.md) tópico para exemplos de implementações de participantes de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-104">See the [Participating in Persistence](https://go.microsoft.com/fwlink/?LinkID=177735) sample and [Store Extensibility](../../../docs/framework/windows-workflow-foundation/store-extensibility.md) topic for sample implementations of persistence participants.</span></span>  
  
1.  <span data-ttu-id="a1801-105">Crie uma classe que deriva de <xref:System.Activities.Persistence.PersistenceParticipant> ou classe de <xref:System.Activities.Persistence.PersistenceIOParticipant> .</span><span class="sxs-lookup"><span data-stu-id="a1801-105">Create a class deriving from the <xref:System.Activities.Persistence.PersistenceParticipant> or the <xref:System.Activities.Persistence.PersistenceIOParticipant> class.</span></span> <span data-ttu-id="a1801-106">A classe de PersistenceIOParticipant oferece os mesmos pontos de extensibilidade que a classe de PersistenceParticipant além de poder participar em operações de e/s.</span><span class="sxs-lookup"><span data-stu-id="a1801-106">The PersistenceIOParticipant class offers the same extensibility points as the PersistenceParticipant class in addition to being able to participate in I/O operations.</span></span> <span data-ttu-id="a1801-107">Execute uma ou mais das seguintes etapas.</span><span class="sxs-lookup"><span data-stu-id="a1801-107">Follow one or more of the following steps.</span></span>  
  
2.  <span data-ttu-id="a1801-108">Implementar o método de <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> .</span><span class="sxs-lookup"><span data-stu-id="a1801-108">Implement the <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> method.</span></span> <span data-ttu-id="a1801-109">O **CollectValues** método tem parâmetros de dicionário de dois, para armazenar os valores de leitura/gravação e o outro é para armazenar valores somente gravação (usados mais tarde em consultas).</span><span class="sxs-lookup"><span data-stu-id="a1801-109">The **CollectValues** method has two dictionary parameters, one for storing read/write values and the other one for storing write-only values (used later in queries).</span></span> <span data-ttu-id="a1801-110">Nesse método, você deve preencher esses dicionários com dados que são específicos para um participante de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-110">In this method, you should populate these dictionaries with data that is specific to a persistence participant.</span></span> <span data-ttu-id="a1801-111">Cada dicionário contém o nome do valor como a chave e o valor próprio como um objeto de <xref:System.Runtime.DurableInstancing.InstanceValue> .</span><span class="sxs-lookup"><span data-stu-id="a1801-111">Each dictionary contains the name of the value as the key and the value itself as an <xref:System.Runtime.DurableInstancing.InstanceValue> object.</span></span>  
  
     <span data-ttu-id="a1801-112">Os valores no dicionário de readWriteValues são empacotados como **InstanceValue** objetos.</span><span class="sxs-lookup"><span data-stu-id="a1801-112">The values in the readWriteValues dictionary are packaged as **InstanceValue** objects.</span></span> <span data-ttu-id="a1801-113">Os valores no dicionário somente gravação são empacotados como **InstanceValue** objetos com o conjunto de conjunto e de Instancevalueoption.</span><span class="sxs-lookup"><span data-stu-id="a1801-113">The values in the write-only dictionary are packaged as **InstanceValue** objects with InstanceValueOptions.Optional and InstanceValueOption.WriteOnly set.</span></span> <span data-ttu-id="a1801-114">Cada **InstanceValue** fornecidas pelo **CollectValues** implementações por todos os participantes de persistência devem ter um nome exclusivo.</span><span class="sxs-lookup"><span data-stu-id="a1801-114">Each **InstanceValue** provided by the **CollectValues** implementations across all persistence participants must have a unique name.</span></span>  
  
    ```  
    protected virtual void CollectValues (out IDictionary<XName,Object> readWriteValues, out IDictionary<XName,Object> writeOnlyValues)  
    ```  
  
3.  <span data-ttu-id="a1801-115">Implementar o método de <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> .</span><span class="sxs-lookup"><span data-stu-id="a1801-115">Implement the <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> method.</span></span> <span data-ttu-id="a1801-116">O **MapValues** método utiliza dois parâmetros que são semelhantes aos parâmetros que o **CollectValues** método recebe.</span><span class="sxs-lookup"><span data-stu-id="a1801-116">The **MapValues** method takes two parameters that are similar to the parameters that the **CollectValues** method receives.</span></span> <span data-ttu-id="a1801-117">Todos os valores coletados na **CollectValues** estágio são passadas para esses parâmetros de dicionário.</span><span class="sxs-lookup"><span data-stu-id="a1801-117">All the values collected in the **CollectValues** stage are passed through these dictionary parameters.</span></span> <span data-ttu-id="a1801-118">Os novos valores adicionados pela **MapValues** estágio são adicionados aos valores somente gravação.</span><span class="sxs-lookup"><span data-stu-id="a1801-118">The new values added by the **MapValues** stage are added to the write-only values.</span></span>  <span data-ttu-id="a1801-119">O dicionário somente gravação é usado para fornecer dados para uma fonte externa associada não diretamente com os valores da instância.</span><span class="sxs-lookup"><span data-stu-id="a1801-119">The write-only dictionary is used to provide data to an external source not directly associated with the instance values.</span></span> <span data-ttu-id="a1801-120">Cada valor fornecido por implementações do **MapValues** método por todos os participantes de persistência deve ter um nome exclusivo.</span><span class="sxs-lookup"><span data-stu-id="a1801-120">Each value provided by implementations of the **MapValues** method across all persistence participants must have a unique name.</span></span>  
  
    ```  
    protected virtual IDictionary<XName,Object> MapValues (IDictionary<XName,Object> readWriteValues,IDictionary<XName,Object> writeOnlyValues)  
    ```  
  
     <span data-ttu-id="a1801-121">O método de <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> fornece funcionalidade que o <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> contrário, que permite uma dependência em outro valor fornecido por outro participante de persistência que não é processado pelo <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> ainda.</span><span class="sxs-lookup"><span data-stu-id="a1801-121">The <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> method provides functionality that <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> does not, in that it allows for a dependency on another value provided by another persistence participant that hasn’t been processed by <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> yet.</span></span>  
  
4.  <span data-ttu-id="a1801-122">Implemente a **PublishValues** método.</span><span class="sxs-lookup"><span data-stu-id="a1801-122">Implement the **PublishValues** method.</span></span> <span data-ttu-id="a1801-123">O **PublishValues** método recebe um dicionário que contém todos os valores carregados do armazenamento de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-123">The **PublishValues** method receives a dictionary containing all the values loaded from the persistence store.</span></span>  
  
    ```  
    protected virtual void PublishValues (IDictionary<XName,Object> readWriteValues)  
    ```  
  
5.  <span data-ttu-id="a1801-124">Implemente a **BeginOnSave** método se o participante é um participante de e/s de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-124">Implement the **BeginOnSave** method if the participant is a persistence I/O participant.</span></span> <span data-ttu-id="a1801-125">Este método é chamado durante uma operação de salvar.</span><span class="sxs-lookup"><span data-stu-id="a1801-125">This method is called during a Save operation.</span></span> <span data-ttu-id="a1801-126">Nesse método, você deve executar o suplemento do e/s de persistência (Salvar) instâncias de fluxo de trabalho.</span><span class="sxs-lookup"><span data-stu-id="a1801-126">In this method, you should perform I/O adjunct to the persisting (saving) workflow instances.</span></span>  <span data-ttu-id="a1801-127">Se o host está usando uma transação para o comando correspondente de persistência, a mesma transação é fornecida em Transaction.Current.</span><span class="sxs-lookup"><span data-stu-id="a1801-127">If the host is using a transaction for the corresponding persistence command, the same transaction is provided in Transaction.Current.</span></span>  <span data-ttu-id="a1801-128">Além disso, PersistenceIOParticipants pode anunciar um requisito transacional de consistência, nesse caso o host cria uma transação para o episódio de persistência se não seria usado de outra maneira.</span><span class="sxs-lookup"><span data-stu-id="a1801-128">Additionally, PersistenceIOParticipants may advertise a transactional consistency requirement, in which case the host creates a transaction for the persistence episode if one would not otherwise be used.</span></span>  
  
    ```  
    protected virtual IAsyncResult BeginOnSave (IDictionary<XName,Object> readWriteValues, IDictionary<XName,Object> writeOnlyValues, TimeSpan timeout, AsyncCallback callback, Object state)  
    ```  
  
6.  <span data-ttu-id="a1801-129">Implemente a **BeginOnLoad** método se o participante é um participante de e/s de persistência.</span><span class="sxs-lookup"><span data-stu-id="a1801-129">Implement the **BeginOnLoad** method if the participant is a persistence I/O participant.</span></span> <span data-ttu-id="a1801-130">Este método é chamado durante uma operação de carregamento.</span><span class="sxs-lookup"><span data-stu-id="a1801-130">This method is called during a Load operation.</span></span> <span data-ttu-id="a1801-131">Nesse método, você deve executar o suplemento de e/s para o carregamento de instâncias de fluxo de trabalho.</span><span class="sxs-lookup"><span data-stu-id="a1801-131">In this method, you should perform I/O adjunct to the loading of workflow instances.</span></span> <span data-ttu-id="a1801-132">Se o host está usando uma transação para o comando correspondente de persistência, a mesma transação é fornecida em Transaction.Current.</span><span class="sxs-lookup"><span data-stu-id="a1801-132">If the host is using a transaction for the corresponding persistence command, the same transaction is provided in Transaction.Current.</span></span> <span data-ttu-id="a1801-133">Além disso, os participantes de e/s de persistência podem anunciar um requisito transacional de consistência, nesse caso o host cria uma transação para o episódio de persistência se um caso contrário, não será usado.</span><span class="sxs-lookup"><span data-stu-id="a1801-133">Additionally, Persistence I/O participants may advertise a transactional consistency requirement, in which case the host creates a transaction for the persistence episode if one would not otherwise be used.</span></span>  
  
    ```  
    protected virtual IAsyncResult BeginOnLoad (IDictionary<XName,Object> readWriteValues, TimeSpan timeout, AsyncCallback callback, Object state)  
    ```

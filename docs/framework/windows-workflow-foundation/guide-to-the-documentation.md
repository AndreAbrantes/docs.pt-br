# <a name="guide-to-the-windows-workflow-documentation"></a>Guie a documentação de fluxo de trabalho do Windows
Este tópico contém informações sobre como usar a documentação do Windows Workflow Foundation. Documentos associados são pontos de partida recomendados agrupados de acordo com interesses e níveis específicos de experiência.  
  
## <a name="new-to-windows-workflow-foundation-programming"></a>Novo para programação do Windows Workflow Foundation  
  
-   Se você for novo para programação com o Windows Workflow Foundation e você quiser apenas ver alguns aplicativos de exemplo que funcionam, consulte os tópicos listados em [exemplos de fluxo de trabalho do Windows](../../../docs/framework/windows-workflow-foundation/samples/index.md).  
  
-   Para obter uma discussão de vários modelos de controle de fluxo do Windows Workflow Foundation, consulte [criando fluxos de trabalho](../../../docs/framework/windows-workflow-foundation/designing-workflows.md).  
  
-   Para obter um tutorial que percorra as etapas básicas de criação de um aplicativo do Windows Workflow Foundation, consulte [Tutorial de Introdução](../../../docs/framework/windows-workflow-foundation/getting-started-tutorial.md).  
  
-   Se você estiver interessado nos conceitos por trás do Windows Workflow Foundation, consulte os tópicos de [visão geral conceitual de um fluxo de trabalho do Windows](../../../docs/framework/windows-workflow-foundation/conceptual-overview.md) seção.  
  
-   Para obter uma lista de termos usados na documentação do Windows Workflow Foundation, consulte [Glossário do Windows Workflow Foundation para o .NET Framework 4.5](../../../docs/framework/windows-workflow-foundation/glossary.md).  
  
-   Para obter uma lista de novos conceitos e funcionalidades, consulte [o que há de novo no Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/whats-new.md).  
  
## <a name="programming-in-depth"></a>Programação detalhada  
  
-   Para obter uma discussão detalhada sobre o modelo de objeto do Windows Workflow Foundation, consulte [arquitetura de fluxo de trabalho do Windows](../../../docs/framework/windows-workflow-foundation/architecture.md).  
  
-   Se você estiver pronto para começar a desenvolver um aplicativo, consulte [programação do Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/programming.md).  
  
-   Se você deseja estender ou personalizar o Windows Workflow Foundation para atender às suas necessidades, consulte [estendendo Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation/extend.md).

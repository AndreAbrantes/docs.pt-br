# <a name="1143---flowchartnextnull"></a>1143 - FlowchartNextNull
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|1143|  
|Palavras-chave|WFActivities|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Indica que o nó seguinte no fluxograma é nulo. A execução do fluxograma será finalizada.  
  
## <a name="message"></a>Mensagem  
 Fluxograma “%1 " /FlowStep - o nó seguir é nulo. A execução do fluxograma será finalizada.

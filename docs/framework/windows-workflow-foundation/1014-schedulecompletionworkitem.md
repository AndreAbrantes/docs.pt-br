|-|-|

|Palavras-chave|WFRuntime|  
|Nível|Detalhado|

## <a name="description"></a>Descrição  
 Indica que um CompletionWorkItem foi agendada.  
  
## <a name="message"></a>Mensagem  
 Um CompletionWorkItem foi agendado para atividades pai '%1', DisplayName: "%2", InstanceId: '%3'.  Atividade counting “%4 ", DisplayName: “%5", InstanceId: “%6".

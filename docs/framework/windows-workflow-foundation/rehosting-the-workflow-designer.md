# <a name="rehosting-the-workflow-designer"></a>Hospedando novamente o designer de fluxo de trabalho
O Windows Designer de Fluxo de Trabalho pode ser rehospedado em ambientes fora do Visual Studio 2012 para fins de criação, modificação e monitoramento de fluxos de trabalho.

 O tipo <xref:System.Activities.Presentation.WorkflowDesigner> é um wrapper da tela, da grade de propriedades e de outros elementos, e expõe um modelo de programação básico para manipular a maioria dos cenários nova hospedagem do designer. Hospedar o <xref:System.Activities.Presentation.WorkflowDesigner> dentro de um aplicativo de Windows Presentation Foundation (WPF) é um cenário de rehospedagem comum para Designer de Fluxo de Trabalho.

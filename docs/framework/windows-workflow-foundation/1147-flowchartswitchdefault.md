# <a name="1147---flowchartswitchdefault"></a>1147 - FlowchartSwitchDefault

|||  
|-|-|  
|ID|1147|  
|Palavras-chave|WFActivities|  
|Nível|Informações do|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  

 Indica que os casos padrão foram selecionados em um interruptor do fluxograma.  
  
## <a name="message"></a>Mensagem  

 Fluxograma “%1 " /FlowSwitch - os casos padrão foram selecionados.

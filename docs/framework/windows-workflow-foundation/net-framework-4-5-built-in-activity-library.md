# <a name="net-framework-45-built-in-activity-library"></a>biblioteca interno de atividade do .NET Framework 4.5
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] inclui uma nova biblioteca de atividade com funcionalidade expandida. Os tipos de atividades incluem o seguinte:  
  
-   [Fluxo de Controle](../../../docs/framework/windows-workflow-foundation/control-flow-activities-in-wf.md)  
  
-   [Fluxograma](../../../docs/framework/windows-workflow-foundation/flowchart-activities-in-wf.md)  
  
-   [Máquina de estado](../../../docs/framework/windows-workflow-foundation/state-machine-activities-in-wf.md)  
  
-   [Atividades de mensagens](../../../docs/framework/wcf/feature-details/messaging-activities.md)  
  
-   [Tempo de execução](../../../docs/framework/windows-workflow-foundation/runtime-activities-in-wf.md)  
  
-   [Primitives](../../../docs/framework/windows-workflow-foundation/primitives-activities-in-wf.md)  
  
-   [Transação](../../../docs/framework/windows-workflow-foundation/transaction-activities-in-wf.md)  
  
-   [Coleção](../../../docs/framework/windows-workflow-foundation/collection-activities-in-wf.md)  
  
-   [Tratamento de erro](../../../docs/framework/windows-workflow-foundation/error-handling-activities-in-wf.md)  
  
-   [Migração](../../../docs/framework/windows-workflow-foundation/migration-activity-in-wf.md)  
  
 Os activites na biblioteca interno de atividade podem ser compostos fluxos de trabalho, ou ser usados ao lado das atividades personalizados. As atividades na biblioteca interno de são; atividade sealed não se destinam ser usados para criar a nova funcionalidade através de herança.

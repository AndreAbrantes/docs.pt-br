|ID|2578|  
|Palavras-chave|WFActivities|  
|Nível|Aviso|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Indica que uma captura ou basicamente atividade apresentou uma exceção.  
  
## <a name="message"></a>Mensagem  
 Uma captura ou finalmente a atividade que está associada com a atividade “%1 " de TryCatch apresentou uma exceção.

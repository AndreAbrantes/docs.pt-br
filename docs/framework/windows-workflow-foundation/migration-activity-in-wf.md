# <a name="migration-activity-in-wf"></a>Atividade de migração em WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] fornece o <xref:System.Activities.Statements.Interop> atividade para executar atividades que derivam de atividade dentro de um fluxo de trabalho que se baseia no <xref:System.Activities.Activity>. Para obter mais informações, consulte o [diretrizes de migração](migration-guidance.md) seção.  
  
> [!NOTE]
>  O <xref:System.Activities.Statements.Interop> não aparecerá na caixa de ferramentas de designer de fluxo de trabalho, a menos que o projeto do fluxo de trabalho tem sua **estrutura de destino** configuração definida como **.Net Framework 4** ou superior.

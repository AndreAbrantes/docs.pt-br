# <a name="xfieldmodifier-directive"></a><span data-ttu-id="aee10-102">Diretiva x:FieldModifier</span><span class="sxs-lookup"><span data-stu-id="aee10-102">x:FieldModifier Directive</span></span>
<span data-ttu-id="aee10-103">Modifica o comportamento de compilação XAML, de modo que os campos para referências de objeto nomeado são definidos com <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> acessar em vez do <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> comportamento padrão.</span><span class="sxs-lookup"><span data-stu-id="aee10-103">Modifies XAML compilation behavior so that fields for named object references are defined with <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> access instead of the <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> default behavior.</span></span>  
  
## <a name="xaml-attribute-usage"></a><span data-ttu-id="aee10-104">Uso do Atributo XAML</span><span class="sxs-lookup"><span data-stu-id="aee10-104">XAML Attribute Usage</span></span>  
  
```xaml  
<object x:FieldModifier="Public".../>  
```

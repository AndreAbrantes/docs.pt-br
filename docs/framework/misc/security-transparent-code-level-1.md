  
 <span data-ttu-id="89fc3-184">Se você desejar misturar código transparente e crítico no mesmo assembly, inicie marcando o assembly com o <xref:System.Security.SecurityCriticalAttribute> atributo para indicar que o assembly pode conter código crítico, da seguinte maneira:</span><span class="sxs-lookup"><span data-stu-id="89fc3-184">If you want to mix critical and transparent code in the same assembly, start by marking the assembly with the <xref:System.Security.SecurityCriticalAttribute> attribute to indicate that the assembly can contain critical code, as follows:</span></span>

## <a name="using-managed-wrapper-classes"></a><span data-ttu-id="587f0-178">Usando classes de wrapper gerenciadas</span><span class="sxs-lookup"><span data-stu-id="587f0-178">Using Managed Wrapper Classes</span></span>

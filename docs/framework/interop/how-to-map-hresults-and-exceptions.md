### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="04513-114">Para criar uma nova classe de exceção e mapeá-la para um HRESULT</span><span class="sxs-lookup"><span data-stu-id="04513-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1.  <span data-ttu-id="04513-115">Use o código a seguir para criar uma nova classe de exceção chamada `NoAccessException` e mapeá-la para o HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="04513-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>  
  
    ```cpp

    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;   
    }  
    }  
    CMyClass::MethodThatThrows  
    {  
    throw new NoAccessException();  
    }  
    ```

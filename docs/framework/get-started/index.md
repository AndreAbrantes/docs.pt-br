- <span data-ttu-id="c11b7-114">Um Common Type System.</span><span class="sxs-lookup"><span data-stu-id="c11b7-114">A common type system.</span></span> <span data-ttu-id="c11b7-115">Em linguagens de programação tradicionais, os tipos básicos são definidos pelo compilador, que complica a interoperabilidade entre linguagens.</span><span class="sxs-lookup"><span data-stu-id="c11b7-115">In traditional programming languages, basic types are defined by the compiler, which complicates cross-language interoperability.</span></span> <span data-ttu-id="c11b7-116">No .NET Framework, os tipos básicos são definidos pelo sistema de tipos do .NET Framework e são comuns a todas as linguagens que segmentam o .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="c11b7-116">In the .NET Framework, basic types are defined by the .NET Framework type system and are common to all languages that target the .NET Framework.</span></span>

- <span data-ttu-id="c11b7-119">Estruturas e tecnologias de desenvolvimento.</span><span class="sxs-lookup"><span data-stu-id="c11b7-119">Development frameworks and technologies.</span></span> <span data-ttu-id="c11b7-120">O .NET Framework inclui bibliotecas para áreas específicas do desenvolvimento de aplicativos, como o ASP.NET para aplicativos Web, o ADO.NET para acesso a dados e o Windows Communication Foundation para aplicativos orientados a serviços e Windows Presentation Foundation para aplicativos de área de trabalho Windows.</span><span class="sxs-lookup"><span data-stu-id="c11b7-120">The .NET Framework includes libraries for specific areas of app development, such as ASP.NET for web apps, ADO.NET for data access, Windows Communication Foundation for service-oriented apps, and Windows Presentation Foundation for Windows desktop apps.</span></span>

- <span data-ttu-id="c11b7-126">Execução lado a lado.</span><span class="sxs-lookup"><span data-stu-id="c11b7-126">Side-by-side execution.</span></span> <span data-ttu-id="c11b7-127">O .NET Framework ajuda a resolver conflitos de versão permitindo que várias versões do CLR existam no mesmo computador.</span><span class="sxs-lookup"><span data-stu-id="c11b7-127">The .NET Framework helps resolve version conflicts by allowing multiple versions of the common language runtime to exist on the same computer.</span></span> <span data-ttu-id="c11b7-128">Isso significa que várias versões dos aplicativos podem coexistir e que um aplicativo pode ser executado na versão do .NET Framework com a qual foi compilada.</span><span class="sxs-lookup"><span data-stu-id="c11b7-128">This means that multiple versions of apps can coexist and that an app can run on the version of the .NET Framework with which it was built.</span></span> <span data-ttu-id="c11b7-129">A execução lado a lado aplica-se aos grupos de versão do .NET Framework 1.0/1.1, 2.0/3.0/3.5 e 4/4.5.x/4.6.x/4.7.x/4.8.</span><span class="sxs-lookup"><span data-stu-id="c11b7-129">Side-by-side execution applies to the .NET Framework version groups 1.0/1.1, 2.0/3.0/3.5, and 4/4.5.x/4.6.x/4.7.x/4.8.</span></span>

- <span data-ttu-id="c11b7-130">Multiplataforma.</span><span class="sxs-lookup"><span data-stu-id="c11b7-130">Multitargeting.</span></span> <span data-ttu-id="c11b7-131">Ao direcionar o [.NET Standard](../../standard/net-standard.md), desenvolvedores criam bibliotecas de classes que funcionam em várias plataformas do .NET Framework com suporte por essa versão do padrão.</span><span class="sxs-lookup"><span data-stu-id="c11b7-131">By targeting [.NET Standard](../../standard/net-standard.md), developers create class libraries that work on multiple .NET Framework platforms supported by that version of the standard.</span></span> <span data-ttu-id="c11b7-132">Por exemplo, as bibliotecas que direcionam o .NET Standard 2.0 podem ser usadas por aplicativos que direcionam .NET Framework 4.6.1, .NET Core 2.0 e UWP 10.0.16299.</span><span class="sxs-lookup"><span data-stu-id="c11b7-132">For example, libraries that target the .NET Standard 2.0 can be used by apps that target the .NET Framework 4.6.1, .NET Core 2.0, and UWP 10.0.16299.</span></span>

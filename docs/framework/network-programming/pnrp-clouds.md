# <a name="pnrp-clouds"></a><span data-ttu-id="e1a39-102">Nuvens PNRP</span><span class="sxs-lookup"><span data-stu-id="e1a39-102">PNRP Clouds</span></span>

 <span data-ttu-id="e1a39-105">A comunicação entre nós nunca deve cruzar de uma nuvem para outra.</span><span class="sxs-lookup"><span data-stu-id="e1a39-105">Communication between nodes should never cross from one cloud to another.</span></span> <span data-ttu-id="e1a39-106">Uma instância <xref:System.Net.PeerToPeer.Cloud> é identificada exclusivamente pelo seu nome, que diferencia maiúsculas de minúsculas.</span><span class="sxs-lookup"><span data-stu-id="e1a39-106">A <xref:System.Net.PeerToPeer.Cloud> instance is uniquely identified by its name, which is case-sensitive.</span></span> <span data-ttu-id="e1a39-107">Um único par ou nó pode estar conectado a mais de uma nuvem.</span><span class="sxs-lookup"><span data-stu-id="e1a39-107">A single peer or node may be connected to more than one cloud.</span></span>  
  
 <span data-ttu-id="e1a39-108">As nuvens estão muito diretamente ligadas a adaptadores de rede.</span><span class="sxs-lookup"><span data-stu-id="e1a39-108">Clouds are tied very closely to network interfaces.</span></span>  <span data-ttu-id="e1a39-109">Em um computador multihomed com duas placas de rede conectadas a diferentes sub-redes, três nuvens serão retornadas: uma para cada um dos endereços locais de link por adaptador e uma única nuvem de escopo global.</span><span class="sxs-lookup"><span data-stu-id="e1a39-109">On a multi-homed machine with two network cards attached to different subnets, three clouds will be returned: one for each of the link local addresses per interface, and a single global scope cloud.</span></span>  
  
 <span data-ttu-id="e1a39-110">O PNRP usa três "escopos" de nuvem, nos quais um escopo é um agrupamento de computadores capazes de localizar uns aos outros:</span><span class="sxs-lookup"><span data-stu-id="e1a39-110">PNRP uses three cloud "scopes", in which a scope is a grouping of computers that are able to find each other:</span></span>  
  
-   <span data-ttu-id="e1a39-111">A nuvem global corresponde ao escopo de endereço IPv6 global e endereços globais e representa todos os computadores em toda a Internet IPv6.</span><span class="sxs-lookup"><span data-stu-id="e1a39-111">The global cloud corresponds to the global IPv6 address scope and global addresses and represents all the computers on the entire IPv6 Internet.</span></span> <span data-ttu-id="e1a39-112">Há apenas uma única nuvem global.</span><span class="sxs-lookup"><span data-stu-id="e1a39-112">There is only a single global cloud.</span></span>  
  
-   <span data-ttu-id="e1a39-113">A nuvem de link local corresponde ao escopo de endereço IPv6 link local e endereços de link local.</span><span class="sxs-lookup"><span data-stu-id="e1a39-113">The link-local cloud corresponds to the link-local IPv6 address scope and link-local addresses.</span></span> <span data-ttu-id="e1a39-114">Uma nuvem de link local é para um link específico, que normalmente é o mesmo que a sub-rede conectada localmente.</span><span class="sxs-lookup"><span data-stu-id="e1a39-114">A link-local cloud is for a specific link, which is typically the same as the locally attached subnet.</span></span> <span data-ttu-id="e1a39-115">Pode haver várias nuvens de link local.</span><span class="sxs-lookup"><span data-stu-id="e1a39-115">There can be multiple link-local clouds.</span></span>  
  
 <span data-ttu-id="e1a39-116">Uma terceira nuvem, a nuvem específica do site, corresponde ao escopo de endereço IPv6 do site e a endereços de site local.</span><span class="sxs-lookup"><span data-stu-id="e1a39-116">A third cloud, the site-specific cloud, corresponds to the site IPv6 address scope and site-local addresses.</span></span> <span data-ttu-id="e1a39-117">Essa nuvem foi preterida, embora ela ainda tenha suporte em PNRP.</span><span class="sxs-lookup"><span data-stu-id="e1a39-117">This cloud has been deprecated, although it is still supported in PNRP.</span></span>  
  
## <a name="clouds"></a><span data-ttu-id="e1a39-118">Nuvens</span><span class="sxs-lookup"><span data-stu-id="e1a39-118">Clouds</span></span>

<span data-ttu-id="99fd4-120">O endereço de loopback `0:0:0:0:0:0:0:1` em formato compactado é `::`1.</span><span class="sxs-lookup"><span data-stu-id="99fd4-120">The loopback address <span data-ttu-id="99fd4-135">O IPv6 define os seguintes tipos de endereço:</span><span class="sxs-lookup"><span data-stu-id="99fd4-135">IPv6 defines the following address types:</span></span>

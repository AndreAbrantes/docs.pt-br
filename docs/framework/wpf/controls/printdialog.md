# <a name="printdialog"></a>PrintDialog
O <xref:System.Windows.Controls.PrintDialog> controle é usado para criar uma instância de uma caixa de diálogo de impressão padrão que configura automaticamente um <xref:System.Printing.PrintTicket> e <xref:System.Printing.PrintQueue> de acordo com a entrada do usuário.

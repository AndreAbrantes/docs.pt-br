# <a name="printdialog"></a>PrintDialog
O <xref:System.Windows.Controls.PrintDialog> controle é usado para instanciar uma caixa de diálogo de impressão padrão que configura automaticamente uma <xref:System.Printing.PrintTicket> e <xref:System.Printing.PrintQueue> de acordo com a entrada do usuário.

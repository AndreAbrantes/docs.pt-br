# <a name="button"></a>Botão
Um <xref:System.Windows.Controls.Button> controle reaja à entrada do usuário de um mouse, teclado, caneta, ou outro dispositivo de entrada e gera um <xref:System.Windows.Controls.Primitives.ButtonBase.Click> eventos. Um <xref:System.Windows.Controls.Button> é um basic [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] componente que pode conter conteúdo simples, como texto e também pode conter conteúdo complexo, como imagens e <xref:System.Windows.Controls.Panel> controles.

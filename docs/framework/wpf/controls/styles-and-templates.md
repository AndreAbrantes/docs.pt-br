# <a name="styles-and-templates"></a>Estilos e modelos
Os estilos e modelos de Windows Presentation Foundation (WPF) referem-se a um conjunto de recursos (estilos, modelos, gatilhos e storyboards) que permitem que um aplicativo, um documento ou um designer de interface do usuário crie aplicativos visualmente atraentes e para padronizar em uma aparência específica para o produto.

# <a name="windows-presentation-foundation"></a>Windows Presentation Foundation
O WPF (Windows Presentation Foundation) no Visual Studio 2015 fornece aos desenvolvedores um modelo de programação unificado para a criação de modernos aplicativos de área de trabalho de linha de negócios no Windows.  
  
> [!NOTE]
>  O navegador Mozilla Firefox desabilita a extensão do Microsoft Framework Assistant. Confira [Como remover o .NET Framework Assistant para Firefox](https://support.microsoft.com/en-us/kb/963707) para obter informações sobre como remover a extensão do Firefox.

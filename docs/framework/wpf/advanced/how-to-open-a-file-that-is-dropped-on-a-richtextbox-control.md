# <a name="how-to-open-a-file-that-is-dropped-on-a-richtextbox-control"></a><span data-ttu-id="48117-102">Como: Abrir um arquivo solto em um controle RichTextBox</span><span class="sxs-lookup"><span data-stu-id="48117-102">How to: Open a File That is Dropped on a RichTextBox Control</span></span>
<span data-ttu-id="48117-103">Na [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)], o <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, e <xref:System.Windows.Documents.FlowDocument> todos os controles têm funcionalidade interna de arrastar e soltar.</span><span class="sxs-lookup"><span data-stu-id="48117-103">In [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)], the <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, and <xref:System.Windows.Documents.FlowDocument> controls all have built-in drag-and-drop functionality.</span></span> <span data-ttu-id="48117-104">A funcionalidade interna permite arrastar e soltar o texto dentro e entre os controles.</span><span class="sxs-lookup"><span data-stu-id="48117-104">The built-in functionality enables drag-and-drop of text within and between the controls.</span></span> <span data-ttu-id="48117-105">No entanto, ela não habilita a abertura de um arquivo soltando-o no controle.</span><span class="sxs-lookup"><span data-stu-id="48117-105">However, it does not enable opening a file by dropping the file on the control.</span></span> <span data-ttu-id="48117-106">Esses controles também marcam os eventos do tipo "arrastar e soltar" como manipulados.</span><span class="sxs-lookup"><span data-stu-id="48117-106">These controls also mark the drag-and-drop events as handled.</span></span> <span data-ttu-id="48117-107">Como resultado, por padrão, não é possível adicionar seus próprios manipuladores de eventos para fornecer funcionalidade para abrir os arquivos soltos.</span><span class="sxs-lookup"><span data-stu-id="48117-107">As a result, by default, you cannot add your own event handlers to provide functionality to open dropped files.</span></span>  
  
 <span data-ttu-id="48117-108">Para adicionar o tratamento adicional de eventos de arrastar e soltar nesses controles, use o <xref:System.Windows.UIElement.AddHandler%28System.Windows.RoutedEvent%2CSystem.Delegate%2CSystem.Boolean%29> método para adicionar seus manipuladores de eventos para os eventos de arrastar e soltar.</span><span class="sxs-lookup"><span data-stu-id="48117-108">To add additional handling for drag-and-drop events in these controls, use the <xref:System.Windows.UIElement.AddHandler%28System.Windows.RoutedEvent%2CSystem.Delegate%2CSystem.Boolean%29> method to add your event handlers for the drag-and-drop events.</span></span> <span data-ttu-id="48117-109">Defina o parâmetro `handledEventsToo` para `true` para que o manipulador especificado seja invocado para um evento roteados que já tenham sido marcado como manipulados por outro elemento na rota de evento.</span><span class="sxs-lookup"><span data-stu-id="48117-109">Set the `handledEventsToo` parameter to `true` to have the specified handler be invoked for a routed event that has already been marked as handled by another element along the event route.</span></span>  
  
> [!TIP]
>  <span data-ttu-id="48117-110">Você pode substituir a funcionalidade interna de arrastar e soltar dos <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, e <xref:System.Windows.Documents.FlowDocument> manipulando as versões prévias dos eventos de arrastar e soltar e marcando os eventos de visualização como manipulados.</span><span class="sxs-lookup"><span data-stu-id="48117-110">You can replace the built-in drag-and-drop functionality of <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, and <xref:System.Windows.Documents.FlowDocument> by handling the preview versions of the drag-and-drop events and marking the preview events as handled.</span></span> <span data-ttu-id="48117-111">No entanto, isso desabilitará a funcionalidade interna do tipo "arrastar e soltar", o que não é recomendado.</span><span class="sxs-lookup"><span data-stu-id="48117-111">However, this will disable the built-in drag-and-drop functionality, and is not recommended.</span></span>

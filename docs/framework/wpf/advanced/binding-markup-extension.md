# <a name="binding-markup-extension"></a><span data-ttu-id="18add-102">Extensão de marcação de associação</span><span class="sxs-lookup"><span data-stu-id="18add-102">Binding Markup Extension</span></span>
<span data-ttu-id="18add-103">Adia um valor da propriedade para ser um valor de associação de dados, criando um objeto de expressão intermediário e interpretando o contexto de dados que se aplica ao elemento e à sua associação em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="18add-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="18add-104">Uso de expressões de associação</span><span class="sxs-lookup"><span data-stu-id="18add-104">Binding Expression Usage</span></span>  
  
```xaml  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```

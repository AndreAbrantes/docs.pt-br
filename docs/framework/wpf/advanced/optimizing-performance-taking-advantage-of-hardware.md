- <span data-ttu-id="3bc24-139">**Camada de Renderização 2** A maioria dos recursos gráficos usa aceleração de hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="3bc24-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="3bc24-140">O nível de versão [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] é maior ou igual à versão 9.0.</span><span class="sxs-lookup"><span data-stu-id="3bc24-140">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 9.0.</span></span>

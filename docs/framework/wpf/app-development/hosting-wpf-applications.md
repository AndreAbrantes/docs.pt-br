# <a name="hosting-wpf-applications"></a>Hospedando aplicativos do WPF
Os aplicativos de navegador XAML do WPF (XBAPs) são aplicativos cliente avançados que podem ser implantados em um servidor Web e iniciados em um navegador. O host do WPF (PresentationHost. exe) é registrado como o Shell e o manipulador de MIME para arquivos XBAP e XAML. Portanto, o Internet Explorer sabe iniciar o host do WPF quando um XBAP é iniciado. Os usuários do Firefox podem instalar complementos do Firefox que habilitam o Firefox a hospedar XBAPs também. Um XBAP pode ser hospedado em outros navegadores ou aplicativos autônomos usando as APIs de hospedagem do navegador nativo fornecidas pelo WPF.

# <a name="windows-in-wpf-applications"></a>Windows em aplicativos WPF
Os usuários interagem com aplicativos por meio do Windows. A finalidade fundamental de uma janela é hospedar e exibir conteúdo. O tipo de conteúdo que uma janela hospeda depende do tipo de dados que um aplicativo opera, o que pode incluir páginas de mídia, Extensible Application Markup Language (XAML), páginas da Web, documentos, tabelas e registros de banco de dados e informações do sistema.

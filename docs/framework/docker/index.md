# <a name="deploying-net-framework-applications-with-docker"></a>Implantação de aplicativos .NET Framework com Docker

Você pode implantar um aplicativo .NET Framework com o Docker usando contêineres do Windows. Você pode saber sobre os requisitos de uso dos [Contêineres do Windows](/virtualization/windowscontainers/about/) e sobre a [Introdução ao Docker para Windows](https://docs.docker.com/docker-for-windows/). 

Você pode começar [executando um aplicativo de console com Docker](console.md).
Para executar aplicativos da Web no Docker, você pode ler [Aplicativos ASP.NET MVC no Docker](/aspnet/mvc/overview/deployment/docker-aspnetmvc).

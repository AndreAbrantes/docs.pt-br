## <a name="login-types"></a><span data-ttu-id="3f059-128">Tipos de logon</span><span class="sxs-lookup"><span data-stu-id="3f059-128">Login Types</span></span>  
 <span data-ttu-id="3f059-129">SQL Server dá suporte a três tipos de logons:</span><span class="sxs-lookup"><span data-stu-id="3f059-129">SQL Server supports three types of logins:</span></span>  
  
-   <span data-ttu-id="3f059-130">Uma conta de usuário local do Windows ou uma conta de domínio confiável.</span><span class="sxs-lookup"><span data-stu-id="3f059-130">A local Windows user account or trusted domain account.</span></span> <span data-ttu-id="3f059-131">SQL Server se baseia em Windows para autenticar as contas de usuário do Windows.</span><span class="sxs-lookup"><span data-stu-id="3f059-131">SQL Server relies on Windows to authenticate the Windows user accounts.</span></span>

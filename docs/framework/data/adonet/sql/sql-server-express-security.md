## <a name="user-instances"></a><span data-ttu-id="abf97-114">Instâncias de usuário</span><span class="sxs-lookup"><span data-stu-id="abf97-114">User Instances</span></span>

 <span data-ttu-id="abf97-118">Uma instância de usuário é gerada de uma instância primária do SQL Server ou do SQL Server Express em nome de um usuário.</span><span class="sxs-lookup"><span data-stu-id="abf97-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="abf97-119">Ela é executada como um processo de usuário no contexto de segurança do Windows do usuário, não como um serviço.</span><span class="sxs-lookup"><span data-stu-id="abf97-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="abf97-120">Os logons do SQL Server não são permitidos; somente logons do Windows têm suporte.</span><span class="sxs-lookup"><span data-stu-id="abf97-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="abf97-121">Isso impede o software que está sendo executado em uma instância de usuário de fazer alterações no sistema que o usuário não teria permissões para fazer.</span><span class="sxs-lookup"><span data-stu-id="abf97-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="abf97-122">Uma instância de usuário também é conhecida como instância filha ou cliente, e às vezes é conhecida pela utilização do acrônimo de RANU (“executar como usuário normal”).</span><span class="sxs-lookup"><span data-stu-id="abf97-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>

## <a name="user-instances"></a><span data-ttu-id="64f13-114">Instâncias de usuário</span><span class="sxs-lookup"><span data-stu-id="64f13-114">User Instances</span></span>  

 <span data-ttu-id="64f13-115">Uma instância de usuário é uma instância separada do mecanismo de banco de dados do SQL Server Express gerada por uma instância pai do SQL Server Express.</span><span class="sxs-lookup"><span data-stu-id="64f13-115">A user instance is a separate instance of the SQL Server Express database engine that is generated by a parent instance of SQL Server Express.</span></span> <span data-ttu-id="64f13-116">A meta principal de uma instância de usuário é permitir que usuários que estão executando o Windows em uma conta de usuário com privilégios mínimos tenham privilégios de administrador do sistema (`sysadmin`) na instância do SQL Server Express no computador local deles.</span><span class="sxs-lookup"><span data-stu-id="64f13-116">The primary goal of a user instance is to allow users who are running Windows under a least-privilege user account to have system administrator (`sysadmin`) privileges on the SQL Server Express instance on their local computer.</span></span> <span data-ttu-id="64f13-117">As instâncias de usuário não são destinadas a usuários que são administradores de sistema nos próprios computadores deles.</span><span class="sxs-lookup"><span data-stu-id="64f13-117">User instances are not intended for users who are system administrators on their own computers.</span></span>  
  
 <span data-ttu-id="64f13-118">Uma instância de usuário é gerada com base em uma instância primária do SQL Server ou do SQL Server Express em nome de um usuário.</span><span class="sxs-lookup"><span data-stu-id="64f13-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="64f13-119">Ela é executada como um processo de usuário no contexto de segurança do usuário do Windows, não como um serviço.</span><span class="sxs-lookup"><span data-stu-id="64f13-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="64f13-120">Os logons do SQL Server não são permitidos; há suporte somente para logons do Windows.</span><span class="sxs-lookup"><span data-stu-id="64f13-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="64f13-121">Isso impede que o software em execução em uma instância de usuário faça alterações em todo o sistema que o usuário não teria permissões para fazer.</span><span class="sxs-lookup"><span data-stu-id="64f13-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="64f13-122">Uma instância de usuário também é conhecida como uma instância filho ou de cliente e, às vezes, é denominada usando o acrônimo RANU ("executar como usuário normal").</span><span class="sxs-lookup"><span data-stu-id="64f13-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>  
  
 <span data-ttu-id="64f13-123">Cada instância de usuário é isolada de sua instância pai e de outra instância de usuário que esteja em execução no mesmo computador.</span><span class="sxs-lookup"><span data-stu-id="64f13-123">Each user instance is isolated from its parent instance and from other user instances running on the same computer.</span></span> <span data-ttu-id="64f13-124">Os bancos de dados instalados em instâncias de usuário são abertos somente no modo de usuário único; vários usuários não podem se conectar a eles.</span><span class="sxs-lookup"><span data-stu-id="64f13-124">Databases installed on user instances are opened in single-user mode only; multiple users cannot connect to them.</span></span> <span data-ttu-id="64f13-125">A replicação, consultas distribuídas e conexões remotas estão desabilitadas para instâncias de usuário.</span><span class="sxs-lookup"><span data-stu-id="64f13-125">Replication, distributed queries and remote connections are disabled for user instances.</span></span> <span data-ttu-id="64f13-126">Quando conectados a uma instância de usuário, os usuários não têm nenhum privilégio especial na instância pai do SQL Server Express.</span><span class="sxs-lookup"><span data-stu-id="64f13-126">When connected to a user instance, users do not have any special privileges on the parent SQL Server Express instance.</span></span>

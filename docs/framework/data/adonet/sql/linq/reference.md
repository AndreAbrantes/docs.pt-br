Esta seção fornece informações de referência para desenvolvedores do [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)].  
  
 Você também é encorajado a procurar o Microsoft Docs por questões específicas e, especialmente, a participar do [Fórum LINQ,](https://social.msdn.microsoft.com/forums/en-us/home?forum=linqtosql)onde você pode discutir tópicos mais complexos em detalhes com especialistas. Além disso, você pode estudar um white paper detalhando a tecnologia do [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)], completo com os exemplos de código do Visual Basic e de C#. Para obter mais informações, consulte [LINQ para SQL: .NET Linguagem-Integrated Query for Relational Data](https://docs.microsoft.com/previous-versions/dotnet/articles/bb425822(v=msdn.10)).

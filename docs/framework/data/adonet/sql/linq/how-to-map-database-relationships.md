 <span data-ttu-id="fe8f6-112">A maioria da relações são um-para-muitos, como no exemplo mais adiante neste tópico.</span><span class="sxs-lookup"><span data-stu-id="fe8f6-112">Most relationships are one-to-many, as in the example later in this topic.</span></span> <span data-ttu-id="fe8f6-113">Você também pode representar relações um-para-um e muitos-para-muitos da seguinte maneira:</span><span class="sxs-lookup"><span data-stu-id="fe8f6-113">You can also represent one-to-one and many-to-many relationships as follows:</span></span>

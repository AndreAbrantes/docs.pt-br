### <a name="schema-owners-and-permissions"></a><span data-ttu-id="f8cff-110">Proprietários e permissões do esquema</span><span class="sxs-lookup"><span data-stu-id="f8cff-110">Schema Owners and Permissions</span></span>

### <a name="built-in-schemas"></a><span data-ttu-id="f8cff-120">Esquemas internos</span><span class="sxs-lookup"><span data-stu-id="f8cff-120">Built-In Schemas</span></span>

-   `guest`  
  
-   `sys`

 <span data-ttu-id="f8cff-125">Se você removê-los do banco de dados modelo, eles não aparecerão em novos bancos de dados.</span><span class="sxs-lookup"><span data-stu-id="f8cff-125">If you drop them from the model database, they will not appear in new databases.</span></span>

#### <a name="the-dbo-schema"></a><span data-ttu-id="f8cff-128">O esquema dbo</span><span class="sxs-lookup"><span data-stu-id="f8cff-128">The dbo Schema</span></span>

 <span data-ttu-id="f8cff-132">Os usuários que são atribuídos ao esquema `dbo` não herdam as permissões da conta de usuário do `dbo`.</span><span class="sxs-lookup"><span data-stu-id="f8cff-132">Users who are assigned the `dbo` schema do not inherit the permissions of the `dbo` user account.</span></span> <span data-ttu-id="f8cff-133">Nenhuma permissão é herdada de um esquema por usuários; as permissões de esquema são herdadas pelos objetos de banco de dados contidos no esquema.</span><span class="sxs-lookup"><span data-stu-id="f8cff-133">No permissions are inherited from a schema by users; schema permissions are inherited by the database objects contained in the schema.</span></span>

# <a name="typed-datasets"></a><span data-ttu-id="4bb66-102">DataSets tipados</span><span class="sxs-lookup"><span data-stu-id="4bb66-102">Typed DataSets</span></span>
<span data-ttu-id="4bb66-103">Juntamente com acesso de associação tardia por meio de valores com variáveis fracamente tipadas, o <xref:System.Data.DataSet> fornece acesso a dados por meio de uma metáfora fortemente tipada.</span><span class="sxs-lookup"><span data-stu-id="4bb66-103">Along with late bound access to values through weakly typed variables, the <xref:System.Data.DataSet> provides access to data through a strongly typed metaphor.</span></span> <span data-ttu-id="4bb66-104">Tabelas e colunas que fazem parte do **conjunto de dados** podem ser acessadas usando nomes amigáveis e variáveis fortemente tipadas.</span><span class="sxs-lookup"><span data-stu-id="4bb66-104">Tables and columns that are part of the **DataSet** can be accessed using user-friendly names and strongly typed variables.</span></span>  
  
 <span data-ttu-id="4bb66-105">Tipado **DataSet** é uma classe que deriva de uma **conjunto de dados**.</span><span class="sxs-lookup"><span data-stu-id="4bb66-105">A typed **DataSet** is a class that derives from a **DataSet**.</span></span> <span data-ttu-id="4bb66-106">Como tal, herda todos os eventos, métodos e propriedades de um **conjunto de dados**.</span><span class="sxs-lookup"><span data-stu-id="4bb66-106">As such, it inherits all the methods, events, and properties of a **DataSet**.</span></span> <span data-ttu-id="4bb66-107">Além disso, um tipado **conjunto de dados** fornece propriedades, eventos e métodos com rigidez de tipos.</span><span class="sxs-lookup"><span data-stu-id="4bb66-107">Additionally, a typed **DataSet** provides strongly typed methods, events, and properties.</span></span> <span data-ttu-id="4bb66-108">Isso significa que você pode acessar tabelas e colunas pelo nome, em vez de usar métodos baseados em coleção.</span><span class="sxs-lookup"><span data-stu-id="4bb66-108">This means you can access tables and columns by name, instead of using collection-based methods.</span></span> <span data-ttu-id="4bb66-109">Além da leitura aprimorada do código, com um tipo **conjunto de dados** também permite que o código do Visual Studio .NET editor preencher linhas automaticamente conforme você digita.</span><span class="sxs-lookup"><span data-stu-id="4bb66-109">Aside from the improved readability of the code, a typed **DataSet** also allows the Visual Studio .NET code editor to automatically complete lines as you type.</span></span>  
  
 <span data-ttu-id="4bb66-110">Além disso, fortemente tipada **conjunto de dados** fornece acesso aos valores como o tipo correto no tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="4bb66-110">Additionally, the strongly typed **DataSet** provides access to values as the correct type at compile time.</span></span> <span data-ttu-id="4bb66-111">Com fortemente tipado **conjunto de dados**, erros de incompatibilidade de tipo são capturados quando o código é compilado em vez de no tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="4bb66-111">With a strongly typed **DataSet**, type mismatch errors are caught when the code is compiled rather than at run time.</span></span>

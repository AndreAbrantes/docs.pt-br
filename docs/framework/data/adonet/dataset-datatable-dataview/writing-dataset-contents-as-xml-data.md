# <a name="writing-dataset-contents-as-xml-data"></a><span data-ttu-id="f938e-102">Gravando o conteúdo do DataSet como dados XML</span><span class="sxs-lookup"><span data-stu-id="f938e-102">Writing DataSet Contents as XML Data</span></span>

 <span data-ttu-id="f938e-106">Quando um <xref:System.Data.DataSet> é gravado como dados XML, as linhas do <xref:System.Data.DataSet> são gravadas nas suas versões atuais.</span><span class="sxs-lookup"><span data-stu-id="f938e-106">When a <xref:System.Data.DataSet> is written as XML data, the rows in the <xref:System.Data.DataSet> are written in their current versions.</span></span> <span data-ttu-id="f938e-107">No entanto, o <xref:System.Data.DataSet> também pode ser gravado como um DiffGram de modo que os valores atuais e originais das linhas sejam incluídos.</span><span class="sxs-lookup"><span data-stu-id="f938e-107">However, the <xref:System.Data.DataSet> can also be written as a DiffGram so that both the current and the original values of the rows will be included.</span></span>  
  
 <span data-ttu-id="f938e-108">A representação XML do <xref:System.Data.DataSet> podem ser gravados em um arquivo, um fluxo, uma **XmlWriter**, ou uma cadeia de caracteres.</span><span class="sxs-lookup"><span data-stu-id="f938e-108">The XML representation of the <xref:System.Data.DataSet> can be written to a file, a stream, an **XmlWriter**, or a string.</span></span> <span data-ttu-id="f938e-109">Essas opções proporcionam excelente flexibilidade para o transporte da representação XML do <xref:System.Data.DataSet>.</span><span class="sxs-lookup"><span data-stu-id="f938e-109">These choices provide great flexibility for how you transport the XML representation of the <xref:System.Data.DataSet>.</span></span> <span data-ttu-id="f938e-110">Para obter a representação XML do <xref:System.Data.DataSet> como uma cadeia de caracteres, use o **GetXml** método conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="f938e-110">To obtain the XML representation of the <xref:System.Data.DataSet> as a string, use the **GetXml** method as shown in the following example.</span></span>  
  
```vb

```  
  
```csharp  
string xmlDS = custDS.GetXml();
```  

 <span data-ttu-id="f938e-111">**GetXml** retorna a representação XML do <xref:System.Data.DataSet> sem informações de esquema.</span><span class="sxs-lookup"><span data-stu-id="f938e-111">**GetXml** returns the XML representation of the <xref:System.Data.DataSet> without schema information.</span></span> <span data-ttu-id="f938e-112">Para gravar as informações de esquema de <xref:System.Data.DataSet> (como um esquema XML) em uma cadeia de caracteres, use **GetXmlSchema**.</span><span class="sxs-lookup"><span data-stu-id="f938e-112">To write the schema information from the <xref:System.Data.DataSet> (as XML Schema) to a string, use **GetXmlSchema**.</span></span>

 <span data-ttu-id="f938e-117">A tabela a seguir mostra as opções para **XmlWriteMode**.</span><span class="sxs-lookup"><span data-stu-id="f938e-117">The following table shows the options for **XmlWriteMode**.</span></span>  
  
|<span data-ttu-id="f938e-118">Opção de XmlWriteMode</span><span class="sxs-lookup"><span data-stu-id="f938e-118">XmlWriteMode option</span></span>|<span data-ttu-id="f938e-119">Descrição</span><span class="sxs-lookup"><span data-stu-id="f938e-119">Description</span></span>|

|<span data-ttu-id="f938e-123">**WriteSchema**</span><span class="sxs-lookup"><span data-stu-id="f938e-123">**WriteSchema**</span></span>|<span data-ttu-id="f938e-124">Grava o conteúdo atual do <xref:System.Data.DataSet> como dados XML com a estrutura relacional como um esquema XML embutido.</span><span class="sxs-lookup"><span data-stu-id="f938e-124">Writes the current contents of the <xref:System.Data.DataSet> as XML data with the relational structure as inline XML Schema.</span></span>|

 <span data-ttu-id="f938e-128">Ao escrever uma representação XML de um <xref:System.Data.DataSet> que contém **DataRelation** objetos, provavelmente você desejará que o XML resultante para que as linhas filho de cada relação aninhada dentro de seus elementos pai relacionado.</span><span class="sxs-lookup"><span data-stu-id="f938e-128">When writing an XML representation of a <xref:System.Data.DataSet> that contains **DataRelation** objects, you will most likely want the resulting XML to have the child rows of each relation nested within their related parent elements.</span></span> <span data-ttu-id="f938e-129">Para fazer isso, defina o **aninhadas** propriedade o **DataRelation** para **true** quando você adiciona o **DataRelation** para o <xref:System.Data.DataSet>.</span><span class="sxs-lookup"><span data-stu-id="f938e-129">To accomplish this, set the **Nested** property of the **DataRelation** to **true** when you add the **DataRelation** to the <xref:System.Data.DataSet>.</span></span> <span data-ttu-id="f938e-130">Para obter mais informações, consulte [aninhamento DataRelations](../../../../../docs/framework/data/adonet/dataset-datatable-dataview/nesting-datarelations.md).</span><span class="sxs-lookup"><span data-stu-id="f938e-130">For more information, see [Nesting DataRelations](../../../../../docs/framework/data/adonet/dataset-datatable-dataview/nesting-datarelations.md).</span></span>

```csharp  
custDS.WriteXml("Customers.xml", XmlWriteMode.WriteSchema);  
```  
  


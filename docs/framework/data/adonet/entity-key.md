# <a name="entity-key"></a><span data-ttu-id="6ffb6-102">chave de entidade</span><span class="sxs-lookup"><span data-stu-id="6ffb6-102">entity key</span></span>
<span data-ttu-id="6ffb6-103">Um *chave da entidade* é um [propriedade](../../../../docs/framework/data/adonet/property.md) ou um conjunto de propriedades de um [tipo de entidade](../../../../docs/framework/data/adonet/entity-type.md) que são usadas para determinar a identidade.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="6ffb6-104">As propriedades que compõem uma chave de entidade são escolhidas em tempo de design.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="6ffb6-105">Os valores das propriedades de chave de entidade devem identificar exclusivamente uma instância do tipo de entidade dentro de um [conjunto de entidades](../../../../docs/framework/data/adonet/entity-set.md) em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="6ffb6-106">As propriedades que compõem uma chave de entidade devem ser escolhidas para garantir a exclusividade de instâncias em um conjunto de entidades.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="6ffb6-107">A seguir estão os requisitos para um conjunto de propriedades ser uma chave de entidade:</span><span class="sxs-lookup"><span data-stu-id="6ffb6-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="6ffb6-108">Nenhuma chave de duas entidades em um conjunto de entidades pode ser idêntica.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="6ffb6-109">Ou seja, para as duas entidades em um conjunto de entidades, os valores para todas as propriedades que constituem uma chave não podem ser os mesmos.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="6ffb6-110">No entanto, alguns (mas não todos os valores) que compõem uma chave de entidade podem ser os mesmos.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="6ffb6-111">Uma chave de entidade deve consistir em um conjunto de não anuláveis, imutável, [propriedades de tipo primitivo](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="6ffb6-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="6ffb6-112">As propriedades que compõem uma chave de entidade para um tipo de dado entidade não pode ser alterado.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="6ffb6-113">Você não pode permitir mais de uma chave possível de entidade para um tipo de dado; entidade as chaves substitutas não são suportadas.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="6ffb6-114">Quando uma entidade é empacotada em uma hierarquia de herança, a entidade raiz deve conter todas as propriedades que compõem a chave de entidade, e a chave de entidade deve ser definida no tipo de entidade raiz.</span><span class="sxs-lookup"><span data-stu-id="6ffb6-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="6ffb6-115">Para obter mais informações, consulte [modelo de dados de entidade: herança](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="6ffb6-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>

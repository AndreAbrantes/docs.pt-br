### <a name="the-connectionstrings-section"></a><span data-ttu-id="5c4ad-111">A seção de connectionStrings</span><span class="sxs-lookup"><span data-stu-id="5c4ad-111">The connectionStrings Section</span></span>

```xml  
  <configuration>  
    <connectionStrings>  
      <clear />  
      <add name="Name"

       connectionString="Valid Connection String;" />  
    </connectionStrings>  
  </configuration>  
```

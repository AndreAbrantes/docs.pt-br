# <a name="oracle-ref-cursors"></a><span data-ttu-id="4239e-102">REF CURSORs do Oracle</span><span class="sxs-lookup"><span data-stu-id="4239e-102">Oracle REF CURSORs</span></span>

-   <span data-ttu-id="4239e-106">Por motivos de desempenho, o provedor de dados para Oracle não associa automaticamente **REF CURSOR** tipos de dados, como faz o MSDAORA, a menos que você especifique explicitamente.</span><span class="sxs-lookup"><span data-stu-id="4239e-106">For performance reasons, the Data Provider for Oracle does not automatically bind **REF CURSOR** data types, as MSDAORA does, unless you explicitly specify them.</span></span>

-   <span data-ttu-id="4239e-111">A obtenção de <xref:System.Data.OracleClient.OracleDataReader> do valor do parâmetro não é suportada.</span><span class="sxs-lookup"><span data-stu-id="4239e-111">Obtaining an <xref:System.Data.OracleClient.OracleDataReader> from the parameter value is not supported.</span></span> <span data-ttu-id="4239e-112">Os valores são do tipo <xref:System.DBNull> após a execução do comando.</span><span class="sxs-lookup"><span data-stu-id="4239e-112">The values are of type <xref:System.DBNull> after command execution.</span></span>  
  
-   <span data-ttu-id="4239e-113">As únicas **CommandBehavior** valor de enumeração que funciona com REF CURSORs (por exemplo, ao chamar <xref:System.Data.OracleClient.OracleCommand.ExecuteReader%2A>) é **CloseConnection**; todos os outros são ignorados.</span><span class="sxs-lookup"><span data-stu-id="4239e-113">The only **CommandBehavior** enumeration value that works with REF CURSORs (for example, when calling <xref:System.Data.OracleClient.OracleCommand.ExecuteReader%2A>) is **CloseConnection**; all others are ignored.</span></span>  
  
-   <span data-ttu-id="4239e-114">A ordem de REF CURSORs na **OracleDataReader** depende da ordem dos parâmetros na **OracleParameterCollection**.</span><span class="sxs-lookup"><span data-stu-id="4239e-114">The order of REF CURSORs in the **OracleDataReader** depends on the order of the parameters in the **OracleParameterCollection**.</span></span> <span data-ttu-id="4239e-115">A propriedade <xref:System.Data.OracleClient.OracleParameter.ParameterName%2A> é ignorada.</span><span class="sxs-lookup"><span data-stu-id="4239e-115">The <xref:System.Data.OracleClient.OracleParameter.ParameterName%2A> property is ignored.</span></span>  
  
-   <span data-ttu-id="4239e-116">PL/SQL **tabela** não há suporte para o tipo de dados.</span><span class="sxs-lookup"><span data-stu-id="4239e-116">The PL/SQL **TABLE** data type is not supported.</span></span> <span data-ttu-id="4239e-117">No entanto, REF CURSORs são mais eficientes.</span><span class="sxs-lookup"><span data-stu-id="4239e-117">However, REF CURSORs are more efficient.</span></span> <span data-ttu-id="4239e-118">Se você precisar usar um **tabela** tipo de dados, use o provedor OLE DB .NET dados com o MSDAORA.</span><span class="sxs-lookup"><span data-stu-id="4239e-118">If you must use a **TABLE** data type, use the OLE DB .NET Data Provider with MSDAORA.</span></span>

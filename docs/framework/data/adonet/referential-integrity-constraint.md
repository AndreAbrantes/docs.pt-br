<span data-ttu-id="993e5-103">Um *restrição de integridade referencial* no modelo de dados de entidade (EDM) é semelhante a uma restrição de integridade referencial em um banco de dados relacional.</span><span class="sxs-lookup"><span data-stu-id="993e5-103">A *referential integrity constraint* in the Entity Data Model (EDM) is similar to a referential integrity constraint in a relational database.</span></span> <span data-ttu-id="993e5-104">Da mesma forma que uma coluna (ou colunas) de uma tabela de banco de dados podem fazer referência a chave primária de outra tabela, uma [propriedade](../../../../docs/framework/data/adonet/property.md) (ou propriedades) de um [tipo de entidade](../../../../docs/framework/data/adonet/entity-type.md) pode fazer referência a [chave de entidade ](../../../../docs/framework/data/adonet/entity-key.md) de outro tipo de entidade.</span><span class="sxs-lookup"><span data-stu-id="993e5-104">In the same way that a column (or columns) from a database table can reference the primary key of another table, a [property](../../../../docs/framework/data/adonet/property.md) (or properties) of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) can reference the [entity key](../../../../docs/framework/data/adonet/entity-key.md) of another entity type.</span></span> <span data-ttu-id="993e5-105">O tipo de entidade que é referenciado é chamado de *final principal* da restrição.</span><span class="sxs-lookup"><span data-stu-id="993e5-105">The entity type that is referenced is called the *principal end* of the constraint.</span></span> <span data-ttu-id="993e5-106">O tipo de entidade que faz referência a extremidade de entidade é chamado de *final dependente* da restrição.</span><span class="sxs-lookup"><span data-stu-id="993e5-106">The entity type that references the principal end is called the *dependent end* of the constraint.</span></span>

-   <span data-ttu-id="993e5-109">O final principal de restrição.</span><span class="sxs-lookup"><span data-stu-id="993e5-109">The principal end of the constraint.</span></span> <span data-ttu-id="993e5-110">(Um tipo de entidade cuja chave de entidade é referenciada pela o final dependente.)</span><span class="sxs-lookup"><span data-stu-id="993e5-110">(An entity type whose entity key is referenced by the dependent end.)</span></span>  
  
-   <span data-ttu-id="993e5-111">A chave de entidade de extremidade principal.</span><span class="sxs-lookup"><span data-stu-id="993e5-111">The entity key of the principal end.</span></span>  
  
-   <span data-ttu-id="993e5-112">O final dependente de restrição.</span><span class="sxs-lookup"><span data-stu-id="993e5-112">The dependent end of the constraint.</span></span> <span data-ttu-id="993e5-113">(Um tipo de entidade que tem uma propriedade ou um propriedades que referenciem a chave de entidade de extremidade principal.)</span><span class="sxs-lookup"><span data-stu-id="993e5-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>  
  
-   <span data-ttu-id="993e5-114">A propriedade ou propriedades referência de extremidade dependente.</span><span class="sxs-lookup"><span data-stu-id="993e5-114">The referencing property or properties of the dependent end.</span></span>  
  
 <span data-ttu-id="993e5-115">O objetivo de restrições de integridade referencial em EDM é garantir que as associações válidos sempre existe.</span><span class="sxs-lookup"><span data-stu-id="993e5-115">The purpose of referential integrity constraints in the EDM is to ensure that valid associations always exist.</span></span> <span data-ttu-id="993e5-116">Para obter mais informações, consulte [propriedade de chave estrangeira](../../../../docs/framework/data/adonet/foreign-key-property.md).</span><span class="sxs-lookup"><span data-stu-id="993e5-116">For more information, see [foreign key property](../../../../docs/framework/data/adonet/foreign-key-property.md).</span></span>

# <a name="referential-integrity-constraint"></a><span data-ttu-id="5f205-103">restrição de integridade referencial</span><span class="sxs-lookup"><span data-stu-id="5f205-103">referential integrity constraint</span></span>
<span data-ttu-id="5f205-104">Uma *restrição de integridade referencial* no modelo de dados de entidade (EDM) é semelhante a uma restrição de integridade referencial em um banco de dados relacional.</span><span class="sxs-lookup"><span data-stu-id="5f205-104">A *referential integrity constraint* in the Entity Data Model (EDM) is similar to a referential integrity constraint in a relational database.</span></span> <span data-ttu-id="5f205-105">Da mesma forma que uma coluna (ou colunas) de uma tabela de banco de dados pode referenciar a chave primária de outra tabela, uma [Propriedade](property.md) (ou Propriedades) de um [tipo de entidade](entity-type.md) pode referenciar a [chave de entidade](entity-key.md) de outro tipo de entidade.</span><span class="sxs-lookup"><span data-stu-id="5f205-105">In the same way that a column (or columns) from a database table can reference the primary key of another table, a [property](property.md) (or properties) of an [entity type](entity-type.md) can reference the [entity key](entity-key.md) of another entity type.</span></span> <span data-ttu-id="5f205-106">O tipo de entidade referenciado é chamado de *extremidade principal* da restrição.</span><span class="sxs-lookup"><span data-stu-id="5f205-106">The entity type that is referenced is called the *principal end* of the constraint.</span></span> <span data-ttu-id="5f205-107">O tipo de entidade que faz referência à extremidade principal é chamado de *extremidade dependente* da restrição.</span><span class="sxs-lookup"><span data-stu-id="5f205-107">The entity type that references the principal end is called the *dependent end* of the constraint.</span></span>

- <span data-ttu-id="5f205-110">O final principal de restrição.</span><span class="sxs-lookup"><span data-stu-id="5f205-110">The principal end of the constraint.</span></span> <span data-ttu-id="5f205-111">(Um tipo de entidade cuja chave de entidade é referenciada pela o final dependente.)</span><span class="sxs-lookup"><span data-stu-id="5f205-111">(An entity type whose entity key is referenced by the dependent end.)</span></span>  
  
- <span data-ttu-id="5f205-112">A chave de entidade de extremidade principal.</span><span class="sxs-lookup"><span data-stu-id="5f205-112">The entity key of the principal end.</span></span>  
  
- <span data-ttu-id="5f205-113">O final dependente de restrição.</span><span class="sxs-lookup"><span data-stu-id="5f205-113">The dependent end of the constraint.</span></span> <span data-ttu-id="5f205-114">(Um tipo de entidade que tem uma propriedade ou um propriedades que referenciem a chave de entidade de extremidade principal.)</span><span class="sxs-lookup"><span data-stu-id="5f205-114">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>  
  
- <span data-ttu-id="5f205-115">A propriedade ou propriedades referência de extremidade dependente.</span><span class="sxs-lookup"><span data-stu-id="5f205-115">The referencing property or properties of the dependent end.</span></span>  
  
 <span data-ttu-id="5f205-116">O objetivo de restrições de integridade referencial em EDM é garantir que as associações válidos sempre existe.</span><span class="sxs-lookup"><span data-stu-id="5f205-116">The purpose of referential integrity constraints in the EDM is to ensure that valid associations always exist.</span></span> <span data-ttu-id="5f205-117">Para obter mais informações, consulte [Propriedade Foreign Key](foreign-key-property.md).</span><span class="sxs-lookup"><span data-stu-id="5f205-117">For more information, see [foreign key property](foreign-key-property.md).</span></span>

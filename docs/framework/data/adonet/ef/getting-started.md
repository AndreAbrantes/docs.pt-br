# <a name="getting-started-entity-framework"></a>Introdução (Entity Framework)
O [!INCLUDE[vstecado](../../../../../includes/vstecado-md.md)] [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] oferece suporte a serviços e aplicativos centrados em dados e fornece uma plataforma para programação em relação aos dados que aumenta o nível de abstração do nível lógico relacional para o nível conceitual. Permitindo que os desenvolvedores trabalhar com dados em um nível mais alto de abstração, o [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] oferece suporte a código que é independente de qualquer mecanismo de armazenamento de dados específico ou um esquema relacional. Para obter mais informações, consulte [visão geral do Entity Framework](../../../../../docs/framework/data/adonet/ef/overview.md).  
  
 Para iniciar rapidamente usando a versão mais recente do Entity Framework, consulte [Introdução](http://go.microsoft.com/fwlink/?LinkId=235280)

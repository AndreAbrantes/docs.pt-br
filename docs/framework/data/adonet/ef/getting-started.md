# <a name="getting-started-entity-framework"></a>Introdução (Entity Framework)
O ADO.NET [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] dá suporte a aplicativos e serviços centrados em dados e fornece uma plataforma para programação em dados que gera o nível de abstração do nível relacional lógico para o nível conceitual. Ao permitir que os desenvolvedores trabalhem com dados em um nível mais recente de abstração, o [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] código oferece suporte que é independente de qualquer mecanismo de armazenamento de dados ou esquema relacional específico. Para obter mais informações, consulte [Entity Framework visão geral](overview.md).  
  
 Para começar rapidamente a usar a versão mais recente do Entity Framework, [consulte Introdução](https://go.microsoft.com/fwlink/?LinkId=235280)

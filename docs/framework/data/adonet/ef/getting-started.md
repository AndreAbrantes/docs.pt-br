# <a name="getting-started-entity-framework"></a>Introdução (Entity Framework)
O [!INCLUDE[vstecado](../../../../../includes/vstecado-md.md)] [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] dá suporte a aplicativos centrados em dados e serviços e fornece uma plataforma para programar com os dados que eleva o nível de abstração do nível relacional lógico para o nível conceitual. Permitir que os desenvolvedores trabalhar com dados em um nível maior de abstração, o [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] dá suporte ao código que é independente de qualquer mecanismo de armazenamento de dados específico ou um esquema relacional. Para obter mais informações, consulte [visão geral do Entity Framework](../../../../../docs/framework/data/adonet/ef/overview.md).  
  
 Para começar rapidamente a usar a versão mais recente do Entity Framework, consulte [começar](https://go.microsoft.com/fwlink/?LinkId=235280)

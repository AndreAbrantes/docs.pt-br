# <a name="entity-framework-resources"></a>Recursos do Entity Framework
Os seguintes recursos externos fornecem informações e suporte para criando aplicativos do Entity Framework.  
  
 [Data Developer Center](https://go.microsoft.com/fwlink/?LinkId=213876)  
 Local central para o desenvolvimento de dados com tecnologias da Microsoft.  
  
 [Blog da equipe do ADO.NET](https://go.microsoft.com/fwlink/?LinkId=91905)  
 Blog que contém atualizações e discussão de recursos e funcionalidade do ADO.NET.  
  
 [Blog de Design do Entity Framework](https://go.microsoft.com/fwlink/?LinkId=186888)  
 Discussões de design e visualizações de recursos para as versões futuras do Entity Framework.  
  
 [Plataforma de dados como faço para? Vídeos: Série do Entity Framework](https://go.microsoft.com/fwlink/?LinkId=124600)  
 Contém um conjunto de screencasts em vídeo que demonstram como criar aplicativos usando vários componentes do Entity Framework.  
  
 [Perguntas Frequentes do Entity Framework](https://social.technet.microsoft.com/wiki/contents/articles/3737.entity-framework-faq.aspx)  
 Seção de wiki que contém perguntas frequentes sobre o Entity Framework.

# <a name="working-with-objects"></a>Trabalhando com objetos
O [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] permite que você consulta, inserção, atualização e exclua dados, que é expresso como tipado objetos common language runtime (CLR) que são instâncias de tipos de entidade. Os tipos de entidade representam as entidades definidas no modelo conceitual. O [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] mapeia as entidades e os relacionamentos que são definidos em um modelo conceitual para uma fonte de dados. O [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] fornece recursos para fazer o seguinte: materializar os dados retornados da fonte de dados como objetos; controlar as alterações feitas aos objetos; manipular a simultaneidade; propagar alterações de objeto para a fonte de dados; e associar objetos a controles.  
  
 Para obter mais informações sobre como trabalhar com objetos na versão mais recente do Entity Framework, consulte, [trabalhando com objetos](https://go.microsoft.com/fwlink/?LinkId=235289).

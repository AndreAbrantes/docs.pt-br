### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="2e3f5-116">Para gerar o modelo da escola para um projeto do Visual Basic usando EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="2e3f5-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="2e3f5-119">No prompt de comando, execute o seguinte comando sem quebras de linha:</span><span class="sxs-lookup"><span data-stu-id="2e3f5-119">At the command prompt, execute the following command without line breaks:</span></span>


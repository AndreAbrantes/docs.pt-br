### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="6c079-116">Para gerar o modelo da escola para um projeto do Visual Basic usando EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="6c079-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

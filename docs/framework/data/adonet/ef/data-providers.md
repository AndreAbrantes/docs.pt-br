# <a name="entity-framework-data-providers"></a>Provedores de dados do Entity Framework
Esta seção fornece informações sobre os provedores de dados que oferecem suporte ao [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
## <a name="in-this-section"></a>Nesta seção  
 [Provedor EntityClient para Entity Framework](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 Descreve o provedor de dados EntityClient. Este provedor transforma consultas em um modelo de dados em uma árvore de comandos canônica. A árvore de comandos pode então ser consumida por um provedor de dados do [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] para o [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)].  
  
 [SqlClient para Entity Framework](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 Descreve o [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] provedor de dados que oferece suporte a [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] para uso com um banco de dados do SQL Server.

# <a name="queries-in-linq-to-entities"></a><span data-ttu-id="3947b-102">Consultas no LINQ to Entities</span><span class="sxs-lookup"><span data-stu-id="3947b-102">Queries in LINQ to Entities</span></span>
<span data-ttu-id="3947b-103">Uma consulta é uma expressão que recupera dados de uma fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="3947b-103">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="3947b-104">Normalmente, as consultas são expressas em uma linguagem de consulta especializada, como o SQL para bancos de dados relacionais e o XQuery para XML.</span><span class="sxs-lookup"><span data-stu-id="3947b-104">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="3947b-105">Portanto, os desenvolvedores precisaram aprender uma nova linguagem de consulta para cada tipo de fonte de dados ou formato de dados que consultam.</span><span class="sxs-lookup"><span data-stu-id="3947b-105">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="3947b-106">O LINQ (Consulta Integrada à Linguagem) oferece um modelo mais simples e consistente para trabalhar com dados em vários tipos de fontes de dados e formatos.</span><span class="sxs-lookup"><span data-stu-id="3947b-106">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="3947b-107">Em uma consulta LINQ, você sempre trabalha com objetos de programação.</span><span class="sxs-lookup"><span data-stu-id="3947b-107">In a LINQ query, you always work with programming objects.</span></span>  
  
 <span data-ttu-id="3947b-108">Uma operação de consulta LINQ consiste em três ações: obter a fonte ou fontes de dados, criar a consulta e executá-la.</span><span class="sxs-lookup"><span data-stu-id="3947b-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="3947b-109">As fontes de dados que implementam a interface genérica <xref:System.Collections.Generic.IEnumerable%601> ou a interface genérica de <xref:System.Linq.IQueryable%601> podem ser consultadas por meio do LINQ.</span><span class="sxs-lookup"><span data-stu-id="3947b-109">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="3947b-110">As instâncias <xref:System.Data.Objects.ObjectQuery%601> da classe genérica, <xref:System.Linq.IQueryable%601> que implementa a interface genérica, servem como fonte de dados para consultas LINQ to Entities.</span><span class="sxs-lookup"><span data-stu-id="3947b-110">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="3947b-111">A classe genérica <xref:System.Data.Objects.ObjectQuery%601> representa uma consulta que retorna uma coleção de zero ou mais objetos tipados.</span><span class="sxs-lookup"><span data-stu-id="3947b-111">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="3947b-112">Você também pode deixar o compilador inferir o tipo de `var` entidade usando a palavra-chave C# (Dim in Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="3947b-112">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="3947b-113">Na consulta, você especifica exatamente as informações que deseja recuperar da fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="3947b-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="3947b-114">Uma consulta também pode especificar como essas informações devem ser classificadas, agrupadas e moldadas antes de serem retornadas.</span><span class="sxs-lookup"><span data-stu-id="3947b-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="3947b-115">No LINQ, uma consulta é armazenada em uma variável.</span><span class="sxs-lookup"><span data-stu-id="3947b-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="3947b-116">Se a consulta retornar uma sequência de valores, a própria variável da consulta deverá ser um tipo consultável.</span><span class="sxs-lookup"><span data-stu-id="3947b-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="3947b-117">Essa variável de consulta não toma nenhuma ação e não retorna nenhum dado, ela apenas armazena as informações da consulta.</span><span class="sxs-lookup"><span data-stu-id="3947b-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="3947b-118">Depois de criar uma consulta, você deve executá-la para recuperar todos os dados.</span><span class="sxs-lookup"><span data-stu-id="3947b-118">After you create a query you must execute that query to retrieve any data.</span></span>

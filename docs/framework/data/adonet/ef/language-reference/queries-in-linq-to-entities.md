<span data-ttu-id="4c6a1-103">Uma consulta é uma expressão que recupera dados de uma fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-103">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="4c6a1-104">Normalmente, as consultas são expressas em uma linguagem de consulta especializada, como o SQL para bancos de dados relacionais e o XQuery para XML.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-104">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="4c6a1-105">Portanto, os desenvolvedores precisaram aprender uma nova linguagem de consulta para cada tipo de fonte de dados ou formato de dados que consultam.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-105">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="4c6a1-106">O LINQ (Consulta Integrada à Linguagem) oferece um modelo mais simples e consistente para trabalhar com dados em vários tipos de fontes de dados e formatos.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-106">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="4c6a1-107">Em uma consulta LINQ, você sempre trabalha com objetos de programação.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-107">In a LINQ query, you always work with programming objects.</span></span>

 <span data-ttu-id="4c6a1-113">Na consulta, você especifica exatamente as informações que deseja recuperar da fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="4c6a1-114">Uma consulta também pode especificar como essas informações devem ser classificadas, agrupadas e moldadas antes de serem retornadas.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="4c6a1-115">No LINQ, uma consulta é armazenada em uma variável.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="4c6a1-116">Se a consulta retornar uma sequência de valores, a própria variável da consulta deverá ser um tipo consultável.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="4c6a1-117">Essa variável de consulta não toma nenhuma ação e não retorna nenhum dado, ela apenas armazena as informações da consulta.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="4c6a1-118">Depois de criar uma consulta, você deve executá-la para recuperar todos os dados.</span><span class="sxs-lookup"><span data-stu-id="4c6a1-118">After you create a query you must execute that query to retrieve any data.</span></span>

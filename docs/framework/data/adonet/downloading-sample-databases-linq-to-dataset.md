# <a name="download-sample-databases-linq-to-dataset"></a>Baixar bancos de dados de exemplo (LINQ para DataSet)

As amostras e passo a passo na documentação LINQ para DataSet usam o banco de dados da amostra AdventureWorks. Você pode baixar este banco de dados de amostras gratuitamente. Navegue pelos [bancos de dados de amostragem adventureworks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks) para baixar o banco de dados. Em seguida, siga as instruções nessa página para anexar ou restaurar um banco de dados.
  
## <a name="sql-server-express"></a>SQL Server Express

As amostras e passo a passo na seção LINQ para DataSet usam o SQL Server 2005 como o armazenamento de dados, mas você pode modificá-las para usar o [SQL Server Express.](https://go.microsoft.com/fwlink/?linkid=866658) O SQL Server Express é gratuito e você pode redistribuí-lo com aplicativos.

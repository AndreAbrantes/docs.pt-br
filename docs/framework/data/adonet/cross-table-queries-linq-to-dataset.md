 <span data-ttu-id="21548-116">O <xref:System.Linq.Enumerable.GroupJoin%2A> operadores não têm equivalente direto em termos de banco de dados relacional; eles implementam um superconjunto de junções internas e junções externas esquerdas.</span><span class="sxs-lookup"><span data-stu-id="21548-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="21548-117">Uma junção externa esquerda é uma junção que retorna cada elemento da primeira (à esquerda) coleção, mesmo se ele não tenha elementos correlacionados na segunda coleção.</span><span class="sxs-lookup"><span data-stu-id="21548-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>

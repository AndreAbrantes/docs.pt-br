 <span data-ttu-id="bd8ed-111">O [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] framework fornece dois operadores, de união <xref:System.Linq.Enumerable.Join%2A> e <xref:System.Linq.Enumerable.GroupJoin%2A>.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-111">The [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] framework provides two join operators, <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="bd8ed-112">Esses operadores executam *junções*: isto é, junções que correspondem a duas fontes somente quando suas chaves são iguais.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-112">These operators perform *equi-joins*: that is, joins that match two data sources only when their keys are equal.</span></span> <span data-ttu-id="bd8ed-113">(Por outro lado, o [!INCLUDE[tsql](../../../../includes/tsql-md.md)] dá suporte a operadores de junção além do `equals`, como o operador `less than`.)</span><span class="sxs-lookup"><span data-stu-id="bd8ed-113">(By contrast, [!INCLUDE[tsql](../../../../includes/tsql-md.md)] supports join operators other than `equals`, such as the `less than` operator.)</span></span>  
  
 <span data-ttu-id="bd8ed-114">Em termos de banco de dados relacional, o <xref:System.Linq.Enumerable.Join%2A> implementa uma junção interna.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-114">In relational database terms, <xref:System.Linq.Enumerable.Join%2A> implements an inner join.</span></span> <span data-ttu-id="bd8ed-115">Uma junção interna é um tipo de junção na qual apenas os objetos que têm uma correspondência no conjunto de dados oposto são retornados.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-115">An inner join is a type of join in which only those objects that have a match in the opposite data set are returned.</span></span>  
  
 <span data-ttu-id="bd8ed-116">O <xref:System.Linq.Enumerable.GroupJoin%2A> operadores não têm equivalente direto em termos de banco de dados relacional; eles implementam um superconjunto de junções internas e junções externas esquerdas.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="bd8ed-117">Uma junção externa esquerda é uma junção que retorna cada elemento da primeira (à esquerda) coleção, mesmo se ele não tenha elementos correlacionados na segunda coleção.</span><span class="sxs-lookup"><span data-stu-id="bd8ed-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>

# <a name="quickstart-wcf-data-services"></a>Guia de Início Rápido (WCF Data Services)
Este guia de início rápido ajuda você a se familiarizar com [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] e [!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] por meio de uma série de tarefas que oferecem suporte os tópicos [Introdução](../../../../docs/framework/data/wcf/getting-started-with-wcf-data-services.md).  
  
## <a name="what-you-will-learn"></a>O que você aprenderá  
 A primeira tarefa neste guia de início rápido mostra como criar um serviço de dados para expor um [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] do feed de dados de exemplo Northwind. Tópicos posteriores, você acessará o [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] feed usando um navegador da Web e também criar um aplicativo de cliente do Windows Presentation Foundation (WPF) que consome o [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] feed usando bibliotecas de cliente.  
  
## <a name="prerequisites"></a>Pré-requisitos  
 Para concluir este guia de início rápido, instale os seguintes componentes:  
  
-   [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)].  
  
-   Uma instância de [!INCLUDE[msCoName](../../../../includes/msconame-md.md)] do SQL Server. Isso inclui o SQL Server Express, que é incluído em uma instalação padrão do Visual Studio.  
  
-   O banco de dados de exemplo Northwind. Para baixar esse banco de dados de exemplo, consulte a página de download [bancos de dados do SQL Server](http://go.microsoft.com/fwlink/?linkid=24758).  
  
## <a name="wcf-data-services-quickstart-tasks"></a>Tarefas do guia de início rápido do WCF Data Services  
 [Creating the Data Service](../../../../docs/framework/data/wcf/creating-the-data-service.md) (Criando o serviço de dados)  
 Defina o aplicativo [!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)], defina o modelo de dados, crie o serviço de dados e habilite o acesso aos recursos.  
  
 [Acessando o serviço de um navegador da Web](../../../../docs/framework/data/wcf/accessing-the-service-from-a-web-browser-wcf-data-services-quickstart.md)  
 Inicie o serviço do Visual Studio e acessar o serviço enviando solicitações HTTP GET por meio de um navegador da Web para o feed exposto.  
  
 [Criando o aplicativo cliente do .NET Framework](../../../../docs/framework/data/wcf/creating-the-dotnet-client-application-wcf-data-services-quickstart.md)  
 Criar um [!INCLUDE[avalon2](../../../../includes/avalon2-md.md)] aplicativo cliente para consumir o [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] feed, associar dados a controles de Windows, alterar dados em controles associados e, em seguida, enviar as alterações de volta para o serviço de dados.  
  
> [!NOTE]
>  Arquivos de projeto de uma versão completa do início rápido podem ser baixados do [exemplos de documentação do WCF Data Services](http://go.microsoft.com/fwlink/?LinkId=179994) página.

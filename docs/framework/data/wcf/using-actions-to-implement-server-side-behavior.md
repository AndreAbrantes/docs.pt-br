 <span data-ttu-id="ebe30-164">As ações são chamadas usando uma solicitação HTTP POST.</span><span class="sxs-lookup"><span data-stu-id="ebe30-164">Actions are invoked using an HTTP POST request.</span></span> <span data-ttu-id="ebe30-165">O URL especifica o recurso seguido pelo nome da ação.</span><span class="sxs-lookup"><span data-stu-id="ebe30-165">The URL specifies the resource followed by the action name.</span></span> <span data-ttu-id="ebe30-166">Os parâmetros são passados no corpo da solicitação.</span><span class="sxs-lookup"><span data-stu-id="ebe30-166">Parameters are passed in the body of the request.</span></span> <span data-ttu-id="ebe30-167">Por exemplo, se houve um serviço chamado MovieService que expôs uma ação chamada Avaliar.</span><span class="sxs-lookup"><span data-stu-id="ebe30-167">For example if there was a service called MovieService which exposed an action called Rate.</span></span> <span data-ttu-id="ebe30-168">Você pode usar a seguinte URL para chamar a ação Avaliar em um filme específico:</span><span class="sxs-lookup"><span data-stu-id="ebe30-168">You could use the following URL to invoke the Rate action on a specific movie:</span></span>  
  
 `http://MovieServer/MovieService.svc/Movies(1)/Rate`
  
 <span data-ttu-id="ebe30-169">Movies(1) especifica o filme que você deseja avaliar e Avaliar especifica a ação.</span><span class="sxs-lookup"><span data-stu-id="ebe30-169">Movies(1) specifies the movie you wish to rate and Rate specifies the Rate action.</span></span> <span data-ttu-id="ebe30-170">O valor real da avaliação estará no corpo da solicitação de HTTP conforme mostrado no seguinte exemplo:</span><span class="sxs-lookup"><span data-stu-id="ebe30-170">The actual value of the rating will be in the body of the HTTP request as shown in the following example:</span></span>

```  
Content-Type: application/json   
Content-Length: 20   
Host: localhost:15238  
{   
   "rating": 4   
}  
```

# <a name="getting-started-with-wcf-data-services"></a>Introdução ao WCF Data Services
Os tópicos nesta seção ajudam você a entender rapidamente Protocolo Open Data (OData) e a usar [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] para expor e consumir feeds OData explicando as tecnologias subjacentes. Esta seção inclui conteúdo conceitual e um tutorial de [início rápido](quickstart-wcf-data-services.md) .  
  
## <a name="in-this-section"></a>Nesta seção  
 Os tópicos a seguir descrevem como criar serviços de dados usando o [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Expondo seus dados como um serviço](exposing-your-data-as-a-service-wcf-data-services.md)  
 Descreve as etapas necessárias para criar um serviço de dados usando o [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Acessando recursos do serviço de dados](accessing-data-service-resources-wcf-data-services.md)  
 Descreve como trabalhar com feeds OData.  
  
 [Usando um serviço de dados em um aplicativo cliente](using-a-data-service-in-a-client-application-wcf-data-services.md)  
 Descreve como trabalhar com um feed OData em um aplicativo cliente .NET Framework.  
  
 [Quickstart](quickstart-wcf-data-services.md) (Início rápido)  
 Mostra como criar e acessar um serviço simples baseado em OData que expõe um feed com base no banco de dados de exemplo Northwind.  
  
 [Application Scenarios](application-scenarios-wcf-data-services.md) (Cenários de aplicativo)  
 Realça um conjunto principal de cenários OData com suporte pelo [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Recursos](wcf-data-services-resources.md)  
 Fornece links para recursos de [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] e OData.

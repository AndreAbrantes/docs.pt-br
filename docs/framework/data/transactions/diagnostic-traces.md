## <a name="enabling-tracing"></a><span data-ttu-id="5a5c1-116">A habilitação do rastreamento</span><span class="sxs-lookup"><span data-stu-id="5a5c1-116">Enabling Tracing</span></span>

```xml  
          <source name="System.Transactions" switchValue="Warning">  
               <listeners>

                     initializeData= "tx.log" />  
               </listeners>  
          </source>  
     </sources>  
</system.diagnostics>  
</configuration>  
```

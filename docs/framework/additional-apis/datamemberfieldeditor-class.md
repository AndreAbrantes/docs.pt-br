# <a name="datamemberfieldeditor-class"></a>Classe DataMemberFieldEditor

Fornece a interface de usuário para editar as propriedades de objetos de associação de dados, listando todas as propriedades do `DataSource` objeto para selecionar a fonte do valor de.  
  
## <a name="syntax"></a>Sintaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> O `DataMemberFieldEditor` classe é interno e não se destina a ser usado diretamente no seu código.
> 
> Microsoft não suporta o uso dessa classe em um aplicativo de produção sob nenhuma circunstância.

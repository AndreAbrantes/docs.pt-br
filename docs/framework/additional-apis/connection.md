# <a name="connection-class"></a>Classe Connection

A `Connection` classe analisa respostas do servidor, solicitações de fila e solicitações de pipeline.

## <a name="syntax"></a>Sintaxe
  
```csharp  
internal class Connection : PooledStream
```

> [!WARNING]
> A `Connection` classe é interna e não deve ser usada diretamente em seu código.
>
> A Microsoft não suporta o uso desta classe em um aplicativo de produção nenhuma circunstância.

## <a name="requirements"></a>Requisitos

**Espaço de nome:**<xref:System.Net>

**Montagem:** Sistema (em System.dll)

**Versões do Framework .NET:** Disponível desde 2.0.

# <a name="datamemberlisteditor-class"></a>Classe DataMemberListEditor

Fornece uma interface de usuário gotejada para editar propriedades de `DataSource` objetos vinculados a `DataSource` dados (objetos que têm propriedade não nula) listando todas as propriedades do objeto para selecionar a fonte de valor de.  
  
## <a name="syntax"></a>Sintaxe
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> A `DataMemberListEditor` classe é interna e não deve ser usada diretamente em seu código.
>
> A Microsoft não suporta o uso desta classe em um aplicativo de produção nenhuma circunstância.

### <a name="to-convert-numeric-input-from-a-web-textbox-control-to-a-number"></a><span data-ttu-id="5c9b0-106">Para converter a entrada numérica de um controle de caixa de texto da Web em um número</span><span class="sxs-lookup"><span data-stu-id="5c9b0-106">To convert numeric input from a Web TextBox control to a number</span></span>  
  
1.  <span data-ttu-id="5c9b0-107">Determine se a matriz de cadeia de caracteres retornada pelo <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> propriedade é preenchida.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-107">Determine whether the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> property is populated.</span></span> <span data-ttu-id="5c9b0-108">Se não estiver, vá para a etapa 6.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-108">If it is not, continue to step 6.</span></span>  
  
2.  <span data-ttu-id="5c9b0-109">Se a matriz de cadeia de caracteres retornada pelo <xref:System.Web.HttpRequest.UserLanguages%2A> propriedade é preenchida, recupere o primeiro elemento.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-109">If the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is populated, retrieve its first element.</span></span> <span data-ttu-id="5c9b0-110">O primeiro elemento indica que o usuário padrão ou idioma e região.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-110">The first element indicates the user's default or preferred language and region.</span></span>  
  
3.  <span data-ttu-id="5c9b0-111">Criar uma instância de um <xref:System.Globalization.CultureInfo> objeto que representa o usuário cultura preferencial do chamando o <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> construtor.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-111">Instantiate a <xref:System.Globalization.CultureInfo> object that represents the user's preferred culture by calling the <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> constructor.</span></span>  
  
4.  <span data-ttu-id="5c9b0-112">Chame o `TryParse` ou o `Parse` método do tipo numérico que você deseja converter o usuário de entrada.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-112">Call either the `TryParse` or the `Parse` method of the numeric type that you want to convert the user's input to.</span></span> <span data-ttu-id="5c9b0-113">Use uma sobrecarga de `TryParse` ou o `Parse` método com um `provider` parâmetro e passe a ele um dos seguintes:</span><span class="sxs-lookup"><span data-stu-id="5c9b0-113">Use an overload of the `TryParse` or the `Parse` method with a `provider` parameter, and pass it either of the following:</span></span>  
  
    -   <span data-ttu-id="5c9b0-114">O <xref:System.Globalization.CultureInfo> objeto criado na etapa 3.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-114">The <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
    -   <span data-ttu-id="5c9b0-115">O <xref:System.Globalization.NumberFormatInfo> objeto que é retornado pelo <xref:System.Globalization.CultureInfo.NumberFormat%2A> propriedade o <xref:System.Globalization.CultureInfo> objeto criado na etapa 3.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-115">The <xref:System.Globalization.NumberFormatInfo> object that is returned by the <xref:System.Globalization.CultureInfo.NumberFormat%2A> property of the <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
5.  <span data-ttu-id="5c9b0-116">Se a conversão falhar, repita as etapas 2 a 4 para cada elemento restante na matriz de cadeia de caracteres retornadas pelo <xref:System.Web.HttpRequest.UserLanguages%2A> propriedade.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-116">If the conversion fails, repeat steps 2 through 4 for each remaining element in the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property.</span></span>  
  
6.  <span data-ttu-id="5c9b0-117">Se a conversão ainda falhar ou se a matriz de cadeia de caracteres retornada pelo <xref:System.Web.HttpRequest.UserLanguages%2A> propriedade estiver vazia, analise a cadeia de caracteres usando a cultura invariável, que é retornado pelo <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> propriedade.</span><span class="sxs-lookup"><span data-stu-id="5c9b0-117">If the conversion still fails or if the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is empty, parse the string by using the invariant culture, which is returned by the <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> property.</span></span>

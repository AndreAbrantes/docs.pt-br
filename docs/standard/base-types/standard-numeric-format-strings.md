  <span data-ttu-id="43188-114">Quando o *especificador de precisão* controla o número de dígitos fracionários na cadeia de caracteres de resultado, ela reflete um número que será arredondado para um resultado representável mais próximo do resultado infinitamente preciso.</span><span class="sxs-lookup"><span data-stu-id="43188-114">When *precision specifier* controls the number of fractional digits in the result string, the result string reflects a number that is rounded to a representable result nearest to the infinitely precise result.</span></span> <span data-ttu-id="43188-115">Se houver dois resultados representáveis igualmente próximos:</span><span class="sxs-lookup"><span data-stu-id="43188-115">If there are two equally near representable results:</span></span>

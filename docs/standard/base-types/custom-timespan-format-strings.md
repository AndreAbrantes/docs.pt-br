<span data-ttu-id="41c92-103">Um <xref:System.TimeSpan> cadeia de caracteres de formato define a representação de cadeia de caracteres de uma <xref:System.TimeSpan> valor resultante de uma operação de formatação.</span><span class="sxs-lookup"><span data-stu-id="41c92-103">A <xref:System.TimeSpan> format string defines the string representation of a <xref:System.TimeSpan> value that results from a formatting operation.</span></span> <span data-ttu-id="41c92-104">Uma cadeia de caracteres de formato personalizado consiste em um ou mais personalizada <xref:System.TimeSpan> Formatar especificadores juntamente com qualquer número de caracteres literal.</span><span class="sxs-lookup"><span data-stu-id="41c92-104">A custom format string consists of one or more custom <xref:System.TimeSpan> format specifiers along with any number of literal characters.</span></span> <span data-ttu-id="41c92-105">Qualquer cadeia de caracteres que não é um [cadeia de caracteres de formato TimeSpan padrão](../../../docs/standard/base-types/standard-timespan-format-strings.md) é interpretado como um personalizado <xref:System.TimeSpan> cadeia de caracteres de formato.</span><span class="sxs-lookup"><span data-stu-id="41c92-105">Any string that is not a [standard TimeSpan format string](../../../docs/standard/base-types/standard-timespan-format-strings.md) is interpreted as a custom <xref:System.TimeSpan> format string.</span></span>  
  
> [!IMPORTANT]
>  <span data-ttu-id="41c92-106">Personalizado <xref:System.TimeSpan> especificadores de formato não incluem símbolos de separador de espaço reservado, como os símbolos que separam os dias de horas, horas, minutos ou segundos de frações de segundo.</span><span class="sxs-lookup"><span data-stu-id="41c92-106">The custom <xref:System.TimeSpan> format specifiers do not include placeholder separator symbols, such as the symbols that separate days from hours, hours from minutes, or seconds from fractional seconds.</span></span> <span data-ttu-id="41c92-107">Em vez disso, esses símbolos devem ser incluídos na cadeia de caracteres de formato personalizado como literais de cadeia de caracteres.</span><span class="sxs-lookup"><span data-stu-id="41c92-107">Instead, these symbols must be included in the custom format string as string literals.</span></span> <span data-ttu-id="41c92-108">Por exemplo, `"dd\.hh\:mm"` define um ponto (.) como o separador entre dias e horas e dois-pontos (:) como o separador entre horas e minutos.</span><span class="sxs-lookup"><span data-stu-id="41c92-108">For example, `"dd\.hh\:mm"` defines a period (.) as the separator between days and hours, and a colon (:) as the separator between hours and minutes.</span></span>  
>   
>  <span data-ttu-id="41c92-109">Personalizar <xref:System.TimeSpan> especificadores de formato também não incluem um símbolo de sinal que lhe permite diferenciar entre os intervalos de tempo positivo e negativo.</span><span class="sxs-lookup"><span data-stu-id="41c92-109">Custom <xref:System.TimeSpan> format specifiers also do not include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="41c92-110">Para incluir um símbolo de sinal, você precisa construir uma cadeia de caracteres de formato usando lógica condicional.</span><span class="sxs-lookup"><span data-stu-id="41c92-110">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="41c92-111">A seção [Outros caracteres](#Other) inclui um exemplo.</span><span class="sxs-lookup"><span data-stu-id="41c92-111">The [Other Characters](#Other) section includes an example.</span></span>

# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Quando implantar contêineres do Windows em sua infraestrutura de VM de IaaS local

- Sua organização pode não estar pronta para passar para a nuvem, ou pode não ser capaz de mover para a nuvem por um motivo comercial. Mas, ainda é possível obter os benefícios do uso de contêineres do Windows em seus próprios data centers.

- Você pode ter outros artefatos que estão sendo usados no local e que podem retardá-lo quando você tenta mover para a nuvem. Por exemplo, segurança ou autenticação dependências com local Windows Server Active Directory ou qualquer outro ativo no local.

- Se você começar a usar contêineres do Windows hoje, você pode fazer uma migração em fases para a nuvem no futuro de uma posição muito melhor. Contêineres do Windows está se tornando uma unidade de implantação para qualquer nuvem, com nenhuma bloqueio-in.

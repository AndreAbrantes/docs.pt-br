<span data-ttu-id="84c7c-122">Um cenário semelhante pode ocorrer em um aplicativo que consome serviços HTTP.</span><span class="sxs-lookup"><span data-stu-id="84c7c-122">A similar scenario might occur in an app that consumes HTTP services.</span></span> <span data-ttu-id="84c7c-123">A rede ou o próprio serviço pode não estar disponível na nuvem durante uma falha transitória, curta.</span><span class="sxs-lookup"><span data-stu-id="84c7c-123">The network or the service itself might not be available in the cloud during a short, transient failure.</span></span>

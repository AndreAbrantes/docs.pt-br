# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Escolhendo entre o .NET Core e do .NET Framework para contêineres do Docker

Há duas implementações com suporte para criar aplicativos do Docker em contêineres do servidor com o .NET: [.NET Framework](https://www.microsoft.com/net/download/framework) e [.NET Core](https://www.microsoft.com/net/download/core). Eles compartilham muitos componentes do .NET Standard e você pode compartilhar o código entre os dois. No entanto, há diferenças fundamentais entre eles e a implementação a ser usada dependerá do que você deseja realizar. Esta seção fornece orientações sobre quando escolher cada implementação.

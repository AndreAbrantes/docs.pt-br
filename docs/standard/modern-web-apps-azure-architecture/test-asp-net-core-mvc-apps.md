<span data-ttu-id="71e38-253">Para aplicativos ASP.NET Core, a classe TestServer facilita muito a gravação de testes funcionais.</span><span class="sxs-lookup"><span data-stu-id="71e38-253">For ASP.NET Core applications, the TestServer class makes functional tests fairly easy to write.</span></span> <span data-ttu-id="71e38-254">Você configura um TestServer usando um WebHostBuilder, assim como faria normalmente para seu aplicativo.</span><span class="sxs-lookup"><span data-stu-id="71e38-254">You configure a TestServer using a WebHostBuilder, just as you normally do for your application.</span></span> <span data-ttu-id="71e38-255">Este WebHostBuilder deve ser configurado da mesma forma que o host real do aplicativo, mas você pode modificar aspectos dele que facilitam os testes.</span><span class="sxs-lookup"><span data-stu-id="71e38-255">This WebHostBuilder should be configured just like your application's real host, but you can modify any aspects of it that make testing easier.</span></span> <span data-ttu-id="71e38-256">Na maioria das vezes, você reutilizará o mesmo TestServer para muitos casos de teste, de modo que você possa encapsulá-lo em um método reutilizável (talvez em uma classe base):</span><span class="sxs-lookup"><span data-stu-id="71e38-256">Most of the time, you'll reuse the same TestServer for many test cases, so you can encapsulate it in a reusable method (perhaps in a base class):</span></span>

```cs
public abstract class BaseWebTest
{
    protected readonly HttpClient _client;
    protected string _contentRoot;

    public BaseWebTest()
    {
        _client = GetClient();
    }
    
    protected HttpClient GetClient()
    {
        var startupAssembly = typeof(Startup).GetTypeInfo().Assembly;
        _contentRoot = GetProjectPath("src", startupAssembly);
        var builder = new WebHostBuilder()
        .UseContentRoot(_contentRoot)
        .UseStartup&lt;Startup&gt;();
        var server = new TestServer(builder);
        var client = server.CreateClient();
        return client;
    }
}
```

```cs
public class CatalogControllerGetImage : BaseWebTest
{
    [Fact]
    public async Task ReturnsFileContentResultGivenValidId()
    {
        var testFilePath = Path.Combine(_contentRoot, "pics//1.png");
        var expectedFileBytes = File.ReadAllBytes(testFilePath);
        var response = await _client.GetAsync("/catalog/pic/1");
        response.EnsureSuccessStatusCode();
        var streamResponse = await response.Content.ReadAsStreamAsync();
        byte[] byteResult;

            byteResult = ms.ToArray();

}
```

### <a name="generics-terminology"></a><span data-ttu-id="f7e93-124">Terminologia de genéricos</span><span class="sxs-lookup"><span data-stu-id="f7e93-124">Generics terminology</span></span>  
 <span data-ttu-id="f7e93-125">Os seguintes termos são usados para discutir genéricos no .NET Framework:</span><span class="sxs-lookup"><span data-stu-id="f7e93-125">The following terms are used to discuss generics in the .NET Framework:</span></span>  
  
-   <span data-ttu-id="f7e93-126">A *definição de tipo genérico* é uma classe, estrutura ou declaração de interface que funciona como um modelo, com espaços reservados para os tipos que ela pode conter ou usar.</span><span class="sxs-lookup"><span data-stu-id="f7e93-126">A *generic type definition* is a class, structure, or interface declaration that functions as a template, with placeholders for the types that it can contain or use.</span></span> <span data-ttu-id="f7e93-127">Por exemplo, a classe <xref:System.Collections.Generic.Dictionary%602?displayProperty=fullName> pode conter dois tipos: chaves e valores.</span><span class="sxs-lookup"><span data-stu-id="f7e93-127">For example, the <xref:System.Collections.Generic.Dictionary%602?displayProperty=fullName> class can contain two types: keys and values.</span></span> <span data-ttu-id="f7e93-128">Como uma definição de tipo genérico é apenas um modelo, não é possível criar instâncias de uma classe, estrutura ou interface que seja uma definição de tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="f7e93-128">Because a generic type definition is only a template, you cannot create instances of a class, structure, or interface that is a generic type definition.</span></span>

-   <span data-ttu-id="f7e93-131">Um *tipo genérico construído* ou um *tipo construído* é o resultado de especificação de tipos para os parâmetros de tipo genérico de uma definição de tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="f7e93-131">A *constructed generic type*, or *constructed type*, is the result of specifying types for the generic type parameters of a generic type definition.</span></span>

-   <span data-ttu-id="f7e93-134">*Covariância* e *contravariância* de parâmetros de tipo genérico permitem que você use tipos genéricos construídos cujos argumentos de tipo sejam mais derivados (covariância) ou menos derivados (contravariância) de um tipo construído de destino.</span><span class="sxs-lookup"><span data-stu-id="f7e93-134">*Covariance* and *contravariance* of generic type parameters enable you to use constructed generic types whose type arguments are more derived (covariance) or less derived (contravariance) than a target constructed type.</span></span> <span data-ttu-id="f7e93-135">A covariância e a contravariância são referidas coletivamente como *variância*.</span><span class="sxs-lookup"><span data-stu-id="f7e93-135">Covariance and contravariance are collectively referred to as *variance*.</span></span> <span data-ttu-id="f7e93-136">Para obter mais informações, consulte [Covariância e contravariância](../../../docs/standard/generics/covariance-and-contravariance.md).</span><span class="sxs-lookup"><span data-stu-id="f7e93-136">For more information, see [Covariance and Contravariance](../../../docs/standard/generics/covariance-and-contravariance.md).</span></span>

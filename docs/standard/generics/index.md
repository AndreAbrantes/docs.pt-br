### <a name="generics-terminology"></a><span data-ttu-id="3ba14-122">Terminologia de genéricos</span><span class="sxs-lookup"><span data-stu-id="3ba14-122">Generics terminology</span></span>  
 <span data-ttu-id="3ba14-123">Os seguintes termos são usados para discutir genéricos no .NET:</span><span class="sxs-lookup"><span data-stu-id="3ba14-123">The following terms are used to discuss generics in .NET:</span></span>  
  
-   <span data-ttu-id="3ba14-124">A *definição de tipo genérico* é uma classe, estrutura ou declaração de interface que funciona como um modelo, com espaços reservados para os tipos que ela pode conter ou usar.</span><span class="sxs-lookup"><span data-stu-id="3ba14-124">A *generic type definition* is a class, structure, or interface declaration that functions as a template, with placeholders for the types that it can contain or use.</span></span> <span data-ttu-id="3ba14-125">Por exemplo, a classe <xref:System.Collections.Generic.Dictionary%602?displayProperty=nameWithType> pode conter dois tipos: chaves e valores.</span><span class="sxs-lookup"><span data-stu-id="3ba14-125">For example, the <xref:System.Collections.Generic.Dictionary%602?displayProperty=nameWithType> class can contain two types: keys and values.</span></span> <span data-ttu-id="3ba14-126">Como uma definição de tipo genérico é apenas um modelo, não é possível criar instâncias de uma classe, estrutura ou interface que seja uma definição de tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="3ba14-126">Because a generic type definition is only a template, you cannot create instances of a class, structure, or interface that is a generic type definition.</span></span>

-   <span data-ttu-id="3ba14-129">Um *tipo genérico construído* ou um *tipo construído* é o resultado de especificação de tipos para os parâmetros de tipo genérico de uma definição de tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="3ba14-129">A *constructed generic type*, or *constructed type*, is the result of specifying types for the generic type parameters of a generic type definition.</span></span>

# <a name="introduction-to-the-docker-application-life-cycle"></a>Introdução ao ciclo de vida do aplicativo Docker
 
O ciclo de vida de aplicativos em contêineres é uma jornada que começa com o desenvolvedor. O desenvolvedor decide implementar contêineres e o Docker porque ele elimina fricções em implantações e operações de TI, o que, por fim, ajuda todos a serem mais ágeis, mais produtivos de ponta a ponta e mais rápidos.

# <a name="introduction-tothe-docker-application-life-cycle"></a>Introdução ao ciclo de vida de aplicativos do Docker

*O ciclo de vida de aplicativos em contêineres é uma jornada que começa com o desenvolvedor. O desenvolvedor decide implementar contêineres e o Docker porque isso elimina o atrito nas implantações e operações de TI e, dessa forma, ajuda todos a serem mais ágeis, mais produtivos de ponta a ponta e mais rápidos.*

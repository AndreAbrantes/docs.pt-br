# <a name="introduction-tothe-docker-application-life-cycle"></a>Introdução ao ciclo de vida do aplicativo Docker

*O ciclo de vida de aplicativos em contêineres é uma jornada que começa com o desenvolvedor. O desenvolvedor decide implementar contêineres e Docker, porque ele elimina fricções em implantações e operações de TI, que, por fim, ajuda todos a serem mais ágeis, mais produtivo ponta a ponta e mais rápido.*

# <a name="net-guide"></a>Guia do .NET

O guia .NET fornece informações sobre .NET. Dependendo da sua familiaridade com o .NET, você pode querer explorar diferentes seções deste guia e outros guias, como [.NET Core](../core/index.md) ou [.NET Framework](../framework/index.md).

## <a name="new-to-net"></a>Novo no .NET?

Se você quiser uma visão geral de alto nível do .NET, consulte [O que é .NET?](https://dotnet.microsoft.com/learn/dotnet/what-is-dotnet).

Se você é novo em .NET, consulte [Começar](get-started.md).

Se você preferir fazer uma visita guiada através das principais características do .NET, consulte [Tour of .NET](tour.md).

Para obter uma visão geral das várias peças do .NET e como elas se encaixam, consulte [os componentes arquitetônicos .NET](components.md).

## <a name="new-to-net-core"></a>Novo no .NET Core?

Se você é novo no .NET Core, confira [Comece com .NET Core](../core/get-started.md).

## <a name="new-to-net-standard"></a>Novo no .NET Standard?

Se você é novo no .NET Standard, confira [.NET Standard](net-standard.md).

## <a name="port-net-framework-code-to-net-standard-or-net-core"></a>Código de estrutura .NET para .NET Standard ou .NET Core

Para portar um pacote de aplicativo, serviço, componente ou NuGet para .NET Standard ou .NET Core, consulte ['Porta para .NET Core' do .NET Framework](../core/porting/index.md). As ferramentas para .NET Standard e .NET Core são compartilhadas, de modo que o conteúdo é relevante para portar para ambas as estruturas.

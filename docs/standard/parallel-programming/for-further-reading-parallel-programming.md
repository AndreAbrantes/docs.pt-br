# <a name="for-further-reading-parallel-programming"></a>Para leitura adicional (programação paralela)
Os recursos a seguir contêm informações adicionais sobre programação paralela no .NET Framework:  
  
-   O [Parallel Computing Developer Center](https://msdn.microsoft.com/vstudio/bb964701) no site da MSDN contém links para o conteúdo e as postagens do Fórum sobre programação paralela no .NET Framework e em C++ nativo.  
  
-   O [blog Parallel Programming with .NET](https://blogs.msdn.microsoft.com/pfxteam/) no site de blog da MSDN contém vários artigos detalhados sobre programação paralela no .NET Framework.  
  
-   O [fórum Parallel Extensions](https://social.msdn.microsoft.com/Forums/en-US/home?forum=parallelextensions) no site de fóruns da MSDN é o local para fazer e responder perguntas sobre programação paralela.  
  
-   A página [Exemplos de Extensões Paralelas](https://code.msdn.microsoft.com/ParExtSamples) no site da Galeria de Códigos do MSDN contém vários exemplos que demonstram técnicas de programação paralela intermediárias e avançadas.

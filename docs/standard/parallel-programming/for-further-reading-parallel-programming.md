# <a name="for-further-reading-parallel-programming"></a>Para leitura adicional (programação paralela)
Os recursos a seguir contêm informações adicionais sobre programação paralela no .NET Framework:  
  
-   O [Parallel Computing Developer Center](http://go.microsoft.com/fwlink/?LinkID=160570) no site da MSDN contém links para o conteúdo e as postagens mais recentes do Fórum sobre programação paralela no .NET Framework e em C++ nativo.  
  
-   O [blog Parallel Programming with .NET](http://go.microsoft.com/fwlink/?LinkID=169627) no site de blog da MSDN contém vários artigos detalhados sobre programação paralela no .NET Framework.  
  
-   O [blog Concurrency Visualizer](http://go.microsoft.com/fwlink/?LinkID=169630) no site de blog da MSDN abrange a nova ferramenta de perfis de desempenho incluída em algumas edições do [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)].  
  
-   O [fórum Parallel Extensions](http://go.microsoft.com/fwlink/?LinkID=169628) no site de fóruns da MSDN é o local para fazer e responder perguntas sobre programação paralela.  
  
-   A página [Exemplos de Extensões Paralelas](http://go.microsoft.com/fwlink/?LinkID=165717) no site da Galeria de Códigos do MSDN contém vários exemplos que demonstram técnicas de programação paralela intermediárias e avançadas.

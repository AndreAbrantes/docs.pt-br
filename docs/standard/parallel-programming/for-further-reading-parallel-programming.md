# <a name="for-further-reading-parallel-programming"></a>Para leitura adicional (programação paralela)
Os recursos a seguir contêm informações adicionais sobre a programação paralela no .NET Framework:  
  
-   O [paralelo Computing Developer Center](http://go.microsoft.com/fwlink/?LinkID=160570) na MSDN Web site contém links para o conteúdo mais recente e postagens do Fórum sobre programação paralela no .NET Framework e em C++ nativo.  
  
-   O [programação paralela com .NET blog](http://go.microsoft.com/fwlink/?LinkID=169627) no MSDN blog site contém vários artigos detalhados sobre programação paralela no .NET Framework.  
  
-   O [blog do Visualizador de simultaneidade](http://go.microsoft.com/fwlink/?LinkID=169630) no blog do MSDN site abrange o desempenho de novos perfis ferramenta está incluída em algumas edições do [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)].  
  
-   O [Fórum de extensões paralelas](http://go.microsoft.com/fwlink/?LinkID=169628) no MSDN site fóruns é onde fazer e responder perguntas sobre programação paralela.  
  
-   O [exemplos de extensões paralelas](http://go.microsoft.com/fwlink/?LinkID=165717) página no site da Web de galeria de código do MSDN contém vários exemplos que demonstram técnicas de programação paralela intermediárias e avançadas.

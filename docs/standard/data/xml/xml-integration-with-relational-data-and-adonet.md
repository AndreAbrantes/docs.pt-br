 <span data-ttu-id="21646-108">Os benefícios de ter os dados disponíveis em dois modos de exibição são:</span><span class="sxs-lookup"><span data-stu-id="21646-108">The benefits of having data available in two views are:</span></span>  
  
-   <span data-ttu-id="21646-109">A parte estruturada de um documento XML pode ser mapeado para um dataset, e com eficiência é armazenada, indexada, e pesquisada.</span><span class="sxs-lookup"><span data-stu-id="21646-109">The structured portion of an XML document can be mapped to a dataset, and be efficiently stored, indexed, and searched.</span></span>  
  
-   <span data-ttu-id="21646-110">As transformações, a validação, e navegação podem ser feitas com eficiência através de um modelo de cursor sobre os dados XML que são armazenados relacional.</span><span class="sxs-lookup"><span data-stu-id="21646-110">Transformations, validation, and navigation can be done efficiently through a cursor model over the XML data that is stored relationally.</span></span> <span data-ttu-id="21646-111">Às vezes, pode ser feito com mais eficiência em estruturas relacionais em comparação ao armazenamento de XML em um modelo **XmlDocument**.</span><span class="sxs-lookup"><span data-stu-id="21646-111">At times, it can be done more efficiently against relational structures than if the XML is stored in an **XmlDocument** model.</span></span>  
  
-   <span data-ttu-id="21646-112">O **Conjunto de Dados** pode armazenar uma parte do XML.</span><span class="sxs-lookup"><span data-stu-id="21646-112">The **DataSet** can store a portion of the XML.</span></span> <span data-ttu-id="21646-113">Ou seja, você pode usar **XPath** ou **XslTransform** para armazenar em um **Conjunto de Dados** somente os elementos e atributos de interesse.</span><span class="sxs-lookup"><span data-stu-id="21646-113">That is, you can use **XPath** or **XslTransform** to store to a **DataSet** only those elements and attributes of interest.</span></span> <span data-ttu-id="21646-114">A partir daí, as alterações podem ser feitas no subconjunto de dados menor e filtrado, com a propagação das alterações nos dados maiores em **XmlDataDocument**.</span><span class="sxs-lookup"><span data-stu-id="21646-114">From there, changes can be made to the smaller, filtered subset of data, with the changes propagating to the larger data in the **XmlDataDocument**.</span></span>

 <span data-ttu-id="0a0cc-157">O exemplo a seguir usa a propriedade <xref:System.Xml.XPath.XPathNavigator.OuterXml%2A> para modificar o valor do elemento `price` e inserir um novo atributo `discount` no primeiro elemento `book` no arquivo `contosoBooks.xml`.</span><span class="sxs-lookup"><span data-stu-id="0a0cc-157">The following example uses the <xref:System.Xml.XPath.XPathNavigator.OuterXml%2A> property to modify the value of the `price` element and insert a new `discount` attribute on the first `book` element in the `contosoBooks.xml` file.</span></span>

```vb  
Dim document As XmlDocument = New XmlDocument()  
document.Load("contosoBooks.xml");

Console.WriteLine(navigator.OuterXml)  
```

# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Removendo os nós, conteúdo, e valores de um documento XML
Uma vez que um modelo de objeto (DOM) de documento XML está na memória, você pode remover os nós de árvore, ou remova o conteúdo e os valores de determinados tipos de nó. Para obter informações sobre como remover um nó folha ou subárvore de todo o nó de um documento, consulte [removendo nós DOM](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md). Para obter informações sobre como remover atributos em um elemento, consulte [remover atributos de um nó de elemento no DOM](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md). Para obter informações sobre como remover o conteúdo de um nó, mas deixando o nó na árvore, consulte [remover conteúdo do nó no DOM](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md).

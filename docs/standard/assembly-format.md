<span data-ttu-id="37d48-115">O formato é independente de CPU e sistema operacional.</span><span class="sxs-lookup"><span data-stu-id="37d48-115">The format is CPU- and OS-agnostic.</span></span> <span data-ttu-id="37d48-116">Ele tem sido usado como parte de implementações do .NET que se destinam a muitos chips e CPUs.</span><span class="sxs-lookup"><span data-stu-id="37d48-116">It has been used as part of .NET implementations that target many chips and CPUs.</span></span> <span data-ttu-id="37d48-117">Embora o formato em si tenha patrimônio do Windows, é implementável em qualquer sistema operacional.</span><span class="sxs-lookup"><span data-stu-id="37d48-117">While the format itself has Windows heritage, it is implementable on any operating system.</span></span> <span data-ttu-id="37d48-118">Sua escolha indiscutivelmente mais significativa para a interoperabilidade do sistema operacional é que a maioria dos valores é armazenada no formato little endian.</span><span class="sxs-lookup"><span data-stu-id="37d48-118">It’s arguably most significant choice for OS interoperability is that most values are stored in little-endian format.</span></span> <span data-ttu-id="37d48-119">Ele não tem uma afinidade específica para o tamanho do ponteiro de computador (por exemplo, 32 bits, 64 bits).</span><span class="sxs-lookup"><span data-stu-id="37d48-119">It doesn’t have a specific affinity to machine pointer size (for example, 32-bit, 64-bit).</span></span>

<span data-ttu-id="37d48-120">O formato do assembly .NET também é muito descritivo sobre a estrutura de um determinado programa ou biblioteca.</span><span class="sxs-lookup"><span data-stu-id="37d48-120">The .NET assembly format is also very descriptive about the structure of a given program or library.</span></span> <span data-ttu-id="37d48-121">Ele descreve os componentes internos de um assembly, especificamente: referências de assembly e tipos definidos e suas estruturas internas.</span><span class="sxs-lookup"><span data-stu-id="37d48-121">It describes the internal components of an assembly, specifically: assembly references and types defined and their internal structure.</span></span> <span data-ttu-id="37d48-122">APIs ou ferramentas podem ler e processar essas informações para exibição ou para tomar decisões de programação.</span><span class="sxs-lookup"><span data-stu-id="37d48-122">Tools or APIs can read and process this information for display or to make programmatic decisions.</span></span>

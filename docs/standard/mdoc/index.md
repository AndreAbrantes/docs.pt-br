# <a name="building-documentation-with-mdoc"></a>Compilar documentação com mdoc

mdoc é um sistema de build de documentação para documentação de referência gerenciada. Ao refletir em seus assemblies, o mdoc cria o formato básico da sua documentação de referência gerenciada, que é complementada com conteúdo original.

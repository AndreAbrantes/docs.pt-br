# <a name="get-started"></a>Introdução

Há várias formas para começar a trabalhar com o .NET. Como o .NET é uma grande plataforma, há vários artigos nesta documentação que podem ajudá-lo a começar a usar o .NET, cada um de uma perspectiva diferente.

## <a name="get-started-using-net-languages"></a>Introdução ao uso de linguagens .NET

* Os artigos e [ C# tutoriais](../csharp/tutorials/index.md) de [ C# introdução](../csharp/getting-started/index.md) fornecem maneiras de começar de C#maneira centralizada.

* Os tutoriais de [ F# introdução](../fsharp/get-started/index.md) abrangem as três principais maneiras que você F#pode usar: com o Visual Studio, Visual Studio Code ou ferramentas de linha de comando.

* Os artigos de [introdução ao Visual Basic](../visual-basic/getting-started/index.md) fornecem guias para usar Visual Basic no Visual Studio.

## <a name="get-started-using-net-core"></a>Introdução ao uso do .NET Core

* A [introdução ao .NET Core](../core/get-started.md) fornece uma visão geral dos artigos que mostram como começar a usar o .NET Core em diferentes sistemas operacionais e o uso de ferramentas diferentes.

* Os [tutoriais do .NET Core](../core/tutorials/index.md) detalham como você pode começar a usar o .NET Core usando seu sistema operacional e as ferramentas de sua escolha.

## <a name="get-started-using-net-standard"></a>Introdução ao uso de .NET Standard

* Para obter um tutorial introdutório, consulte [criar uma biblioteca de .net Standard no Visual Studio](../core/tutorials/library-with-visual-studio.md).

## <a name="get-started-using-net-core-on-docker"></a>Introdução ao uso do .NET Core no Docker

[Introdução ao .NET e o Docker](../core/docker/introduction.md) mostra como você pode usar o .NET Core em contêineres do Docker do Windows.

## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="7bb11-193">Como especificar estruturas de destino</span><span class="sxs-lookup"><span data-stu-id="7bb11-193">How to specify target frameworks</span></span>

<span data-ttu-id="7bb11-194">As estruturas de destino são especificadas no arquivo de projeto.</span><span class="sxs-lookup"><span data-stu-id="7bb11-194">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="7bb11-195">Quando uma única estrutura de destino é especificada, use o elemento **TargetFramework**.</span><span class="sxs-lookup"><span data-stu-id="7bb11-195">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="7bb11-196">O seguinte arquivo de projeto de aplicativo do console demonstra como destinar ao .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="7bb11-196">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml
    <TargetFramework>netcoreapp2.0</TargetFramework>
```

```xml
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->

  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->

    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

```

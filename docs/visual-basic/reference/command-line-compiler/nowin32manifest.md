# <a name="nowin32manifest-visual-basic"></a>/nowin32manifest (Visual Basic)
Instrui o compilador não incorporar nenhum manifesto do aplicativo para o arquivo executável.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
/nowin32manifest  
```  
  
## <a name="remarks"></a>Comentários  
 Quando essa opção for usada, o aplicativo estará sujeito à virtualização no Windows Vista, a menos que você forneça um manifesto de aplicativo em um arquivo de recurso Win32 ou durante uma etapa de compilação posterior. Para obter mais informações sobre virtualização, consulte [implantação do ClickOnce no Windows Vista](https://docs.microsoft.com/visualstudio/deployment/clickonce-deployment-on-windows-vista).  
  
 Para obter mais informações sobre a criação de manifesto, consulte [/win32manifest (Visual Basic)](../../../visual-basic/reference/command-line-compiler/win32manifest.md).

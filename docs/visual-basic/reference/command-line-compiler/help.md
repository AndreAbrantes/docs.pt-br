# <a name="-help---visual-basic"></a>/help, /? (Visual Basic)
Exibe as opções do compilador.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
-help  
' -or-  
-?  
```  
  
## <a name="remarks"></a>Comentários  
 Se você incluir essa opção em uma compilação, nenhum arquivo de saída é criado e nenhum compilação ocorre.  
  
> [!NOTE]
>  O `-help` opção não está disponível de dentro do [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] ambiente de desenvolvimento; está disponível somente quando estiver compilando na linha de comando.  
  
## <a name="example"></a>Exemplo  
 O código a seguir exibe a Ajuda da linha de comando.  
  
```  
vbc -help  
```

# <a name="querying-xml-trees-visual-basic"></a>Consultando árvores XML (Visual Basic)
Esta seção fornece exemplos de consultas [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)].  
  
 Para obter mais informações sobre como escrever [!INCLUDE[vbteclinq](~/includes/vbteclinq-md.md)] consultas, consulte [Introdução a LINQ no Visual Basic](../../../../visual-basic/programming-guide/concepts/linq/getting-started-with-linq.md).

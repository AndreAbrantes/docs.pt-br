# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>Visual Studio IDE e ferramentas de suporte para LINQ (Visual Basic)
O ambiente de desenvolvimento integrado (IDE) do Visual Studio fornece os seguintes recursos que oferecem suporte ao desenvolvimento de aplicativos do LINQ:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 O Object Relational Designer é uma ferramenta de design visual que você pode usar em [LINQ to SQL](https://msdn.microsoft.com/library/bb386976) aplicativos para gerar classes no Visual Basic que representam os dados relacionais em um banco de dados subjacente. Para obter mais informações, consulte [LINQ to SQL Tools no Visual Studio](https://docs.microsoft.com/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2).  
  
## <a name="sqlmetal-command-line-tool"></a>Ferramenta de linha de comando SQLMetal  
 SQLMetal é uma ferramenta de linha de comando que pode ser usada em processos de compilação para gerar classes de bancos de dados existentes para uso em LINQ para aplicativos do SQL. Para obter mais informações, consulte [SqlMetal.exe (ferramenta de geração de código)](http://msdn.microsoft.com/library/819e5a96-7646-4fdb-b14b-fe31221b0614).  
  
## <a name="linq-aware-code-editor"></a>Editor de códigos com reconhecimento de LINQ  
 Editor de código do Visual Basic oferece suporte a LINQ com o IntelliSense e recursos de formatação.  
  
## <a name="visual-studio-debugger-support"></a>Suporte de depurador do Visual Studio  
 O depurador do Visual Studio oferece suporte à depuração de expressões de consulta. Para obter mais informações, consulte [depuração LINQ](https://docs.microsoft.com/visualstudio/debugger/debugging-linq).

# <a name="join-operations-visual-basic"></a><span data-ttu-id="61f5b-102">Ingressar em operações (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="61f5b-102">Join Operations (Visual Basic)</span></span>
<span data-ttu-id="61f5b-103">A *junção* de duas fontes de dados é a associação de objetos em uma fonte de dados com objetos que compartilham um atributo comum em outra fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="61f5b-103">A *join* of two data sources is the association of objects in one data source with objects that share a common attribute in another data source.</span></span>  
  
 <span data-ttu-id="61f5b-104">A junção é uma operação importante em consultas que têm como destino fontes de dados cujas relações entre si não podem ser seguidas diretamente.</span><span class="sxs-lookup"><span data-stu-id="61f5b-104">Joining is an important operation in queries that target data sources whose relationships to each other cannot be followed directly.</span></span> <span data-ttu-id="61f5b-105">Na programação orientada a objeto, isso pode significar uma correlação entre objetos que não são modelados, como a direção retroativa de uma relação unidirecional.</span><span class="sxs-lookup"><span data-stu-id="61f5b-105">In object-oriented programming, this could mean a correlation between objects that is not modeled, such as the backwards direction of a one-way relationship.</span></span> <span data-ttu-id="61f5b-106">Um exemplo de uma relação unidirecional é uma classe Cliente que tem uma propriedade do tipo Cidade, mas a classe Cidade ainda não tem uma propriedade que é uma coleção de objetos Cliente.</span><span class="sxs-lookup"><span data-stu-id="61f5b-106">An example of a one-way relationship is a Customer class that has a property of type City, but the City class does not have a property that is a collection of Customer objects.</span></span> <span data-ttu-id="61f5b-107">Se você tem uma lista de objetos Cidade e você quer encontrar todos os clientes em cada cidade, você pode usar uma operação de junção para encontrá-los.</span><span class="sxs-lookup"><span data-stu-id="61f5b-107">If you have a list of City objects and you want to find all the customers in each city, you could use a join operation to find them.</span></span>  
  
 <span data-ttu-id="61f5b-108">Os métodos de associação fornecidos no framework LINQ são <xref:System.Linq.Enumerable.Join%2A>e <xref:System.Linq.Enumerable.GroupJoin%2A>.</xref:System.Linq.Enumerable.GroupJoin%2A> </xref:System.Linq.Enumerable.Join%2A></span><span class="sxs-lookup"><span data-stu-id="61f5b-108">The join methods provided in the LINQ framework are <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="61f5b-109">Esses métodos executam junções equivalentes ou junções que correspondem a duas fontes de dados com base na igualdade de suas chaves.</span><span class="sxs-lookup"><span data-stu-id="61f5b-109">These methods perform equijoins, or joins that match two data sources based on equality of their keys.</span></span> <span data-ttu-id="61f5b-110">(Para comparação, o Transact-SQL oferece suporte a operadores de junção diferentes de 'equals', por exemplo, o 'less than'.) Em termos de banco de dados relacional, <xref:System.Linq.Enumerable.Join%2A>implementa uma junção interna, um tipo de junção em que apenas os objetos que têm uma correspondência no conjunto de dados são retornados.</xref:System.Linq.Enumerable.Join%2A></span><span class="sxs-lookup"><span data-stu-id="61f5b-110">(For comparison, Transact-SQL supports join operators other than 'equals', for example the 'less than' operator.) In relational database terms, <xref:System.Linq.Enumerable.Join%2A> implements an inner join, a type of join in which only those objects that have a match in the other data set are returned.</span></span> <span data-ttu-id="61f5b-111">O <xref:System.Linq.Enumerable.GroupJoin%2A>método não tem equivalente em termos de banco de dados relacional, mas ele implementa um superconjunto de junções internas e junções externas esquerdas.</xref:System.Linq.Enumerable.GroupJoin%2A></span><span class="sxs-lookup"><span data-stu-id="61f5b-111">The <xref:System.Linq.Enumerable.GroupJoin%2A> method has no direct equivalent in relational database terms, but it implements a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="61f5b-112">Uma junção externa esquerda é uma associação que retorna cada elemento da primeira fonte de dados (esquerda), mesmo que ele não tenha elementos correlacionados na fonte de dados.</span><span class="sxs-lookup"><span data-stu-id="61f5b-112">A left outer join is a join that returns each element of the first (left) data source, even if it has no correlated elements in the other data source.</span></span>  
  
 <span data-ttu-id="61f5b-113">A ilustração a seguir mostra uma visão conceitual de dois conjuntos e os elementos dentro desses conjuntos que estão incluídos em uma junção interna ou externa à esquerda.</span><span class="sxs-lookup"><span data-stu-id="61f5b-113">The following illustration shows a conceptual view of two sets and the elements within those sets that are included in either an inner join or a left outer join.</span></span>

# <a name="how-to-build-linq-to-xml-examples-visual-basic"></a>Como: compilar exemplos LINQ to XML (Visual Basic)
Os vários trechos e exemplos nesta documentação usam classes e tipos de uma variedade de namespaces. Para compilar o código em Visual Basic, você precisará fornecer instruções apropriadas de `Imports`.  
  
## <a name="example"></a>Exemplo  
 O código a seguir contém instruções de `Imports` que os exemplos de Visual Basic exigem para compilar e executar. Nem todas as instruções de `Imports` são necessárias para todos os exemplos.  
  
```vb  
Imports System  
Imports System.Diagnostics  
Imports System.Collections  
Imports System.Collections.Generic  
Imports System.Collections.Specialized  
Imports System.Text  
Imports System.Linq  
Imports System.Xml  
Imports System.Xml.Linq  
Imports System.Xml.Schema  
Imports System.Xml.XPath  
Imports System.Xml.Xsl  
Imports System.IO  
Imports System.Threading  
Imports System.Reflection  
Imports System.IO.Packaging  
```

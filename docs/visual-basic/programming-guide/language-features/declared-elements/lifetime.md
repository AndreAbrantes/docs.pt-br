 <span data-ttu-id="163b0-109">A *variável membro* (declarado no nível de módulo, fora de qualquer procedimento) geralmente tem a mesma duração do elemento no qual ela é declarada.</span><span class="sxs-lookup"><span data-stu-id="163b0-109">A *member variable* (declared at module level, outside any procedure) typically has the same lifetime as the element in which it is declared.</span></span> <span data-ttu-id="163b0-110">Existe uma variável não compartilhada declarada em uma classe ou estrutura como uma cópia separada para cada instância da classe ou estrutura na qual ela é declarada.</span><span class="sxs-lookup"><span data-stu-id="163b0-110">A nonshared variable declared in a class or structure exists as a separate copy for each instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="163b0-111">Cada variável como tem a mesma duração sua instância.</span><span class="sxs-lookup"><span data-stu-id="163b0-111">Each such variable has the same lifetime as its instance.</span></span> <span data-ttu-id="163b0-112">No entanto, um `Shared` variável tem apenas um único ciclo de vida, que dura durante todo o tempo de execução do seu aplicativo.</span><span class="sxs-lookup"><span data-stu-id="163b0-112">However, a `Shared` variable has only a single lifetime, which lasts for the entire time your application is running.</span></span>  
  
 <span data-ttu-id="163b0-113">A *variável local* (declaradas dentro de um procedimento) existe somente enquanto o procedimento no qual ela é declarada é executado.</span><span class="sxs-lookup"><span data-stu-id="163b0-113">A *local variable* (declared inside a procedure) exists only while the procedure in which it is declared is running.</span></span> <span data-ttu-id="163b0-114">Isso se aplica também aos parâmetros do procedimento e para qualquer função de retornada.</span><span class="sxs-lookup"><span data-stu-id="163b0-114">This applies also to that procedure's parameters and to any function return.</span></span> <span data-ttu-id="163b0-115">No entanto, se esse procedimento chama outros procedimentos, as variáveis locais retêm seus valores durante a execução de procedimentos chamados.</span><span class="sxs-lookup"><span data-stu-id="163b0-115">However, if that procedure calls other procedures, the local variables retain their values while the called procedures are running.</span></span>  
  
## <a name="beginning-of-lifetime"></a><span data-ttu-id="163b0-116">Início do tempo de vida</span><span class="sxs-lookup"><span data-stu-id="163b0-116">Beginning of Lifetime</span></span>

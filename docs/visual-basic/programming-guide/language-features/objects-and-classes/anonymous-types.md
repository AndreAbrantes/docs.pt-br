## <a name="an-anonymous-type-definition"></a><span data-ttu-id="87922-194">Uma definição de tipo anônimo</span><span class="sxs-lookup"><span data-stu-id="87922-194">An Anonymous Type Definition</span></span>

 <span data-ttu-id="87922-196">Se o tipo anônimo contém pelo menos uma propriedade de chave, a definição substitui três membros herdados <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>e <xref:System.Object.ToString%2A>.</xref:System.Object.ToString%2A> </xref:System.Object.GetHashCode%2A> </xref:System.Object.Equals%2A> </xref:System.Object></span><span class="sxs-lookup"><span data-stu-id="87922-196">If the anonymous type contains at least one key property, the definition overrides three members inherited from <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>, and <xref:System.Object.ToString%2A>.</span></span> <span data-ttu-id="87922-197">O código produzido para teste de igualdade e determinar que o valor de código hash considera apenas as propriedades de chave.</span><span class="sxs-lookup"><span data-stu-id="87922-197">The code produced for testing equality and determining the hash code value considers only the key properties.</span></span> <span data-ttu-id="87922-198">Se o tipo anônimo contiver sem propriedades chave, somente <xref:System.Object.ToString%2A>é substituído.</xref:System.Object.ToString%2A></span><span class="sxs-lookup"><span data-stu-id="87922-198">If the anonymous type contains no key properties, only <xref:System.Object.ToString%2A> is overridden.</span></span> <span data-ttu-id="87922-199">Propriedades explicitamente nomeadas de um tipo anônimo não entram em conflito com esses métodos gerados.</span><span class="sxs-lookup"><span data-stu-id="87922-199">Explicitly named properties of an anonymous type cannot conflict with these generated methods.</span></span> <span data-ttu-id="87922-200">Ou seja, você não pode usar `.Equals`, `.GetHashCode`, ou `.ToString` para nomear uma propriedade.</span><span class="sxs-lookup"><span data-stu-id="87922-200">That is, you cannot use `.Equals`, `.GetHashCode`, or `.ToString` to name a property.</span></span>

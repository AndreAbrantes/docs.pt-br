1. <span data-ttu-id="b5241-105">**Acessibilidade.**</span><span class="sxs-lookup"><span data-stu-id="b5241-105">**Accessibility.**</span></span> <span data-ttu-id="b5241-106">Elimina qualquer sobrecarga com um nível de acesso que impeça o código de chamada de chamá-lo.</span><span class="sxs-lookup"><span data-stu-id="b5241-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="b5241-107">**Número de parâmetros.**</span><span class="sxs-lookup"><span data-stu-id="b5241-107">**Number of Parameters.**</span></span> <span data-ttu-id="b5241-108">Elimina qualquer sobrecarga que defina um número diferente de parâmetros do que são fornecidos na chamada.</span><span class="sxs-lookup"><span data-stu-id="b5241-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>

     <span data-ttu-id="b5241-113">Nesta etapa, elimina qualquer sobrecarga para a qual os tipos de dados dos argumentos de chamada não podem ser convertidos para os tipos de parâmetros definidos na sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="b5241-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>

5. <span data-ttu-id="b5241-117">**Pelo menos ampliando.**</span><span class="sxs-lookup"><span data-stu-id="b5241-117">**Least Widening.**</span></span> <span data-ttu-id="b5241-118">O compilador considera as sobrecargas restantes em pares.</span><span class="sxs-lookup"><span data-stu-id="b5241-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="b5241-119">Para cada par, ele compara os tipos de dados dos parâmetros definidos.</span><span class="sxs-lookup"><span data-stu-id="b5241-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="b5241-120">Se os tipos em uma das sobrecargas se expandirem para os tipos correspondentes no outro, o compilador elimina este último.</span><span class="sxs-lookup"><span data-stu-id="b5241-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="b5241-121">Ou seja, retém a sobrecarga que requer a menor quantidade de alargamento.</span><span class="sxs-lookup"><span data-stu-id="b5241-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="b5241-122">**Candidato único.**</span><span class="sxs-lookup"><span data-stu-id="b5241-122">**Single Candidate.**</span></span> <span data-ttu-id="b5241-123">Ele continua considerando sobrecargas em pares até que apenas uma sobrecarga permaneça, e resolve a chamada para essa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="b5241-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="b5241-124">Se o compilador não puder reduzir as sobrecargas a um único candidato, ele gerará um erro.</span><span class="sxs-lookup"><span data-stu-id="b5241-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>

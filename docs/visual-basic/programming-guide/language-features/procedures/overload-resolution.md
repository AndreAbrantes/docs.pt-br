# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="78890-102">Resolução de sobrecarga (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="78890-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="78890-103">Quando o compilador de Visual Basic encontra uma chamada para um procedimento que é definido em várias versões sobrecarregadas, o compilador deve decidir qual das sobrecargas a chamar.</span><span class="sxs-lookup"><span data-stu-id="78890-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="78890-104">Ele faz isso executando as seguintes etapas:</span><span class="sxs-lookup"><span data-stu-id="78890-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="78890-105">**Acessibilidade.**</span><span class="sxs-lookup"><span data-stu-id="78890-105">**Accessibility.**</span></span> <span data-ttu-id="78890-106">Ele elimina qualquer sobrecarga com um nível de acesso que impede o código de chamada de chamá-lo.</span><span class="sxs-lookup"><span data-stu-id="78890-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="78890-107">**Número de parâmetros.**</span><span class="sxs-lookup"><span data-stu-id="78890-107">**Number of Parameters.**</span></span> <span data-ttu-id="78890-108">Ele elimina qualquer sobrecarga que define um número diferente de parâmetros que são fornecidos na chamada.</span><span class="sxs-lookup"><span data-stu-id="78890-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="78890-109">**Tipos de dados de parâmetro.**</span><span class="sxs-lookup"><span data-stu-id="78890-109">**Parameter Data Types.**</span></span> <span data-ttu-id="78890-110">O compilador fornece preferência de métodos de instância sobre métodos de extensão.</span><span class="sxs-lookup"><span data-stu-id="78890-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="78890-111">Se qualquer método de instância for encontrado e exigir apenas conversões ampliadas para corresponder à chamada de procedimento, todos os métodos de extensão serão descartados e o compilador continuará com apenas os candidatos do método de instância.</span><span class="sxs-lookup"><span data-stu-id="78890-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="78890-112">Se nenhum método de instância desse tipo for encontrado, ele continuará com os métodos de instância e extensão.</span><span class="sxs-lookup"><span data-stu-id="78890-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="78890-113">Nesta etapa, ele elimina qualquer sobrecarga para a qual os tipos de dados dos argumentos de chamada não podem ser convertidos nos tipos de parâmetro definidos na sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="78890-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="78890-114">**Refinando conversões.**</span><span class="sxs-lookup"><span data-stu-id="78890-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="78890-115">Ele elimina qualquer sobrecarga que exija uma conversão de restrição dos tipos de argumento de chamada para os tipos de parâmetro definidos.</span><span class="sxs-lookup"><span data-stu-id="78890-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="78890-116">Isso é verdadeiro se a opção de verificação de tipo ([instrução Option Strict](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) é `On` ou `Off`.</span><span class="sxs-lookup"><span data-stu-id="78890-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="78890-117">**Menos alargamento.**</span><span class="sxs-lookup"><span data-stu-id="78890-117">**Least Widening.**</span></span> <span data-ttu-id="78890-118">O compilador considera as sobrecargas restantes em pares.</span><span class="sxs-lookup"><span data-stu-id="78890-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="78890-119">Para cada par, ele compara os tipos de dados dos parâmetros definidos.</span><span class="sxs-lookup"><span data-stu-id="78890-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="78890-120">Se os tipos em uma das sobrecargas forem ampliados para os tipos correspondentes no outro, o compilador eliminará o último.</span><span class="sxs-lookup"><span data-stu-id="78890-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="78890-121">Ou seja, ele retém a sobrecarga que requer a menor quantidade de alargamento.</span><span class="sxs-lookup"><span data-stu-id="78890-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="78890-122">**Único candidato.**</span><span class="sxs-lookup"><span data-stu-id="78890-122">**Single Candidate.**</span></span> <span data-ttu-id="78890-123">Ele continua considerando sobrecargas em pares até que apenas uma sobrecarga permaneça e resolva a chamada para essa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="78890-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="78890-124">Se o compilador não puder reduzir as sobrecargas para um único candidato, ele gerará um erro.</span><span class="sxs-lookup"><span data-stu-id="78890-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>

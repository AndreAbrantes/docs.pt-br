# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="a0dc0-102">Resolução de sobrecarga (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="a0dc0-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="a0dc0-103">Quando o compilador do Visual Basic encontra uma chamada para um procedimento que é definido em várias versões sobrecarregadas, o compilador deve decidir qual das sobrecargas para chamar.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="a0dc0-104">Ele faz isso executando as seguintes etapas:</span><span class="sxs-lookup"><span data-stu-id="a0dc0-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="a0dc0-105">**Acessibilidade.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-105">**Accessibility.**</span></span> <span data-ttu-id="a0dc0-106">Elimina qualquer sobrecarregamento com um nível de acesso que impede que o código de chamada chamando-o.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="a0dc0-107">**Número de parâmetros.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-107">**Number of Parameters.**</span></span> <span data-ttu-id="a0dc0-108">Elimina qualquer sobrecarregamento que define um número diferente de parâmetros que são fornecidos na chamada.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="a0dc0-109">**Tipos de dados do parâmetro.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-109">**Parameter Data Types.**</span></span> <span data-ttu-id="a0dc0-110">O compilador dá preferência de métodos de instância sobre métodos de extensão.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="a0dc0-111">Se qualquer método de instância que requer somente conversões para corresponder a chamada de procedimento de expansão, todos os métodos de extensão são descartados e o compilador continua com apenas os candidatos de método de instância.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="a0dc0-112">Se esse método de instância não for encontrado, ele continuará com a instância e métodos de extensão.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="a0dc0-113">Nesta etapa, ela elimina qualquer sobrecarga para o qual os tipos de dados dos argumentos de chamada não podem ser convertidos para tipos de parâmetro definido na sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="a0dc0-114">**Conversões de estreitamento.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="a0dc0-115">Elimina qualquer sobrecarregamento que requer uma conversão de estreitamento dos tipos de argumento de chamada para os tipos de parâmetro definido.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="a0dc0-116">Isso é verdadeiro se a verificação de tipo muda ([instrução Option Strict](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) é `On` ou `Off`.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="a0dc0-117">**Mínimo de ampliação.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-117">**Least Widening.**</span></span> <span data-ttu-id="a0dc0-118">O compilador considera as sobrecargas restantes em pares.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="a0dc0-119">Para cada par, ele compara os tipos de dados dos parâmetros definidos.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="a0dc0-120">Se os tipos em uma das sobrecargas todos ampliam para os tipos correspondentes no outro, o compilador elimina o último.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="a0dc0-121">Ou seja, ele retém a sobrecarga que exige o mínimo de ampliação.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="a0dc0-122">**Candidato único.**</span><span class="sxs-lookup"><span data-stu-id="a0dc0-122">**Single Candidate.**</span></span> <span data-ttu-id="a0dc0-123">Continua considerando sobrecargas em pares até que somente uma sobrecarga permanece e resolve a chamada para essa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="a0dc0-124">Se o compilador não pode reduzir as sobrecargas a um candidato único, ele gera um erro.</span><span class="sxs-lookup"><span data-stu-id="a0dc0-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>

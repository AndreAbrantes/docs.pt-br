# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="3f3d6-102">Resolução de sobrecarga (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="3f3d6-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="3f3d6-103">Quando o compilador do Visual Basic encontra uma chamada para um procedimento que é definido em várias versões sobrecarregadas, o compilador deve decidir qual das sobrecargas para chamar.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="3f3d6-104">Ele faz isso executando as seguintes etapas:</span><span class="sxs-lookup"><span data-stu-id="3f3d6-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="3f3d6-105">**Acessibilidade.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-105">**Accessibility.**</span></span> <span data-ttu-id="3f3d6-106">Ele elimina qualquer sobrecarga com um nível de acesso que impede que o código de chamada chamando-o.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="3f3d6-107">**Número de parâmetros.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-107">**Number of Parameters.**</span></span> <span data-ttu-id="3f3d6-108">Ele elimina qualquer sobrecarga que define um número diferente de parâmetros que são fornecidos na chamada.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="3f3d6-109">**Tipos de dados do parâmetro.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-109">**Parameter Data Types.**</span></span> <span data-ttu-id="3f3d6-110">O compilador dá preferência de métodos de instância sobre métodos de extensão.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="3f3d6-111">Se qualquer método de instância for encontrado que requer somente conversões de acordo com a chamada de procedimento de expansão, todos os métodos de extensão são descartados e o compilador continua com apenas os candidatos a método de instância.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="3f3d6-112">Se nenhum método de instância desse tipo for encontrado, ele continua com a instância e métodos de extensão.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="3f3d6-113">Nesta etapa, ele elimina qualquer sobrecarga para o qual os tipos de dados dos argumentos de chamada não podem ser convertidos para os tipos de parâmetro definidos na sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="3f3d6-114">**Conversões de estreitamento.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="3f3d6-115">Ele elimina qualquer sobrecarga que requer uma conversão redutora entre os tipos de argumento de chamada para os tipos de parâmetro definidas.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="3f3d6-116">Isso é verdadeiro se a verificação de tipo alternar ([instrução Option Strict](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) é `On` ou `Off`.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="3f3d6-117">**Mínimo de ampliação.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-117">**Least Widening.**</span></span> <span data-ttu-id="3f3d6-118">O compilador considera as sobrecargas restantes em pares.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="3f3d6-119">Para cada par, ele compara os tipos de dados dos parâmetros definidos.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="3f3d6-120">Se os tipos em uma das sobrecargas todos são ampliados com os tipos correspondentes no outro, o compilador elimina o último.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="3f3d6-121">Ou seja, ele retém a sobrecarga que exige o mínimo de ampliação.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="3f3d6-122">**Única de candidato.**</span><span class="sxs-lookup"><span data-stu-id="3f3d6-122">**Single Candidate.**</span></span> <span data-ttu-id="3f3d6-123">Ele continua considerando sobrecargas em pares até que apenas uma sobrecarregam permanece, além de resolver a chamada para essa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="3f3d6-124">Se o compilador não é possível reduzir as sobrecargas a um candidato único, ele gera um erro.</span><span class="sxs-lookup"><span data-stu-id="3f3d6-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>

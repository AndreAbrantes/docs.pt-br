## <a name="modifiable-and-nonmodifiable-arguments"></a><span data-ttu-id="4d447-119">Argumentos modificáveis e</span><span class="sxs-lookup"><span data-stu-id="4d447-119">Modifiable and Nonmodifiable Arguments</span></span>  
 <span data-ttu-id="4d447-120">Um *argumento modificável* é aquele com um elemento subjacente modificável.</span><span class="sxs-lookup"><span data-stu-id="4d447-120">A *modifiable argument* is one with a modifiable underlying element.</span></span> <span data-ttu-id="4d447-121">O código de chamada pode armazenar um novo valor a qualquer momento, e se você passar o argumento [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), o código no procedimento também pode modificar o elemento subjacente no código de chamada.</span><span class="sxs-lookup"><span data-stu-id="4d447-121">The calling code can store a new value at any time, and if you pass the argument [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), the code in the procedure can also modify the underlying element in the calling code.</span></span>  
  
 <span data-ttu-id="4d447-122">Um *argumento não modificável* tem um elemento subjacente não modificável ou é passado [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span><span class="sxs-lookup"><span data-stu-id="4d447-122">A *nonmodifiable argument* either has a nonmodifiable underlying element or is passed [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span></span> <span data-ttu-id="4d447-123">O procedimento não pode modificar o elemento subjacente no código de chamada, mesmo se for um elemento modificável.</span><span class="sxs-lookup"><span data-stu-id="4d447-123">The procedure cannot modify the underlying element in the calling code, even if it is a modifiable element.</span></span> <span data-ttu-id="4d447-124">Se for um elemento não modificável, o código de chamada em si não é possível modificá-lo.</span><span class="sxs-lookup"><span data-stu-id="4d447-124">If it is a nonmodifiable element, the calling code itself cannot modify it.</span></span>  
  
 <span data-ttu-id="4d447-125">O procedimento chamado pode modificar sua cópia local de um argumento não modificável, mas essa modificação não afeta o elemento subjacente no código de chamada.</span><span class="sxs-lookup"><span data-stu-id="4d447-125">The called procedure might modify its local copy of a nonmodifiable argument, but that modification does not affect the underlying element in the calling code.</span></span>

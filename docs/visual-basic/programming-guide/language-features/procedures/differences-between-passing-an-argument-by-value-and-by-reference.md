## <a name="passing-by-value"></a><span data-ttu-id="1a113-106">passagem por valor</span><span class="sxs-lookup"><span data-stu-id="1a113-106">Passing by Value</span></span>  
 <span data-ttu-id="1a113-107">Passar um argumento *pelo valor* especificando o [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) palavra-chave para o parâmetro correspondente na definição do procedimento.</span><span class="sxs-lookup"><span data-stu-id="1a113-107">You pass an argument *by value* by specifying the [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="1a113-108">Quando você usa este mecanismo de passagem, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] copia o valor do elemento de programação subjacente em uma variável local no procedimento.</span><span class="sxs-lookup"><span data-stu-id="1a113-108">When you use this passing mechanism, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] copies the value of the underlying programming element into a local variable in the procedure.</span></span> <span data-ttu-id="1a113-109">O código do procedimento não tem nenhum acesso ao elemento subjacente no código de chamada.</span><span class="sxs-lookup"><span data-stu-id="1a113-109">The procedure code does not have any access to the underlying element in the calling code.</span></span>  
  
## <a name="passing-by-reference"></a><span data-ttu-id="1a113-110">Passagem por referência</span><span class="sxs-lookup"><span data-stu-id="1a113-110">Passing by Reference</span></span>  
 <span data-ttu-id="1a113-111">Passar um argumento *por referência* especificando o [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) palavra-chave para o parâmetro correspondente na definição do procedimento.</span><span class="sxs-lookup"><span data-stu-id="1a113-111">You pass an argument *by reference* by specifying the [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="1a113-112">Quando você usa este mecanismo de passagem, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] fornece o procedimento uma referência direta para o elemento de programação subjacente no código de chamada.</span><span class="sxs-lookup"><span data-stu-id="1a113-112">When you use this passing mechanism, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] gives the procedure a direct reference to the underlying programming element in the calling code.</span></span>  
  
## <a name="passing-mechanism-and-element-type"></a><span data-ttu-id="1a113-113">Mecanismo de passagem e tipo de elemento</span><span class="sxs-lookup"><span data-stu-id="1a113-113">Passing Mechanism and Element Type</span></span>  
 <span data-ttu-id="1a113-114">A escolha do mecanismo de passagem não é o mesmo que a classificação de tipo do elemento.</span><span class="sxs-lookup"><span data-stu-id="1a113-114">The choice of passing mechanism is not the same as the classification of the underlying element type.</span></span> <span data-ttu-id="1a113-115">Passagem por valor ou referência se refere ao que [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] fornece ao código do procedimento.</span><span class="sxs-lookup"><span data-stu-id="1a113-115">Passing by value or by reference refers to what [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] supplies to the procedure code.</span></span> <span data-ttu-id="1a113-116">Um tipo de valor ou tipo de referência se refere a como um elemento de programação é armazenado na memória.</span><span class="sxs-lookup"><span data-stu-id="1a113-116">A value type or reference type refers to how a programming element is stored in memory.</span></span>  
  
 <span data-ttu-id="1a113-117">No entanto, o mecanismo de passagem e o tipo de elemento são correlacionados.</span><span class="sxs-lookup"><span data-stu-id="1a113-117">However, the passing mechanism and element type are interrelated.</span></span> <span data-ttu-id="1a113-118">O valor de um tipo de referência é um ponteiro para os dados em outro lugar na memória.</span><span class="sxs-lookup"><span data-stu-id="1a113-118">The value of a reference type is a pointer to the data elsewhere in memory.</span></span> <span data-ttu-id="1a113-119">Isso significa que, quando você passar um tipo de referência por valor, o código do procedimento tem um ponteiro para os dados do elemento, mesmo que ele não pode acessar o elemento subjacente.</span><span class="sxs-lookup"><span data-stu-id="1a113-119">This means that when you pass a reference type by value, the procedure code has a pointer to the underlying element's data, even though it cannot access the underlying element itself.</span></span> <span data-ttu-id="1a113-120">Por exemplo, se o elemento é uma variável de matriz, o código do procedimento não tem acesso à variável em si, mas ele pode acessar os membros da matriz.</span><span class="sxs-lookup"><span data-stu-id="1a113-120">For example, if the element is an array variable, the procedure code does not have access to the variable itself, but it can access the array members.</span></span>

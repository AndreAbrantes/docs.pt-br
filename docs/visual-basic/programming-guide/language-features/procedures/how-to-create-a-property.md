# <a name="how-to-create-a-property-visual-basic"></a><span data-ttu-id="f2205-102">Como criar uma propriedade (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="f2205-102">How to: Create a Property (Visual Basic)</span></span>
<span data-ttu-id="f2205-103">Coloque uma definição de propriedade entre uma `Property` instrução e um `End Property` instrução.</span><span class="sxs-lookup"><span data-stu-id="f2205-103">You enclose a property definition between a `Property` statement and an `End Property` statement.</span></span> <span data-ttu-id="f2205-104">Nessa definição você define um `Get` procedimento, uma `Set` procedimento, ou ambos.</span><span class="sxs-lookup"><span data-stu-id="f2205-104">Within this definition you define a `Get` procedure, a `Set` procedure, or both.</span></span> <span data-ttu-id="f2205-105">Todo o código da propriedade está situado nesses procedimentos.</span><span class="sxs-lookup"><span data-stu-id="f2205-105">All the property's code lies within these procedures.</span></span>  
  
 <span data-ttu-id="f2205-106">O `Get` procedimento recupera o valor da propriedade e o `Set` procedimento armazena um valor.</span><span class="sxs-lookup"><span data-stu-id="f2205-106">The `Get` procedure retrieves the property's value, and the `Set` procedure stores a value.</span></span> <span data-ttu-id="f2205-107">Se você quiser que a propriedade para ter acesso de leitura/gravação, você deve definir ambos os procedimentos.</span><span class="sxs-lookup"><span data-stu-id="f2205-107">If you want the property to have read/write access, you must define both procedures.</span></span> <span data-ttu-id="f2205-108">Para uma propriedade somente leitura, você define apenas `Get`, e para uma propriedade somente gravação, você define apenas `Set`.</span><span class="sxs-lookup"><span data-stu-id="f2205-108">For a read-only property, you define only `Get`, and for a write-only property, you define only `Set`.</span></span>  
  
### <a name="to-create-a-property"></a><span data-ttu-id="f2205-109">Para criar uma propriedade</span><span class="sxs-lookup"><span data-stu-id="f2205-109">To create a property</span></span>  
  
1.  <span data-ttu-id="f2205-110">Fora de qualquer propriedade ou procedimento, utilize uma [declaração de propriedade](../../../../visual-basic/language-reference/statements/property-statement.md), seguido por um `End Property` instrução.</span><span class="sxs-lookup"><span data-stu-id="f2205-110">Outside any property or procedure, use a [Property Statement](../../../../visual-basic/language-reference/statements/property-statement.md), followed by an `End Property` statement.</span></span>  
  
2.  <span data-ttu-id="f2205-111">Se a propriedade usa parâmetros, execute o `Property` palavra-chave com o nome do procedimento, a lista de parâmetros entre parênteses.</span><span class="sxs-lookup"><span data-stu-id="f2205-111">If the property takes parameters, follow the `Property` keyword with the name of the procedure, then the parameter list in parentheses.</span></span>  
  
3.  <span data-ttu-id="f2205-112">Siga os parênteses com uma `As` cláusula para especificar o tipo de dados do valor da propriedade.</span><span class="sxs-lookup"><span data-stu-id="f2205-112">Follow the parentheses with an `As` clause to specify the data type of the property's value.</span></span> <span data-ttu-id="f2205-113">Você deve especificar o tipo de dados, mesmo para uma propriedade somente gravação.</span><span class="sxs-lookup"><span data-stu-id="f2205-113">You must specify the data type even for a write-only property.</span></span>  
  
4.  <span data-ttu-id="f2205-114">Adicionar `Get` e `Set` procedimentos, conforme apropriado.</span><span class="sxs-lookup"><span data-stu-id="f2205-114">Add `Get` and `Set` procedures, as appropriate.</span></span> <span data-ttu-id="f2205-115">Consulte as instruções a seguir.</span><span class="sxs-lookup"><span data-stu-id="f2205-115">See the following directions.</span></span>  
  
### <a name="to-create-a-get-procedure-that-retrieves-a-property-value"></a><span data-ttu-id="f2205-116">Para criar um procedimento Get que recupera um valor de propriedade</span><span class="sxs-lookup"><span data-stu-id="f2205-116">To create a Get procedure that retrieves a property value</span></span>  
  
1.  <span data-ttu-id="f2205-117">Entre o `Property` e `End Property` escrever instruções, uma [obter instrução](../../../../visual-basic/language-reference/statements/get-statement.md), seguido por um `End Get` instrução.</span><span class="sxs-lookup"><span data-stu-id="f2205-117">Between the `Property` and `End Property` statements, write a [Get Statement](../../../../visual-basic/language-reference/statements/get-statement.md), followed by an `End Get` statement.</span></span> <span data-ttu-id="f2205-118">Você não precisa definir nenhum parâmetro para o `Get` procedimento.</span><span class="sxs-lookup"><span data-stu-id="f2205-118">You do not need to define any parameters for the `Get` procedure.</span></span>  
  
2.  <span data-ttu-id="f2205-119">Coloque as declarações de código para recuperar o valor da propriedade entre o `Get` e `End Get` instruções.</span><span class="sxs-lookup"><span data-stu-id="f2205-119">Place the code statements to retrieve the property's value between the `Get` and `End Get` statements.</span></span> <span data-ttu-id="f2205-120">Esse código pode incluir outros cálculos e manipulações de dados, além de gerar e retornar o valor da propriedade.</span><span class="sxs-lookup"><span data-stu-id="f2205-120">This code can include other calculations and data manipulations in addition to generating and returning the property's value.</span></span>  
  
3.  <span data-ttu-id="f2205-121">Use um `Return` instrução para retornar o valor da propriedade para o código de chamada.</span><span class="sxs-lookup"><span data-stu-id="f2205-121">Use a `Return` statement to return the property's value to the calling code.</span></span>  
  
 <span data-ttu-id="f2205-122">Você deve escrever uma `Get` procedimento para uma propriedade de leitura-gravação e para uma propriedade somente leitura.</span><span class="sxs-lookup"><span data-stu-id="f2205-122">You must write a `Get` procedure for a read-write property and for a read-only property.</span></span> <span data-ttu-id="f2205-123">Você não deve definir um `Get` procedimento para uma propriedade somente gravação.</span><span class="sxs-lookup"><span data-stu-id="f2205-123">You must not define a `Get` procedure for a write-only property.</span></span>  
  
### <a name="to-create-a-set-procedure-that-writes-a-propertys-value"></a><span data-ttu-id="f2205-124">Para criar um procedimento Set que grava um valor de propriedade</span><span class="sxs-lookup"><span data-stu-id="f2205-124">To create a Set procedure that writes a property's value</span></span>  
  
1.  <span data-ttu-id="f2205-125">Entre o `Property` e `End Property` instruções, gravar um [instrução Set](../../../../visual-basic/language-reference/statements/set-statement.md), seguido por um `End Set` instrução.</span><span class="sxs-lookup"><span data-stu-id="f2205-125">Between the `Property` and `End Property` statements, write a [Set Statement](../../../../visual-basic/language-reference/statements/set-statement.md), followed by an `End Set` statement.</span></span>  
  
2.  <span data-ttu-id="f2205-126">No `Set` instrução, siga o `Set` palavra-chave com uma lista de parâmetros entre parênteses.</span><span class="sxs-lookup"><span data-stu-id="f2205-126">In the `Set` statement, follow the `Set` keyword with a parameter list in parentheses.</span></span> <span data-ttu-id="f2205-127">Esta lista de parâmetros deve incluir pelo menos um parâmetro de valor para o valor passado pelo código de chamada.</span><span class="sxs-lookup"><span data-stu-id="f2205-127">This parameter list must include at least a value parameter for the value passed by the calling code.</span></span> <span data-ttu-id="f2205-128">O nome padrão para este parâmetro de valor é `Value`, mas você pode usar um nome diferente, se apropriado.</span><span class="sxs-lookup"><span data-stu-id="f2205-128">The default name for this value parameter is `Value`, but you can use a different name if appropriate.</span></span> <span data-ttu-id="f2205-129">O parâmetro de valor deve ter os mesmos dados de tipo da propriedade.</span><span class="sxs-lookup"><span data-stu-id="f2205-129">The value parameter must have the same data type as the property itself.</span></span>  
  
3.  <span data-ttu-id="f2205-130">Coloque as declarações de código para armazenar um valor na propriedade entre o `Set` e `End Set` instruções.</span><span class="sxs-lookup"><span data-stu-id="f2205-130">Place the code statements to store a value in the property between the `Set` and `End Set` statements.</span></span> <span data-ttu-id="f2205-131">Esse código pode incluir outros cálculos e manipulações de dados, além de validação e o valor da propriedade de armazenamento.</span><span class="sxs-lookup"><span data-stu-id="f2205-131">This code can include other calculations and data manipulations in addition to validating and storing the property's value.</span></span>  
  
4.  <span data-ttu-id="f2205-132">Use o parâmetro de valor para aceitar o valor fornecido pelo código de chamada.</span><span class="sxs-lookup"><span data-stu-id="f2205-132">Use the value parameter to accept the value supplied by the calling code.</span></span> <span data-ttu-id="f2205-133">Você pode armazenar esse valor diretamente em uma instrução de atribuição ou usá-lo em uma expressão para calcular o valor interno a ser armazenado.</span><span class="sxs-lookup"><span data-stu-id="f2205-133">You can either store this value directly in an assignment statement, or use it in an expression to calculate the internal value to be stored.</span></span>  
  
 <span data-ttu-id="f2205-134">Você deve escrever uma `Set` procedimento para uma propriedade de leitura-gravação e para uma propriedade somente gravação.</span><span class="sxs-lookup"><span data-stu-id="f2205-134">You must write a `Set` procedure for a read-write property and for a write-only property.</span></span> <span data-ttu-id="f2205-135">Você não deve definir um `Set` procedimento para uma propriedade somente leitura.</span><span class="sxs-lookup"><span data-stu-id="f2205-135">You must not define a `Set` procedure for a read-only property.</span></span>

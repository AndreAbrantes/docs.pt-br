## <a name="perform-a-left-outer-join-by-using-the-group-join-clause"></a><span data-ttu-id="985e0-124">Realizar uma junção externa esquerda, usando a cláusula Group Join</span><span class="sxs-lookup"><span data-stu-id="985e0-124">Perform a Left Outer Join by Using the Group Join Clause</span></span>

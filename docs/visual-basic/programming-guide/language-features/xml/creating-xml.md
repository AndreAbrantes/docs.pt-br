# <a name="creating-xml-in-visual-basic"></a>Criando XML no Visual Basic
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]permite que você use *literais XML* diretamente no seu código. A sintaxe literal XML representa [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] objetos e é semelhante à sintaxe XML 1.0. Isso facilita criar elementos XML, documentos e fragmentos por programação porque seu código possui a mesma estrutura que o XML final.

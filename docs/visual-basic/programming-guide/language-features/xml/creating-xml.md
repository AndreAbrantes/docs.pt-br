# <a name="creating-xml-in-visual-basic"></a>Criando XML no Visual Basic
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]permite que você use *literais XML* diretamente no seu código. A sintaxe de literais XML representa [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] objetos e é semelhante à sintaxe XML 1.0. Isso torna mais fácil criar elementos XML, documentos e fragmentos por programação porque seu código tem a mesma estrutura que o XML final.

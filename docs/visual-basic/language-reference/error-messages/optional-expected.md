# <a name="39optional39-expected"></a>&#39; opcional &#39; esperado
Um argumento opcional em uma declaração de procedimento é seguido por um argumento necessário. Cada argumento após um argumento opcional também deve ser opcional.  
  
 **ID do erro:** BC30202  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Se o argumento deve ser necessários, movê-la para preceder o primeiro argumento opcional na lista de argumentos.  
  
2.  Se o argumento deve ser opcional, use o `Optional` palavra-chave.

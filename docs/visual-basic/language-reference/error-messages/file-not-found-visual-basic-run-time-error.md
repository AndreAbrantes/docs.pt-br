# <a name="file-not-found-visual-basic-run-time-error"></a>Arquivo não localizado (erro de tempo de execução do Visual Basic)
O arquivo não foi encontrado na qual especificada. O erro tem as seguintes causas possíveis:  
  
-   Uma declaração se refere a um arquivo que não existe.  
  
-   Foi feita uma tentativa para chamar um procedimento em uma biblioteca de vínculo dinâmico (DLL), mas a biblioteca especificada na `Lib` cláusula do `Declare` instrução não pode ser encontrada.  
  
-   Você tentou abrir um projeto ou carregar um arquivo de texto que não existe.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a ortografia do nome do arquivo e a especificação do caminho.

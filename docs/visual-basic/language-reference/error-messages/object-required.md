# <a name="object-required-visual-basic"></a>Objeto obrigatório (Visual Basic)
Referências a propriedades e métodos geralmente exigem um qualificador de objeto explícito. Este é o caso.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se as referências a um método ou propriedade do objeto tem o qualificador de objeto válido. Especifique um qualificador de objeto, se você não fornecer uma.  
  
2.  Verifique a ortografia do qualificador do objeto e verifique se que o objeto está visível na parte do programa no qual você está fazendo referência a ele.  
  
3.  Se um caminho é fornecido para um aplicativo de host **abrir arquivo** de comando, verifique se os argumentos estão corretos.  
  
4.  Verifique a documentação do objeto e assegure-se de que a ação é válida.

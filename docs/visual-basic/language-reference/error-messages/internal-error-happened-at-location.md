# <a name="internal-error-happened-at-ltlocationgt"></a>Erro interno ocorreu em &lt;local&gt;
Erro interno. A linha na qual ele ocorreu está contida na mensagem de erro.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se esse erro não foi gerado pelo `Error` instrução ou `Raise` método; se não, estava entre em contato com o Atendimento Microsoft para relatar as condições sob as quais a mensagem apareceu.

## <a name="to-correct-this-error"></a><span data-ttu-id="067ae-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="067ae-108">To correct this error</span></span>

    ```vb
        Struct.demoArray(2) = 777
    End Sub
    ```

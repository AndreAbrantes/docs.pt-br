# <a name="subscript-out-of-range-visual-basic"></a>Subscrito fora do intervalo (Visual Basic)
Um subscrito de matriz não é válido porque está fora do intervalo permitido. O valor mais baixo de subscrito para uma dimensão é sempre 0 e o valor de subscrito mais alto é retornado pelo `GetUpperBound` método para a dimensão.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o subscrito para que ele esteja dentro do intervalo válido.

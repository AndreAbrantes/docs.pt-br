# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Não é possível localizar o arquivo &#39; &lt;nome de arquivo&gt;&#39;

 **ID do erro:** BC30655  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reinstale o Visual Studio.

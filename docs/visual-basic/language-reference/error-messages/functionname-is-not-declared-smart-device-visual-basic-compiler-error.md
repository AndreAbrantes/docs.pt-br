# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>&#39; &lt;functionname&gt;&#39; não é declarada (dispositivo inteligente/básico compilador erro Visual)
<`functionname`> não está declarado. A funcionalidade de e/s de arquivo está disponível na `Microsoft.VisualBasic` namespace, mas a versão de destino do .NET Compact Framework não suporta.  
  
 **ID do erro:** BC30766  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Executar operações de arquivo com funções definidas no `System.IO` namespace.

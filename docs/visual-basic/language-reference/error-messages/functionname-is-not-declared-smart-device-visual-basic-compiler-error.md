# <a name="functionname-is-not-declared-smart-devicevisual-basic-compiler-error"></a>'\<functionname >' não está declarado (inteligente dispositivo/Visual Basic erro do compilador)
<`functionname`> não é declarado. Funcionalidade de e/s de arquivo está normalmente disponível no `Microsoft.VisualBasic` namespace, mas a versão de destino do .NET Compact Framework não dar suporte a ele.  
  
 **ID do erro:** BC30766  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Executar operações de arquivo com funções definidas no `System.IO` namespace.

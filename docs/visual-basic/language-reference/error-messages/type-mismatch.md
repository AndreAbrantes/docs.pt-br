# <a name="type-mismatch-visual-basic"></a>Tipos incompatíveis (Visual Basic)
Tentativa de converter um valor em outro tipo de forma que não é válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a atribuição para garantir que ele é válido.  
  
2.  Verifique se que você não passar um objeto para um procedimento que espera um valor ou uma única propriedade.  
  
3.  Verifique se que você não usou um nome de módulo ou projeto em que uma expressão era esperada.

 <span data-ttu-id="596af-107">Se você passar o argumento [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic, às vezes, copia o valor do argumento para uma variável local no procedimento em vez de passar uma referência.</span><span class="sxs-lookup"><span data-stu-id="596af-107">If you pass the argument [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic sometimes copies the argument value into a local variable in the procedure instead of passing a reference.</span></span> <span data-ttu-id="596af-108">Nesse caso, quando o procedimento retorna, Visual Basic deve, em seguida, copie o valor da variável local volta para o argumento no código de chamada.</span><span class="sxs-lookup"><span data-stu-id="596af-108">In such a case, when the procedure returns, Visual Basic must then copy the local variable value back into the argument in the calling code.</span></span>  
  
 <span data-ttu-id="596af-109">Se um `ByRef` o valor do argumento é copiado para o procedimento e o argumento e parâmetro são do mesmo tipo, nenhuma conversão é necessária.</span><span class="sxs-lookup"><span data-stu-id="596af-109">If a `ByRef` argument value is copied into the procedure and the argument and parameter are of the same type, no conversion is necessary.</span></span> <span data-ttu-id="596af-110">Mas se os tipos forem diferentes, o Visual Basic deve converter em ambas as direções.</span><span class="sxs-lookup"><span data-stu-id="596af-110">But if the types are different, Visual Basic must convert in both directions.</span></span> <span data-ttu-id="596af-111">Se um dos tipos é o tipo de classe ou estrutura, o Visual Basic deve convertê-lo para e o outro tipo.</span><span class="sxs-lookup"><span data-stu-id="596af-111">If one of the types is your class or structure type, Visual Basic must convert it both to and from the other type.</span></span> <span data-ttu-id="596af-112">Se um dessas conversões está ampliando, a conversão reversa pode estar restringindo.</span><span class="sxs-lookup"><span data-stu-id="596af-112">If one of these conversions is widening, the reverse conversion might be narrowing.</span></span>

-   <span data-ttu-id="596af-117">Se você precisar retornar um valor para o argumento de chamada, defina o operador de conversão inversa como [Widening](../../../visual-basic/language-reference/modifiers/widening.md), se possível.</span><span class="sxs-lookup"><span data-stu-id="596af-117">If you need to return a value into the calling argument, define the reverse conversion operator as [Widening](../../../visual-basic/language-reference/modifiers/widening.md), if possible.</span></span>

# <a name="reference-required-to-assembly-assemblyname-containing-the-base-class-classname"></a>Referência obrigatória ao assembly '\<assemblyname >' que contém a classe base\<classname >'
Referência obrigatória ao assembly '\<assemblyname >' que contém a classe base\<classname >'. Adicione uma ao seu projeto.  
  
 A classe é definida em uma biblioteca de vínculo dinâmico (DLL) ou assembly que não é referenciado diretamente em seu projeto. O compilador do Visual Basic exige uma referência para evitar ambiguidade no caso da classe é definida em mais de uma DLL ou assembly.  
  
 **ID do erro:** BC30007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Inclua o nome da DLL ou assembly não referenciado em suas referências do projeto.

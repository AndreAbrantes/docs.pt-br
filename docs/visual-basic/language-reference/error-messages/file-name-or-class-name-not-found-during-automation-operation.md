# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>O nome do arquivo ou da classe não foi localizado durante a operação de automação (Visual Basic)
O nome especificado para o nome de arquivo ou a classe em uma chamada para a função `GetObject` não pôde ser encontrado.  
  
## <a name="to-correct-this-error"></a>Para corrigir esse erro  
  
- Verifique os nomes e tente novamente. Verifique se o nome usado para o parâmetro de `class` corresponde ao que foi registrado com o sistema.

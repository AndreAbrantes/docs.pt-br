# <a name="expression-does-not-produce-a-value"></a>A expressão não produz um valor
Você tentou usar uma expressão que produz um valor em um contexto de produção de valor, por exemplo, chamar um `Sub` em um contexto onde um `Function` é esperado.  
  
 **ID do erro:** BC30491  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a expressão para uma que gera um valor.

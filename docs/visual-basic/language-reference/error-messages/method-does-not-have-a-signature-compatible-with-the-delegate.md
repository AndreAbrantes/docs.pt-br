# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>O método não tem uma assinatura compatível com o delegado
Há uma incompatibilidade entre as assinaturas do método e o delegado que você está tentando usar. O `Delegate` instrução define os tipos de parâmetro e tipos de retorno de uma classe de delegado. Qualquer procedimento que possui parâmetros de tipos compatíveis e retornar tipos podem ser usados para criar uma instância desse tipo de delegado.  
  
 **ID do erro**: BC36563

# <a name="declaration-expected"></a>Declaração esperada
Uma instrução nondeclarative, como uma declaração de loop, ou atribuição ocorre fora de qualquer procedimento. Apenas declarações são permitidas fora procedimentos.  
  
 Como alternativa, um elemento de programação está declarado sem uma palavra-chave de declaração, como `Dim` ou `Const`.  
  
 **ID do erro:** BC30188  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a declaração nondeclarative para o corpo de um procedimento.  
  
-   Comece a declaração com uma palavra-chave de declaração adequado.  
  
-   Certifique-se de que não foi digitado incorretamente uma palavra-chave de declaração.

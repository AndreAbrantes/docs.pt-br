# <a name="declaration-expected"></a>Declaração esperada
Ocorre uma instrução nondeclarative, como uma atribuição ou instrução loop, fora de qualquer procedimento. Apenas declarações são permitidas procedimentos externos.  
  
 Como alternativa, um elemento de programação é declarado sem uma palavra-chave de declaração, como `Dim` ou `Const`.  
  
 **ID do erro:** BC30188  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a instrução nondeclarative para o corpo de um procedimento.  
  
-   Comece a declaração com uma palavra-chave de declaração adequada.  
  
-   Certifique-se de que uma palavra-chave de declaração não está incorreto.

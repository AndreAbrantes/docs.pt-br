# <a name="property-not-found"></a>Propriedade não localizada
Este objeto não suporta a propriedade especificada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a ortografia do nome da propriedade.  
  
2.  Verifique a documentação do objeto para certificar-se de que você não está tentando acessar algo parecido com uma propriedade "texto" quando o objeto dá suporte a um "legenda" ou a propriedade nomeada da mesma forma.

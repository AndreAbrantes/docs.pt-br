# <a name="too-many-files"></a>Muitos arquivos
O mais arquivos foram criados no diretório raiz que o sistema operacional permite, ou mais arquivos foram abertos que o número especificado no **arquivos =** configuração na sua configuração. Arquivo SYS.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Se o programa é abrir, fechar ou salvar arquivos no diretório raiz, altere seu programa para que ele use um subdiretório.  
  
2.  Aumentar o número de arquivos especificados em seu **arquivos =** configuração na sua configuração. Sistema de arquivos e reinicie o computador.

# <a name="path-not-found"></a>demarcador não localizado
Durante uma operação de acesso a arquivos ou o acesso ao disco, o sistema operacional não pôde localizar o caminho especificado. O caminho para um arquivo inclui a especificação de unidade mais os diretórios e subdiretórios que devem ser percorridos para localizar o arquivo. Um caminho pode ser relativo ou absoluto.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique e especificar novamente o caminho.

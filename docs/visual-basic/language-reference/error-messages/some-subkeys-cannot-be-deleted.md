# <a name="some-subkeys-cannot-be-deleted"></a>Não é possível excluir algumas subchaves
Foi feita uma tentativa para excluir uma chave do registro, mas a operação falhou porque não não possível excluir algumas subchaves. Normalmente, isso é devido à falta de permissões.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que você tem permissões suficientes para excluir as subchaves especificadas.

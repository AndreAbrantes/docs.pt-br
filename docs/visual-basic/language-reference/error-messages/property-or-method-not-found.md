# <a name="property-or-method-not-found"></a>Propriedade ou método não localizado
O objeto referenciado método ou propriedade de objeto não está definida.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Você pode ter digitado incorretamente o nome do objeto. Para ver quais propriedades e métodos são definidos para um objeto, exiba o Pesquisador de objetos. Selecione a biblioteca de objetos apropriada para exibir uma lista de propriedades e métodos disponíveis.

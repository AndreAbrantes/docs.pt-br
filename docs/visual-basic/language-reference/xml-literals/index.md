|-----------|-----------------|

|[Literal CDATA XML](xml-cdata-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XCData>.|  
|[Literal de Comentário XML](xml-comment-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XComment>.|

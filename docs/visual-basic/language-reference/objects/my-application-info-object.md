# <a name="myapplicationinfo-object"></a>Objeto My.Application.Info
Fornece propriedades para obter informações sobre o aplicativo, como o número de versão, descrição, assemblies carregados e assim por diante.  
  
## <a name="remarks"></a>Comentários  
 Para obter informações sobre os métodos e as propriedades do objeto `My.Application.Info`, consulte <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
> Você pode usar as propriedades da classe <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> para obter informações sobre um arquivo em disco.

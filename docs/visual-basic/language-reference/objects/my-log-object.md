# <a name="mylog-object"></a>Objeto My.Log
Fornece uma propriedade e métodos para gravar informações de evento e de exceção para os ouvintes de log do aplicativo.  
  
## <a name="remarks"></a>Comentários  
 Para obter informações sobre os métodos e propriedades do `My.Log` de objeto, consulte <xref:Microsoft.VisualBasic.Logging.AspLog>.</xref:Microsoft.VisualBasic.Logging.AspLog>  
  
 O `My.Log` objeto está disponível para apenas a aplicativos ASP.NET. Para aplicativos cliente, use [My.Application.Log Object](../../../visual-basic/language-reference/objects/my-application-log-object.md).

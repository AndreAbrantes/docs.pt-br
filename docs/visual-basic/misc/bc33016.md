# <a name="operator-operator-must-have-either-one-or-two-parameters"></a>Operador '\<operador >' deve ter um ou dois parâmetros
Um operador que pode ser unária ou binária é definido sem parâmetros ou com mais de dois parâmetros.  
  
 Um operador unário/binário deve ter um ou dois parâmetros.  
  
 **ID do erro:** BC33016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Ajuste a definição para especificar um ou dois parâmetros.  
  
- Se você precisar sem parâmetros ou maior que dois, você deve usar o [instrução Function](../../visual-basic/language-reference/statements/function-statement.md) para definir um `Function` procedimento em vez de um operador.

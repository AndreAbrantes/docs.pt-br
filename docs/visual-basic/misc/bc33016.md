# <a name="operator-39ltoperatorgt39-must-have-either-one-or-two-parameters"></a>Operador &#39; &lt;operador&gt; &#39; deve ter um ou dois parâmetros
Um operador que pode ser um unário ou binário é definido sem parâmetros ou com mais de dois parâmetros.  
  
 Um operador unário/binário deve ter um ou dois parâmetros.  
  
 **ID do erro:** BC33016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Ajuste a definição para especificar um ou dois parâmetros.  
  
-   Se você precisar sem parâmetros ou com mais de dois, você deve usar o [instrução Function](../../visual-basic/language-reference/statements/function-statement.md) para definir um `Function` procedimento em vez de um operador.

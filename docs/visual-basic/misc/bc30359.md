# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>'Default' pode ser aplicado a apenas um nome de propriedade em um \<tipo >
Você tentou aplicar `Default` para mais de um nome de propriedade dentro de uma classe, interface ou estrutura.  
  
 **ID do erro:** BC30359  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remover tudo, exceto um do `Default` declarações.

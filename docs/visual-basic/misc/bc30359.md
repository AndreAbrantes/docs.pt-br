# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>&#39; padrão &#39; pode ser aplicado a apenas um nome de propriedade em um &lt;tipo&gt;
Você tentou aplicar `Default` para mais de um nome de propriedade em uma classe, interface ou estrutura.  
  
 **ID do erro:** BC30359  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova todos, exceto um do `Default` declarações.

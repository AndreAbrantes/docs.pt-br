# <a name="if-block-must-end-with-a-matching-end-if"></a>Bloco '#If' deve encerrar com '#End If' correspondente
`#If` é uma diretiva de compilação condicional. Uma `#If` bloco não é terminado por um `#End If` diretiva.  
  
 **ID do erro:** BC30012  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `#End If` diretiva até o final do bloco de compilação condicional.

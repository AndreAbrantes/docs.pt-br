# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39; bloco deve terminar com uma correspondência &#39;#End se&#39;
`#If` é uma diretiva de compilação condicional. Um `#If` bloco não é terminado por um `#End If` diretiva.  
  
 **ID do erro:** BC30012  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `#End If` diretiva até o final do bloco de compilação condicional.

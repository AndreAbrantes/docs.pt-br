# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' Try ' não são válidas na janela Immediate
`Try` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30717  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Try` instruções do código de depuração.

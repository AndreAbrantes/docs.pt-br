# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Tente &#39; instruções não são válidas na janela imediata
`Try`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30717  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Try` instruções do código de depuração.

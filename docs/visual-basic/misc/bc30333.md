# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-39lttype3gt39-is-not-a-reference-type"></a>O valor do tipo &#39; &lt;type1&gt; &#39; não pode ser convertido em &#39; &lt;type2&gt; &#39; porque &#39; &lt;type3&gt; &#39; não é um tipo de referência
Uma instrução tenta converter um tipo de matriz para outro tipo de matriz, onde o tipo de dados dos elementos da matriz de destino não é um tipo de referência.  
  
 **ID do erro:** BC30333  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique os tipos de dados dos elementos de matrizes para determinar a origem do erro.

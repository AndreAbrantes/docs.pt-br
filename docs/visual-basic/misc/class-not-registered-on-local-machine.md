# <a name="class-not-registered-on-local-machine"></a>Classe não registrada no computador local
Este erro geralmente ocorre como resultado de um arquivo corrompido. Talvez seja necessário reinstalar o Visual Studio.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reinstale o Visual Studio.

# <a name="get-is-already-declared"></a>'Get' já está declarado
Um `Get` instrução aparece mais de uma vez em um bloco de código. O `Get` instrução declara um `Get` procedimento de propriedade usado para recuperar um valor de uma propriedade.  
  
 **ID do erro:** BC30443  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `Get` instrução.

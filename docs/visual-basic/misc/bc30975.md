# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Para&#39; variável de controle de loop já está em uso por um delimitador &#39;para&#39; loop
Aninhados `For` loops devem usar variáveis de controle exclusivo.  
  
 **ID do erro:** BC30975  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere um dos nomes de variável de controle.

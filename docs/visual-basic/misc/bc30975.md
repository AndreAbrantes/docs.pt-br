# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>'Variável de controle de loop já está em uso por um 'For' loop for'
Aninhado `For` loops devem usar variáveis de controle exclusiva.  
  
 **ID do erro:** BC30975  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere um dos nomes de variável de controle.

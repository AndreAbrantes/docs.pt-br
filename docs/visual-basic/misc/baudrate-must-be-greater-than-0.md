# <a name="baudrate-must-be-greater-than-0"></a>BaudRate deve ser maior que 0
O `BaudRate` argumento especificado para o `My.Computer.Ports.OpenSerialPort` método deve ser maior que zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere o valor da `BaudRate` argumento para um número positivo.

# <a name="data-types-of-the-type-parameters-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="8ef0d-102">Tipos de dados dos parâmetros de tipo no método de extensão &#39; &lt;methodname&gt;&#39; definido no &#39;&lt; TypeName&gt;&#39; não pode ser inferido a partir destes argumentos porque há mais de um tipo possível</span><span class="sxs-lookup"><span data-stu-id="8ef0d-102">Data type(s) of the type parameter(s) in extension method &#39;&lt;methodname&gt;&#39; defined in &#39;&lt;typename&gt;&#39; cannot be inferred from these arguments because more than one type is possible</span></span>
<span data-ttu-id="8ef0d-103">Tipos de dados dos parâmetros de tipo no método de extensão '\<methodname >' definido em '\<typename >' não pode ser inferido a partir destes argumentos porque há mais de um tipo possível.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="8ef0d-104">Especificando os dados tipos explicitamente talvez corrija esse erro.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="8ef0d-105">Tentativa de usar a inferência de tipo para determinar o tipo (ou tipos) do parâmetro de tipo (ou parâmetros) em uma chamada para um método de extensão genérica.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="8ef0d-106">O compilador encontra mais de um tipo de dados para um ou mais dos parâmetros de tipo e relata esse erro.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="8ef0d-107">Quando especificar argumentos não é uma opção (por exemplo, para operadores de consulta em expressões de consulta), a mensagem de erro aparece sem a segunda frase.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="8ef0d-108">O código a seguir demonstra o erro.</span><span class="sxs-lookup"><span data-stu-id="8ef0d-108">The following code demonstrates the error.</span></span>  
  
```vb  
Option Strict Off  
Imports System.Runtime.CompilerServices  
Module Module1  
    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.  
        'caller.targetExtension(1, "2")  
  
    End Sub  
  
    <Extension()> _  
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
    Class Class1  
    End Class  
  
End Module  
```  
  
 <span data-ttu-id="8ef0d-109">**ID do erro:** BC36655 (dentro de [!INCLUDE[vbteclinq](~/includes/vbteclinq-md.md)] consultas) e BC36652 (fora de consultas)</span><span class="sxs-lookup"><span data-stu-id="8ef0d-109">**Error ID:** BC36655 (within [!INCLUDE[vbteclinq](~/includes/vbteclinq-md.md)] queries) and BC36652 (outside queries)</span></span>

# <a name="too-few-type-arguments-to-extension-method-methodname-defined-in-typename"></a>Muito poucos argumentos de tipo para o método de extensão '\<methodname >' definido em '\<typename >'
Um método de extensão genéricos terá sido chamado com menos argumentos de tipo que o número de parâmetros de tipo na definição do método.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC36590  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione argumentos de tipo para sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo do método de extensão que você está invocando.

# <a name="interface-ltinterfacenamegt-can-be-implemented-only-once-by-this-type"></a>Interface '&lt;interfacename&gt;' pode ser implementada somente uma vez por este tipo
A interface especificada já é implementada por esse tipo e não pode ser implementada novamente.  
  
 **ID do erro:** BC31033  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Implemente essa interface usando esse tipo de uma só vez.

# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Interface &#39; &lt;interfacename&gt; &#39; pode ser implementada somente uma vez por este tipo
A interface especificada já é implementada por este tipo e não pode ser implementada novamente.  
  
 **ID do erro:** BC31033  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Implemente essa interface usando esse tipo apenas uma vez.

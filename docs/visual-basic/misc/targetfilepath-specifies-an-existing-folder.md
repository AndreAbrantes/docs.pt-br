# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath Especifica uma pasta existente
O parâmetro `TargetFilePath` Especifica um diretório existente, mas deve especificar o novo nome do diretório.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alteração `TargetFilePath` para especificar um diretório novo, em vez de um existente.

# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'End' não são válidas na janela imediata
O `Stop` e `End` suspender a execução de instruções e não são permitidas em um contexto de depuração.  
  
 **ID do erro:** BC30123  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita uma `End` ou `Stop` instrução na **imediato** janela.

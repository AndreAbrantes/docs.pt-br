# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Final &#39; instruções não são válidas na janela imediata
O `Stop` e `End` instruções suspenda a execução e não são permitidas em um contexto de depuração.  
  
 **ID do erro:** BC30123  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita um `End` ou `Stop` instrução o **imediato** janela.

# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;Compartilhado&#39; não pode ser combinado com &#39; &lt;especificador&gt; &#39; em uma declaração de método
Você tentou combinar o `Shared` modificador com um especificador como `Overridable`, `NotOverridable`, ou `MustOverride`, que é não é válido em uma combinação como uma declaração de método.  
  
 **ID do erro:** BC30501  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

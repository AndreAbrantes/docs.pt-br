# <a name="no-method-ltmethodnamegt-can-accept-an-argument-of-type-lttypenamegt-for-parameter-ltparameternamegt"></a>Nenhum método '&lt;methodname&gt;'pode aceitar um argumento do tipo'&lt;typename&gt;'para o parâmetro'&lt;parametername&gt;'
Uma chamada de método contém um argumento de um tipo que não é válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os tipos dos argumentos.  
  
2.  Corresponder aos tipos de argumento contra os tipos de parâmetro do método para garantir que eles estão de acordo. Altere os tipos conforme necessário.

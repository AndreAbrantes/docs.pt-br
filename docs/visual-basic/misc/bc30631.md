# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;Obter&#39; deve finalizar com uma correspondência &#39;End Get&#39;
`Get` procedimentos de propriedade devem terminar com um `End Get` instrução.  
  
 **ID do erro:** BC30631  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `Get` procedimento de propriedade termina com um `End Get` construir.

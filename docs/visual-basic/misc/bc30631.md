# <a name="get-statement-must-end-with-a-matching-end-get"></a>A instrução 'Get' deve encerrar com um 'End Get' correspondente
`Get` procedimentos de propriedade devem terminar com um `End Get` instrução.  
  
 **ID do erro:** BC30631  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o `Get` procedimento de propriedade termina com um `End Get` construir.

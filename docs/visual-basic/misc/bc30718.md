# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Selecionar &#39; instruções não são válidas na janela imediata
`Select`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30718  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Select` instruções do código de depuração.

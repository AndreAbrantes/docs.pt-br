# <a name="specifier-is-not-valid-on-a-withevents-declaration"></a>'\<especificador >' não é válido em uma declaração WithEvents
Um `Dim` instrução contém uma palavra-chave, como `ReadOnly`, que é inválido em combinação com o `WithEvents` palavra-chave.  
  
 **ID do erro:** BC30234  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Remova a palavra-chave inválida do `Dim` instrução.

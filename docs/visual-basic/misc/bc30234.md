# <a name="39ltspecifiergt39-is-not-valid-on-a-withevents-declaration"></a>&#39;&lt;especificador de&gt; &#39; não é válido em declaração WithEvents
Um `Dim` instrução contém uma palavra-chave, como `ReadOnly`, que é inválido em combinação com o `WithEvents` palavra-chave.  
  
 **ID do erro:** BC30234  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a palavra-chave inválida do `Dim` instrução.

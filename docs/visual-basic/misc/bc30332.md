# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-lttype3gt-is-not-derived-from-lttype4gt"></a>Valor do tipo '&lt;type1&gt;'não pode ser convertido em'&lt;type2&gt;'porque'&lt;type3&gt;'não é derivado de'&lt;type4&gt;'
Uma instrução tenta converter um tipo de matriz em outro tipo de matriz em uma situação em que os tipos de dados dos elementos de matrizes não são tipos de referência, ou onde uma conversão de ampliação ou redução, não é possível entre os tipos de elementos das duas matrizes.  
  
 **ID do erro:** BC30332  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique os tipos de dados dos elementos de ambas as matrizes para determinar a origem do erro.

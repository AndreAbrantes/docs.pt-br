# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39;&lt;TypeName&gt; &#39; não pode ser usado como um atributo porque está declarado como &#39;MustInherit&#39;
Classes de atributos personalizados não podem ser declarados como `MustInherit`.  
  
 **ID do erro:** BC31506  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `MustInherit` modificador de atributos personalizados.

# <a name="exponent-is-not-valid"></a>O expoente é inválido
Você tentou usar um expoente que não é válido com uma literal de ponto flutuante. Esse erro geralmente ocorre quando o tipo de um literal de ponto flutuante é conflitante com o tamanho necessário para manter o literal de ponto flutuante.  
  
 **ID do erro:** BC30495  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Tipo do literal de ponto flutuante e alterá-lo para um tipo válido.

# <a name="exponent-is-not-valid"></a>Expoente não é válido
Você tentou usar um expoente que não é válido com um literal de ponto flutuante. Esse erro normalmente ocorre quando o tipo de um literal de ponto flutuante está em desacordo com o tamanho necessário para manter o literal de ponto flutuante.  
  
 **ID do erro:** BC30495  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verificar tipo de literal de ponto flutuante e altere-o para um tipo válido.

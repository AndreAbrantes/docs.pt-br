# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' continue ' não são válidas na janela Immediate
Instruções ' continue ' só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30780  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a instrução ' continue ' do código de depuração.

# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Continue' não são válidas na janela imediata
`Continue`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30780  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `Continue` instrução do código de depuração.

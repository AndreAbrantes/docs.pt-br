# <a name="object-doesnt-support-this-action-visual-basic"></a>Objeto não dá suporte a essa ação (Visual Basic)
Você referenciou um método ou propriedade que não é suportada por este objeto.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Leia a documentação do objeto para obter mais informações sobre o objeto.  
  
2. Certifique-se de que os nomes de propriedades e métodos estão escritos corretamente.

# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-it-could-be-identical-to-interface-39ltinterfacename2gt39-from-which-the-interface-39ltinterfacename3gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="f198e-102">Não é possível herdar a interface &#39; &lt;interfacename1&gt;&#39; porque ela pode ser idêntica à interface &#39;&lt; interfacename2&gt;&#39; a partir do qual a interface &#39;&lt; interfacename3&gt;&#39; herda para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="f198e-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because it could be identical to interface &#39;&lt;interfacename2&gt;&#39; from which the interface &#39;&lt;interfacename3&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="f198e-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças podem entrar em conflito para certos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="f198e-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="f198e-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="f198e-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="f198e-105">Se `derivedInterface` é construída ou implementada fornecendo o mesmo tipo para ambos `t1` e `t2`, ele deve herdar duas versões do `interfaceA` com argumentos de tipo idênticos.</span><span class="sxs-lookup"><span data-stu-id="f198e-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="f198e-106">Isso geraria ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="f198e-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="f198e-107">**ID do erro:** BC32123</span><span class="sxs-lookup"><span data-stu-id="f198e-107">**Error ID:** BC32123</span></span>

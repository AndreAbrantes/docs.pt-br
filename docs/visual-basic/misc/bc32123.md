# <a name="cannot-inherit-interface-ltinterfacename1gt-because-it-could-be-identical-to-interface-ltinterfacename2gt-from-which-the-interface-ltinterfacename3gt-inherits-for-some-type-arguments"></a><span data-ttu-id="726dd-102">Não é possível herdar a interface '&lt;interfacename1&gt;'porque ela poderia ser idêntica à interface'&lt;interfacename2&gt;' da qual a interface '&lt;interfacename3&gt;' herda para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="726dd-102">Cannot inherit interface '&lt;interfacename1&gt;' because it could be identical to interface '&lt;interfacename2&gt;' from which the interface '&lt;interfacename3&gt;' inherits for some type arguments</span></span>
<span data-ttu-id="726dd-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças poderia entrar em conflito para determinados valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="726dd-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="726dd-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="726dd-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="726dd-105">Se `derivedInterface` é construída ou implementada fornecendo o mesmo tipo para ambos `t1` e `t2`, ela deve herdar duas versões do `interfaceA` com argumentos de tipo idêntico.</span><span class="sxs-lookup"><span data-stu-id="726dd-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="726dd-106">Isso produziria uma ambiguidade sobre a versão que deseja acessar.</span><span class="sxs-lookup"><span data-stu-id="726dd-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="726dd-107">**ID do erro:** BC32123</span><span class="sxs-lookup"><span data-stu-id="726dd-107">**Error ID:** BC32123</span></span>

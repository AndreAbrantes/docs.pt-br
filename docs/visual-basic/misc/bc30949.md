# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="24463-102">Inicializador de matriz não pode ser especificado para uma dimensão não constante; Use o inicializador vazio '{}'</span><span class="sxs-lookup"><span data-stu-id="24463-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="24463-103">Uma matriz inicializa de uma dimensão que não é conhecida em tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="24463-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="24463-104">O código a seguir gera este erro.</span><span class="sxs-lookup"><span data-stu-id="24463-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="24463-105">O código a seguir evita o erro.</span><span class="sxs-lookup"><span data-stu-id="24463-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="24463-106">**ID do erro:** BC30949</span><span class="sxs-lookup"><span data-stu-id="24463-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="24463-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="24463-107">To correct this error</span></span>  
  
- <span data-ttu-id="24463-108">Se possível, especifique uma dimensão constante na declaração de matriz.</span><span class="sxs-lookup"><span data-stu-id="24463-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="24463-109">Se você não pode especificar uma dimensão constante, em seguida, você deve inicializar a matriz usando um loop de quando a dimensão não constante se tornam conhecida.</span><span class="sxs-lookup"><span data-stu-id="24463-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>

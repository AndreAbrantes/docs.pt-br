# <a name="39resume39-or-39goto39-expected"></a>'Resume' ou 'GoTo' esperado
Um `On Error` instrução não contenha um `Resume` cláusula ou um `GoTo` cláusula. Ele deve usar uma cláusula ou outro.  
  
 **ID do erro:** BC32019  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione um `Resume` cláusula ou `GoTo` cláusula para o `On Error` instrução.

# <a name="resume-or-goto-expected"></a>'Resume' ou 'GoTo' esperado
Uma `On Error` instrução não contém uma `Resume` cláusula ou um `GoTo` cláusula. Ele deve usar uma cláusula ou outro.  
  
 **ID do erro:** BC32019  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma `Resume` cláusula ou uma `GoTo` cláusula para o `On Error` instrução.

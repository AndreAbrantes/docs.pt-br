# <a name="array-modifiers-cannot-be-specified-on-lambda-expression-parameters-name-only-on-its-type"></a>Modificadores de matriz não podem ser especificados em nome de parâmetros de expressões lambda, somente em seu tipo
Argumentos de matriz podem ser enviados para expressões lambda, mas a declaração de parâmetro deve incluir o tipo de elemento.  
  
```vb  
' Not valid.  
' Dim arrayExample1 = Function(arrayPara()) 2  
  
' Valid.  
Dim arrayExample2 = Function(arrayPara() As Integer) arrayPara.Length > 0  
Dim arrayexample3 = Function(arrayPara As Integer()) arrayPara.Length > 0  
```  
  
 **ID do erro:** BC36643  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique o tipo dos elementos no parâmetro de matriz.

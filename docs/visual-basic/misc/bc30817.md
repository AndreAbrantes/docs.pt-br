# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39;On GoTo&#39; e &#39;no GoSub&#39; instruções não têm mais suporte
Ele não é mais válido para usar o valor de uma variável ou expressão com o `GoTo` e `GoSub` instruções para controlar o fluxo de programa.  
  
 **ID do erro:** BC30817  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reestruturar seu aplicativo para usar `If...Then...Else` ou `Case` instruções.

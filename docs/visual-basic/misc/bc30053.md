# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="417cc-102">Matrizes não podem ser declaradas com 'New'</span><span class="sxs-lookup"><span data-stu-id="417cc-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="417cc-103">O `New` palavra-chave pode aparecer somente em parte de inicialização de uma declaração de matriz.</span><span class="sxs-lookup"><span data-stu-id="417cc-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="417cc-104">Isso significa `New` deve ser no lado direito do sinal de igual (`=`) para que ele possa criar um novo tipo de matriz a ser atribuído à variável de matriz.</span><span class="sxs-lookup"><span data-stu-id="417cc-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="417cc-105">O atalho para a classe de inicialização não está disponível para matrizes.</span><span class="sxs-lookup"><span data-stu-id="417cc-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="417cc-106">As seguintes duas linhas de código são válidas e são equivalentes porque eles inicializarem um objeto de uma classe.</span><span class="sxs-lookup"><span data-stu-id="417cc-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="417cc-107">No entanto, a inicialização de matriz não pode usar o mesmo atalho da classe de inicialização.</span><span class="sxs-lookup"><span data-stu-id="417cc-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="417cc-108">Observe que o `New` cláusula para uma matriz deve conter ambos os parênteses `()`e chaves, `{}`.</span><span class="sxs-lookup"><span data-stu-id="417cc-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="417cc-109">Os parênteses especificam que o novo tipo é uma matriz, e as chaves fornecem os valores de inicialização.</span><span class="sxs-lookup"><span data-stu-id="417cc-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="417cc-110">O compilador requer as chaves, mesmo se elas estiverem vazias, ou seja, mesmo se você não estiver inicializando qualquer um dos valores de matriz.</span><span class="sxs-lookup"><span data-stu-id="417cc-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="417cc-111">**ID do erro:** BC30053</span><span class="sxs-lookup"><span data-stu-id="417cc-111">**Error ID:** BC30053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="417cc-112">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="417cc-112">To correct this error</span></span>  
  
-   <span data-ttu-id="417cc-113">Substituir uma instrução, como `Dim myDates() As New Date` com uma instrução como `Dim myDates() As Date = New Date() {}`.</span><span class="sxs-lookup"><span data-stu-id="417cc-113">Replace a statement such as `Dim myDates() As New Date` with a statement such as `Dim myDates() As Date = New Date() {}`.</span></span>

# <a name="39as39-comma-or-3939-expected"></a>&#39; Como &#39; vírgula ou &#39;) &#39; esperado
O parâmetro de tipo que foi fornecido está faltando um `As`, vírgula (`,`), ou o parêntese de fechamento ( `)` ).  
  
 **ID do erro:** BC32100  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça a `As`, vírgula ou o parêntese de fechamento.

# <a name="as-comma-or--expected"></a>'As', vírgula ou ')' esperado
O parâmetro de tipo que foi fornecido está faltando uma `As`, vírgula (`,`), ou o parêntese de fechamento ( `)` ).  
  
 **ID do erro:** BC32100  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Forneça a `As`, vírgula ou um parêntese de fechamento.

# <a name="expected-closing-39--gt39-for-xml-comment"></a>Esperado o fechamento &#39; –&gt;&#39; para o comentário XML
Um literal de comentário XML tem os caracteres '-->' de fechamento necessários.  
  
 **ID do erro:** BC31161  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione os caracteres '-->' de fechamento necessários ao final do literal de comentário XML.

# <a name="39get39-statements-are-no-longer-supported"></a>&#39; Obter &#39; Não há suporte para instruções
`Get`Não há suporte para instruções. A funcionalidade de e/s de arquivo está disponível na `Microsoft.VisualBasic` namespace.  
  
 `Get`Não há suporte para operações de arquivo e só pode ser usado na sintaxe do procedimento de propriedade.  
  
 **ID do erro:** BC30829  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Executar operações de arquivo usando os membros `System.IO`, `FileSystemObject`, e [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] funções de tempo de execução.

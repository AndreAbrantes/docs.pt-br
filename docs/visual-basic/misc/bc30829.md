# <a name="get-statements-are-no-longer-supported"></a>Não há suporte para instruções 'Get'
`Get` Não há suporte para instruções. Funcionalidade de e/s de arquivo está disponível no `Microsoft.VisualBasic` namespace.  
  
 `Get` Não há suporte para operações de arquivo e só pode ser usado na sintaxe do procedimento de propriedade.  
  
 **ID do erro:** BC30829  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Executar operações de arquivo usando os membros `System.IO`, `FileSystemObject`e funções de tempo de execução do Visual Basic.

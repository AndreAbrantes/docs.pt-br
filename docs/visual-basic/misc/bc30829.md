# <a name="get-statements-are-no-longer-supported"></a>Não há mais suporte para instruções ' Get '
Não há mais suporte para instruções `Get`. A funcionalidade de e/s de arquivo está disponível no namespace `Microsoft.VisualBasic`.  
  
 Não há suporte para `Get` para operações de arquivo e só pode ser usada na sintaxe de procedimento de propriedade.  
  
 **ID do erro:** BC30829  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Execute operações de arquivo usando os membros de `System.IO`, `FileSystemObject`e Visual Basic funções de tempo de execução.

# <a name="option-optionname-requires-argument"></a>Opção \<optionname > requer \<argumento >
Um argumento obrigatório foi omitido.  
  
 **ID do erro:** BC2006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Fornece o argumento ausente.

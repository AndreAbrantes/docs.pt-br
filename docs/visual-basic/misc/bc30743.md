# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Tipo de &#39; &lt;typename&gt;&#39; não está definido ou o módulo que o contém não está carregado na sessão de depuração
Foi mencionado um tipo que não está disponível ou não está definido.  
  
 **ID do erro:** BC30743  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se que o nome do tipo está escrito corretamente.  
  
2.  Defina o tipo dentro da classe atual ou o módulo.

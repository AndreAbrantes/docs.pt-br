# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>'Sub Main' está declarado mais de uma vez em '\<namespace >': \<mensagem >
`Sub Main` só pode ser declarado uma vez dentro de um namespace.  
  
 **ID do erro:** BC30738  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se há um único `Sub Main` procedimento em seu projeto.

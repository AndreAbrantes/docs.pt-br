# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Marca de comentário XML '&lt;marca&gt;' aparece com atributos idênticos mais de uma vez no mesmo bloco de comentário XML
Marca de comentário XML \<marca > aparece com atributos idênticos mais de uma vez no mesmo bloco de comentário XML. O comentário XML será ignorado.  
  
 O comentário XML contém mais de uma marca com atributos idênticos.  
  
 **ID do erro:** BC42305  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a marca duplicada.

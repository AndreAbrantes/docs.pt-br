# <a name="user-interrupt-occurred"></a>Interrupção feita pelo usuário
Um usuário pressionou CTRL + BREAK ou outra chave de interrupção.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Repita a operação.

# <a name="no-mouse-is-present"></a>Nenhum mouse está presente
Uma das propriedades do `My.Computer.Mouse` objeto foi chamado, mas o computador não tem nenhum mouse ou a porta de mouse instalado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Try...Catch` bloco em torno da chamada à propriedade do `My.Computer.Mouse` objeto.  
  
     – ou —  
  
-   Instale um mouse no computador.

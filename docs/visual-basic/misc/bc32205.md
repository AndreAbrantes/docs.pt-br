# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>Comprimento do nome de variável de WithEvents não pode exceder 1019 caracteres
O nome ou identificador de cada variável declarada usando `WithEvents` está limitado a 1019 caracteres.  
  
 **ID do erro:** BC32205  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reduza o comprimento do identificador.

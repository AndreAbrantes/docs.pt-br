# <a name="addhandler-is-already-declared"></a>'AddHandler' já está declarado
Mais de um `AddHandler` declaração aparece em uma declaração de evento personalizado. Um `AddHandler` declaração declara o procedimento usado para adicionar um manipulador de eventos.  
  
 **ID do erro:** BC31127  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a `AddHandler` instrução.

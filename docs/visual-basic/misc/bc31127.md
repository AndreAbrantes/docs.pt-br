# <a name="39addhandler39-is-already-declared"></a>&#39; AddHandler &#39; já está declarado
Mais de um `AddHandler` declaração é exibida em uma declaração de evento personalizado. Um `AddHandler` declaração declara procedimento usado para adicionar um manipulador de eventos.  
  
 **ID do erro:** BC31127  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `AddHandler` instrução.

# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Número incorreto de argumentos de tipo passado para o método '\<genericprocedurename >'
Um procedimento genérico é chamado com um número de argumentos de tipo que não coincide com o número de parâmetros de tipo com o qual ele está definido.  
  
 **ID do erro:** BC30951  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Fornece um argumento de tipo para cada parâmetro de tipo definido para o procedimento genérico.  
  
     -ou-  
  
-   Chame o procedimento genérico sem argumentos de tipo em todos os e deixar que o compilador tenta inferir os argumentos de tipo.

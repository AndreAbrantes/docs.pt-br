# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Nomes de tipos não intrínsecos não são permitidos em expressões de compilação condicional
A expressão de compilação condicional refere-se a um tipo não intrínseco, mas somente tipos intrínsecos são permitidos.  
  
 Para obter mais informações sobre o tipo intrínseco em Visual Basic, consulte [Resumo de tipos de dados](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **ID do erro:** BC31426  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Substitua o tipo não intrínseco por um tipo intrínseco.

# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Nomes de tipo não intrínsecos não são permitidos em expressões de compilação condicional
A expressão de compilação condicional se refere a um tipo não intrínsecos, mas somente tipos intrínsecos são permitidos.  
  
 Para obter mais informações sobre o tipo intrínseco no [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)], consulte [resumo de tipos de dados](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **ID do erro:** BC31426  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o tipo não intrínsecos com um tipo intrínseco.

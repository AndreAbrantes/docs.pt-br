# <a name="friend-access-was-denied-to-assembly-if-you-want-the-compiler-to-consider-friend-types-in-assembly-add-the-public-key-information-in-the-project-settings-or-through-a-command-line-switch-to-the-compiler-for-the-project-project"></a>O acesso Friend foi negado para \<assembly >. Se você quiser que o compilador considere tipos Friend em \<assembly >, adicione as informações de chave pública nas configurações do projeto ou por meio de uma opção de linha de comando ao compilador para o projeto \<project >.

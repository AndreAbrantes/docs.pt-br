# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Número de aviso &#39; &lt;número&gt; &#39; para a opção &#39; &lt;optionname&gt; &#39; não é configurável ou não é válido
Uma ID de aviso não é válida. IDs devem ser valores de número inteiro maiores que 0.  
  
 **ID do erro:** BC2026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a ID de aviso fornecida para certificar-se de que ele é válido.

# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Número de aviso '&lt;número&gt;'para a opção'&lt;optionname&gt;' não é configurável ou não é válido
Uma ID de aviso não é válida. IDs devem ser valores de número inteiro maiores que 0.  
  
 **ID do erro:** BC2026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a ID do aviso fornecida para certificar-se de que ele é válido.

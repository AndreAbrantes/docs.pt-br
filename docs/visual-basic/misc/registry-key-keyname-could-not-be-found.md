# <a name="registry-key-keyname-could-not-be-found"></a>Chave do registro '\<keyname >' não pôde ser encontrado
Uma chave do registro não foi encontrada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique a ortografia do nome da chave.  
  
2. Certifique-se de que você tenha permissões de segurança suficientes para acessar a chave.

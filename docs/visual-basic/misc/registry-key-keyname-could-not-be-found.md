# <a name="registry-key-39ltkeynamegt39-could-not-be-found"></a>Chave do registro &#39; &lt;keyname&gt; &#39; não pôde ser encontrado
Não foi encontrada uma chave do registro.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a ortografia do nome da chave.  
  
2.  Certifique-se de que você tem permissões de segurança suficientes para acessar a chave.

# <a name="argument-cannot-be-less-than-zero"></a>Argumento não pode ser menor que zero
Um valor que é menor que zero foi fornecido, mas o argumento não pode ser menor que zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique como o valor em questão é calculado, para garantir que ele está correto.

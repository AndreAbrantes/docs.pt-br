# <a name="module-modulename-cannot-be-used-as-a-type"></a>O módulo '\<modulename>' não pode ser usado como um tipo

Você tentou usar um módulo como um tipo. Um módulo não pode ser usado, exceto como um `Get` tipo.  
  
 **ID do erro:** BC30371  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Substitua o módulo por algo que possa ser usado como um tipo válido.

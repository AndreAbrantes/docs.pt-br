# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39; End AddHandler &#39; deve ser a primeira instrução em uma linha
Um `End AddHandler` instrução segue um separador de declaração dois-pontos (:). `End AddHandler`deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC31118  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

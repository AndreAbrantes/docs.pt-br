# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Novo &#39; não pode ser usado na classe &#39; &lt;classname&gt;&#39; porque ele contém um &#39; MustOverride &#39; membro que não tenha sido substituído
Você tentou usar `New` em uma classe que contém um `MustOverride` membro que não foi substituído.  
  
 **ID do erro:** BC30376  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `New` instrução.

# <a name="new-cannot-be-used-on-class-ltclassnamegt-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' não pode ser usado na classe&lt;classname&gt;' porque ele contém um membro 'MustOverride' que não foi substituído
Você tentou usar `New` em uma classe que contém um `MustOverride` membro que não foi substituído.  
  
 **ID do erro:** BC30376  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `New` instrução.

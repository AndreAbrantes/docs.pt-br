# <a name="bracketed-identifier-is-missing-closing-3939"></a>Ausente de identificador entre colchetes fechamento &#39;] &#39;
Colchetes em um nome com caracteres de escape devem ocorrer em pares correspondentes.  
  
 **ID do erro:** BC30034  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Coloque um colchete de fechamento no final do nome de escape.

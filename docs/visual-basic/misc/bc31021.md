# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilador do Visual Basic é não é possível recuperar o seguinte erro: &lt;erro&gt;
O compilador do Visual Basic é não é possível recuperar o seguinte erro: <`error`>.  
  
 Alguns recursos do IDE foram desabilitados.  
  
 Erro interno no compilador do Visual Basic.  
  
 **ID do erro:** BC31021  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Salve seu trabalho e reinicie o Visual Studio.  
  
2.  Se o erro persistir, reinstale o Visual Studio.  
  
3.  Se o erro persistir após a reinstalação, notifique o Microsoft Product Support Services.

# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilador do Visual Basic é capaz de recuperar o seguinte erro: &lt;erro&gt;
O [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador é capaz de recuperar o seguinte erro: `error`>.  
  
 Alguns recursos IDE foram desabilitados.  
  
 Ocorreu um erro interno no [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador.  
  
 **ID do erro:** BC31021  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Salve seu trabalho e reinicie o [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
2.  Se o erro persistir, reinstale o [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Se o erro persistir após a reinstalação, notifique o Microsoft Product Support Services.

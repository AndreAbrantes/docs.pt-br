# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;nome&gt; &#39; é ambíguo entre declarações nos módulos &#39; &lt;modulename1&gt; &#39; e &#39; &lt;modulename2&gt;&#39;
Você forneceu um nome que é ambíguo e, portanto, está em conflito com outro nome. O compilador do Visual Basic não tem nenhuma regra de resolução de conflitos; Você deve resolver a ambiguidade de nomes por conta própria.  
  
 **ID do erro:** BC30562  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Qualifica totalmente o nome.

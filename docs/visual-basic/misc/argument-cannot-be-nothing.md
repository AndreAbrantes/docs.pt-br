# <a name="argument-cannot-be-nothing"></a>Argumento não pode ser Nothing
Um valor nulo foi fornecido para um argumento que deve ter um valor.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Você pode ter tentou usar um objeto sem fornecer uma instância do objeto. Nesse caso, use o `New` palavra-chave para criar a instância.  
  
-   Verifique se o valor é calculado corretamente.

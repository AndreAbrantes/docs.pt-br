# <a name="variable-ltvariablenamegt-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Variável '&lt;variablename&gt;' é usada antes que ele foi atribuído um valor (erro do Visual Basic)
Uma variável que está sendo usada antes que ele foi atribuído um valor, resultando em um <xref:System.NullReferenceException>.  
  
 **ID do erro:** BC42109  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que a variável foi atribuída um valor.

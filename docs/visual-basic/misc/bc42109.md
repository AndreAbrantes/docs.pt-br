# <a name="variable-39ltvariablenamegt39-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Variável &#39; &lt;variablename&gt;&#39; é usada antes de receber um valor (erro do Visual Basic)
Uma variável está sendo usada antes de receber um valor, resultando em um <xref:System.NullReferenceException>.  
  
 **ID do erro:** BC42109  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que a variável foi atribuída um valor.

# <a name="static-local-variable-39ltvariablenamegt39-is-already-declared"></a>Variável local estática &#39; &lt;variablename&gt; &#39; já está declarado
Outra variável local estática com o mesmo nome já foi declarado.  
  
 **ID do erro:** BC31401  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova redundantes declarações locais estáticas.  
  
2.  Dê um nome exclusivo de cada variável local estática.

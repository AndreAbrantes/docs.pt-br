# <a name="static-local-variable-variablename-is-already-declared"></a>Variável local estática '\<variablename >' já está declarado
Outra variável local estática com o mesmo nome já foi declarado.  
  
 **ID do erro:** BC31401  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova as declarações redundantes de local estáticas.  
  
2.  Dê um nome exclusivo de cada variável local estática.

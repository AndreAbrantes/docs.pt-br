# <a name="instance-of-restricted-type-lttypenamegt-cannot-be-used-in-a-query-expression"></a>Instância de tipo restrito '&lt;typename&gt;' não pode ser usado em uma expressão de consulta
Instâncias de tipos, restritos, como <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, e <xref:System.TypedReference>, não pode ser usado em uma expressão lambda ou expressão de consulta.  
  
 **ID do erro:** BC36598

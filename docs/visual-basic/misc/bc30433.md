# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Métodos em um módulo não podem ser declarados &#39; &lt;especificador&gt;&#39;
Você usou um especificador que não é válido em um método dentro de um `Module` instrução. Módulos nunca podem ser instanciados, não oferecem suporte a herança e não podem implementar interfaces.  
  
 **ID do erro:** BC30433  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

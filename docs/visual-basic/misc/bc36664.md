# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Aninhamento de parênteses na lista de parâmetros da expressão lambda excede a profundidade máxima permitida
Parênteses aninhados na lista de parâmetros de uma expressão lambda não podem exceder uma profundidade de 128.  
  
 **ID do erro:** BC36664

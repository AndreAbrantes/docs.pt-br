# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>Aninhamento de parênteses na expressão lambda&#39;lista de parâmetros de s excede a profundidade máxima permitida
Parênteses aninhados na lista de parâmetros de uma expressão lambda não podem exceder uma profundidade de 128.  
  
 **ID do erro:** BC36664

# <a name="set-not-supported-read-only-property"></a>Conjunto não tem suportado (propriedade somente leitura)
Você tentou definir ou alterar uma propriedade que é somente leitura.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a referência à propriedade do seu código.  
  
2.  Altere a referência para retornar somente o valor da propriedade em tempo de execução.

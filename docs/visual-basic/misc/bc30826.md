# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>Não há suporte para instruções 'EndIf'; Use 'End If' em vez disso
O `EndIf` instrução não é mais suportada.  
  
 **ID do erro:** BC30826  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use `End If` encerrar `If` instruções.

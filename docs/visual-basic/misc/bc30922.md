# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="ab870-102">' \<derivedtypename> ' não pode herdar de \<type> ' \<constructedbasetypename> ' porque ele expande o acesso do tipo ' \<internaltypename> ' fora do assembly</span><span class="sxs-lookup"><span data-stu-id="ab870-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="ab870-103">Uma classe ou interface derivada tenta expandir o nível de acesso de um tipo restrito usando-o como um argumento de tipo para uma classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="ab870-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="ab870-104">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="ab870-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="ab870-105">O código fora do assembly não tem permissão para acessar `restrictedStructure` .</span><span class="sxs-lookup"><span data-stu-id="ab870-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="ab870-106">No entanto, o `derivedClass` pode ser acessado de qualquer código que possa fazer referência a ele.</span><span class="sxs-lookup"><span data-stu-id="ab870-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="ab870-107">Portanto, `derivedClass` não é possível herdar `baseClass` se ele usa `restrictedStructure` como um argumento de tipo, porque isso pode ser exposto `restrictedStructure` ao código em qualquer assembly.</span><span class="sxs-lookup"><span data-stu-id="ab870-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="ab870-108">**ID do erro:** BC30922</span><span class="sxs-lookup"><span data-stu-id="ab870-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ab870-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="ab870-109">To correct this error</span></span>  
  
- <span data-ttu-id="ab870-110">Ajuste os níveis de acesso das classes ou interfaces para que o tipo derivado não expanda o nível de acesso do tipo restrito.</span><span class="sxs-lookup"><span data-stu-id="ab870-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="ab870-111">-ou-</span><span class="sxs-lookup"><span data-stu-id="ab870-111">-or-</span></span>  
  
- <span data-ttu-id="ab870-112">Se você não puder ajustar os níveis de acesso, não use o tipo restrito como um argumento de tipo ao construir a classe base ou a interface.</span><span class="sxs-lookup"><span data-stu-id="ab870-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>

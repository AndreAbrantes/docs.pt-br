# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-outside-the-assembly"></a><span data-ttu-id="710db-102">&#39; &lt;derivedtypename&gt;&#39; não pode herdar de &lt;tipo&gt; &#39;&lt; constructedbasetypename&gt;&#39; porque ele expande o acesso do tipo &#39;&lt; internaltypename&gt;&#39; fora do assembly</span><span class="sxs-lookup"><span data-stu-id="710db-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; outside the assembly</span></span>
<span data-ttu-id="710db-103">Uma classe derivada ou interface tenta expandir o nível de acesso de um tipo restrito usando-o como um argumento de tipo a uma classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="710db-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="710db-104">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="710db-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="710db-105">Código fora do assembly não tem permissão para acessar `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="710db-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="710db-106">No entanto, `derivedClass` pode ser acessado de qualquer código que pode fazer referência a ele.</span><span class="sxs-lookup"><span data-stu-id="710db-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="710db-107">Portanto, `derivedClass` não é possível herdar `baseClass` se ele usa `restrictedStructure` como um argumento de tipo, porque que poderia expor `restrictedStructure` a código em qualquer assembly.</span><span class="sxs-lookup"><span data-stu-id="710db-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="710db-108">**ID do erro:** BC30922</span><span class="sxs-lookup"><span data-stu-id="710db-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="710db-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="710db-109">To correct this error</span></span>  
  
-   <span data-ttu-id="710db-110">Ajuste os níveis de acesso das interfaces ou classes para que o tipo derivado não expande o nível de acesso do tipo restrito.</span><span class="sxs-lookup"><span data-stu-id="710db-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="710db-111">-ou-</span><span class="sxs-lookup"><span data-stu-id="710db-111">-or-</span></span>  
  
-   <span data-ttu-id="710db-112">Se você não pode ajustar os níveis de acesso, não use o tipo restrito como um argumento de tipo ao construir a classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="710db-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>

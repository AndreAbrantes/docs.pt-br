# <a name="is-operand-of-type-typename-can-only-be-compared-to-nothing-because-typename-is-a-nullable-type"></a><span data-ttu-id="7d791-102">O operando 'Is' do tipo 'typename' só pode ser comparado a 'Nothing', porque 'typename' é um tipo que permite valor nulo</span><span class="sxs-lookup"><span data-stu-id="7d791-102">'Is' operand of type 'typename' can only be compared to 'Nothing', because 'typename' is a nullable type</span></span>
<span data-ttu-id="7d791-103">Uma variável declarada como anulável foi comparada com uma expressão diferente de `Nothing` usando o `Is` operador.</span><span class="sxs-lookup"><span data-stu-id="7d791-103">A variable declared as nullable has been compared to an expression other than `Nothing` by using the `Is` operator.</span></span>  
  
 <span data-ttu-id="7d791-104">**ID do erro:** BC32127</span><span class="sxs-lookup"><span data-stu-id="7d791-104">**Error ID:** BC32127</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7d791-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="7d791-105">To correct this error</span></span>
  
1. <span data-ttu-id="7d791-106">Para comparar um tipo de valor anulável com uma expressão que não seja `Nothing` usando o `Is` operador, chame o `GetType` método no tipo de valor anulável e compare o resultado com a expressão, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="7d791-106">To compare a nullable value type to an expression other than `Nothing` by using the `Is` operator, call the `GetType` method on the nullable value type and compare the result to the expression, as shown in the following example.</span></span>  
  
```vb  
Dim number? As Integer = 5  

If number IsNot Nothing Then  
  If number.GetType() Is Type.GetType("System.Int32") Then

  End If  
End If  
```

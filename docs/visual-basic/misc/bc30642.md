# <a name="optional-and-paramarray-cannot-be-combined"></a>'Optional' e 'ParamArray' não podem ser combinados
O `Optional` e `ParamArray` modificadores são mutuamente exclusivos.  
  
 **ID do erro:** BC30642  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especificar `ParamArray` ou `Optional`, mas não ambos.

# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' Case ' não são válidas na janela Immediate
`Case` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30719  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Case` instruções da janela **imediata** .

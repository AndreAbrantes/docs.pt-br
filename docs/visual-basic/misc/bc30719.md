# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Caso&#39; instruções não são válidas na janela imediata
`Case` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30719  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Case` instruções de **imediato** janela.

# <a name="39removehandler39-is-already-declared"></a>&#39;RemoveHandler&#39; já está declarado
Mais de um `RemoveHandler` declaração apareceu na declaração de evento personalizado. Um `RemoveHandler` declaração declara procedimento usado para remover um manipulador de eventos.  
  
 **ID do erro:** BC31128  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `RemoveHandler` instrução.

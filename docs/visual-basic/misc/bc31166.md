# <a name="expected-closing--for-xml-tag"></a>Fechamento esperado ' >' para uma marca XML
A marca XML para um literal XML está faltando o fechamento necessários ' >' caracteres.  
  
 **ID do erro:** BC31166  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento ' >' caractere ao final da marca XML para o literal XML.

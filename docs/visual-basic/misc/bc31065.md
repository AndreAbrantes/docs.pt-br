# <a name="set-parameter-cannot-be-declared-ltspecifiergt"></a>O parâmetro 'Set' não pode ser declarado '&lt;especificador&gt;'
Um modificador inválido foi especificado com um parâmetro em uma `Set` procedimento de propriedade.  
  
 **ID do erro:** BC31065  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova os modificadores inválidos, como `ByRef`, `ParamArray`, e `Optional`.

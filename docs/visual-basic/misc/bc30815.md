# <a name="functionname-is-not-declared-visual-basic-error"></a>'\<functionname >' não está declarado (erro do Visual Basic)
'\<functionname >' não está declarado. Funcionalidade de e/s de arquivo está disponível no namespace Microsoft. VisualBasic.  
  
 **ID do erro:** BC30815  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Use as funções de e/s de arquivo definidas `My.Computer.FileSystem`, o `System.IO` namespace, funções de tempo de execução do Visual Basic, ou o `Microsoft.VisualBasic.FileSystem` objeto.

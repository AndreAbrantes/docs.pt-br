# <a name="object-doesnt-support-this-property-or-method-visual-basic"></a>Objeto não dá suporte a esta propriedade ou método (Visual Basic)
Você especificou um método ou propriedade não existe para este objeto de automação.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Leia a documentação do objeto para obter mais informações sobre o objeto e verifique a ortografia de propriedades e métodos.

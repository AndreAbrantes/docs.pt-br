# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>Caractere de tipo '\<charactername >' não corresponde ao tipo de dados declarado '\<tipo >'
Uma variável é declarada com um tipo de dados mas chamada com um caractere de tipo que representa um tipo de dados incompatíveis, por exemplo `K$ = 10` quando `K` é declarado `Integer`.  
  
 **ID do erro:** BC30277  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Alterar o tipo de dados declarado da variável, ou altere ou remova o caractere de tipo na referência.

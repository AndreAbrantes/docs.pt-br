<span data-ttu-id="a01ac-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças podem entrar em conflito para certos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="a01ac-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="a01ac-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="a01ac-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="a01ac-105">Se `derivedInterface` é construída ou implementada fornecendo o mesmo tipo para ambos `t1` e `t2`, ele deve herdar duas versões do `interfaceX` com argumentos de tipo idênticos.</span><span class="sxs-lookup"><span data-stu-id="a01ac-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="a01ac-106">Isso geraria ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="a01ac-106">Doing so would produce an ambiguity about which version to access.</span></span>

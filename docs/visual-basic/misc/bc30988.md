<span data-ttu-id="f448e-103">Quando você declara uma instância de uma classe, o `New` palavra-chave deve ser seguido por um nome de tipo ou por `With`.</span><span class="sxs-lookup"><span data-stu-id="f448e-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="f448e-104">Por exemplo, as instruções seguintes declaram `client` deve ser uma instância do `Customer` classe.</span><span class="sxs-lookup"><span data-stu-id="f448e-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="f448e-105">O nome do tipo `Customer` segue `New`.</span><span class="sxs-lookup"><span data-stu-id="f448e-105">The type name `Customer` follows `New`.</span></span>  
  
```

Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="f448e-106">Começando com [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], você pode declarar um objeto deve ser uma instância de um tipo anônimo, nesse caso, você não especificar um tipo de dados.</span><span class="sxs-lookup"><span data-stu-id="f448e-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="f448e-107">Em declarações de tipo anônimo, a palavra-chave `With` segue `New`.</span><span class="sxs-lookup"><span data-stu-id="f448e-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>

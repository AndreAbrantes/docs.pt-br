<span data-ttu-id="bc1dd-103">Quando você declara uma instância de uma classe, a `New` palavra-chave deve ser seguida por um nome de `With`tipo ou por.</span><span class="sxs-lookup"><span data-stu-id="bc1dd-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="bc1dd-104">Por exemplo, as instruções a seguir declaram `client` ser uma instância `Customer` da classe.</span><span class="sxs-lookup"><span data-stu-id="bc1dd-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="bc1dd-105">O nome `Customer` do tipo `New`é o seguinte.</span><span class="sxs-lookup"><span data-stu-id="bc1dd-105">The type name `Customer` follows `New`.</span></span>


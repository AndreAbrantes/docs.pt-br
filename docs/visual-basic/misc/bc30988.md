# <a name="type-or-with-expected"></a>Tipo ou ' with ' esperado
Quando você declara uma instância de uma classe, a `New` palavra-chave deve ser seguida por um nome de `With`tipo ou por. Por exemplo, as instruções a seguir declaram `client` ser uma instância `Customer` da classe. O nome `Customer` do tipo `New`é o seguinte.  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 A partir do Visual Basic 2008, você pode declarar um objeto para ser uma instância de um tipo anônimo; nesse caso, você não especifica um tipo de dados. Em declarações de tipo anônimo, a `With` palavra `New`-chave é a seguinte.  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 **ID do erro:** BC30988  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere a declaração para que `With` o ou um nome de `New`tipo seja mostrado a seguir.

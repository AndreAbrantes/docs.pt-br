<span data-ttu-id="c32bf-103">Quando você declara uma instância de uma classe, o `New` palavra-chave deve ser seguido por um nome de tipo ou por `With`.</span><span class="sxs-lookup"><span data-stu-id="c32bf-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="c32bf-104">Por exemplo, as instruções seguintes declaram `client` deve ser uma instância do `Customer` classe.</span><span class="sxs-lookup"><span data-stu-id="c32bf-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="c32bf-105">O nome do tipo `Customer` segue `New`.</span><span class="sxs-lookup"><span data-stu-id="c32bf-105">The type name `Customer` follows `New`.</span></span>

```

```

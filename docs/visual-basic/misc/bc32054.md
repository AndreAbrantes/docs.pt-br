# <a name="39ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>&#39; &lt;membername&gt; tem o mesmo nome como um parâmetro de tipo
Um membro de uma classe genérica ou estrutura tem o mesmo nome que um parâmetro de tipo de classe ou estrutura.  
  
 Cada variável, constante, enumeração, procedimento, propriedade, evento e delegado devem ter um nome distinto de cada parâmetro de tipo.  
  
 **ID do erro:** BC32054  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro de classe ou estrutura para que ele é diferente de cada nome de parâmetro de tipo.

# <a name="get-not-supported-at-run-time"></a>Obter não tem suporte em tempo de execução
Você tentou ler uma propriedade em tempo de execução que só está acessível no tempo de design.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique a propriedade e determine em quais condições pode ser definido.  
  
2. Remova a referência à propriedade.

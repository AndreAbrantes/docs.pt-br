# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>'Option Explicit' pode ser seguido apenas por 'On' ou 'Off'
`Option Explicit` instruções devem especificar `On` ou `Off`. Todos os outros valores são inválidos.  
  
 **ID do erro:** BC30640  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique `Option Explicit On` ou `Option Explicit Off`.

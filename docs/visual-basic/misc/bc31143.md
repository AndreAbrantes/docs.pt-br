 <span data-ttu-id="f3302-105">O código a seguir ilustra as conversões com falha.</span><span class="sxs-lookup"><span data-stu-id="f3302-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="f3302-106">O representante é `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="f3302-106">The delegate is `FunDel`.</span></span>

 <span data-ttu-id="f3302-107">As funções a seguir diferem `FunDel` de forma que fará com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="f3302-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

```  
  
 <span data-ttu-id="f3302-108">Cada uma das seguintes instruções de atribuição faz com que o erro.</span><span class="sxs-lookup"><span data-stu-id="f3302-108">Each of the following assignment statements causes the error.</span></span>

    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2

```

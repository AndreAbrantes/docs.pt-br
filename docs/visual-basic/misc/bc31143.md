# <a name="method-methodname-does-not-have-a-signature-compatible-with-delegate-delegatename"></a><span data-ttu-id="57208-102">O método ' \<methodname> ' não tem uma assinatura compatível com delegado \<'delegatename'></span><span class="sxs-lookup"><span data-stu-id="57208-102">Method '\<methodname>' does not have a signature compatible with delegate \<'delegatename'></span></span>

<span data-ttu-id="57208-103">Esse erro ocorre quando uma conversão é necessária entre um método e um delegado que não é possível.</span><span class="sxs-lookup"><span data-stu-id="57208-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="57208-104">A causa do erro pode ser a conversão entre parâmetros ou, quando o método e o delegado são funções, conversão nos valores de retorno.</span><span class="sxs-lookup"><span data-stu-id="57208-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="57208-105">O código a seguir ilustra as conversões com falha.</span><span class="sxs-lookup"><span data-stu-id="57208-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="57208-106">O delegado é `FunDel` .</span><span class="sxs-lookup"><span data-stu-id="57208-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="57208-107">As funções a seguir diferem de `FunDel` uma maneira que causará esse erro.</span><span class="sxs-lookup"><span data-stu-id="57208-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>  
  
```vb  
Function ExampleMethod1(ByVal m As Integer, ByVal aDate As Date) As Integer  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="57208-108">Cada uma das instruções de atribuição a seguir causa o erro.</span><span class="sxs-lookup"><span data-stu-id="57208-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb  
Sub Main()  
    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2  
  
End Sub  
```  
  
 <span data-ttu-id="57208-109">**ID do erro:** BC31143</span><span class="sxs-lookup"><span data-stu-id="57208-109">**Error ID:** BC31143</span></span>

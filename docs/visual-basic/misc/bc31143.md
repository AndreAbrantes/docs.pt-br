<span data-ttu-id="7edc9-103">Esse erro ocorre quando uma conversão é exigida entre um método e um delegado que não é possível.</span><span class="sxs-lookup"><span data-stu-id="7edc9-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="7edc9-104">A causa do erro pode ser conversão entre parâmetros ou, quando o método e delegado são funções, conversão de valores de retorno.</span><span class="sxs-lookup"><span data-stu-id="7edc9-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="7edc9-105">O código a seguir ilustra conversões que falharam.</span><span class="sxs-lookup"><span data-stu-id="7edc9-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="7edc9-106">O representante é `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="7edc9-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="7edc9-107">As funções a seguir diferem das `FunDel` de forma que irá causar esse erro.</span><span class="sxs-lookup"><span data-stu-id="7edc9-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>  
  
```vb  
Function ExampleMethod1(ByVal m As Integer, ByVal aDate As Date) As Integer  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="7edc9-108">Cada uma das seguintes declarações de designação causa o erro.</span><span class="sxs-lookup"><span data-stu-id="7edc9-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb

    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2  
  
End Sub  
```

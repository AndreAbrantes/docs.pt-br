# <a name="comma-or-3939-expected-procedure-parameter-list"></a>Vírgula ou &#39;)&#39; esperado (lista de parâmetros de procedimento)
Uma lista de parâmetros contém um elemento inesperado. Use vírgulas (`,`) para separar parâmetros na lista e use um parêntese de fechamento (`)`) para finalizar a lista.  
  
 **ID do erro:** BC30213  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma vírgula e continuar a lista de parâmetros de tipo ou adicionar um fechamento parêntese de abertura para finalizar a lista.

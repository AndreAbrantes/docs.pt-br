# <a name="unable-to-open-module-file-ltfilenamegt-lterrorgt"></a>Não é possível abrir o arquivo de módulo '&lt;filename&gt;': &lt;erro&gt;
Um arquivo que contém um módulo para este projeto não pôde ser aberto.  
  
 **ID do erro:** BC31007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verificar se o arquivo está no local especificado e não está bloqueado por outro aplicativo.

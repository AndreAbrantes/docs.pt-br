# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Evento personalizado&#39; deve terminar com uma correspondência &#39;fim do evento&#39;
Um `Custom Event` declaração deve terminar com um `End Event` instrução.  
  
 **ID do erro:** BC31114  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `Custom Event` declaração termina com um `End Event` instrução.

# <a name="custom-event-must-end-with-a-matching-end-event"></a>'Custom Event' deve encerrar com um 'End Event' correspondente
Um `Custom Event` declaração deve terminar com um `End Event` instrução.  
  
 **ID do erro:** BC31114  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `Custom Event` declaração termina com um `End Event` instrução.

## <a name="to-correct-this-error"></a><span data-ttu-id="6c1c4-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="6c1c4-111">To correct this error</span></span>  
  
-   <span data-ttu-id="6c1c4-112">Se a matriz de parâmetros não especificar o tipo de dados que você deseja estender, adicione um novo primeiro parâmetro que especifica este tipo.</span><span class="sxs-lookup"><span data-stu-id="6c1c4-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```

    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="6c1c4-113">Se a matriz de parâmetros especificar o tipo de dados que você deseja estender, considere alterá-la para uma matriz regular, que requer um argumento, em vez de uma matriz de parâmetros.</span><span class="sxs-lookup"><span data-stu-id="6c1c4-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="6c1c4-114">Matrizes regulares podem ser estendidas.</span><span class="sxs-lookup"><span data-stu-id="6c1c4-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>

        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```

# <a name="extension-method-methodname-defined-in-modulename-does-not-have-a-signature-compatible-with-delegate-delegatename"></a>Método de extensão '\<methodname >' definido em '\<modulename >' não tem uma assinatura compatível com o delegado '\<delegatename >'
Há uma incompatibilidade entre as assinaturas do método de extensão e o delegado que você está tentando usar. O `Delegate` instrução define os tipos de parâmetro e tipos de retorno de uma classe de delegado. Qualquer procedimento com parâmetros compatíveis, tipos de parâmetro e tipo de retorno pode ser usado para criar uma instância desse tipo de delegado.  
  
 **ID do erro:** BC36710

# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>Método de extensão &#39; &lt;methodname&gt;&#39; definido no &#39;&lt; ModuleName&gt;&#39; não tem uma assinatura compatível com delegado &#39;&lt; delegateName&gt;&#39;
Há uma incompatibilidade entre as assinaturas do método de extensão e o representante que você está tentando usar. O `Delegate` instrução define os tipos de parâmetro e tipos de retorno de uma classe delegate. Qualquer procedimento com parâmetros compatíveis, tipos de parâmetro e tipo de retorno pode ser usado para criar uma instância do tipo delegado.  
  
 **ID do erro:** BC36710

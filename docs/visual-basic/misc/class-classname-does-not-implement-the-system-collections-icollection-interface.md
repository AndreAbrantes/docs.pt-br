# <a name="class-39ltclassnamegt39-does-not-implement-the-systemcollectionsicollection-interface"></a>Classe &#39; &lt;classname&gt; &#39; não implementa a interface Collections
A classe não implementa o `System.Collections.ICollection` interface, que define tamanho, enumeradores e métodos de sincronização para todas as coleções. Ele pode ser necessário reinstalar o Visual Studio.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Reinstale o Visual Studio.

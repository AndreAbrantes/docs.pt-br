# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Tipo de argumento '\<argumentname >' é '\<typename >', que não é numérico
Um argumento não numérico ocorre em uma situação em que um argumento numérico é esperado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Certifique-se de que o argumento é numérico.

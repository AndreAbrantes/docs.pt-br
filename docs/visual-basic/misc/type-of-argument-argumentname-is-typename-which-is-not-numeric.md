# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Tipo de argumento '&lt;argumentname&gt;'é'&lt;typename&gt;', que não é numérico
Um argumento não numérico ocorre em uma situação em que um argumento numérico é esperado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o argumento é numérico.

# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicializadores de tipo anônimo não são válidos em janelas de depuração
Foi inserido em um inicializador de tipo anônimo a **Watch** ou **imediato** janela durante uma sessão de depuração. Inicializadores de tipo anônimo não estão disponíveis em janelas de depuração.  
  
 **ID do erro:** BC36577  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Teste os inicializadores de tipo anônimo no Editor de códigos.

# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicializadores de tipo anônimo não são válidos nas janelas de depuração
Um inicializador de tipo anônimo foi inserido no **inspecionar** ou **imediato** durante uma sessão de depuração. Inicializadores de tipo anônimo não estão disponíveis no windows de depuração.  
  
 **ID do erro:** BC36577  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Inicializadores de tipo anônimo de teste no Editor de códigos.

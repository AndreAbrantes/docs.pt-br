# <a name="try-must-have-at-least-one-catch-or-a-finally"></a>Try deve ter pelo menos um 'Catch' ou 'Finally'
Um `Try` bloco deve incluir uma `Finally` bloco ou pelo menos um `Catch` bloquear antes do `End Try` instrução.  
  
 **ID do erro:** BC30030  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Catch` ou `Finally` bloquear dentro a `Try` bloco.

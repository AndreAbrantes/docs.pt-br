# <a name="try-must-have-at-least-one-39catch39-or-a-39finally39"></a>Try deve ter pelo menos um &#39;Catch&#39; ou um &#39;Finally&#39;
Um `Try` bloco deve incluir um `Finally` bloco ou pelo menos um `Catch` bloquear antes do `End Try` instrução.  
  
 **ID do erro:** BC30030  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Catch` ou `Finally` bloquear dentro de `Try` bloco.

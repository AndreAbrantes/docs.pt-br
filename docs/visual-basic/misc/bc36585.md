# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<elementname >' não é um parâmetro de método de extensão '\<methodname >' definido em '\<typename >'
Uma chamada de procedimento fornece um argumento adicional que não está definido pelo método de extensão.  
  
 **ID do erro:** BC36585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o argumento supérfluo da lista de argumentos.

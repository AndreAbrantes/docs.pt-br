# <a name="ltelementnamegt-is-not-a-parameter-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>'&lt;elementname&gt;'não é um parâmetro de método de extensão'&lt;methodname&gt;'definido em'&lt;typename&gt;'
Uma chamada de procedimento fornece um argumento adicional que não está definido pelo método de extensão.  
  
 **ID do erro:** BC36585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento supérfluo da lista de argumentos.

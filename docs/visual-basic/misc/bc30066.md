# <a name="39exit-property39-is-not-valid-in-a-function-or-sub"></a>&#39; Propriedade de saída &#39; não é válido em Function ou Sub
`Exit Property`é exibida em uma `Function` procedimento ou uma `Sub` procedimento. Um `Exit` instrução deve coincidir com o bloco no qual ele ocorre.  
  
 **ID do erro:** BC30066  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o `Exit Property` com o `Exit Function` ou `Exit Sub` instrução conforme apropriado.

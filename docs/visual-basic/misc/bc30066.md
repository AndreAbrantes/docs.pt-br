# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>'Exit Property' não é válido em Function ou Sub
`Exit Property` é exibida em uma `Function` procedimento ou uma `Sub` procedimento. Um `Exit` instrução deve coincidir com o bloco no qual ele ocorre.  
  
 **ID do erro:** BC30066  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua os `Exit Property` com o `Exit Function` ou `Exit Sub` instrução conforme apropriado.

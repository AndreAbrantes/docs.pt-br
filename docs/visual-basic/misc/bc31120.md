# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>'End RaiseEvent' deve ser a primeira instrução em uma linha
Um `End RaiseEvent` instrução segue um separador de instrução de dois-pontos (:). `End RaiseEvent` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC31120  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39; End RaiseEvent &#39; deve ser a primeira instrução em uma linha
Um `End RaiseEvent` instrução segue um separador de declaração dois-pontos (:). `End RaiseEvent`deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC31120  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

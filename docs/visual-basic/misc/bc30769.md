# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>A versão de destino do .NET Compact Framework não dá suporte a &#39; Final &#39; instrução
A versão do .NET Compact Framework você está trabalhando com não oferece suporte a `End` instrução.  
  
 **ID do erro:** BC30769  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a instrução.

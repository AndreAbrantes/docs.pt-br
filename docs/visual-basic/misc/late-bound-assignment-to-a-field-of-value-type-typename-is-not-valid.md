# <a name="late-bound-assignment-to-a-field-of-value-type-typename-is-not-valid-when-name-is-the-result-of-a-late-bound-expression"></a>Atribuição de associação tardia a um campo de tipo de valor '\<typename >' não é válida quando '\<nome >' é o resultado de uma expressão de associação tardia
Você tentou uma atribuição de associação tardia que não é válida.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a atribuição no início associado.

# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>Atribuição de associação tardia a um campo de tipo de valor '&lt;typename&gt;'não é válida quando'&lt;nome&gt;' é o resultado de uma expressão de associação tardia
Você tentou uma atribuição de associação tardia que não é válida.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a atribuição no início associado.

# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>Objeto não existe mais devido a erro de compilação ou exclusão
Ocorreu um erro de acesso do elemento de código dentro de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador.  
  
 **ID do erro:** BC32300  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Compile o programa novamente para verificar se o erro persiste.  
  
2.  Se o erro persistir, reinstale o compilador [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
3.  Se o erro persistir após a reinstalação, colete informações sobre as circunstâncias e notifique o Microsoft Product Support Services.

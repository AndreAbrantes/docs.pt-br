# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>Arquivo de e/s de uma estrutura com campo &#39; &lt;fieldname&gt;&#39; do tipo &#39;&lt; TypeName&gt;&#39; não é válido
A estrutura que você está tentando gravar em um arquivo contém um tipo sem suporte.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a estrutura para determinar onde o tipo sem suporte ocorre e, em seguida, altere-o para um tipo válido.

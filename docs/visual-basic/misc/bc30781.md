<span data-ttu-id="35389-103">Uma `Continue` instrução deve ser seguida por `Do` , `For` ou `While` , dependendo de se a `Continue` instrução aparecer dentro de um `Do...Loop` loop, `For...Next` loop ou loop `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="35389-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="35389-104">**ID do erro:** BC30781</span><span class="sxs-lookup"><span data-stu-id="35389-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="35389-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="35389-105">To correct this error</span></span>

4. <span data-ttu-id="35389-109">Caso contrário, remova a `Continue` instrução.</span><span class="sxs-lookup"><span data-stu-id="35389-109">Otherwise, remove the `Continue` statement.</span></span>

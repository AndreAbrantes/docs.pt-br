# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="ddfce-102">'Continue' deve ser seguido de 'Do', 'For' ou 'While'</span><span class="sxs-lookup"><span data-stu-id="ddfce-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>

<span data-ttu-id="ddfce-103">Uma `Continue` instrução deve ser seguida por `Do` , `For` ou `While` , dependendo de se a `Continue` instrução aparecer dentro de um `Do...Loop` loop, `For...Next` loop ou loop `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="ddfce-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="ddfce-104">**ID do erro:** BC30781</span><span class="sxs-lookup"><span data-stu-id="ddfce-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ddfce-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="ddfce-105">To correct this error</span></span>  
  
1. <span data-ttu-id="ddfce-106">Se a `Continue` instrução estiver em um `Do...Loop` loop, altere a instrução para `Continue Do` .</span><span class="sxs-lookup"><span data-stu-id="ddfce-106">If the `Continue` statement is in a `Do...Loop` loop, change the statement to `Continue Do`.</span></span>  
  
2. <span data-ttu-id="ddfce-107">Se a `Continue` instrução estiver em um `For...Next` loop, altere a instrução para `Continue For` .</span><span class="sxs-lookup"><span data-stu-id="ddfce-107">If the `Continue` statement is in a `For...Next` loop, change the statement to `Continue For`.</span></span>  
  
3. <span data-ttu-id="ddfce-108">Se a `Continue` instrução estiver em um `While...End While` loop, altere a instrução para `Continue While` .</span><span class="sxs-lookup"><span data-stu-id="ddfce-108">If the `Continue` statement is in a `While...End While` loop, change the statement to `Continue While`.</span></span>  
  
4. <span data-ttu-id="ddfce-109">Caso contrário, remova a `Continue` instrução.</span><span class="sxs-lookup"><span data-stu-id="ddfce-109">Otherwise, remove the `Continue` statement.</span></span>

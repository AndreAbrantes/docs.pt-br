# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argumento &#39;acesso&#39; não é válido. Os valores válidos para o modo Input são &#39;OpenAccess&#39; e &#39;OpenAccess&#39;
Um valor de acesso não é válido em `Input` modo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o valor para `OpenAccess.Read` ou `OpenAccess.Default`.

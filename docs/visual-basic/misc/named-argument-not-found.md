# <a name="named-argument-not-found-visual-basic"></a>Argumento nomeado não encontrado (Visual Basic)
Um argumento nomeado não pode ser usado em uma invocação de procedimento, a menos que ele apareça na definição do procedimento. Você especificou um argumento nomeado, mas o procedimento não foi definido para aceitar um argumento por esse nome.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique a ortografia do nome do argumento.

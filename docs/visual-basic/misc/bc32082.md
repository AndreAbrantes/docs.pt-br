# <a name="type-argument-typeargumentname-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-typeparametername"></a><span data-ttu-id="3d7cb-102">O argumento de tipo ' \<typeargumentname> ' é declarado como ' MustInherit ' e não satisfaz a restrição ' New ' para o parâmetro de tipo ' \<typeparametername> '</span><span class="sxs-lookup"><span data-stu-id="3d7cb-102">Type argument '\<typeargumentname>' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '\<typeparametername>'</span></span>

<span data-ttu-id="3d7cb-103">Um tipo genérico é invocado com uma `MustInherit` classe como um argumento de tipo, enquanto o parâmetro de tipo correspondente é declarado com a `New` restrição.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="3d7cb-104">A `New` restrição requer que o tipo passado no argumento de tipo correspondente tenha suporte à criação de objetos.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="3d7cb-105">No entanto, uma classe *abstrata* , ou seja, uma classe declarada como `MustInherit` , não expõe nenhum construtor porque você não pode criar nenhum objeto a partir dele.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="3d7cb-106">**ID do erro:** BC32082</span><span class="sxs-lookup"><span data-stu-id="3d7cb-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3d7cb-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="3d7cb-107">To correct this error</span></span>  
  
1. <span data-ttu-id="3d7cb-108">Se a classe usada no argumento de tipo não precisar ser abstrata, remova a `MustInherit` palavra-chave de sua declaração.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>  
  
2. <span data-ttu-id="3d7cb-109">Se a classe usada no argumento de tipo precisar ser abstrata, mas não precisar ser usada para construir o tipo genérico, passe uma classe diferente no argumento de tipo.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>  
  
3. <span data-ttu-id="3d7cb-110">Se o parâmetro de tipo correspondente não precisar criar nenhum objeto do tipo passado para ele, remova a `New` restrição de sua declaração.</span><span class="sxs-lookup"><span data-stu-id="3d7cb-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>

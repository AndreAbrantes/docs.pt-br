# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>Não é possível obter informações de tipo para &#39; &lt;typename&gt;&#39;
Foi referenciado um tipo que não tenha sido carregado pelo common language runtime.  
  
 **ID do erro:** BC30751  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reestruture seu código para que o tipo é definido dentro do escopo atual.

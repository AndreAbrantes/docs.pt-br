# <a name="unable-to-get-type-information-for-lttypenamegt"></a>Não é possível obter informações de tipo para '&lt;typename&gt;'
Um tipo foi referenciado que não tenha sido carregado pelo common language runtime.  
  
 **ID do erro:** BC30751  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reestruture o código para que o tipo é definido dentro do escopo atual.

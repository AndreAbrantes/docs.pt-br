# <a name="cast-from-string-quotltstringgtquot-to-type-lttypenamegt-is-not-valid"></a>Conversão de cadeia de caracteres &quot; &lt;cadeia de caracteres&gt; &quot; para o tipo '&lt;typename&gt;' não é válido
Uma tentativa de converter uma variável de cadeia de caracteres para outro tipo não é válida.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os tipos envolvidos na conversão e verifique se que você está tentando converter para um tipo significativo.

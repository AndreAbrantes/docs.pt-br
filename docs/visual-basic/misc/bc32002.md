# <a name="keyword-is-valid-only-within-a-class"></a>'\<palavra-chave >' é válido somente dentro de uma classe
Uma palavra-chave relacionadas às classes, como `Me` ou `MyClass`, é usado fora de uma definição de classe.  
  
 **ID do erro:** BC32002  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Se o código usando a palavra-chave envolve instâncias de classe, mova-o para uma implementação de classe.  
  
-   Se o código usando a palavra-chave não se aplica a classes, remova a palavra-chave inválida.

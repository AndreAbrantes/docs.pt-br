# <a name="39ltkeywordgt39-is-valid-only-within-a-class"></a>&#39; &lt;palavra-chave&gt;&#39; é válido somente dentro de uma classe
Uma palavra-chave relacionada a classes, como `Me` ou `MyClass`, é usada fora de uma definição de classe.  
  
 **ID do erro:** BC32002  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Se o código usando a palavra-chave envolve instâncias de classe, mova-o para uma implementação da classe.  
  
-   Se o código usando a palavra-chave não se aplica a classes, remova a palavra-chave inválida.

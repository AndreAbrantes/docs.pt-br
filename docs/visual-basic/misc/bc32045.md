# <a name="type-39lttypenamegt39-has-no-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="f3f2b-102">Tipo de &#39; &lt;typename&gt;&#39; não tem um tipo parâmetros e, portanto, não podem ter argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="f3f2b-102">Type &#39;&lt;typename&gt;&#39; has no type parameters and so cannot have type arguments</span></span>
<span data-ttu-id="f3f2b-103">Uma declaração ou instrução de atribuição inclui um [de](../../visual-basic/language-reference/statements/of-clause.md) cláusula ao invocar um tipo não genérico.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-103">A declaration or assignment statement includes an [Of](../../visual-basic/language-reference/statements/of-clause.md) clause when invoking a non-generic type.</span></span>  
  
 <span data-ttu-id="f3f2b-104">Por sua definição, um *tipo genérico* é uma classe, estrutura, interface, um procedimento ou representante que funciona em tipos de dados, você pode especificar por meio de um ou mais *parâmetros de tipo*.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-104">By its definition, a *generic type* is a class, structure, interface, procedure, or delegate that operates on data types you can specify through one or more *type parameters*.</span></span> <span data-ttu-id="f3f2b-105">Quando o uso de código cria um tipo desse tipo genérico, ele fornece um *argumento de tipo* para cada parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-105">When the using code creates a type from this generic type, it supplies a *type argument* to each type parameter.</span></span> <span data-ttu-id="f3f2b-106">Como parte da criação do tipo, cada argumento de tipo substitui todas as ocorrências do seu parâmetro de tipo correspondente no código gerado.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-106">As part of the creation of the type, each type argument replaces every occurrence of its corresponding type parameter in the generated code.</span></span>  
  
 <span data-ttu-id="f3f2b-107">Parâmetros de tipo são definidos com um `Of` cláusula dentro de parênteses e argumentos de tipo são fornecidos usando um `Of` cláusula dentro de parênteses.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-107">Type parameters are defined with an `Of` clause inside parentheses, and type arguments are supplied using an `Of` clause inside parentheses.</span></span> <span data-ttu-id="f3f2b-108">O `Of` cláusula é usada apenas ao lidar com tipos genéricos.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-108">The `Of` clause is used only when dealing with generic types.</span></span>  
  
 <span data-ttu-id="f3f2b-109">Tipos não genérico não aceitam parâmetros de tipo, e você não pode especificar argumentos de tipo quando você invoca desse tipo.</span><span class="sxs-lookup"><span data-stu-id="f3f2b-109">Non-generic types do not accept type parameters, and you cannot specify any type arguments when you invoke such a type.</span></span>

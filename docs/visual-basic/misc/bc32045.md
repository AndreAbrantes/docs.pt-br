# <a name="type-typename-has-no-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="8bf75-102">O tipo '\<typename>' não tem parâmetros de tipo e, portanto, não pode ter argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="8bf75-102">Type '\<typename>' has no type parameters and so cannot have type arguments</span></span>

<span data-ttu-id="8bf75-103">Uma declaração ou instrução de atribuição inclui uma cláusula [of](../language-reference/statements/of-clause.md) ao invocar um tipo não genérico.</span><span class="sxs-lookup"><span data-stu-id="8bf75-103">A declaration or assignment statement includes an [Of](../language-reference/statements/of-clause.md) clause when invoking a non-generic type.</span></span>  
  
 <span data-ttu-id="8bf75-104">Por sua definição, um *tipo genérico* é uma classe, estrutura, interface, procedimento ou delegado que opera em tipos de dados que você pode especificar por meio de um ou mais *parâmetros de tipo*.</span><span class="sxs-lookup"><span data-stu-id="8bf75-104">By its definition, a *generic type* is a class, structure, interface, procedure, or delegate that operates on data types you can specify through one or more *type parameters*.</span></span> <span data-ttu-id="8bf75-105">Quando o código de uso cria um tipo desse tipo genérico, ele fornece um *argumento de tipo* para cada parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="8bf75-105">When the using code creates a type from this generic type, it supplies a *type argument* to each type parameter.</span></span> <span data-ttu-id="8bf75-106">Como parte da criação do tipo, cada argumento de tipo substitui cada ocorrência de seu parâmetro de tipo correspondente no código gerado.</span><span class="sxs-lookup"><span data-stu-id="8bf75-106">As part of the creation of the type, each type argument replaces every occurrence of its corresponding type parameter in the generated code.</span></span>  
  
 <span data-ttu-id="8bf75-107">Os parâmetros de tipo são definidos com uma `Of` cláusula dentro dos parênteses, e os argumentos de tipo são fornecidos usando uma `Of` cláusula dentro dos parênteses.</span><span class="sxs-lookup"><span data-stu-id="8bf75-107">Type parameters are defined with an `Of` clause inside parentheses, and type arguments are supplied using an `Of` clause inside parentheses.</span></span> <span data-ttu-id="8bf75-108">A `Of` cláusula é usada somente ao lidar com tipos genéricos.</span><span class="sxs-lookup"><span data-stu-id="8bf75-108">The `Of` clause is used only when dealing with generic types.</span></span>  
  
 <span data-ttu-id="8bf75-109">Tipos não genéricos não aceitam parâmetros de tipo e você não pode especificar nenhum argumento de tipo ao invocar esse tipo.</span><span class="sxs-lookup"><span data-stu-id="8bf75-109">Non-generic types do not accept type parameters, and you cannot specify any type arguments when you invoke such a type.</span></span>

# <a name="39exit-function39-is-not-valid-in-a-sub-or-property"></a>&#39; Função de saída &#39; não é válido em Sub ou Property
`Exit Function`é exibida em uma `Sub` procedimento ou uma `Property` procedimento. Um `Exit` instrução deve coincidir com o bloco no qual ele ocorre.  
  
 **ID do erro:** BC30067  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o `Exit Function` com o `Exit Sub` ou `Exit Property` instrução conforme apropriado.

# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>Apenas os oito primeiros caracteres de um nome de log personalizado são significativos
Ao verificar os nomes de log de eventos de exclusividade, apenas os oito primeiros caracteres são considerados. Um conflito pode resultar nos logs de eventos que compartilham seus primeiros oito caracteres.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Nomeie o log de eventos nos quais os oito primeiros caracteres são exclusivos.

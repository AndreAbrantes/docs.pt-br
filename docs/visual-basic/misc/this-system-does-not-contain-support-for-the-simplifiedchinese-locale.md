# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Este sistema não há suporte para a localidade SimplifiedChinese
Você está tentando usar o `VbStrConv.SimplifiedChinese` membro de enumeração, mas seu sistema não dá suporte para a localidade de chinês simplificado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Atualize a máquina para oferecer suporte à localidade de chinês simplificado ou executar o aplicativo em um computador que dá suporte à localidade.

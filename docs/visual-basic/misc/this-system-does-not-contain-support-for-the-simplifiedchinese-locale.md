# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Este sistema não tem suporte para localidade chinês simplificado
Você está tentando usar o `VbStrConv.SimplifiedChinese` membro de enumeração, mas o sistema não dá suporte para localidade chinês simplificado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade chinês simplificado, ou execute o aplicativo em uma máquina que dá suporte a localidade.

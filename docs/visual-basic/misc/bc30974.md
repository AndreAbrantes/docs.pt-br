# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Instruções de comentário não podem ser avaliadas (não no Sumário)
Uma instrução de comentário foi inserida na janela Immediate durante uma sessão de depuração. Instruções de comentário na janela Immediate não são avaliadas.  
  
 **ID do erro:** BC30974

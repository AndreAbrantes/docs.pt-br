# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>Não é possível incluir fragmento XML &#39; &lt;elementpath&gt; &#39; do arquivo &#39; &lt;commentfile&gt;&#39;
Um `<include>` marca em um comentário XML contém uma expressão XPath para um elemento que não existe ou o arquivo contém XML mal formado.  
  
 **ID do erro:** BC42320  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o XML no arquivo é válido.  
  
2.  Verifique se o fragmento XML existe no arquivo.

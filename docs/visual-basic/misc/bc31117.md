# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>A declaração 'RaiseEvent' deve encerrar com um 'End RaiseEvent'
Um `RaiseEvent` declaração deve terminar com um `End RaiseEvent` instrução.  
  
 **ID do erro:** BC31117  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se o `RaiseEvent` declaração termina com um `End RaiseEvent` instrução.

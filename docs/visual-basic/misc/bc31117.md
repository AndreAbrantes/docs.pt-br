# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent&#39; declaração deve terminar com uma correspondência &#39;End RaiseEvent&#39;
Um `RaiseEvent` declaração deve terminar com um `End RaiseEvent` instrução.  
  
 **ID do erro:** BC31117  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `RaiseEvent` declaração termina com um `End RaiseEvent` instrução.

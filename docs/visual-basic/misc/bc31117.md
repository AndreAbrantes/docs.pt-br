# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>A declaração 'RaiseEvent' deve finalizar com 'End RaiseEvent' correspondente
A `RaiseEvent` declaração deve terminar com um `End RaiseEvent` instrução.  
  
 **ID do erro:** BC31117  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `RaiseEvent` declaração termina com um `End RaiseEvent` instrução.

# <a name="39end-operator39-expected"></a>&#39;End Operator&#39; esperado
Um `End` instrução deve corresponder ao bloco encerra. Um `Operator` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC33005  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar um `End Operator` instrução até o final do procedimento.

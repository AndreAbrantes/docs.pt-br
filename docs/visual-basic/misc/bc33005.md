# <a name="end-operator-expected"></a>'End Operator' esperado
Um `End` instrução deve corresponder ao bloco em que ele é encerrado. Um `Operator` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC33005  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar um `End Operator` instrução até o final do procedimento.

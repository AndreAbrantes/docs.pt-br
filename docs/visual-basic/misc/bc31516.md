# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Tipo de '\<typename >' não pode ser usado como um atributo porque não está declarado como 'Public'
Atributos devem ser públicos.  
  
 **ID do erro:** BC31516  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Adicionar o `Public` modificador para a classe de atributo.

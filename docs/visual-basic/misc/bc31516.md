# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Tipo de '&lt;typename&gt;' não pode ser usado como um atributo porque não está declarado como 'Public'
Atributos devem ser públicos.  
  
 **ID do erro:** BC31516  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar o `Public` modificador para a classe de atributo.

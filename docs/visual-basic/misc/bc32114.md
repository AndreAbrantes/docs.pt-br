# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="e3e4f-102">Parâmetro de tipo com um &#39;estrutura&#39; restrição não pode ser usada como uma restrição</span><span class="sxs-lookup"><span data-stu-id="e3e4f-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="e3e4f-103">Um parâmetro de tipo com um `Structure` restrição é usada como a restrição para outro parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="e3e4f-104">O `Structure` restrição requer que o argumento de tipo passado para o parâmetro de tipo seja um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="e3e4f-105">No entanto, um tipo de valor não pode ser implementado ou herdado, portanto não faz sentido usá-lo como uma restrição, que exige que o outro parâmetro de tipo para implementá-lo ou herdar dele.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="e3e4f-106">A interpretação só é significativa essa situação é que ambos os argumentos de tipo devem ser do mesmo tipo exato.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="e3e4f-107">Se esse for o caso, o tipo genérico precisa apenas um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="e3e4f-108">A instrução a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="e3e4f-109">O tipo passado para `t2` não é possível implementar ou herdar o tipo passado para `t1`, porque o tipo passado para `t1` deve ser um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="e3e4f-110">**ID do erro:** BC32114</span><span class="sxs-lookup"><span data-stu-id="e3e4f-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e3e4f-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="e3e4f-111">To correct this error</span></span>  
  
-   <span data-ttu-id="e3e4f-112">Remova o parâmetro de tipo restringido a `Structure` da lista de restrições no parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="e3e4f-113">Se ambos os parâmetros de tipo requerem o mesmo tipo de valor, defina o tipo genérico com apenas um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="e3e4f-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>

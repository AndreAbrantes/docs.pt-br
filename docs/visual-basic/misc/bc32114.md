# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="b3725-102">Parâmetro de tipo com uma restrição não pode ser usada como uma restrição 'Structure'</span><span class="sxs-lookup"><span data-stu-id="b3725-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="b3725-103">Um parâmetro de tipo com uma `Structure` restrição é usada como a restrição para outro parâmetro do tipo.</span><span class="sxs-lookup"><span data-stu-id="b3725-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="b3725-104">O `Structure` restrição requer que o argumento passado para o parâmetro do tipo seja um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="b3725-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="b3725-105">No entanto, um tipo de valor não pode ser implementado ou herdado, portanto não faz sentido usá-lo como uma restrição, que exigiria o outro parâmetro do tipo para implementá-lo ou herdar a partir dele.</span><span class="sxs-lookup"><span data-stu-id="b3725-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="b3725-106">A interpretação somente significativa nesta situação é que ambos os argumentos de tipo devem ser do mesmo tipo exato.</span><span class="sxs-lookup"><span data-stu-id="b3725-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="b3725-107">Se esse for o caso, o tipo genérico precisa apenas de um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="b3725-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="b3725-108">A instrução a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="b3725-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="b3725-109">O tipo passado para `t2` não pode implementar ou herdar o tipo passado para `t1`, porque o tipo passado para `t1` deve ser um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="b3725-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="b3725-110">**ID do erro:** BC32114</span><span class="sxs-lookup"><span data-stu-id="b3725-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b3725-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="b3725-111">To correct this error</span></span>  
  
-   <span data-ttu-id="b3725-112">Remova o parâmetro do tipo restringido a `Structure` da lista de restrições no parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="b3725-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="b3725-113">Se ambos os parâmetros de tipo exigirem o mesmo tipo de valor, defina o tipo genérico com apenas um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="b3725-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>

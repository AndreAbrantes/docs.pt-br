# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="92d25-102">O parâmetro de tipo com uma restrição 'Structure' não pode ser usado como uma restrição</span><span class="sxs-lookup"><span data-stu-id="92d25-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="92d25-103">Um parâmetro de tipo com uma `Structure` restrição é usado como a restrição para outro parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="92d25-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="92d25-104">A `Structure` restrição requer que o argumento de tipo passado para seu parâmetro de tipo seja um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="92d25-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="92d25-105">No entanto, um tipo de valor não pode ser implementado ou herdado, portanto não é significativo usá-lo como uma restrição, o que exigiria que o outro parâmetro de tipo o implemente ou herde dele.</span><span class="sxs-lookup"><span data-stu-id="92d25-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="92d25-106">A única interpretação significativa dessa situação é que ambos os argumentos de tipo devem ser exatamente do mesmo tipo.</span><span class="sxs-lookup"><span data-stu-id="92d25-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="92d25-107">Se esse for o caso, seu tipo genérico precisará apenas de um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="92d25-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="92d25-108">A instrução a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="92d25-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="92d25-109">O tipo passado para `t2` não pode implementar ou herdar o tipo passado para `t1` , porque o tipo passado para `t1` deve ser um tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="92d25-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="92d25-110">**ID do erro:** BC32114</span><span class="sxs-lookup"><span data-stu-id="92d25-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="92d25-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="92d25-111">To correct this error</span></span>  
  
- <span data-ttu-id="92d25-112">Remova o parâmetro de tipo restrito à `Structure` da lista de restrições no outro parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="92d25-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="92d25-113">Se ambos os parâmetros de tipo exigirem o mesmo tipo de valor, defina o tipo genérico com apenas um parâmetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="92d25-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>

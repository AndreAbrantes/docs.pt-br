# <a name="membername-is-not-declared"></a>'\<MemberName > ' não está declarado
'`<membername>`' não está declarado. `Debug` funcionalidade de objeto está disponível em `System.Diagnostics.Debug` no assembly `System`.  
  
 O nome de membro especificado não pôde ser encontrado.  
  
 **ID do erro:** BC30816  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique a ortografia do membro.  
  
2. Use uma instrução `Imports` ou membros totalmente qualificados definidos em outros namespaces. Por exemplo, a função `WriteLine` é declarada no namespace `System.Diagnostics.Debug`.

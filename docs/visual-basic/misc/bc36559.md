# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="8cc10-102">A propriedade de membro de\<tipo anônimo ' PropertyName > ' não pode ser usada para inferir o tipo de outra propriedade de\<membro porque o tipo de ' PropertyName > ' ainda não foi estabelecido</span><span class="sxs-lookup"><span data-stu-id="8cc10-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="8cc10-103">Até que o tipo de uma propriedade de tipo anônimo seja estabelecido, ele não pode ser usado para estabelecer o tipo de outra propriedade.</span><span class="sxs-lookup"><span data-stu-id="8cc10-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="8cc10-104">Por exemplo, na Declaração `.IDName = .LastName` a seguir não é válida porque `.LastName` ainda não foi inicializada.</span><span class="sxs-lookup"><span data-stu-id="8cc10-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.

```

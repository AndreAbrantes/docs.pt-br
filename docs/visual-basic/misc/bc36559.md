# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="f7a39-102">A propriedade de membro de tipo anônimo '\<propertyname>' não pode ser usada para deduzir o tipo de uma outra propriedade de membro porque o tipo de '\<propertyname>' ainda não está estabelecido</span><span class="sxs-lookup"><span data-stu-id="f7a39-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="f7a39-103">Até que o tipo de uma propriedade de tipo anônimo seja estabelecido, ele não pode ser usado para estabelecer o tipo de outra propriedade.</span><span class="sxs-lookup"><span data-stu-id="f7a39-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="f7a39-104">Por exemplo, na declaração a seguir `.IDName = .LastName` não é válida porque `.LastName` ainda não foi inicializada.</span><span class="sxs-lookup"><span data-stu-id="f7a39-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="f7a39-105">**ID do erro:** BC36559</span><span class="sxs-lookup"><span data-stu-id="f7a39-105">**Error ID:** BC36559</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f7a39-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="f7a39-106">To correct this error</span></span>

- <span data-ttu-id="f7a39-107">Estabeleça o tipo da propriedade antes de usá-la para inicializar outra propriedade.</span><span class="sxs-lookup"><span data-stu-id="f7a39-107">Establish the type of the property before using it to initialize another property.</span></span>


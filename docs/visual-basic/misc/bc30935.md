# <a name="member-39ltclassnamegtltprocedurenamegt39-that-matches-this-signature-cannot-be-overridden-because-the-class-39ltclassnamegt39-contains-multiple-members-with-this-same-name-and-signature-ltsignaturelistgt"></a><span data-ttu-id="bf6b1-102">Membro &#39; &lt;classname&gt;.&lt; procedurename&gt;&#39; que corresponde a esta assinatura não pode ser substituída porque a classe &#39;&lt; nome da classe&gt;&#39; contém vários membros com o mesmo nome e assinatura: &lt;signaturelist&gt;</span><span class="sxs-lookup"><span data-stu-id="bf6b1-102">Member &#39;&lt;classname&gt;.&lt;procedurename&gt;&#39; that matches this signature cannot be overridden because the class &#39;&lt;classname&gt;&#39; contains multiple members with this same name and signature: &lt;signaturelist&gt;</span></span>
<span data-ttu-id="bf6b1-103">Um procedimento ou propriedade tenta substituir um procedimento ou propriedade herdados, mas o compilador encontra mais de uma versão do procedimento básico ou propriedade com o mesmo nome e assinatura.</span><span class="sxs-lookup"><span data-stu-id="bf6b1-103">A procedure or property attempts to override an inherited procedure or property, but the compiler finds more than one version of the base procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="bf6b1-104">Esse erro pode ocorrer em uma situação com tipos genéricos construídos, como as seguintes declarações de esqueleto ilustram.</span><span class="sxs-lookup"><span data-stu-id="bf6b1-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```  
Public Class baseClass(Of t)  
    Public Overridable Sub doSomething(ByVal inputValue As String)  
    End Sub  
    Public Overridable Sub doSomething(ByVal inputValue As t)  
    End Sub  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of String)  
    Overrides Sub doSomething(ByVal inputValue As String)  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="bf6b1-105">Porque `derivedClass` herda `baseClass` fornecendo `String` para seu parâmetro de tipo `t`, as duas versões do `doSomething` na `baseClass` assumem assinaturas idênticas, até onde `derivedClass` está interessado.</span><span class="sxs-lookup"><span data-stu-id="bf6b1-105">Because `derivedClass` inherits `baseClass` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseClass` take on identical signatures as far as `derivedClass` is concerned.</span></span> <span data-ttu-id="bf6b1-106">Como resultado, o compilador não pode determinar a versão que deseja substituir.</span><span class="sxs-lookup"><span data-stu-id="bf6b1-106">As a result, the compiler cannot determine which version to override.</span></span>  
  
 <span data-ttu-id="bf6b1-107">**ID do erro:** BC30935</span><span class="sxs-lookup"><span data-stu-id="bf6b1-107">**Error ID:** BC30935</span></span>

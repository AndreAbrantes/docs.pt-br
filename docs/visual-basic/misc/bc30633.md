# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;Definir&#39; deve finalizar com uma correspondência &#39;final definido&#39;
`Set` procedimentos de propriedade devem terminar com um `End Set` instrução.  
  
 **ID do erro:** BC30633  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `Set` procedimento de propriedade termina com um `End Set` construir.

# <a name="mustinherit-cannot-be-specified-for-partial-type-partialtypename-because-it-cannot-be-combined-with-notinheritable-specified-for-one-of-its-other-partial-types"></a><span data-ttu-id="bab6e-102">'MustInherit' não pode ser especificado para o tipo parcial '\<partialtypename >' porque ele não pode ser combinado com 'NotInheritable' especificado para um de seus outros tipos parciais</span><span class="sxs-lookup"><span data-stu-id="bab6e-102">'MustInherit' cannot be specified for partial type '\<partialtypename>' because it cannot be combined with 'NotInheritable' specified for one of its other partial types</span></span>
<span data-ttu-id="bab6e-103">Uma classe é definida em várias declarações parciais, um dos quais especifica `MustInherit` e outra que especifica `NotInheritable`.</span><span class="sxs-lookup"><span data-stu-id="bab6e-103">A class is defined in multiple partial declarations, one of which specifies `MustInherit` and another of which specifies `NotInheritable`.</span></span>  
  
 <span data-ttu-id="bab6e-104">Quando você divide a definição de uma classe entre várias declarações parciais, o compilador trata a classe como a união de todas as suas declarações parciais.</span><span class="sxs-lookup"><span data-stu-id="bab6e-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="bab6e-105">Isso se aplica não apenas aos membros, mas também para a implementação, a herança e o nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="bab6e-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="bab6e-106">Uma classe não pode ser ambos *abstrata* e *lacrado*, que significa que ele não pode exigir tanto proibir herança.</span><span class="sxs-lookup"><span data-stu-id="bab6e-106">A class cannot be both *abstract* and *sealed*, meaning it cannot both require and forbid inheritance.</span></span> <span data-ttu-id="bab6e-107">Portanto, não é possível especificar ambos `MustInherit` e `NotInheritable` para a mesma classe.</span><span class="sxs-lookup"><span data-stu-id="bab6e-107">Therefore you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="bab6e-108">**ID do erro:** BC30926</span><span class="sxs-lookup"><span data-stu-id="bab6e-108">**Error ID:** BC30926</span></span>

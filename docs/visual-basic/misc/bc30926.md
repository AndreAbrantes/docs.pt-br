 <span data-ttu-id="5f623-104">Quando você divide a definição de uma classe entre várias declarações parciais, o compilador trata a classe como a união de todas as suas declarações parciais.</span><span class="sxs-lookup"><span data-stu-id="5f623-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="5f623-105">Isso se aplica não apenas aos membros, mas também para a implementação, herança e nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="5f623-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="5f623-106">Uma classe não pode ser *abstrata* e *lacrado*, que significa que ela não pode exigir tanto proibir herança.</span><span class="sxs-lookup"><span data-stu-id="5f623-106">A class cannot be both *abstract* and *sealed*, meaning it cannot both require and forbid inheritance.</span></span> <span data-ttu-id="5f623-107">Portanto, não é possível especificar `MustInherit` e `NotInheritable` para a mesma classe.</span><span class="sxs-lookup"><span data-stu-id="5f623-107">Therefore you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>

# <a name="method-39ltmethodnamegt39-cannot-be-called-with-ltnumbergt-arguments"></a>Método &#39; &lt;methodname&gt; &#39; não pode ser chamado com &lt;número&gt; argumentos
Uma chamada de método contém o número errado de argumentos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Contagem de argumentos que você forneceu.  
  
2.  Corresponde ao número em relação ao número de parâmetros no método que ser chamado.  
  
3.  Se os dois números não concordar, ajuste o número de argumentos.

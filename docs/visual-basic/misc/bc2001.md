# <a name="file-ltfilenamegt-could-not-be-found"></a>Arquivo &lt;filename&gt; não pôde ser encontrado
O arquivo especificado não pôde ser localizado. Isso pode ser resultado de fornecer um caminho de arquivo incorreto.  
  
 **ID do erro:** BC2001  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o nome do arquivo e o caminho estão corretos.

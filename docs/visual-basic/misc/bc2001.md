# <a name="file-filename-could-not-be-found"></a>Arquivo \<filename > não pôde ser encontrado
O arquivo especificado não pôde ser localizado. Isso pode ser um resultado de fornecer um caminho de arquivo incorreto.  
  
 **ID do erro:** BC2001  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o nome de arquivo e caminho estão corretos.

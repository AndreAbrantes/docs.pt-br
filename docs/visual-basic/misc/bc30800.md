# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumentos de método devem ser colocados entre parênteses
As regras que controlam a chamadas de procedimento são mais simples em versões mais recentes do Visual Basic. Todos os argumentos devem ser delimitados por parênteses.  
  
 **ID do erro:** BC30800  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Coloque a lista de argumentos entre parênteses; Por exemplo:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```

# <a name="sub-or-function-expected"></a>'Sub' ou 'Function' esperado
Um `Declare` instrução não especifica nem `Sub` ou `Function`. `Declare` identifica um procedimento externo e deve especificar qual tipo de procedimento.  
  
 **ID do erro:** BC30215  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione a `Sub` ou `Function` palavra-chave para o `Declare` instrução.

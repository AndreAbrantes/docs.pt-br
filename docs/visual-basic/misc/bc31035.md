# <a name="interface-39ltinterfacenamegt39-is-not-implemented-by-this-class"></a>Interface &#39; &lt;interfacename&gt;&#39; não é implementada por esta classe
Um membro dessa classe ou estrutura tenta implementar um membro de uma interface que a classe ou estrutura não implementa.  
  
 **ID do erro:** BC31035  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Implements` instrução no início da classe ou estrutura de modo que ele implementa a interface apropriada.  
  
-   Remover o `Implements` palavra-chave do membro que causa esse erro.

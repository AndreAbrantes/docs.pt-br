# <a name="security-attribute-attributename-is-not-valid-error"></a>Atributo de segurança '\<attributename >' não é válido: \<erro >
O common language runtime não aceita o atributo em um contexto de segurança.

**ID do erro:** BC30128

## <a name="to-correct-this-error"></a>Para corrigir este erro

1. Remova o atributo da declaração ou substituí-lo com um atributo que é válido para o contexto.

2. Examine a mensagem de erro entre aspas e tomar as devidas providências.

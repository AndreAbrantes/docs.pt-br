 <span data-ttu-id="96b0a-104">Quando o código usa o endereço de um procedimento, o compilador tenta encontrar uma versão desse procedimento com uma lista de parâmetros que corresponde à do delegado.</span><span class="sxs-lookup"><span data-stu-id="96b0a-104">When code uses the address of a procedure, the compiler tries to find a version of that procedure with a parameter list that matches that of the delegate.</span></span> <span data-ttu-id="96b0a-105">Se o procedimento for definido em várias versões sobrecarregadas, o compilador tentará encontrar uma única versão com uma assinatura correspondente.</span><span class="sxs-lookup"><span data-stu-id="96b0a-105">If the procedure is defined in several overloaded versions, the compiler tries to find a single version with a matching signature.</span></span> <span data-ttu-id="96b0a-106">Para obter mais informações, consulte [sobrecarga de resolução](../programming-guide/language-features/procedures/overload-resolution.md).</span><span class="sxs-lookup"><span data-stu-id="96b0a-106">For more information, see [Overload Resolution](../programming-guide/language-features/procedures/overload-resolution.md).</span></span>

# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39; RemoveHandler &#39; a declaração deve terminar com uma correspondência &#39; End RemoveHandler &#39;
Um `RemoveHandler` declaração deve terminar com um `End RemoveHandler` instrução.  
  
 **ID do erro:** BC31116  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `RemoveHandler` declaração termina com um `End RemoveHandler` instrução.

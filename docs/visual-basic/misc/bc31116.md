# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>A declaração 'RemoveHandler' deve encerrar com um 'End RemoveHandler' correspondente
Um `RemoveHandler` declaração deve terminar com um `End RemoveHandler` instrução.  
  
 **ID do erro:** BC31116  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `RemoveHandler` declaração termina com um `End RemoveHandler` instrução.

# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>Prefixo de namespace XML 'prefixo' já está declarado
Um prefixo de namespace XML definido por um `Imports` instrução corresponde a um prefixo de namespace XML definido anteriormente.  
  
 **ID do erro:** BC30573  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use um nome exclusivo para o prefixo de namespace XML.

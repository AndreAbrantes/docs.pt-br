## <a name="example"></a><span data-ttu-id="db676-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="db676-111">Example</span></span>

<span data-ttu-id="db676-112">O exemplo a seguir usa `AttributeUsage` o atributo para especificar a quais elementos de programa a `ExtensionAttribute` nova versão do pode ser aplicada.</span><span class="sxs-lookup"><span data-stu-id="db676-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="db676-113">O exemplo especifica `AttributeTargets` três membros da enumeração: `Assembly`, `Class`e `Method`.</span><span class="sxs-lookup"><span data-stu-id="db676-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="db676-114">A omissão de qualquer um desses elementos causará esse erro.</span><span class="sxs-lookup"><span data-stu-id="db676-114">The omission of any one of these elements will cause this error.</span></span>

        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace

## <a name="example"></a><span data-ttu-id="cf06a-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="cf06a-111">Example</span></span>  
 <span data-ttu-id="cf06a-112">O exemplo a seguir usa o `AttributeUsage` atributo para especificar quais elementos do programa a nova versão do `ExtensionAttribute` pode aplicar a.</span><span class="sxs-lookup"><span data-stu-id="cf06a-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="cf06a-113">O exemplo especifica três membros do `AttributeTargets` enumeração: `Assembly`, `Class`, e `Method`.</span><span class="sxs-lookup"><span data-stu-id="cf06a-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="cf06a-114">A omissão de qualquer um desses elementos fará com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="cf06a-114">The omission of any one of these elements will cause this error.</span></span>  
  
```  
Namespace System.Runtime.CompilerServices  
    <AttributeUsage(AttributeTargets.Assembly Or _  
        AttributeTargets.Class Or AttributeTargets.Method)>  
    Class ExtensionAttribute  
        Inherits System.Attribute  
        ' Definitions of methods, fields, and properties.  
    End Class  
End Namespace  
```

```  
  
 <span data-ttu-id="cf06a-116">Excluindo o `AttributeUsage` linha, conforme mostrado no código a seguir produz o mesmo resultado.</span><span class="sxs-lookup"><span data-stu-id="cf06a-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>

    End Class

```

## <a name="example"></a><span data-ttu-id="86d7a-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="86d7a-111">Example</span></span>  
 <span data-ttu-id="86d7a-112">O exemplo a seguir usa o `AttributeUsage` atributo para especificar quais elementos do programa a nova versão do `ExtensionAttribute` pode aplicar a.</span><span class="sxs-lookup"><span data-stu-id="86d7a-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="86d7a-113">O exemplo especifica três membros do `AttributeTargets` enumeração: `Assembly`, `Class`, e `Method`.</span><span class="sxs-lookup"><span data-stu-id="86d7a-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="86d7a-114">A omissão de qualquer um desses elementos fará com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="86d7a-114">The omission of any one of these elements will cause this error.</span></span>

        AttributeTargets.Class Or AttributeTargets.Method)>

    End Class

# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Erro ao inserir o manifesto Win32: o arquivo de manifesto \<file > não pode ser encontrado
O arquivo especificado para a opção de compilador `/win32manifest` não pode ser encontrado.  
  
 **ID do erro:** BC31192  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Forneça um nome de arquivo e um caminho válidos para a opção de compilador `/win32manifest`.

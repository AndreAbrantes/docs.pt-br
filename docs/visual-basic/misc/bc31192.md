# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Erro ao inserir o manifesto Win32: Arquivo de manifesto &lt;arquivo&gt; não pode ser encontrado
O arquivo especificado para o `/win32manifest` opção de compilador não pode ser encontrada.  
  
 **ID do erro:** BC31192  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Forneça um nome de arquivo válido e o caminho para o `/win32manifest` opção de compilador.

# <a name="39ltassemblynamegt39-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'&lt;assemblyname&gt;' é um assembly e não pode ser referenciado como um módulo
Foi feita uma tentativa para usar um assembly como um módulo.  
  
 **ID do erro:** BC31077  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a referência a um módulo.

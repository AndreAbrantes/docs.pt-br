# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Nome do arquivo já declarado com um valor GUID e soma de verificação diferente
Foi feita uma tentativa para declarar um arquivo, mas o arquivo já existe com um valor GUID e soma de verificação diferente.  
  
 **ID do erro:** BC42034  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique o nome do arquivo e o caminho para garantir que cada um está correto.  
  
- Exclua o arquivo de redundante se ele não é necessário.

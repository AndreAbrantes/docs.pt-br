# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Nome de arquivo já declarado com um valor diferente de GUID e soma de verificação
Foi feita uma tentativa para declarar um arquivo, mas o arquivo já existe com um valor GUID e soma de verificação diferente.  
  
 **ID do erro:** BC42034  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o nome do arquivo e o caminho para certificar-se de que cada um está correto.  
  
-   Exclua o arquivo redundante se ele não for necessário.

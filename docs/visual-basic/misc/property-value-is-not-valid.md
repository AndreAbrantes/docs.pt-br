# <a name="property-value-is-not-valid"></a>Valor da propriedade não é válido
Uma das propriedades de um objeto ou o controle foi definida como um valor fora do intervalo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere o valor da propriedade para uma configuração válida.

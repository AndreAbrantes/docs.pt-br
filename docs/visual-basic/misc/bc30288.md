# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>Variável local '\<variablename >' já está declarada no bloco atual
Duas variáveis com o mesmo nome são declaradas no mesmo bloco (que pode ser um procedimento ou um bloco de controle, como `If...Then`).  
  
 **ID do erro:** BC30288  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Mover uma das declarações para um bloco diferente ou alterar o nome de uma das variáveis.

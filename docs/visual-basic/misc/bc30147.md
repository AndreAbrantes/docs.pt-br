# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Return' não são válidas na janela imediata
O `Return` instrução executa a ramificação e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30147  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita uma `Return` instrução em de **imediato** janela.

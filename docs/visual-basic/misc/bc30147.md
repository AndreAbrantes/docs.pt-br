# <a name="39return39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Retorno &#39; instruções não são válidas na janela imediata
O `Return` instrução executa a ramificação e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30147  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita um `Return` instrução o **imediato** janela.

# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>A versão de destino do .NET Compact Framework não oferece suporte a resolução de sobrecarga latebound
A versão de destino do .NET Compact Framework não dá suporte a resolução de sobrecarga com associação tardia.  
  
 **ID do erro:** BC30764  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Parâmetros de conversão do tipo de objeto para o tipo apropriado.

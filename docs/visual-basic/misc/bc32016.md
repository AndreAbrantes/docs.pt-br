# <a name="39ltprocedurenamegt39-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>&#39; &lt;procedurename&gt;&#39; não tem parâmetros e seu tipo de retorno não pode ser indexado.
Uma chamada para um `Function` ou `Sub` procedimento fornece um ou mais argumentos, mas o procedimento não aceita argumentos e seu tipo de retorno (se for um `Function`) não é um tipo de matriz.  
  
 **ID do erro:** BC32016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento ou argumentos de chamada de procedimento.

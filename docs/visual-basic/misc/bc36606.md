# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-39object39-class"></a>Nome da variável de intervalo não pode coincidir com o nome de um membro do &#39;objeto&#39; classe
Uma variável de intervalo em uma consulta LINQ corresponde ao nome de um membro do <xref:System.Object> classe. Isso cria um conflito com os objetos criados pelo Visual Basic para a consulta LINQ.  
  
 **ID do erro:** BC36606  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Escolha um novo nome para a variável de intervalo que não coincide com o nome de qualquer membro de <xref:System.Object> classe.

# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-object-class"></a>Nome de variável de intervalo não pode corresponder ao nome de um membro da classe 'Object'
O nome de um membro da corresponde a uma variável de intervalo em uma consulta LINQ a <xref:System.Object> classe. Isso cria um conflito com os objetos criados pelo Visual Basic para a consulta LINQ.  
  
 **ID do erro:** BC36606  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Escolha um novo nome para a variável de intervalo que não coincide com o nome de qualquer membro do <xref:System.Object> classe.

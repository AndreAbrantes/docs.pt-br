# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Método '\<methodname >' já está declarado na interface '\<interfacename >'
Você declarou um método de interface uma segunda vez, duplicá-la.  
  
 **ID do erro:** BC30377  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a declaração duplicada.

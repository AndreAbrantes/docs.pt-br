# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>'MyBase' não pode ser usado com o método '\<methodname >' porque ele está declarado como 'MustOverride'
Você tentou usar `MyBase` com um método que foi declarado `MustOverride`.  
  
 **ID do erro:** BC30399  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remover o `MustOverride` declaração.

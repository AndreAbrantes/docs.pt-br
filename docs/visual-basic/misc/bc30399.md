# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39; MyBase &#39; não pode ser usado com o método &#39; &lt;methodname&gt;&#39; porque está declarado como &#39; MustOverride &#39;
Você tentou usar `MyBase` com um método que foi declarado `MustOverride`.  
  
 **ID do erro:** BC30399  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `MustOverride` declaração.

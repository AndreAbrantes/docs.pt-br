# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Alias Imports &#39; &lt;nome&gt; &#39; está em conflito com &#39; &lt;name2&gt; &#39; declarado no namespace raiz
O nome de alias usado no `Imports` instrução entra em conflito com outro nome declarado no namespace raiz.  
  
 **ID do erro:** BC31403  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Escolha um nome exclusivo para o alias usado pelo `Imports` instrução.

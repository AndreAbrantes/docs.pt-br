# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Alias de importações\<nome >' está em conflito com '\<nome2 >' declarado no namespace raiz
O nome de alias usado no `Imports` instrução entra em conflito com outro nome declarado no namespace raiz.  
  
 **ID do erro:** BC31403  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Escolha um nome exclusivo para o alias usado pelo `Imports` instrução.

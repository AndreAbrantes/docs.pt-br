# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Alias de importações&lt;nome&gt;'está em conflito com'&lt;nome2&gt;' declarado no namespace raiz
O nome de alias usado no `Imports` instrução entra em conflito com outro nome declarado no namespace raiz.  
  
 **ID do erro:** BC31403  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Escolha um nome exclusivo para o alias usado pelo `Imports` instrução.

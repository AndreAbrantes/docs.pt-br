<span data-ttu-id="7f56b-103">Você não pode usar um caractere de tipo em um nome de propriedade quando você declara uma instância de um tipo anônimo.</span><span class="sxs-lookup"><span data-stu-id="7f56b-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="7f56b-104">O tipo de dados da propriedade é inferido do valor atribuído a ele.</span><span class="sxs-lookup"><span data-stu-id="7f56b-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="7f56b-105">Por exemplo, as seguintes declarações não são válidas.</span><span class="sxs-lookup"><span data-stu-id="7f56b-105">For example, the following declarations are not valid.</span></span>  
  
```vb

'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="7f56b-106">**ID do erro:** BC36560</span><span class="sxs-lookup"><span data-stu-id="7f56b-106">**Error ID:** BC36560</span></span>

# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Efeitos colaterais inválidos durante a avaliação da expressão neste contexto
Instruções que alteram o estado de um objeto não são permitidas neste contexto.  
  
 **ID do erro:** BC30700  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere seu código para que as instruções sejam avaliadas sem alterar o estado dos outros itens.

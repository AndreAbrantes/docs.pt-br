# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Por fim&#39; instruções não são válidas na janela imediata
`Finally` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30716  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Finally` instruções do código de depuração.

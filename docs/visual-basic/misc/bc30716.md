# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' Finally ' não são válidas na janela Immediate
`Finally` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30716  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Finally` instruções do código de depuração.

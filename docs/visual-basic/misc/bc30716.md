# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>'Finally' instruções não são válidas na janela imediata
`Finally` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30716  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Finally` instruções do código de depuração.

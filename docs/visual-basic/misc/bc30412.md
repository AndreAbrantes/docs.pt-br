# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39;WithEvents&#39; variáveis devem ter um &#39;como&#39; cláusula
Você não forneceu um `As` cláusula com a palavra-chave `WithEvents`. Declare a variável como a classe específica que pode gerar eventos.  
  
 **ID do erro:** BC30412  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar o `As` cláusula que especifica a classe que pode gerar eventos.

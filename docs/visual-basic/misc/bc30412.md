# <a name="withevents-variables-must-have-an-as-clause"></a>Variáveis 'WithEvents' devem ter uma cláusula 'As'
Você não forneceu uma `As` cláusula com a palavra-chave `WithEvents`. Declare a variável como a classe específica que pode gerar eventos.  
  
 **ID do erro:** BC30412  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar o `As` cláusula que especifica a classe que pode lançar os eventos.

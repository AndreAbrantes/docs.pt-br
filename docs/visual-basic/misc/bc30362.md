# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt; &#39; não é possível substituir &#39; &lt;propertyname2&gt; &#39; porque diferem por &#39;ReadOnly&#39; ou &#39;WriteOnly&#39;
Você tentou substituir uma propriedade com uma segunda que difere da primeira somente pelo `ReadOnly` ou `WriteOnly`.  
  
 **ID do erro:** BC30362  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se as propriedades são diferenciadas por mais de `ReadOnly` e `WriteOnly`.

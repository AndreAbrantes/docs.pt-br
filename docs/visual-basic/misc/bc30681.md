# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39;#Region&#39; deve finalizar com uma correspondência &#39;#End região&#39;
Use o `#Region` diretiva para especificar um bloco de código que você pode expandir ou recolher ao usar o recurso de estrutura de tópicos do Editor de código do Visual Studio. O início e fim de `#Region` instruções devem estar no mesmo bloco de código.  
  
 **ID do erro:** BC30681  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Inserir `#End Region` no local apropriado do código após a `#Region` instrução.

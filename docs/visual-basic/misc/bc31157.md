# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>Atributo XML 'attribute1' deve aparecer antes do atributo XML 'attribute2'
Atributos em um literal de documento XML são especificados em uma ordem inválida. Ordem de atributo válido para um literal de documento XML se baseia na especificação XML 1.0. Por exemplo, o `encoding` atributo de um literal de documento XML deve vir logo após o `version` atributo.  
  
 **ID do erro:** BC31157  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atualize a ordem de atributo para o literal de documento XML usando as diretrizes especificadas na especificação XML 1.0.

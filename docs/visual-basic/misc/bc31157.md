# <a name="xml-attribute-39attribute139-must-appear-before-xml-attribute-39attribute239"></a>Atributo XML &#39; attribute1 &#39; deve aparecer antes do atributo XML &#39; attribute2 &#39;
Atributos em um literal de documento XML são especificados em uma ordem inválida. Ordem de atributo válido para um literal de documento XML é baseada na especificação XML 1.0. Por exemplo, o `encoding` atributo de um literal de documento XML deve seguir imediatamente a `version` atributo.  
  
 **ID do erro:** BC31157  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atualize a ordem de atributo para o literal de documento XML usando as instruções especificadas na especificação XML 1.0.

# <a name="nullable-parameters-must-specify-a-type"></a>Parâmetros que permitem valor nulos devem especificar um tipo
Um parâmetro inclui o modificador de tipo anulável (?), mas não especifica um tipo.  
  
 **ID do erro:** BC36632  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique um tipo ao declarar o parâmetro que permite valor nulo.

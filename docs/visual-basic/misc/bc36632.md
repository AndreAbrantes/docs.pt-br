# <a name="nullable-parameters-must-specify-a-type"></a>Os parâmetros que permitem valor nulo devem especificar um tipo
Um parâmetro inclui o modificador de tipo anulado (?) mas não especifica um tipo.  
  
 **ID de erro:** BC36632  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Especifique um tipo ao declarar o parâmetro de tipo de valor anulado.

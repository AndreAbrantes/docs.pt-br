# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39; &lt;namespacename&gt;&#39; é ambíguo
Você forneceu um nome que é ambíguo e, portanto, está em conflito com outro nome. O [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compilador não tem quaisquer regras de resolução de conflitos; você deve resolver a ambiguidade de nomes por conta própria.  
  
 **ID do erro:** BC30554  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Qualifica totalmente o nome.

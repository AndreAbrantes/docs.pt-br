# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Valor do atributo não é válido. esperando 'value1' ou 'value2'
Um valor inválido para um atributo em um literal de documento XML foi especificado. Valores de atributo válidos para um literal de documento XML se baseiam na especificação XML 1.0.  
  
 **ID do erro:** BC31182  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o valor do atributo para um valor que é válido de acordo com a especificação XML 1.0.

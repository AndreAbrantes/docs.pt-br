# <a name="keyword-is-not-valid-as-an-identifier"></a>Palavra-chave não é válido como um identificador
Uma palavra-chave ocorre em que é necessário um nome de elemento. O contexto de chamadas para um elemento como uma variável ou um procedimento.  
  
 **ID do erro:** BC30183  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Substitua a palavra-chave com um nome de elemento válido.

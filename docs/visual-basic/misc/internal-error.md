# <a name="internal-error-visual-basic"></a>Erro interno (Visual Basic)
Ocorreu um mau funcionamento interno em [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se esse erro não foi gerado pelo `Error` instrução ou `Raise` método. A menos que ela foi, entre em contato com o Atendimento Microsoft para relatar as condições sob as quais a mensagem apareceu.

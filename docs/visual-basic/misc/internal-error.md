# <a name="internal-error-visual-basic"></a>Erro interno (Visual Basic)
Um mau funcionamento interno ocorreu no Visual Basic.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se esse erro não foi gerado pelo `Error` instrução ou `Raise` método. A menos que ele era, entre em contato com o Atendimento Microsoft para relatar as condições sob as quais a mensagem foi exibida.

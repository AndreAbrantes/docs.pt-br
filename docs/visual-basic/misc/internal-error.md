# <a name="internal-error-visual-basic"></a>Erro interno (Visual Basic)
Ocorreu um problema interno no Visual Basic.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se esse erro não foi gerado pela instrução `Error` ou pelo método `Raise`. A menos que tenha sido, entre em contato com o atendimento Microsoft para relatar as condições sob as quais a mensagem apareceu.

# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'\<procedurename >' tem um tipo de retorno que não é suportado ou tipos de parâmetro que não são suportados
Um procedimento é declarado com parâmetros ou um tipo de retorno não pode ser representado pelo Visual Studio.  
  
 **ID do erro:** BC30657  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Defina os parâmetros ou tipos de retorno como um tipo com suporte.  
  
-   Desenvolva um novo tipo que descreve os dados usados por esse procedimento.

# <a name="a-nullable-type-cannot-be-inferred-for-variable-variablename"></a>Um tipo anulado não pode ser\<inferido para variável ' variablename>'
Um tipo anulado não pode ser inferido a partir de um `String`tipo de referência, como uma matriz, uma classe ou um . O valor a partir do qual o tipo de dados é inferido deve ser um tipo de valor. O código a seguir ilustra este erro.  
  
```vb  
'' Not valid.
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **ID de erro:** BC36628  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a designação anulada.

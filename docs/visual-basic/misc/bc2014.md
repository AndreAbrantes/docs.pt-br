# <a name="the-value-value-is-invalid-for-option-optionname"></a>o valor '\<valor >' é inválido para a opção '\<optionname >'
Um valor foi fornecido para uma opção de linha de comando que não era válida.  
  
 **ID do erro:** BC2014  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os valores para encontrar aquele que não é válido.

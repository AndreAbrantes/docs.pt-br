# <a name="the-value-ltvaluegt-is-invalid-for-option-ltoptionnamegt"></a>o valor '&lt;valor&gt;'é inválido para a opção'&lt;optionname&gt;'

 **ID do erro:** BC2014  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os valores para encontrar aquele que não é válido.

# <a name="unable-to-create-a-net-runtime-interface-error"></a>Não é possível criar uma interface .NET Runtime: \<erro >
Ocorreu um erro interno do compilador criando uma interface ou uma chamada de método em uma interface falhou.  
  
 **ID do erro:** BC31024  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Salve seu trabalho e reinicie o Visual Studio.  
  
2.  Se o erro persistir, reinstale o Visual Studio.  
  
3.  Se o erro persistir após a reinstalação, notifique o Microsoft Product Support Services.

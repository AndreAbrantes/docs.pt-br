# <a name="xml-namespace-prefix-39prefix39-is-not-defined"></a>Prefixo de namespace XML &#39;prefixo&#39; não está definido
Um prefixo de namespace XML indefinido foi especificado para um literal XML ou uma propriedade de eixo XML.  
  
 **ID do erro:** BC31148  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Definir o prefixo de namespace XML usando o `Imports` instrução.  
  
 -ou-  
  
-   Modifique o prefixo de namespace XML para o XML literal ou uma propriedade de eixo XML para um prefixo de namespace XML definido anteriormente.

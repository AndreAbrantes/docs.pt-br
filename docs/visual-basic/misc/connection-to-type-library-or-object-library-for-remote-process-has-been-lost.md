# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Conexão para o tipo de biblioteca de objetos para processo remoto foi perdida
Você perdeu sua conexão à biblioteca de objetos de processo remoto ou biblioteca de tipos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Reinicie o `Application` objeto.  
  
2. Na caixa de diálogo de erro por meio do qual você entrou nesse tópico de Ajuda, clique em **Okey** para exibir o **referências** caixa de diálogo. A referência perdida aparece com a palavra "Ausente" à sua esquerda.  
  
3. Exclua a referência perdida.  
  
4. No **referências** caixa de diálogo, selecione a caixa de seleção o `Application` objeto.

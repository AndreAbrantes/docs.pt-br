# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Bloco de comentário XML não pode ser associado a qualquer elemento de linguagem que ofereça suporte a aplicação de comentários de documentação XML
Bloco de comentário XML não pode ser associado a qualquer elemento de linguagem válido. O comentário XML será ignorado. Esse erro também ocorre se uma marca XML no bloco de comentário não tem nenhum elemento de linguagem correspondente.  
  
 **ID do erro:** BC42312  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Localizar e remova a marca XML estranhas ou fornecer o elemento de linguagem que ele descreve.

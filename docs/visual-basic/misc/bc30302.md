# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Caractere de tipo '\<caractere >' não pode ser usado em uma declaração com um tipo explícito
Você usou um caractere de tipo, como `$`, ao especificar um tipo explicitamente.  
  
 **ID do erro:** BC30302  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere de tipo ou remova o `As <Type>` cláusula da declaração.

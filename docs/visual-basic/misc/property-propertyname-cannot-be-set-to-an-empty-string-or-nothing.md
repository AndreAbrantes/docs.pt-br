# <a name="property-ltpropertynamegt-cannot-be-set-to-an-empty-string-or-nothing"></a>Propriedade &lt;propertyname&gt; não pode ser definido como uma cadeia de caracteres vazia ou Nothing
A propriedade requer um valor diferente de uma cadeia de caracteres vazia ("") ou `Nothing`. Isso pode ser resultado de calcular incorretamente o valor.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça um valor válido.

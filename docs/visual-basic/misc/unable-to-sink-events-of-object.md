# <a name="unable-to-sink-events-of-object-because-the-object-is-already-firing-events-to-the-maximum-number-of-event-receivers-it-supports"></a>Não é possível coletar eventos de objeto porque o objeto já está acionando eventos para o número máximo de receptores de eventos que ele suporta
O número máximo de receptores de eventos com suporte do objeto foi excedido.  
  
### <a name="to-correct-the-error"></a>Para corrigir o erro  
  
-   Reduza o número de receptores de eventos.

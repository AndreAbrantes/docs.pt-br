# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Estrutura &#39; &lt;structurename&gt; &#39; deve conter pelo menos uma variável de membro de instância ou declaração de evento
Uma estrutura funcionalmente vazia ocorre. Pelo menos um membro de variável deve ser declarado entre a `Structure` instrução e o `End Structure` instrução.  
  
 **ID do erro:** BC30281  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Dim` instrução ou um `Event` instrução à estrutura.

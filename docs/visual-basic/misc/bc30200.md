# <a name="39new39-is-not-valid-in-this-context"></a>&#39;Novo&#39; não é válido neste contexto
O `New` palavra-chave ocorre em que é necessário um nome de tipo. O contexto pede um tipo de valor não pode ser combinado com `New`.  
  
 **ID do erro:** BC30200  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `New` palavra-chave.

# <a name="option-option-cannot-be-followed-by--or---ignored"></a>opção '\<opção >' não pode ser seguido por '+' ou '-'; ignorado
A opção de compilador foi especificada com o valor `+` ou `-`; não há suporte para esses valores.  
  
 Para obter mais informações sobre as opções do compilador e os valores suportados, consulte [opções de compilador Visual Basic listados alfabeticamente](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **ID do erro:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Para corrigir esse erro e usar a opção  
  
-   Especifique a opção de compilador com nenhum valor.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Para corrigir esse erro e não usar a opção  
  
-   Remova a opção de compilador.

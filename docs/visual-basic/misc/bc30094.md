# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Rótulo &#39; &lt;labelname&gt; &#39; já está definida no atual lambda do método/várias linhas
O nome do rótulo é duplicado dentro do procedimento. Todos os nomes dentro de um procedimento devem ser exclusivos.  
  
 **ID do erro:** BC30094  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Renomear o rótulo duplicado.

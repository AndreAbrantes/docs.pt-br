# <a name="label-ltlabelnamegt-is-already-defined-in-the-current-methodmultiline-lambda"></a>Rótulo '&lt;labelname&gt;' já está definido na lambda multilinha/método atual
O nome do rótulo está duplicado dentro do procedimento. Todos os nomes dentro de um procedimento devem ser exclusivos.  
  
 **ID do erro:** BC30094  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Renomear o rótulo duplicado.

# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Especificadores e atributos não são válidos nesta instrução
Uma declaração que não pode ser modificada, `Inherits`como, contém um ou mais atributos, ou uma ou mais palavras-chave `Public` , como `ReadOnly`ou. Somente declarações modificáveis podem conter tais palavras-chave ou atributos.  
  
 **ID do erro:** BC30193  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o atributo ou a palavra-chave da declaração.

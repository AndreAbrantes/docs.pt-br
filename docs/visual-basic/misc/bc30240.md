# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39;Saída&#39; deve ser seguido por &#39;Sub&#39;, &#39;função&#39;, &#39;propriedade&#39;, &#39;fazer&#39;, &#39;para&#39;, &#39;enquanto&#39;, &#39;selecione&#39;, ou &#39;tente&#39;
Um `Exit` instrução contém uma palavra-chave inválida. `Exit` Especifique o bloco do qual o controle a ser transferida para a instrução que segue o bloco; Por exemplo, `End While`.  
  
 **ID do erro:** BC30240  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione a seguinte palavra-chave válida `Exit`, ou remova o `Exit` instrução.

# <a name="readonly-and-writeonly-cannot-be-combined"></a>'ReadOnly' e 'WriteOnly' não podem ser combinados
O `ReadOnly` e `WriteOnly` modificadores são mutuamente exclusivos.  
  
 **ID do erro:** BC30635  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especificar `ReadOnly` ou `WriteOnly`, mas não ambos.

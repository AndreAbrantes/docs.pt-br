# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39; Somente leitura &#39; e &#39; WriteOnly &#39; não podem ser combinados
O `ReadOnly` e `WriteOnly` modificadores são mutuamente exclusivos.  
  
 **ID do erro:** BC30635  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique o `ReadOnly` ou `WriteOnly`, mas não ambos.

# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Este sistema não tem suporte para a localidade especificada
Você está tentando usar o `VbStrConv` enumeração, mas o sistema não dá suporte para a localidade especificada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade, ou execute o aplicativo em um computador que oferece suporte.

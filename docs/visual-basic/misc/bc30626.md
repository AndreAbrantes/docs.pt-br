# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>A instrução 'Namespace' deve encerrar com 'End Namespace' correspondente
`Namespace` blocos devem terminar com um `End Namespace` instrução.  
  
 **ID do erro:** BC30626  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Namespace` instrução até o final do `Namespace` bloco.

# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Valor do tipo '&lt;type1&gt;'não pode ser convertido em'&lt;type2&gt;' porque os tipos de matriz têm números diferentes de dimensões
Você tentou converter uma matriz para outro com um número diferente de dimensões.  
  
 **ID do erro:** BC30414  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar uma ou ambas as matrizes para que corresponda ao número de dimensões.

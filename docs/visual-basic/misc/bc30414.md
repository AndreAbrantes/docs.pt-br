# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Valor do tipo &#39; &lt;type1&gt;&#39; não pode ser convertido para &#39;&lt; type2&gt;&#39; porque os tipos de matriz têm números diferentes de dimensões
Você tentou converter uma matriz para outro com um número diferente de dimensões.  
  
 **ID do erro:** BC30414  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar uma ou ambas as matrizes para que coincida com o número de dimensões.

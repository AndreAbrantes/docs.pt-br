# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Valor do tipo '\<type1 >' não pode ser convertido em '\<type2 >' porque os tipos de matriz têm números diferentes de dimensões
Você tentou converter uma matriz para outro com um número diferente de dimensões.  
  
 **ID do erro:** BC30414  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar uma ou ambas as matrizes para que corresponda ao número de dimensões.

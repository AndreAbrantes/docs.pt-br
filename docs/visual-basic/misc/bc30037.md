# <a name="character-is-not-valid"></a>Caractere não é válido
O contexto atual não permite que esse caractere. Por exemplo, um til (`~`) ou barra vertical (`|`) não pode ser parte de um nome de elemento declarado.  
  
 **ID do erro:** BC30037  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se um caractere extra acidentalmente tenha sido inserido no elemento de programação.  
  
2.  Verifique se o elemento de programação está escrito ou construída usando apenas caracteres válidos para esse tipo de elemento.

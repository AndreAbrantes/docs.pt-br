## <a name="example"></a><span data-ttu-id="d4723-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d4723-109">Example</span></span>  
 <span data-ttu-id="d4723-110">A classe a seguir contém uma propriedade padrão, `defaultProp`, que requer um argumento inteiro.</span><span class="sxs-lookup"><span data-stu-id="d4723-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```


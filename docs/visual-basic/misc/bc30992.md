## <a name="example"></a><span data-ttu-id="8418f-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8418f-109">Example</span></span>  
 <span data-ttu-id="8418f-110">A seguinte classe contém uma propriedade padrão, `defaultProp`, que requer um argumento integer.</span><span class="sxs-lookup"><span data-stu-id="8418f-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="8418f-111">Como a propriedade padrão requer um argumento, a seguinte declaração causa um erro.</span><span class="sxs-lookup"><span data-stu-id="8418f-111">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```  
' Dim strs As New SomeStrings(2) With { .defaultProp = "One" }  
```

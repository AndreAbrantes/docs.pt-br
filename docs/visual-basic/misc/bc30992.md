## <a name="example"></a><span data-ttu-id="34cbf-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="34cbf-109">Example</span></span>  
 <span data-ttu-id="34cbf-110">A classe a seguir contém uma propriedade padrão, `defaultProp`, que requer um argumento integer.</span><span class="sxs-lookup"><span data-stu-id="34cbf-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```


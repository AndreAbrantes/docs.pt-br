# <a name="39end-sub39-must-be-the-first-statement-on-a-line"></a>&#39; End Sub &#39; deve ser a primeira instrução em uma linha
Um `End Sub` instrução segue um separador de declaração dois-pontos (:). `End Sub`deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC32031  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>'End Sub' deve ser a primeira instrução em uma linha
Um `End Sub` instrução segue um separador de instrução de dois-pontos (:). `End Sub` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC32031  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

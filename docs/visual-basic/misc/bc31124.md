# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>'End AddHandler' deve ser precedido por uma declaração 'AddHandler' correspondente
Uma `End AddHandler` instrução ocorreu sem um correspondente `AddHandler` instrução. `End AddHandler` deve ser precedida por uma `AddHandler` instrução.  
  
 **ID do erro:** BC31124  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se precedidos `AddHandler` instrução é válida e foi digitada corretamente.

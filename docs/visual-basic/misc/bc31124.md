# <a name="39end-addhandler39-must-be-preceded-by-a-matching-39addhandler39-declaration"></a>&#39;End AddHandler&#39; deve ser precedido por uma correspondência &#39;AddHandler&#39; declaração
Um `End AddHandler` instrução ocorreu sem correspondente `AddHandler` instrução. `End AddHandler` deve ser precedida por uma `AddHandler` instrução.  
  
 **ID do erro:** BC31124  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se precedidos `AddHandler` instrução é válida e foi digitada corretamente.

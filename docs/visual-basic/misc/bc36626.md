# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-type-lttypenamegt-is-declared-mustinherit-and-cannot-be-created"></a>Expressão lambda não pode ser convertido em '&lt;typename&gt;' porque o tipo '&lt;typename&gt;' é declarado 'como MustInherit' e não pode ser criado
Uma instrução tenta converter uma expressão lambda em um tipo que só pode ser uma classe base e não pode ser usado para criar uma instância.  
  
 **ID do erro:** BC36626

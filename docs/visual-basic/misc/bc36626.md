# <a name="lambda-expression-cannot-be-converted-to-39lttypenamegt39-because-type-39lttypenamegt39-is-declared-39mustinherit39-and-cannot-be-created"></a>Expressão lambda não pode ser convertida para &#39; &lt;typename&gt; &#39; porque o tipo &#39; &lt;typename&gt; &#39; é declarado &#39;MustInherit&#39; e não pode ser criado
Uma instrução tenta converter uma expressão lambda em um tipo que só pode ser uma classe base e não pode ser usado para criar uma instância.  
  
 **ID do erro:** BC36626

# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Expressão lambda não pode ser convertido em '\<typename >' porque o tipo '\<typename >' é declarado 'como MustInherit' e não pode ser criado
Uma instrução tenta converter uma expressão lambda em um tipo que só pode ser uma classe base e não pode ser usado para criar uma instância.  
  
 **ID do erro:** BC36626

# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>A expressão lambda não pode ser convertida em '\<TypeName > ' porque o tipo '\<TypeName > ' está declarado como ' MustInherit ' e não pode ser criado
Uma instrução tenta converter uma expressão lambda em um tipo que só pode ser uma classe base e não pode ser usada para criar uma instância.  
  
 **ID do erro:** BC36626

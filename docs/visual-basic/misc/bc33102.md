<span data-ttu-id="4d9aa-103">O modificador de tipo anulável (?) é incluído na variável em uma declaração de variável na qual os modificadores de matriz (parênteses) são incluídos no tipo de variável especificado.</span><span class="sxs-lookup"><span data-stu-id="4d9aa-103">The nullable type modifier (?) is included on the variable in a variable declaration in which the array modifiers (parentheses) are included on the specified variable type.</span></span> <span data-ttu-id="4d9aa-104">Ou, o modificador anulável é incluído em um tipo de variável especificado em uma declaração de variável na qual os modificadores de matriz são incluídos na variável.</span><span class="sxs-lookup"><span data-stu-id="4d9aa-104">Or, the nullable type modifier is included on the specified variable type in a variable declaration in which the array modifiers are included on the variable.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="4d9aa-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="4d9aa-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="4d9aa-107">Especifica o modificador de tipo anulável (?) e os modificadores de matriz (parênteses) na variável declarada ou o tipo de variável especificado, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="4d9aa-107">Specify both the nullable type modifier (?) and the array modifiers (parentheses) on either the declared variable or the specified variable type, as shown in the following example.</span></span>

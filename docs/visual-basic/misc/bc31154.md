# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>Declaração XML não permite atributo 'attributeName'
Um atributo inválido para um literal de documento XML foi especificado. Valores de atributo válidos para um literal de documento XML se baseiam na especificação XML 1.0.  
  
 **ID do erro:** BC31154  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o atributo inválido do documento XML literal.

# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>Declaração XML não permite atributo &#39;attributeName&#39;
Um atributo inválido para um literal de documento XML foi especificado. Valores de atributo válidos para um literal de documento XML baseiam-se na especificação XML 1.0.  
  
 **ID do erro:** BC31154  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o atributo inválido do literal documento XML.

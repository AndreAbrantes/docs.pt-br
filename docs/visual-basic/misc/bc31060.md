# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>'\<propertyname >' define implicitamente '\<membername >', que está em conflito com um membro do mesmo nome no \<tipo > '\<typename >'
O nome de um conflitos de membro de tipo com o nome de um membro implicitamente criado para uma propriedade. Propriedades implicitamente criam diversas variáveis implícitas que podem entrar em conflito com os membros definidos.  
  
 **ID do erro:** BC31060  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Renomeie o membro declarado explicitamente para remover o conflito de nomenclatura.

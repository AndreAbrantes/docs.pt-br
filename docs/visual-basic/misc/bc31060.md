# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;PropertyName&gt; &#39; define implicitamente &#39; &lt;membername&gt;&#39;, que está em conflito com um membro do mesmo nome em &lt;tipo&gt; &#39; &lt; TypeName&gt;&#39;
O nome de um tipo membro está em conflito com o nome de um membro implicitamente criado para uma propriedade. Propriedades implicitamente criam diversas variáveis implícitas que podem entrar em conflito com membros definidos.  
  
 **ID do erro:** BC31060  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Renomeie o membro declarado explicitamente para remover o conflito de nomes.

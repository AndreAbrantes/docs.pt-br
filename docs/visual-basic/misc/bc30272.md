# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>'&lt;argumentname&gt;'não é um parâmetro de'&lt;methoddeclaration&gt;'
Uma chamada de procedimento fornece um argumento adicional que não está definido pelo procedimento.  
  
 **ID do erro:** BC30272  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento supérfluo da lista de argumentos.

# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>'\<argumentname >' não é um parâmetro da '\<methoddeclaration >'
Uma chamada de procedimento fornece um argumento adicional que não está definido pelo procedimento.  
  
 **ID do erro:** BC30272  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento supérfluo da lista de argumentos.

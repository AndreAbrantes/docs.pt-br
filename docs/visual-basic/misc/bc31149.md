# <a name="duplicate-xml-attribute-39attributename39"></a>Duplicar o atributo XML &#39; attributeName &#39;
Um atributo especificado para um literal de elemento XML corresponde àquele de um atributo que foi previamente definido para o elemento literal XML. Nomes de atributo em literais de elemento XML devem ser exclusivos.  
  
 **ID do erro:** BC31149  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Renomeie o atributo com um nome exclusivo para o elemento literal XML.

# <a name="duplicate-xml-attribute-attributename"></a>Duplicar atributo XML 'attributeName'
Um atributo especificado para um literal de elemento XML corresponde ao de um atributo que foi previamente definido para o elemento XML literal. Nomes de atributo em literais de elemento XML devem ser exclusivos.  
  
 **ID do erro:** BC31149  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Renomeie o atributo com um nome exclusivo para o elemento XML literal.

# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39; Caso &#39; não é possível seguir uma &#39; Caso outro &#39; o mesmo &#39; Selecionar &#39; instrução
Um `Case Else` instrução introduz as instruções a serem executadas se nenhuma correspondência for encontrada para o inicial `Case`. Um `Case` instrução foi encontrada após uma `Case Else` na mesma `Select` bloco.  
  
 **ID do erro:** BC30321  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `Case Else` para o local apropriado após o `Case` instrução.

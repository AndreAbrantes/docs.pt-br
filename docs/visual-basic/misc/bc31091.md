# <a name="import-of-type-39lttypenamegt39-from-assembly-or-module-39ltnamegt39-failed"></a>Importação do tipo &#39; &lt;typename&gt; &#39; do assembly ou módulo &#39; &lt;nome&gt; &#39; falha
Certifique-se de que todas as bibliotecas de referência são compatíveis entre si.  
  
 **ID do erro:** BC31091  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se os arquivos que contém o tipo de referência não estão corrompidos.

# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;tipo&gt; não pode herdar de uma &lt;specifier2&gt; &lt;tipo&gt; porque ele expande o acesso da base de &lt;tipo&gt;
Você tentou fazer uma classe pública herdar de uma classe base marcada `Private` ou `Friend`.  
  
 **ID do erro:** BC30509  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declarar a classe base `Public` ou declara a classe herdeira `Private` ou `Friend`.

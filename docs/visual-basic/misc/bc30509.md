# <a name="specifier1-type-cannot-inherit-from-a-specifier2-type-because-it-expands-the-access-of-the-base-type"></a>\<specifier1 > \<tipo > não pode herdar de uma \<specifier2 > \<tipo > porque ele expande o acesso da base de \<tipo >
Você tentou fazer uma classe pública herdar de uma classe base marcada `Private` ou `Friend`.  
  
 **ID do erro:** BC30509  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declarar a classe base `Public` ou declara a classe herdeira `Private` ou `Friend`.

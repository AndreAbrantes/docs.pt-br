# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Está em conflito com '\<name1 > ', que é implicitamente criado para'\<nome2 >' no \<tipo > '\<nome3 >'
O nome de um tipo membro está em conflito com outro membro criado implicitamente.  
  
 **ID do erro:** BC31058  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Renomeie o membro declarado explicitamente para remover o conflito de nomenclatura.

# <a name="statement-cannot-appear-within-an-enum-body"></a>A instrução não pode aparecer dentro do corpo de Enum
Instrução não pode ocorrer dentro do corpo de Enum. Fim de enumeração assumido.  
  
 Uma construção de linguagem inesperada foi encontrada. Supõe-se que um `End Enum` construção está falta e que qualquer código-fonte depois deste ponto está fora da enumeração.  
  
 **ID do erro:** BC30619  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a sintaxe do código usado na enumeração.  
  
2.  Certifique-se de que a definição de interface termina com um `End Enum` construir.

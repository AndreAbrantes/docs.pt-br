# <a name="field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-39readonly39"></a>Campo &#39; &lt;fieldname&gt;&#39; do tipo &#39;&lt; TypeName&gt;&#39; é &#39; Somente leitura &#39;
Você tentou gravar em um campo que é declarado `ReadOnly`.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o campo para que ele não está declarado `ReadOnly`.  
  
-   —ou—  
  
-   Não tente gravar nesse campo.

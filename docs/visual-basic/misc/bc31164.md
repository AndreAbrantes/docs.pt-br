# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Aspas duplas para o valor do atributo XML de fechamento esperadas
Um valor de atributo é delimitado por aspas duplas está faltando o fechamento caractere de aspas.  
  
 **ID do erro:** BC31164  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o caractere de aspas duplas de fechamento para delimitar o valor do atributo.

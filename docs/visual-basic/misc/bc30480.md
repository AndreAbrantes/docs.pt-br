# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>Compartilhado &#39;Sub New&#39; não podem ser declarados &#39; &lt;especificador&gt;&#39;
Você tentou usar um especificador em um `Shared Sub New` declaração, que não é válida.  
  
 **ID do erro:** BC30480  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

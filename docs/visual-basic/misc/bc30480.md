# <a name="shared-sub-new-cannot-be-declared-ltspecifiergt"></a>Compartilhado 'Sub New' não pode ser declarado '&lt;especificador&gt;'
Você tentou usar um especificador em um `Shared Sub New` declaração, que não é válida.  
  
 **ID do erro:** BC30480  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

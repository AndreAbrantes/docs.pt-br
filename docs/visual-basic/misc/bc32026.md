# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Especificadores e atributos não são válidos em instruções 'Namespace'

Os namespaces são usados para a organização hierárquica de blocos de programas e elementos de programação. Eles não usam modificadores nem atributos.  
  
 **ID do erro:** BC32026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova todas as palavras-chave e os blocos de atributo da `Namespace` instrução.

# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Especificadores e atributos não são válidos em instruções 'Namespace'
Namespaces são usados para organização hierárquica de blocos de programa e elementos de programação. Eles não têm qualquer modificadores nem atributos.  
  
 **ID do erro:** BC32026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova todas as palavras-chave e blocos a partir de atributos a `Namespace` instrução.

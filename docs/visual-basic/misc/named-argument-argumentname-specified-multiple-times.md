# <a name="named-argument-argumentname-specified-multiple-times"></a>Argumento nomeado '\<argumentname >' especificado várias vezes
Você especificou um argumento nomeado mais de uma vez.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova todas, exceto uma das especificações.

# <a name="xml-comment-parameter-must-have-a-39name39-attribute"></a>Parâmetro de comentário XML deve ter um &#39;nome&#39; atributo
Parâmetro de comentário XML deve ter um atributo 'name'. O comentário XML será ignorado.  
  
 A marca XML está faltando um necessária `name` atributo.  
  
 **ID do erro:** BC42308  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça o necessário `name` atributo para a marca.

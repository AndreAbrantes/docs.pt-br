# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>'End RaiseEvent' deve ser precedido por uma declaração 'RaiseEvent' correspondente
Uma `End RaiseEvent` instrução ocorreu sem um correspondente `RaiseEvent` instrução. `End RaiseEvent` deve ser precedida por uma `RaiseEvent` instrução.  
  
 **ID do erro:** BC31126  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o anterior `RaiseEvent` instrução é válida e foi digitada corretamente.

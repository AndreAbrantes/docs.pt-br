# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39;End RaiseEvent&#39; deve ser precedido por uma correspondência &#39;RaiseEvent&#39; declaração
Um `End RaiseEvent` instrução ocorreu sem correspondente `RaiseEvent` instrução. `End RaiseEvent` deve ser precedida por uma `RaiseEvent` instrução.  
  
 **ID do erro:** BC31126  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o anterior `RaiseEvent` instrução é válida e foi digitada corretamente.

# <a name="expected-closing-39gt39-for-xml-processor-instruction"></a>Fechamento esperado &#39;? &gt;&#39; para obter instruções de processador XML
O fechamento necessário ausente do literal de instrução de processamento XML `?>` caracteres.  
  
 **ID do erro:** BC31160  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento necessários `?>` caracteres ao final do literal de instrução de processamento XML.

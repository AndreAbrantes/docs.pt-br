# <a name="expected-closing--for-xml-processor-instruction"></a>Fechamento esperado '? >' para obter instruções de processador XML
Um literal está ausente no fechamento necessário de instrução de processamento de XML `?>` caracteres.  
  
 **ID do erro:** BC31160  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicionar o fechamento necessários `?>` caracteres até o final da instrução de processamento XML literal.

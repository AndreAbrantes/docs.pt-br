# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars deve ser maior que zero
Ao usar o `PeekChars` método o `TextFieldParser` do objeto, você deve fornecer um `NumberOfChars` valor que é maior do que `0`.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar `NumberOfChars` para um valor que é maior do que `0`.

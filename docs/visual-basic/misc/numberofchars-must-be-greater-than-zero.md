# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars deve ser maior que zero
Ao usar o `PeekChars` método da `TextFieldParser` do objeto, você deve fornecer um `NumberOfChars` valor é maior que `0`.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Alteração `NumberOfChars` como um valor que é maior que `0`.

# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Somente um dos &#39;pública&#39;, &#39;privada&#39;, &#39;protegidos&#39;, &#39;Friend&#39;, ou &#39;Protected Friend&#39; pode ser especificado
Acessibilidade conflitante ocorre em uma declaração de elemento. Acessibilidade apenas uma pode ser especificada.  
  
 **ID do erro:** BC30176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Escolha uma acessibilidade e remova outras palavras-chave da declaração.

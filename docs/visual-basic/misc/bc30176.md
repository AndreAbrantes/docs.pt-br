# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Somente um dos &#39; Público &#39; &#39; Privada &#39; &#39; Protegido &#39; &#39; Friend &#39; ou &#39; Friend protegido &#39; pode ser especificado
Acessibilidade conflitante ocorre em uma declaração de elemento. Acessibilidade apenas uma pode ser especificada.  
  
 **ID do erro:** BC30176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Escolha uma acessibilidade e remova outras palavras-chave da declaração.

# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Somente um dos 'Public', 'Private', 'Protected', 'Friend' ou 'Protected Friend' pode ser especificado
Acessibilidade conflitante ocorre em uma declaração de elemento. Acessibilidade apenas uma pode ser especificada.  
  
 **ID do erro:** BC30176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Escolha uma acessibilidade e remova as outras palavras-chave da declaração.

# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>Apenas um dos 'Public', 'Private', 'Protected', 'Friend' ou 'Protected Friend' pode ser especificado.
Acessibilidades conflitantes ocorrem em uma declaração de elemento. Acessibilidade apenas uma pode ser especificada.  
  
 **ID do erro:** BC30176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Escolha uma acessibilidade e remova as outras palavras-chave da declaração.

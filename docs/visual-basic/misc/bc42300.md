# <a name="xml-comment-block-must-immediately-precede-the-language-element-to-which-it-applies"></a>Bloco de comentário XML deve preceder imediatamente o elemento de linguagem ao qual se aplica
Bloco de comentário XML deve preceder imediatamente o elemento de linguagem ao qual se aplica. O comentário XML será ignorado.  
  
 Esse erro também ocorrerá se uma marca está no local errado ou caso contrário malformado.  
  
 **ID do erro:** BC42300  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Mova o bloco de comentário deve preceder o elemento de linguagem ao qual se aplica. Certifique-se de que caracteres estranhos não foram inseridos acidentalmente antes da marca inicial.  
  
2.  Verifique se que a marca é válida.

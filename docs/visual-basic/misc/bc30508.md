# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39;&lt;nome&gt; &#39; não pode expor o tipo &lt;type1&gt; na &lt;especificador&gt; &lt;type2&gt; &#39; &lt;typename&gt; &#39; por meio de &lt;nome&gt;
Você tentou expor um tipo privado fora da classe pública `MyClass`.  
  
 **ID do erro:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public`.

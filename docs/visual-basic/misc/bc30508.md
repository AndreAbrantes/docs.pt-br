# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<nome >' não pode expor o tipo \<type1 > na \<especificador > \<type2 > '\<typename >' por meio de \<nome >
Você tentou expor um tipo particular fora da classe pública `MyClass`.  
  
 **ID do erro:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declare o tipo `Public`.

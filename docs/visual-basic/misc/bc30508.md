# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>'&lt;nome&gt;' não pode expor o tipo &lt;type1&gt; na &lt;especificador&gt; &lt;type2&gt; '&lt;typename&gt;' por meio de &lt;nome&gt;
Você tentou expor um tipo particular fora da classe pública `MyClass`.  
  
 **ID do erro:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public`.

# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Saída &#39; instruções não são válidas na janela imediata
`Exit`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30713  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Exit` instruções do código de depuração.

# <a name="too-many-type-arguments-to-ltgenericmethodnamegt"></a>Muitos argumentos de tipo para '&lt;genericMethodName&gt;'
Um método genérico terá sido chamado com mais argumentos de tipo de parâmetros de tipo.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC32043  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova os argumentos de tipo da sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo no método genérico que você está invocando.

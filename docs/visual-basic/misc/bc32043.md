# <a name="too-many-type-arguments-to-39ltgenericmethodnamegt39"></a>Muitos argumentos de tipo para &#39; &lt;genericMethodName&gt;&#39;
Um método genérico ter sido chamado com argumentos de tipo mais do que o número de parâmetros de tipo.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC32043  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova argumentos de tipo a sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo no método genérico que está chamando.

# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39;Padrão&#39; não pode ser combinado com &#39; &lt;especificador&gt;&#39;
Você tentou combinar o `Default` palavra-chave com um especificador como `Shared` ou `Private`, que é não é válido neste contexto.  
  
 **ID do erro:** BC30490  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova `Default`.  
  
     -ou-  
  
-   Remova o especificador conflitante.

# <a name="default-cannot-be-combined-with-specifier"></a>'Default' não pode ser combinado com '\<especificador >'
Você tentou combinar o `Default` palavra-chave com um especificador como `Shared` ou `Private`, que é não é válido neste contexto.  
  
 **ID do erro:** BC30490  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Default`.  
  
     - ou -  
  
- Remova o especificador conflitante.

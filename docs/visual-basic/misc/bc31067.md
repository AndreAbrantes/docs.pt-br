# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Método em uma estrutura não pode ser declarado como 'Protected' ou 'Protected Friend'
Um método em uma estrutura foi declarado como `Protected` ou `Protected Friend`.  
  
 **ID do erro:** BC31067  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare métodos dentro de estruturas como `Public` ou `Private`.

# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Método em uma estrutura não pode ser declarado como 'Protected' ou 'Protected Friend'
Um método em uma estrutura foi declarado como `Protected` ou `Protected Friend`.  
  
 **ID do erro:** BC31067  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare métodos dentro de estruturas conforme `Public` ou `Private`.

# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>Não é possível converter o argumento '&lt;argumentname&gt;'do tipo'&lt;type1&gt;' no tipo'&lt;type2&gt;'
Uma operação de conversão tenta converter um argumento para um tipo incompatível.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que o tipo ao qual você está tentando converter o argumento é válido.

# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Propriedade&gt; parâmetros não podem ser declarados &#39;ByRef&#39;
Parâmetros para propriedades devem ser declarados usando o `ByVal` modificador.  
  
 **ID do erro:** BC30651  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Definir parâmetros de propriedade como `ByVal`.

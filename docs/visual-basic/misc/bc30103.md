# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-lttypegt"></a>'!' requer sua esquerda operando tenha um parâmetro de tipo, classe ou tipo de interface, mas este operando tem o tipo '&lt;tipo&gt;'
Acesso de membro de dicionário, que usa um ponto de exclamação (`!`) em vez de um período (`.`), só é válida em uma classe ou uma interface.  
  
 **ID do erro:** BC30103  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua a expressão à esquerda do ponto de exclamação por um que é avaliada em um tipo de interface ou classe definida.

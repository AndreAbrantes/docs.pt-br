<span data-ttu-id="1f2a7-103">Um procedimento ou propriedade é declarada como `MustOverride` dentro de uma classe que é definida em várias declarações parciais, mas uma das definições parciais especifica `NotInheritable` para a classe.</span><span class="sxs-lookup"><span data-stu-id="1f2a7-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="1f2a7-104">Quando você divide a definição de uma classe entre várias declarações parciais, o compilador trata a classe como a união de todas as suas declarações parciais.</span><span class="sxs-lookup"><span data-stu-id="1f2a7-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="1f2a7-105">Isso se aplica não apenas aos membros, mas também para a implementação, herança e nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="1f2a7-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>

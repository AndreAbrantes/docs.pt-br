<span data-ttu-id="edfd0-103">Um procedimento ou propriedade é declarada como `MustOverride` dentro de uma classe que é definida em várias declarações parciais, mas uma das definições parciais especifica `NotInheritable` para a classe.</span><span class="sxs-lookup"><span data-stu-id="edfd0-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="edfd0-104">Quando você divide a definição de uma classe entre várias declarações parciais, o compilador trata a classe como a união de todas as suas declarações parciais.</span><span class="sxs-lookup"><span data-stu-id="edfd0-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="edfd0-105">Isso se aplica não apenas aos membros mas também para a implementação, herança e nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="edfd0-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="edfd0-106">Para substituir um procedimento ou uma propriedade, uma classe deve herdá-la de uma classe base.</span><span class="sxs-lookup"><span data-stu-id="edfd0-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="edfd0-107">Portanto, para especificar `MustOverride` para um procedimento ou propriedade em uma classe base, você deve especificar `MustInherit` para a classe.</span><span class="sxs-lookup"><span data-stu-id="edfd0-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="edfd0-108">Como eles são mutuamente contraditórios, não é possível especificar `MustInherit` e `NotInheritable` para a mesma classe.</span><span class="sxs-lookup"><span data-stu-id="edfd0-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>

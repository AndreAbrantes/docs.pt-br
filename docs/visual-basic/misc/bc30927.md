# <a name="mustoverride-cannot-be-specified-on-ltprocedurenamegt-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="be906-102">'MustOverride' não pode ser especificado em '&lt;procedurename&gt;' porque ele está em um tipo parcial que é declarado como 'NotInheritable' em outra definição parcial</span><span class="sxs-lookup"><span data-stu-id="be906-102">'MustOverride' cannot be specified on '&lt;procedurename&gt;' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>
<span data-ttu-id="be906-103">Um procedimento ou propriedade é declarada como `MustOverride` dentro de uma classe que é definida em várias declarações parciais, mas uma das definições parciais especifica `NotInheritable` para a classe.</span><span class="sxs-lookup"><span data-stu-id="be906-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="be906-104">Quando você divide a definição de uma classe entre várias declarações parciais, o compilador trata a classe como a união de todas as suas declarações parciais.</span><span class="sxs-lookup"><span data-stu-id="be906-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="be906-105">Isso se aplica não apenas aos membros, mas também para a implementação, a herança e o nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="be906-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="be906-106">Para substituir um procedimento ou propriedade, uma classe deve herdá-la de uma classe base.</span><span class="sxs-lookup"><span data-stu-id="be906-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="be906-107">Portanto, para especificar `MustOverride` para um procedimento ou propriedade em uma classe base, você deve especificar `MustInherit` para a classe.</span><span class="sxs-lookup"><span data-stu-id="be906-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="be906-108">Como eles são mutuamente contraditórios, não é possível especificar ambos `MustInherit` e `NotInheritable` para a mesma classe.</span><span class="sxs-lookup"><span data-stu-id="be906-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="be906-109">**ID do erro:** BC30927</span><span class="sxs-lookup"><span data-stu-id="be906-109">**Error ID:** BC30927</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="be906-110">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="be906-110">To correct this error</span></span>  
  
-   <span data-ttu-id="be906-111">Se a propriedade ou procedimento deve ser substituído, em seguida, remova o `NotInheritable` palavra-chave da declaração parcial na qual ele aparece.</span><span class="sxs-lookup"><span data-stu-id="be906-111">If the property or procedure must be overridden, then remove the `NotInheritable` keyword from the partial declaration in which it appears.</span></span>

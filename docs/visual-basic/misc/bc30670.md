# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39; ReDim &#39; instruções exigem uma lista delimitada por parênteses dos novos limites de cada dimensão da matriz
Você deve especificar o novo tamanho de uma matriz como parte de um `ReDim` instrução.  
  
 **ID do erro:** BC30670  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça o tamanho de cada classificação de matriz; Por exemplo:  
  
    ```  
    ReDim arr(5, 6)  
    ```

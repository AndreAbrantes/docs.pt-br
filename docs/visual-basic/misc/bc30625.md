# <a name="module-statement-must-end-with-a-matching-end-module"></a>A instrução 'Module' deve encerrar com 'End Module' correspondente
Blocos de módulo devem terminar com um `End Module` instrução.  
  
 **ID do erro:** BC30625  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Module` instrução até o final do `Module` bloco.

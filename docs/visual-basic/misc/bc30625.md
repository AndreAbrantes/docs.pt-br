# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Módulo&#39; instrução deve terminar com uma correspondência &#39;End Module&#39;
Blocos de módulo devem terminar com um `End Module` instrução.  
  
 **ID do erro:** BC30625  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Module` instrução até o final do `Module` bloco.

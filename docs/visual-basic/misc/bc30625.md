# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>Instrução 'Module' deve finalizar com 'End Module' correspondente
Blocos de módulo devem terminar com um `End``Module` instrução.  
  
 **ID do erro:** BC30625  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar uma `End Module` instrução ao final do `Module` bloco.

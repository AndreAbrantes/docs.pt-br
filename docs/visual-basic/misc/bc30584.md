# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>'&lt;tipo&gt;' não podem ser herdadas de mais de uma vez
Ter duplicado um `Inherits` instrução.  
  
 **ID do erro:** BC30584  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `Inherits` instrução.

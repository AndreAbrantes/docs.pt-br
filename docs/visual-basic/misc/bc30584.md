# <a name="type-cannot-be-inherited-more-than-once"></a>'\<tipo >' não pode ser herdado mais de uma vez
Você duplicou um `Inherits` instrução.  
  
 **ID do erro:** BC30584  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `Inherits` instrução.

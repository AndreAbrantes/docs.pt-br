# <a name="39catch39-must-end-with-a-matching-39end-try39"></a>&#39; Catch &#39; deve terminar com uma correspondência &#39; End Try &#39;
Um `Catch` instrução aparece no seu código sem uma `End Try` instrução. `Catch`instruções devem ser concluídas com um `End Try` instrução.  
  
 **ID do erro:** BC30441  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Catch` instrução.  
  
2.  Adicionar um `End Try` instrução para concluir o bloco.

# <a name="3939-expected-declaration"></a>'=' esperado (declaração)
A `Mid` instrução ou um `#Const` diretiva de compilação não incluem uma atribuição de um valor.  
  
 **ID do erro:** BC30249  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um sinal de igual (`=`) e um valor a ser atribuído para a cadeia de caracteres ou a constante de compilação.

# <a name="type1-typename-cannot-be-declared-overrides-because-it-does-not-override-a-type1-in-a-base-type2"></a>\<type1 > '\<TypeName > ' não pode ser declarada ' Overrides ' porque \<não substitui uma > type1 \<em um type2 base >
Uma `Sub`instrução `Function`,, `Property` ou especifica`Overrides` quando nenhum tipo com o mesmo nome existe em uma classe base.  
  
 **ID do erro:** BC30284  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se o nome do tipo está grafado corretamente.  
  
2. Remova a palavra `Overrides` -chave supérflua.

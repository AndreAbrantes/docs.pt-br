# <a name="format-not-valid-in-resource-file"></a>Formato inválido no arquivo de recurso
Um arquivo de recurso em seu projeto não segue o formato padrão para um arquivo de recurso do Windows ou há uma incompatibilidade entre o arquivo de recurso e o [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] projeto.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Recriar a fonte de recursos e arquivos de recursos e de acordo com a sintaxe documentada no Software Development Kit do Windows.  
  
2.  Remova o arquivo de recurso inválido do projeto.

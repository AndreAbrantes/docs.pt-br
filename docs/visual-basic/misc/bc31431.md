<span data-ttu-id="42b98-103">O modificador de acesso `Private` é necessário nas declarações de método parcial.</span><span class="sxs-lookup"><span data-stu-id="42b98-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="42b98-104">O exemplo a seguir mostra o uso de `Private` na assinatura do método e sua implementação.</span><span class="sxs-lookup"><span data-stu-id="42b98-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb

    ' The body of the signature is empty.  
End Sub  
```  
  
```vb

End Sub
```  

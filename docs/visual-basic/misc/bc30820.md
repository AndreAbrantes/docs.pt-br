# <a name="39ltlsetstmtgt39-is-not-declared"></a>'&lt;lsetstmt&gt;' não está declarado
'\<lsetstmt >' não está declarado. Não há mais suporte a instruções LSet; Use Microsoft.VisualBasic.LSet ao invés.  
  
 Várias funções que foram intrínsecas a [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] em versões anteriores foram movidas para o <xref:Microsoft.VisualBasic?displayProperty=fullName>namespace.</xref:Microsoft.VisualBasic?displayProperty=fullName> Isso torna a funcionalidade mais disponível para todas as linguagens de programação.  
  
 **ID do erro:** BC30820  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `LSet` funcionar a `Microsoft.VisualBasic` namespace em vez disso.

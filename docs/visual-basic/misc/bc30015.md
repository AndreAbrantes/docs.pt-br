# <a name="inheriting-from-39systemltclassnamegt39-is-not-valid"></a>Herdando &#39; Sistema. &lt;classname&gt;&#39; não é válido
Uma classe não pode herdar de `System.Array`, `System.Delegate`, `System.Enum`, ou `System.ValueType`.  
  
 **ID do erro:** BC30015  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `Inherits` instrução ou alterá-la para herdar de uma classe base válida.

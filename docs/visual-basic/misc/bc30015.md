# <a name="inheriting-from-systemltclassnamegt-is-not-valid"></a>Herdar de ' System. &lt;classname&gt;' não é válido
Uma classe não pode herdar de `System.Array`, `System.Delegate`, `System.Enum`, ou `System.ValueType`.  
  
 **ID do erro:** BC30015  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `Inherits` instrução ou alterá-lo para herdar de uma classe base válida.

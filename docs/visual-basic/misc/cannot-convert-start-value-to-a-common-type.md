# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>Não é possível converter o valor inicial do tipo &#39; &lt;type1&gt;&#39;, valor de limite do &#39; &lt;type2&gt; &#39; e o valor de step do &#39; &lt;type3&gt; &#39; para um tipo comum
Especificado `For...Next` valores de controle têm tipos diferentes e, portanto, não são válidos porque eles não podem ser convertidos em um tipo comum.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Altere o tipo de um ou mais dos valores, portanto, tudo pode ser convertido em um tipo comum.

# <a name="cannot-convert-start-value-of-type-lttype1gt-limit-value-of-lttype2gt-and-step-value-of-lttype3gt-to-a-common-type"></a>Não é possível converter o valor inicial do tipo '&lt;type1&gt;', valor limite de '&lt;type2&gt;'e valor de step de'&lt;type3&gt;' em um tipo comum
Especificado `For...Next` valores de controle têm tipos diferentes e, portanto, não são válidos porque eles não podem ser convertidos em um tipo comum.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Altere o tipo de um ou mais dos valores, portanto, tudo pode ser convertido em um tipo comum.

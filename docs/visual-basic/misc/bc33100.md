# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Não é possível especificar o modificador anulável em uma variável e seu tipo
O modificador de tipo anulável (?) foi incluído para uma variável declarada e seu tipo declarado.  
  
 **ID do erro:** BC33100  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o modificador de tipo anulável (?) ou da variável ou seu tipo declarado.

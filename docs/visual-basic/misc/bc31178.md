# <a name="expected-closing--for-xml-entity"></a>Fechamento esperado ';' para a entidade XML
Uma referência de entidade XML (por exemplo, `>`) está faltando um ponto e vírgula de fechamento (;).  
  
 **ID do erro:** BC31178  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Inclua o ponto e vírgula no final da referência de entidade XML.

# <a name="expected-closing-3939-for-xml-entity"></a>Esperado o fechamento &#39; &#39; entidade XML
Uma referência de entidade XML (por exemplo, `>`) está faltando um fechamento-e-vírgula (;).  
  
 **ID do erro:** BC31178  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Inclua o ponto e vírgula no final da referência de entidade do XML.

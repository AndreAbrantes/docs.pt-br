<span data-ttu-id="b11c2-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças podem entrar em conflito para determinados valores dos argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="b11c2-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="b11c2-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="b11c2-104">The following statements can generate this error.</span></span>

```vb
Public Interface interfaceA(Of u)
End Interface
Public Interface interfaceX(Of v)
    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)
End Interface
Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="b11c2-105">Se `derivedInterface` é construído ou implementado fornecendo o mesmo tipo para ambos `t1` e `t2`, ele deve herdar duas versões do `interfaceA` com argumentos de tipo idênticos.</span><span class="sxs-lookup"><span data-stu-id="b11c2-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="b11c2-106">Isso produziria uma ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="b11c2-106">Doing so would produce an ambiguity about which version to access.</span></span>

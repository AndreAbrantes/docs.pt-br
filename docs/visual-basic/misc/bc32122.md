# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-from-which-the-interface-interfacename4-inherits-for-some-type-arguments"></a><span data-ttu-id="6d44b-102">Não é possível herdar a interface '\<interfacename1 >' porque a interface '\<interfacename2 >' da qual ela herda poderia ser idêntica à interface '\<interfacename3 >' da qual a interface '\< interfacename4 >' herda para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="6d44b-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' from which the interface '\<interfacename4>' inherits for some type arguments</span></span>
<span data-ttu-id="6d44b-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças poderia entrar em conflito para determinados valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="6d44b-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="6d44b-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="6d44b-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface interfaceY(Of w)  
    Inherits interfaceA(Of w)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceX(Of t1), interfaceY(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="6d44b-105">Se `derivedInterface` é construída ou implementada fornecendo o mesmo tipo para ambos `t1` e `t2`, ela deve herdar duas versões do `interfaceA` com argumentos de tipo idêntico.</span><span class="sxs-lookup"><span data-stu-id="6d44b-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="6d44b-106">Isso produziria uma ambiguidade sobre a versão que deseja acessar.</span><span class="sxs-lookup"><span data-stu-id="6d44b-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="6d44b-107">**ID do erro:** BC32122</span><span class="sxs-lookup"><span data-stu-id="6d44b-107">**Error ID:** BC32122</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6d44b-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="6d44b-108">To correct this error</span></span>  
  
- <span data-ttu-id="6d44b-109">Altere um dos argumentos de tipo fornecidos para a interface derivada de modo que haja conflito.</span><span class="sxs-lookup"><span data-stu-id="6d44b-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="6d44b-110">- ou -</span><span class="sxs-lookup"><span data-stu-id="6d44b-110">-or-</span></span>

 <span data-ttu-id="0ede2-104">Em tempo de compilação, o Visual Basic considera uma conversão predefinida existente de qualquer tipo de referência para qualquer tipo na sua hierarquia de herança, ou seja, qualquer tipo do qual ele deriva ou que deriva.</span><span class="sxs-lookup"><span data-stu-id="0ede2-104">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="0ede2-105">Tal conversão pode falhar em tempo de execução, mas o compilador não pode prever resultados de tempo de execução, que permite que qualquer conversão desse tipo compilar.</span><span class="sxs-lookup"><span data-stu-id="0ede2-105">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>

# <a name="option-statements-must-precede-any-declarations-or-imports-statements"></a>Instruções 'Option' devem preceder quaisquer declarações ou instruções 'Imports'
`Option` instruções devem ser a primeira sem comentário no seu código-fonte.  
  
 **ID do erro:** BC30627  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Mover o `Option` instruções na parte superior da fonte de código, imediatamente antes `Imports` e `Namespace` instruções.

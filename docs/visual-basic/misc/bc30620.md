# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>'Option Strict' pode ser seguido apenas por 'On' ou 'Off'
`Option Strict` instruções devem especificar `On` ou `Off`. Todos os outros valores são inválidos.  
  
 **ID do erro:** BC30620  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especificar `On` ou `Off` para o `Option Strict` instrução.

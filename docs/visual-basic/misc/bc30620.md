# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Opção Strict&#39; pode ser seguida somente por &#39;em&#39; ou &#39;Off&#39;
`Option Strict` as instruções devem especificar um `On` ou `Off`. Todos os outros valores são inválidos.  
  
 **ID do erro:** BC30620  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique o `On` ou `Off` para o `Option Strict` instrução.

# <a name="element-is-missing-an-end-tag"></a>Elemento não tem uma marca de fim
Um literal de elemento XML não tem uma marca de fim XML ou um identificador de marca vazia '/' (por exemplo, `<tagName />`).  
  
 **ID do erro:** BC31151  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar a marca de fim de marca XML, ou adicionar um identificador de marca vazia ' /' para o marcador XML.

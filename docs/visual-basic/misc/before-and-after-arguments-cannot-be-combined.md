# <a name="39before39-and-39after39-arguments-cannot-be-combined"></a>&#39; Antes de &#39; e &#39; Depois de &#39; argumentos não podem ser combinados
Uma lista de argumentos inclui tanto o `Before` e `After` argumentos que são mutuamente exclusivos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Before` ou `After` argumento.

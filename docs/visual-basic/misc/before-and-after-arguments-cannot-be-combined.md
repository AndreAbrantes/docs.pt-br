# <a name="before-and-after-arguments-cannot-be-combined"></a>'Before' e 'After' argumentos não podem ser combinados
Uma lista de argumentos inclui tanto o `Before` e `After` argumentos, que são mutuamente exclusivos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Remover o `Before` ou `After` argumento.

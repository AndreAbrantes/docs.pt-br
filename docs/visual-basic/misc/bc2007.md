# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>Opção não reconhecida &lt;optionname&gt;; ignorado
Foi especificada uma opção que não é válido na linha de comando.  
  
 **ID do erro:** BC2007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verificar opções especificadas para encontrar o que não é válido.

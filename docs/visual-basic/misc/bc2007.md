# <a name="unrecognized-option-optionname-ignored"></a>Opção não reconhecida \<optionname >; ignorado
Foi especificada uma opção que não é válido a partir da linha de comando.  
  
 **ID do erro:** BC2007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique as opções especificadas para encontrar aquele que não é válido.

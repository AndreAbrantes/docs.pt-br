# <a name="cannot-calculate-rate-using-the-arguments-provided"></a>Não é possível calcular a taxa usando os argumentos fornecidos
O cálculo de taxa não contém os argumentos necessários.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a ortografia dos argumentos na expressão. Um nome de variável incorreta implicitamente pode criar uma variável numérica que é inicializada a zero.  
  
2.  Verifique se as operações anteriores em variáveis na expressão, especialmente aquelas de outros procedimentos passadas para o procedimento como argumentos.

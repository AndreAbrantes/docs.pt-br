# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>Arquivo &#39; &lt;filename&gt;&#39; não pode ser excluído porque ele está aberto
Uma exclusão está sendo tentada em um arquivo aberto.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Feche o arquivo antes de excluí-lo.

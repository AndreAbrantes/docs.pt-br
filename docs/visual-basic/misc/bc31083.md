# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Métodos em um módulo não podem implementar membros de interface
Membros de interface devem ser implementados por classes ou estruturas.  
  
 **ID do erro:** BC31083  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Implemente membros de interface de classes ou estruturas.

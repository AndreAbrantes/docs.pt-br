# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argumento &#39; Acessar &#39; não é válido. Valores válidos para acrescentar modo são #39;OpenAccess.Write &#39; e &#39;OpenAccess.Default &#39;
Um valor de acesso não é válido no modo de acréscimo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Altere o valor para `OpenAccess.Write` ou `OpenAccess.Default`.

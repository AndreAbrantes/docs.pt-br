# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argumento &#39;acesso&#39; não é válido. Os valores válidos para o modo Append são &#39;OpenAcess&#39; e &#39;OpenAccess&#39;
Um valor de acesso não é válido no modo de acréscimo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Altere o valor para `OpenAccess.Write` ou `OpenAccess.Default`.

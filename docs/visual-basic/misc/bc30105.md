# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>O número de índices é menor que o número de dimensões da matriz indexada
O número de índices usados para acessar um elemento de matriz deve ser exatamente o mesmo que a classificação da matriz, ou seja, o número de dimensões declarado para ele.  
  
 **ID do erro:** BC30105  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione subscritos para a referência de matriz até que o número total de subscritos é igual a classificação da matriz.

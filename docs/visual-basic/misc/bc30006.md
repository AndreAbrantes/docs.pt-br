# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Referência necessária para o módulo &#39; &lt;modulename&gt;&#39; contendo a definição do evento &#39;&lt; EventName&gt;&#39;
Referência necessária para o módulo ' <`modulename`>' que contém a definição para o evento ' <`eventname`>'. Adicione uma ao seu projeto.  
  
 O evento é definido em um módulo que não é referenciado diretamente em seu projeto. O [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compilador requer uma referência para evitar ambiguidade no caso do evento é definido em mais de um módulo.  
  
 **ID do erro:** BC30006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Inclua o nome do módulo sem referência em suas referências do projeto.

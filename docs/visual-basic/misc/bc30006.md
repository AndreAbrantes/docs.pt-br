# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>Referência obrigatória ao módulo '\<modulename >' que contém a definição para o evento '\<eventname >'
Referência obrigatória ao módulo ' <`modulename`>' que contém a definição para o evento ' <`eventname`>'. Adicione uma ao seu projeto.  
  
 O evento é definido em um módulo que não é referenciado diretamente em seu projeto. O compilador do Visual Basic exige uma referência para evitar ambiguidade no caso do evento é definido em mais de um módulo.  
  
 **ID do erro:** BC30006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Inclua o nome do módulo sem referência em suas referências do projeto.

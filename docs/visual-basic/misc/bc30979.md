# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Opção Infer&#39; pode ser seguida somente por &#39;em&#39; ou &#39;Off&#39;
`Option Infer` as instruções devem especificar um `On` ou `Off`. Outros valores não são válidos.  
  
 **ID do erro:** BC30979  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique `Option Infer On` ou `Option Infer Off`.

# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Este sistema não contém suporte para a localidade do chinês tradicional
Você tentou usar o membro de enumeração `VbStrConv.TraditionalChinese`, mas seu sistema não oferece suporte à localidade chinês tradicional.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Atualize o computador para dar suporte à localidade do chinês tradicional ou execute o aplicativo em um computador que ofereça suporte à localidade.

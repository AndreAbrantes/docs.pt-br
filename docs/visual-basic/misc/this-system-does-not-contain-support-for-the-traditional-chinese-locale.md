# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Este sistema não há suporte para localidade chinês tradicional
Você tentou usar o `VbStrConv.TraditionalChinese` membro de enumeração, mas seu sistema não dá suporte para localidade chinês tradicional.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Atualize a máquina para dar suporte à localidade chinês tradicional, ou executar o aplicativo em um computador que dá suporte à localidade.

# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Este sistema não tem suporte para localidade chinês tradicional
Você tentou usar o `VbStrConv.TraditionalChinese` membro de enumeração, mas o sistema não dá suporte para localidade chinês tradicional.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade de chinês tradicional, ou execute o aplicativo em uma máquina que dá suporte a localidade.

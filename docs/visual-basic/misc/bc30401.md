# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39;&lt;interfacename1&gt; &#39; não é possível implementar &#39; &lt;methodname&gt; &#39; porque não há nenhuma correspondência &lt;método&gt; na interface &#39; &lt; interfacename2&gt;&#39;
Você tentou implementar um método em uma interface quando não há nenhum método correspondente na interface do qual a primeira interface tem sido derivada.  
  
 **ID do erro:** BC30401  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que você não ter digitado o nome do método, ou fornecer o método ausente.

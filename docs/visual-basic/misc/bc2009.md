# <a name="option-ltoptionnamegt-can-be-followed-only-by--or--"></a>opção &lt;optionname&gt; pode ser seguida somente por '+' ou '-'
O texto que segue a opção não é um sinal de adição (+) ou um sinal de subtração (-).  
  
 **ID do erro:** BC2009  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o texto que segue a opção com um sinal de adição (+) para ativar a opção.  
  
     – ou —  
  
-   Substitua o texto que segue a opção com um sinal de subtração (-) para desativar a opção.

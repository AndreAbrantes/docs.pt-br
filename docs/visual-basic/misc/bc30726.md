# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>'Instruções e contextos with' não são válidas em janelas de depuração
`With` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30726  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `With` instruções do código de depuração.

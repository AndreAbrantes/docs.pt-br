# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;Com&#39; instruções e contextos não são válidas nas janelas de depuração
`With` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30726  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `With` instruções do código de depuração.

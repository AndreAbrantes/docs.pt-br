# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39;Por fim&#39; só pode aparecer uma vez em um &#39;tente&#39; instrução
`Finally` é usado para concluir um `Try...Catch...Finally` bloco; portanto ele pode aparecer apenas uma vez no final do bloco.  
  
 **ID do erro:** BC30381  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Localize e remova a `Finally`.

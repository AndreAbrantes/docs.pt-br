# <a name="finally-can-only-appear-once-in-a-try-statement"></a>'Finally' só pode aparecer uma vez em uma instrução 'Try'
`Finally` é usado para concluir uma `Try...Catch...Finally` block; portanto ele pode aparecer apenas uma vez no final do bloco.  
  
 **ID do erro:** BC30381  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Localize e remova a `Finally`.

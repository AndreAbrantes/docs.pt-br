# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Variável de controle de loop for '\<variablename >' já está em uso por um loop For
Aninhado `For` loops todos devem usar variáveis de controle exclusiva.  
  
 **ID do erro:** BC30069  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere um dos nomes de variável de controle.

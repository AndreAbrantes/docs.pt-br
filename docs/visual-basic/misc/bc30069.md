# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>Variável de controle de loop for '&lt;variablename&gt;' já está em uso por um loop For
Aninhado `For` loops todos devem usar variáveis de controle exclusiva.  
  
 **ID do erro:** BC30069  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere um dos nomes de variável de controle.

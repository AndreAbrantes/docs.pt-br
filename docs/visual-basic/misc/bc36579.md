# <a name="too-many-type-arguments"></a>Muitos argumentos de tipo
Um método genérico terá sido chamado com mais argumentos de tipo de parâmetros de tipo.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC36579  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova os argumentos de tipo da sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo do método genérico que você está invocando.

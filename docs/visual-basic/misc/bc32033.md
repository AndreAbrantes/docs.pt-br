# <a name="end-get-must-be-the-first-statement-on-a-line"></a>'End Get' deve ser a primeira instrução em uma linha
Um `End Get` instrução segue um separador de instrução de dois-pontos (:). `End Get` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC32033  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Separe várias declarações em linhas diferentes.

# <a name="operator-ltoperatornamegt-is-not-defined-for-types-lttype1gt-and-lttype2gt"></a>Operador '&lt;operatorname&gt;'não está definido para tipos'&lt;type1&gt;'e'&lt;type2&gt;'
Você tentou usar um operador binário para executar uma operação que não é válida para os dois retornando o valor elementos de código envolvidos.  
  
 **ID do erro:** BC30452  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique os dois elementos e certifique-se de que eles são compatíveis.

# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>Operador &#39; &lt;operatorname&gt;&#39; não está definido para tipos de &#39;&lt; type1&gt;&#39; e &#39;&lt; type2&gt;&#39;
Você tentou usar um operador binário para executar uma operação que não é válida para os dois retornando valor código elementos envolvidos.  
  
 **ID do erro:** BC30452  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique os dois elementos e certifique-se de que eles são compatíveis.

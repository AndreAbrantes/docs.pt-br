# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>Propriedade '\<propertyname >' deve ser declarado como 'Default' porque ela substitui uma propriedade padrão na base de '\<nome >'
Um `Property` instrução em seu código substitui um `Default` propriedade na classe base, estrutura, ou de interface e, portanto, precisa ser declarada `Default`.  
  
 **ID do erro:** BC30360  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declarar a propriedade `Default`.

# <a name="comma--or-a-valid-expression-continuation-expected"></a>Vírgula, ')', ou uma continuação de expressão válida esperada
Uma lista, como uma lista de argumentos em uma definição de procedimento ou uma chamada de procedimento, não separe ou encerrar seus membros. Uma causa possível é dois membros da lista que está sendo separados apenas por um ou mais espaços.  
  
 **ID do erro:** BC32017  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use uma vírgula (,) para separar os membros da lista e um parênteses de fechamento ()) para finalizar a lista.

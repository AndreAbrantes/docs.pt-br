# <a name="39ltmembergt39-is-already-declared-in-this-structure"></a>&#39;&lt;membro&gt; &#39; já está declarado nesta estrutura
Um membro de enumeração foi declarado mais de uma vez nessa estrutura.  
  
 **ID do erro:** BC31421  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a declaração irrelevante.

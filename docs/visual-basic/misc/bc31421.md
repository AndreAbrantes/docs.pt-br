# <a name="ltmembergt-is-already-declared-in-this-structure"></a>'&lt;membro&gt;' já está declarado nesta estrutura
Um membro de enumeração foi declarado mais de uma vez nessa estrutura.  
  
 **ID do erro:** BC31421  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a declaração de estranhas.

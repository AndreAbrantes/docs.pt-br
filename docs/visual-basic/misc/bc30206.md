# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>'Option' deve ser seguido por 'Compare', 'Explicit', 'Infer' ou 'Strict'
As únicas opções que podem ser definidas com o `Option` instrução estão `Compare`, `Explicit`, `Infer`, e `Strict`.  
  
 **ID do erro:** BC30206  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se uma palavra-chave option (`Compare`, `Explicit`, `Infer`, ou `Strict`) está incorreto.  
  
- Verifique se o `Option` instrução está incompleta.

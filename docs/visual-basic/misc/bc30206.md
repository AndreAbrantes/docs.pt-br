# <a name="39option39-must-be-followed-by-39compare39-39explicit39-39infer39-or-39strict39"></a>&#39;Opção&#39; deve ser seguido por &#39;comparar&#39;, &#39;Explicit&#39;, &#39;Infer&#39;, ou &#39;Strict&#39;
As únicas opções que podem ser definidas com o `Option` instrução são `Compare`, `Explicit`, `Infer`, e `Strict`.  
  
 **ID do erro:** BC30206  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se uma palavra-chave option (`Compare`, `Explicit`, `Infer`, ou `Strict`) está incorreto.  
  
-   Verifique se o `Option` instrução está incompleta.

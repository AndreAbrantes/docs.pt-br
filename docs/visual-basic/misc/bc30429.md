# <a name="39end-sub39-must-be-preceded-by-a-matching-39sub39"></a>&#39; End Sub &#39; deve ser precedido por uma correspondência &#39; Sub &#39;
Um `End Sub` instrução aparece no seu código com nenhuma correspondência `Sub` definição do procedimento anterior.  
  
 **ID do erro:** BC30429  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `End Sub` instrução se ela é redundante.  
  
-   Forneça a `Sub` procedimento se um estiver ausente.  
  
-   Mover o `End Sub` até o local apropriado no código.

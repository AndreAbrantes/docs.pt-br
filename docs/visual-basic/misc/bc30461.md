# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>Classes não podem ser declaradas &#39; &lt;especificador&gt;&#39;
Você tentou declarar uma `Class` ou `Module` com um especificador inválido.  
  
 **ID do erro:** BC30461  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

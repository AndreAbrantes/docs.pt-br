# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Compartilhado &#39;Sub New&#39; não pode ter nenhum parâmetro
Você incluiu um ou mais parâmetros em um compartilhado `Sub New` instrução, que não é válida.  
  
 **ID do erro:** BC30479  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o parâmetro ou parâmetros.

# <a name="end-module-must-be-preceded-by-a-matching-module"></a>'End Module' deve ser precedido por um 'Module' correspondente
Blocos de módulo começam com um `Module` instrução e terminam com um `End Module` construir.  
  
 **ID do erro:** BC30622  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Certifique-se de que o bloco de módulo começa com um `Module` instrução.

# <a name="statement-cannot-appear-within-an-event-body"></a>A instrução não pode aparecer dentro do corpo de evento
A instrução não pode aparecer dentro do corpo de evento. Fim do evento assumido.  
  
 Uma instrução que não é válida dentro de um corpo de evento é exibido dentro de um.  
  
 **ID do erro:** BC31112  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Event` antes da instrução.

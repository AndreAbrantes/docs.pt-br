# <a name="locale-id-ltnamegt-is-not-supported-on-this-system"></a>Identificação de localidade '&lt;nome&gt;' não tem suporte neste sistema
Você está tentando usar `VbStrConv`, mas seu sistema não oferece suporte para a localidade especificada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade, ou executar o aplicativo em um computador que dá suporte à localidade especificada.

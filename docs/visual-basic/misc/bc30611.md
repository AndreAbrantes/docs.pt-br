# <a name="array-dimensions-cannot-have-a-negative-size"></a>Dimensões de matriz não podem ter um tamanho negativo
Um ou mais dos limites de matriz especificados são um número negativo. Você pode especificar um subscrito negativo somente quando você usa um limite superior de -1 para declarar uma matriz vazia. Como uma matriz com zero elementos, mas não é `Nothing`.  
  
 **ID do erro:** BC30611  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Substitua a matriz negativa associada com um número positivo.

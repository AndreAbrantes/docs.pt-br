# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="f2ec5-102">Tipo de '\<variablename >' é ambíguo porque os limites do loop e a variável step não são ampliados para o mesmo tipo</span><span class="sxs-lookup"><span data-stu-id="f2ec5-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="f2ec5-103">O código contiver um `For...Next` loop em que o compilador não é possível inferir um tipo de dados para a variável de controle de loop porque as seguintes condições é verdadeira:</span><span class="sxs-lookup"><span data-stu-id="f2ec5-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="f2ec5-104">O tipo de dados da variável de controle de loop não é especificado com um `As` cláusula.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="f2ec5-105">Os limites do loop e a variável step contêm pelo menos dois tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="f2ec5-106">Existe mais de uma conversão possível entre os tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="f2ec5-107">Não há nenhum tipo melhor entre os candidatos, para que a escolha do tipo para a variável de controle de loop é ambígua.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="f2ec5-108">Por exemplo, o seguinte loop tem um limite de tipo `Integer` e um loop de vinculado do tipo `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="f2ec5-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="f2ec5-109">Há uma conversão possível entre `Integer` e `UInteger`e uma conversão possível entre `UInteger` e `Integer`, mas ambos são conversões de estreitamento portanto, nenhum dos dois é a melhor opção.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="f2ec5-110">No próximo exemplo, uma terceira variável do tipo `Double` é adicionado.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="f2ec5-111">Ambos `Integer` e `UInteger` ter conversões de expansão padrão para `Double`, que faz a conversão para `Double` a melhor opção.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="f2ec5-112">Inferência de tipo atribui à variável de controle de loop `i` o tipo de dados `Double`.</span><span class="sxs-lookup"><span data-stu-id="f2ec5-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```

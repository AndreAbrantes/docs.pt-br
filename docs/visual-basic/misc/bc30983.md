# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="9acac-102">O tipo de '\<variablename>' é ambíguo porque os limites de loop e a variável step não são expandidos para o mesmo tipo</span><span class="sxs-lookup"><span data-stu-id="9acac-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="9acac-103">Seu código contém um `For...Next` loop no qual o compilador não pode inferir um tipo de dados para a variável de controle de loop porque as seguintes condições são verdadeiras:</span><span class="sxs-lookup"><span data-stu-id="9acac-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="9acac-104">O tipo de dados da variável de controle de loop não é especificado com uma `As` cláusula.</span><span class="sxs-lookup"><span data-stu-id="9acac-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="9acac-105">Os limites do loop e a variável Step contêm pelo menos dois tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="9acac-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="9acac-106">Há mais de uma conversão possível entre os tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="9acac-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="9acac-107">Não há um melhor tipo entre os candidatos, para que a escolha do tipo para a variável de controle de loop seja ambígua.</span><span class="sxs-lookup"><span data-stu-id="9acac-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="9acac-108">Por exemplo, o loop a seguir tem um limite de loop do tipo `Integer` e um limite de loop do tipo `UInteger` :</span><span class="sxs-lookup"><span data-stu-id="9acac-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="9acac-109">Há uma possível conversão entre `Integer` e e `UInteger` uma possível conversão entre `UInteger` e `Integer` , mas ambas são conversões estreitas, portanto, nenhuma é a melhor opção.</span><span class="sxs-lookup"><span data-stu-id="9acac-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="9acac-110">No exemplo a seguir, uma terceira variável do tipo `Double` é adicionada.</span><span class="sxs-lookup"><span data-stu-id="9acac-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="9acac-111">Ambos `Integer` e `UInteger` têm conversões de alargamento padrão para `Double` o, o que faz a conversão para `Double` a melhor escolha.</span><span class="sxs-lookup"><span data-stu-id="9acac-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="9acac-112">A inferência de tipos atribui à variável `i` de controle de loop o tipo de dados `Double` .</span><span class="sxs-lookup"><span data-stu-id="9acac-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```

# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="367cc-102">O tipo de ' \<variablename > ' é ambíguo porque os limites do loop e a variável Step não se ampliam para o mesmo tipo</span><span class="sxs-lookup"><span data-stu-id="367cc-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="367cc-103">Seu código contém um loop `For...Next` no qual o compilador não pode inferir um tipo de dados para a variável de controle de loop porque as seguintes condições são verdadeiras:</span><span class="sxs-lookup"><span data-stu-id="367cc-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
- <span data-ttu-id="367cc-104">O tipo de dados da variável de controle de loop não é especificado com uma cláusula `As`.</span><span class="sxs-lookup"><span data-stu-id="367cc-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
- <span data-ttu-id="367cc-105">Os limites do loop e a variável Step contêm pelo menos dois tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="367cc-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
- <span data-ttu-id="367cc-106">Há mais de uma conversão possível entre os tipos de dados.</span><span class="sxs-lookup"><span data-stu-id="367cc-106">More than one possible conversion exists between the data types.</span></span>  
  
- <span data-ttu-id="367cc-107">Não há um melhor tipo entre os candidatos, para que a escolha do tipo para a variável de controle de loop seja ambígua.</span><span class="sxs-lookup"><span data-stu-id="367cc-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="367cc-108">Por exemplo, o loop a seguir tem um limite de loop do tipo `Integer` e um único loop do tipo `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="367cc-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="367cc-109">Há uma possível conversão entre `Integer` e `UInteger` e uma possível conversão entre `UInteger` e `Integer`, mas ambos são conversões redutoras, portanto, nenhuma é a melhor opção.</span><span class="sxs-lookup"><span data-stu-id="367cc-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="367cc-110">No exemplo a seguir, uma terceira variável do tipo `Double` é adicionada.</span><span class="sxs-lookup"><span data-stu-id="367cc-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="367cc-111">@No__t-0 e `UInteger` têm conversões de ampliação padrão para `Double`, o que torna a conversão em `Double` a melhor opção.</span><span class="sxs-lookup"><span data-stu-id="367cc-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="367cc-112">A inferência de tipos atribui a variável de controle de loop `i` o tipo de dados `Double`.</span><span class="sxs-lookup"><span data-stu-id="367cc-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```

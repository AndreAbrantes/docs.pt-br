# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Campo '&lt;fieldname&gt;' é de um tipo sem suporte
Um campo está declarado como um tipo que não pode ser representado por [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **ID do erro:** BC30656  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Defina o campo como um tipo com suporte.  
  
2.  Desenvolva um novo tipo que descreve os dados no campo.

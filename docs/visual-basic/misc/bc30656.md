# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Campo &#39; &lt;fieldname&gt; &#39; é de um tipo sem suporte
Um campo está declarado como um tipo que não pode ser representado pelo Visual Studio.  
  
 **ID do erro:** BC30656  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Defina o campo como um tipo com suporte.  
  
2.  Desenvolva um novo tipo que descreve os dados no campo.

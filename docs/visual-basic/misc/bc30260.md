# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>'\<nome >' já está declarado como\<declaração >' neste \<declarationspace >
Dois elementos de programação são declarados com o mesmo nome no mesmo espaço de declaração (uma classe, módulo, interface ou estrutura).  
  
 **ID do erro:** BC30260  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Mover uma das declarações para um espaço de declaração diferente ou alterar o nome de um dos elementos declarados.

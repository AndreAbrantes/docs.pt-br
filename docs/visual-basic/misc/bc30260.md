# <a name="39ltnamegt39-is-already-declared-as-39ltdeclarationgt39-in-this-ltdeclarationspacegt"></a>&#39;&lt;nome&gt; &#39; já está declarado como &#39; &lt;declaração&gt; &#39; na &lt;declarationspace&gt;
Dois elementos de programação são declarados com o mesmo nome no mesmo espaço de declaração (uma classe, módulo, interface ou estrutura).  
  
 **ID do erro:** BC30260  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover uma das declarações para um espaço de declaração diferente ou altere o nome de um dos elementos declarados.

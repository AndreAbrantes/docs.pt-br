# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>Argumento correspondente ao parâmetro &#39; &lt;parametername&gt; &#39; limita-se a &#39; &lt;typename&gt;&#39;
Foi feita uma chamada para um método sobrecarregado, mas o compilador não pode localizar um método que pode ser chamado sem uma conversão de restrição. Um *conversão de estreitamento* altera um valor para um tipo de dados que pode não ser capaz de armazenar alguns dos possíveis valores.  
  
 **ID do erro:** BC36711

# <a name="argument-matching-parameter-ltparameternamegt-narrows-to-lttypenamegt"></a>Argumento correspondente ao parâmetro '&lt;parametername&gt;'restringe-se para'&lt;typename&gt;'
Foi feita uma chamada para um método sobrecarregado, mas o compilador não encontra um método que pode ser chamado sem uma conversão de estreitamento. Um *conversão de estreitamento* altera um valor para um tipo de dados que pode não ser capaz de armazenar alguns dos possíveis valores.  
  
 **ID do erro:** BC36711

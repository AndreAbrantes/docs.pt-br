# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>'&lt;nome&gt;'não é um membro de'&lt;classname&gt;'
O membro que você forneceu não é um membro da classe.  
  
 **ID do erro:** BC30456  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique o nome do membro para garantir sua precisão.  
  
2.  Use um membro real da classe.

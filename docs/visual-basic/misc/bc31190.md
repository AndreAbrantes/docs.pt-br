# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literais XML e propriedades de eixo XML não estão disponíveis
Literais XML e propriedades de eixo XML não estão disponíveis. Adicione referências a System. XML, LINQ e System. Core.  
  
 O código que está sendo compilado inclui literais XML ou propriedades de eixo XML, mas ele não faz referência os assemblies que são necessárias para compilar literais XML ou propriedades de eixo XML.  
  
 **ID do erro:** BC31190  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione referências aos assemblies DLL, XML e dll.

# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="91233-102">&#39;&lt;derivedtypename&gt; &#39; não pode herdar de &lt;tipo&gt; &#39; &lt;constructedbasetypename&gt; &#39; porque ele expande o acesso do tipo &#39; &lt; internaltypename&gt; &#39; para &lt;região&gt; &#39; &lt;regionname&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="91233-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="91233-103">Uma classe derivada ou interface tenta expandir o nível de acesso de um tipo interno ao usá-lo como um argumento de tipo a uma classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="91233-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="91233-104">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="91233-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="91233-105">Código fora `containingClass` não tem permissão para acessar `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="91233-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="91233-106">No entanto, `derivedClass` pode ser acessado de qualquer código no mesmo assembly.</span><span class="sxs-lookup"><span data-stu-id="91233-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="91233-107">Portanto, `derivedClass` não é possível herdar `baseClass` se ele usa `internalStructure` como um argumento de tipo, porque que poderia expor `internalStructure` em toda a região de código de definição.</span><span class="sxs-lookup"><span data-stu-id="91233-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="91233-108">**ID do erro:** BC30921</span><span class="sxs-lookup"><span data-stu-id="91233-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="91233-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="91233-109">To correct this error</span></span>  
  
-   <span data-ttu-id="91233-110">Ajuste os níveis de acesso das interfaces ou classes de forma que o tipo derivado não expande o nível de acesso do tipo interno.</span><span class="sxs-lookup"><span data-stu-id="91233-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="91233-111">-ou-</span><span class="sxs-lookup"><span data-stu-id="91233-111">-or-</span></span>

# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="51c90-102">' \<derivedtypename> ' não pode herdar de \<type> ' \<constructedbasetypename> ' porque ele expande o acesso do tipo ' \<internaltypename> ' para \<region> ' \<regionname> '</span><span class="sxs-lookup"><span data-stu-id="51c90-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>

<span data-ttu-id="51c90-103">Uma classe ou interface derivada tenta expandir o nível de acesso de um tipo interno usando-o como um argumento de tipo para uma classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="51c90-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="51c90-104">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="51c90-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="51c90-105">O código fora `containingClass` não tem permissão para acessar `internalStructure` .</span><span class="sxs-lookup"><span data-stu-id="51c90-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="51c90-106">No entanto, `derivedClass` o pode ser acessado de qualquer código no mesmo assembly.</span><span class="sxs-lookup"><span data-stu-id="51c90-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="51c90-107">Portanto, `derivedClass` não é possível herdar `baseClass` se ele usa `internalStructure` como um argumento de tipo, porque isso pode ser exposto `internalStructure` em toda a região de código de definição.</span><span class="sxs-lookup"><span data-stu-id="51c90-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="51c90-108">**ID do erro:** BC30921</span><span class="sxs-lookup"><span data-stu-id="51c90-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="51c90-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="51c90-109">To correct this error</span></span>  
  
- <span data-ttu-id="51c90-110">Ajuste os níveis de acesso das classes ou interfaces para que o tipo derivado não expanda o nível de acesso do tipo interno.</span><span class="sxs-lookup"><span data-stu-id="51c90-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="51c90-111">- ou -</span><span class="sxs-lookup"><span data-stu-id="51c90-111">-or-</span></span>  
  
- <span data-ttu-id="51c90-112">Se você não puder ajustar os níveis de acesso, não use o tipo interno como um argumento de tipo ao construir a classe base ou a interface.</span><span class="sxs-lookup"><span data-stu-id="51c90-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>

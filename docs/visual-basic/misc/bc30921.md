# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="df991-102">'&lt;derivedtypename&gt;' não pode herdar de &lt;tipo&gt; '&lt;constructedbasetypename&gt;'porque ele expande o acesso do tipo'&lt;internaltypename&gt;' para &lt;região&gt; '&lt;regionname&gt;'</span><span class="sxs-lookup"><span data-stu-id="df991-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="df991-103">Uma classe derivada ou interface tenta expandem o nível de acesso de um tipo interno ao usá-lo como um argumento de tipo a uma classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="df991-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="df991-104">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="df991-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="df991-105">Código fora `containingClass` não tem permissão para acessar `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="df991-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="df991-106">No entanto, `derivedClass` podem ser acessados de qualquer código no mesmo assembly.</span><span class="sxs-lookup"><span data-stu-id="df991-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="df991-107">Portanto, `derivedClass` não é possível herdar `baseClass` se ele usa `internalStructure` como um argumento de tipo, porque que poderia expor `internalStructure` em toda a região de código de definição.</span><span class="sxs-lookup"><span data-stu-id="df991-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="df991-108">**ID do erro:** BC30921</span><span class="sxs-lookup"><span data-stu-id="df991-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="df991-109">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="df991-109">To correct this error</span></span>  
  
-   <span data-ttu-id="df991-110">Ajuste os níveis de acesso das interfaces ou classes, de modo que o tipo derivado não expande o nível de acesso do tipo interno.</span><span class="sxs-lookup"><span data-stu-id="df991-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="df991-111">-ou-</span><span class="sxs-lookup"><span data-stu-id="df991-111">-or-</span></span>  
  
-   <span data-ttu-id="df991-112">Se você não pode ajustar os níveis de acesso, não use o tipo interno como um argumento de tipo ao construir a classe base ou interface.</span><span class="sxs-lookup"><span data-stu-id="df991-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>

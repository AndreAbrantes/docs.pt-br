# <a name="inherits-not-valid-in-modules"></a>'Inherits' não é válido em módulos
Um `Inherits` instrução ocorre dentro de um módulo. Módulos não podem herdar de classes.  
  
 **ID do erro:** BC30230  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remover o `Inherits` instrução ou redigite o módulo como uma classe.

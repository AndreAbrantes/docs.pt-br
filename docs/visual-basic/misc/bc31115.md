# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39; AddHandler &#39; a declaração deve terminar com uma correspondência &#39; End AddHandler &#39;
Um `AddHandler` declaração deve terminar com um `End AddHandler` instrução.  
  
 **ID do erro:** BC31115  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `AddHandler` declaração termina com um `End AddHandler` instrução.

# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>A declaração 'AddHandler' deve encerrar com 'End AddHandler' correspondente
Uma `AddHandler` declaração deve terminar com um `End AddHandler` instrução.  
  
 **ID do erro:** BC31115  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `AddHandler` declaração termina com um `End AddHandler` instrução.

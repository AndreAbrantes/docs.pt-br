# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="41449-102">Falha na inferência de argumento de tipo para o parâmetro de tipo ' \<typeparametername > ' de ' \<genericproceduresignature > '</span><span class="sxs-lookup"><span data-stu-id="41449-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="41449-103">**ID do erro:** BC32051</span><span class="sxs-lookup"><span data-stu-id="41449-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="41449-104">Falha na inferência de argumento de tipo para o parâmetro de tipo ' \<typeparametername > ' de ' \<genericproceduresignature > '.</span><span class="sxs-lookup"><span data-stu-id="41449-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="41449-105">Não foi possível inferir o argumento de tipo a partir do argumento passado para o parâmetro ' \<parametername > '.</span><span class="sxs-lookup"><span data-stu-id="41449-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="41449-106">Um procedimento genérico é chamado sem fornecer nenhum argumento de tipo, e o compilador não pode inferir o tipo a ser passado para um dos parâmetros.</span><span class="sxs-lookup"><span data-stu-id="41449-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="41449-107">Normalmente, quando você chama um procedimento genérico, você fornece um argumento de tipo para cada parâmetro de tipo que o procedimento genérico define.</span><span class="sxs-lookup"><span data-stu-id="41449-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="41449-108">Se você não fornecer nenhum argumento de tipo, o compilador tentará inferir os tipos a serem passados para os parâmetros de tipo.</span><span class="sxs-lookup"><span data-stu-id="41449-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="41449-109">Se o contexto da chamada fornecer informações de tipo de dados conflitantes para um parâmetro de tipo, a inferência de tipo falhará.</span><span class="sxs-lookup"><span data-stu-id="41449-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="41449-110">O código a seguir pode gerar esse erro:</span><span class="sxs-lookup"><span data-stu-id="41449-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="41449-111">No exemplo anterior, o compilador infere o tipo `Integer` para `t` com base no valor de 42 passado para `arg2`.</span><span class="sxs-lookup"><span data-stu-id="41449-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="41449-112">No entanto, essa inferência exigiria que `arg1` fosse do tipo `Integer()`, ou seja, uma matriz de `Integer`, e o valor 6 passado para `arg1` não corresponda a esse tipo.</span><span class="sxs-lookup"><span data-stu-id="41449-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="41449-113">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="41449-113">To correct this error</span></span>

- <span data-ttu-id="41449-114">Forneça argumentos de tipo para o procedimento genérico, para que o compilador não precise inferir-lhes.</span><span class="sxs-lookup"><span data-stu-id="41449-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>

- <span data-ttu-id="41449-115">Forneça argumentos normais com tipos que correspondam aos argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="41449-115">Supply normal arguments with types that match those of the type arguments.</span></span>

# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="9a58d-102">Falha na inferência de argumento de tipo para o parâmetro de tipo ' \<typeparametername > ' de ' \<genericproceduresignature > '</span><span class="sxs-lookup"><span data-stu-id="9a58d-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>
<span data-ttu-id="9a58d-103">Falha na inferência de argumento de tipo para o parâmetro de tipo ' \<typeparametername > ' de ' \<genericproceduresignature > '.</span><span class="sxs-lookup"><span data-stu-id="9a58d-103">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="9a58d-104">Não foi possível inferir o argumento de tipo a partir do argumento passado para o parâmetro ' \<parametername > '.</span><span class="sxs-lookup"><span data-stu-id="9a58d-104">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>  
  
 <span data-ttu-id="9a58d-105">Um procedimento genérico é chamado sem fornecer nenhum argumento de tipo, e o compilador não pode inferir o tipo a ser passado para um dos parâmetros.</span><span class="sxs-lookup"><span data-stu-id="9a58d-105">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>  
  
 <span data-ttu-id="9a58d-106">Normalmente, quando você chama um procedimento genérico, você fornece um argumento de tipo para cada parâmetro de tipo que o procedimento genérico define.</span><span class="sxs-lookup"><span data-stu-id="9a58d-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="9a58d-107">Se você não fornecer nenhum argumento de tipo, o compilador tentará inferir os tipos a serem passados para os parâmetros de tipo.</span><span class="sxs-lookup"><span data-stu-id="9a58d-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="9a58d-108">Se o contexto da chamada fornecer informações de tipo de dados conflitantes para um parâmetro de tipo, a inferência de tipo falhará.</span><span class="sxs-lookup"><span data-stu-id="9a58d-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="9a58d-109">O código a seguir pode gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="9a58d-109">The following code can generate this error.</span></span>  
  
```vb  
Public Sub doSomething(Of t)(ByVal arg1 As t(), ByVal arg2 As t)  
End Sub  
Call doSomething(6, 42)  
```  
  
 <span data-ttu-id="9a58d-110">No exemplo anterior, o compilador infere o tipo `Integer` para `t` com base no valor de 42 passado para `arg2`.</span><span class="sxs-lookup"><span data-stu-id="9a58d-110">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="9a58d-111">No entanto, essa inferência exigiria que `arg1` fosse do tipo `Integer()`, ou seja, uma matriz de `Integer`, e o valor 6 passado para `arg1` não corresponda a esse tipo.</span><span class="sxs-lookup"><span data-stu-id="9a58d-111">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>  
  
 <span data-ttu-id="9a58d-112">**ID do erro:** BC32051</span><span class="sxs-lookup"><span data-stu-id="9a58d-112">**Error ID:** BC32051</span></span>

# <a name="argument-argumentname-is-nothing-or-empty"></a>Argumento '\<argumentname >' é Nothing ou vazio
Um nome de argumento não é válido porque ele é inicializado para `Nothing` ou está vazio. Um argumento deve ter um tipo de dados específico e um nome válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione um nome de argumento válido.

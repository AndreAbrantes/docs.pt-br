# <a name="argument-path-is-nothing-or-empty"></a>O argumento 'Path' é Nothing ou vazio
Um nome de caminho não é válido porque ele é inicializado para `Nothing` ou está vazio. Um caminho deve ter um tipo de dados específico e um nome válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione um nome de caminho válido.

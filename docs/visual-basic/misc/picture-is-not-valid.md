# <a name="picture-is-not-valid"></a>Imagem não é válida
Você tentou atribuir um formato de gráfico que não seja um bitmap, ícone ou metarquivo do Windows para o `Picture` propriedade de um formulário ou controle.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que o arquivo que você está tentando carregar para o `Picture` propriedade é um arquivo de gráficos válido com suporte pelo Visual Basic.

# <a name="local-variable-ltvariablenamegt-cannot-be-referred-to-before-it-is-declared"></a>Variável local '&lt;variablename&gt;' não pode ser referida antes de ser declarada
Uma variável dentro de um procedimento é usada antes de sua declaração.  
  
 **ID do erro:** BC32000  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a declaração para preceder a primeira referência à variável.

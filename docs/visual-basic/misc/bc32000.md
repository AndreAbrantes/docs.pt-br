# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Variável local &#39; &lt;variablename&gt;&#39; não pode ser referida antes de ser declarada
Uma variável dentro de um procedimento é usada antes da sua declaração.  
  
 **ID do erro:** BC32000  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a declaração para preceder a primeira referência à variável.

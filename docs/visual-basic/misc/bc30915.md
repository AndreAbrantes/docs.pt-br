O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30915  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public`.  
  
     -ou-  
  
-   Use outro tipo.

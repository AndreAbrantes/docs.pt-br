# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>'\<nome >' não pode expor o tipo delegado subjacente '\<delegatetype >' do evento que está implementando fora do projeto por meio \<especificador > '\<tipo >'
O código expõe um tipo particular fora da classe pública.  
  
 **ID do erro:** BC30915  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declare o tipo `Public`.  
  
     - ou -  
  
- Use outro tipo.

# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttypegt39"></a>&#39; &lt;nome&gt;&#39; não pode expor o tipo delegado subjacente &#39;&lt; delegatetype&gt;&#39; do evento que está implementando fora do projeto por meio de &lt;especificador&gt; &#39;&lt; tipo de&gt;&#39;
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30915  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public`.  
  
     -ou-  
  
-   Use outro tipo.

# <a name="unable-to-open-key-file-filename-error"></a>Não é possível abrir o arquivo de chave '\<filename >': \<erro >
O arquivo de chave especificado para este projeto não pôde ser aberto.  
  
 **ID do erro:** BC31025  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verificar se o arquivo de chave existe e não está bloqueado por outro aplicativo.

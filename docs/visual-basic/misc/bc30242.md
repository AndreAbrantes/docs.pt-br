# <a name="modifier-is-not-valid-on-a-method-declaration"></a>'\<modificador >' não é válido em uma declaração de método
Um `Function` ou `Sub` instrução contém uma palavra-chave inválida.  
  
 **ID do erro:** BC30242  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a palavra-chave inválida do `Function` ou `Sub` instrução.

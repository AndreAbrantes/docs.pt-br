# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="ec32e-102">Membro de tipo anônimo ou propriedade '&lt;propertyname&gt;' já está declarado</span><span class="sxs-lookup"><span data-stu-id="ec32e-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="ec32e-103">Um nome de propriedade pode ser estabelecido com apenas uma vez na declaração de um tipo anônimo.</span><span class="sxs-lookup"><span data-stu-id="ec32e-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="ec32e-104">Por exemplo, as declarações a seguir não são válidas:</span><span class="sxs-lookup"><span data-stu-id="ec32e-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="ec32e-105">**ID do erro:** BC36547</span><span class="sxs-lookup"><span data-stu-id="ec32e-105">**Error ID:** BC36547</span></span>

```vb  
' Valid.

```

# <a name="anonymous-type-member-or-property-39ltpropertynamegt39-is-already-declared"></a><span data-ttu-id="93a8a-102">Membro de tipo anônimo ou propriedade &#39; &lt;propertyname&gt; &#39; já está declarado</span><span class="sxs-lookup"><span data-stu-id="93a8a-102">Anonymous type member or property &#39;&lt;propertyname&gt;&#39; is already declared</span></span>
<span data-ttu-id="93a8a-103">Um nome de propriedade pode ser estabelecido apenas uma vez na declaração de um tipo anônimo.</span><span class="sxs-lookup"><span data-stu-id="93a8a-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="93a8a-104">Por exemplo, as seguintes declarações não são válidas:</span><span class="sxs-lookup"><span data-stu-id="93a8a-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="93a8a-105">**ID do erro:** BC36547</span><span class="sxs-lookup"><span data-stu-id="93a8a-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="93a8a-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="93a8a-106">To correct this error</span></span>

```vb  
' Valid.

```

```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```

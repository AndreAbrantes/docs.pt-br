# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="d06df-102">O membro ou propriedade de tipo anônimo '\<propertyname>' já foi declarada</span><span class="sxs-lookup"><span data-stu-id="d06df-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>

<span data-ttu-id="d06df-103">Um nome de propriedade só pode ser estabelecido uma vez na declaração de um tipo anônimo.</span><span class="sxs-lookup"><span data-stu-id="d06df-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="d06df-104">Por exemplo, as seguintes declarações não são válidas:</span><span class="sxs-lookup"><span data-stu-id="d06df-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="d06df-105">**ID do erro:** BC36547</span><span class="sxs-lookup"><span data-stu-id="d06df-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d06df-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="d06df-106">To correct this error</span></span>  
  
- <span data-ttu-id="d06df-107">Escolha um nome diferente para uma das propriedades.</span><span class="sxs-lookup"><span data-stu-id="d06df-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="d06df-108">Forneça novos nomes para as variáveis ou nomes de propriedade dos quais você está inferindo nomes e valores.</span><span class="sxs-lookup"><span data-stu-id="d06df-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```

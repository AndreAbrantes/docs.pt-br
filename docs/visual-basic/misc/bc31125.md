# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>'End RemoveHandler' deve ser precedido por uma declaração 'RemoveHandler' correspondente
Uma `End RemoveHandler` instrução ocorreu sem um correspondente `RemoveHandler` instrução. `End RemoveHandler` deve ser precedida por uma `RemoveHandler` instrução.  
  
 **ID do erro:** BC31125  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se o anterior `RemoveHandler` instrução é válida e foi digitada corretamente.

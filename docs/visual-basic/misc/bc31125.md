# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39; End RemoveHandler &#39; deve ser precedido por uma correspondência &#39; RemoveHandler &#39; declaração
Um `End RemoveHandler` instrução ocorreu sem correspondente `RemoveHandler` instrução. `End RemoveHandler`deve ser precedida por uma `RemoveHandler` instrução.  
  
 **ID do erro:** BC31125  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o anterior `RemoveHandler` instrução é válida e foi digitada corretamente.

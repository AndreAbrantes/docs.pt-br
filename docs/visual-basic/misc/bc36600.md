# <a name="range-variable-variable-is-already-declared"></a>Variável de intervalo \<variável > já está declarado
Um nome de variável de intervalo especificado em uma `Select` cláusula, ou o `Into` faz parte de uma `Aggregate`, `Group By`, ou `Group Join` cláusula, duplique o nome de uma variável de intervalo já especificado na cláusula de consulta.  
  
 **ID do erro:** BC36600  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que todas as variáveis de intervalo em uma cláusula de consulta tenham nomes exclusivos.

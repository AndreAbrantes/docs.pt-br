# <a name="expected-closing--for-embedded-expression"></a>Fechamento esperado ' % >' para a expressão inserida
Uma expressão inserida em um literal XML está ausente do fechamento necessários `%>` caracteres.  
  
 **ID do erro:** BC31159  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento necessários `%>` caracteres ao final da expressão inserida.

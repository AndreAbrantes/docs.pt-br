# <a name="expected-closing-39gt39-for-embedded-expression"></a>Fechamento esperado &#39;%&gt; &#39; na expressão inserida
Uma expressão inserida em um literal XML está faltando o fechamento necessários `%>` caracteres.  
  
 **ID do erro:** BC31159  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento necessários `%>` caracteres até o fim da expressão inserida.

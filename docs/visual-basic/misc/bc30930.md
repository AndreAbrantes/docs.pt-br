# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;nome&gt;&#39;, necessário para a compilação não pode ser encontrado
Não foi encontrado um elemento necessário para compilação.  
  
 **ID do erro:** BC30930  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Forneça o elemento necessário auxiliar.  
  
2.  Verifique se que o código é um código gerenciado válido.

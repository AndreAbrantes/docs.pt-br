# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>'\<nome >', necessário para compilação deste constructo, não pode ser encontrado
Não foi encontrado um elemento necessário para compilação.  
  
 **ID do erro:** BC30930  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Forneça o elemento necessário auxiliar.  
  
2.  Verifique se que o código é um código gerenciado válido.

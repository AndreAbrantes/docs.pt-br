# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39; &lt;typename&gt;&#39; é um tipo de &#39;&lt; ContainerName&gt;&#39; e não pode ser usado como uma expressão
Um dos nomes que está sendo usados já está declarado como o nome de um tipo e não pode ser usado em uma expressão.  
  
 **ID do erro:** BC30691  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Nome exclusivamente o membro.  
  
-   Use colchetes em torno do nome do membro.

# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>'&lt;typename&gt;'é um tipo em'&lt;containername&gt;' não pode ser usado como uma expressão
Um dos nomes que está sendo usados já está declarado como o nome de um tipo e não pode ser usado em uma expressão.  
  
 **ID do erro:** BC30691  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Nome exclusivamente o membro.  
  
-   Use colchetes em torno do nome do membro.

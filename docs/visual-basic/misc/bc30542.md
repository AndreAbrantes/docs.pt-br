# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Ocorreu uma divisão por zero durante a avaliação desta expressão
Avaliar esta expressão, você tentou dividir um número por zero.  
  
 **ID do erro:** BC30542  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os valores de constante e variável para determinar como zero ocorre no denominador da sua expressão.

# <a name="types-declared-private-must-be-inside-another-type"></a>Tipos declarados 'Private' devem estar dentro de outro tipo
O `Private` modificador foi usado em um tipo não dentro de outro tipo.  
  
 **ID do erro:** BC31089  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Use um modificador de acesso menos restritivo, como `Friend`.  
  
2.  Declare o tipo dentro de outro tipo.

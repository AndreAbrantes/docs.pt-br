# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Nome da função de agregação não pode ser usado com um caractere de tipo
Um caractere de tipo foi incluído no final do nome de uma função de agregação.  
  
 **ID do erro:** BC36617  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o caractere de tipo do nome da função de agregação.

# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39;&lt;VARIABLENAME&gt; &#39; não é uma variável local ou parâmetro e não pode ser usado como um &#39;Catch&#39; variável
Variáveis no `Try...Catch...Finally` instruções devem ser declaradas localmente ou ser parâmetros para o procedimento atual.  
  
 **ID do erro:** BC31082  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Declarar variáveis locais ou parâmetros para `Try...Catch...Finally` instruções.

# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39; SyncLock &#39; instrução deve terminar com uma correspondência &#39; End SyncLock &#39;
`SyncLock`blocos começam com o `SyncLock` palavra-chave e terminar com o `End SyncLock` construir.  
  
 **ID do erro:** BC30675  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se `SyncLock` bloco termina com um `End SyncLock` construir.

# <a name="file-io-with-type-typename-is-not-valid"></a>E/s de arquivo com o tipo '\<typename >' não é válido
Você está tentando gravar um tipo sem suporte em um arquivo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique a estrutura para determinar onde o tipo sem suporte ocorre e, em seguida, alterá-lo para um tipo válido.

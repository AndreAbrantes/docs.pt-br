# <a name="expected-3939-at-start-of-an-embedded-expression"></a>Esperado &#39;% =&#39; no início de uma expressão inserida
O identificador de início de uma expressão inserida, `<%=`, não foi codificado corretamente. Observe que você não pode usar um caractere de porcentagem (%) no nome de um literal de elemento XML.  
  
 **ID do erro:** BC31179  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o identificador de início para a expressão inserida é codificado como `<%=`.

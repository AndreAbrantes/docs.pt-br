# <a name="labels-are-not-valid-in-the-immediate-window"></a>Rótulos não são válidos na janela imediata
Rótulos de linha habilitar a ramificação e não são permitidos em um contexto de depuração.  
  
 **ID do erro:** BC30134  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não insira um rótulo de linha na **imediato** janela.

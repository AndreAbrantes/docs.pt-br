# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="9e5bd-102">Operandos 'If' não podem ser argumentos nomeados</span><span class="sxs-lookup"><span data-stu-id="9e5bd-102">&#39;If&#39; operands cannot be named arguments</span></span>
<span data-ttu-id="9e5bd-103">Usando argumentos com nome nos operandos do `If` operador não é válido.</span><span class="sxs-lookup"><span data-stu-id="9e5bd-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="9e5bd-104">O exemplo a seguir causa esse erro:</span><span class="sxs-lookup"><span data-stu-id="9e5bd-104">The following example causes this error:</span></span>  
  
```

Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="9e5bd-105">Isso difere do `IIf` função, permitindo que os argumentos nomeada, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="9e5bd-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>


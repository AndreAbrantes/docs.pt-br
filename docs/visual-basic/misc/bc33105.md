# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="4c37a-102">&#39;Se&#39; operandos não podem ser argumentos nomeados</span><span class="sxs-lookup"><span data-stu-id="4c37a-102">&#39;If&#39; operands cannot be named arguments</span></span>
<span data-ttu-id="4c37a-103">Usando argumentos com nome nos operandos do `If` operador não é válido.</span><span class="sxs-lookup"><span data-stu-id="4c37a-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="4c37a-104">O exemplo a seguir faz com que esse erro:</span><span class="sxs-lookup"><span data-stu-id="4c37a-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="4c37a-105">Isso difere de `IIf` função, que permite argumentos com nome, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="4c37a-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="4c37a-106">**ID do erro:** BC33105</span><span class="sxs-lookup"><span data-stu-id="4c37a-106">**Error ID:** BC33105</span></span>

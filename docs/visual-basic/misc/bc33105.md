# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="d4c20-102">Os operandos 'If' não podem ser argumentos nomeados</span><span class="sxs-lookup"><span data-stu-id="d4c20-102">'If' operands cannot be named arguments</span></span>

<span data-ttu-id="d4c20-103">O uso de argumentos nomeados nos operandos do `If` operador não é válido.</span><span class="sxs-lookup"><span data-stu-id="d4c20-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="d4c20-104">O exemplo a seguir causa esse erro:</span><span class="sxs-lookup"><span data-stu-id="d4c20-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="d4c20-105">Isso difere da `IIf` função, que permite argumentos nomeados, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="d4c20-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="d4c20-106">**ID do erro:** BC33105</span><span class="sxs-lookup"><span data-stu-id="d4c20-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d4c20-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="d4c20-107">To correct this error</span></span>


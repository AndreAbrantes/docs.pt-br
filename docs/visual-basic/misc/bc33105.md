# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="e7dfd-102">Os operandos 'If' não podem ser argumentos nomeados</span><span class="sxs-lookup"><span data-stu-id="e7dfd-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="e7dfd-103">Usando argumentos nos operandos do nomeados a `If` operador não é válido.</span><span class="sxs-lookup"><span data-stu-id="e7dfd-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="e7dfd-104">O exemplo a seguir faz com que esse erro:</span><span class="sxs-lookup"><span data-stu-id="e7dfd-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="e7dfd-105">Isso difere do `IIf` função, que permitem argumentos nomeada, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="e7dfd-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="e7dfd-106">**ID do erro:** BC33105</span><span class="sxs-lookup"><span data-stu-id="e7dfd-106">**Error ID:** BC33105</span></span>

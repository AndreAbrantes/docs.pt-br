# <a name="stop-statement-encountered"></a>Instrução Stop encontrada
Um `Stop` instrução, que suspende a execução, foi encontrada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Stop` instrução.

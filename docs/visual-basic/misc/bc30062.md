# <a name="me-cannot-be-the-target-of-an-assignment"></a>'Me' não pode ser o destino de uma atribuição
O `Me` palavra-chave sempre faz referência à instância específica de uma classe em que o código está sendo executado. Seu valor não pode ser alterado.  
  
 **ID do erro:** BC30062  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a instrução que atribui um valor para `Me`.

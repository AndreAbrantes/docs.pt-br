# <a name="method-call-did-not-return-a-value"></a>Chamada de método não retornou um valor
O método chamado não retornou um valor, conforme o esperado.  
  
 **ID do erro:** BC30723  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se o método chamado é uma função que retorna um valor.

# <a name="class-classname-has-no-accessible-sub-new-and-cannot-be-inherited"></a><span data-ttu-id="3ab05-102">A classe '\<classname>' não tem 'Sub New' acessível e não pode ser herdada</span><span class="sxs-lookup"><span data-stu-id="3ab05-102">Class '\<classname>' has no accessible 'Sub New' and cannot be inherited</span></span>

<span data-ttu-id="3ab05-103">Uma classe usa uma [instrução Inherits](../language-reference/statements/inherits-statement.md) para especificar uma classe base, mas não pode acessar nenhum construtor na classe base pretendida.</span><span class="sxs-lookup"><span data-stu-id="3ab05-103">A class uses an [Inherits Statement](../language-reference/statements/inherits-statement.md) to specify a base class, but it cannot access any constructor on the intended base class.</span></span>  
  
 <span data-ttu-id="3ab05-104">Isso pode acontecer se a classe base pretendida não tiver nenhum construtor ou se tiver construtores com níveis de acesso que impeçam o acesso de outra classe.</span><span class="sxs-lookup"><span data-stu-id="3ab05-104">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>  
  
 <span data-ttu-id="3ab05-105">Quando você herda uma classe, seu construtor deve chamar o construtor da classe base usando [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span><span class="sxs-lookup"><span data-stu-id="3ab05-105">When you inherit a class, your constructor should call the base class constructor using [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span></span> <span data-ttu-id="3ab05-106">Se você não fizer essa chamada, ou se não gravar um construtor explícito, Visual Basic gerará um Construtor implícito que chama `MyBase.New()` .</span><span class="sxs-lookup"><span data-stu-id="3ab05-106">If you do not make this call, or if you do not even write an explicit constructor, Visual Basic generates an implicit constructor that calls `MyBase.New()`.</span></span>  
  
 <span data-ttu-id="3ab05-107">**ID do erro:** BC31399</span><span class="sxs-lookup"><span data-stu-id="3ab05-107">**Error ID:** BC31399</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3ab05-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="3ab05-108">To correct this error</span></span>  
  
1. <span data-ttu-id="3ab05-109">Se você tiver controle do código-fonte sobre a classe base pretendida, altere o nível de acesso de pelo menos um de seus construtores para que outra classe possa acessá-los.</span><span class="sxs-lookup"><span data-stu-id="3ab05-109">If you have source control over the intended base class, then change the access level of at least one of its constructors so that another class can access them.</span></span>  
  
2. <span data-ttu-id="3ab05-110">Se você não puder alterar os níveis de acesso dos construtores de classe base pretendidos, herde de uma classe diferente ou não.</span><span class="sxs-lookup"><span data-stu-id="3ab05-110">If you cannot change the access levels of the intended base class constructors, then inherit from a different class or not at all.</span></span>

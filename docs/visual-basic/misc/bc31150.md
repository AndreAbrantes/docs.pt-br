# <a name="end-tag-lttagnamegt-expected"></a>Marca de fim &lt;/tagName&gt; esperado
A marca de fim em um literal de elemento XML não coincide com a marca de início correspondente.  
  
 **ID do erro:** BC31150  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Modificar o marcação XML para coincidir com a marca de XML de início para o elemento XML literal final.

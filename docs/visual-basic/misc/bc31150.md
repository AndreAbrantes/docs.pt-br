# <a name="end-tag-lttagnamegt-expected"></a>Marca de fim &lt;/tagName&gt; esperado
A marca final em um literal de elemento XML não coincide com a marca inicial correspondente.  
  
 **ID do erro:** BC31150  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Modificar marca XML para corresponder a marca XML de início para o elemento XML literal final.

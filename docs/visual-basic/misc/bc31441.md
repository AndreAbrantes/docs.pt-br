# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="c939d-102">O método ' \<methodname1> ' deve ser declarado como ' Private ' para implementar o método parcial ' \<methodname2> '</span><span class="sxs-lookup"><span data-stu-id="c939d-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>

<span data-ttu-id="c939d-103">A implementação de um método parcial deve ser declarada `Private` .</span><span class="sxs-lookup"><span data-stu-id="c939d-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="c939d-104">Por exemplo, o código a seguir causa esse erro.</span><span class="sxs-lookup"><span data-stu-id="c939d-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="c939d-105">**ID do erro:** BC31441</span><span class="sxs-lookup"><span data-stu-id="c939d-105">**Error ID:** BC31441</span></span>


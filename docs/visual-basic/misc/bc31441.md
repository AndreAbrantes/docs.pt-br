# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a>O\<método 'methodname1>' deve ser declarado 'Privado'\<para implementar o método parcial ' methodname2>'
A implementação de um método `Private`parcial deve ser declarada . Por exemplo, o código a seguir causa esse erro.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID de erro:** BC31441  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Use o modificador `Private` de acesso na implementação do método parcial, como mostrado no exemplo a seguir.  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```

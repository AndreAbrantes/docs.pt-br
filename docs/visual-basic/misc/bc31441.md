# <a name="method-ltmethodname1gt-must-be-declared-private-in-order-to-implement-partial-method-ltmethodname2gt"></a>Método '&lt;methodname1&gt;'deve ser declarado como 'Private' para implementar o método parcial'&lt;methodname2&gt;'
A implementação de um método parcial deve ser declarada `Private`. Por exemplo, o código a seguir faz com que esse erro.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,   
    ' causes this error.   
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID do erro:** BC31441  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Use o modificador de acesso `Private` na implementação do método parcial, conforme mostrado no exemplo a seguir.  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```

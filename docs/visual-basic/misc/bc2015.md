# <a name="the-file-39ltfilenamegt39-is-not-a-text-file"></a>O arquivo &#39; &lt;filename&gt;&#39; não é um arquivo de texto
O arquivo especificado não pode ser gravado porque é um arquivo binário.  
  
 **ID do erro:** BC2015  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se você está fornecendo o caminho e nome de arquivo correto.

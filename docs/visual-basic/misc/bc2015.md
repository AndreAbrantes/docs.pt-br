# <a name="the-file-ltfilenamegt-is-not-a-text-file"></a>O arquivo '&lt;filename&gt;' não é um arquivo de texto
O arquivo especificado não pode ser gravado porque é um arquivo binário.  
  
 **ID do erro:** BC2015  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se você está fornecendo o caminho e nome de arquivo correto.

# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39;End Operator&#39; deve ser precedido por uma correspondência &#39;operador&#39;
Um `End Operator` instrução aparece sem associação `Operator` declaração anterior.  
  
 **ID do erro:** BC33007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `End Operator` instrução se ela é redundante.  
  
-   Forneça a `Operator` procedimento se um estiver ausente.  
  
-   Mover o `End Operator` instrução até o local apropriado no código.

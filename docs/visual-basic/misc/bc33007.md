# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>'End Operator' deve ser precedido por um 'Operator' correspondente
Uma `End Operator` instrução aparece sem associação `Operator` declaração anterior a ele.  
  
 **ID do erro:** BC33007  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `End Operator` instrução se ela é redundante.  
  
-   Forneça a `Operator` procedimento se uma estiver ausente.  
  
-   Mover o `End Operator` instrução até o local apropriado no código.

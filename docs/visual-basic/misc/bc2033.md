# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Erro ao inserir o manifesto Win32: opção /win32manifest entra em conflito com /nowin32manifest
O compilador do Visual Basic tem sido passado ambos um `/win32manifest` opção de compilador e uma `/nowin32manifest` opção de compilador. Somente uma dessas opções pode ser passada para o compilador do Visual Basic.  
  
 **ID do erro:** BC2033  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `/win32manifest` opção de compilador ou `/nowin32manifest` opção de compilador.

# <a name="maximum-number-of-errors-has-been-exceeded"></a>Número máximo de erros foi excedido
O número máximo de erros de compilação é 100. O compilador impõe esse limite para reduzir o tamanho dos relatórios de erros.  
  
 **ID do erro:** BC30041  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Revise os outros erros gerados por esta compilação e resolva tantos quanto possível. Se o código-fonte contiver mais de 100 erros, você poderá ver mais erros na próxima compilação.

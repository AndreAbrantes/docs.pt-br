# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>Operador &#39; &lt;operatorname&gt; &#39; não está definido para o tipo &#39; &lt;typename&gt;&#39;
Você tentou usar um operador unário (`+`, `-`, ou `Not`) em um tipo, como um `String`, para que ele não está definido.  
  
 **ID do erro:** BC30487  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o operador unário por um operador apropriado para a expressão.

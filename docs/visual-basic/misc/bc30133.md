# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'GoTo' não são válidas na janela imediata
O `GoTo` instrução executa a ramificação e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30133  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita uma `GoTo` instrução em de **imediato** janela.

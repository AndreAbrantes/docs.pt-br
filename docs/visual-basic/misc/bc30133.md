# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>&#39;GoTo&#39; instruções não são válidas na janela imediata
O `GoTo` instrução executa a ramificação e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30133  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita um `GoTo` instrução o **imediato** janela.

# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Argumento não especificado para o parâmetro &#39; &lt;parametername&gt;&#39; do método de extensão &#39;&lt; MethodName&gt;&#39; definido no &#39;&lt; TypeName&gt;&#39;
Não foi fornecido um argumento para um parâmetro obrigatório em um método de extensão.  
  
 **ID do erro:** BC36586  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça um argumento para o parâmetro especificado.

# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Argumento não especificado para o parâmetro '&lt;parametername&gt;'do método de extensão '&lt;methodname&gt;'definido em'&lt;typename&gt;'
Não foi fornecido um argumento para um parâmetro obrigatório em um método de extensão.  
  
 **ID do erro:** BC36586  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Fornece um argumento para o parâmetro especificado.

# <a name="properties-in-a-module-cannot-be-declared-ltspecifiergt"></a>Propriedades em um módulo não podem ser declaradas como '&lt;especificador&gt;'
Você tentou usar um especificador, tais como `MustOverride`, `NotOverridable`, ou `Overridable`, em uma propriedade dentro de um módulo.  
  
 **ID do erro:** BC30503  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

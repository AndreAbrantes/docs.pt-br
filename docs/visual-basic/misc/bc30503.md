# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Propriedades em um módulo não podem ser declaradas &#39; &lt;especificador&gt;&#39;
Você tentou usar um especificador, como `MustOverride`, `NotOverridable`, ou `Overridable`, em uma propriedade dentro de um módulo.  
  
 **ID do erro:** BC30503  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

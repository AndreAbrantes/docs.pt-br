# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39declare-sub39-language-element"></a>Marca de comentário XML &#39;retorna&#39; não é permitida em um &#39;declare sub&#39; elemento de linguagem
A marca de comentário XML 'returns' não é permitida em um elemento de linguagem 'declare sub'. O comentário XML será ignorado.  
  
 Um comentário XML para um `Declare Sub` declaração não pode conter um `<`retorna`>` marca.  
  
 **ID do erro:** BC42315  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o suporte `<`retorna`>` marca.

# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Os parâmetros do método ' AddHandler ' e ' RemoveHandler ' não podem ser declarados ' ByRef '
Os parâmetros dos `AddHandler` métodos e `RemoveHandler` não podem ser declarados `ByRef` com o modificador.  
  
 **ID do erro:** BC31134  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a `ByRef` palavra-chave do parâmetro.

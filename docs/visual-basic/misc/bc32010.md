# <a name="name-cannot-be-named-as-a-parameter-in-an-attribute-specifier-because-it-is-not-a-field-or-property"></a>'\<nome >' não pode ser nomeado como um parâmetro em um especificador de atributo porque não é um campo ou propriedade
Um bloco de atributo define um valor para um membro não varie do atributo. Você pode atribuir valores a membros de variável, como campos ou propriedades.  
  
 **ID do erro:** BC32010  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que os nomes de atributo e membro foram escritos corretamente.  
  
2.  Remova o argumento do bloco de atributo se o membro é não varie.

# <a name="39ltnamegt39-cannot-be-named-as-a-parameter-in-an-attribute-specifier-because-it-is-not-a-field-or-property"></a>'&lt;nome&gt;' não pode ser nomeado como um parâmetro em um especificador de atributo porque ele não é um campo ou propriedade
Um bloco de atributo define um valor para um membro não varie do atributo. Você pode atribuir valores somente para membros das variáveis, como campos ou propriedades.  
  
 **ID do erro:** BC32010  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que os nomes de atributo e membro estão escritos corretamente.  
  
2.  Remova o argumento o bloco de atributo se o membro for não varie.

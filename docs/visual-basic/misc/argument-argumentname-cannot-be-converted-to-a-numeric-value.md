# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-a-numeric-value"></a>Argumento &#39; &lt;argumentname&gt; &#39; não pode ser convertido em um valor numérico
Uma conversão tentou converter uma variável, como uma cadeia de caracteres em um valor numérico.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Localize uma conversão que é significativa.

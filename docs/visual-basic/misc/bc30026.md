# <a name="end-sub-expected"></a>'End Sub' esperado
Um `End` instrução deve corresponder ao bloco em que ele é encerrado. Um `Sub` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC30026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Sub` instrução até o final do procedimento.

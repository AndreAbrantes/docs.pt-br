# <a name="39end-sub39-expected"></a>&#39; End Sub &#39; esperado
Um `End` instrução deve corresponder ao bloco encerra. Um `Sub` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC30026  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Sub` instrução até o final do procedimento.

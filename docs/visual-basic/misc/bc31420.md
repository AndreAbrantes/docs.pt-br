# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>&#39;&lt;membro&gt; &#39; está em conflito com o membro reservado por este nome que é declarado implicitamente em todos os enums
O nome de um tipo membro está em conflito com o nome de um membro declarado implicitamente em todas as enumerações.  
  
 O compilador do Visual Basic cria membros implícitos correspondentes a certos elementos de programação que você declarar. Enumerações declara implicitamente o membro `value__member`.  
  
 **ID do erro:** BC31420  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Modifique o nome do membro.

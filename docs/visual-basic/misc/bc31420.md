# <a name="ltmembergt-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>'&lt;membro&gt;' está em conflito com o membro reservado por este nome que é declarado implicitamente em todos os enums
O nome de um tipo de conflitos de membro com o nome de um membro implicitamente declarado em todas as enumerações.  
  
 O compilador do Visual Basic cria membros implícitos correspondentes a determinados elementos de programação que você declarar. Enumerações implicitamente declaram o membro `value__member`.  
  
 **ID do erro:** BC31420  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Modifique o nome do membro.

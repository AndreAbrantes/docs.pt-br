# <a name="expected-one-of-39dim39-39const39-39public39-39private39-39protected39-39friend39-39shadows39-39readonly39-or-39shared39"></a>Esperado um de &#39; Dim &#39; &#39; Const &#39; &#39; Público &#39; &#39; Privada &#39; &#39; Protegido &#39; &#39; Friend &#39; &#39; Sombras &#39; &#39; Somente leitura &#39; ou &#39; Compartilhado &#39;
Uma instrução de declaração está faltando uma palavra-chave de declaração. Uma possível causa é uma declaração de atributo chama um método.  
  
 **ID do erro:** BC30195  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se um método é declarado dentro de uma declaração de atributo.  
  
2.  Inicie a instrução com a palavra-chave de declaração adequado.

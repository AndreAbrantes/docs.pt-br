 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="f01d2-105">Se `derivedInterface` é construída ou implementada fornecendo o mesmo tipo para ambos `t1` e `t2`, ele deve herdar duas versões do `interfaceA` com argumentos de tipo idênticos.</span><span class="sxs-lookup"><span data-stu-id="f01d2-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="f01d2-106">Isso geraria ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="f01d2-106">Doing so would produce an ambiguity about which version to access.</span></span>

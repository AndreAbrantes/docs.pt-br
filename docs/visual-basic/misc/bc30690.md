# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Estrutura &#39; &lt;structurename&gt;&#39; não pode ser indexada porque não tem nenhuma propriedade padrão
Foi feita uma tentativa para usar a sintaxe da propriedade padrão com uma estrutura que não tem propriedade padrão.  
  
 **ID do erro:** BC30690  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use a sintaxe de propriedade padrão para acessar as propriedades da estrutura.

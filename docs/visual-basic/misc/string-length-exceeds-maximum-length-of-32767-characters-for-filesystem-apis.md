# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-39filesystem39-apis"></a>Comprimento de cadeia de caracteres excede o comprimento máximo de 32.767 caracteres para &#39;FileSystem&#39; APIs
Uma cadeia de caracteres excede o comprimento máximo de 32.767 caracteres.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Reduza a cadeia de caracteres.

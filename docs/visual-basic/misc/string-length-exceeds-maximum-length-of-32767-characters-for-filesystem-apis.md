# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-filesystem-apis"></a>Comprimento de cadeia de caracteres excede o comprimento máximo de 32.767 caracteres para APIs 'FileSystem'
Uma cadeia de caracteres excede o comprimento máximo de 32.767 caracteres.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Diminua a cadeia de caracteres.

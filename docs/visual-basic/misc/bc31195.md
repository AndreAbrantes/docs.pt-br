# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>O operador 'operator' não está definido para tipos 'type1' e 'type2'
O operador 'operator' não está definido para tipos 'type1' e 'type2'. Você pode usar a propriedade 'Value' para obter o valor de cadeia de caracteres do primeiro elemento de '\<parentElement >'.  
  
 Um literal XML foi usado em um binário ou `Boolean` comparação.  
  
 **ID do erro:** BC31195  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `Value` propriedade do XML literal para o binário ou `Boolean` comparação.

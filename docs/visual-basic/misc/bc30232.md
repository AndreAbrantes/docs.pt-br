# <a name="implemented-type-must-be-an-interface"></a>O tipo implementado deve ser uma interface
Um `Implements` instrução não especifica uma interface. Uma classe pode implementar apenas uma interface.  
  
 **ID do erro:** BC30232  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que o nome da interface está escrito corretamente.  
  
2.  Se a declaração especifica uma classe, considere usar o `Inherits` instrução.

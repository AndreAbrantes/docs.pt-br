  
 <span data-ttu-id="93434-105">As restrições indiretas `Structure` e `Class` causam um conflito para o parâmetro de tipo `t1` , porque a `Structure` restrição requer que o argumento de tipo correspondente seja um tipo de valor, enquanto `Class` requer que seja um tipo de referência.</span><span class="sxs-lookup"><span data-stu-id="93434-105">The indirect constraints `Structure` and `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>

# <a name="paramarray-parameters-must-be-declared-39byval39"></a>Parâmetros ParamArray devem ser declarados 'ByVal'
`ParamArray`não é possível usar parâmetros de `ByRef` modificador.  
  
 **ID do erro:** BC30667  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declarar `ParamArray` parâmetros usando o `ByVal` modificador.

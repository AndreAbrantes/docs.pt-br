# <a name="implicit-variable-variablename-is-invalid-because-of-message"></a>Variável implícita '\<variablename >' é inválida devido '\<mensagem >'
Uma variável implícita inválida foi gerada durante a depuração.  
  
 **ID do erro:** BC36016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Examine a mensagem citada para o motivo e uma solução possível.  
  
2. Se o erro persistir, colete informações sobre as circunstâncias e notifique o Microsoft Product Support services.

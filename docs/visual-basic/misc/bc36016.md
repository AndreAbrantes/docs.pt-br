# <a name="implicit-variable-39ltvariablenamegt39-is-invalid-because-of-39ltmessagegt39"></a>Variável implícita &#39; &lt;variablename&gt;&#39; é inválido porque &#39;&lt; mensagem&gt;&#39;
Uma variável implícita inválida foi gerada durante a depuração.  
  
 **ID do erro:** BC36016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Examine a mensagem citada para o motivo e uma solução possível.  
  
2.  Se o erro persistir, colete informações sobre as circunstâncias e notificar o Microsoft Product Support services.

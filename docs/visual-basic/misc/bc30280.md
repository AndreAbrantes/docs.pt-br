# <a name="enum-39ltenumnamegt39-must-contain-at-least-one-member"></a>Enum &#39; &lt;enumname&gt;&#39; deve conter pelo menos um membro
Uma enumeração vazia ocorre. Pelo menos um membro deve ser declarado entre a `Enum` instrução e o `End Enum` instrução.  
  
 **ID do erro:** BC30280  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicione uma declaração de membro para a enumeração.

# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="3dd26-102">Um literal do XML não pode aparecer aqui, a menos que seja colocada entre parênteses</span><span class="sxs-lookup"><span data-stu-id="3dd26-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>

<span data-ttu-id="3dd26-103">Uma declaração literal XML é usada em uma expressão em um local ambíguo para o compilador Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="3dd26-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="3dd26-104">Ou seja, o compilador Visual Basic não pode determinar se o caractere menor que (<) é destinado como um operador comparativa ou o início de um literal XML.</span><span class="sxs-lookup"><span data-stu-id="3dd26-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="3dd26-105">O código a seguir mostra um exemplo:</span><span class="sxs-lookup"><span data-stu-id="3dd26-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="3dd26-106">**ID do erro:** BC31198</span><span class="sxs-lookup"><span data-stu-id="3dd26-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3dd26-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="3dd26-107">To correct this error</span></span>

                  Select element

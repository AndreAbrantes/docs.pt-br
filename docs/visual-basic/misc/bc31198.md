<span data-ttu-id="40479-103">Uma declaração literal XML é usada em uma expressão em um local que é ambíguo para o compilador do Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="40479-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="40479-104">Ou seja, o compilador do Visual Basic não pode determinar se o menor-que o caractere (<) destina-se como um operador de comparação ou o início de um literal XML.</span><span class="sxs-lookup"><span data-stu-id="40479-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="40479-105">O código a seguir mostra um exemplo:</span><span class="sxs-lookup"><span data-stu-id="40479-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="40479-106">**ID do erro:** BC31198</span><span class="sxs-lookup"><span data-stu-id="40479-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="40479-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="40479-107">To correct this error</span></span>

                  Select element

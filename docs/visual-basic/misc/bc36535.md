# <a name="instance-members-and-39me39-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="a37f0-102">Membros de instância e 'Me' não pode ser usado em uma expressão de consulta</span><span class="sxs-lookup"><span data-stu-id="a37f0-102">Instance members and &#39;Me&#39; cannot be used in a query expression</span></span>
<span data-ttu-id="a37f0-103">Uma consulta LINQ em uma `Structure` inclui uma referência a `Me` ou a um membro de instância da estrutura.</span><span class="sxs-lookup"><span data-stu-id="a37f0-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="a37f0-104">As referências a `Me` ou membros de instância não são permitidos em expressões de consulta dentro de um `Structure`.</span><span class="sxs-lookup"><span data-stu-id="a37f0-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="a37f0-105">**ID do erro:** BC36535</span><span class="sxs-lookup"><span data-stu-id="a37f0-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a37f0-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="a37f0-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="a37f0-107">Criar uma cópia de membro de instância ou o valor retornado por referência a `Me` e usar a cópia na expressão de consulta, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="a37f0-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```

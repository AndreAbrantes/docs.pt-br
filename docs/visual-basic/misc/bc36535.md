# <a name="instance-members-and-me-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="75bfd-102">Os membros da instância e 'Me' não podem ser usados em uma expressão de consulta</span><span class="sxs-lookup"><span data-stu-id="75bfd-102">Instance members and 'Me' cannot be used in a query expression</span></span>
<span data-ttu-id="75bfd-103">Uma consulta LINQ em um `Structure` inclui uma referência a `Me` ou a um membro de instância da estrutura.</span><span class="sxs-lookup"><span data-stu-id="75bfd-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="75bfd-104">Referências a `Me` ou membros de instância não são permitidos em expressões de consulta em um `Structure` .</span><span class="sxs-lookup"><span data-stu-id="75bfd-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="75bfd-105">**ID do erro:** BC36535</span><span class="sxs-lookup"><span data-stu-id="75bfd-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="75bfd-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="75bfd-106">To correct this error</span></span>  
  
1. <span data-ttu-id="75bfd-107">Crie uma cópia do membro da instância ou o valor retornado pela referência para `Me` e use a cópia na expressão de consulta, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="75bfd-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```

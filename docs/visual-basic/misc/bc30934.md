# <a name="conversion-from-39lttype1gt39-to-39lttype2gt39-cannot-occur-in-a-constant-expression-used-as-an-argument-to-an-attribute"></a><span data-ttu-id="4d624-102">Conversão de '&lt;type1&gt;'para'&lt;type2&gt;' não pode ocorrer em uma expressão constante usada como um argumento para um atributo</span><span class="sxs-lookup"><span data-stu-id="4d624-102">Conversion from &#39;&lt;type1&gt;&#39; to &#39;&lt;type2&gt;&#39; cannot occur in a constant expression used as an argument to an attribute</span></span>
<span data-ttu-id="4d624-103">Uma expressão usada para um argumento de atributo avalia para um tipo de dados diferente do parâmetro de atributo correspondente, e [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] não permite a conversão de tipos necessária para argumentos de atributo.</span><span class="sxs-lookup"><span data-stu-id="4d624-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="4d624-104">Um atributo fornece metadados para o elemento para que é aplicado, e o compilador deve ser capaz de construir todos os metadados em tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="4d624-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="4d624-105">Por esse motivo, cada atributo deve usar valores que são constantes no tempo de compilação e, portanto, cada argumento de atributo deve ser avaliada como um valor constante de tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="4d624-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="4d624-106">Determinadas conversões de tipos não podem produzir valores que são constantes no tempo de compilação.</span><span class="sxs-lookup"><span data-stu-id="4d624-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="4d624-107">Por exemplo, converter um `String` para um `Double` ou um `Date` depende da configuração de localidade no tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="4d624-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="4d624-108">Outras conversões, como uma matriz de um tipo derivado para uma matriz de `Object`, apresentam uma variedade de problemas que não permitem que o compilador as permita em argumentos de atributo.</span><span class="sxs-lookup"><span data-stu-id="4d624-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>  
  
 <span data-ttu-id="4d624-109">**ID do erro:** BC30934</span><span class="sxs-lookup"><span data-stu-id="4d624-109">**Error ID:** BC30934</span></span>

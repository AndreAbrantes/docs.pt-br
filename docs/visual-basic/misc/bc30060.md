# <a name="conversion-from-lttype1gt-to-lttype2gt-cannot-occur-in-a-constant-expression"></a>Conversão de '&lt;type1&gt;'para'&lt;type2&gt;' não pode ocorrer em uma expressão constante
A expressão de inicialização em uma `Const` instrução é avaliada como um tipo de dados não pode ser convertido para o tipo declarado da constante.  
  
 **ID do erro:** BC30060  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Inicialize a constante com uma expressão de um tipo de dados que pode ser convertido para o tipo declarado para a constante.

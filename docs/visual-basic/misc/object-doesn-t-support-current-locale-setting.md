# <a name="object-doesnt-support-current-locale-setting-visual-basic"></a>Objeto não dá suporte à configuração de localidade atual (Visual Basic)
Você tentou acessar um objeto que não oferece suporte a configuração de localidade para o projeto atual.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique as configurações de localidade que o objeto dá suporte.

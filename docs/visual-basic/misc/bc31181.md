# <a name="attribute-value-is-not-valid-expecting-39value39"></a>Valor de atributo não é válido. esperando &#39;valor&#39;
Foi especificado um valor inválido para um atributo em um documento XML. Valores de atributo válidos para um literal de documento XML são especificados na especificação XML 1.0.  
  
 **ID do erro:** BC31181  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atualize o valor de atributo para um valor válido, conforme especificado na especificação XML 1.0.

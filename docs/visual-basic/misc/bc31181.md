# <a name="attribute-value-is-not-valid-expecting-value"></a>Valor do atributo não é válido. esperando 'value'
Um valor inválido para um atributo em um literal de documento XML foi especificado. Valores de atributo válidos para um literal de documento XML são definidos na especificação XML 1.0.  
  
 **ID do erro:** BC31181  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atualize o valor do atributo para um valor válido, conforme especificado na especificação XML 1.0.

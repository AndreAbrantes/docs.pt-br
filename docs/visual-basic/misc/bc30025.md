# <a name="property-missing-39end-property39"></a>Propriedade ausente &#39;End Property&#39;
Um `End` instrução deve corresponder ao bloco encerra. Um `Property` definição não foi terminada corretamente.  
  
 **ID do erro:** BC30025  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Property` instrução até o final da definição de propriedade.

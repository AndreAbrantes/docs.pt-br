# <a name="property-missing-end-property"></a>Propriedade ausente 'End Property'
Um `End` instrução deve corresponder ao bloco em que ele é encerrado. Um `Property` definição não foi terminada corretamente.  
  
 **ID do erro:** BC30025  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Property` instrução até o final da definição de propriedade.

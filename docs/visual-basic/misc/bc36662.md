# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-39ltdelegatenamegt39"></a><span data-ttu-id="ccd2d-102">Option Strict On não permite restringir em conversões implícitas de tipo entre a expressão lambda e o delegado '&lt;delegatename&gt;'</span><span class="sxs-lookup"><span data-stu-id="ccd2d-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate &#39;&lt;delegatename&gt;&#39;</span></span>
<span data-ttu-id="ccd2d-103">Com `Option Strict` , você não pode ter uma conversão de restrição entre o tipo de dados de um parâmetro em um representante e o parâmetro correspondente de uma expressão lambda atribuído a uma variável do tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="ccd2d-104">Por exemplo, no código a seguir, delegar `Del` tem um parâmetro do tipo `Integer`.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="ccd2d-105">Portanto, o parâmetro correspondente de qualquer expressão lambda atribuído a uma variável do tipo `Del` pode ser um `Integer` ou qualquer tipo de dados para o qual há uma conversão de ampliação de `Integer`.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="ccd2d-106">**ID do erro:** BC36662</span><span class="sxs-lookup"><span data-stu-id="ccd2d-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ccd2d-107">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="ccd2d-107">To correct this error</span></span>  
  
-   <span data-ttu-id="ccd2d-108">Altere o tipo de dados do parâmetro em que o representante ou a expressão lambda para que a relação de ampliação necessária exista.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
-   <span data-ttu-id="ccd2d-109">Não Especifica tipos de dados de parâmetro na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="ccd2d-110">Tipos serão inferidos dos parâmetros correspondentes no representante.</span><span class="sxs-lookup"><span data-stu-id="ccd2d-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"  
```

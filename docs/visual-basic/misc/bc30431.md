# <a name="39end-property39-must-be-preceded-by-a-matching-39property39"></a>&#39; Propriedade end &#39; deve ser precedido por uma correspondência &#39; Propriedade &#39;
Um `End Property` instrução aparece no seu código com nenhuma correspondência `Property` declaração anterior.  
  
 **ID do erro:** BC30431  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `End Property` instrução se ela é redundante.  
  
-   Forneça a `Property` procedimento se um estiver ausente.  
  
-   Mover o `End Property` até o local apropriado no código.

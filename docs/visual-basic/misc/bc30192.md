# <a name="end-of-parameter-list-expected"></a>Final da lista de parâmetros esperada
Um `ParamArray` especificação é seguida por um ou mais outras especificações de argumentos em uma declaração de procedimento.  
  
 **ID do erro:** BC30192  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `ParamArray` até o final da lista de argumentos.

# <a name="end-of-parameter-list-expected"></a>Fim da lista de parâmetros esperada
Um `ParamArray` especificação é seguida por um ou mais outras argumento especificações na declaração de procedimento.  
  
 **ID do erro:** BC30192  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `ParamArray` ao final da lista de argumentos.

# <a name="modifier-is-not-valid-on-an-interface-declaration"></a>'\<Modifier > ' não é válido em uma declaração de interface
Você usou um modificador que não é válido em uma `Interface` declaração. Os únicos modificadores válidos para `Sub`instruções `Function`,, `Property` ou declaradas `Interface` em uma declaração `Overloads` são `Default` as palavras-chave e. Outros modificadores, `Public`como `Friend`, `Private` ,,,,`Protected`, e`Overridable`,não são válidos. `MustOverride` `Shared` `Static` `Overrides`  
  
 **ID do erro:** BC30397  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o modificador.

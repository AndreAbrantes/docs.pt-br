# <a name="modifier-is-not-valid-on-an-interface-declaration"></a>'\<modificador >' não é válido em uma declaração de Interface
Você usou um modificador que não é válido em um `Interface` declaração. Os modificadores só é válidos para `Sub`, `Function`, ou `Property` instruções declaradas em uma `Interface` declaração são os `Overloads` e `Default` palavras-chave. Outros modificadores, tais como `Public`, `Private`, `Friend`, `Protected`, `Shared`, `Static`, `Overrides`, `MustOverride` e `Overridable`, não são válidos.  
  
 **ID do erro:** BC30397  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o modificador.

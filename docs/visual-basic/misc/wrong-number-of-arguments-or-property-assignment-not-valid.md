# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>Número incorreto de argumentos ou atribuição de propriedade não é válida
Foi tentada uma atribuição que não é válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o número de argumentos que você forneceu corresponde ao número exigido pelo destino.  
  
2.  Verifique a atribuição de propriedade.

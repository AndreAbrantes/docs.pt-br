# <a name="unable-to-load-referenced-library-39ltfilenamegt39-lterrorgt"></a>Não é possível carregar a biblioteca referenciada '&lt;filename&gt;': &lt;erro&gt;
Uma biblioteca referenciada por este projeto não pôde ser carregada.  
  
 **ID do erro:** BC31011  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verificar se o arquivo está no local especificado e não está bloqueado por outro aplicativo.

# <a name="unable-to-load-referenced-library-ltfilenamegt-lterrorgt"></a>Não é possível carregar a biblioteca referenciada '&lt;filename&gt;': &lt;erro&gt;
Não foi possível carregar uma biblioteca referenciada por este projeto.  
  
 **ID do erro:** BC31011  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verificar se o arquivo está no local especificado e não está bloqueado por outro aplicativo.

# <a name="unable-to-load-referenced-library-filename-error"></a>Não é possível carregar a biblioteca referenciada ' \<filename> ': \<error>

Não foi possível carregar uma biblioteca referenciada por este projeto.  
  
 **ID do erro:** BC31011  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se o arquivo está no local especificado e não está bloqueado por outro aplicativo.

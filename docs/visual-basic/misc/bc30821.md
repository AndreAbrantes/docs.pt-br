# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39; não é declarada
'\<rsetstmt >' não está declarado. Não há suporte para instruções de RSet; Use Microsoft.  
  
 Várias funções que foram intrínsecas do Visual Basic em versões anteriores foram movidas para o <xref:Microsoft.VisualBasic?displayProperty=nameWithType> namespace. Isso torna a funcionalidade mais disponível para todas as linguagens de programação.  
  
 **ID do erro:** BC30821  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `RSet` funcionar a `Microsoft.VisualBasic` namespace em vez disso.

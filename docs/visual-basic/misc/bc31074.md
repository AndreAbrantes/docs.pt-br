# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Construtor delegate para o tipo &#39; &lt;typename&gt;&#39;, necessário para implementação de delegates, não pode ser encontrado
O assembly que contém o tipo pode ser danificado ou não é válido.  
  
 **ID do erro:** BC31074  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o assembly que contém esse tipo é válido.  
  
2.  Compile o programa novamente para verificar se o erro persiste.  
  
3.  Se o erro persistir, salve seu trabalho e reinicie o Visual Studio.

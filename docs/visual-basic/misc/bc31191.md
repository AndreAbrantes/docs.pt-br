# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Erro ao incorporar o manifesto Win32: &lt;de manifesto&gt;
O arquivo especificado para o `/win32manifest` opção de compilador não pode ser inserida. Isso pode ser resultado de privilégios de segurança insuficientes para o arquivo de manifesto.  
  
 **ID do erro:** BC31191  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que a identidade que executa o comando do compilador Visual Basic tem acesso ao arquivo de manifesto Win32.

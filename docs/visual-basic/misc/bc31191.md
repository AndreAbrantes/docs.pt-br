# <a name="error-embedding-win32-manifest-manifest"></a>Erro ao inserir o manifesto Win32: \<manifesto >
O arquivo especificado para o `/win32manifest` opção de compilador não pode ser inserida. Isso pode ser um resultado de privilégios de segurança insuficientes para o arquivo de manifesto.  
  
 **ID do erro:** BC31191  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que a identidade executando o comando do compilador Visual Basic tem acesso ao arquivo de manifesto Win32.

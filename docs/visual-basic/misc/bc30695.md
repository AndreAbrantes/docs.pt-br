# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<TypeName > '\<membername >' deve ser declarado como 'Shadows' porque outro membro com esse nome está declarado 'como Shadows'
Esta propriedade ou procedimento tem o mesmo nome que outra propriedade ou procedimento marcado com o `Shadows` modificador.  
  
 **ID do erro:** BC30695  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Use o `Shadows` modificador.

# <a name="39end-interface39-must-be-preceded-by-a-matching-39interface39"></a>&#39; Interface end &#39; deve ser precedido por uma correspondência &#39; Interface &#39;
Um `End Interface` instrução ocorre sem correspondente `Interface` instrução. `End Interface`deve ser precedida por uma `Interface` instrução.  
  
 **ID do erro:** BC30252  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se precedidos `Interface` instrução está incorreta ou inválidos.  
  
2.  Certifique-se de que o `Interface` membros estão formatados corretamente.

# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>'End Interface' deve ser precedido por um 'Interface' correspondente
Uma `End Interface` declaração ocorre sem um correspondente `Interface` instrução. `End Interface` deve ser precedida por uma `Interface` instrução.  
  
 **ID do erro:** BC30252  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se precedidos `Interface` instrução está incorreta ou inválidos.  
  
2.  Certifique-se de que o `Interface` membros estão formatados corretamente.

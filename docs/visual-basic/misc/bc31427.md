# <a name="syntax-error-in-conditional-compilation-expression"></a>Erro de sintaxe na expressão de compilação condicional
A expressão de compilação condicional gera um erro de sintaxe não específico.  
  
 **ID do erro:** BC31427  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Examine a documentação para uma compilação condicional e para as palavras-chave usadas na expressão. Compare o exemplo de código na documentação com a linha de origem gera este erro.  
  
2.  Se você não pode identificar a causa do erro, colete informações sobre as circunstâncias e notifique Microsoft Product Support Services.

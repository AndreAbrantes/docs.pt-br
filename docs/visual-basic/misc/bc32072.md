# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="ad995-102">Não é possível implementar a interface '\<interfacename1 >' porque sua implementação poderia entrar em conflito com a implementação de outra interface implementada '\<interfacename2 >' para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="ad995-102">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>
<span data-ttu-id="ad995-103">Uma declaração de classe inclui um `Implements` declaração que especifica duas ou mais interfaces, mas pelo menos uma das interfaces é genérico e duas das implementações pode entrar em conflito para determinados valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="ad995-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="ad995-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="ad995-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="ad995-105">Porque `iFace2` é construído usando `String`, `testClass` deve implementar duas versões do `testSub` com assinaturas idênticas.</span><span class="sxs-lookup"><span data-stu-id="ad995-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="ad995-106">Isso produziria uma ambiguidade sobre a versão que deseja acessar.</span><span class="sxs-lookup"><span data-stu-id="ad995-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="ad995-107">**ID do erro:** BC32072</span><span class="sxs-lookup"><span data-stu-id="ad995-107">**Error ID:** BC32072</span></span>

-   <span data-ttu-id="ad995-109">Altere o argumento de tipo fornecido para a interface genérica, de modo que haja conflito.</span><span class="sxs-lookup"><span data-stu-id="ad995-109">Change the type argument supplied to the generic interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="ad995-110">-ou-</span><span class="sxs-lookup"><span data-stu-id="ad995-110">-or-</span></span>

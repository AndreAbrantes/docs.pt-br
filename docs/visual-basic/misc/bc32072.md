# <a name="cannot-implement-interface-39ltinterfacename1gt39-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-39ltinterfacename2gt39-for-some-type-arguments"></a><span data-ttu-id="3b463-102">Não é possível implementar a interface &#39; &lt;interfacename1&gt;&#39; porque sua implementação poderia entrar em conflito com a implementação de outra interface implementada &#39;&lt; interfacename2&gt;&#39; para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="3b463-102">Cannot implement interface &#39;&lt;interfacename1&gt;&#39; because its implementation could conflict with the implementation of another implemented interface &#39;&lt;interfacename2&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="3b463-103">Uma declaração de classe inclui um `Implements` instrução que especifica duas ou mais interfaces, mas pelo menos uma das interfaces é genérico e duas das implementações pode entrar em conflito para certos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="3b463-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="3b463-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="3b463-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="3b463-105">Porque `iFace2` é construído usando `String`, `testClass` deve implementar duas versões do `testSub` com assinaturas idênticas.</span><span class="sxs-lookup"><span data-stu-id="3b463-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="3b463-106">Isso geraria ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="3b463-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="3b463-107">**ID do erro:** BC32072</span><span class="sxs-lookup"><span data-stu-id="3b463-107">**Error ID:** BC32072</span></span>

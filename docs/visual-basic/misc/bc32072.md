# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="54b1e-102">Não é possível implementar\<a interface ' interfacename1 > ' porque sua implementação pode entrar em conflito com a implementação\<de outra interface implementada ' interfacename2 > ' para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="54b1e-102">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>

<span data-ttu-id="54b1e-103">Uma declaração de classe inclui `Implements` uma instrução que especifica duas ou mais interfaces, mas pelo menos uma das interfaces é genérica e duas das implementações podem entrar em conflito para determinados valores dos argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="54b1e-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>

<span data-ttu-id="54b1e-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="54b1e-104">The following statements can generate this error.</span></span>

```vb
Public Interface iFace1
    Sub testSub(ByVal arg As String)
End Interface
Public Interface iFace2(Of t)
    Sub testSub(ByVal arg As t)
End Interface
Public Class testClass
    Implements iFace1, iFace2(Of String)
End Class
```

<span data-ttu-id="54b1e-105">Como `iFace2` o é construído `String`usando `testClass` , o deve implementar duas `testSub` versões do com assinaturas idênticas.</span><span class="sxs-lookup"><span data-stu-id="54b1e-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="54b1e-106">Isso produziria uma ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="54b1e-106">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="54b1e-107">**ID do erro:** BC32072</span><span class="sxs-lookup"><span data-stu-id="54b1e-107">**Error ID:** BC32072</span></span>

 <span data-ttu-id="8ca5b-104">O código a seguir pode gerar essa mensagem de erro.</span><span class="sxs-lookup"><span data-stu-id="8ca5b-104">The following code can generate this error message.</span></span>  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="8ca5b-105">No código anterior, classe `c1` herda da classe `c4`, mas `c4` está aninhado em `c3`, que herda de `c2`, aninhado em `c1`.</span><span class="sxs-lookup"><span data-stu-id="8ca5b-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>

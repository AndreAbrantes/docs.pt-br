# <a name="type-character-cannot-be-used-in-a-39sub39-declaration-because-a-39sub39-doesn39t-return-a-value"></a>Caractere de tipo não pode ser usado em um &#39; Sub &#39; declaração porque um &#39; Sub &#39; &#39; um valor de retorno de t
Um `Sub` procedimento, como um `Function` procedimento, é um procedimento separado que pode ter argumentos e executar uma série de instruções. Ao contrário de um `Function` procedimento, um `Sub` não retorna um valor e, portanto, não pode conter uma declaração de tipo.  
  
 **ID do erro:** BC30303  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar o `Sub` procedimento para uma `Function` procedimento.

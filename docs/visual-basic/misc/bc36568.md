# <a name="ltelementnamegt-is-not-a-method-parameter"></a>'&lt;elementname&gt;' não é um parâmetro de método
Uma chamada de procedimento fornece um argumento adicional que não está definido pelo método.  
  
 **ID do erro:** BC36568  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento supérfluo da lista de argumentos.

# <a name="white-space-may-not-appear-here"></a>Espaço em branco não pode aparecer aqui
Um ou mais espaços são incluídos no nome de um literal de elemento XML.  
  
 **ID do erro:** BC31177  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o nome do elemento XML literal a ou mais espaços.

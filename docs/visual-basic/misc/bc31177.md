# <a name="white-space-may-not-appear-here"></a>Espaço em branco não pode aparecer aqui
Um ou mais espaços são incluídos no nome de um elemento XML literal.  
  
 **ID do erro:** BC31177  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o espaço ou espaços do nome do elemento XML literal.

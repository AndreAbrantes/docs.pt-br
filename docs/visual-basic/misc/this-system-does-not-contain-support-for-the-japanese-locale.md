# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Este sistema não há suporte para a localidade japonês
Você está tentando usar o `VbStrConv.Japanese` membro de enumeração, mas seu sistema não dá suporte para a localidade japonês.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Atualize a máquina para oferecer suporte a localidade japonês ou executar o aplicativo em um computador que dá suporte à localidade.

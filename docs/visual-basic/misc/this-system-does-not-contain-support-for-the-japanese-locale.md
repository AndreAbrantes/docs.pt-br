# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Este sistema não contém suporte para a localidade japonesa
Você está tentando usar o membro de enumeração `VbStrConv.Japanese`, mas o sistema não oferece suporte à localidade japonesa.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Atualize o computador para dar suporte à localidade japonesa ou execute o aplicativo em um computador que ofereça suporte à localidade.

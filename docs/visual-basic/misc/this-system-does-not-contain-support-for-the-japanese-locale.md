# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Este sistema não tem suporte para localidade japonês
Você está tentando usar o `VbStrConv.Japanese` membro de enumeração, mas o sistema não dá suporte para localidade japonês.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade japonês, ou execute o aplicativo em uma máquina que dá suporte a localidade.

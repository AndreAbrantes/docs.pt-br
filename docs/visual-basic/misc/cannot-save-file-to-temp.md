<span data-ttu-id="95e99-103">Um componente não pode localizar um diretório chamado TEMP ou a unidade ou partição que contém o diretório TEMP não tem espaço suficiente para salvar as informações.</span><span class="sxs-lookup"><span data-stu-id="95e99-103">Either a component cannot find a directory named TEMP, or the drive or partition containing the TEMP directory lacks sufficient space to save information.</span></span>

# <a name="xml-name-expected"></a>Nome XML esperado
Um literal XML ou uma propriedade de eixo XML foi especificada sem um nome.  
  
 **ID do erro:** BC31146  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione um nome para o literal XML ou uma propriedade de eixo XML.

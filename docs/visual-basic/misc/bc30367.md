# <a name="class-ltclassnamegt-cannot-be-indexed-because-it-has-no-default-property"></a>Classe de&lt;classname&gt;' não pode ser indexada porque não tem nenhuma propriedade padrão
Falha ao fornecer uma propriedade padrão para uma classe.  
  
 **ID do erro:** BC30367  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifica uma propriedade como a propriedade padrão, começando sua declaração com o `Default` palavra-chave.

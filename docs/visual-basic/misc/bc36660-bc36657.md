<span data-ttu-id="c1cf6-103">Não é possível inferir os tipos de dados dos parâmetros de tipo no método\<' MethodName > ' com base nesses argumentos porque eles não são convertidos no mesmo tipo.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="c1cf6-104">A especificação explícita dos tipos de dados pode corrigir esse erro.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="c1cf6-105">Foi feita uma tentativa de usar a inferência de tipos para determinar o tipo de dados ou os tipos de parâmetro de tipo ou parâmetros ao avaliar uma chamada para um procedimento genérico.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="c1cf6-106">O compilador não pôde localizar um tipo de dados que atenda às restrições de todos os argumentos.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="c1cf6-107">Portanto, ele relatou esse erro.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="c1cf6-108">Ao especificar argumentos não é uma opção (por exemplo, para operadores de consulta em expressões de consulta), a mensagem de erro é exibida sem a segunda sentença.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="c1cf6-109">O código a seguir demonstra o erro.</span><span class="sxs-lookup"><span data-stu-id="c1cf6-109">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module1
    Sub Main()

        '' Not valid. Integer and Date do not convert to the same type.
        'targetMethod(19, #3/4/2007#)

    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

End Module
```

<span data-ttu-id="c1cf6-110">**ID do erro:** BC36660 e BC36657</span><span class="sxs-lookup"><span data-stu-id="c1cf6-110">**Error ID:** BC36660 and BC36657</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="c1cf6-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="c1cf6-111">To correct this error</span></span>

  ```vb

  ```

- <span data-ttu-id="c1cf6-113">Você pode especificar um tipo de dados para o parâmetro de tipo ou parâmetros para os quais os argumentos são convertidos, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="c1cf6-113">You may be able to specify a data type for the type parameter or parameters to which the arguments convert, as shown in the following code:</span></span>


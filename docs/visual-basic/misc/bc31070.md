# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Classe em uma interface não pode ser declarada '&lt;especificador&gt;'
Um modificador inválido foi especificado com uma classe declarada em uma interface.  
  
 **ID do erro:** BC31070  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remove modifiers such as `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, or `Overridable`.

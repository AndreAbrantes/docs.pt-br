# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>'DllImportAttribute' não pode ser aplicado a Sub, Function ou Operator com um corpo vazio
O `DllImportAttribute` atributo foi aplicado a um `Sub`, `Function`, ou `Operator` que não está vazio.  
  
 **ID do erro:** BC31522  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova todo o código do `Sub`, `Function`, ou `Operator` para usar este atributo.

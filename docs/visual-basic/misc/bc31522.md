# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>'DllImportAttribute' não pode ser aplicado a um Sub, função ou operador com um corpo não vazio
O `DllImportAttribute` atributo foi aplicado a um `Sub`, `Function`, ou `Operator` que não está vazio.  
  
 **ID do erro:** BC31522  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova todos os códigos do `Sub`, `Function`, ou `Operator` para usar este atributo.

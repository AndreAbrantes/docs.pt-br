# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>Elemento de matriz referenciado tem um valor de &#39; Nada &#39;
O elemento da matriz que está sendo referenciado não foi definido, ou ele foi previamente configurado para `Nothing`.  
  
 **ID do erro:** BC30744  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se os valores na matriz estão sendo definidos corretamente.

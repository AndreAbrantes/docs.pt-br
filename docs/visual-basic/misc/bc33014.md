# <a name="operator-39ltoperatorgt39-must-have-one-parameter"></a>Operador &#39; &lt;operador&gt; &#39; deve ter um parâmetro
Um operador unário é definido sem parâmetros ou com mais de um parâmetro.  
  
 Um operador unário deve ter exatamente um parâmetro.  
  
 **ID do erro:** BC33014  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Ajuste a definição para especificar exatamente um parâmetro.  
  
-   Se você precisar de dois parâmetros, você deve definir um operador binário.  
  
-   Se você precisar sem parâmetros ou com mais de dois, você deve usar o [instrução Function](../../visual-basic/language-reference/statements/function-statement.md) para definir um `Function` procedimento em vez de um operador.

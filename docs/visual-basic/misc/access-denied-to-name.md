# <a name="access-denied-to-name"></a>Acesso negado para '\<nome >'
Um dispositivo, arquivo ou disco não está acessível porque as permissões de segurança do seu computador não são suficientes.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que você tenha as permissões necessárias.

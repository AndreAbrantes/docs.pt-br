# <a name="access-denied-to-39ltnamegt39"></a>Acesso negado a &#39; &lt;nome&gt;&#39;
Um dispositivo, arquivo ou disco não está acessível porque as permissões de segurança do seu computador são insuficientes.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que você tem as permissões necessárias.

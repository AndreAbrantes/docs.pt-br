# <a name="index-39ltindexnumbergt39-for-dimension-39ltdimensionnumbergt39-is-out-of-range"></a>Índice '&lt;número_de_índice&gt;'da dimensão'&lt;dimensionnumber&gt;' está fora do intervalo
Um dos índices excede o intervalo definido.  
  
 **ID do erro:** BC30702  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a lógica do procedimento de forma que não excedam os limites da matriz.  
  
-   Redefina a matriz para permitir uma maior variedade de valores.

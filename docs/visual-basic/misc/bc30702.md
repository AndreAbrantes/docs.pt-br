# <a name="index-39ltindexnumbergt39-for-dimension-39ltdimensionnumbergt39-is-out-of-range"></a>Índice &#39; &lt;número_de_índice&gt;&#39; para dimensão &#39;&lt; dimensionnumber&gt;&#39; está fora do intervalo
Um dos índices excede o intervalo definido.  
  
 **ID do erro:** BC30702  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a lógica do procedimento de forma que não excedam os limites da matriz.  
  
-   Redefina a matriz para permitir uma maior variedade de valores.

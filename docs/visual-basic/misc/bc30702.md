# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>Índice '\<número_de_índice >' para a dimensão '\<dimensionnumber >' está fora do intervalo
Um dos índices excede o intervalo definido.  
  
 **ID do erro:** BC30702  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere a lógica do procedimento para que os limites da matriz não são excedidos.  
  
- Redefina a matriz para permitir uma maior variedade de valores.

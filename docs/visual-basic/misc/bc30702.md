# <a name="index-ltindexnumbergt-for-dimension-ltdimensionnumbergt-is-out-of-range"></a>Índice '&lt;número_de_índice&gt;'para a dimensão'&lt;dimensionnumber&gt;' está fora do intervalo
Um dos índices excede o intervalo definido.  
  
 **ID do erro:** BC30702  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a lógica do procedimento para que os limites da matriz não são excedidos.  
  
-   Redefina a matriz para permitir uma maior variedade de valores.

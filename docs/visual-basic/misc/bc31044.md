# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Eventos declarados com uma cláusula 'As' devem ter um tipo de delegado
Foi especificado um tipo que não é um delegado.  
  
 **ID do erro:** BC31044  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `As` cláusula.  
  
2.  Especifique um delegado com o `As` cláusula.

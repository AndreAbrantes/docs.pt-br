# <a name="parameter-39ltparameternamegt39-of-39ltmethodnamegt39-already-has-a-matching-argument"></a>Parâmetro &#39; &lt;parametername&gt;&#39; do &#39;&lt; MethodName&gt;&#39; já tem um argumento correspondente
Um argumento é passado por nome mais de uma vez na mesma chamada de procedimento.  
  
 **ID do erro:** BC30274  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se um dos nomes de argumento está incorreto.  
  
-   Remova o argumento duplicado da lista de argumentos.

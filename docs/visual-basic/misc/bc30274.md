# <a name="parameter-ltparameternamegt-of-ltmethodnamegt-already-has-a-matching-argument"></a>Parâmetro '&lt;parametername&gt;'de'&lt;methodname&gt;' já tem um argumento correspondente
Um argumento é passado por nome mais de uma vez na mesma chamada de procedimento.  
  
 **ID do erro:** BC30274  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se um dos nomes de argumento está incorreto.  
  
-   Remova o argumento duplicado da lista de argumentos.

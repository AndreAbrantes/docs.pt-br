# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39; MyClass &#39; não pode ser usado fora de uma classe
Você usou o `MyClass` palavra-chave fora de uma estrutura de classe.  
  
 **ID do erro:** BC30470  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `MyClass` declaração para a estrutura de classe.

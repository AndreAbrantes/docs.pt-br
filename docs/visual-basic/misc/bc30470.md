# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>'MyClass' não pode ser usado fora de uma classe
Você usou o `MyClass` palavra-chave fora de uma estrutura de classe.  
  
 **ID do erro:** BC30470  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova o `MyClass` declaração à sua estrutura de classe.

# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>Módulos não podem ser declarados &#39; &lt;especificador&gt;&#39;
Um módulo foi declarado com um modificador de acesso inválido.  
  
 **ID do erro:** BC31052  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o modificador de acesso com um modificador de acesso válido, como `Public` ou `Friend`.

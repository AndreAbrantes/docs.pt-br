# <a name="modules-cannot-be-declared-specifier"></a>Módulos não podem ser declarados como '\<especificador >'
Um módulo foi declarado com um modificador de acesso inválido.  
  
 **ID do erro:** BC31052  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua o modificador de acesso com um modificador de acesso válido, como `Public` ou `Friend`.

# <a name="label-ltlabelnamegt-is-not-defined"></a>Rótulo '&lt;labelname&gt;' não está definido
Um rótulo de linha ou o número de linha é referenciado mas não definido dentro do escopo da referência. O rótulo deve ser dentro do procedimento que contém a referência.  
  
 **ID do erro:** BC30132  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reestruture o código de modo que o rótulo de linha e a referência estejam no mesmo procedimento.

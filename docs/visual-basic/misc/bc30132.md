# <a name="label-39ltlabelnamegt39-is-not-defined"></a>Rótulo &#39; &lt;labelname&gt;&#39; não está definido
Um rótulo de linha ou um número de linha é referenciado mas não definido dentro do escopo da referência. O rótulo deve estar dentro do procedimento que contém a referência.  
  
 **ID do erro:** BC30132  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reestruture o código para que o rótulo de linha e a referência estejam no mesmo procedimento.

# <a name="39ltmembernamegt39-cannot-be-declared-39shadows39-outside-of-a-class-structure-or-interface"></a>&#39; &lt;membername&gt;&#39; não pode ser declarado como &#39; Sombras &#39; fora de uma classe, estrutura ou interface
O `Shadows` palavra-chave aparece em uma declaração no namespace, módulo ou nível de arquivo. Sombreamento é útil somente em um elemento de programação que pode herdar de um elemento base.  
  
 **ID do erro:** BC32200  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `Shadows` palavra-chave, ou mover a declaração dentro de uma classe, estrutura ou interface.

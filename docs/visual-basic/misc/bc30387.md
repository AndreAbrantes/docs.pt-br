# <a name="class-classname-must-declare-a-sub-new-because-its-base-class-classname2-does-not-have-an-accessible-sub-new-that-can-be-called-with-no-arguments"></a><span data-ttu-id="7f5ce-102">Classe de\<classname >' deve declarar um 'Sub New' porque sua classe base\<classname2 >' não tem um 'Sub New' acessível que pode ser chamado sem argumentos</span><span class="sxs-lookup"><span data-stu-id="7f5ce-102">Class '\<classname>' must declare a 'Sub New' because its base class '\<classname2>' does not have an accessible 'Sub New' that can be called with no arguments</span></span>
<span data-ttu-id="7f5ce-103">Uma classe derivada não declara um construtor, e o Visual Basic não é possível gerar um porque não há nenhum construtor de classe base pode chamar.</span><span class="sxs-lookup"><span data-stu-id="7f5ce-103">A derived class does not declare a constructor, and Visual Basic cannot generate one because there is no base class constructor it can call.</span></span>

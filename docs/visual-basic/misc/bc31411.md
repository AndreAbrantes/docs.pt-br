# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39; &lt;typename&gt;&#39; deve ser declarado como &#39; MustInherit &#39; porque contém métodos declarados como &#39; MustOverride &#39;
Tipos que contêm `MustOverride` membros devem ser declarados `MustInherit`.  
  
 **ID do erro:** BC31411  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o `MustInherit` modificador no tipo.

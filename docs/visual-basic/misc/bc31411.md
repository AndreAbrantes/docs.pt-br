# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<typename >' deve ser declarado 'como MustInherit' porque ela contém métodos declarados como 'MustOverride'
Tipos que contêm `MustOverride` membros devem ser declarados `MustInherit`.  
  
 **ID do erro:** BC31411  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicionar o `MustInherit` modificador no tipo.

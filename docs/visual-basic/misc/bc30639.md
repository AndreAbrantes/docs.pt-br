# <a name="properties-cannot-be-declared-modifiername"></a>Propriedades não podem ser declaradas como '\<modifiername >'
Há um erro na sintaxe usada no `Property` declaração. Este erro geralmente indica que uma palavra-chave especificada, tais como `WithEvents`, não pode ser usado em um `Property` declaração.  
  
 **ID do erro:** BC30639  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a sintaxe para o `Property` declaração.

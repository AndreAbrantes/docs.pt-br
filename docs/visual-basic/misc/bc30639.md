# <a name="properties-cannot-be-declared-39ltmodifiernamegt39"></a>Propriedades não podem ser declaradas &#39; &lt;modifiername&gt;&#39;
Há um erro na sintaxe usada no `Property` declaração. Este erro geralmente indica que uma palavra-chave especificada, como `WithEvents`, não pode ser usado em um `Property` declaração.  
  
 **ID do erro:** BC30639  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a sintaxe para a `Property` declaração.

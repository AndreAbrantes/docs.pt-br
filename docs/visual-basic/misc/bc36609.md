# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Expressões de consulta não são válidas em janelas de depuração
Uma consulta LINQ foi inserida no **inspecionar** ou **imediato** durante uma sessão de depuração. Consultas não estão disponíveis nas janelas de depuração.  
  
 **ID do erro:** BC36609

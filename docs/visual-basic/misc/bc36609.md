# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Expressões de consulta não são válidas em janelas de depuração
Foi inserida em uma consulta LINQ a **Watch** ou **imediato** janela durante uma sessão de depuração. Consultas não estão disponíveis em janelas de depuração.  
  
 **ID do erro:** BC36609

# <a name="attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Atributo &#39; &lt;attributename&gt; &#39; não é válido: &lt;erro&gt;
O common language runtime não aceita o atributo no contexto atual.  
  
 **ID do erro:** BC30127  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o atributo da declaração ou substituí-lo com um atributo que é válido para o contexto.  
  
2.  Examine a mensagem de erro e tomar as devidas providências.

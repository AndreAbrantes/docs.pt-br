# <a name="keyword-does-not-name-a-type"></a>Palavra-chave não nomeia um tipo
Uma palavra-chave aparece onde um nome de tipo é necessário. O contexto de chamadas para um tipo de dados elementar ou um nome de elemento declarado representando um tipo.  
  
 **ID do erro:** BC30180  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua a palavra-chave com um nome de tipo válido.

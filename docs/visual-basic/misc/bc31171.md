# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Caractere de sequência de &#39; – &#39; não é permitida em um comentário XML
Um literal de comentário XML contém a sequência de caracteres '-', que não é suportado.  
  
 **ID do erro:** BC31171  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover a sequência de caracteres '-' do literal de comentário XML.

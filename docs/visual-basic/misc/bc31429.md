# <a name="elementname-is-ambiguous-because-multiple-kinds-of-members-with-this-name-exist-in-type-typename"></a><span data-ttu-id="04900-102">' \<elementname> ' é ambíguo porque existem vários tipos de membros com esse nome em \<type> ' \<typename> '</span><span class="sxs-lookup"><span data-stu-id="04900-102">'\<elementname>' is ambiguous because multiple kinds of members with this name exist in \<type> '\<typename>'</span></span>
<span data-ttu-id="04900-103">Uma expressão acessa um elemento de programação definido em uma classe, estrutura, módulo ou interface que contém mais de um membro com o mesmo nome.</span><span class="sxs-lookup"><span data-stu-id="04900-103">An expression accesses a programming element defined in a class, structure, module, or interface that contains more than one member with the same name.</span></span>  
  
 <span data-ttu-id="04900-104">A causa mais provável desse erro é a *diferenciação de maiúsculas e minúsculas*.</span><span class="sxs-lookup"><span data-stu-id="04900-104">The most likely cause of this error is *case sensitivity*.</span></span> <span data-ttu-id="04900-105">Os nomes de Visual Basic não diferenciam maiúsculas de minúsculas, o que significa que você pode capitalizar deles de forma diferente em locais diferentes em seu código.</span><span class="sxs-lookup"><span data-stu-id="04900-105">Visual Basic names are case-insensitive, which means you can capitalize them differently at different places in your code.</span></span> <span data-ttu-id="04900-106">Por exemplo, se você definir uma variável com o nome `XYZ` e, posteriormente, acessá-la como `xyz` , o compilador considerará que os dois nomes são equivalentes.</span><span class="sxs-lookup"><span data-stu-id="04900-106">For example, if you define a variable with the name `XYZ` and later access it as `xyz`, the compiler considers the two names to be equivalent.</span></span>  
  
 <span data-ttu-id="04900-107">No entanto, outras linguagens, como [C#](../../csharp/index.yml) e [Visual C++](/cpp/index), diferenciam maiúsculas de minúsculas.</span><span class="sxs-lookup"><span data-stu-id="04900-107">However, other languages, such as [C#](../../csharp/index.yml) and [Visual C++](/cpp/index), are case-sensitive.</span></span> <span data-ttu-id="04900-108">Nesse idioma, `XYZ` e `xyz` não são considerados o mesmo nome.</span><span class="sxs-lookup"><span data-stu-id="04900-108">In such a language, `XYZ` and `xyz` are not considered to be the same name.</span></span> <span data-ttu-id="04900-109">Portanto, uma classe escrita em tal linguagem poderia definir uma variável chamada `XYZ` e uma propriedade chamada `xyz` .</span><span class="sxs-lookup"><span data-stu-id="04900-109">Therefore, a class written in such a language could define a variable named `XYZ` and a property named `xyz`.</span></span> <span data-ttu-id="04900-110">O Common Language Runtime (CLR) preserva a diferenciação de maiúsculas e minúsculas nos assemblies.</span><span class="sxs-lookup"><span data-stu-id="04900-110">The common language runtime (CLR) preserves case sensitivity in assemblies.</span></span> <span data-ttu-id="04900-111">No entanto, se um aplicativo Visual Basic acessar um assembly com nomes `XYZ` e `xyz` , eles aparecerão como o mesmo nome.</span><span class="sxs-lookup"><span data-stu-id="04900-111">However, if a Visual Basic application accesses an assembly with names `XYZ` and `xyz`, they appear as the same name.</span></span>  
  
 <span data-ttu-id="04900-112">**ID do erro:** BC31429</span><span class="sxs-lookup"><span data-stu-id="04900-112">**Error ID:** BC31429</span></span>

1. <span data-ttu-id="04900-114">Se você tiver controle sobre o código-fonte do tipo de definição, considere renomear os membros para que eles difiram mais do que apenas maiúsculas e minúsculas.</span><span class="sxs-lookup"><span data-stu-id="04900-114">If you have control over the source code of the defining type, consider renaming the members so that they differ by more than only casing.</span></span> <span data-ttu-id="04900-115">Isso pode não ser possível se o tipo de definição já tiver sido publicado e estiver sendo usado por outros aplicativos.</span><span class="sxs-lookup"><span data-stu-id="04900-115">This might not be possible if the defining type has already been published and is being used by other applications.</span></span>

# <a name="could-not-find-library-39ltlibrarynamegt39"></a>Não foi possível localizar a biblioteca &#39; &lt;libraryname&gt;&#39;
Não foi possível encontrar a biblioteca especificada.  
  
 Esse problema pode ocorrer quando o arquivo de compilação do prompt de comando e o uso de uma resposta (. rsp) faz referência a vários assemblies  
  
 **ID do erro:** BC2017  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o nome da biblioteca para verificar se que ela está correta.

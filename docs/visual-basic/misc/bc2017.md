# <a name="could-not-find-library-libraryname"></a>Não foi possível encontrar biblioteca '\<libraryname >'
A biblioteca especificada não pôde ser encontrada.  
  
 Esse problema pode ocorrer quando o arquivo de compilação do prompt de comando e o uso de uma resposta (. rsp) que faz referência a muitos assemblies  
  
 **ID do erro:** BC2017  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o nome da biblioteca para garantir que ele está correto.

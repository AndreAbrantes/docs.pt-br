# <a name="implicit-conversion-from-39lttypename1gt39-to-39lttypename2gt39"></a>Conversão implícita de &#39; &lt;typename1&gt;&#39; para &#39;&lt; typename2&gt;&#39;
Uma expressão ou uma instrução de atribuição leva um valor de um tipo de dados e o converte em outro tipo. Como nenhuma palavra-chave de conversão for usado, a conversão é *implícita*.  
  
 Por padrão, esta mensagem é um aviso. Para obter informações sobre como ocultar avisos ou tratar avisos como erros, consulte [Configurando avisos no Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID do erro:** BC42016  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Se possível, use os valores do mesmo tipo de dados, portanto [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] não precisa fazer nenhuma conversão.  
  
-   Use `CType` ou um dos outros conversão palavras-chave para que a conversão seja *explícita*.

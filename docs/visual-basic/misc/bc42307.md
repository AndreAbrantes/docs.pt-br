# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parâmetro de comentário XML &#39; &lt;parâmetro&gt; &#39; não coincide com um parâmetro em correspondente &lt;palavra-chave&gt; instrução
Parâmetro de comentário XML \<parâmetro > não coincide com um parâmetro em correspondente \<palavra-chave > demonstrativo. O comentário XML será ignorado.  
  
 Uma marca de comentário XML foi especificada para alguns, mas nem todos os parâmetros em um método.  
  
 **ID do erro:** BC42307  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça a marca de comentário XML para todos os parâmetros no método.

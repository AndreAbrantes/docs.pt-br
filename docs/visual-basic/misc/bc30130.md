# <a name="module-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Atributo de módulo '&lt;attributename&gt;' não é válido: &lt;erro&gt;
O common language runtime não aceita o atributo em um módulo.

**ID do erro:** BC30130

## <a name="to-correct-this-error"></a>Para corrigir este erro

1. Remova o atributo da declaração ou substituí-lo com um atributo que é válido para o contexto.

2. Examine a mensagem de erro entre aspas e tomar as devidas providências.

# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Instruções de loop não são válidas na janela imediata
`Loop`instruções não podem ser usadas no **imediato** janela.  
  
 **Error ID:** BC30708  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Loop` instruções do código de depuração.

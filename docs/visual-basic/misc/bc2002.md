# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Arquivo de origem &lt;filename&gt; especificado várias vezes
Um arquivo foi especificado várias vezes.  
  
 **ID do erro:** BC2002  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova os especificadores de arquivo redundantes.

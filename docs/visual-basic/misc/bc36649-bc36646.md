# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments"></a><span data-ttu-id="53680-102">Tipos de dados dos parâmetros de tipo no método de extensão '\<methodname>' definidos em '\<typename>' não podem ser inferidos destes argumentos</span><span class="sxs-lookup"><span data-stu-id="53680-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments</span></span>

<span data-ttu-id="53680-103">Não é possível inferir os tipos de dados dos parâmetros de tipo no método de extensão ' \<methodname> ' definido em ' \<typename> ' com base nesses argumentos.</span><span class="sxs-lookup"><span data-stu-id="53680-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments.</span></span> <span data-ttu-id="53680-104">A especificação explícita dos tipos de dados pode corrigir esse erro.</span><span class="sxs-lookup"><span data-stu-id="53680-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="53680-105">Foi feita uma tentativa de usar a inferência de tipos para determinar o tipo de dados (ou tipos) do parâmetro de tipo (ou parâmetros) ao avaliar uma chamada para um método de extensão genérico.</span><span class="sxs-lookup"><span data-stu-id="53680-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="53680-106">No entanto, o compilador não é capaz de localizar um tipo de dados para os parâmetros de tipo nesse método e relata o erro.</span><span class="sxs-lookup"><span data-stu-id="53680-106">However, the compiler is not able to find a data type for the type parameters in this method, and it reports the error.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="53680-107">Ao especificar argumentos não é uma opção (por exemplo, para operadores de consulta em expressões de consulta), a mensagem de erro é exibida sem a segunda sentença.</span><span class="sxs-lookup"><span data-stu-id="53680-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="53680-108">O código a seguir demonstra o erro.</span><span class="sxs-lookup"><span data-stu-id="53680-108">The following code demonstrates the error.</span></span>  
  
```vb  
Module Module1  
  
    Sub Main()  
  
        Dim classInstance As ClassExample  
  
        '' Not valid.  
        'classInstance.GenericExtensionMethod("Hello", "World")  
  
    End Sub  
  
    <System.Runtime.CompilerServices.Extension()> _  
    Sub GenericExtensionMethod(Of T)(ByVal classEx As ClassExample, _  
                                     ByVal x As String, ByVal y As _  
                                     InterfaceExample(Of T))  
    End Sub  
  
End Module  
  
Interface InterfaceExample(Of T)  
End Interface  
  
Class ClassExample  
End Class  
```  
  
 <span data-ttu-id="53680-109">**ID do erro:** BC36649 e BC36646</span><span class="sxs-lookup"><span data-stu-id="53680-109">**Error ID:** BC36649 and BC36646</span></span>

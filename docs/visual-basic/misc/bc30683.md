# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39; herda &#39; instrução deve preceder todas as declarações em uma classe
O `Inherits` instruções devem estar no início de definições de classe.  
  
 **ID do erro:** BC30683  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `Inherits` à parte superior da sua classe.

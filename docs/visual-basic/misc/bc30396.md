# <a name="39ltspecifiergt39-is-not-valid-on-an-enum-declaration"></a>&#39;&lt;especificador de&gt; &#39; não é válido em uma declaração Enum
Você usou um especificador que não é válido em um `Enum` declaração.  
  
 **ID do erro:** BC30396  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o especificador.

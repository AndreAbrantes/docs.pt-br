# <a name="events-cannot-have-a-return-type"></a>Eventos não podem ter um tipo de retorno
Eventos podem aceitar argumentos, mas não podem retornar valores diretamente.  
  
 **ID do erro:** BC30032  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o tipo de retorno do `Event` instrução.

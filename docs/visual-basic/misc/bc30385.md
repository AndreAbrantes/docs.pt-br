# <a name="modifier-is-not-valid-on-a-delegate-declaration"></a>'\<modificador >' não é válido em uma declaração Delegate
Você tentou usar um modificador, tais como `Shared`, em uma declaração delegate. Os delegados são objetos usados para chamar os métodos de outros objetos, definindo um construtor que é passado para a especificação de um método de objeto. Não é válido para especificar um modificador em uma declaração delegate.  
  
 **ID do erro:** BC30385  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o modificador.

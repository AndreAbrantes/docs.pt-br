# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Acesso de propriedade deve ser atribuído à propriedade ou usar seu valor
Você tentou acessar uma propriedade sem atribuí-lo ou usar seu valor.
  
 **ID do erro:** BC30545  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atribua um valor à propriedade.  
  
     \- ou -  
  
-   Use o valor da propriedade.

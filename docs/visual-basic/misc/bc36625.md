# <a name="lambda-expression-cannot-be-converted-to-39lttypenamegt39-because-39lttypenamegt39-is-not-a-delegate-type"></a><span data-ttu-id="10c50-102">Expressão lambda não pode ser convertida em '&lt;typename&gt;'porque'&lt;typename&gt;' não é um tipo de delegado</span><span class="sxs-lookup"><span data-stu-id="10c50-102">Lambda expression cannot be converted to &#39;&lt;typename&gt;&#39; because &#39;&lt;typename&gt;&#39; is not a delegate type</span></span>
<span data-ttu-id="10c50-103">Expressões lambda podem ser usadas onde os delegados são válidos.</span><span class="sxs-lookup"><span data-stu-id="10c50-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="10c50-104">Eles podem ser convertidos para tipos delegados compatível, mas não para qualquer outro tipo.</span><span class="sxs-lookup"><span data-stu-id="10c50-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="10c50-105">Por exemplo, você pode definir um tipo de delegado e atribuir uma expressão lambda para ele ou enviar uma expressão lambda como argumento para um <xref:System.Func%601>parâmetro.</xref:System.Func%601></span><span class="sxs-lookup"><span data-stu-id="10c50-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="10c50-106">Esses exemplos são mostrados no código a seguir.</span><span class="sxs-lookup"><span data-stu-id="10c50-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```

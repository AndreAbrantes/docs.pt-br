# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' RaiseEvent ' não são válidas na janela Immediate
`RaiseEvent` instruções não podem ser usadas na janela **Immediate** .  
  
 **ID do erro:** BC30739  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Teste `RaiseEvent` instruções do código no **Editor de código**.

# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent&#39; instruções não são válidas na janela imediata
`RaiseEvent` instruções não podem ser usadas no **imediato** janela.  
  
 **ID do erro:** BC30739  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Teste `RaiseEvent` instruções no código do **Editor de código**.

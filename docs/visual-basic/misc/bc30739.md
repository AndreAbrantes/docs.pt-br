# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'RaiseEvent' não são válidas na janela imediata
`RaiseEvent`instruções não podem ser usadas os **imediato** janela.  
  
 **ID do erro:** BC30739  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Teste `RaiseEvent` instruções de código a **Editor de código**.

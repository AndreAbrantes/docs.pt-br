# <a name="xml-processing-instruction-name-name-is-not-valid"></a>O nome' nome de instrução de processamento de XML' não é válido
O nome fornecido para uma instrução de processamento de XML é inválido.  
  
 **ID do erro:** BC31174  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Corrija os erros no nome da instrução de processamento de XML ou removê-lo.

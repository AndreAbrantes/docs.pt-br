# <a name="specified-registry-key-is-not-valid-because-it-contains-two-or-more-consecutive-backslashes"></a>Chave do Registro especificada não é válido porque contém duas ou mais barras invertidas consecutivas
Uma chave do Registro especificada com um caminho contém dois ou mais barras invertidas consecutivas. Isso pode ser um resultado de combinar várias cadeias de caracteres para formar o caminho e incluir inadvertidamente muitas barras invertidas.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Examine a chave do registro que está sendo especificada para determinar onde e por que as barras invertidas extras estão sendo inseridas.

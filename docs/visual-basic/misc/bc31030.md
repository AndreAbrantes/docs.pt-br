# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>Constante de compilação condicional no nível de projeto '\<constantname >' não é válido: \<erro >
Uma opção de compilação condicional inválido é especificada para o projeto.  
  
 **ID do erro:** BC31030  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se as constantes de compilação condicional usadas com este projeto.

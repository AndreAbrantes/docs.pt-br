# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese e VbStrConv. TraditionalChinese não podem ser combinados
Seu aplicativo está tentando combinar os membros de enumeração de `VbStrConv` `SimplifiedChinese` e `TraditionalChinese`, que são mutuamente exclusivos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `VbStrConv.SimplifiedChinese` ou `VbStrConv.TraditionalChinese`.

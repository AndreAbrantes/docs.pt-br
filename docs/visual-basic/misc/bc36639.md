# <a name="39byref39-parameter-39ltparameternamegt39-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="9b4a3-102">Parâmetro 'ByRef' '&lt;parametername&gt;' não pode ser usado em uma expressão lambda</span><span class="sxs-lookup"><span data-stu-id="9b4a3-102">&#39;ByRef&#39; parameter &#39;&lt;parametername&gt;&#39; cannot be used in a lambda expression</span></span>
<span data-ttu-id="9b4a3-103">Uma expressão lambda declarada dentro de um `Sub` ou função não pode usar qualquer `ByRef` parâmetros de que `Sub` ou função.</span><span class="sxs-lookup"><span data-stu-id="9b4a3-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="9b4a3-104">Por exemplo, o código a seguir irá causar esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="9b4a3-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="9b4a3-105">**ID do erro:** BC36639</span><span class="sxs-lookup"><span data-stu-id="9b4a3-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9b4a3-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="9b4a3-106">To correct this error</span></span>

    ```  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```

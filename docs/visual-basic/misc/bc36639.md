<span data-ttu-id="c29a4-103">Uma expressão lambda declarada dentro de um `Sub` ou função não pode usar qualquer `ByRef` parâmetros que `Sub` ou função.</span><span class="sxs-lookup"><span data-stu-id="c29a4-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="c29a4-104">Por exemplo, o código a seguir fará com que esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="c29a4-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>

'' Not valid.

'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="c29a4-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="c29a4-106">To correct this error</span></span>

        Dim lambda = Function(p As Integer) p + temp

    ```

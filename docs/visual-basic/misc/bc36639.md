<span data-ttu-id="cf822-103">Uma expressão lambda declarada em uma `Sub` ou função não pode usar nenhum parâmetro `ByRef` desse `Sub` ou função.</span><span class="sxs-lookup"><span data-stu-id="cf822-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="cf822-104">Por exemplo, o código a seguir causará esse erro porque o parâmetro `ByRef` `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="cf822-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>

'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="cf822-105">**ID do erro:** BC36639</span><span class="sxs-lookup"><span data-stu-id="cf822-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="cf822-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="cf822-106">To correct this error</span></span>

        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```

<span data-ttu-id="d1c6b-103">Uma expressão lambda é declarada dentro de um `Sub` ou função não é possível usar qualquer `ByRef` parâmetros de que `Sub` ou função.</span><span class="sxs-lookup"><span data-stu-id="d1c6b-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="d1c6b-104">Por exemplo, o código a seguir fará com que esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="d1c6b-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>

```  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```

# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="c5ced-102">Não é possível usar o parâmetro 'ByRef' '\<parametername>' em uma expressão lambda</span><span class="sxs-lookup"><span data-stu-id="c5ced-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>

<span data-ttu-id="c5ced-103">Uma expressão lambda declarada em uma `Sub` função ou não pode usar nenhum `ByRef` parâmetro dessa `Sub` função.</span><span class="sxs-lookup"><span data-stu-id="c5ced-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="c5ced-104">Por exemplo, o código a seguir causará esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="c5ced-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="c5ced-105">**ID do erro:** BC36639</span><span class="sxs-lookup"><span data-stu-id="c5ced-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c5ced-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="c5ced-106">To correct this error</span></span>  
  
- <span data-ttu-id="c5ced-107">Atribua o `ByRef` parâmetro a uma variável local e use a variável local na expressão lambda, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="c5ced-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```

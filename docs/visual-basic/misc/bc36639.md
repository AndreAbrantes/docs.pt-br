# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="1b0d9-102">'ByRef' parâmetro\<' parâmetro nome>' não pode ser usado em uma expressão lambda</span><span class="sxs-lookup"><span data-stu-id="1b0d9-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="1b0d9-103">Uma expressão lambda declarada `Sub` dentro de `ByRef` uma função `Sub` ou função não pode usar quaisquer parâmetros disso ou função.</span><span class="sxs-lookup"><span data-stu-id="1b0d9-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="1b0d9-104">Por exemplo, o seguinte código causará esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="1b0d9-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="1b0d9-105">**ID de erro:** BC36639</span><span class="sxs-lookup"><span data-stu-id="1b0d9-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1b0d9-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="1b0d9-106">To correct this error</span></span>

    ```vb  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```

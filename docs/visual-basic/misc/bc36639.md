# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="1f676-102">Parâmetro 'ByRef' '\<parametername >' não pode ser usado em uma expressão lambda</span><span class="sxs-lookup"><span data-stu-id="1f676-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="1f676-103">Uma expressão lambda é declarada dentro de um `Sub` ou função não é possível usar qualquer `ByRef` parâmetros de que `Sub` ou função.</span><span class="sxs-lookup"><span data-stu-id="1f676-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="1f676-104">Por exemplo, o código a seguir fará com que esse erro porque o `ByRef` parâmetro `n` é usado na expressão lambda.</span><span class="sxs-lookup"><span data-stu-id="1f676-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="1f676-105">**ID do erro:** BC36639</span><span class="sxs-lookup"><span data-stu-id="1f676-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1f676-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="1f676-106">To correct this error</span></span>  
  
- <span data-ttu-id="1f676-107">Atribuir o `ByRef` parâmetro para uma variável local e use a variável local na expressão lambda, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="1f676-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```

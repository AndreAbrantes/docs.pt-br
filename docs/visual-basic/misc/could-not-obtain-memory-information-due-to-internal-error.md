# <a name="could-not-obtain-memory-information-due-to-internal-error"></a>Não foi possível obter informações de memória devido a erro interno
Uma chamada para uma das propriedades de informações de memória a `My.Computer.Info` falha do objeto.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Try...Catch` bloco em torno da chamada à propriedade de informações de memória a `My.Computer.Info` objeto.

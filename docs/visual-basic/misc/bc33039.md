 <span data-ttu-id="67da8-104">A variável de contador deve ser de um tipo que dá suporte a operadores de adição ( `+` ) e de subtração ( `-` ) que operam completamente em seu tipo recipiente.</span><span class="sxs-lookup"><span data-stu-id="67da8-104">The counter variable must be of a type that supports addition (`+`) and subtraction (`-`) operators that operate completely on their containing type.</span></span> <span data-ttu-id="67da8-105">Isso significa que ambos os operandos e o valor de retorno devem ser do tipo da variável de contador.</span><span class="sxs-lookup"><span data-stu-id="67da8-105">This means both of the operands and the return value must be of the type of the counter variable.</span></span>

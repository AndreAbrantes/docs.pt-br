# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Método parcial &#39; &lt;methodname&gt;&#39; não é possível usar o &#39; implementa &#39; a palavra-chave
Uma definição de método parcial não pode implementar uma interface ou membro de interface.  
  
 **ID do erro:** BC31439  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Implements` instrução da definição de método parcial.

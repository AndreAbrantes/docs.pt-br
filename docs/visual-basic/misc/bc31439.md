# <a name="partial-method-ltmethodnamegt-cannot-use-the-implements-keyword"></a>Método parcial '&lt;methodname&gt;' não é possível usar a palavra-chave 'Implements'
Uma definição de método parcial não pode implementar uma interface ou um membro de interface.  
  
 **ID do erro:** BC31439  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Implements` instrução da definição de método parcial.

# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39; Moeda &#39; não é um tipo com suporte; usar o &#39; decimais &#39; em vez disso, digite
O `Currency` tipo de dados foi substituído pelo `Decimal` tipo de dados.  
  
 **ID do erro:** BC30803  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `Decimal` de tipo de dados em vez de `Currency`.

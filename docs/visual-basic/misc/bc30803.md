# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>'Currency' não é mais um tipo com suporte; Use o tipo 'Decimal'
O `Currency` tipo de dados foi substituído pelo `Decimal` tipo de dados.  
  
 **ID do erro:** BC30803  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `Decimal` em vez de `Currency`.

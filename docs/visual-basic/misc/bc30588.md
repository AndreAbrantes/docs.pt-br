# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Argumento omitido não pode corresponder a um parâmetro ParamArray
Você omitiu um argumento necessário para corresponder um `ParamArray` parâmetro. `ParamArray`, que permite que você passe um número arbitrário de argumentos para um procedimento pode ser usado apenas no último argumento de uma lista de argumentos.  
  
 **ID do erro:** BC30588  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Fornece o argumento omitido.

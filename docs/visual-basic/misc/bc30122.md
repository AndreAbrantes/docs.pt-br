# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Stop' não são válidas na janela imediata
O `Stop` e `End` suspender a execução de instruções e não são permitidas em um contexto de depuração.  
  
 **ID do erro:** BC30122  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita uma `Stop` ou `End` instrução na **imediato** janela.

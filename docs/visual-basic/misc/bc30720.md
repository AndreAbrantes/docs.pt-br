# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' On Error ' não são válidas na janela Immediate
`On Error` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30720  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `On Error` instruções do código de depuração.

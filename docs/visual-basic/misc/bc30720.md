# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Em erro&#39; instruções não são válidas na janela imediata
`On Error` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30720  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `On Error` instruções do código de depuração.

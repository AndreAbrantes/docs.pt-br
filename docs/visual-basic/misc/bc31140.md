# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39; Personalizar &#39; modificador só pode ser usado imediatamente antes de um &#39; Evento &#39; declaração
O `Custom` palavra-chave deve preceder apenas o `Event` palavra-chave em uma declaração de evento personalizado.  
  
 **ID do erro:** BC31140  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `Custom` palavra-chave deve preceder o `Event` palavra-chave.  
  
     —ou—  
  
-   Remover o `Custom` palavra-chave.

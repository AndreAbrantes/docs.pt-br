# <a name="39ltnamegt39-cannot-expose-type-39lttype1gt39-used-in-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttype2gt39"></a>'&lt;nome&gt;'não pode expor o tipo'&lt;type1&gt;'usado no tipo delegado subjacente'&lt;delegatetype&gt;' do evento que está implementando fora do projeto por meio de &lt;especificador&gt; '&lt;type2&gt;'
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30924  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public` ou use outro tipo.

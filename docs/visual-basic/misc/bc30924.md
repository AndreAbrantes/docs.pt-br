# <a name="39ltnamegt39-cannot-expose-type-39lttype1gt39-used-in-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttype2gt39"></a>&#39; &lt;nome&gt;&#39; não pode expor o tipo de &#39;&lt; type1&gt;&#39; usado no tipo delegado subjacente &#39;&lt; delegatetype&gt;&#39; do evento que está implementando fora do projeto por meio de &lt;especificador&gt; &#39;&lt; type2&gt;&#39;
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30924

# <a name="name-cannot-expose-type-type1-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type2"></a>'\<nome >' não pode expor o tipo '\<type1 >' usado no tipo delegado subjacente '\<delegatetype >' do evento que está implementando fora do projeto por meio \<especificador > '\<type2 >'
O código expõe um tipo particular fora da classe pública.  
  
 **ID do erro:** BC30924  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declare o tipo `Public` ou use outro tipo.

# <a name="compare-expected"></a>'Compare' esperado
Uma `Option` declaração especifica `Text` ou `Binary`, mas não `Compare`.  
  
 **ID do erro:** BC30208  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione a `Compare` palavra-chave para o `Option` instrução; por exemplo, `Option Compare Binary`.

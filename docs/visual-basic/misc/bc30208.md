# <a name="39compare39-expected"></a>&#39;Comparar&#39; esperado
Um `Option` instrução especifica `Text` ou `Binary`, mas não `Compare`.  
  
 **ID do erro:** BC30208  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o `Compare` palavra-chave para o `Option` instrução; por exemplo, `Option Compare Binary`.

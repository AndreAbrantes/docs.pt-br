# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>'\<name1 >' está em conflito com um \<tipo > com o mesmo nome definido em '\<nome2 >'
Foi encontrado um nome que está em conflito com um tipo com o mesmo nome.  
  
 **ID do erro:** BC31073  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Declare um nome exclusivo.  
  
2. Use colchetes em torno do nome de membro em conflito.

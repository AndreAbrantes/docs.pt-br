# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39;&lt;Name1&gt; &#39; está em conflito com um &lt;tipo&gt; com o mesmo nome definido no &#39; &lt;name2&gt;&#39;
Foi encontrado um nome que está em conflito com um tipo com o mesmo nome.  
  
 **ID do erro:** BC31073  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Declare um nome exclusivo.  
  
2.  Use colchetes em torno do nome de membro em conflito.

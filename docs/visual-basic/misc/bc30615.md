# <a name="39end39-statement-cannot-be-used-in-class-library-projects"></a>Instrução 'End' não pode ser usada em projetos de biblioteca de classe
Projetos de biblioteca de classes usados para criar DLLs não permitem o uso do `End` palavra-chave para interromper a execução de código em um procedimento.  
  
 **ID do erro:** BC30615  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use estruturas de controle como `While` e `For` para controlar o fluxo de execução do programa.

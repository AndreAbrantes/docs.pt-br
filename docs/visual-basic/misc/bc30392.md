# <a name="39catch39-cannot-catch-type-39lttypenamegt39-because-it-is-not-39systemexception39-or-a-class-that-inherits-from-39systemexception39"></a>'Catch' não pode capturar o tipo '&lt;typename&gt;' porque ele não é 'System. Exception' ou uma classe que herda de 'System. Exception'
`Catch`só pode interceptar exceções, e você tentou capturar algo que não é derivado de uma exceção.  
  
 **ID do erro:** BC30392  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover o `Catch` declaração, ou alterar o destino do `Catch` para uma exceção real.

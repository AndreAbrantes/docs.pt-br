<span data-ttu-id="4fb11-103">Não é possível inferir um tipo comum para os primeiro e segundo operandos do operador 'If' binário.</span><span class="sxs-lookup"><span data-stu-id="4fb11-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="4fb11-104">Um deve ter uma conversão de ampliação para outro.</span><span class="sxs-lookup"><span data-stu-id="4fb11-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="4fb11-105">O binário `If` operador requer que haja uma conversão de ampliação entre um dos argumentos e o outro argumento.</span><span class="sxs-lookup"><span data-stu-id="4fb11-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="4fb11-106">Por exemplo, porque não há uma conversão de ampliação em ambas as direções entre `Integer` e `String`, o código a seguir faz com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="4fb11-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

'' Not valid.  
' Console.WriteLine(If(first, second))

 <span data-ttu-id="4fb11-107">**ID do erro:** BC33110</span><span class="sxs-lookup"><span data-stu-id="4fb11-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4fb11-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="4fb11-108">To correct this error</span></span>

    ```

        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```

<span data-ttu-id="e86b3-103">Não é possível inferir um tipo comum para os primeiro e segundo operandos do operador 'If' binário.</span><span class="sxs-lookup"><span data-stu-id="e86b3-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="e86b3-104">Um deve ter uma conversão de ampliação para outro.</span><span class="sxs-lookup"><span data-stu-id="e86b3-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="e86b3-105">O binário `If` operador requer que haja uma conversão de ampliação entre um dos argumentos e o outro argumento.</span><span class="sxs-lookup"><span data-stu-id="e86b3-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="e86b3-106">Por exemplo, porque não há uma conversão de ampliação em ambas as direções entre `Integer` e `String`, o código a seguir faz com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="e86b3-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="e86b3-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="e86b3-108">To correct this error</span></span>

    Else  
        Console.WriteLine(second)

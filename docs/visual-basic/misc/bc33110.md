# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="f7345-102">Não é possível deduzir um tipo comum para o primeiro e o segundo operandos do operador ' If ' binário</span><span class="sxs-lookup"><span data-stu-id="f7345-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="f7345-103">Não é possível inferir um tipo comum para o primeiro e o segundo operandos do operador ' If ' binário.</span><span class="sxs-lookup"><span data-stu-id="f7345-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="f7345-104">Um deve ter uma conversão de ampliação para a outra.</span><span class="sxs-lookup"><span data-stu-id="f7345-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="f7345-105">O operador `If` Binary requer que haja uma conversão de ampliação entre um dos argumentos e o outro argumento.</span><span class="sxs-lookup"><span data-stu-id="f7345-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="f7345-106">Por exemplo, como não há uma conversão de alargamento em qualquer direção entre `Integer` e `String`, o código a seguir causa esse erro.</span><span class="sxs-lookup"><span data-stu-id="f7345-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="f7345-107">**ID do erro:** BC33110</span><span class="sxs-lookup"><span data-stu-id="f7345-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f7345-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="f7345-108">To correct this error</span></span>

  ```vb

  ```

- <span data-ttu-id="f7345-110">Reescreva o código usando uma construção condicional diferente.</span><span class="sxs-lookup"><span data-stu-id="f7345-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb

      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```

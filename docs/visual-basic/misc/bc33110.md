# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="5a263-102">Não é possível inferir um tipo comum para o primeiro e segundo operandos do operador 'If' binário</span><span class="sxs-lookup"><span data-stu-id="5a263-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="5a263-103">Não é possível inferir um tipo comum para o primeiro e o segundo operandos do operador ' If ' binário.</span><span class="sxs-lookup"><span data-stu-id="5a263-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="5a263-104">Um deve ter uma conversão de ampliação para a outra.</span><span class="sxs-lookup"><span data-stu-id="5a263-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="5a263-105">O `If` operador Binary requer que haja uma conversão de ampliação entre um dos argumentos e o outro argumento.</span><span class="sxs-lookup"><span data-stu-id="5a263-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="5a263-106">Por exemplo, como não há uma conversão de alargamento em qualquer direção entre `Integer` e `String` , o código a seguir causa esse erro.</span><span class="sxs-lookup"><span data-stu-id="5a263-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="5a263-107">**ID do erro:** BC33110</span><span class="sxs-lookup"><span data-stu-id="5a263-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="5a263-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="5a263-108">To correct this error</span></span>

- <span data-ttu-id="5a263-109">Forneça uma conversão explícita para um dos operandos, se isso for possível em seu código:</span><span class="sxs-lookup"><span data-stu-id="5a263-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="5a263-110">Reescreva o código usando uma construção condicional diferente.</span><span class="sxs-lookup"><span data-stu-id="5a263-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```

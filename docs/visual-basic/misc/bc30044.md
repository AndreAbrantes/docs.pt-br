# <a name="39ltkeywordgt39-is-not-valid-within-a-structure"></a>&#39;&lt;palavra-chave&gt; &#39; não é válido dentro de uma estrutura
Estruturas são tipos de valor, não tipos de referência. Uma estrutura não é uma instância criada de uma classe, portanto, o `Me`, `MyClass`, e `MyBase` palavras-chave são sem sentido em uma estrutura.  
  
 **ID do erro:** BC30044  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar a estrutura para uma classe, ou remova a palavra-chave do procedimento.

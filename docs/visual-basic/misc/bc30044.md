# <a name="keyword-is-not-valid-within-a-structure"></a>'\<palavra-chave > ' não é válida dentro de uma estrutura
Estruturas são tipos de valor, não tipos de referência. Uma estrutura não é uma instância criada a partir de uma classe, `Me`portanto `MyClass`, as `MyBase` palavras-chave, e não têm significado em uma estrutura.  
  
 **ID do erro:** BC30044  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Altere a estrutura para uma classe ou remova a palavra-chave do procedimento.

# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Caractere 'caractere' (0x\<hexadecimal >) não é permitido em XML literal
Um caractere sem suporte foi usado em um literal XML.  
  
 **ID do erro:** BC31176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o caractere sem suporte.

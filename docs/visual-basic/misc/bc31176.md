# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Caractere 'caractere' (0x&lt;hexadecimal&gt;) não é permitido em XML literal
Um caractere sem suporte foi usado em um literal XML.  
  
 **ID do erro:** BC31176  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere sem suporte.

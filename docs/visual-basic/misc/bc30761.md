# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>Projeto &#39; &lt;projectname&gt;&#39; não é possível referenciar o projeto &#39;&lt; ProjectName&gt;&#39; porque &#39;&lt; ProjectName&gt;&#39; direta ou indiretamente referências &#39;&lt; ProjectName&gt;&#39;
Um projeto inclui uma referência a um segundo projeto. Por sua vez, o segundo projeto inclui uma referência para o projeto no qual ele é referenciado.  
  
 **ID do erro:** BC30761  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova uma das referências.

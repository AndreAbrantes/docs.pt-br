# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Projeto '&lt;NomeDoProjeto&gt;'não pode referenciar o projeto'&lt;projectname&gt;'porque'&lt;projectname&gt;'direta ou indiretamente referencia'&lt;projectname &gt;'
Um projeto inclui uma referência a um segundo projeto. Por sua vez, o segundo projeto inclui uma referência ao projeto no qual ele é referenciado.  
  
 **ID do erro:** BC30761  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova uma das referências.

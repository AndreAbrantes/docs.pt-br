# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Projeto '\<projectname >' não pode referenciar o projeto '\<projectname >' porque '\<projectname >' direta ou indiretamente, referencia '\<projectname >'
Um projeto inclui uma referência a um segundo projeto. Por sua vez, o segundo projeto inclui uma referência ao projeto no qual ele é referenciado.  
  
 **ID do erro:** BC30761  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova uma das referências.

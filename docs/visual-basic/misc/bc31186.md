# <a name="-is-not-allowed"></a>':' não é permitido
':' não é permitido. XML qualificado nomes não podem ser usados neste contexto.  
  
 Um literal XML foi especificado com um nome qualificado em um local inválido, por exemplo, um destino em um literal de instrução de processamento.  
  
 **ID do erro:** BC31186  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o literal com o nome qualificado XML ou movê-lo para um local válido.  
  
     Para obter informações sobre quais caracteres são válidos em nomes XML, consulte [Extensible Markup Language (XML) 1.0](https://www.w3.org/TR/xml).

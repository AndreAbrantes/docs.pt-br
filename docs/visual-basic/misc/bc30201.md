# <a name="expression-expected"></a>Expressão esperada
Um operador como `+` ou `=` requer uma expressão para o operando à direita, mas não reconhecível expressão aparece.  
  
 **ID do erro:** BC30201  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Forneça uma expressão válida para o operando à direita do operador.

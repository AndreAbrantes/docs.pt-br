# <a name="members-in-a-module-cannot-be-declared-ltspecifiergt"></a>Membros em um módulo não podem ser declarados como '&lt;especificador&gt;'
Você usou um especificador que não é válido em um membro dentro de um `Module` instrução. Módulos nunca podem ser instanciados, não dão suporte a herança e não podem implementar interfaces.  
  
 **ID do erro:** BC30436  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

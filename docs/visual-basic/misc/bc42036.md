 <span data-ttu-id="95d45-107">As expressões em um `Select`... `Case` construção todos deve ser do mesmo tipo de dados ou de dados relacionados tipos que podem ser convertidos entre si.</span><span class="sxs-lookup"><span data-stu-id="95d45-107">The expressions in a `Select`...`Case` construction should all be of the same data type or of closely related data types that can be converted to each other.</span></span> <span data-ttu-id="95d45-108">Isso ocorre porque cada `Case` instrução compara a pelo menos um valor em relação à expressão de teste no qual o `Select`... `Case` com base em construção.</span><span class="sxs-lookup"><span data-stu-id="95d45-108">This is because each `Case` statement compares at least one value against the test expression on which the `Select`...`Case` construction is based.</span></span>

# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39; não há suporte para instruções; Use &#39;End While&#39; instruções em vez disso
O `End While` construir substitui o `Wend` palavra-chave.  
  
 **ID do erro:** BC30809  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use `End While` para encerrar um `While` bloco.

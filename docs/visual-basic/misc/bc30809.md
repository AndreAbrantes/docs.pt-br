# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Não há suporte para instruções 'Wend'; Use instruções 'End While'
O `End While` construir substitui o `Wend` palavra-chave.  
  
 **ID do erro:** BC30809  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use `End While` para encerrar um `While` bloco.

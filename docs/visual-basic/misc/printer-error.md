# <a name="printer-error"></a>Erro de impressora
Ocorreu um erro na impressora, mas nenhuma outra informação foi retornada para o computador que enviou o arquivo.  
  
### <a name="to-correct-the-error"></a>Para corrigir o erro  
  
-   Examine fisicamente a impressora. Verifique se todas as conexões entre o computador e a impressora estão sólidas. A maioria das impressoras fornecem uma exibição para obter informações de erro, como "Offline" ou "sem"papel.

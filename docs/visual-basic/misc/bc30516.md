# <a name="overload-resolution-failed-because-no-accessible-method-accepts-this-number-of-arguments"></a>Resolução de sobrecarga falhou porque nenhum '\<método >' aceita este número de argumentos
Você fez uma chamada para um método sobrecarregado, mas o compilador não é possível encontrar um método com uma lista de argumentos correspondentes.  
  
 **ID do erro:** BC30516  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a lista de argumentos que você forneceu.

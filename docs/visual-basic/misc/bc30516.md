# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>Resolução de sobrecarga falhou porque nenhum &#39; &lt;método&gt; &#39; aceita este número de argumentos
Você fez uma chamada para um método sobrecarregado, mas o compilador não pode localizar um método com uma lista de argumento correspondente.  
  
 **ID do erro:** BC30516  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a lista de argumentos que você forneceu.

# <a name="character-constant-must-contain-exactly-one-character"></a>Constante de caractere deve conter exatamente um caractere
Uma constante de caractere não é válida porque ele contém mais de um caractere ou um caractere nulo (todos os bits são zero).  
  
 **ID do erro:** BC30004  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a constante de caractere em um único caractere ou o caractere nulo para um caractere válido.

# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="5ee2e-102">Tipos de dados dos parâmetros de tipo no método de extensão '\<methodname >' definido em 'typename' não pode ser inferido destes argumentos porque eles não convertem para o mesmo tipo</span><span class="sxs-lookup"><span data-stu-id="5ee2e-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>
<span data-ttu-id="5ee2e-103">Tipos de dados dos parâmetros de tipo no método de extensão '\<methodname >' definido em 'typename' não pode ser inferido destes argumentos porque eles não convertem para o mesmo tipo.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="5ee2e-104">Especificar os dados de tipo (s) explicitamente talvez corrija esse erro.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="5ee2e-105">Foi feita uma tentativa de usar inferência de tipo para determinar o tipo dados (ou tipos) do parâmetro de tipo (ou parâmetros) ao avaliar uma chamada para um método de extensão genéricos.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="5ee2e-106">O compilador não foi possível encontrar um tipo de dados que atende às restrições de todos os argumentos.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="5ee2e-107">Portanto, ele relatado esse erro.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-107">Therefore, it reported this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="5ee2e-108">Ao especificar argumentos não é uma opção (por exemplo, para operadores de consulta em expressões de consulta), a mensagem de erro é exibido sem a segunda frase.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="5ee2e-109">O código a seguir demonstra o erro.</span><span class="sxs-lookup"><span data-stu-id="5ee2e-109">The following code demonstrates the error.</span></span>  
  
```vb  
Option Strict Off  
Module Module3  
    Sub Main()  
  
        Dim c1 As New Class1  
  
        '' Not valid. Integer and Date do not convert to the same type.  
        'c1.targetMethod(19, #3/4/2007#)  
  
    End Sub  
  
    <System.Runtime.CompilerServices.Extension()> _  
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
    Class Class1  
    End Class  
  
End Module  
```  
  
 <span data-ttu-id="5ee2e-110">**ID do erro:** BC36661 e BC36658</span><span class="sxs-lookup"><span data-stu-id="5ee2e-110">**Error ID:** BC36661 and BC36658</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5ee2e-111">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="5ee2e-111">To correct this error</span></span>  
  
- <span data-ttu-id="5ee2e-112">Você pode ser capaz de converter um ou mais argumentos explicitamente em um tipo compatível, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="5ee2e-112">You might be able to convert one or more arguments explicitly to a compatible type, as shown in the following code:</span></span>  
  
    ```  
    c1.targetMethod(19, #3/4/2007#.ToOADate)  
    ```  
  
- <span data-ttu-id="5ee2e-113">Você poderá especificar um tipo de dados para o parâmetro de tipo ou os parâmetros para o qual os argumentos de converter, conforme mostrado no código a seguir:</span><span class="sxs-lookup"><span data-stu-id="5ee2e-113">You might be able to specify a data type for the type parameter or parameters to which the arguments convert, as shown in the following code:</span></span>  
  
    ```  
    c1.targetMethod(Of String)(19, #3/4/2007#)  
    ```

# <a name="method-39ltmethodnamegt39-has-no-parameter-named-39ltparameternamegt39"></a>Método &#39; &lt;methodname&gt;&#39; tem nenhum parâmetro denominado &#39;&lt; nome do parâmetro&gt;&#39;
Um argumento de nome fornecido não corresponde a nenhum dos argumentos de método de destino.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que o argumento nomeado foi fornecido corretamente.

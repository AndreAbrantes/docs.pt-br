# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>'Shared' não pode ser combinado com '&lt;especificador&gt;' em uma declaração de propriedade
Você tentou combinar o `Shared` modificador com um especificador que não é válido em uma combinação como uma declaração de propriedade.  
  
 **ID do erro:** BC30502  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

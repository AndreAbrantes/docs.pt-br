# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;Compartilhado&#39; não pode ser combinado com &#39; &lt;especificador&gt; &#39; em uma declaração de propriedade
Você tentou combinar o `Shared` modificador com um especificador que não é válido em uma combinação como uma declaração de propriedade.  
  
 **ID do erro:** BC30502  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Objeto referenciado '\<objectname >' tem um valor de 'Nothing'
O objeto referenciado não foi definido ou foi definido como o valor `Nothing`.  
  
 **ID do erro:** BC30705  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se o nome do objeto está escrito corretamente e que o objeto tenha sido definido.

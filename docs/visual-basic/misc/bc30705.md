# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Objeto referenciado &#39; &lt;objectname&gt;&#39; possui um valor de &#39; Nada &#39;
O objeto que está sendo referenciado não foi definido ou foi definido como o valor `Nothing`.  
  
 **ID do erro:** BC30705  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o nome do objeto está escrito corretamente e que o objeto tenha sido definido.

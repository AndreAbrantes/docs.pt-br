# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>O objeto referenciado '\<ObjectName > ' tem um valor de ' Nothing '
O objeto que está sendo referenciado não foi definido ou foi definido para o valor `Nothing`.  
  
 **ID do erro:** BC30705  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Verifique se o nome do objeto está grafado corretamente e se o objeto foi definido.

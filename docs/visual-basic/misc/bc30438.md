# <a name="constants-must-have-a-value"></a>Constantes devem ter um valor
Você declarou uma constante sem atribuindo um valor.  
  
 **ID do erro:** BC30438  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Atribua um valor para a constante.

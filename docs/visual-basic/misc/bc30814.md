# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39;GoSub&#39; instruções não são mais suportadas
`GoSub` não pode ser usado para chamar um procedimento.  
  
 **ID do erro:** BC30814  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Chamar procedimentos diretamente sem usar `GoSub`; por exemplo:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```

# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39; Estrutura &#39; instrução deve terminar com uma correspondência &#39; End Structure &#39;
Estruturas começam com um `Structure` instrução e terminam com um `End Structure` construir.  
  
 **ID do erro:** BC30624  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o bloco de estrutura termina com um `End Structure` instrução.

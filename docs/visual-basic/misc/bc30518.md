# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>Resolução sobrecarregada falhou porque não acessível '&lt;método&gt;' pode ser chamado com esses argumentos:&lt;erro&gt;
Você fez uma chamada para um método sobrecarregado, mas o compilador não pode localizar um método com uma lista de argumentos correspondentes.  
  
 **ID do erro:** BC30518  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a lista de argumentos que você forneceu.

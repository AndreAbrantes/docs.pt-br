# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>Resolução de sobrecarga falhou porque nenhum &#39; &lt;método&gt; &#39; pode ser chamado com estes argumentos:&lt;erro&gt;
Você fez uma chamada para um método sobrecarregado, mas o compilador não pode localizar um método com uma lista de argumento correspondente.  
  
 **ID do erro:** BC30518  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a lista de argumentos que você forneceu.

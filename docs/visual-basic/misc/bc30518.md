# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>Resolução de sobrecarga falhou porque nenhum '&lt;método&gt;' pode ser chamado com estes argumentos:&lt;erro&gt;
Você fez uma chamada para um método sobrecarregado, mas o compilador não é possível encontrar um método com uma lista de argumentos correspondentes.  
  
 **ID do erro:** BC30518  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a lista de argumentos que você forneceu.

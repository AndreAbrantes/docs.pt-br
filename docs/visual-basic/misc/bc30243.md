# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39; &lt;modificador&gt;&#39; não é válido em declaração de evento
Um `Event` instrução contém uma palavra-chave inválida, como `ReadOnly`.  
  
 **ID do erro:** BC30243  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a palavra-chave inválida do `Event` instrução.

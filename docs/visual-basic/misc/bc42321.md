# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>XML mal formado. Arquivo '\<commentfile >' não pode ser incluído
Um `<include>` marca em um dos comentários de XML se refere a um arquivo que não existe ou um arquivo que contém mal formada XML.  
  
 **ID do erro:** BC42321  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o arquivo existe.  
  
2.  Verifique se o XML no arquivo é válido.

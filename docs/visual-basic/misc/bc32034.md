# <a name="39end-set39-must-be-the-first-statement-on-a-line"></a>&#39; Conjunto de término &#39; deve ser a primeira instrução em uma linha
Um `End Set` instrução segue um separador de declaração dois-pontos (:). `End Set`deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC32034  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

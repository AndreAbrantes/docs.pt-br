# <a name="expected--for-xml-end-tag"></a>Marca de fim XML ' /' esperado
A marca de fim XML para um literal XML está faltando o obrigatório '/' caracteres.  
  
 **ID do erro:** BC31167  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o caractere '/' para o início da marca de fim XML para o XML literal.

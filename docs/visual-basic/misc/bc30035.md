# <a name="syntax-error-visual-basic-error"></a>Erro de sintaxe (erro do Visual Basic)
Linha do código-fonte gera um erro de sintaxe não específico.  
  
 **ID do erro:** BC30035  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Examine a documentação para cada uma das palavras-chave usadas na linha de código-fonte. Compare o código de exemplo na documentação com a linha de código-fonte gera este erro.  
  
2.  Se você não pode identificar a causa do erro, colete informações sobre as circunstâncias e notifique o Atendimento Microsoft.

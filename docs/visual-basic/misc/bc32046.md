<span data-ttu-id="2e5da-103">Uma instrução de declaração usa um [novo operador](../../visual-basic/language-reference/operators/new-operator.md) cláusula especificando um parâmetro de tipo como o tipo a ser criado e o parâmetro de tipo é declarado sem um `New` restrição.</span><span class="sxs-lookup"><span data-stu-id="2e5da-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="2e5da-104">Um *restrição* em um tipo de parâmetro impõe uma necessidade em qualquer argumento de tipo passado para esse parâmetro de tipo quando o tipo genérico é criado.</span><span class="sxs-lookup"><span data-stu-id="2e5da-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="2e5da-105">O `New` restrição Especifica que o argumento de tipo deve expor um construtor sem parâmetros que o código de criação pode acessar.</span><span class="sxs-lookup"><span data-stu-id="2e5da-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="2e5da-106">Isso é o que permite que um `New` cláusula em uma instrução de declaração para criar uma instância desse tipo.</span><span class="sxs-lookup"><span data-stu-id="2e5da-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="2e5da-107">**ID do erro:** BC32046</span><span class="sxs-lookup"><span data-stu-id="2e5da-107">**Error ID:** BC32046</span></span>

# <a name="new-cannot-be-used-on-a-type-parameter-that-does-not-have-a-new-constraint"></a><span data-ttu-id="f5076-102">Não é possível usar 'New' em um parâmetro de tipo que não tenha uma restrição 'New'</span><span class="sxs-lookup"><span data-stu-id="f5076-102">'New' cannot be used on a type parameter that does not have a 'New' constraint</span></span>

<span data-ttu-id="f5076-103">Uma instrução de declaração usa uma [nova cláusula Operator](../language-reference/operators/new-operator.md) especificando um parâmetro de tipo como o tipo a ser criado, e o parâmetro de tipo é declarado sem uma `New` restrição.</span><span class="sxs-lookup"><span data-stu-id="f5076-103">A declaration statement uses a [New Operator](../language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="f5076-104">Uma *restrição* em um parâmetro de tipo impõe um requisito em qualquer argumento de tipo passado para esse parâmetro de tipo quando o tipo genérico é criado.</span><span class="sxs-lookup"><span data-stu-id="f5076-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="f5076-105">A `New` restrição especifica que o argumento de tipo deve expor um construtor sem parâmetros que o código de criação possa acessar.</span><span class="sxs-lookup"><span data-stu-id="f5076-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="f5076-106">Isso é o que permite que uma `New` cláusula em uma instrução de declaração crie uma instância desse tipo.</span><span class="sxs-lookup"><span data-stu-id="f5076-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="f5076-107">**ID do erro:** BC32046</span><span class="sxs-lookup"><span data-stu-id="f5076-107">**Error ID:** BC32046</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f5076-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="f5076-108">To correct this error</span></span>

# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' e &lt;type2&gt; '&lt;typename&gt;' estão em conflito em &lt;namespace&gt; '&lt;namespacename &gt;'
Dois elementos declarados, como uma classe e um módulo, têm o mesmo nome no mesmo namespace.  
  
 **ID do erro:** BC30179  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a declaração de um dos elementos para um namespace diferente.  
  
-   Altere o nome de um dos elementos.

# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-zero"></a>Argumento &#39; &lt;argumentname&gt;&#39; deve ser maior que ou igual a zero
Um argumento não é válido porque ele é menor que zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a ortografia dos argumentos. Um nome de variável incorreta implicitamente pode criar uma variável numérica que é inicializada a zero.  
  
2.  Verifique se as operações anteriores em variáveis na expressão, especialmente aquelas de outros procedimentos passadas para o procedimento como argumentos.

# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefixo de namespace XML &#39;prefixo&#39; é reservado para uso por XML e o namespace URI não pode ser alterado.
Um reservado prefixo de namespace XML, como `xmlns`, foi usado em um literal XML.  
  
 **ID do erro:** BC31152  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o prefixo de namespace XML para um valor que não está reservado.

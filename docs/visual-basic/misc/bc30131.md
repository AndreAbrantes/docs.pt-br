# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>Atributo de segurança '\<attributename >' não pode ser aplicado a um módulo
O common language runtime não aceita o atributo de segurança em um módulo.

**ID do erro:** BC30131

## <a name="to-correct-this-error"></a>Para corrigir este erro

- Remova o atributo da declaração.

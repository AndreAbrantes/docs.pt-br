# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Instruções 'Get' não são mais suportadas (Visual Basic)
`Get` Não há suporte para instruções. Funcionalidade de e/s de arquivo está normalmente disponível no `Microsoft.VisualBasic` namespace, mas a versão de destino do .NET Compact Framework não dar suporte a ele.  
  
 **ID do erro:** BC30767  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Executar operações de arquivo com funções definidas no <xref:System.IO> namespace.

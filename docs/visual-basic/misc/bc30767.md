# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Não há mais suporte para instruções ' Get ' (Visual Basic)
Não há mais suporte para instruções `Get`. A funcionalidade de e/s de arquivo normalmente está disponível no namespace `Microsoft.VisualBasic`, mas a versão de destino do .NET Compact Framework não oferece suporte a ele.  
  
 **ID do erro:** BC30767  
  
## <a name="to-correct-this-error"></a>Para corrigir esse erro  
  
- Execute operações de arquivo com funções definidas no namespace <xref:System.IO>.

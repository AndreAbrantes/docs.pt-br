# <a name="39get39-statements-are-no-longer-supported-visual-basic"></a>&#39;Obter&#39; instruções não são mais suportadas (Visual Basic)
`Get` Não há suporte para instruções. A funcionalidade de e/s de arquivo está disponível na `Microsoft.VisualBasic` namespace, mas a versão de destino do .NET Compact Framework não suporta.  
  
 **ID do erro:** BC30767  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Executar operações de arquivo com funções definidas no <xref:System.IO> namespace.

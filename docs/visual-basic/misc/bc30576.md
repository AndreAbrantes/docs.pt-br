# <a name="statement-does-not-declare-a-39get39-or-39set39-method"></a>A instrução não declara um &#39;obter&#39; ou &#39;definir&#39; método
Sua instrução falha ao não fornecer uma um `Get` ou `Set` declaração em torno de uma `Property` procedimento. Uma propriedade é definida como um bloco de código entre as `Property` e `End Property` instruções. Dentro deste bloco, cada `Property` procedimento aparece como um bloco interno incluído dentro de uma instrução de declaração e uma instrução end.  
  
 **ID do erro:** BC30576  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça um `Get` ou `Set` declaração.

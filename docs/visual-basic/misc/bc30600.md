# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'Variável de WithEvents não gera nenhum evento de instância que são acessível para'\<containername >'
O objeto que você está fazendo referência não gera eventos acessíveis.  
  
 **ID do erro:** BC30600  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Modifique os modificadores de acesso para o evento ou escolha um evento que é acessível no contexto atual.

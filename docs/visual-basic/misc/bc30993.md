# <a name="property-39ltpropertynamegt39-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="10832-102">Propriedade '&lt;propertyname&gt;' não pode ser inicializado em uma expressão de inicializador de objeto porque todas as sobrecargas acessíveis requerem argumentos</span><span class="sxs-lookup"><span data-stu-id="10832-102">Property &#39;&lt;propertyname&gt;&#39; cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="10832-103">Os membros inicializados em uma lista de inicializadores de objeto devem ser campos ou propriedades.</span><span class="sxs-lookup"><span data-stu-id="10832-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="10832-104">Além disso, propriedades em uma lista de inicializadores não podem ter parâmetros.</span><span class="sxs-lookup"><span data-stu-id="10832-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="10832-105">A propriedade causando o erro está sobrecarregada, e cada uma das suas versões requer argumentos.</span><span class="sxs-lookup"><span data-stu-id="10832-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="10832-106">Portanto, a propriedade não pode ser inicializada em uma lista de inicializadores de objeto.</span><span class="sxs-lookup"><span data-stu-id="10832-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="10832-107">**ID do erro:** BC30993</span><span class="sxs-lookup"><span data-stu-id="10832-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="10832-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="10832-108">To correct this error</span></span>  
  
-   <span data-ttu-id="10832-109">Remova a propriedade que requer argumentos da lista do inicializador.</span><span class="sxs-lookup"><span data-stu-id="10832-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="10832-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="10832-110">Example</span></span>  
 <span data-ttu-id="10832-111">A seguinte classe contém três definições de propriedade: uma para `TotalItems` e dois para `Item`, que está sobrecarregado.</span><span class="sxs-lookup"><span data-stu-id="10832-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="10832-112">O `TotalItems` propriedade não requer nenhum argumento e pode ser inicializada em uma lista de inicialização de objeto, conforme mostrado na seguinte declaração.</span><span class="sxs-lookup"><span data-stu-id="10832-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="10832-113">O `Item` propriedade está sobrecarregada, e cada sobrecarga exige um argumento.</span><span class="sxs-lookup"><span data-stu-id="10832-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="10832-114">Portanto, `Item` não pode aparecer em uma lista de inicializadores de objeto.</span><span class="sxs-lookup"><span data-stu-id="10832-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="10832-115">Para evitar esse erro, inicialize o `Item` propriedade fora do inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="10832-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```

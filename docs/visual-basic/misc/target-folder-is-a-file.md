# <a name="target-folder-is-a-file"></a>Pasta de destino é um arquivo
Especificar um arquivo durante a tentativa de executar uma ação de pasta.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o caminho que você está fornecendo para certificar-se de que especificar uma pasta em vez de um arquivo.

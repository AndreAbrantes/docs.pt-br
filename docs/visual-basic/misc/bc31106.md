# <a name="property-accessors-cannot-be-declared-39ltaccessmodifiergt39-in-a-39notoverridable39-property"></a><span data-ttu-id="ec707-102">Acessadores de propriedade não podem ser declarados &#39; &lt;accessmodifier&gt; &#39; em um &#39;NotOverridable&#39; propriedade</span><span class="sxs-lookup"><span data-stu-id="ec707-102">Property accessors cannot be declared &#39;&lt;accessmodifier&gt;&#39; in a &#39;NotOverridable&#39; property</span></span>
<span data-ttu-id="ec707-103">Um [obter instrução](../../visual-basic/language-reference/statements/get-statement.md) ou [instrução Set](../../visual-basic/language-reference/statements/set-statement.md) em uma `NotOverridable` propriedade inclui o `Private` palavra-chave.</span><span class="sxs-lookup"><span data-stu-id="ec707-103">A [Get Statement](../../visual-basic/language-reference/statements/get-statement.md) or [Set Statement](../../visual-basic/language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="ec707-104">A seguinte linha de raciocínio explica por que `NotOverridable` e `Private` não podem ser combinados em um [declaração de propriedade](../../visual-basic/language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="ec707-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../../visual-basic/language-reference/statements/property-statement.md):</span></span>  
  
1.  <span data-ttu-id="ec707-105">Uma propriedade ou procedimento que não substitui uma propriedade de classe base ou um procedimento tem uma configuração padrão de [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="ec707-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span></span>  
  
2.  <span data-ttu-id="ec707-106">No entanto, uma propriedade ou procedimento em uma classe derivada que substitui uma propriedade de classe base ou um procedimento tem uma configuração padrão de [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span><span class="sxs-lookup"><span data-stu-id="ec707-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="ec707-107">Para encerrar a hierarquia de substituição, você pode declará-la `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="ec707-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="ec707-108">Este é o contexto apenas nos quais você pode usar `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="ec707-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="ec707-109">Ou seja, você pode usar `NotOverridable` apenas em combinação com [substitui](../../visual-basic/language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="ec707-109">That is, you can use `NotOverridable` only in combination with [Overrides](../../visual-basic/language-reference/modifiers/overrides.md).</span></span>  
  
3.  <span data-ttu-id="ec707-110">Se uma propriedade de classe base ou um procedimento é declarado [privada](../../visual-basic/language-reference/modifiers/private.md), uma classe derivada não pode substituir essa propriedade ou procedimento porque ele não pode acessá-lo.</span><span class="sxs-lookup"><span data-stu-id="ec707-110">If a base class property or procedure is declared [Private](../../visual-basic/language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="ec707-111">Por isso, você não pode usar `Private` em combinação com `Overridable`.</span><span class="sxs-lookup"><span data-stu-id="ec707-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4.  <span data-ttu-id="ec707-112">Para substituir uma propriedade ou procedimento, a propriedade ou procedimento de substituição deve ter não apenas a assinatura idêntica, mas também o mesmo nível de acesso.</span><span class="sxs-lookup"><span data-stu-id="ec707-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="ec707-113">Isso significa que uma propriedade ou procedimento de substituição não pode especificar `Private`, porque uma propriedade substituível ou procedimento não é possível especificar `Private`.</span><span class="sxs-lookup"><span data-stu-id="ec707-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5.  <span data-ttu-id="ec707-114">Porque você pode especificar `NotOverridable` somente em uma propriedade ou procedimento de substituição, você não pode combinar com `Private`.</span><span class="sxs-lookup"><span data-stu-id="ec707-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="ec707-115">Pelo mesmo raciocínio, os procedimentos de propriedade individuais (`Get` e `Set`) de uma propriedade de substituição não pode ser `Private`.</span><span class="sxs-lookup"><span data-stu-id="ec707-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>  
  
 <span data-ttu-id="ec707-116">**ID do erro:** BC31106</span><span class="sxs-lookup"><span data-stu-id="ec707-116">**Error ID:** BC31106</span></span>

-   <span data-ttu-id="ec707-118">Remover o `Private` palavra-chave do `Get` ou `Set` instrução, ou remova o `Overrides` e `NotOverridable` palavras-chave do `Property` instrução.</span><span class="sxs-lookup"><span data-stu-id="ec707-118">Remove the `Private` keyword from the `Get` or `Set` statement, or remove the `Overrides` and `NotOverridable` keywords from the `Property` statement.</span></span>

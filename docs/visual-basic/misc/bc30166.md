# <a name="cannot-create-an-instance-of-module-ltmodulenamegt"></a>Não é possível criar uma instância do módulo '&lt;modulename&gt;'
Um módulo existe apenas como uma única instância compartilhada e não não possível criar instâncias adicionais.  
  
 **ID do erro:** BC30166  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o módulo para uma classe ou substituí-lo no `New` cláusula com um nome de classe.

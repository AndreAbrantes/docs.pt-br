# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>Argumento &#39; &lt;argumentname&gt;&#39; não pode ser uma matriz multidimensional
A classificação de um argumento de matriz é maior que 1, mas ele só pode ser uma matriz unidimensional.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a ordem argumento para 1.

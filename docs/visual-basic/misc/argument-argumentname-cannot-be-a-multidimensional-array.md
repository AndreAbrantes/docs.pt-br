# <a name="argument-ltargumentnamegt-cannot-be-a-multidimensional-array"></a>Argumento '&lt;argumentname&gt;' não pode ser uma matriz multidimensional
A classificação de um argumento de matriz é maior que 1, mas ele só pode ser uma matriz unidimensional.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a classificação de argumento para 1.

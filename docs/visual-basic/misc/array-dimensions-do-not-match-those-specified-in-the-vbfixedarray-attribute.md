# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>As dimensões da matriz não correspondem às especificadas no atributo ' VBFixedArray '
As dimensões da matriz não correspondem às dimensões que você especificou no `VBFixedArray` atributo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Faça com que as dimensões da matriz `Dim` na instrução de matriz correspondam `VBFixedArray` às dimensões especificadas pelo atributo.

# <a name="array-dimensions-do-not-match-those-specified-in-the-39vbfixedarray39-attribute"></a>Dimensões de matriz não coincidem com aqueles especificados no &#39;VBFixedArray&#39; atributo
Dimensões de matriz não coincidem com as dimensões que você especificou no `VBFixedArray` atributo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique as dimensões de matriz na matriz `Dim` instrução corresponde as dimensões especificadas pelo `VBFixedArray` atributo.

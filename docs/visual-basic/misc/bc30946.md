# <a name="stop-request-is-pending"></a>Solicitação de interrupção está pendente
No depurador do Visual Studio, uma expressão especifica uma chamada de procedimento, mas há uma solicitação para parar o thread.  
  
 O depurador não tenta chamar um procedimento em um thread que não está ativo.  
  
 **ID do erro:** BC30946  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Se possível, determine a origem da solicitação para parar o thread, para que você pode impedir que isso aconteça novamente.  
  
-   Encerrar e reinicie a depuração.

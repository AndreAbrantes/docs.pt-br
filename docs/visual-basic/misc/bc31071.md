# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Estrutura em uma interface não pode ser declarada &#39; &lt;especificador&gt;&#39;
Um modificador inválido foi especificado com uma estrutura declarada em uma interface.  
  
 **ID do erro:** BC31071  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova modificadores como `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, ou `Overridable`.

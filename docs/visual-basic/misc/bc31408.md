# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>'&lt;specifier1&gt;'e'&lt;specifier2&gt;' não podem ser combinados
Dois especificadores incompatíveis, tais como `Shadows` e `Overrides` são usados juntos.  
  
 **ID do erro:** BC31408  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova um ou o especificador de outro.

# <a name="specifier1-and-specifier2-cannot-be-combined"></a>'\<specifier1 >' e '\<specifier2 >' não podem ser combinados
Dois especificadores incompatíveis, tais como `Shadows` e `Overrides` são usados juntos.  
  
 **ID do erro:** BC31408  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova um ou o especificador de outro.

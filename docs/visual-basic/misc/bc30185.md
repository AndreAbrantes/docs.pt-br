# <a name="enum-must-end-with-a-matching-end-enum"></a>'Enum' deve encerrar com um 'End Enum' correspondente
Uma `Enum` declaração ocorre sem um correspondente `End Enum` instrução. Um `End Enum` declaração deve ser usada para encerrar a enumeração.  
  
 **ID do erro:** BC30185  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se o `Enum` membros estão formatados corretamente.  
  
2. Adicionar um `End Enum` instrução até o final da enumeração.

# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;Enumeração&#39; deve terminar com uma correspondência &#39;End Enum&#39;
Um `Enum` instrução ocorre sem correspondente `End Enum` instrução. Um `End Enum` declaração deve ser usada para encerrar a enumeração.  
  
 **ID do erro:** BC30185  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o `Enum` membros estão formatados corretamente.  
  
2.  Adicionar um `End Enum` instrução até o fim da enumeração.

# <a name="the-value-of-ltargumentnamegt-must-be-a-positive-number"></a>O valor de &lt;argumentname&gt; deve ser um número positivo
O valor da <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> propriedade deve ser maior que zero.  
  
 O <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> propriedade especifica a quantidade de espaço livre em disco, em bytes, necessários antes que as mensagens podem ser gravadas no arquivo de log  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Defina o <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> propriedade como um número positivo.

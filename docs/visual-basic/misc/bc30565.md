# <a name="array-initializer-has-too-few-dimensions"></a>Inicializador de matriz tem um número insuficiente de dimensões
Você não forneceu dimensões suficientes no inicializador para sua matriz.  
  
 **ID do erro:** BC30565  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o inicializador de matriz para determinar quantas dimensões são necessário.  
  
-   Forneça o argumento ou argumentos ausentes.

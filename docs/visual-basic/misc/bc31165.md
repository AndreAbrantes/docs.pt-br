# <a name="expected-beginning-lt-for-an-xml-tag"></a>Abertura esperada '&lt;' em uma marca XML
A marca XML para um literal XML não tem início obrigatório ' <' caracteres.  
  
 **ID do erro:** BC31165  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o ' <' caractere para o início da marca XML para o literal XML.

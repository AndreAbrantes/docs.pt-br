# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>Abertura esperada &#39; &lt; &#39; para uma marca XML
A marca XML para um literal XML tem início necessário ' <' caracteres.  
  
 **ID do erro:** BC31165  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar a ' <' caractere para o início da marca XML para o literal XML.

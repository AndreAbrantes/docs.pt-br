# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39;&lt;nome&gt; &#39; já está declarado como um parâmetro deste método
O nome do parâmetro especificado já foi declarado para este método.  
  
 **ID do erro:** BC30734  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Renomeie o parâmetro para que ele seja exclusivo.

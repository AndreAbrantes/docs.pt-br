# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Catch&#39; instruções não são válidas na janela imediata
`Catch` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30715  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Catch` instruções do código de depuração.

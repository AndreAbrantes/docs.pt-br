# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' não está acessível neste contexto porque ele é '\<modificador >'
Você tentou acessar um método que não está acessível neste contexto devido a um modificador que foi aplicado a ele.  
  
 **ID do erro:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova o modificador.

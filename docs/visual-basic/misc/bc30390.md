# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>'&lt;qualifiedmethod&gt;'não está acessível neste contexto porque ele é'&lt;modificador&gt;'
Você tentou acessar um método que não está acessível neste contexto devido a um modificador que foi aplicado a ele.  
  
 **ID do erro:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o modificador.

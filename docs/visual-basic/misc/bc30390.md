# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39; &lt;qualifiedmethod&gt;&#39; não está acessível neste contexto porque é &#39;&lt; modificador&gt;&#39;
Você tentou acessar um método que não está acessível neste contexto devido a um modificador que foi aplicado a ele.  
  
 **ID do erro:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o modificador.

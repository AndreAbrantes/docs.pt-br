# <a name="39end-function39-expected"></a>&#39;Função end&#39; esperado
Um `End` instrução deve corresponder ao bloco encerra. Um `Function` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC30027  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `End Function` instrução até o final do procedimento.

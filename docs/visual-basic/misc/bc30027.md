# <a name="end-function-expected"></a>'End Function' esperado
Um `End` instrução deve corresponder ao bloco em que ele é encerrado. Um `Function` definição do procedimento não foi terminada corretamente.  
  
 **ID do erro:** BC30027  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicionar um `End Function` instrução até o final do procedimento.

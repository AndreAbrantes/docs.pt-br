# <a name="-expected"></a><span data-ttu-id="41448-102">'} ' esperado</span><span class="sxs-lookup"><span data-stu-id="41448-102">'}' expected</span></span>
<span data-ttu-id="41448-103">Um inicializador de matriz ou uma lista de restrições não foi encerrado de maneira válida.</span><span class="sxs-lookup"><span data-stu-id="41448-103">An array initializer or a constraint list has not been ended in a valid fashion.</span></span>

<span data-ttu-id="41448-104">Os valores de elemento com os quais inicializar uma matriz devem ser colocados entre chaves (`{}`).</span><span class="sxs-lookup"><span data-stu-id="41448-104">The element values with which to initialize an array must be enclosed in braces (`{}`).</span></span>

```vb

```

<span data-ttu-id="41448-105">Da mesma forma, as restrições em uma lista de restrições para um parâmetro de tipo genérico devem ser colocadas entre chaves.</span><span class="sxs-lookup"><span data-stu-id="41448-105">Similarly, the constraints in a constraint list for a generic type parameter must be enclosed in braces.</span></span>

```vb

```

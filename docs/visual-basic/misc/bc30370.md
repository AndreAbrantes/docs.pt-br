# <a name="-expected"></a><span data-ttu-id="3e86e-102">'}' esperado</span><span class="sxs-lookup"><span data-stu-id="3e86e-102">'}' expected</span></span>
<span data-ttu-id="3e86e-103">Um inicializador de matriz ou uma lista de restrição não foi encerrada de maneira válida.</span><span class="sxs-lookup"><span data-stu-id="3e86e-103">An array initializer or a constraint list has not been ended in a valid fashion.</span></span>  
  
 <span data-ttu-id="3e86e-104">Os valores de elemento com o qual inicializar uma matriz devem ser colocados entre chaves (`{}`).</span><span class="sxs-lookup"><span data-stu-id="3e86e-104">The element values with which to initialize an array must be enclosed in braces (`{}`).</span></span>

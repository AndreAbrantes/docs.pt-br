# <a name="-expected"></a><span data-ttu-id="78da6-102">'}' esperado</span><span class="sxs-lookup"><span data-stu-id="78da6-102">'}' expected</span></span>
<span data-ttu-id="78da6-103">Um inicializador de matriz ou uma lista de restrição não foi encerrada de maneira válida.</span><span class="sxs-lookup"><span data-stu-id="78da6-103">An array initializer or a constraint list has not been ended in a valid fashion.</span></span>  
  
 <span data-ttu-id="78da6-104">Os valores de elemento com o qual inicializar uma matriz devem ser colocados entre chaves (`{}`).</span><span class="sxs-lookup"><span data-stu-id="78da6-104">The element values with which to initialize an array must be enclosed in braces (`{}`).</span></span>  
  
```  
Dim demoArray() As Integer = New Integer() {1, 2, 3}   
```  
  
 <span data-ttu-id="78da6-105">Da mesma forma, as restrições em uma lista de restrição para um parâmetro de tipo genérico devem ser colocadas entre chaves.</span><span class="sxs-lookup"><span data-stu-id="78da6-105">Similarly, the constraints in a constraint list for a generic type parameter must be enclosed in braces.</span></span>  
  
```  
Public Class dictionaryMaker(Of t As {IComparable, IDisposable, New})   
```

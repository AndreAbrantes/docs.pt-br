# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;método&gt; &#39; &lt;methodname1&gt; &#39; devem ser declarados &#39;sobrecargas&#39; porque outro &#39; &lt;methodname2&gt; &#39; é declarado &#39; Sobrecargas&#39;
Outra propriedade ou método com o mesmo nome está declarado com o `Overloads` modificador.  
  
 **ID do erro:** BC31409  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicionar o `Overloads` modificador a propriedade ou método.  
  
2.  Remover o `Overloads` modificador de todas as propriedades ou métodos com o mesmo nome.

# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>'Declare' instruções em uma estrutura não podem ser declaradas '\<especificador >'
Um `Declare` declaração tem um especificador que não é válido dentro de um `Structure` declaração.  
  
 **ID do erro:** BC30791  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;Declarar&#39; instruções em uma estrutura não podem ser declaradas &#39; &lt;especificador&gt;&#39;
Um `Declare` declaração tem um especificador que não é válido dentro de um `Structure` declaração.  
  
 **ID do erro:** BC30791  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

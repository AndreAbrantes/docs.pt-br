# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace ou tipo '\<nome >' já foi importado
Foi feita uma tentativa para importar um namespace mais de uma vez.  
  
 **ID do erro:** BC31051  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover redundante `Imports` instruções.

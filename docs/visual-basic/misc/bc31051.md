# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace ou tipo &#39; &lt;nome&gt; &#39; já foi importado
Foi feita uma tentativa de importar um namespace mais de uma vez.  
  
 **ID do erro:** BC31051  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover redundante `Imports` instruções.

# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="be593-102">Não não possível inicializar propriedades expandidas</span><span class="sxs-lookup"><span data-stu-id="be593-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="be593-103">Uma propriedade implementada automaticamente pode ser inicializada como parte de sua declaração, mas não pode ser uma propriedade expandida.</span><span class="sxs-lookup"><span data-stu-id="be593-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="be593-104">**ID do erro:** BC36714</span><span class="sxs-lookup"><span data-stu-id="be593-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="be593-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="be593-105">To correct this error</span></span>  
  
-   <span data-ttu-id="be593-106">Use uma propriedade implementada automaticamente ou remova a inicialização da declaração de propriedade.</span><span class="sxs-lookup"><span data-stu-id="be593-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="be593-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="be593-107">Example</span></span>  
 <span data-ttu-id="be593-108">Os exemplos a seguir mostram ambas autoimplementadas e expandido de propriedades.</span><span class="sxs-lookup"><span data-stu-id="be593-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="be593-109">Propriedades autoimplementadas podem ser inicializadas usando a atribuição ou um `New` não podem ser a cláusula, mas as propriedades expandidas.</span><span class="sxs-lookup"><span data-stu-id="be593-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```

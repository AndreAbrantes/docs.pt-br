# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>VbStrConv. Wide e VbStrConv. Narrow não são aplicáveis à localidade especificada
O aplicativo está tentando usar os membros de enumeração de `VbStrConv` `Wide` ou `Narrow`, que não são aplicáveis à localidade especificada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Remova `VbStrConv.Wide` ou `VbStrConv.Narrow`.

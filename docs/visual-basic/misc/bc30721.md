# <a name="evaluation-of-expression-or-statement-stopped"></a>Avaliação de expressão ou instrução interrompida
O **parar avaliação** item de menu foi selecionado, fazendo com que a avaliação parar.  
  
 **ID do erro:** BC30721  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Permitir a execução continue sem selecionar **parar avaliação**.

# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parâmetro '&lt;parametername&gt;'no método de extensão'&lt;methodname&gt;'definido no'&lt;typename&gt;' já tem um argumento correspondente
Um argumento é passado por nome mais de uma vez na mesma chamada de procedimento.  
  
 **ID do erro:** BC36584  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se um dos nomes dos argumentos está incorreto.  
  
-   Remova o argumento duplicado da lista de argumentos.

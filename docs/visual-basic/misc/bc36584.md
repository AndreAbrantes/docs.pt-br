# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parâmetro &#39; &lt;parametername&gt;&#39; no método de extensão &#39;&lt; MethodName&gt;&#39; definido no &#39;&lt; TypeName&gt;&#39; já tem um argumento correspondente
Um argumento é passado por nome mais de uma vez na mesma chamada de procedimento.  
  
 **ID do erro:** BC36584  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se um dos nomes de argumento está incorreto.  
  
-   Remova o argumento duplicado da lista de argumentos.

# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>Parâmetro já declarado com o nome &#39; &lt;argumentname&gt;&#39;
Uma declaração de procedimento define dois argumentos com o mesmo nome. Todos os nomes de argumento em uma declaração de procedimento devem ser exclusivos.  
  
 **ID do erro:** BC30237  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome de um dos argumentos.

# <a name="sub-new-cannot-implement-interface-members"></a>'Sub New' não pode implementar membros de interface
`Sub New` é um construtor e ele não pode implementar membros.  
  
 **ID do erro:** BC31042  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova `Implements` instruções de `Sub New` procedimentos.

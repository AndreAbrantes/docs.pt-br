# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>'Sub New' não pode ser declarado '&lt;modificador&gt;'
Você tentou declarar uma `Sub New` com um modificador, como `Overrides`, que é inválido.  
  
 **ID do erro:** BC30364  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o modificador.

# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39; Sub novo &#39; não é possível declarar &#39; &lt;modificador&gt;&#39;
Você tentou declarar uma `Sub New` com um modificador, como `Overrides`, que é não é válido.  
  
 **ID do erro:** BC30364  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o modificador.

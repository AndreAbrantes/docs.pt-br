# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>'Sub New' não pode ser declarado como '&lt;modificador&gt;'
Você tentou declarar uma `Sub New` com um modificador, tais como `Overrides`, que é não é válido.  
  
 **ID do erro:** BC30364  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o modificador.

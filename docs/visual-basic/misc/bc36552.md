## <a name="example"></a><span data-ttu-id="5ec61-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="5ec61-110">Example</span></span>  
 <span data-ttu-id="5ec61-111">O primeiro parâmetro no exemplo a seguir indica que o método `Print` estende o tipo de dados `String`.</span><span class="sxs-lookup"><span data-stu-id="5ec61-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb

Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="5ec61-112">Quando o método de extensão é chamado da seguinte maneira, o parâmetro `str` no método é associado a `greeting`, a instância de `String` que chama `Print`.</span><span class="sxs-lookup"><span data-stu-id="5ec61-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="5ec61-113">O compilador usará `greeting` como o argumento para o método de extensão `Print`.</span><span class="sxs-lookup"><span data-stu-id="5ec61-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```vb  
greeting.Print()  
```

# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Referência necessária para o módulo &#39; &lt;modulename&gt; &#39; que contém o tipo &#39; &lt;membername&gt;&#39;
Referência necessária para o módulo '\<modulename >' contendo o tipo '\<membername >'. Adicione uma ao seu projeto.  
  
 O projeto contém um tipo que depende de outro tipo de um assembly que não está incluído nas referências de projeto.  
  
 **ID do erro:** BC30653  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência de projeto para os tipos dependentes no projeto.

# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>Referência obrigatória ao módulo '\<modulename >' que contém o tipo '\<membername >'
Referência obrigatória ao módulo '\<modulename >' que contém o tipo '\<membername >'. Adicione uma ao seu projeto.  
  
 O projeto contém um tipo que depende de outro tipo de um assembly que não está incluído nas referências do projeto.  
  
 **ID do erro:** BC30653  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência de projeto para os tipos dependentes no projeto.

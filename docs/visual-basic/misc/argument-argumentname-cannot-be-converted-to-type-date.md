# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>Argumento &#39; &lt;argumentname&gt; &#39; não pode ser convertido no tipo &#39;data&#39;
Uma conversão tentou alterar uma variável para o tipo `Date`, mas o valor não é reconhecido como uma data válida.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Localize uma conversão que é significativa.

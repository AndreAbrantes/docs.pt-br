# <a name="set-method-cannot-have-more-than-one-parameter"></a>Método 'Set' não pode ter mais de um parâmetro
Mais de um parâmetro é definido para uma propriedade `Set` procedimentos.  
  
 **ID do erro:** BC31063  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover parâmetros extras de propriedade `Set` procedimentos.

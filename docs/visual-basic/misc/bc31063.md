# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39; Definir &#39; método não pode ter mais de um parâmetro
Mais de um parâmetro está definido para uma propriedade `Set` procedimentos.  
  
 **ID do erro:** BC31063  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover parâmetros extras de propriedade `Set` procedimentos.

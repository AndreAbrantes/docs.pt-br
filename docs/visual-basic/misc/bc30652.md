# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Referência necessária para o assembly &#39; &lt;assemblyname&gt; &#39; que contém o tipo &#39; &lt;membername&gt;&#39;
Referência necessária para o assembly '\<assemblyname >' contendo o tipo '\<membername >'. Adicione uma ao seu projeto.  
  
 O projeto contém um tipo que depende de outro tipo de um assembly que não está incluído nas referências de projeto.  
  
 **ID do erro:** BC30652  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência de projeto para os tipos dependentes no projeto.

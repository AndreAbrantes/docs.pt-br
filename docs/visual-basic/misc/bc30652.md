# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>Referência obrigatória ao assembly '\<assemblyname >' que contém o tipo '\<membername >'
Referência obrigatória ao assembly '\<assemblyname >' que contém o tipo '\<membername >'. Adicione uma ao seu projeto.  
  
 O projeto contém um tipo que depende de outro tipo de um assembly que não está incluído nas referências do projeto.  
  
 **ID do erro:** BC30652  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência de projeto para os tipos dependentes no projeto.

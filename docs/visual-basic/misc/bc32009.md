# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Instruções de declaração de método devem ser o primeiro em uma linha lógica
Um `Function` ou `Sub` instrução segue um separador de instrução de dois-pontos (:). `Function` ou `Sub` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC32009  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39; &lt;nome&gt;&#39; é declarado no projeto &#39;&lt; projectname1&gt;&#39; que não é referenciado pelo projeto &#39;&lt; projectname2&gt;&#39;
Um elemento de programação é definido em um projeto ao qual o projeto atual não tem uma referência.  
  
 **ID do erro:** BC32004  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   No projeto atual, adicione uma referência ao projeto que contém o elemento desejado.

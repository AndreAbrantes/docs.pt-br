# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>'&lt;nome&gt;'é declarado no projeto'&lt;projectname1&gt;', que não é referenciado pelo projeto'&lt;projectname2&gt;'
Um elemento de programação é definido em um projeto ao qual o projeto atual não tem uma referência.  
  
 **ID do erro:** BC32004  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   No projeto atual, adicione uma referência ao projeto que contém o elemento desejado.

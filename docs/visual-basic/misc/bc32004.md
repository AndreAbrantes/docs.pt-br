# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>'\<nome >' é declarado no projeto '\<projectname1 > ', que não é referenciado pelo projeto'\<projectname2 >'
Um elemento de programação é definido em um projeto ao qual o projeto atual não tem uma referência.  
  
 **ID do erro:** BC32004  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   No projeto atual, adicione uma referência ao projeto que contém o elemento desejado.

<span data-ttu-id="caf76-103">Um argumento de tipo foi especificado em uma chamada para um método de extensão que não tem nenhum parâmetro genérico ou não tem nenhum parâmetro genérico para o qual um tipo não tiver sido especificado.</span><span class="sxs-lookup"><span data-stu-id="caf76-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="caf76-104">Por exemplo, o código a seguir faz com que esse erro.</span><span class="sxs-lookup"><span data-stu-id="caf76-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.

    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```

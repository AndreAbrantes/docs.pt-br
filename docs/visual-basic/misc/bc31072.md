# <a name="warning-treated-as-error-lterrorgt"></a>Aviso tratado como erro: &lt;erro&gt;
Foi encontrado um aviso quando executar um assembly compilado com o **tratar avisos como erros** opção ativada.  
  
 **ID do erro:** BC31072  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Corrija o aviso.  
  
2.  Compile o assembly com o **tratar avisos como erros** opção desativada.

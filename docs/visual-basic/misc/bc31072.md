# <a name="warning-treated-as-error-error"></a>Aviso tratado como erro: \<erro >
Foi encontrado um aviso quando a execução de um assembly compilado com o **tratar avisos como erros** opção ativada.  
  
 **ID do erro:** BC31072  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Corrigi o aviso.  
  
2.  Compila o assembly com o **tratar avisos como erros** opção desativada.

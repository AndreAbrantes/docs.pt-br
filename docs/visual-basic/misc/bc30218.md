# <a name="39lib39-expected"></a>&#39; Lib &#39; esperado
Um `Declare` instrução Especifica um nome de biblioteca sem usar o `Lib` palavra-chave.  
  
 **ID do erro:** BC30218  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o `Lib` palavra-chave antes do nome da biblioteca.

# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;Herda&#39; instruções devem preceder todas as declarações em uma interface
Um `Inherits` instrução deve ser a primeira linha não vazia e que não seja de comentários em uma definição de classe.  
  
 **ID do erro:** BC30357  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mover o `Inherits` declaração de modo que ela preceda todas as declarações.

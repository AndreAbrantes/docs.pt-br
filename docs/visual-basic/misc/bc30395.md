# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>'\<especificador >' não é válido em uma declaração de estrutura
Você usou como um especificador `Dim` ou `Override`, que é não é válido em um `Structure` declaração.  
  
 **ID do erro:** BC30395  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

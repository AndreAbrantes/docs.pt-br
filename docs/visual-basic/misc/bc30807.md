# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39; Permitir que o &#39; e &#39; Definir &#39; Não há suporte para instruções de atribuição
`Let`e `Set` não pode ser usado em atribuições.  
  
 **ID do erro:** BC30807  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Omitir `Let` e `Set` das instruções de atribuição; por exemplo:  
  
     `X = 2`

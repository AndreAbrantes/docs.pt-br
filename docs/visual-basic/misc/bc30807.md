# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>'Let' e 'Set' não há suporte para instruções de atribuição
`Let` e `Set` não pode ser usado em atribuições.  
  
 **ID do erro:** BC30807  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Omitir `Let` e `Set` das instruções de atribuição; por exemplo:  
  
     `X = 2`

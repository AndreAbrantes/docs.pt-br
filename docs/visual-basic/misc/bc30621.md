# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39; End Structure &#39; deve ser precedido por uma correspondência &#39; Estrutura &#39;
O `End Structure` instrução só pode ser usada no final de um bloco de estrutura que comece com um `Structure` instrução.  
  
 **ID do erro:** BC30621  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o bloco de estrutura atual começa com um `Structure` instrução.

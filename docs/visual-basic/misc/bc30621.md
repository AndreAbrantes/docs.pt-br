# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>'End Structure' deve ser precedido por uma 'Structure' correspondente
O `End Structure` instrução só pode ser usada no final de um bloco de estrutura que começa com um `Structure` instrução.  
  
 **ID do erro:** BC30621  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o bloco de estrutura atual começa com um `Structure` instrução.

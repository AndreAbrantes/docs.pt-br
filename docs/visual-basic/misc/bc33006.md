# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39;End Operator&#39; deve ser a primeira instrução em uma linha
Um `End Operator` instrução segue um dois-pontos (`:`) separador de instrução. `End Operator` deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC33006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>'End Operator' deve ser a primeira instrução em uma linha
Uma `End Operator` instrução segue um dois-pontos (`:`) separador de instrução. `End Operator` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC33006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

# <a name="xml-dtds-are-not-supported"></a>Não há suporte para DTDs de XML
Uma definição de tipo de documento (DTD) foi especificado como um literal XML.  
  
 **ID do erro:** BC31175  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o DTD. Use a definição de esquema XML (XSD) para especificar o esquema de um documento XML.

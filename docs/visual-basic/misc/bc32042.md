# <a name="too-few-type-arguments-to-genericmethodname"></a>Muito poucos argumentos de tipo para '\<genericMethodName >'
Um método genérico terá sido chamado com menos argumentos de tipo de parâmetros de tipo.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC32042  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione argumentos de tipo para sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo no método genérico que você está invocando.

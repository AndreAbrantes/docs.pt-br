# <a name="too-few-type-arguments-to-39ltgenericmethodnamegt39"></a>Muito poucos argumentos de tipo para &#39; &lt;genericMethodName&gt;&#39;
Um método genérico ter sido chamado com menos argumentos de tipo de parâmetros de tipo.  
  
 Quando você invoca um método genérico, você deve fornecer um argumento de tipo para cada parâmetro de tipo definido pelo método.  
  
 **ID do erro:** BC32042  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione argumentos de tipo para a lista de argumentos de tipo para que haja um para cada parâmetro de tipo no método genérico que está chamando.

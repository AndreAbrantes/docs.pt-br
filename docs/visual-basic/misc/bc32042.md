# <a name="too-few-type-arguments-to-genericmethodname"></a>Poucos argumentos de tipo para ' \<genericMethodName> '

Um método genérico foi invocado com menos argumentos de tipo do que os parâmetros de tipo.  
  
 Quando você invoca um método genérico, deve fornecer um argumento de tipo para cada parâmetro de tipo definido por esse método.  
  
 **ID do erro:** BC32042  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione argumentos de tipo à sua lista de argumentos de tipo para que haja um para cada parâmetro de tipo no método genérico que você está invocando.

# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39; &lt;modulename&gt;&#39; é um módulo e não pode ser referenciado como um assembly
Foi feita uma tentativa para usar um módulo como um assembly.  
  
 **ID do erro:** BC31076  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a referência a um assembly.

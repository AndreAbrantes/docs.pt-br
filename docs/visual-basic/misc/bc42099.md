# <a name="unused-local-constant-ltconstantgt"></a>Constante local não utilizada: '&lt;constante&gt;'
Uma constante local foi declarada que não é usado em qualquer lugar no seu código.  
  
 **ID do erro:** BC42099  
  
### <a name="to-remove-this-warning"></a>Para remover este aviso  
  
1.  Esse erro é um aviso e pode ser ignorado. Para remover o aviso, remova a declaração de constante não utilizada do seu código.

# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;nome&gt; &#39; é ambíguo nos objetos do aplicativo &#39; &lt;lista&gt;&#39;
Você forneceu um nome que é ambíguo e, portanto, está em conflito com outro nome. O compilador do Visual Basic não tem nenhuma regra de resolução de conflitos; Você deve resolver a ambiguidade de nomes por conta própria. A lista fornecida nos detalhes da mensagem de erro, os objetos de aplicativo onde o nome é ambíguo.  
  
 **ID do erro:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Qualifica totalmente o nome.

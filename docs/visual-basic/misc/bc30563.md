# <a name="name-is-ambiguous-in-the-application-objects-list"></a>'\<nome >' é ambíguo nos objetos de aplicativo\<lista >'
Você forneceu um nome que é ambíguo e, portanto, entra em conflito com outro nome. O compilador do Visual Basic não tem nenhuma regra de resolução de conflitos; Você deve resolver a ambiguidade de nomes por conta própria. A lista fornecida nos detalhes da mensagem de erro, os objetos de aplicativo em que o nome é ambíguo.  
  
 **ID do erro:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Qualifica totalmente o nome.

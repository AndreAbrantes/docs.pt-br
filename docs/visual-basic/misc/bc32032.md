# <a name="end-function-must-be-the-first-statement-on-a-line"></a>'End Function' deve ser a primeira instrução em uma linha
Um `End Function` instrução segue um separador de instrução de dois-pontos (:). `End Function` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC32032  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Separe várias declarações em linhas diferentes.

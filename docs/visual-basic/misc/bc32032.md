# <a name="39end-function39-must-be-the-first-statement-on-a-line"></a>'End Function' deve ser a primeira instrução em uma linha
Um `End Function` instrução segue um separador de declaração dois-pontos (:). `End Function`deve ser a única declaração na sua linha de código.  
  
 **ID do erro:** BC32032  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe as várias declarações em linhas diferentes.

<span data-ttu-id="eca68-103">Você fez uma chamada para um método sobrecarregado, mas o compilador encontrou dois ou mais sobrecargas com listas de parâmetros ao qual a lista de argumentos pode ser convertida e ele não é possível selecionar entre eles.</span><span class="sxs-lookup"><span data-stu-id="eca68-103">You have made a call to an overloaded method, but the compiler has found two or more overloads with parameter lists to which your argument list can be converted, and it cannot select among them.</span></span>  
  
 <span data-ttu-id="eca68-104">O compilador tenta corresponder ao máximo os tipos de dados na lista de argumentos de chamada e a lista de parâmetros de sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="eca68-104">The compiler attempts to match the data types in the calling argument list and the overload parameter list as closely as possible.</span></span> <span data-ttu-id="eca68-105">Ele requer uma conversão de ampliação de cada um dos seus argumentos em seu parâmetro correspondente, se a opção de verificação de tipo ([instrução Option Strict](../../visual-basic/language-reference/statements/option-strict-statement.md)) é `On` ou `Off`.</span><span class="sxs-lookup"><span data-stu-id="eca68-105">It requires a widening conversion from every one of your arguments to its corresponding parameter, whether the type checking switch ([Option Strict Statement](../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
 <span data-ttu-id="eca68-106">Se o compilador não encontra mais de uma sobrecarga que satisfaz o requisito de ampliação, em seguida, procura a sobrecarga que é mais específica para os tipos de dados do argumento, ou seja, que chama para a menor quantidade de ampliação.</span><span class="sxs-lookup"><span data-stu-id="eca68-106">If the compiler finds more than one overload satisfying the widening requirement, it then looks for the overload that is most specific for the argument data types, that is, that calls for the least amount of widening.</span></span> <span data-ttu-id="eca68-107">Ele gera essa mensagem de erro quando uma sobrecarga é mais específica tipo do argumento de um de dados, enquanto outra sobrecarga é mais específica para o tipo de dados do argumento para outra.</span><span class="sxs-lookup"><span data-stu-id="eca68-107">It generates this error message when one overload is more specific for one argument's data type while another overload is more specific for another argument's data type.</span></span> <span data-ttu-id="eca68-108">Para obter mais informações e um exemplo, consulte [resolução de sobrecarga](../../visual-basic/programming-guide/language-features/procedures/overload-resolution.md).</span><span class="sxs-lookup"><span data-stu-id="eca68-108">For more information and an example, see [Overload Resolution](../../visual-basic/programming-guide/language-features/procedures/overload-resolution.md).</span></span>  
  
 <span data-ttu-id="eca68-109">**ID do erro:** BC30521</span><span class="sxs-lookup"><span data-stu-id="eca68-109">**Error ID:** BC30521</span></span>

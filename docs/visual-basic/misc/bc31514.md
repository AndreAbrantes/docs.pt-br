# <a name="project-39ltprojectnamegt39-cannot-generate-a-reference-to-file-39ltfilenamegt39"></a>Projeto &#39; &lt;projectname&gt; &#39; não é possível gerar uma referência ao arquivo &#39; &lt;nome de arquivo&gt;&#39;
Projeto ' <`projectname`>' não é possível gerar uma referência ao arquivo ' <`filename`>'. Talvez seja necessário excluir e adicionar novamente essa referência, fazer uma recompilação completa da solução.  
  
 Ocorreu um erro ao criar uma referência a um arquivo.  
  
 **ID do erro:** BC31514  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a referência clicando com o item do **referências de projeto** no **Solution Explorer** e clicando em **remover**.  
  
2.  Restaure a referência com um clique direito **referências** no **Solution Explorer** e, em seguida, clicando em **adicionar referência**.

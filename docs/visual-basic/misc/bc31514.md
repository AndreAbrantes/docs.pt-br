# <a name="project-projectname-cannot-generate-a-reference-to-file-filename"></a>Projeto '\<projectname >' não é possível gerar uma referência ao arquivo '\<filename >'
Projeto ' <`projectname`>' não é possível gerar uma referência ao arquivo ' <`filename`>'. Talvez você precise excluir e adicionar novamente essa referência e, em seguida, fazer uma recompilação completa da solução.  
  
 Ocorreu um erro durante a criação de uma referência a um arquivo.  
  
 **ID do erro:** BC31514  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a referência clicando com o item da **referências de projeto** na **Gerenciador de soluções** e clicando em **remover**.  
  
2.  Restaure a referência ao clicar com o botão direito **referências** na **Gerenciador de soluções** e, em seguida, clicando em **Add Reference**.

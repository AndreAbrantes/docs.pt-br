# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Variável de controle de loop não pode ser uma propriedade ou uma matriz indexada de associação tardia
A variável usada para iterar por meio de um `For` loop deve ser do tipo de dados numérico.  
  
 **ID do erro:** BC30039  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere a declaração da variável de controle de loop para especificar `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, ou `Decimal`.

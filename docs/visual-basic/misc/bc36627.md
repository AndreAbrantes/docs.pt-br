# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Avaliação de expressões lambda não é válida no depurador
Foi inserida em uma expressão lambda a **Watch** ou **imediato** janela durante uma sessão de depuração. Expressões lambda não estão disponíveis em janelas de depuração.  
  
 **ID do erro:** BC36627  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Teste as expressões lambda no Editor de códigos.

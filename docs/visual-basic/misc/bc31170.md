# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>Caractere &#39; caractere &#39; (0 x&lt;hexadecimal&gt;) não é permitido em um nome XML
Um caractere não suportado está incluído no nome de um literal XML ou uma propriedade de eixo XML.  
  
 **ID do erro:** BC31170  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere sem suporte do nome do literal XML ou propriedade de eixo XML.

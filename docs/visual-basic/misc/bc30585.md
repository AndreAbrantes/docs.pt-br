# <a name="event-lteventnamegt-cannot-be-handled-because-it-is-not-accessible-from-ltnamegt"></a>Evento '&lt;eventname&gt;'não pode ser manipulado, porque ele não é acessível de'&lt;nome&gt;'
Tentativa de manipular um evento que não está acessível. Por exemplo, se um `Handles` variável é compartilhada, o método que manipula os eventos também deve ser compartilhado.  
  
 **ID do erro:** BC30585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se que o evento está acessível.

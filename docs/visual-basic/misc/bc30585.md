# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>Evento &#39; &lt;eventname&gt;&#39; não pode ser manipulada porque ele não está acessível &#39;&lt; nome&gt;&#39;
Você tentou manipular um evento que não está acessível. Por exemplo, se um `Handles` variável for compartilhada, o método de manipulação de eventos também deve ser compartilhado.  
  
 **ID do erro:** BC30585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o evento está acessível.

# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>Evento '\<eventname >' não pode ser manipulada porque ele não é acessível de '\<nome >'
Tentativa de manipular um evento que não está acessível. Por exemplo, se um `Handles` variável é compartilhada, o método que manipula os eventos também deve ser compartilhado.  
  
 **ID do erro:** BC30585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Verifique se que o evento está acessível.

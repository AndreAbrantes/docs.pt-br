# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="848e9-102">A variável 'ReadOnly' não pode ser o destino de uma atribuição em uma expressão lambda em um construtor</span><span class="sxs-lookup"><span data-stu-id="848e9-102">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>

<span data-ttu-id="848e9-103">Você fez referência a uma `ReadOnly` variável de dentro de uma expressão lambda, o que não é permitido.</span><span class="sxs-lookup"><span data-stu-id="848e9-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="848e9-104">O código a seguir causa esse erro enviando a `ReadOnly` variável `m` como o argumento para um `ByRef` parâmetro.</span><span class="sxs-lookup"><span data-stu-id="848e9-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="848e9-105">**ID do erro:** BC36602</span><span class="sxs-lookup"><span data-stu-id="848e9-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="848e9-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="848e9-106">To correct this error</span></span>  
  
- <span data-ttu-id="848e9-107">Se possível, altere `n` o parâmetro na função `Test` para um `ByVal` parâmetro, conforme mostrado no código a seguir.</span><span class="sxs-lookup"><span data-stu-id="848e9-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="848e9-108">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="848e9-108">To correct this error</span></span>  
  
- <span data-ttu-id="848e9-109">Atribua a `ReadOnly` variável `m` a uma variável local no construtor e use a variável local no lugar de `m` , conforme mostrado no código a seguir.</span><span class="sxs-lookup"><span data-stu-id="848e9-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```

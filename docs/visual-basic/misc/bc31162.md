# <a name="expected-closing-39gt39-for-xml-cdata-section"></a>Fechamento esperado &#39;]] &gt;&#39; seção CDATA do XML
O fechamento necessário ausente de um literal XML CDATA ']] >' caracteres.  
  
 **ID do erro:** BC31162  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento necessários ']] >' caracteres ao final do literal XML CDATA.

# <a name="expected-closing--for-xml-cdata-section"></a>Fechamento esperado ']] >' para a seção CDATA XML
Um literal CDATA XML está faltando o fechamento necessários ']] >' caracteres.  
  
 **ID do erro:** BC31162  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento necessários ']] >' caracteres até o final do literal CDATA XML.

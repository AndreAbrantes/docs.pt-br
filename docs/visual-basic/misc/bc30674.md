# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39; End SyncLock &#39; deve ser precedido por uma correspondência &#39; SyncLock &#39;
`SyncLock`blocos começam com o `SyncLock` palavra-chave e terminar com o `End``SyncLock` construir.  
  
 **ID do erro:** BC30674  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o `SyncLock` bloco começa com um `SyncLock` instrução.

# <a name="argument-39ltargumentnamegt39-is-not-valid-for-the-array"></a>Argumento &#39; &lt;argumentname&gt;&#39; não é válido para a matriz
Uma matriz contém um argumento que não é válido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique a ortografia dos argumentos na expressão. Um nome de variável incorreta implicitamente pode criar uma variável numérica que é inicializada a zero.

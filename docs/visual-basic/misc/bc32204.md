# <a name="event-name-length-cannot-exceed-1011-characters"></a>Comprimento do nome do evento não pode exceder 1011 caracteres
O nome ou identificador de todos os eventos declarados com uma `Event` instrução é limitada a 1011 caracteres.  
  
 **ID do erro:** BC32204  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reduza o comprimento do identificador.

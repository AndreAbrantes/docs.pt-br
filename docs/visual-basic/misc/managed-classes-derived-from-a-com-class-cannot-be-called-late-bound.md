# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Classes gerenciadas derivadas de uma classe COM não podem ser chamados de associação tardia.
Você tentou fazer uma chamada de associação tardia para uma classe gerenciada derivada de uma classe COM; Não há suporte para essas chamadas.  
  
### <a name="to-correct-the-problem"></a>Para corrigir o problema  
  
-   Tornar a chamada de associação inicial.

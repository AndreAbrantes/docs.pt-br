# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39; &lt;modificador&gt;&#39; não é válido em uma declaração de variável local
Um `Dim` instrução dentro de um procedimento contém uma palavra-chave inválida, como `Overloads`.  
  
 **ID do erro:** BC30247  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a palavra-chave inválida do `Dim` instrução.

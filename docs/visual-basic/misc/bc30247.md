# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>'\<modificador >' não é válido em uma declaração de variável local
Um `Dim` instrução dentro de um procedimento contém uma palavra-chave inválida, como `Overloads`.  
  
 **ID do erro:** BC30247  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a palavra-chave inválida do `Dim` instrução.

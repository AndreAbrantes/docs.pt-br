# <a name="39redim39-preserve-operand-cannot-be-nothing"></a>&#39;ReDim&#39; preservar operando não pode ser Nothing
Um `ReDim` instrução tenta usar o `Preserve` palavra-chave para alterar uma dimensão de uma matriz que não é da última dimensão, mas não fornece um valor válido para seu operando.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Alterar o `Preserve` operando como um valor válido.

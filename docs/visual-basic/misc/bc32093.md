# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>&#39; do &#39; necessário ao especificar argumentos de tipo para um tipo ou método genérico
Uma declaração tenta construir um tipo de um tipo genérico ou chamar um método genérico, sem usar um [de](../../visual-basic/language-reference/statements/of-clause.md) cláusula.  
  
 O [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] sintaxe para tipos genéricos chamadas para os parâmetros de tipo e argumentos de tipo para que sejam introduzidos pelo `Of` palavra-chave. Além disso, a lista de parâmetros de tipo ou a lista de argumentos de tipo deve ser colocada entre parênteses.  
  
 **ID do erro:** BC32093  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Incluir o `Of` palavra-chave no início do argumento de tipo lista e coloque a lista por parênteses.

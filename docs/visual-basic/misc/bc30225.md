# <a name="option-specifier-statement-can-only-appear-once-per-file"></a>' Opção \<especificador >' instrução só pode aparecer uma vez por arquivo
O `Option Compare`, `Option Explicit`, e `Option Strict` declarações podem ser utilizadas somente uma vez no mesmo arquivo de origem.  
  
 **ID do erro:** BC30225  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova os duplicados `Option` instrução.

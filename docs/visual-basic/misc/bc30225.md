# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39; opção &lt;especificador&gt;&#39; instrução pode aparecer apenas uma vez por arquivo
O `Option Compare`, `Option Explicit`, e `Option Strict` instruções podem ser usadas somente uma vez no mesmo arquivo de origem.  
  
 **ID do erro:** BC30225  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover a duplicação `Option` instrução.

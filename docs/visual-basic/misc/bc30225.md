# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>' Opção &lt;especificador&gt;' instrução só pode aparecer uma vez por arquivo
O `Option Compare`, `Option Explicit`, e `Option Strict` instruções podem ser usadas somente uma vez no mesmo arquivo de origem.  
  
 **ID do erro:** BC30225  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover a duplicação `Option` instrução.

# <a name="disk-full-visual-basic"></a>Disco cheio (Visual Basic)
Não há espaço suficiente no disco para completar uma `Print`, `Write`, ou `FileClose` operação, ou para criar os arquivos necessários.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Crie mais espaço no disco por mover alguns arquivos para outro disco ou excluir alguns arquivos.

# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="7896e-102">Instrução 'Next' nomeia mais variáveis do que 'as instruções For' correspondente</span><span class="sxs-lookup"><span data-stu-id="7896e-102">'Next' statement names more variables than there are matching 'For' statements</span></span>
<span data-ttu-id="7896e-103">Loops aninhados são finalizados com um único `Next` declaração que especifica as variáveis de loop mais que os loops in a aninhada, como no exemplo a seguir:</span><span class="sxs-lookup"><span data-stu-id="7896e-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```

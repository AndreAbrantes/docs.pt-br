# <a name="property-accessors-cannot-be-declared-39ltkeywordgt39"></a>Acessadores de propriedade não podem ser declarados '&lt;palavra-chave&gt;'
A `Get` ou `Set` declaração de procedimento inclui uma palavra-chave não é válida em um procedimento de propriedade.  
  
 O [instrução Get](../../visual-basic/language-reference/statements/get-statement.md) e [instrução Set](../../visual-basic/language-reference/statements/set-statement.md) permitir um modificador de acesso (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **ID do erro:** BC31099  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a palavra-chave inválida do `Get` ou `Set` instrução.

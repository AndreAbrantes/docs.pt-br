# <a name="object-doesn39t-support-named-arguments"></a>Objeto &#39; suporta argumentos nomeados
Você tentou acessar um objeto cujos métodos não dão suporte a argumentos nomeado.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifica argumentos por posição ao acessar métodos neste objeto. Consulte a documentação do objeto para obter mais informações sobre as posições de argumento e tipos.

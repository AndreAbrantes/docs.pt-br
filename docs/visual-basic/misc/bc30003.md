# <a name="next-expected"></a>'Next' esperado
Uma `On Error Resume` constructo ocorre sem um Concluindo `Next`.  
  
 **ID do erro:** BC30003  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione a `Next` palavra-chave para o final do `On Error` instrução.

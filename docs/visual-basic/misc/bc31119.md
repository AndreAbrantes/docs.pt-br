# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>'End RemoveHandler' deve ser a primeira instrução em uma linha
Um `End RemoveHandler` instrução segue um separador de declaração dois-pontos (:). `End RemoveHandler`deve ser a única declaração na sua linha de código.  
  
 **ID do erro:** BC31119  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe as várias declarações em linhas diferentes.

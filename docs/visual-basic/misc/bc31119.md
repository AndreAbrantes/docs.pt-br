# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39;End RemoveHandler&#39; deve ser a primeira instrução em uma linha
Um `End RemoveHandler` instrução segue um separador de declaração dois-pontos (:). `End RemoveHandler` deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC31119  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

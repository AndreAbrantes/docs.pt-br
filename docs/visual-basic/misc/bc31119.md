# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>'End RemoveHandler' deve ser a primeira instrução em uma linha
Um `End RemoveHandler` instrução segue um separador de instrução de dois-pontos (:). `End RemoveHandler` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC31119  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.

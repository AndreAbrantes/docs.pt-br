# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Throw &#39; instruções não são válidas na janela imediata
`Throw`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30725  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Throw` instruções do código de depuração.

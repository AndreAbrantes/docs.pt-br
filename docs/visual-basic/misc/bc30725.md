# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' Throw ' não são válidas na janela Immediate
`Throw` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30725  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Throw` instruções do código de depuração.

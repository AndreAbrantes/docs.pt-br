# <a name="loop-control-variable-of-type-39lttypenamegt39-does-not-implement-the-39systemicomparable39-interface"></a>Loop de variável de controle do tipo &#39; &lt;typename&gt;&#39; não implementa o &#39; System. IComparable &#39; interface
Uma variável é de um tipo que não implementa o `System.IComparable` interface. `System.IComparable`deve ser implementada por uma classe ou um tipo de valor.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que a variável é um tipo de valor.

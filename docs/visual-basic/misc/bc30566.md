# <a name="array-initializer-has-too-many-dimensions"></a>Inicializador de matriz tem muitas dimensões
Número excessivo de dimensões no inicializador foram fornecidas para sua matriz.  
  
 **ID do erro:** BC30566  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique seu inicializador de matriz para determinar quantas dimensões são necessárias.  
  
-   Remova o argumento extra ou os argumentos.

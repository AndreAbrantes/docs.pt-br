# <a name="project-ltprojectnamegt-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-ltclassnamegt"></a>Projeto '&lt;NomeDoProjeto&gt;'faz uma referência indireta ao assembly'&lt;assemblyname&gt;que contém&lt;classname&gt;'
Projeto '\<projectname >' faz uma referência indireta ao assembly '\<assemblyname >' que contém '\<classname >'. Adicione uma referência a '\<assemblyname >' ao seu projeto.  
  
 Este projeto contém membros que dependem de um assembly que não está contido nas referências do projeto.  
  
 **ID do erro:** BC31515  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência ao assembly clicando **referências** na **Gerenciador de soluções** e, em seguida, clicando em **Add Reference**.

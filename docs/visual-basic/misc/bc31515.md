# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Projeto &#39; &lt;projectname&gt;&#39; faz uma referência indireta ao assembly &#39;&lt; AssemblyName&gt;&#39; que contém &#39;&lt; nome da classe&gt;&#39;
Projeto '\<projectname >' faz uma referência indireta ao assembly '\<assemblyname >' que contém '\<classname >'. Adicione uma referência a '\<assemblyname >' ao seu projeto.  
  
 Este projeto contém membros que dependem de um assembly que não está contido em referências do projeto.  
  
 **ID do erro:** BC31515  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma referência ao assembly **referências** no **Solution Explorer** e, em seguida, clicando em **adicionar referência**.

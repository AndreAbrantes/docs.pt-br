# <a name="unable-to-find-entry-point-39ltnamegt39-in-dll-39ltfilenamegt39-lterrorgt"></a>Não é possível localizar o ponto de entrada &#39; &lt;nome&gt;&#39; no DLL &#39;&lt; nome de arquivo&gt;&#39;: &lt;erro&gt;
O nome do procedimento não pôde ser encontrado no DLL especificado.  
  
 **ID do erro:** BC31014  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se que o nome do procedimento esteja escrito corretamente e se o uso de maiusculas e minúsculas corresponde da função na DLL.

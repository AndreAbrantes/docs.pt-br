# <a name="unable-to-find-entry-point-ltnamegt-in-dll-ltfilenamegt-lterrorgt"></a>Não é possível localizar o ponto de entrada '&lt;nome&gt;'na DLL'&lt;filename&gt;': &lt;erro&gt;
O nome do procedimento não pôde ser encontrado na DLL especificada.  
  
 **ID do erro:** BC31014  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se que o nome do procedimento esteja escrito corretamente e se as maiusculas e minúsculas corresponde da função na DLL.

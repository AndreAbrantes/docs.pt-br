# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Caractere &#39; caractere &#39; (0 x&lt;hexadecimal&gt;) não é permitido no início de um nome XML
Um caractere não suportado está incluído no início do nome de um literal XML ou uma propriedade de eixo XML.  
  
 **ID do erro:** BC31169  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere sem suporte do início do nome do literal XML ou propriedade de eixo XML.

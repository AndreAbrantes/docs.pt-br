# <a name="character-character-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Caractere 'caractere' (0x&lt;hexadecimal&gt;) não é permitido no início de um nome XML
Um caractere sem suporte está incluído no início do nome de um literal XML ou uma propriedade de eixo XML.  
  
 **ID do erro:** BC31169  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere sem suporte desde o início do nome do literal XML ou a propriedade de eixo XML.

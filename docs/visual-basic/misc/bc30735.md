# <a name="type-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Tipo em um módulo não pode ser declarado &#39; &lt;especificador&gt;&#39;
Um tipo com modificadores inválidos foi declarado dentro de um módulo.  
  
 **ID do erro:** BC30735  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não use `Protected` ou `Protected Friend` modificadores em tipos declarados dentro de um módulo.

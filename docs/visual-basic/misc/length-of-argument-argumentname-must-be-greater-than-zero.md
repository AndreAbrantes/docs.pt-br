# <a name="length-of-argument-ltargumentnamegt-must-be-greater-than-zero"></a>Comprimento do argumento '&lt;argumentname&gt;' deve ser maior que zero
Um argumento tem um valor que é igual ou menor que zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça um argumento que é maior que zero.

# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>Comprimento do argumento '\<argumentname >' deve ser maior que zero
Um argumento tem um valor que é igual ou menor que zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Forneça um argumento que é maior que zero.

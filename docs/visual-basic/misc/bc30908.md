<span data-ttu-id="33f40-103">Uma definição de classe ou interface inclui um [instrução Inherits](../../visual-basic/language-reference/statements/inherits-statement.md) que especifica um tipo aninhado dentro dele.</span><span class="sxs-lookup"><span data-stu-id="33f40-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="33f40-104">Herança deve ser linear, não circular.</span><span class="sxs-lookup"><span data-stu-id="33f40-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="33f40-105">Um tipo não pode herdar de um tipo que herda dela.</span><span class="sxs-lookup"><span data-stu-id="33f40-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="33f40-106">Uma restrição relacionada é que um tipo não pode herdar de um tipo que ainda não foi definido.</span><span class="sxs-lookup"><span data-stu-id="33f40-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="33f40-107">Herança envolve a capacidade de reutilizar os membros da classe base, que por sua vez requer que esses membros estejam definidos.</span><span class="sxs-lookup"><span data-stu-id="33f40-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="33f40-108">Portanto, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] não é possível compilar o código como o exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="33f40-108">Therefore, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] cannot compile code such as the following example.</span></span>  
  
```

    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```

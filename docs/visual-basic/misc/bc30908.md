<span data-ttu-id="e08c2-103">Uma definição de classe ou interface inclui um [instrução Inherits](../../visual-basic/language-reference/statements/inherits-statement.md) que especifica um tipo aninhado dentro dele.</span><span class="sxs-lookup"><span data-stu-id="e08c2-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="e08c2-104">Herança deve ser linear, circular não.</span><span class="sxs-lookup"><span data-stu-id="e08c2-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="e08c2-105">Um tipo não pode herdar de um tipo que herda dela.</span><span class="sxs-lookup"><span data-stu-id="e08c2-105">A type cannot inherit from a type that inherits from it.</span></span>

```  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```

# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>A instrução não pode aparecer fora de uma expressão lambda multiline/corpo de método
Instruções executáveis devem ser usadas dentro de procedimentos.  
  
 **ID do erro:** BC30689  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Mova a instrução em um procedimento ou uma sub-rotina.

# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Instrução não pode aparecer fora de uma lambda multilinha/corpo de método
Instruções executáveis devem ser usadas dentro de procedimentos.  
  
 **ID do erro:** BC30689  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Mova a instrução em um procedimento ou uma sub-rotina.

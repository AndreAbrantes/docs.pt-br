# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="4dc54-102">O tipo de ' \<variablename> ' não pode ser deduzido a partir de uma expressão contendo ' \<variablename> '</span><span class="sxs-lookup"><span data-stu-id="4dc54-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>

<span data-ttu-id="4dc54-103">O compilador não poderá inferir o tipo de dados de uma variável se a variável for usada para estabelecer seu valor inicial na declaração.</span><span class="sxs-lookup"><span data-stu-id="4dc54-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="4dc54-104">Por exemplo, com `Option Infer` definido como `On` , os exemplos a seguir não compilam:</span><span class="sxs-lookup"><span data-stu-id="4dc54-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="4dc54-105">Declarações</span><span class="sxs-lookup"><span data-stu-id="4dc54-105">Declarations</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="4dc54-106">`For` While</span><span class="sxs-lookup"><span data-stu-id="4dc54-106">`For` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="4dc54-107">`For Each` While</span><span class="sxs-lookup"><span data-stu-id="4dc54-107">`For Each` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="4dc54-108">**ID do erro:** BC30980</span><span class="sxs-lookup"><span data-stu-id="4dc54-108">**Error ID:** BC30980</span></span>

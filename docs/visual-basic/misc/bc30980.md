<span data-ttu-id="0f903-103">O compilador não pode inferir o tipo de dados de uma variável se a variável é usada no estabelecimento de seu valor inicial na declaração.</span><span class="sxs-lookup"><span data-stu-id="0f903-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>

    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="0f903-106">`For` loop</span><span class="sxs-lookup"><span data-stu-id="0f903-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="0f903-107">`For Each` loop</span><span class="sxs-lookup"><span data-stu-id="0f903-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.

    Next  
    ```

    ```  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="85f0a-106">`For`loop</span><span class="sxs-lookup"><span data-stu-id="85f0a-106">`For` loop</span></span>

    ```  
    ' Does not compile with Option Infer on.

    Next  
    ```


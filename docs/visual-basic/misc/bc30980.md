 <span data-ttu-id="8856f-104">Por exemplo, com `Option Infer` definida como `On`, não compilam os exemplos a seguir:</span><span class="sxs-lookup"><span data-stu-id="8856f-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="8856f-105">Declarações</span><span class="sxs-lookup"><span data-stu-id="8856f-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="8856f-106">`For`loop</span><span class="sxs-lookup"><span data-stu-id="8856f-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="8856f-107">`For Each`loop</span><span class="sxs-lookup"><span data-stu-id="8856f-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="8856f-108">**ID do erro:** BC30980</span><span class="sxs-lookup"><span data-stu-id="8856f-108">**Error ID:** BC30980</span></span>

<span data-ttu-id="f4872-103">O compilador não pode inferir o tipo de dados de uma variável se a variável é usada no estabelecimento de seu valor inicial na declaração.</span><span class="sxs-lookup"><span data-stu-id="f4872-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="f4872-104">Por exemplo, com `Option Infer` definido como `On`, os exemplos a seguir não são compilados:</span><span class="sxs-lookup"><span data-stu-id="f4872-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="f4872-105">Declarações</span><span class="sxs-lookup"><span data-stu-id="f4872-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="f4872-106">`For` loop</span><span class="sxs-lookup"><span data-stu-id="f4872-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="f4872-107">`For Each` loop</span><span class="sxs-lookup"><span data-stu-id="f4872-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```

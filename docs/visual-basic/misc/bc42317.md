# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>O parâmetro de tipo de comentário XML &lt;parâmetro&gt; não coincide com um parâmetro de tipo em correspondente &lt;palavra-chave&gt; instrução
Parâmetro de comentário XML \<parâmetro > não coincide com um parâmetro de tipo em correspondente \<palavra-chave > demonstrativo. O comentário XML será ignorado.  
  
 Uma marca de comentário XML foi especificada para alguns, mas nem todos os parâmetros de tipo em um método.  
  
 **ID do erro:** BC42317  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Fornece marcas de comentário XML para todos os parâmetros de tipo no método.

# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Conversões de &#39; &lt;typename1&gt; &#39; para &#39; &lt;typename2&gt; &#39; devem ser explícitas
Essa conversão não pode ser realizada implicitamente.  
  
 **ID do erro:** BC30664  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `CType`, ou uma conversão específica, como funções `CInt` ou `CDec`.

# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>Conversões de '&lt;typename1&gt;'para'&lt;typename2&gt;' devem ser explícitas
Essa conversão não pode ser realizada implicitamente.  
  
 **ID do erro:** BC30664  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `CType`, ou uma da conversão específica, como funções `CInt` ou `CDec`.

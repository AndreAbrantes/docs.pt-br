# <a name="event-39lteventnamegt39-cannot-be-found"></a>Evento '&lt;eventname&gt;' não pode ser encontrado
Você tentou manipular um evento que o compilador não consegue localizar.  
  
 **ID do erro:** BC30590  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que o nome do evento está digitado corretamente.

# <a name="event-eventname-cannot-be-found"></a>Evento '\<eventname >' não pode ser encontrado
Tentativa de manipular um evento que o compilador não pode localizar.  
  
 **ID do erro:** BC30590  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que o nome do evento está escrito corretamente.

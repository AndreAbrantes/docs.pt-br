# <a name="39raiseevent39-is-already-declared"></a>&#39;RaiseEvent&#39; já está declarado
Mais de um `RaiseEvent` declaração apareceu na declaração de evento personalizado. Um `RaiseEvent` declaração declara um procedimento usado para gerar o evento.  
  
 **ID do erro:** BC31129  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova a `RaiseEvent` instrução.

# <a name="type-lttypenamegt-is-imported-from-different-versions-of-assembly-ltassemblynamegt"></a>Tipo '&lt;typename&gt;'é importado de diferentes versões do assembly'&lt;assemblyname&gt;'
Tipo '\<typename >' é importado de diferentes versões do assembly '\<assemblyname >'. Versões diferentes do mesmo assembly não podem ser usadas no mesmo projeto.  
  
 Um tipo foi importado de diferentes versões do mesmo assembly, criando ambiguidade.  
  
 **ID do erro:** BC31525  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova referências a várias versões do mesmo assembly.

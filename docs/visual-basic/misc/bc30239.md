# <a name="relational-operator-expected"></a>Operador relacional esperado
Um `Case` instrução contém uma `Is` cláusula, mas nenhum operador de comparação, como `=` ou `>`. Se um `Case` instrução não inclui um operador `=` presume-se e `Is` não é usado.  
  
 **ID do erro:** BC30239  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remover o `Is` palavra-chave ou segui-lo com um operador de comparação.

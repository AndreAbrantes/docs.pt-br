# <a name="required-attribute-version-missing-from-xml-declaration"></a>O atributo 'version' obrigatório está ausente da declaração XML

O `version` atributo de um literal de documento XML está ausente.  
  
 **ID do erro:** BC31153  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione o `version` atributo à declaração de documento XML, como no exemplo a seguir: `<?xml version="1.0"?>`

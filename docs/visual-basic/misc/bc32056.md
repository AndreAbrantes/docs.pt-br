<span data-ttu-id="a9d30-103">Um `Implements` cláusula em um tipo genérico especifica um parâmetro de tipo como o membro a ser implementada.</span><span class="sxs-lookup"><span data-stu-id="a9d30-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="a9d30-104">Um `Implements` cláusula deve especificar uma interface e um membro.</span><span class="sxs-lookup"><span data-stu-id="a9d30-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="a9d30-105">Ele pode passar um parâmetro de tipo para a interface, mas ele não pode passá-lo para o membro ou usá-lo como o nome do membro.</span><span class="sxs-lookup"><span data-stu-id="a9d30-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>

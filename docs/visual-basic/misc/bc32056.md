<span data-ttu-id="e74f5-103">Um `Implements` cláusula em um tipo genérico especifica um parâmetro de tipo como o membro a ser implementada.</span><span class="sxs-lookup"><span data-stu-id="e74f5-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="e74f5-104">Um `Implements` cláusula deve especificar uma interface e um membro.</span><span class="sxs-lookup"><span data-stu-id="e74f5-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="e74f5-105">Ele pode passar um parâmetro de tipo para a interface, mas ele não pode passá-lo para o membro, nem usá-lo como o nome do membro.</span><span class="sxs-lookup"><span data-stu-id="e74f5-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="e74f5-106">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="e74f5-106">The following statements can generate this error.</span></span>

```  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="e74f5-107">**ID do erro:** BC32056</span><span class="sxs-lookup"><span data-stu-id="e74f5-107">**Error ID:** BC32056</span></span>

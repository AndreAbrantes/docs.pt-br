# <a name="type-parameter-not-allowed-in-39implements39-clause"></a><span data-ttu-id="74f68-102">Parâmetro de tipo não permitido em &#39;implementa&#39; cláusula</span><span class="sxs-lookup"><span data-stu-id="74f68-102">Type parameter not allowed in &#39;Implements&#39; clause</span></span>
<span data-ttu-id="74f68-103">Um `Implements` cláusula em um tipo genérico especifica um parâmetro de tipo como o membro a ser implementada.</span><span class="sxs-lookup"><span data-stu-id="74f68-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="74f68-104">Um `Implements` cláusula deve especificar uma interface e um membro.</span><span class="sxs-lookup"><span data-stu-id="74f68-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="74f68-105">Ele pode passar um parâmetro de tipo para a interface, mas ele não pode passá-lo para o membro, nem usá-lo como o nome do membro.</span><span class="sxs-lookup"><span data-stu-id="74f68-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="74f68-106">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="74f68-106">The following statements can generate this error.</span></span>

```  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```

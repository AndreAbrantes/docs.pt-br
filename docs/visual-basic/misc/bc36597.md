# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>Ir para o rótulo &#39; &lt;labelname&gt; &#39; não pode ser usado dentro de expressões lambda
O corpo de uma expressão lambda deve ser uma expressão que calcula e retorna um único valor. Não há instruções são permitidas, incluindo `GoTo` instruções.  
  
 **ID do erro:** BC36597

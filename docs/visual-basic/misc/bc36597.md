# <a name="goto-to-label-ltlabelnamegt-cannot-be-used-within-lambda-expressions"></a>Ir para o rótulo '&lt;labelname&gt;' não pode ser usado dentro de expressões lambda
O corpo de uma expressão lambda deve ser uma expressão que calcula e retorna um único valor. Não há instruções são permitidas, incluindo `GoTo` instruções.  
  
 **ID do erro:** BC36597

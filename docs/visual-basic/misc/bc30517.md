# <a name="overload-resolution-failed-because-no-39ltmethodgt39-is-accessible"></a>Resolução de sobrecarga falhou porque nenhum &#39; &lt;método&gt; &#39; está acessível
Você fez uma chamada para um método sobrecarregado, mas o compilador não é possível encontrar um método acessível com uma lista de parâmetros que faz a correspondência em termos de número de parâmetros.  
  
 **ID do erro:** BC30517  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique a lista de argumentos que você forneceu.  
  
2.  Verifique o método que você está tentando chamar para determinar se ele está acessível.

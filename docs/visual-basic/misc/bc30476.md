# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>Variáveis 'WithEvents' não podem ser digitadas como matrizes
Você tentou declarar uma matriz usando `WithEvents`. Você pode declarar como muitas variáveis individuais que você goste `WithEvents`, mas você não pode declarar matrizes dessa maneira.  
  
 **ID do erro:** BC30476  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Declare as variáveis individualmente.

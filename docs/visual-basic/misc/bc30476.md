# <a name="39withevents39-variables-cannot-be-typed-as-arrays"></a>&#39;WithEvents&#39; variáveis não podem ser digitadas como matrizes
Você tentou declarar uma matriz usando `WithEvents`. Você pode declarar como muitas variáveis individuais que você como usando `WithEvents`, mas você não pode declarar matrizes dessa maneira.  
  
 **ID do erro:** BC30476  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Declare as variáveis individualmente.

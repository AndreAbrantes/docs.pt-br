# <a name="type-expected"></a>Tipo esperado
Caracteres de código fonte ocorrerem, que não são reconhecidos como um nome de tipo. O contexto de chamadas para um tipo de dados elementar ou um nome de elemento declarado representando um tipo.  
  
 **ID do erro:** BC30182  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o tipo foi definido.  
  
-   Verifique se o nome do tipo está escrito corretamente.

# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39; ByVal &#39; e &#39; ByRef &#39; não podem ser combinados
O `ByVal` e `ByRef` palavras-chave são mutuamente exclusivas.  
  
 **ID do erro:** BC30641  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique o `ByVal` ou `ByRef`, mas não ambos.

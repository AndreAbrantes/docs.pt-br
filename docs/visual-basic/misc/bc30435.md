# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Membros em uma estrutura não podem ser declarados como '\<especificador >'
Você usou um especificador que não é válido em um membro dentro de um `Structure` instrução.  
  
 **ID do erro:** BC30435  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

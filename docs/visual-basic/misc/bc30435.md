# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Membros em uma estrutura não podem ser declarados &#39; &lt;especificador&gt;&#39;
Você usou um especificador que não é válido em um membro em um `Structure` instrução.  
  
 **ID do erro:** BC30435  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o especificador.

# <a name="argument-value-39ltpathnamegt39-contains-characters-that-are-not-valid-in-a-path-name"></a>O valor do argumento '&lt;pathname&gt;' contém caracteres que não são válidos em um nome de caminho
Um nome de caminho contém um ou mais caracteres que não são válidos.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o caractere do nome do caminho.

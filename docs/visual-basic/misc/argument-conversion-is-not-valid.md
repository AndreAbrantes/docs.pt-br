# <a name="argument-conversion-is-not-valid"></a>O argumento 'Conversion' não é válido
Um valor para o `VbStrConv` enumeração não é válida, possivelmente porque ele é menor que zero. Valores de enumeração devem ser maior que ou igual a zero.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Certifique-se de que o valor é maior que ou igual a zero.

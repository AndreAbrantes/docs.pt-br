# <a name="comma-or--expected-type-argument-list"></a>Vírgula ou ')' esperado (lista de argumentos de tipo)
Uma lista de argumentos de tipo contém um elemento inesperado. Use vírgulas (`,`) para separar os argumentos de tipo em uma lista desse tipo e use um parêntese de fechamento (`)`) para finalizar a lista.  
  
 **ID do erro:** BC32094  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione uma vírgula e continuar a lista de argumentos de tipo ou adicionar um fechamento de parêntese de abertura para finalizar a lista.

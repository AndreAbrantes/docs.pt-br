# <a name="comma-or-3939-expected-type-argument-list"></a>Vírgula ou &#39;) &#39; esperado (lista de argumentos de tipo)
Uma lista de argumentos de tipo contém um elemento inesperado. Use vírgulas (`,`) para separar os argumentos de tipo na lista e use um parêntese de fechamento (`)`) para finalizar a lista.  
  
 **ID do erro:** BC32094  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione uma vírgula e continuar a lista de argumentos de tipo ou adicionar um fechamento parêntese de abertura para finalizar a lista.

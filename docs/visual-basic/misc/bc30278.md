# <a name="sub-or-function-expected-after-delegate"></a>'Sub' ou 'Function' esperado após 'Delegate'
Um `Delegate` instrução não especifica um `Sub` ou `Function` procedimento. O `Sub` ou `Function` palavra-chave deve vir logo após o `Delegate` palavra-chave.  
  
 **ID do erro:** BC30278  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Adicione a `Sub` ou `Function` palavra-chave imediatamente após `Delegate`.  
  
2.  Especifique um nome do procedimento, uma lista de argumentos e retornar o tipo, conforme apropriado.

# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' error ' não são válidas na janela Immediate
`Error` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30724  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Error` instruções do código de depuração.

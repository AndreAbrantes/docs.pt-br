# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Erro &#39; instruções não são válidas na janela imediata
`Error`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30724  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Error` instruções do código de depuração.

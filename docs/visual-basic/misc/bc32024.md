# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>Valores padrão não podem ser fornecidos para parâmetros não declarados &#39; opcional &#39;
Os argumentos necessários devem ser fornecidos quando o procedimento é chamado, e eles não têm valores padrão na declaração.  
  
 **ID do erro:** BC32024  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o argumento seja `Optional` ou remova o valor padrão.

# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Referência necessária para o assembly &#39; &lt;assemblyname&gt; &#39; que contém a interface implementada &#39; &lt;interfacename&gt;&#39;
Referência necessária para o assembly '\<assemblyname >' que contém a interface implementada '\<interfacename >'. Adicione uma ao seu projeto.  
  
 A interface é definida em uma biblioteca de vínculo dinâmico (DLL) ou assembly que não é referenciado diretamente em seu projeto. O compilador do Visual Basic exige uma referência para evitar ambiguidades caso a interface é definida em mais de uma DLL ou assembly.  
  
 **ID do erro:** BC30009  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Inclua o nome da DLL ou assembly não referenciado em suas referências do projeto.

# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Método '\<methodname >' não pode ser declarado como 'Partial' porque somente um método '\<methodname >' pode ser marcado como 'Partial'
Você não pode ter duas declarações de método parcial que têm o mesmo nome e assinatura.  
  
 **ID do erro:** BC31433  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Altere o nome ou a assinatura de uma das declarações de método parcial.

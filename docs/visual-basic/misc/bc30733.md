# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<membername >' já foi declarado por '\<procedurename >', que foi gerado para este \<containername >
Um membro foi declarado que tem o mesmo nome que um membro que é declarado pelo tempo de execução de linguagem comom.  
  
 **ID do erro:** BC30733  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro para remover o conflito.

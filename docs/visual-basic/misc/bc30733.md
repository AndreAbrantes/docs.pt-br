# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39;&lt;membername&gt; &#39; já foi declarado por &#39; &lt;procedurename&gt;&#39;, que foi gerado para este &lt;containername&gt;
Um membro foi declarado que tem o mesmo nome como um membro que é declarado pelo tempo de execução de linguagem comom.  
  
 **ID do erro:** BC30733  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro para remover o conflito.

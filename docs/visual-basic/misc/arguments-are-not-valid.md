# <a name="arguments-are-not-valid"></a>Argumentos não são válidos
Argumentos que não são válidos estão sendo passados para uma função ou sub-rotina.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique os argumentos para determinar por que eles são inválidos.  
  
2.  Se os argumentos são fornecidos pelo cálculos, verifique os valores que processam os cálculos.

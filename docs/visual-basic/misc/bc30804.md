# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39;Variant&#39; não é um tipo com suporte; Use o &#39;objeto&#39; em vez disso, digite
O `Object` substitui o tipo de dados `Variant`.  
  
 **ID do erro:** BC30804  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declarar o item como `Object`.

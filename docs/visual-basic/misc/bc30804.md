# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>'Variant' não é mais um tipo com suporte; Use o tipo 'Object'
O `Object` substitui o tipo de dados `Variant`.  
  
 **ID do erro:** BC30804  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declarar o item como `Object`.

# <a name="statement-is-not-valid-in-an-interface"></a>Instrução não é válida em uma interface
Uma instrução inválida foi encontrada na interface.  
  
 **ID do erro:** BC31041  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que instruções na interface definem elementos, mas não fornecem implementação.

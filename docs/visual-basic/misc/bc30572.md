# <a name="alias-39ltnamespacenamegt39-is-already-declared"></a>Alias &#39; &lt;namespacename&gt; &#39; já está declarado
Você tentou usar `Imports` para importar um namespace que já está declarado.  
  
 **ID do erro:** BC30572  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o namespace que você está tentando importar.

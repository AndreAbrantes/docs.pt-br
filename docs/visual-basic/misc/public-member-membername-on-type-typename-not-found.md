# <a name="public-member-membername-on-type-typename-not-found"></a>Membro público '\<membername >' no tipo '\<typename >' não encontrado
O membro que você está tentando acessar não foi encontrado. Esse erro pode ocorrer ao acessar membros não públicos dos objetos de associação tardia.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Se o membro existe, certifique-se de que ele é declarado como `Public`.

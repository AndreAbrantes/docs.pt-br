# <a name="public-member-39ltmembernamegt39-on-type-39lttypenamegt39-not-found"></a>Membro público &#39; &lt;membername&gt;&#39; no tipo &#39;&lt; TypeName&gt;&#39; não encontrado
O membro que você está tentando acessar não foi encontrado. Esse erro pode ocorrer ao acessar membros não-públicos de objetos de associação tardia.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Se o membro existe, certifique-se de que ele é declarado como `Public`.

# <a name="withevents-variable-does-not-raise-any-events"></a>Variável 'WithEvents' não gera nenhum evento
A variável de objeto declarada com o `WithEvents` palavra-chave não faz referência a uma instância da classe que pode gerar eventos.  
  
 **ID do erro:** BC30591  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Certifique-se de que a variável de objeto se refere a uma instância da classe que pode gerar eventos.

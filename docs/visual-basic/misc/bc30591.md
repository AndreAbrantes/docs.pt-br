# <a name="withevents-variable-does-not-raise-any-events"></a>A variável ' WithEvents ' não gera nenhum evento
A variável de objeto declarada com a `WithEvents` palavra-chave não se refere a uma instância de classe que pode gerar eventos.  
  
 **ID do erro:** BC30591  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Certifique-se de que a variável de objeto se refere a uma instância de classe que pode gerar eventos.

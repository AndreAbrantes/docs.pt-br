# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Resume' não são válidas na janela imediata
Instruções de retomar somente podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30714  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Resume` instruções de depuração de código.

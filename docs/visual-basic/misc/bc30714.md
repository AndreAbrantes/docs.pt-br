# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Reiniciar &#39; instruções não são válidas na janela imediata
Instruções Resume só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30714  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Resume` instruções do código de depuração.

# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Instruções ' Resume ' não são válidas na janela Immediate
As instruções resume só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30714  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova `Resume` instruções do código de depuração.

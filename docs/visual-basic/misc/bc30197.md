# <a name="39as39-expected"></a>&#39; Como &#39; esperado
Ocorre uma declaração que não fornece um `As` cláusula para um ou mais dos elementos que está sendo declarados.  
  
 **ID do erro:** BC30197  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça o `As` cláusula conforme necessário, ou ative `Option Strict Off`.

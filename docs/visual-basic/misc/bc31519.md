# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>&#39; &lt;filename&gt;&#39; não pode ser referenciado porque não é um assembly
Uma referência foi feita para um arquivo que não é um assembly .NET.  
  
 **ID do erro:** BC31519  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o nome e o local do assembly.  
  
2.  Verifique se o assembly não está danificado ou corrompido.

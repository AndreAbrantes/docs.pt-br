# <a name="ltfilenamegt-cannot-be-referenced-because-it-is-not-an-assembly"></a>'&lt;filename&gt;' não pode ser referenciado porque ele não é um assembly
Uma referência foi feita para um arquivo que não é um assembly .NET.  
  
 **ID do erro:** BC31519  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o nome e local do assembly.  
  
2.  Verifique se o assembly não está danificado ou corrompido.

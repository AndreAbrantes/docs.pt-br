# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>A referência de assembly padrão '\<referência >' tem já foi adicionada automaticamente, ignorado
Foi feita uma tentativa de adicionar a referência de assembly padrão, mas o assembly já foi adicionado, portanto, a tentativa é ignorada.  
  
 **ID do erro:** BC2024  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o código supérfluo.

# <a name="statement-cannot-appear-within-a-method-body"></a>A instrução não pode aparecer dentro de um corpo de método
Um procedimento contiver uma instrução inválida, como outra declaração de procedimento. Declarações de procedimento não podem ser aninhadas.  
  
 **ID do erro:** BC30289  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Remova a declaração de procedimento ou movê-lo fora do procedimento.

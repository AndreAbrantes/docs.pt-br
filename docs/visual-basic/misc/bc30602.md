# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Membros de interface devem ser métodos, propriedades, eventos ou definições de tipo
Aninhados apenas tipos, métodos, propriedades e eventos podem ser membros de uma interface.  
  
 **ID do erro:** BC30602  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se que o item é um método, propriedade, evento ou um tipo aninhado, como uma enumeração.

# <a name="if-operator-requires-either-two-or-three-operands"></a>Operador 'If' requer dois ou três operandos
O `If` operador aceita apenas dois ou três argumentos.  
  
 **ID do erro:** BC33104  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Examinar o `If` expressão para ver se o número de operadores pode ser reduzido ou estendido para dois ou três argumentos.  
  
- Considere usar outra construção condicional, como um `If...Then...Else` instrução ou um aninhada `If...Then...Else` instrução para organizar os argumentos.

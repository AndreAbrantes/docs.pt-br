# <a name="cannot-convert-to-type"></a>Não é possível converter para '\<tipo >'
Foi tentada uma conversão inválida.  
  
 **ID do erro:** BC30748  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Escolha um tipo de conversão compatível.

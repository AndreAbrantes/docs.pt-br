# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>Nome de recurso &#39; &lt;resourcename&gt;&#39; não pode ser usado mais de uma vez
O nome de um recurso especificado em uma biblioteca de vínculo dinâmico ou arquivo executável foi usado mais de uma vez.  
  
 **ID do erro:** BC31502  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Recursos de nome de forma exclusiva.

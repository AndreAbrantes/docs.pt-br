# <a name="39nothing39-cannot-be-evaluated"></a>&#39; Nada &#39; não pode ser avaliada
O `Nothing` palavra-chave que foi usada na janela de depuração.  
  
 **ID do erro:** BC30701  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não use `Nothing` durante a depuração.

# <a name="nothing-cannot-be-evaluated"></a>'Nothing' não pode ser avaliada
O `Nothing` palavra-chave que foi usada na janela de depuração.  
  
 **ID do erro:** BC30701  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não use `Nothing` durante a depuração.

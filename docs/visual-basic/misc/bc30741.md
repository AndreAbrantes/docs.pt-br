# <a name="39lttypename1gt39-cannot-be-converted-to-39lttypename2gt39"></a>'&lt;typename1&gt;'não pode ser convertido em'&lt;typename2&gt;'
Foi tentada uma conversão de tipos entre dois tipos incompatíveis.  
  
 **ID do erro:** BC30741  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se os tipos que estão sendo convertidos.

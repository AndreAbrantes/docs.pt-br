# <a name="typename1-cannot-be-converted-to-typename2"></a>'\<typename1 >' não pode ser convertido em '\<typename2 >'
Foi tentada uma conversão de tipo entre dois tipos incompatíveis.  
  
 **ID do erro:** BC30741  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se os tipos que estão sendo convertidos.

# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>O número máximo de erros foi excedido
O máximo de compilador de 100 erros foi alcançado.  
  
 **ID do erro:** BC2020  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Diminua o número de erros gerados em depuração e endereçamento isoladamente.

# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39;Interface&#39; deve terminar com uma correspondência &#39;End Interface&#39;
Um `Interface` instrução ocorre sem correspondente `End Interface` instrução. Você deve usar um `End Interface` instrução para finalizar a definição de interface.  
  
 **ID do erro:** BC30253  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se o `Interface` membros estão formatados corretamente.  
  
2.  Adicionar um `End Interface` instrução no fim da definição da interface.

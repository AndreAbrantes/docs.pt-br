# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>Marca de comentário XML &#39;incluem&#39; deve ter um &#39; &lt;atributo&gt; &#39; atributo
A marca de comentário XML 'include' deve ter um \<atributo > atributo. O comentário XML será ignorado.  
  
 Um `file` ou `path` atributo, passado para o `<`incluem`>` marca, estava ausente ou incompleta.  
  
 **ID do erro:** BC42310  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione ou complete o atributo obrigatório para o \<incluem > marca.

# <a name="method-39ltmethodnamegt39-does-not-have-the-same-signature-as-delegate-39ltdelegatenamegt39"></a>Método &#39; &lt;methodname&gt;&#39; não tem a mesma assinatura do delegado &#39;&lt; delegateName&gt;&#39;
Há uma incompatibilidade entre as assinaturas de método e do delegado que você está tentando usar. O `Delegate` instrução define os tipos de parâmetro e tipos de retorno de uma classe delegate. Qualquer procedimento com parâmetros, tipos e tipo de retorno de correspondência pode ser usado para criar uma instância do tipo delegado. Cada classe de delegado define um construtor que é passado para a especificação de um método do objeto.  
  
 **ID do erro:** BC30408  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique as assinaturas para encontrar a incompatibilidade e corrigi-lo.

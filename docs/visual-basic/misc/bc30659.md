# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>Propriedade ou campo &#39; &lt;nome&gt; &#39; não tem um tipo de atributo válido
Talvez você esteja usando um construtor de atributo que não aceita um atributo válido de tipo, mesmo que o atributo tenha outros construtores apropriados.  
  
 **ID do erro:** BC30659  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique o uso deste atributo.

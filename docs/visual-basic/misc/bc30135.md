# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39;SyncLock&#39; instruções não são válidas na janela imediata
O `SyncLock` instrução sincroniza segmentos e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30135  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita um `SyncLock` instrução o **imediato** janela.

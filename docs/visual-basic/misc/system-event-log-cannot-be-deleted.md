# <a name="system-event-log-cannot-be-deleted"></a>Log de eventos do sistema não pode ser excluído.
Tentativa de excluir o log de eventos do sistema, não pode ser excluído. O log do sistema rastreia eventos do sistema, como falhas de hardware e de inicialização do sistema.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Considere fazer com que o seu aplicativo gravar em um aplicativo ou log personalizado, em vez do log de eventos do sistema.  
  
-   Não tente excluir o log de eventos do sistema.

# <a name="xml-comment-type-parameter-must-have-a-39name39-attribute"></a>O parâmetro de tipo de comentário XML deve ter um atributo 'name'
Parâmetro de tipo de comentário XML deve ter um atributo "name". O comentário XML será ignorado.  
  
 A marca de comentário XML requer um `name` atributo, que está ausente ou incompleta.  
  
 **ID do erro:** BC42318  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione ou complete necessários `name` atributo à marca.

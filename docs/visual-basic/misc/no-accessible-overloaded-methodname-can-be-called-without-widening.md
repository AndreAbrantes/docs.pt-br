# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>Não acessível sobrecarregado &#39; &lt;methodname&gt;&#39; pode ser chamado com estes argumentos sem uma conversão de ampliação: &lt;lista&gt;
Um método sobrecarregado foi chamado, mas nenhum método foi compatível com a lista dos argumentos fornecidos sem uma conversão de ampliação.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique `Option Strict Off`.  
  
-   Altere os argumentos para coincidir com as assinaturas do método sobrecarregado.

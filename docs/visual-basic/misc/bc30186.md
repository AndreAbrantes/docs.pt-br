# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'Using' não são válidas na janela imediata
`Using` instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30186  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover o `Using` instrução from a **imediato** janela.

# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>Linha \<número > não pode ser analisada usando os Delimiters atuais
A linha especificada não pode ser analisada porque ele usa delimitadores que não sejam aqueles especificados.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Ajustar `Delimiters` para que a linha possa ser processada corretamente, ou insira o código de manipulação de exceção para lidar com a linha.

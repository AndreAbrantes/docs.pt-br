# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>'&lt;property1&gt;'e'&lt;property2&gt;' não podem sobrecarregar um ao outro porque somente um está declarado como 'Default'
Se uma propriedade que especifica `Default`, todas as propriedades sobrecarregadas nesse nome também devem especificar `Default`.  
  
 **ID do erro:** BC30361  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de todas as propriedades sobrecarregadas são declaradas `Default`.

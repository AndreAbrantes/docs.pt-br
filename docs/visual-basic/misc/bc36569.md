# <a name="argument-not-specified-for-parameter-parametername"></a>Argumento não especificado para o parâmetro '\<parametername >'
Um argumento para um parâmetro necessário não foi fornecido.  
  
 **ID do erro:** BC36569  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Fornece um argumento para o parâmetro especificado.

# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>Argumento não especificado para o parâmetro '&lt;parametername&gt;'
Um argumento para um parâmetro necessário não foi fornecido.  
  
 **ID do erro:** BC36569  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Forneça um argumento para o parâmetro especificado.

# <a name="you-must-specify-a-file-name"></a>Você deve especificar um nome de arquivo
Você tentou uma operação, como mover ou copiar um arquivo, que requer que um nome de arquivo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Forneça o nome de arquivo necessário.  
  
- Verifique se o caminho não termina com um caractere separador de diretório (\\).

### <a name="to-create-a-data-bound-datarepeater"></a><span data-ttu-id="4e826-109">Para criar um associação de dados DataRepeater</span><span class="sxs-lookup"><span data-stu-id="4e826-109">To create a data-bound DataRepeater</span></span>  
  
1.  <span data-ttu-id="4e826-110">Arraste um <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controlar do **Visual Basic PowerPacks** guia o **caixa de ferramentas** a um controle de formulário ou contêiner.</span><span class="sxs-lookup"><span data-stu-id="4e826-110">Drag a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control from the **Visual Basic PowerPacks** tab in the **Toolbox** to a form or container control.</span></span>  
  
2.  <span data-ttu-id="4e826-111">Arraste as alças de dimensionamento e a posição para o tamanho e posição do controle.</span><span class="sxs-lookup"><span data-stu-id="4e826-111">Drag the sizing and position handles to size and position the control.</span></span>  
  
     <span data-ttu-id="4e826-112">Observe que o controle tem duas regiões retangulares.</span><span class="sxs-lookup"><span data-stu-id="4e826-112">Note that the control has two rectangular regions.</span></span> <span data-ttu-id="4e826-113">A região superior é o *modelo de item*; controles adicionados para o modelo serão repetidos em cada item a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controle em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="4e826-113">The upper region is the *item template*; controls added to the template will be repeated in each item in the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control at run time.</span></span> <span data-ttu-id="4e826-114">A região inferior é o *visor*, onde os itens serão exibidos.</span><span class="sxs-lookup"><span data-stu-id="4e826-114">The lower region is the *viewport*, where the items will be displayed.</span></span>  
  
     <span data-ttu-id="4e826-115">Você também pode dimensionar e posicionar o controle ou o modelo de item, alterando o **tamanho** e **posição** propriedades na janela Propriedades.</span><span class="sxs-lookup"><span data-stu-id="4e826-115">You can also size and position the control or the item template by changing the **Size** and **Position** properties in the Properties window.</span></span>  
  
3.  <span data-ttu-id="4e826-116">Sobre o **dados** menu, clique em **Mostrar fontes de dados**.</span><span class="sxs-lookup"><span data-stu-id="4e826-116">On the **Data** menu, click **Show Data Sources**.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="4e826-117">Se o **fontes de dados** janela estiver vazia, adicionar uma fonte de dados a ele.</span><span class="sxs-lookup"><span data-stu-id="4e826-117">If the **Data Sources** window is empty, add a data source to it.</span></span> <span data-ttu-id="4e826-118">Para obter mais informações, consulte [Adicionar novas fontes de dados](/visualstudio/data-tools/add-new-data-sources).</span><span class="sxs-lookup"><span data-stu-id="4e826-118">For more information, see [Add new data sources](/visualstudio/data-tools/add-new-data-sources).</span></span>  
  
4.  <span data-ttu-id="4e826-119">No **fontes de dados** janela, selecione o nó de nível superior para a tabela que contém os dados que você deseja vincular.</span><span class="sxs-lookup"><span data-stu-id="4e826-119">In the **Data Sources** window, select the top-level node for the table that contains the data that you want to bind.</span></span>  
  
5.  <span data-ttu-id="4e826-120">Altere o tipo subjacente da tabela para `Details` clicando `Details` na lista suspensa no nó da tabela.</span><span class="sxs-lookup"><span data-stu-id="4e826-120">Change the drop type of the table to `Details` by clicking `Details` in the drop-down list on the table node.</span></span>  
  
6.  <span data-ttu-id="4e826-121">Selecione o nó de tabela e arraste-o para a região de modelo de item do <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controle.</span><span class="sxs-lookup"><span data-stu-id="4e826-121">Select the table node and drag it onto the item template region of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span>  
  
     <span data-ttu-id="4e826-122">Você pode especificar quais tipos de controles são exibidos para cada campo.</span><span class="sxs-lookup"><span data-stu-id="4e826-122">You can specify which types of controls are displayed for each field.</span></span> <span data-ttu-id="4e826-123">Para obter mais informações, consulte [definir o controle a ser criado quando arrastado da janela fontes de dados](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span><span class="sxs-lookup"><span data-stu-id="4e826-123">For more information, see [Set the control to be created when dragging from the Data Sources window](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span></span>

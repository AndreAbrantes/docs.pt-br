    > <span data-ttu-id="2e9b4-111">Nem todo assembly tem um arquivo de configuração.</span><span class="sxs-lookup"><span data-stu-id="2e9b4-111">Not every assembly has a configuration file.</span></span>

    - <span data-ttu-id="2e9b4-131">Os ouvintes <xref:System.Diagnostics.DelimitedListTraceListener?displayProperty=nameWithType> e <xref:System.Diagnostics.XmlWriterTraceListener?displayProperty=nameWithType> gravam no arquivo especificado no parâmetro `initializeData`.</span><span class="sxs-lookup"><span data-stu-id="2e9b4-131">The <xref:System.Diagnostics.DelimitedListTraceListener?displayProperty=nameWithType> and <xref:System.Diagnostics.XmlWriterTraceListener?displayProperty=nameWithType> listeners write to the file specified in the `initializeData` parameter.</span></span>

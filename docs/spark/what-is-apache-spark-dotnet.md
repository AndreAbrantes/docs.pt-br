# <a name="what-is-net-for-apache-spark"></a>O que é o .NET para o Apache Spark?

[Apache Spark](https://spark.apache.org/) é um mecanismo de processamento distribuído de finalidade geral para análise em conjuntos de dados grandes – normalmente terabytes ou petabytes de dados. O .NET para Apache Spark torna Apache Spark acessível para desenvolvedores, trazendo o poder do Spark para seus aplicativos com suporte para idiomas que você já conhece.

Com C# o F#e o, você pode acessar:

* Dataframe e SparkSQL para trabalhar com dados estruturados.
* Streaming estruturado do Spark para trabalhar com dados de streaming.

O .NET para Apache Spark é compatível com o .NET Standard, uma especificação formal das APIs do .NET que são comuns em implementações do .NET. Isso significa que você pode usar o .NET para Apache Spark em qualquer lugar em que você escrever código .NET, permitindo que você reutilize todo o conhecimento, as habilidades, o código e as bibliotecas que você já tem como um desenvolvedor do .NET.

O .NET para Apache Spark é executado no Windows, no Linux e no macOS usando o .NET Core. Ele também é executado no Windows usando .NET Framework. Você pode implantar seus aplicativos em todos os principais provedores de nuvem, incluindo Azure HDInsight Spark, Amazon EMR Spark, Azure Databricks e databricks no AWS.

## <a name="net-foundation"></a>.NET Foundation

O projeto .NET para Apache Spark faz parte do [.net Foundation](https://www.dotnetfoundation.org/).

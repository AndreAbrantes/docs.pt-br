# <a name="net-for-apache-spark-tutorials"></a>Tutoriais do .NET para Apache Spark

Com os tutoriais a seguir, você aprenderá a usar o [.NET para Apache Spark](../index.yml) para criar soluções de Big Data personalizadas com o Apache Spark e integrá-las aos seus aplicativos .NET:

* [Introdução](get-started.md): demonstra o build e a execução de um aplicativo .NET para Apache Spark simples.
* [Implantar um aplicativo .NET para Apache Spark no Azure HDInsight](hdinsight-deployment.md): demonstra como implantar seu aplicativo .NET para Apache Spark no Azure HDInsight.
* [Implantar um aplicativo .NET para Apache Spark no Amazon EMR Spark](amazon-emr-spark-deployment.md): demonstra como implantar seu aplicativo .NET para Apache Spark no Amazon EMR Spark.
* [Implantar um aplicativo .NET para Apache Spark no Databricks](databricks-deployment.md): demonstra como implantar seu aplicativo .NET para Apache Spark no Databricks.

## <a name="next-steps"></a>Próximas etapas

## <a name="github-issues-and-announcements"></a>Problemas e anúncios do GitHub

Você também pode visualizar problemas individuais que detalham as alterações de quebra introduzidas no .NET Core nos seguintes repositórios do GitHub:

- Para .NET Core, o repositório [dotnet/docs.](https://github.com/dotnet/docs/issues?q=is%3Aissue+label%3Abreaking-change)
- Para ASP.NET Core, o repositório [aspnet/Announcements.](https://github.com/aspnet/Announcements/issues?q=is%3Aissue+is%3Aopen+label%3A%22Breaking+change%22+label%3A3.0.0)
- Para Entity Framework Core, o repositório [dotnet/efcore.](https://github.com/dotnet/efcore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change)

# <a name="download-and-install-net-core"></a>Baixar e instalar o .NET Core

Este artigo fornece informações sobre como baixar e instalar o .NET Core. Existem duas partes para o .NET Core, o **tempo de execução** que é usado para executar aplicativos, e o **SDK** que é usado para criar aplicativos. O SDK inclui o tempo de execução.

- Se você é um usuário que precisa do .NET Core para executar aplicativos, consulte [Instale o .NET Core Runtime](runtime.md).

- Se você é um desenvolvedor que precisa do .NET Core para criar aplicativos, consulte [o Install the .NET Core SDK](sdk.md).

> [!NOTE]
> O .NET Core SDK inclui o tempo de execução.

## <a name="dependencies"></a>Dependências

.NET Core é suportado no Windows, Linux e macOS. Para obter uma lista completa de requisitos, consulte [sistemas operacionais suportados](dependencies.md).

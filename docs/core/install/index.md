# <a name="download-and-install-net-core"></a>Baixar e instalar o .NET Core

Este artigo fornece informações sobre como baixar e instalar o .NET Core. Há duas partes no .NET Core, o **tempo de execução** usado para executar aplicativos e o **SDK** que é usado para criar aplicativos. O SDK inclui o tempo de execução.

- Se você for um usuário que precisa do .NET Core para executar aplicativos, consulte [instalar o tempo de execução do .NET Core](runtime.md).

- Se você for um desenvolvedor que precisa do .NET Core para criar aplicativos, consulte [instalar o SDK do .NET Core](sdk.md).

> [!NOTE]
> O SDK do .NET Core inclui o tempo de execução.

## <a name="dependencies"></a>Dependências

O .NET Core tem suporte no Windows, Linux e macOS. Para obter uma lista completa dos requisitos, consulte [sistemas operacionais com suporte](dependencies.md).

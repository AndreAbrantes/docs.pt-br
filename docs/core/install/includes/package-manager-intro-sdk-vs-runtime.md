Instale o SDK (que inclui o tempo de execução) se você quiser desenvolver aplicativos .NET Core. Ou, se você só precisa executar aplicativos, instale o tempo de execução. Se você estiver instalando o tempo de execução, sugerimos que você instale o [tempo de execução do ASP.NET Core,](#install-the-aspnet-core-runtime)pois inclui os tempos de execução do .NET Core e do ASP.NET Core.

Se você já instalou o SDK ou `dotnet --list-sdks` o `dotnet --list-runtimes` tempo de execução, use os comandos e para ver quais versões estão instaladas. Para obter mais informações, consulte [Como verificar se o .NET Core já está instalado](../how-to-detect-installed-versions.md?pivots=os-linux).

Ao instalar o pacote do .NET Core, você poderá ver um erro semelhante a `signature verification failed for file 'repomd.xml' from repository 'packages-microsoft-com-prod'`. Em termos gerais, esse erro significa que o feed de pacote do .NET Core está sendo atualizado com versões mais recentes do pacote e que você deve tentar novamente mais tarde. Durante uma atualização, o feed de pacote não deve ficar indisponível por mais de duas horas. Se você receber esse erro continuamente por mais de 2 horas, registre um problema em <https://github.com/dotnet/core/issues>.

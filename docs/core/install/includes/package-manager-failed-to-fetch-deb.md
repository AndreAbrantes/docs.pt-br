Ao instalar o pacote do .NET Core, você poderá ver um erro semelhante a `Failed to fetch ... File has unexpected size ... Mirror sync in progress?`. Em termos gerais, esse erro significa que o feed de pacote do .NET Core está sendo atualizado com versões mais recentes do pacote e que você deve tentar novamente mais tarde. Durante uma atualização, o feed de pacote não deve ficar indisponível por mais de 30 minutos. Se você receber esse erro continuamente por mais de 30 minutos, registre um problema em <https://github.com/dotnet/core/issues>.
